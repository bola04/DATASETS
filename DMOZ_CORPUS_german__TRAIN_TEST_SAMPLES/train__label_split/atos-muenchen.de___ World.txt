       <h1 class="">
        ATOS Klinik München – Herzlich willkommen auf unserer Webseite!
       </h1>
       <p>
        Seit 2008 bietet die ATOS Klinik München Spitzenmedizin gepaart mit Spitzenservice für privatversicherte Patienten – ganz gemäß des ATOS Konzepts, das 1991 in Heidelberg aus der Wiege gehoben wurde. Für unsere Patienten bedeutet dies, dass sie höchste Behandlungsqualität von ausgezeichneten Spezialisten erwarten können, ebenso wie umfassende, individuelle Pflege. Damit aber noch lange nicht genug: Anders als gewöhnliche Krankenhäuser bieten wir Ihnen in der ATOS Klinik München ein Höchstmaß an Komfort. Dies fängt bei der Zimmerausstattung an und geht von modernen medizinischen Geräten über höchste Hygienestandards bis hin zu einer freundlichen, harmonischen und respektvollen Atmosphäre. So soll sich ein Aufenthalt bei uns mehr wie in einem Hotel als in einer Klinik anfühlen. Die Idee dahinter ist ganz einfach: Wir möchten, dass Sie sich bei uns wohlfühlen – denn eine schnelle Genesung hängt nicht nur von exzellenter medizinischer Behandlung und herausragender Pflege ab, sondern auch vom seelischen Wohlbefinden.
       </p>

     function decryptCharcode(n,start,end,offset) {
         n = n + offset;
         if (offset > 0 && n > end) {
             n = start + (n - end - 1);
         } else if (offset < 0 && n < start) {

     function decryptString(enc,offset) {
         var dec = "";
         var len = enc.length;
         for(var i=0; i < len; i++) {
             var n = enc.charCodeAt(i);
             if (n >= 0x2B && n <= 0x3A) {
                 dec += decryptCharcode(n,0x2B,0x3A,offset);	// 0-9 . , - + / :
             } else if (n >= 0x40 && n <= 0x5A) {

             } else if (n >= 0x61 && n <= 0x7A) {

        <h1 class="">
         ATOS Klinik München – Herzlich willkommen auf unserer Webseite!
        </h1>
        <p>
         Seit 2008 bietet die ATOS Klinik München Spitzenmedizin gepaart mit Spitzenservice für privatversicherte Patienten – ganz gemäß des ATOS Konzepts, das 1991 in Heidelberg aus der Wiege gehoben wurde. Für unsere Patienten bedeutet dies, dass sie höchste Behandlungsqualität von ausgezeichneten Spezialisten erwarten können, ebenso wie umfassende, individuelle Pflege. Damit aber noch lange nicht genug: Anders als gewöhnliche Krankenhäuser bieten wir Ihnen in der ATOS Klinik München ein Höchstmaß an Komfort. Dies fängt bei der Zimmerausstattung an und geht von modernen medizinischen Geräten über höchste Hygienestandards bis hin zu einer freundlichen, harmonischen und respektvollen Atmosphäre. So soll sich ein Aufenthalt bei uns mehr wie in einem Hotel als in einer Klinik anfühlen. Die Idee dahinter ist ganz einfach: Wir möchten, dass Sie sich bei uns wohlfühlen – denn eine schnelle Genesung hängt nicht nur von exzellenter medizinischer Behandlung und herausragender Pflege ab, sondern auch vom seelischen Wohlbefinden.
        </p>

        <h2 class="">
         ATOS Check-Up – Prävention statt Reaktion
        </h2>
        <p>
         Zwar stehen Sie als Patient voll und ganz im Mittelpunkt unseres Tuns – noch lieber ist es uns allerdings, wenn Sie gar nicht erst zum Patienten werden. Aus diesem Grund bieten wir Ihnen in der ATOS Klinik München einen medizinischen Check-Up an, der Ihnen dabei hilft, gesund zu bleiben anstatt gesund zu werden. An unserem Check-Up-Programm sind Ärzte verschiedener Disziplinen beteiligt, sodass wir Ihnen eine ganzheitliche Rundum-Vorsorge anbieten können – mit viel Zeit für Sie und einem festen medizinischen Ansprechpartner.
         <br/>
         Um Jedem die passende Art von Gesundheitsvorsorge anbieten zu können, setzt sich unser Check-Up aus verschiedenen Modulen zusammen. Neben dem Basis-Check und dem erweiterten Check, ist es auch möglich, einzelne Bausteine individuell zusammenzustellen.
        </p>

         Finden Sie heraus, welche Untersuchungen Teil unseres Präventionsprogramms sind!
