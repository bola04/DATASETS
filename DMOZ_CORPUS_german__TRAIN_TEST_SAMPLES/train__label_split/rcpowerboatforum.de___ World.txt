            Hallo Powerboot-Freunde,
            <br/>
            <br/>
            nun ist es soweit: Das Jahr der Auszeit von Tom Lorenz ist zu Ende. Er wird im Juni seinen Shop unter
            <br/>
            <a href="http://www.modellbaucenterlorenz.de" target="_blank">
             www.modellbaucenterlorenz.de
            </a>
            wie vereinbart wieder übernehmen. Start des Vertriebes wird vermutlich die
            <br/>
            zweite Juni-Woche sein. Mit der Übergabe des Shops gehen aber auch einige Veränderungen einher. Dazu später
            <br/>
            mehr.
            <br/>
            <br/>
            Zunächst bedanke ich mich an dieser Stelle bei Tom. Die Chance, so einen tollen Shop zu managen, war eine
            <br/>
            einmalige Erfahrung und hat mich viel gelehrt. Ich konnte in diesem Jahr sehr viele nette Kontakte innerhalb
            <br/>
            unserer fantastischen Powerboot-Szene knüpfen. Vielen Dank auch an alle Kunden, die hin und wieder auf Ihre
            <br/>
            Bestellungen warten mussten. Im Nebenberuf ist der pünktliche Versand der Ware eines so großen Online-Shops
            <br/>
            nicht immer ganz einfach.
            <br/>
            <br/>
            Ferner konnte ich in diesem Jahr einen engen Kontakt zu MTC Marl-Hüls und insbesondere dessen Inhaber Rudi
            <br/>
            Kappe knüpfen. Rudi hat mittlerweile ein Alter erreicht, in dem er sich allmählich auf seinen Ruhestand
            <br/>
            vorbereiten möchte. In diesem Zusammenhang unterbreitete er mir das Angebot, die Marke MTC fortzuführen und
            <br/>
            seine Produkte zunächst zu vertreiben und später herstellen zu dürfen. Als leidenschaftlicher Modellbauer
            <br/>
            habe ich dieses Angebot natürlich gerne angenommen.
            <br/>
            <br/>
            Was heißt das für Euch:
            <br/>
            <br/>
            1. Es wird die Marke MTC auch nach 2019 weiter geben.
            <br/>
            2. MTC-Hardware wird ab dem 05.08.2018 wieder über
            <a href="http://www.mtc-powerboat.de" target="_blank">
             http://www.mtc-powerboat.de
            </a>
            vertrieben.
            <br/>
            3. Es ändert sich nichts an der Qualität. Rudi Kappe wird weiterhin die Teile produzieren. Ab 2019 werde ich,
            <br/>
            Christoph Raschke, in die Produktion mit einsteigen. Die Qualitätskontrolle liegt aber weiterhin bei Rudi.
            <br/>
            4. Gemeinsam werden wir dafür sorgen, dass MTC das bleibt, was es schon immer war: Qualität - Made in
            <br/>
            Germany. Ohne Kompromisse.
            <br/>
            5. Da ich auch MTC nur im Nebengewerbe betreiben werde, wird es nur zweimal wöchentlich einen Versand geben.
            <br/>
            6. In der Umstellungsphase kann und wird es zu Engpässen kommen. Wenn Ihr ein Projekt plant, solltet Ihr
            <br/>
            frühzeitig bestellen oder mir eine E-Mail an:
            <a href="mailto:info@mtc-powerboat.de">
             info@mtc-powerboat.de
            </a>
            schreiben.
            <br/>
            <br/>
            <br/>
            Viele Grüße
            <br/>
            <br/>
            Christoph Raschke
