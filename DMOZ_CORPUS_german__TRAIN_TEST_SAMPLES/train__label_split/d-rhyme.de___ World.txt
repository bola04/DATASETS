     <article class="dr-text dr-inner-container">
      <h1>
       We
       <span class="dr-org icon-dr-heart">
       </span>
       rhymes
      </h1>
      <h2>
       Reime suchen und finden!
      </h2>
      <p class="left">
       <em>
        Was reimt sich auf?
       </em>
       … Gute Frage, einfache Antwort!
       <strong class="plain">
        Reime
       </strong>
       finden leicht gemacht mit
       <span class="dr-tx">
        d-rhyme
       </span>
       , deinem
       <strong>
        Reimlexikon
       </strong>
       . Gib ein Wort ein, starte die
       <strong class="plain">
        Reimsuche
       </strong>
       und freue dich über eine Vielzahl gefundener
       <strong class="plain">
        Reimwörter
       </strong>
       , mit denen du
       <b>
        Verse
       </b>
       schmiedest, einen coolen
       <strong>
        Rhyme
       </strong>
       machst, zum Rapper wirst oder einfach deine
       <b>
        Gedichte
       </b>
       verschönerst.
      </p>
      <div class="dr-ball left">
      </div>
     </article>
     <!-- Animation -->
     <article class="dr-icon dr-inner-container">
      <div id="dr-icon-rhymesearch-container">
       <span class="dr-icon-inner" id="dr-icon-word">
        Suchbegriff
       </span>
       <span class="dr-icon-arrow-search dr-icon-inner left icon-dr-arrow-circle-right">
       </span>
       <span class="dr-icon-inner icon-dr-permute scrollme animateme" data-easing="linear" data-from="0" data-rotatez="360" data-to="1" id="dr-icon-word-search">
       </span>
       <span class="dr-icon-arrow-search dr-icon-inner right icon-dr-arrow-circle-right">
       </span>
       <span class="dr-icon-inner" id="dr-icon-searchresult">
        Reimwörter
       </span>
      </div>
     </article>
     <article class="dr-text dr-inner-container">
      <div class="dr-ball right">
      </div>
      <h6>
       … Wofür braucht man das?
      </h6>
      <p class="right">
       <strong class="plain">
        Reime
       </strong>
       eignen sich optimal dazu, deine selbstverfassten und kreativen Texte anzureichern. Egal, ob als Dichter oder Songschreiber, als Schriftsteller oder Texter in der Werbung – nutze
       <span class="dr-tx">
        d-rhyme
       </span>
       dazu, deine Kreativität zu unterstützen, effizienter zu arbeiten und dich aufs Wesentliche zu konzentrieren.
      </p>
      <div>
      </div>
      <h6>
       Und so funktioniert's …
      </h6>
      <p class="right">
       Gib in das obige
       <b>
        Textfeld
       </b>
       ein vollständiges
       <strong class="plain">
        Reimwort
       </strong>
       ein und starte die Reimsuche. Auf Groß- und Kleinschreibung braucht nicht geachtet zu werden. Das
       <b>
        Suchergebnis
       </b>
       kannst du über den
       <span class="icon-dr-hamburger dr-td">
       </span>
       <b>
        Einstellungen
       </b>
       -Filter an deine Bedürfnisse anpassen. Neben der Silbenzahl lässt sich nach Worttyp filtern – oder die alphabetische Sortierung ändern.
      </p>
     </article>

       Liebe sich fühlte, darin geborgen,
       <br/>
       traurig Ende, nicht vorhersehbar.
       <br/>
       <br/>
       Ein fröhlich Lächeln in sich kehrt,
       <br/>
       möchte nach außen, nicht tragen,
       <br/>
       <br/>
       denn Stille, gelegentlich verwehrt,
       <br/>
       das ein Lächeln muss, sich plagen.
       <br/>
       <br/>
       Stille, zu atmen mich, nicht wage,
       <br/>
       zart ein Rauschen nur vom Winde
       <br/>
       <br/>
       Gedanken nun, dorthin ich trage,
       <br/>
       wo noch Leben, statt Stille finde.

       Ein Kuss ist ein Kuss ist ein ..
      </h3>
      <blockquote>
       <span class="icon-dr-quotes-left">
       </span>
       Was ist ein Kuss -
       <br/>
       der Austausch von Gefühlen?
       <br/>
       Ich hab dich lieb' und
       <br/>
       du mich bitte auch?
       <br/>
       Es ist der Kuss, ich weiß,
       <br/>
       ein schöner alter Brauch:
       <br/>
       <br/>
       Mal ist er nass und manchmal
       <br/>
       ziemlich spröde,
       <br/>
       mal kommt er spitz daher,
       <br/>
       doch seltener als Witz.
       <br/>
       Und hin und wieder denkst du:
       <br/>
       Was für 'n zarter Frühlingshauch!
       <br/>
       Nur im Extremfall kriegst du Angst:
       <br/>
       Der frisst mich auf …
       <br/>
       <br/>
       Mal seufzt er leise vor sich hin,
       <br/>
       mal gibt 's 'nen lauten Schmatz,
       <br/>
       mal flammt von Lippenpaar zu
       <br/>
       Lippenpaar die Wollust auf;
       <br/>
       der Bruderkuss verbindet gar
       <br/>
       von Menschewist zu Menschewist
       <br/>
       so manch bankrottes Land;
       <br/>
       ein Kuss versetzt den Teenie
       <br/>
       außer Rand und Band.
       <br/>
       <br/>
       Gehaucht von Mund zu Mund
       <br/>
       verhilft ein zarter Kuss
       <br/>
       selbst einer Greisin,
       <br/>
       einem Greise noch
       <br/>
       und dir und mir
       <br/>
       zu einem unvergesslichen - Genuss.
       <span class="icon-dr-quotes-right">
       </span>
      </blockquote>
      <cite>
       von
       <b>
        Annelie Kelch
       </b>
       , veröffentlicht am
       <b>
        7. März 2017
       </b>
      </cite>
