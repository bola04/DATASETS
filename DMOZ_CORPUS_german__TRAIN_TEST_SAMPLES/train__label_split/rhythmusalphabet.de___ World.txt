       <h1 id="post-34">
        Ein Rhythmuskurs für musikalische Anfänger
       </h1>
       <div class="entrytext">
        <p align="left">
         Hier stelle ich Ihnen meinen Rhythmuskurs „Rhythmusalphabet“ für musikalische Anfänger vor.
        </p>
        <p align="left">
         Dieses Rhythmustraining habe ich im Lauf meiner Arbeit als Gitarrenlehrer an Musikschulen im Rhein-Main-Gebiet entwickelt. Ich persönlich fand die Verbesserung der rhythmischen Fähigkeiten meiner Schüler als so deutlich, und die Kollegenresonanz war so positiv, dass die Idee aufkam, dieses Training zu vermarkten. Zuerst als CD, ab 2008 als Downloadangebot.
        </p>
        <p align="left">
         Da dieses Trainings aus zwei Teilen besteht und ursprünglich als CD gedacht war, ist des Downloadangebot bzw. Versandangebot so gestaltet, dass jeder Teil des Kurses mit wenig Aufwand auf eine CD gebrannt werden kann.
        </p>
        <p align="left">
         Auf Grund rechtlicher Gründe gilt bei Downloadangeboten und dieser Art von Versand nicht das 14-tägige Widerrufsrecht für Fernabsatzgeschäfte. Deswegen versuche ich ihnen ausführliche Informationen zur Verfügung zu stellen, so dass Sie nicht die Katze im Sack kaufen müssen.
        </p>
        <ul>
         <li>
          Über das pädagogische Prinzip können Sie sich unter „
          <a href="http://www.rhythmusalphabet.de/das-paedagogische-prinzip-des-rhythmusalphabetes/" title="Pädagogischer Hintergrund">
           <strong>
            Prinzip
           </strong>
          </a>
          “ informieren.
         </li>
         <li>
          Sie können sich den Inhalt von „
          <a href="http://www.rhythmusalphabet.de/produktinfo-teil-1/" title="Inhalt Teil 1">
           <strong>
            Teil 1
           </strong>
          </a>
          “ und „
          <a href="http://www.rhythmusalphabet.de/produktinfo-teil-2/" title="Inhalt Teil 2">
           <strong>
            Teil 2
           </strong>
          </a>
          “ ansehen und teilweise anhören.
         </li>
         <li>
          Unter dem Punkt „
          <a href="http://www.rhythmusalphabet.de/haeufig-gestellte-fragen/" title="Was Sie vielleicht interessiert.">
           <strong>
            Fragen
           </strong>
          </a>
          “ beantworte ich die meist gestellten Fragen zum Rhythmusalphabet.
         </li>
         <li>
          Unter dem Punkt „
          <strong>
           <a href="http://www.rhythmusalphabet.de/kauf-hinweise/">
            Hinweise zum Kauf (Download)
           </a>
          </strong>
          “ finden Sie wichtige Informationen zum Kaufvorgang und technischen Voraussetzungen für das Downloadangebot.
         </li>
         <li>
          Unter dem Punkt „
          <strong>
           <a href="http://www.rhythmusalphabet.de/hinweise-zum-kauf-versand/">
            Hinweise zum Kauf (Versand)
           </a>
          </strong>
          “ finden Sie wichtige Informationen zum Kaufvorgang und technischen Voraussetzungen für das Downloadangebot.
         </li>
         <li>
          Meine Multiplikatoren dürften wahrscheinlich Kollegen sein. Diese können unter „
          <a href="http://www.rhythmusalphabet.de/tipps-fuer-kollegen/" title="Was Sie beachten sollten!">
           <strong>
            Tipps für Kollegen
           </strong>
          </a>
          “ nachlesen, ob das Rhythmusalphabet für Sie geeignet ist und wie sie mit dem Rhythmusalphabet arbeiten sollten.
         </li>
         <li>
          Wenn Sie Fragen haben, finden Sie Kontaktmöglichkeiten unter dem Punkt „
          <a href="http://www.rhythmusalphabet.de/impressum/" title="Wenn Sie Fragen haben.">
           <strong>
            Impressum
           </strong>
          </a>
          „.
         </li>
        </ul>
