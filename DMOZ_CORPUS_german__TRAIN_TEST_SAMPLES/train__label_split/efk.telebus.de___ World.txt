             In unserem Kindergarten wird die Welt für die
             <br/>
             <strong>
              Kinder
             </strong>
             richtig interessant! Denn sie
             <strong>
              erleben
             </strong>
             <br/>
             viel, sowohl im Kindergarten als auch außerhalb
             <br/>
             auf verschiedensten Ausflügen.
             <br/>
             Unser Kindergarten schafft eine
             <strong>
              geborgen
             </strong>
             e
             <br/>
             Basis für Kinder, wo sie die Sicherheit und die
             <br/>
             Freiheit haben, sich zu
             <strong>
              entwickeln
             </strong>
             , damit sie
             <br/>
             nach der Kindergartenzeit selbstbewusst auf
             <br/>
             Entdeckungsreise gehen können.
            </p>

            </p>

              Eltern
             </strong>

             <br/>
             sondern sehr
             <strong>
              willkommen
             </strong>

             <br/>
             Privileg, einmal im Monat am Kindergartenalltag im Form eines
             <br/>
             Elterndienstes
             <strong>
              teilzunehmen
             </strong>
             : Man bereitet das Frühstück, hat
             <br/>
             die Möglichkeit, seine eigene Talente
             <strong>
              einzubringen
             </strong>

             <br/>

              Die Vorteile unseres Kindergartens sind:

            <ul>

              eine Gruppe mit 20-25 Kindern, altersgemischt ab 3 Jahren
             </li>
             <li>
              großes Außengelände mitten in der Friedrichsau
             </li>
             <li>
              freundliche Innenräume mit vielen Spielmöglichkeiten
