          <h1>
           Gelenkzentrum Saar
          </h1>
          <p>
          </p>
          <a class="jcepopup" href="/component/content/article/2-uncategorised/3-praxis-illingen" title="Standort Illingen">
           <img alt="Standort Illingen" height="30" src="/images/stories/standorte/standort_illingen.png" style="margin-right: 5px;" width="150"/>
          </a>
          <a href="/component/content/article/2-uncategorised/35-praxis-heusweiler" title="Praxis Heusweiler">
           <img alt="standort heusweiler" height="30" src="/images/stories/standorte/standort_heusweiler.png" style="margin-right: 5px;" width="150"/>
          </a>
          <br/>
          <br/>
          Das Gelenkzentrum Saar ist eine überörtliche
          <strong>
           orthopädisch-unfallchirurgische Gemeinschaftspraxis an den Standorten Illingen und Heusweiler.
          </strong>
          <br/>
          Im Gelenkzentrum Saar werden Sie von folgenden Fachärtzten für Orthopädie und Unfallchirurgie behandelt:
          <br/>
          <ul>
           <li>
            <a href="/component/content/article/8-aerzte/17-dr-med-henrik-lind" title="Dr. med. Henrik Lind">
             Dr. med. Henrik Lind
            </a>
           </li>
           <li>
            <strong>
             <a href="/component/content/article/8-aerzte/19-regis-meggiorini" title="Regis Meggiorini">
              Dr. med.Regis Meggiorini
             </a>
            </strong>
           </li>
           <li>
            <strong>
             <a href="/component/content/article/8-aerzte/16-egbert-mohr" title="Egbert Mohr">
              Egbert Mohr
             </a>
            </strong>
           </li>
           <li>
            <a href="/component/content/article/8-aerzte/40-umair-ahmad-bajwa">
             Umair Ahmad Bajwa
            </a>
           </li>
           <li>
            <a href="/aerzte">
             Dr. med Wolfgang Schneider
            </a>
           </li>
          </ul>
          <p>
           <br/>
           Wir bieten Ihnen ein
           <strong>
            diagnostisches
           </strong>
           und
           <strong>
            therapeutisches Leistungsspektrum
           </strong>
           auf
           <strong>
            höchstem Niveau
           </strong>
           .
           <br/>
