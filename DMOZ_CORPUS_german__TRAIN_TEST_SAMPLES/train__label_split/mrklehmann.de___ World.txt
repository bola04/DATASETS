           <h3>
            Werbebotschaft per SMS
           </h3>
           <p>
            Der Konsument sendet eine SMS mit einer Zahl oder einem Stichwort an eine Kurzwahlnummer, um beispielweise ein Produkt zum Testen zu erhalten oder um an einem Gewinnspiel teilzunehmen. Der Code kann auf einer Verpackung (On-pack-Promotion) aufgedruckt sein oder auch auf einer Anzeige stehen. Wenn die Kunden den Code einsenden, erfahren sie, ob sie etwas gewonnen haben. Alternativ werden sie auf eine mobile Landing Page oder ein mobiles Portal weitergeleitet, wo sie mobilen Content (Klingelton, Screensaver u.Ä.) herunterladen können. Der Fachausdruck dafür lautet „Direkt Response Marketing“. Die Kontakte, die aus diesen Promotions resultieren, werden für weitere Direktmarketing-Maßnahmen eingesetzt.

           <p>
            <strong>
             Wie man Attribute verbessert (lernt)
            </strong>
           </p>
           <p>
            Ganz offensichtlich erfordert das Lernen der Lern- und Attributskills eine gewissen Zeit. Das Primärattribut, das für die „Learning“ Gruppe benötigt wird ist „Memory“ und das Sekundärattribut ist „Intelligence“.
           </p>
           <p>
            Deswegen erspart Ihnen das Lernen von zunächst „Instant Recall“ (Memory) und „Analytical Mind“ (Intelligence) eine Menge Zeit beim Lernen der anderen Fertigkeiten. Die Attribute „Memory“ und „Intelligence“ sind auch für das Erlernen anderer Fertigkeitsgruppen wichtig, so dass Sie in Erwägung ziehen sollten, sie mindestens auf Stufe 3 – oder besser noch auf Stufe 4 zu lernen.
           </p>
