     <h1>
      Willkommen zu den Microsoft Windows Server Update Services
     </h1>
     <div class="style106">
      Bei den Windows Server Update Services (WSUS) handelt es sich um eine Patch- und Updatesoftware von Microsoft, bestehend aus einer Server- und einer Clientkomponente. WSUS unterstützt die Administratoren, Microsoft-Updates in großen lokalen Netzwerken auszuliefern. Er lädt Updatepakete aus dem Internet (Microsoft Update) und bietet sie den Windows-Clients zur Installation an. Der WSUS-Administrator kann am Server festlegen, welche Computer welche Updates installieren sollen. Microsoft stellt die Windows Server Update Services kostenlos zur Verfügung. Erstmals wurden die Update Services vollständig als Rolle in den Server Manager des Windows Server 2012 integriert.
     </div>
     <table cellpadding="0" cellspacing="0" class="style111" style="width: 100%">
      <tbody>
       <tr>
        <td style="width: 75%" valign="top">
         <div class="style106">

Darüber hinaus kann der WSUS Server als Updatequelle für untergeordnete WSUS Server dienen.
          <br/>
         </div>
        </td>
