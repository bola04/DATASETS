        <div class="page-header">
         <h1>
          Verbau
         </h1>
        </div>
        <div itemprop="articleBody">
         <p style="text-align: center;">
          <strong>
           <span style="color: #000099; font-family: Arial, Palatino; font-size: x-large;">
            <br/>
           </span>
           <span style="color: #000099; font-family: Arial; font-size: x-large;">
            <strong>
             <span style="color: #000099; font-family: Georgia, Palatino, Times\ New\ Roman, Times, serif, serif;">
              VERBAUGERÄTE ROTTKAMP-KIMMEL
              <br/>
             </span>
             <span style="color: #000099;">
              Vermietung - Verkauf - Service
              <br/>
              SBH Verbau
             </span>
            </strong>
           </span>
          </strong>
         </p>
         <p style="text-align: center;">
          <strong>
           <span style="color: #000099; font-family: Arial; font-size: x-large;">
           </span>
          </strong>
         </p>
         <p>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Unsere Fa. Verbaugeräte Rottkamp-Kimmel ist seit über zweiJahrzehnten Vertragspartner der Fa. SBH Tiefbautechnik GmbH in 52525 Heinsberg, Ferdinand Porsche Str. 8.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Firma verfügt über eine umfangreichen Erfahrungsschatz was Verbauarbeiten im Tiefbau angeht. Der Einsatz von verschiedenen Verbaugeräten ist immer abhängig von der Verbautiefe:
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Bis 2m Verbautiefe kommt folgendes Verbausystem zum Einsatz:
            <br/>
            Schnellverbau, 260er Serie.
           </strong>
           Dieses Verbausystem bieten wir nur zum Kauf an und es wird bei standfesten Böden eingesetzt.
          </span>
          <br/>
          <br/>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Bis 3m Verbautiefe kommen folgendes Verbausystem zum Einsatz:
           </strong>
           <br/>
           <strong>
            Aluverbau 250er Serie
           </strong>
           und
           <strong>
            Leichtverbau 100er Serie
           </strong>
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Der Aluminiumverbau 250er Serie wird nur zum Kauf angeboten, während die Leichtverbau 100er Serie
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           auch gemietet werden kann. Hiermit werden Gräben für Fernwärme-, Wasser- und Stromleitungen mit verbaut.
           <br/>
           Wir erreichen mit unserem 100er Verbausystem eine lichte Breite von 3,08m. In der Hauptsache kommen Plattenlängen von 2,0 2,5 und 3m zum Einsatz. Die Stärke der Platten beträgt 60mm.
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Der Leichtverbau wird im Einsetzverfahren verbaut.
          </span>
         </p>
         <p>
          <strong>
           <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
            Bis 4m Verbautiefe kommen folgendes Verbausystem zum Einsatz:
           </span>
          </strong>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Standardbox 600er Serie
           </strong>
           wird als Hauptumsatzträger in der Miete eingesetzt.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Box ist sogar bis Tiefen von 6m zugelassen.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Plattenstärke beträgt 100mm, wobei das lichte Maß bis zu einer Breite von 4,28m möglich ist.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Verbauplatten sind in den Abmaßen 2,0 2,5  3,0  3,5  4m im Mietpark erhältlich. Längere Platten können bei uns käuflich erworben werden.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Der Standardboxverbau wird im Absenkverfahren eingesetzt. Die Rohrdurchlaßhöhe beträgt 1,5m. Mithilfe von Überziehprofilen kann man die Rohrdurchlasshöhe auf 2,5m erhöhen.
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Rollenschlittenbox 780er Serie
           </strong>
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Rollenschlittenbox zeichnet sich durch große Plattenhöhe aus, so dass der Einsatz eines Aufstockelementes entfällt.
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Einfachgleitschiene 790er Serie
           </strong>
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Die Einfachgleitschiene bildet die moderne Alternative zum Boxenverbau, die sich durch das extrem leichte Ziehen der Elemente auszeichnet.
          </span>
          <br/>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           <strong>
            Über 4m kommen folgende Verbausystem zum Einsatz:
           </strong>
           <br/>
           <br/>
           <strong>
            Doppelgleitschiene, 750er Serie.
            <br/>
           </strong>
           Hier können Tiefen bis 10m realisiert werden. Verbaubreiten bis zu 12m sind möglich.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Durch das von uns patentierte Rollenschlittensystem ist man bzgl. der Rohrdurchlasshöhen flexibel.
          </span>
         </p>
         <p>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Für die immer größer werdende Anzahl von Querungen kommen unsere
           <strong>
            Dielenkammerverbau Systeme 400er
           </strong>
           Serie zum Einsatz. Diese können einerseits als normales Dielenkammerelement mit Plattenlängen zwischen 2 und 6m zum Einsatz kommen, andererseits finden unsere Dielenkammerplatten auch den Einsatz in unseren 100% kompatiblen Gleitschienensystemen. Diese Kompatibilität unserer Systeme ist die Stärke der SBH - VerbauSysteme. Bei Verwendung von entsprechenden Gurtungssystemen sind Tiefen bis 10 m realisierbar.
          </span>
          <br/>
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Außerdem können wir Ihnen durch unser eigenes Kaltwalzwerk die Walzprofile KD6/8, Leichtprofil Typ LP, Omega und Z- Profil zur Verfügung stellen.
          </span>
         </p>
         <p>
         </p>
         <p>
         </p>
         <p style="text-align: center;">
          <img alt="Bauma 2013" src="/images/Bauma%202013.jpg"/>
         </p>
         <p style="text-align: center;">
          <strong>
           <span style="color: #000099; font-family: Times\ New\ Roman;">

             Dipl.- Ing.

          </strong>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Als Vertragspartner der Fa. SBH Tiefbautechnik GmbH mit Sitz in
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           52525 Heinsberg, Ferdinand Porsche Str. 8 sind wir die direkte Verbindung zum Hersteller.
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Innerhalb von Stunden bearbeiten wir Ihre Anfragen mit Angeboten und können Ihnen die für Sie beste Verbaulösung anbieten.
           <br/>
           <br/>
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Kurzfristig können wir Ihnen eine Baustelleneinweisung vor Ort zusagen. Wir bieten Weiterbildungsmaßnahmen vor Ort oder bei Ihnen in der Firma, über das Thema Verbau an.
           <br/>
           <br/>
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Weiterhin bieten wir Weiterbildungsmaßnahmen im Stammwerk an.
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Durch unsere geringen Overheadkosten können wir Ihnen preisaggressive Angebote unterbreiten, die bei Submissionen Ihnen einen Konkurrenzvorteil bieten.
           <br/>
           <br/>
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Durch unsere 20- jährige Erfahrungen im Sektor Verbau können wir Sie bei Miete, Kauf, Mietkauf, Ratenkauf, Ersatzteilbedarf fachspezifisch und kostenlos beraten.
           <br/>
           <br/>
          </span>
         </p>
         <p style="margin-bottom: 0cm;">
          <span style="font-family: arial, helvetica, sans-serif; font-size: 14pt; color: #000080;">
           Statische Berechnungen für Ihren Verbau beinhaltet ebenso unser Portfolio, wie der faire Abrechnungsmodus mit 20 Arbeitstagen bei Mietaufträgen.
          </span>
         </p>
