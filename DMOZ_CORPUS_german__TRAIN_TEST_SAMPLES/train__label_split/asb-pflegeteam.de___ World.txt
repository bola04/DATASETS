            <h1>
             Ihr ASB Regionalverband Rhein-Erft/Düren e.V.
            </h1>
            <p>
             <img alt="" class="bildlinks" src="/images/stories/aktuelles/asb-erft-dueren-gebaeude-25.jpg"/>
             Der ASB Regionalverband Rhein-Erft/Düren e.V. ist zuständig für den Rhein-Erft-Kreis, die Kreise Düren und Heinsberg sowie die Städte-Region Aachen.
            </p>
            <p>
            </p>
            <p>
             <span>
              <strong>
               Kompetent und herzlich.
              </strong>
             </span>
            </p>
            <p>
             Neben dem Service-Wohnen bietet der Regionalverband den Bürgern ein breites Leistungsspektrum: Pflegezentren, Sozialstationen, mobile soziale Dienste, Menüservice, Haus-Notruf, ISA (Individuelle Schwerstbehindertenassistenz), familienunterstützende Dienste, Schuldner- und Krebsberatungsstellen, Wohnheime und Wohngemeinschaften für psychisch Kranke, Kindergärten, ein Jugendkulturzentrum, Erste Hilfe-Ausbildungen und den Katastrophenschutz. Etwa 500 hauptamtliche und eine Vielzahl ehrenamtlicher Mitarbeiter engagieren sich im Regionalverband.
            </p>
            <p>
             <span style="font-size: 8pt;">
              <span style="font-size: 8pt;">
              </span>
             </span>
             <span>
              <strong>
               <br/>
               Wir helfen hier und jetzt.
              </strong>
             </span>
            </p>
            <p>
             Bei Fragen rund um den ASB und seine Hilfs- und Dienstleistungen rufen Sie uns einfach unverbindlich an.
             <br/>
             Die zentrale Telefonnummer des ASB Regionalverbandes Rhein-Erft/Düren e.V. ist  0 22 35-4 60 20.
            </p>
            <p>
             Die zentrale Mailadresse:
             <a href="mailto:info@asb-erft.de">
              <img alt="mail150px" height="53" src="/images/stories/aktuelles/2015/mail150px.png" width="53"/>
             </a>
            </p>

			function setCookie(c_name,value,exdays)
			{
				var exdate=new Date();
				exdate.setDate(exdate.getDate() + exdays);
				var c_value=escape(value) + ((exdays==null) ? "" : "; expires="+exdate.toUTCString()) + "; path=/";

			function readCookie(name) {
				var nameEQ = name + "=";
				var jb = document.cookie.split(';');

					var c = jb[i];
					while (c.charAt(0)==' ') c = c.substring(1,c.length);
						if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
					}
				return null;
