        <h2>
         Willkommen
        </h2>
        <p class="left clearfix">
         <a class="fancybox fancybox.image" href="/upload/glaze/872/GastgeberKlein_original.JPG" rel="artikel" title="">
          <img alt="" src="/upload/glaze/872/GastgeberKlein_zoom.JPG" width="146"/>
         </a>
         Wir begreifen Campingurlaub als ein
         <b>
          Sommerglück
         </b>
         und gestalten Ihnen dazu bei uns einen unkomplizierten Aufenthalt:
         <br/>
         Unsere Vorstellung von 'leben und leben lassen' macht die Atmosphäre auf unserem Platz aus - hier finden Sie gemeinsam mit vielen sympathischen Gästen Ruhe und Erholung
         <br/>
         <b>
          - ganz einfach Zeit füreinander.
         </b>
         <br/>
         <font color="FF0033">
          <br/>
         </font>
         <b>
          <font color="FF0033">
           AKTUELL:
          </font>

          <a href="http://camping-suedstrand.de/alle-platzinfos/video's">
           <font color="FF0000">
            Drohnenrundflug
           </font>
          </a>
          über unseren Platz!
          <br/>
          <br/>
         </b>
         <b>
          <font color="FF0033">
           NEU seit 2016:
          </font>
          Die
          <a href="http://camping-suedstrand.de/freizeit/tauchschule">
           <font color="0000FF">
            Tauchschule
           </font>
          </a>
          für Sie!
         </b>
         <br/>
         <br/>
         <b>
          Bitte beachten Sie unsere begehrten
         </b>
         <a href="http://camping-suedstrand.de/alle-platzinfos/mietzelte">
          <b>
           MIETZELTE
          </b>
         </a>
         <b>
          - ein preisgerechtes Angebot für's echte Camping-Erleben!
          <br/>
          <br/>
          EBENFALLS TOLL: zentral gelegene
         </b>
         <a href="http://camping-suedstrand.de/alle-platzinfos/zelten">
          <b>
           Zeltwiese
          </b>
         </a>
         <b>
          , preisgünstig für Familienzelte bis 5m x 5m Größe.
          <br/>
          <br/>
         </b>
         Ellen &amp; Jens-Peter Bumann und Team
         <br/>
         <br/>
         <font color="FF0000">
          <b>
