       <h1>
        Herzlich Willkommen auf der Website der NNNPN e.V. - niedergelassene Neurologen, Nervenärzte und Psychiater in Niederbayern
       </h1>
       <div class="box-content-text">
        <p>
         Neurologen, Nervenärzte und Psychiater in Niederbayern haben sich im NNNPN e.V. zusammengeschlossen.
        </p>
        <p>
         Der Verein wurde 2005 von engagierten Neurologen, Nervenärzten und Psychiatern in Niederbayern - kurz NNNPN e.V. gegründet und hat seinen Sitz in Bogen (Landkreis Straubing / Bogen).
        </p>
        <p>
         Wir sind in erster Linie niedergelassene Kassenärzte, die sich eine Verbesserung der Qualität der ambulanten medizinischen Versorgung auf dem nervenärztlichen Fachgebiet zum Ziel gesetzt haben.
        </p>
        <h2>
         Dies betrifft zum Beispiel folgende Krankheitsbilder:
        </h2>
        <ul>
         <li>
          Schlaganfall
         </li>
         <li>
          Multiple Sklerose
         </li>
         <li>
          Morbus Parkinson
         </li>
         <li>
          Nervenverletzungen / Nervenkompressionssyndrome
          <br/>
          <ul>
           <li>
            zum Beispiel Bandscheibenvorfall
           </li>
          </ul>
         </li>
         <li>
          Neuropathien
         </li>
         <li>
          Borreliose
         </li>
         <li>
          Kopfschmerzen / Gesichtsschmerzen
          <br/>
          <ul>
           <li>
            Migräne
           </li>
           <li>
            Trigeminus-Neuralgie
           </li>
           <li>
            Cluster-Kopfschmerz
           </li>
           <li>
            Spannungs-Kopfschmerz
           </li>
           <li>
            Schwindel
           </li>
          </ul>
         </li>
         <li>
          Anfallsleiden
          <ul>
           <li>
            zum Beispiel Epilepsie
           </li>
          </ul>
         </li>
         <li>
          Depressionen
          <br/>
          <ul>
           <li>
            Bipolare Störungen
           </li>
           <li>
            Manie
           </li>
          </ul>
         </li>
         <li>
          Demenzielle Störungen
          <br/>
          <ul>
           <li>
            zum Beispiel Alzheimer
           </li>
          </ul>
         </li>
         <li>
          Angststörungen
         </li>
         <li>
          Zwangsstörungen
         </li>
         <li>
          Schlafstörungen
         </li>
         <li>
          Psychosen aus dem schizophrenen Formenkreis
         </li>
        </ul>
        <h2>
         Der Verein ist hierzu unter anderem aktiv durch
        </h2>
        <ul>
         <li>
          Vertretung von Patienteninteressen auf gesundheitspolitischer Ebene
         </li>
         <li>
          Kontakt und Zusammenarbeit mit Patienten und Selbsthilfegruppen
         </li>
         <li>
          Presse- und Öffentlichkeitsarbeit
         </li>
         <li>
          Verbesserung der Patientenversorgung in Senioren-Wohn- und Pflegeheimen
         </li>
         <li>
          interne Fortbildung für Ärzte
         </li>
         <li>
          Fortbildungsangebote für Arzthelferinnen / medizinischen Fachangestellten
         </li>
         <li>
          Durchführung ärztlicher Qualitätszirkel unter Einbeziehung aller Fachrichtungen
         </li>
         <li>
          Erfahrungsaustausch unter den ärztlichen Mitgliedern
         </li>
         <li>
          Sicherung der medizinischen Qualität durch Aufbau geeigneter Qualitätsmanagementverfahren
         </li>
         <li>
          internes Fachforum
         </li>
        </ul>
        <p>
         Für Fragen aller Art stehen wir Patienten, Angehörigen sowie Ärzten gerne zur Verfügung, nehmen Sie
         <a href="/kontakt/">
          Kontakt
         </a>
         mit uns auf.
        </p>
