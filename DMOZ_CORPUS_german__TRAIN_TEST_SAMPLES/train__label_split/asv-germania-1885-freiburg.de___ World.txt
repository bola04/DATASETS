               Willkommen beim ASV Germania 1885 Freiburg
              </h3>
              <h3 style="text-align: center;">
               Mitgliederversammlung der ASV Germania 1885 Freiburg e.V.
               <br/>
               am 20.06.2018 um 20:00 Uhr
               <br/>
               im Vereinsheim Ensisheimer Straße 20, 79110 Freiburg
              </h3>
              <h3>
               Tagesordnung:
              </h3>
              <p>
               – Begrüßung
               <br/>
               – Totenehrung
               <br/>
               – Genehmigung der Tagesordnung
               <br/>
               – Verlesen des letztjährigen Protokolls
               <br/>
               – Berichte für das Geschäftsjahr 2017
               <br/>
               – Kassenbericht und Entlastung des Hauptkassierers
               <br/>
               – Entlastung des Gesamtvorstandes
               <br/>
               – Wahl eines Wahlleiters
               <br/>
               – Neuwahlen
               <br/>
               1. Vorsitzender
               <br/>
               Geschäftsführer
               <br/>
               Hauptkassierer
               <br/>
               Technischer Leiter
               <br/>
               Beisitzer
               <br/>
               – Bestätigung der Jugendleiterin
               <br/>
               – Anpassung der Mitgliedsbeiträge mit Abstimmung
               <br/>
               – Anträge
               <br/>
               – Angelegenheiten die sich aus der Versammlung ergeben
               <br/>
               – Informationen / Bekanntmachungen
               <br/>
               – Schlusswort
              </p>
              <p>
               <strong>
                Anträge müssen laut Vereinssatzung mindestens 6 Tage vor der Mitgliederversammlung schriftlich

               </strong>

                <br/>

                <br/>

                <br/>

                <br/>
                Technischer Leiter
                <br/>
                Beisitzer
                <br/>
                – Bestätigung der Jugendleiterin
                <br/>
                – Anpassung der Mitgliedsbeiträge mit Abstimmung
                <br/>
                – Anträge
                <br/>
                – Angelegenheiten die sich aus der Versammlung ergeben
                <br/>
                – Informationen / Bekanntmachungen
                <br/>
