          <font color="#000000" face="Georgia" size="+1">
           <b>
            Tour Service Ruhr -
            <br/>
            Ihr Spezialist für Touren im Ruhrgebiet
            <br/>
           </b>
          </font>
          <font color="#000000" face="Georgia" size="-1">
           (mit Empfehlung von dem aktuellen Baedeker-Reiseführer Ruhrgebiet sowie dem Pressedienst der ITB Berlin)
           <br/>
           <br/>
           „Wenn einer eine Reise tut, dann kann er was erzählen.“
           <br/>
           <br/>
           Mit einem kompetenten Gästeführer oder Reiseleiter erfahren Sie sogar noch mehr!
           <br/>
           <br/>
           Wir bieten Ihnen ein umfassendes Programm abwechslungsreicher Touren zu Fuß, mit dem Fahrrad, per Schiff, Bus oder Straßenbahn. Unsere Angebote richten sich sowohl an Kinder, Jugendliche, Familien sowie die ältere Generation, an Einzelpersonen als auch an Gruppen. Ein Schwerpunkt unserer Arbeit ist die Betreuung von Vereins- oder Betriebsausflügen.
           <br/>
           Bestimmte Angebote und Thementouren - wie z.B. die GPS-Fahrradrallyes, die Fußballtouren und die Sagen und Legenden-Touren durch das Ruhr- bzw. Emschertal - finden Sie
           <strong>
            nur
           </strong>
           bei uns. Außerdem bieten wir spezielle kostengünstige Touren für Kleingruppen bis maximal fünf Personen inkl. Beförderung an.
           <br/>
           <br/>
           Sie können uns auch für Führungen an einzelnen ausgewählten Orten wie dem Landschaftspark Nord in Duisburg, der Kokerei Zollverein in Essen oder dem Gasometer in Oberhausen buchen. 
 							Oder wie wäre es mit einem Stadtrundgang durch Mülheim oder die Altstadt von Hattingen?
           <br/>
           <br/>
           Unser komplettes Angebot an
           <a href="./touren.html">
            <font color="#000000" face="Georgia">
             <b>
              Touren
             </b>
            </font>
           </a>
          </font>
          ,
          <a href="./fuehrung.html">
           <font color="#000000" face="Georgia">
            <b>
             Führungen
            </b>
           </font>
          </a>
          und Rallyes finden Sie zusammengefasst und sortiert unter der Rubrik Touren oder gleich hier am Ende dieser Seite. Die meisten Touren bieten wir ebenso auf Englisch an.
          <br/>
          Auf Anfrage können wir Ihnen auch niederländisch-, französisch-, spanisch- oder russischsprachige Fahrten durch das Ruhrgebiet vermitteln.
          <br/>
          <br/>
          <strong>
           Achtung: Unser komplettes Angebot an Krimitouren finden Sie im Netz auf eigenen Seiten unter
           <a href="http://www.krimitouren-ruhrgebiet.de" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              www.krimitouren-ruhrgebiet.de
             </b>
            </font>
           </a>
           .
          </strong>
          <br/>
          <br/>
          <strong>
           Daneben vermitteln wir auch Touren mit Hotte Schibulski auf ruhrdeutsch durch das Ruhrgebiet! Diese Angebote finden Sie unter
           <a href="http://www.ruhrpott-touren.de/schibulski.html" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              www.ruhrpott-touren.de
             </b>
            </font>
           </a>
           . Wie die Touren ungefähr sind, können Sie sich auf
           <a href="https://www.youtube.com/channel/UCAdlIcWknwTsK0BI-QJV1Zg" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              Hottes YouTube-Kanal
             </b>
            </font>
           </a>
           anschauen.
          </strong>
          <br/>
          <br/>
          <strong>
           Falls Sie ein Indoor-Event für kleinere oder größere Gruppen suchen, dann können wir Ihnen das große Ruhrgebiets-Quiz empfehlen.
           <br/>
           Das härteste, umfangreichste und spektakulärste Quiz über das Ruhrgebiet finden Sie unter
           <a href="http://www.ruhrgebietsquiz.de/index.html" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              www.ruhrgebietsquiz.de
             </b>
            </font>
           </a>
           , in der ruhrdeutschen Version unter
           <a href="http://www.ruhrpott-quiz.de/index.html" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              www.ruhrpott-quiz.de
             </b>
            </font>
           </a>
           .
          </strong>
          <br/>
          <br/>
          <strong>
           Neu! In Kooperation mit street-safari gibt es jetzt auch die
           <a href="https://www.street-safari.de/de/safari/essen/138-essen-de" target="_blank">
            <font color="#000000" face="Georgia">
             <b>
              Street-Safari Essen
             </b>
            </font>
           </a>
           , eine Smartphone-Stadtführung durch die Essener Innenstadt! Per Schnitzeljagd werden Sie auf ihrer Smartphone-geführten Route Essen und seine Attraktionen erkunden. Es gilt dabei Fragen und Aufgaben zu lösen und so möglichst viele safari-Punkte zu erspielen. Jetzt buchen und jederzeit starten!
          </strong>
          <br/>
          <br/>
          Sie wissen noch gar nicht so genau, was Sie sich anschauen möchten? Sie finden in unserem Angebot keine passende Tour für sich? Kein Problem! Wir stellen Ihnen auch eine 
 							maßgeschneiderte Tour nach Ihren persönlichen Wünschen zusammen. Sie müssen uns nur mitteilen, wann Sie etwas unternehmen möchten, wie viel Zeit Sie mitbringen, wohin Sie mit wie 
 							vielen Personen wollen oder was Ihr spezielles Interesse ist. In den meisten Fällen können wir Ihnen sofort weiterhelfen. Sollte das ausnahmsweise einmal nicht möglich sein, dann 
 							wissen wir wahrscheinlich, an wen Sie sich wenden können.
          <br/>
          <br/>
          <strong>
           Gästeführer bzw. Reiseleiter für das Ruhrgebiet gesucht?
          </strong>
          <br/>
          <br/>
          Falls Sie einen Gästeführer für das Ruhrgebiet oder einen Reiseleiter für eine Ausflugstour in die nähere Umgebung oder für eine (mehrtägige) Fahrt in eine andere europäische Metropolenregion suchen, können Sie selbst mit 
 							diesem Wunsch bei uns fündig werden. Fragen Sie ruhig danach.
          <br/>
          <br/>
          Alle nötigen Informationen, um uns zu kontaktieren, finden Sie gleich hier am Ende dieser Seite.
          <br/>
          <br/>
          Am besten erreichen Sie uns per E-Mail, weil wir bewusst auf hohe Bürokosten verzichtet haben, um Ihnen unsere Touren - bei gleichbleibend guter Qualität und bestmöglichem Service -  
 							zu einem günstigen Preis anbieten zu können. Gleichwohl ist unser Telefon ständig für Sie freigeschaltet. Falls Sie uns nicht persönlich erreichen, rufen wir Sie umgehend zurück.
          <br/>
          <br/>
          Ihre Anregungen und Vorschläge sowie Lob und Kritik nehmen wir dankbar entgegen, denn wir haben es uns zum Ziel gesetzt, unsere Produkte so kundenorientiert wie möglich anzubieten.
