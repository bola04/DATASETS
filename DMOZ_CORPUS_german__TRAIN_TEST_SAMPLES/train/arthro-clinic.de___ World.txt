         <h1>
          <span style="font-size: 1.4em;">
           Orthopäde in Hamburg gesucht?
          </span>
         </h1>
         <p>
         </p>
         <p>
          <img alt="Das Arthro Clinic Team" src="/images/stories/personal/team_panorama_klein.jpg"/>
         </p>
         <p>
          <br/>
          Sie suchen einen Orthopäden - am besten einen Schulterspezialisten, einen Hüftspezialisten, einen Kniespezialisten oder einen Fußspezialisten in Hamburg? Dann sind Sie bei uns richtig. Wir sind eine Gemeinschaftspraxis für Orthopädie mit dem Schwerpunkt operative Therapie und Dr. med. Kai Uwe Jensen gehört laut FOCUS Gesundheit schon seit Jahren zu Deutschlands Top-Medizinern (siehe
          <a href="/focus-aerzteliste">
           FOCUS Ärzteliste
          </a>
          ).
          <br/>
          Wir sind spezialisiert auf arthroskopische Operationen (Schulterarthroskopie, Hüftarthroskopie, Knie-Arthroskopie) und die Versorgung mit Endoprothesen an den großen Gelenken (Schulter-TEP, Hüft-TEP, Knie-TEP und Sprunggelenks-TEP).
          <br/>
          Zum Erstgespräch und zur weiteren Diagnostik melden Sie sich bitte direkt in einer unserer beiden Praxen in Hamburg Rahlstedt oder Hamburg Bergedorf an (siehe
          <a href="/kontakt">
           Kontakt
          </a>
          oder
          <a href="/termin-chrono">
           Terminvereinbarung
          </a>
          ). Ist eine Operation erforderlich, dann operieren wir Sie in der Park-Klinik Manhagen, in der Klinik Fleetinsel, in der Facharztklinik Hamburg oder im Evangelischen Krankehaus Alsterdorf. Wir wünschen gute Besserung!
          <br/>
          <br/>
         </p>

           Arthro Rehabilitation 2018
           <br/>
           14. Symposium für Physiotherapeuten - 7. Februar 2018
          </strong>
          <br/>
          Wird zuviel operiert? Grenzen der operativen Therapie - Chancen für die Physiotherapie. Mit diesem Thema startete das diesjährige Symposium für Physiotherpeiten. Das Symposium war komplett ausgebucht.  (
          <a href="/aktuelles#Arthro-Rehabilitation_2018" title="14. Symposium für Physiotherapeuten">
           Mehr...
          </a>
          )
          <br/>
          <b>
           <br/>
          </b>
         </p>
         <p>
          <b>
           Expertentipp von Drs. (NL) Bongaerts in der Wochenzeitung
           <br/>
           DIE ZEIT, 2. März 2017, Seite 8H (Hamburg)
           <br/>
          </b>
          <a href="/aktuelles#Expertentipp_DIE_ZEIT_Maerz_2017">
           <img alt="Expertentipp von Drs. (NL) Bongaerts in DIE ZEIT" height="184" src="/images/stories/veroeffentlichungen/2017_DIE_ZEIT_Expertentipp_Arthro_Clinic_Hamburg_400" width="400"/>

          <br/>
          <br/>
          <strong>
           Akupunktur hilft bei Kniearthrose!
          </strong>
          <br/>
          <a href="/knieprobleme/akupunktur" title="Akupunktur hilft bei Kniearthrose">
           <img alt="" height="262" src="/images/stories/aku_tcm/Schmerz.jpg" width="312"/>
          </a>
          <br/>
          Im Jahr 2016 wurden in der Arthro Clinic Hamburg 178 Akupunktursitzungen zur Schmerzbehandlung bei Patienten mit Kniearthrose durchgeführt. Die Patienten gaben eine deutliche Besserung um 32 % an. Der Schmerz reduzierte sich durchschnittlich um 2 Punkte auf einer Schmerzskala von 0 bis 10 (von 6,3 auf 4,2). Die allgemeine Zufriedenheit mit der Behandlung der Schmerzen war hoch. Zwei Drittel der Patienten vergaben 9 oder 10 Punkte von 10 möglichen Punkten.
          <br/>
          Fast identische Ergebnisse zeigte unsere  Studie von 2015. (
          <a href="/knieprobleme/akupunktur" title="Akupunktur hilft bei Kniearthrose">
           Mehr…
          </a>
          )
          <br/>
          <br/>
          <strong>
           <br/>
           FOCUS Ärzteliste
          </strong>
          <br/>
          Wieder einmal wurde Dr. med. Kai-Uwe Jensen in der renommierten FOCUS Ärzteliste gleich zweimal genannt. Er wurde in den Bereichen Kniechirurgie und Schulterchirurgie als TOP-Mediziner bewertet. (
          <a href="/focus-aerzteliste">
           mehr...
          </a>
          )
          <br/>
          <br/>
          <a href="/focus-aerzteliste">
           <img alt="Focus Top Mediziner 2018 Kiespezialist Dr. Kai-Uwe Jensen" height="153" src="/mages/stories/focus/focus_2018_Siegel_Kniechirurgie_neu.jpg" title="FOCUS TOP Mediziner 2018" width="78"/>
          </a>
          <a href="/focus-aerzteliste">
           <img alt="Focus Top Mediziner 2018 Schulterspezialist Dr. Kai-Uwe Jensen" height="153" src="/images/stories/focus/focus_2018_Siegel_Schulterchirurgie.jpg" title="FOCUS TOP Mediziner 2018" width="74"/>
          </a>
          <br/>
          <br/>
          <strong>
           <br/>
           <br/>
           Film "Der vordere Zugang zum Hüftgelenk"
           <br/>
          </strong>
          <a href="/video/Hueftprothese_Vorderer_Zugang_Arthro_Clinic_Hamburg.mp4" rel="noopener noreferrer" target="_blank" title="Hüftprothese: Der vordere Zugang zum Hüftgelenk">
           <img alt="Hüftprothese: Der vordere Zugang zum hüftgelenk" height="237" src="/video/Vorderer_Zugang_Film_Startbild.jpg" width="400"/>
          </a>
          <br/>
          <a href="/video/Vorderer%20Zugang_Arthro_Clinic_Hamburg.mp4" rel="noopener noreferrer" target="_blank" title="Hüftprothese: Der vordere Zugang zum Hüftgelenk">
           Video: Der vordere Zugang zum Hüftgelenk
          </a>
          <strong>
           <br/>
          </strong>
          Der muskelschonende Zugang zum Hüftgelenk ermöglicht eine schnelle und intensive Mobilisation nach Einsetzen einer Hüftprothese. Intraoperativ nutzt der Operateur natürliche Muskellücken um zum Hüftgelenk zu gelangen. Es werden keine Muskeln vom Knochen abgelöst. Dadurch hat der Patient nach der Operation kaum Schmerzen und ist schneller wieder fit. Die Arthro Clinic Hamburg hat sich auf den minimalinvasiven "ventralen" Zugang spezialisiert. (
          <a href="/hueftschmerzen/endoprothesen">
           mehr zum Thema...
          </a>
          )
          <strong>
           <br/>
          </strong>
          Keywords: Hüft-TEP, Ventraler Zugang, Anteriorer minimal-invasiver Zugang, direct anterior approach, DAA
          <br/>
          <strong>
           <br/>
           <br/>
           Filmbeitrag im NDR Schleswig-Holstein Magazin, 11.8.2014
           <br/>
           "Neue Behandlungsmethoden für Hüft-Operationen"
          </strong>
          <a href="http://www.abendblatt.de/ratgeber/article132163368/Wenn-die-Gelenke-schmerzen.html" rel="noopener noreferrer" target="_blank">
           <strong>
            <br/>
           </strong>
          </a>
          Das NDR Schleswig-Hosltein Magazin berichtet über unsere muskelschonende Operationstechnik beim vorderen Zugang zur Hüftprothese. Die neue Operationsmethode ohne Muskelablösung revolutioniert die Rehabilitation nach Hüftprothesen-OP. Bereits drei Stunden nach der Operation sitzt der Patient wieder an der Bettkante. Dank der neuen OP-Technik sind der Reha keine Grenzen gesetzt. Der Patient darf sofort aufstehen. Nach einer Woche läuft er kurze Distanzen ohne Gehstützen und kann schon Treppen steigen.
          <br/>
          Keywords: Hüftprothese, Hüft-TEP, Ventraler Zugang, Anteriorer minimal-invasiver Zugang, minimalinvasiv, direct anterior approach, DAA (
          <a href="http://arthro-clinic.rohwerder-cloud.de/20140811NDR/" rel="noopener noreferrer" target="_blank">
           Filmbeitrag
          </a>
          )
          <br/>
          <a href="http://arthro-clinic.rohwerder-cloud.de/20140811NDR/" rel="noopener noreferrer" target="_blank">
