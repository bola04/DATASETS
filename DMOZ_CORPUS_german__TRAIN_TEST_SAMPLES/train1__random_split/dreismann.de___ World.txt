Wir sehen den Patienten als gleichberechtigten Partner, nicht als Kunden. Wir stehen dafür, dass jeder Patient eine individuelle, an seinen Beschwerden orientierte optimale Behandlung erfährt. Wir machen dabei keinen Unterschied, ob der Patient gesetzlich oder privat versichert ist, die Wahrung der Menschenwürde unabhängig vom sozialen Status ist uns wichtig. Den Gegensatz zwischen sogenannter Schulmedizin und alternativer Medizin sehen wir dabei aber als wenig zielführend an, haben dabei keine Berührungsängste vor der Naturheilkunde.
                <br/>

Hierzu zählen beispielsweise die reisemedizinische Beratung und Metabolic Balance, ein Stoffwechselprogramm zur Gewichtsregulation.

Wir behandeln Patienten aus allen Kulturen, allen Nationen, allen Religionen und allen Sprachkreisen mit dem gebotenen Respekt vor deren jeweiligen Eigenarten. Die multikulturelle Zusammensetzung unseres Teams betrachten wir dementsprechend nicht als Nachteil, sondern als Gewinn.

 /**
  * Fix the z-index bug
  */
 jQuery(document).ready(function() 
 {
 	var api = jQuery('.my_revolutionslider_18').revolution();
 	api.revolution().bind("revolution.slide.onchange",function (e) 
 	{
 		var arrSlides = jQuery(e.currentTarget).find('.slide');
 		if(arrSlides == undefined || arrSlides.length < 1)
 		{
 			return;
 		}
 		
 		var index  = Math.round(api.revcurrentslide() + 1);
 		if(index >= api.revmaxslide())
 		{
 			index = 0;
 		}
 		
 		arrSlides.css('z-index',10);
 		jQuery(arrSlides[index]).css({'z-index':20});
 	});
 });
