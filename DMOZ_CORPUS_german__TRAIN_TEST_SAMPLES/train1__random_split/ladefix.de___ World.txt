         <h2 style="text-align: center;">
          <strong>
           <span style="font-size: 12pt;">
           </span>
          </strong>
          <strong>
           <span style="font-size: 12pt;">
            Wir freuen uns über Ihren Besuch auf unserer Webpräsenz und versuchen Ihnen unsere sehr umfangreiche Produktpalette möglichst genau und dennoch übersichtlich zu vermitteln.
           </span>
          </strong>
         </h2>
         <hr/>
         <ul>
          <li>
           Das
           <strong>
            Produktinfo-Menü
           </strong>
           auf der linken Seite führt Sie direkt zu
           <em>
            "Ihrer"
           </em>
           Rampe, wenn Sie schon wissen, welches Rampenmodell aus unserem Produktsortiment für Sie in Frage kommt. Dort finden Sie auch mögliche Optionen, technische Daten und Preise.
          </li>
          <li>
           <strong>
            Rechtliches
           </strong>
           finden Sie gleich darunter.
          </li>
          <li>
           Im
           <strong>
            Top-Menü
           </strong>
           unter "Grundsätzliches" finden Sie beachtenswertes zu allen Ladeauffahrten. Der Knopf
           <em>
            "Produkte"
           </em>
           ruft eine Kurzübersicht über alle Ladefix Ladehilfen auf.  Unsere
           <strong>
            Vertriebs- und Montagepartner
           </strong>
           sind im
           <em>
            "Partnermenü"
           </em>
           , in der Desktopansicht auf der rechten Seite, aufgelistet. Der jeweilige Link führt Sie direkt zur Partnerhomepage.
          </li>
          <li>
           Ganz unten, in der Fußzeile, finden Sie auf allen Seiten den
           <strong>
            "Zurück zur vorherigen Seite"
           </strong>
           Link. Der bringt Sie immer auf die Seite zurück, von der Sie gerade gekommen sind.
          </li>
         </ul>
         <hr/>
         <p class="typo-info">
          Wenn Sie noch keine Ladefix Laderampe oder noch gar keine Laderampe haben, folgen Sie am Besten dem
          <em>
           "Rundgang"
          </em>
          Link am Seitenende.
          <br/>
          <span style="text-decoration: underline;">
           Übrigens:
          </span>
          Immer wenn Sie so eine
          <img alt="Lupe 69x57" height="NaN" src="/images/Piktogramme/Lupe_69x57.png" width="4%"/>
          Lupe in einem Bild sehen, lässt sich dieses vergrößern.
         </p>
         <p style="text-align: center;">
          <strong>
           Damit Sie auch wirklich die richtige Laderampe bekommen, muss diese nicht nur zu Ihrem Fahrzeug, sondern auch zu den zu transportierenden Ladegütern passen.
          </strong>
         </p>
