               <br/>
               <img alt="" border="0" class="spacer-gif" height="5" src="clear.gif" title="" width="1"/>
               <br/>
               <p style="margin:0 0 0 0;">
                Meine Damen und Herren,
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                Stress und Hektik bestimmen unseren Alltag, nicht nur im Beruf, sondern auch im Privatbereich. Hinzu kommen die hohen Belastungen durch unsere Umwelt.
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                Dies alles hinterlässt seine Spuren und kostet viel Kraft. Umso wichtiger ist es, ab und zu neue Energie zu tanken.
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                Ohne Verpflichtung, ohne Familie und ohne Garderoben- zwang lassen sich die unliebsamen Begleiterscheinungen des täglichen Einerleis in unserer kleinen Oase abstreifen.
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                Die Tagesschönheitsfarm liegt am östlichen Stadtrand von Gera. Sie ist in 10 Minuten zu Fuß vom Zentrum aus zu erreichen und ist ganztägig von Montag - Mittwoch geöffnet. Donnerstag bis Samstag nach vorheriger Absprache.
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                In ungezwungener, jedoch ruhiger Atmosphäre und in einem harmonischen Ambiente werden Sie sich bald rundherum Wohlfühlen.
               </p>
               <p style="margin:0 0 0 0;">
               </p>
               <p style="margin:0 0 0 0;">
                Ihre
               </p>
               <p style="margin:0 0 0 0;">
                Kerstin Zschäckel
               </p>
