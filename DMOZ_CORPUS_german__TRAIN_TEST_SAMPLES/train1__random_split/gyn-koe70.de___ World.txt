       <h1>
        Praxis für Frauenheilkunde
        <br/>
        und Geburtshilfe in Stuttgart
       </h1>
       <h4>
        <a class="internal-link" href="schwangerschaft/" title="Schwangerenbetreuung, Praxis Dostal, Stuttgart">
         Schwangerenbetreuung
        </a>
       </h4>
       <p class="bodytext">
        umfassende Mutterschaftsvorsorge
        <br/>
        3D/4D-Ultraschall / Baby-Fernsehen
        <br/>
        erweiterte Labordiagnostik
        <br/>
        interaktiver Schwangerschaftskalender
        <br/>
        Ultraschalldokumentation über
        <a href="schwangerschaft/fetview/" title="FetView, Ultraschall">
         FetView
        </a>
       </p>
       <h4>
        <a class="internal-link" href="schwangerschaft/praenatale-untersuchungen/" title="Pränatale Diagnostik, Praxis Dostal, Stuttgart">
         Pränatale Diagnostik (DEGUM 2)
        </a>
       </h4>
       <p class="bodytext">
        Pränataler DNA-Test auf Trisomien (Harmony-Test)
        <br/>
        Ersttrimester-Screening mit Nackentransparenzmessung
        <br/>
        und Risikoberechnung auf Down-Syndrom
        <br/>
        pränatale Feindiagnostik (
        <a class="external-link-new-window" href="http://www.degum.de/index.php?id=23" target="_blank" title="Degum 2, Praxis Dostal, Stuttgart">
         DEGUM 2
        </a>
        )
        <br/>
        fetale Echokardiographie
        <br/>
        farbcodierte Dopplersonographie
       </p>
       <h4>
        <a class="internal-link" href="kinderwunsch/" title="Kinderwunsch, Praxis Dostal, Stuttgart">
         Kinderwunschsprechstunde
        </a>
       </h4>
       <p class="bodytext">
        Hormontherapie
        <br/>
        Follikelmessung
        <br/>
        individueller Fruchtbarkeitsindex
       </p>
       <h4>
        <a class="internal-link" href="untersuchung/individuelle-gesundheitsleistung/" title="Krebsvorsorge, Praxis Dostal, Stuttgart">
         Umfassende Krebsfrüherkennung
        </a>
       </h4>
       <p class="bodytext">
        Brust- und Beckenultraschall
        <br/>
        Dünnschichtzytologie (verbesserter Pap-Abstrich)
        <br/>
        Test auf Humane Papillomviren (HPV-Abstrich)
        <br/>
        immunologischer Stuhltest
        <br/>
        Blasenkrebstest
       </p>
       <h4>
        <a class="internal-link" href="operationen/" title="ambulante Operationen, Praxis Dostal, Stuttgart">
         Ambulante Operationen
        </a>
       </h4>
       <p class="bodytext">
        Ausschabung (Abrasio)
        <br/>
        Abortkürettage / genetische Diagnostik
        <br/>
        Einlage der Spirale in kurzer Vollnarkose
        <br/>
        Sterilisation der Frau
       </p>
