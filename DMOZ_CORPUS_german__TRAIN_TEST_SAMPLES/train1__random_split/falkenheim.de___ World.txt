                 Eine spannende Zeitreise in das Mittelalter
                </strong>
               </font>
               <br/>

              <p>
               <span class="hasCaption">
                <font size="3">
                 <strong>
                  Montag 16.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Mittelalter: Wann war das denn eigentlich?
                 <br/>
                 Geschichten und Bilder zum Thema
                 <br/>
                 Wir malen uns eine große Burg als Ferienkulisse
                 <br/>
                 <br/>
                 <strong>
                  Dienstag 17.07
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Welche Berufe gab es im Mittelalter:
                 <br/>
                 Wir basten uns Schilde und Wappen, wer erfindet den lustigsten Ritternamen.
                 <br/>
                 <br/>
                 <strong>
                  Mittwoch 18.07
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Wir basteln unsere Kleidung:
                 <br/>
                 Kegelhauben für  die Burgfräuleins und Kittel für die Ritter
                 <br/>
                 <br/>
                 <strong>
                  Donnerstag 19.07
                  <br/>
                  10-18 Uhr
                 </strong>
                 <br/>
                 Ausflug zur Burg Altena
                 <br/>
                 Fahrt mit öffentlichen Verkehrsmitteln
                 <br/>
                 <br/>
                 <strong>
                  Freitag 20.07
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Strohpuppen und Amulette werden heute gewerkelt.
                 <br/>
                 Bei einem Quiz testen wir unser Wissen über das Mittelalter
                 <br/>
                 <br/>
                 <strong>
                  Montag 23.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Heute geht es weiter mit Helmen für die Ritter und Ritterinnen
                 <br/>
                 Alles was Krach macht: Rasseln und Schellen, Bombaß und Trommeln für das Burgfest.
                 <br/>
                 Wir versuchen einen Ofen aus Lehm zu bauen für den Festschmauß
                 <br/>
                 <br/>
                 <strong>
                  Dienstag 24.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Blütenkränze für die Haare,
                 <br/>
                 Mosaiken aus Papier zum schmücken der Burg.
                 <br/>
                 Natürlich brauchen wir auch Pfeile und Bogen für die Jagd.
                 <br/>
                 <br/>
                 <strong>
                  Mittwoch 25.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Ausflug zur Burgruine Hardenstein.
                 <br/>
                 Fahrt mit öffentlichen Verkehrsmitteln
                 <br/>
                 <br/>
                 <strong>
                  Donnerstag 26.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Jetzt noch eine Zielscheibe basten und jetzt können wir üben für die Jagd
                 <br/>
                 <br/>
                 <strong>
                  Freitag 27.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Dann brauchen wir noch Pferde um zu reisen.
                 <br/>
                 Wir bauen uns Steckenpferde
                 <br/>
                 Stoßlanzen für die RitterInnen
                 <br/>
                 Batiken von bunten Kopftüchern
                 <br/>
                 <br/>
                 <strong>
                  Montag 30.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Jetzt brauchen wir noch Amulette und Siegel, basteln mit Wachs und Ton
                 <br/>
                 <br/>
                 <strong>
                  Dienstag 31.07.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Wir legen einen Kräutergarten an, brauen Tee und lernen Zauberformeln
                 <br/>
                 <br/>
                 <strong>
                  Mittwoch 01.08.
                  <br/>
                  10-18 Uhr
                 </strong>
                 <br/>
                 Ausflug zum Freilichtmuseum nach Hagen
                 <br/>
                 Fahrt mit öffentlichen Verkehrsmitteln
                 <br/>
                 <br/>
                 <strong>
                  Donnerstag 02.08.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Weben von kleinen Teppichen, Gefäße aus Ton
                 <br/>
                 Vorbereitung für das Mittelalterfest
                 <br/>
                 <br/>
                 <strong>
                  Freitag 03.08.
                  <br/>
                  11-17 Uhr
                 </strong>
                 <br/>
                 Großes Mittelalterfest mit Wettkämpfen, leckerem Schmauß und einem Quiz, Spielen und Zaubereien
                </font>
