       </h2>
       <h2>
        <strong>
         Gesunde, leckere Nahrung für jeden &amp; überall…
        </strong>
       </h2>
       <h3>
        <strong>
         Was ist Growtainer?
        </strong>
       </h3>
       <p>
        <strong>
        </strong>
        Ein Pflanzkübel, der durch einfache physikalische Kniffe das Bodenleben im Kübel und die Versorgung der Pflanze optimiert.
        <a href="http://www.growtainer.de/growtainer-prinzip/" title="prinzip">
         Siehe Prinzip
        </a>
       </p>
       <h3>
        <strong>
         Welche Vorteile hat das Growtainern?
        </strong>
       </h3>
       <ul>
        <li>
         <strong>
          Gesunde Pflanzen
         </strong>
         : natürliche Nährstoffversorgung, optimale Luft- und Wasserversorgung und natürliche Mikroorganismen statt ausgelaugter Böden oder Hydro-Anbau
        </li>
        <li>
         <strong>
          Gesunde Nahrung:
         </strong>
         Vollspektum Mineralstoff- und Spurenelemente, frisch geerntet für vollen Vitamin- und Enzymgehalt
        </li>
       </ul>
       <ul>
        <li>
         <strong>
          Überall Anbauen
         </strong>
         : wo es genug Licht und Wärme gibt – auf Balkon, Terasse, Dächern, Betonbrachen etc.
        </li>
       </ul>
       <ul>
        <li>
         <strong>
          Super einfach:
         </strong>
         selten gießen (Reservoir), keine Beikräuter und quasi nie nachdüngen (Düngervorrat für die gesamte Wachstumssaison)
        </li>
       </ul>
       <ul>
        <li>
         <strong>
          Volle Kontrolle:
         </strong>
         keine Pestizide, Herbizide, Fungizide, Genveränderungen oder Bestrahlungen, die Sie nicht wollen
        </li>
       </ul>
       <ul>
        <li>
         <strong>
          Ressourcen- &amp; umweltschonend
         </strong>
         : minimaler Wasserverbrauch, kein Auswaschen oder Einsickern des Düngers in Grundwasser oder  Gewässer,  alles recycelbar und aus lokalen Kreisläufen nutzbar
        </li>
       </ul>
       <p>
        <strong>
        </strong>
       </p>
       <h3>
        <strong>
         Wie komme ich an einen?
        </strong>
       </h3>
       <ul>
        <li>
         einfach selbst bauen
         <a href="http://www.growtainer.de/?page_id=46" title="Eigenbauanleitung Growtainer">
          siehe DIY-Anleitung
         </a>
        </li>
        <li>
         einen Tauschring für Growtainer (Komponenten) oder Gemeinschafts-Growtainergärten gründen
        </li>
        <li>
         Unternehmen motiveren, endlich Bio-Growtainer-Sets anzubieten oder selbst ein Unternehmen gründen, um Sets und nützliches Microgardenig-Zubehör zu verkaufen
        </li>
       </ul>
       <h3>
        <strong>
         Was macht der Verein?
        </strong>
       </h3>
       <p>
        Vernetzen, promoten, Kampagnen initiieren &amp; unterstützen. Siehe
        <a href="http://www.growtainer.de/?cat=1" target="_self">
         Neuigkeiten
        </a>
        und
        <a href="http://www.growtainer.de/?page_id=2">
         Verein
        </a>
       </p>
