             Kinderbehandlung
            </a>
            und der
            <a href="/erwachsene" title="Gehe zu: Erwachsene">
             Erwachsenenbehandlung
            </a>
            über die
            <a href="http://www.cmd-ibb.de" target="_blank" title="Gehe zu: www.cmd-ibb.de">
             Funktionsanalyse
            </a>
            bis hin zur
            <a href="http://www.schnarchen-ibb.de" title="Gehe zu: www.schnarchen-ibb.de">
             Schnarchtherapie
            </a>
            vor. Im Mittelpunkt stehen dabei für uns moderne, schonende und ästhetische Verfahren, z. B.
           </p>
           <ul class="auflistung">
            <li>
             Behandlung mit
             <a href="/kinder-und-jugendliche/herausnehmbare-zahnspange" title="Gehe zu: Herausnehmbare Zahnspangen">
              herausnehmbaren Zahnspangen
             </a>
            </li>
            <li>
             <a href="/erwachsene/aesthetik/selbstligierende-brackets" title="Gehe zu: 
 			">
              selbstligierende Brackets
             </a>
            </li>
            <li>
             die
             <a href="/erwachsene/aesthetik/lingualtechnik" title="Gehe zu: Lingualtechnik">
              Lingualtechnik
             </a>
             (innenliegende Zahnspange)
            </li>
            <li>
             und
             <a href="/erwachsene/aesthetik/aligner" title="Gehe zu: Aligner">
              durchsichtige Zahnschienen
             </a>
             (Aligner).
            </li>
           </ul>
           <p>
            Außerdem können Sie die behandelnden
            <a href="/praxis/ihr-praxisteam/aerzteteam" title="Gehe zu: Ärzteteam">
             Zahnärztinnen/Fachzahnärztinnen für Kieferorthopädie
            </a>
            sowie unser
            <a href="/praxis/ihr-praxisteam/anmeldung-verwaltung" title="Gehe zu: Anmeldung / Verwaltung">
             Team
            </a>
            kennenlernen und erfahren mehr über unsere
            <a href="/praxis/praxisphilosophie" title="Gehe zu: Praxisphilosophie">
             Praxisphilosophie
            </a>
            .
           </p>
           <p>
            Wir wünschen Ihnen viel Spaß beim Lesen und stehen Ihnen selbstverständlich bei Fragen gerne persönlich zur Verfügung.
           </p>
           <p>
            Ihre Dr. Sabine Mirsberger &amp; das Praxisteam

    function setCookie(cname,cvalue,exdays) { 
   var d = new Date(); 
   d.setTime(d.getTime() + (exdays * 24 * 60 * 60 * 1000)); 
   var expires = "expires=" + d.toGMTString(); 
   document.cookie = cname + "=" + cvalue + ";"+ expires + ";path=/" 
 } 
 function getCookie(cname) { 
   var name = cname; 
   var decodedCookie = decodeURIComponent(document.cookie); 
   var ca = decodedCookie.split(';'); 
   for(var i = 0; i < ca.length; i++) { 
     var c = ca[i]; 
     while (c.charAt(0) == ' ') { 
       c = c.substring(1); 
     } 
     if (c.indexOf(name) == 0) { 
       return c.substring(name.length, c.length); 
     } 
   } 
   return  ""; 
 }
