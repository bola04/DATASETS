     <h1>
      alter ego
     </h1>
     <p>
      alter ego ist eine als Buch konzipierte Arbeit
      <br/>
      aus Fotos, Texten und Zeichnungen, die eine
      <br/>
      frühe Standortbestimmung des künstlerischen
      <br/>
      ego von Hansik Gebert einleiten.
     </p>
     <h1>
      Konzepte
     </h1>
     <p>
      Die Konzepte zeigen eine Sammlung zeichner
      <br/>
      -ischer und schriftlicher Notizen, die den Fotos
      <br/>
      und Fotografischen Arbeiten vorausgingen, d.h.
      <br/>
      realisiert wurden, zugleich als reines Konzept
      <br/>
      eine eigenständigeArbeit darstellen.
     </p>
     <h1>
      LIFETIME
     </h1>
     <p>
      LIFETIME- ist eine Publikationreihe (5 Nummern),
      <br/>
      die sich mit Alltagsrealität anhand von Fotos, Texten,
      <br/>
      Dokumenten auseinandersetzt. Typisch sind der
      <br/>
      Fortsetzungscharakter, sowie die vom Rezipienten
      <br/>
      eingeforderte aktive Mitarbeit. Dies gilt auch für die
      <br/>
      Reihe WHOS WHO (publiziert mit Angelika Bliese)
      <br/>
      hier untersuchen Fotos, Texte, Fragebögen etc.
      <br/>
      das Spannungsfeld: Kunst und Alltagsleben.
     </p>
     <h1>
      Fotografischen Arbeiten
     </h1>
     <p>
      Die Fotografischen Arbeiten im Werk von Hansik
      <br/>
      Gebert stellen sich als künstlerische Formulierungen
      <br/>
      dar, die sich auch der Fotografie bedienen, d.h. es
      <br/>
      handelt sich entweder um überarbeitete Fotografien,
      <br/>
      oder die Fotografien sind Teil bzw. Bestandteil einer
      <br/>
      Arbeit.
     </p>
     <h1>
      Fotografien
     </h1>
     <p>
      Fotografien spielen als eigenständige Arbeit
      <br/>
      innerhalb des Werks von Hansik Gebert ebenso
      <br/>
      eine Rolle, wie als Versatzstücke innerhalb
      <br/>
      anderer Arbeiten wie z.B. den Inszenierungen.
      <br/>
      Einzeln oder in Serien zeigen sie oft in zeichen-
      <br/>
      hafter Formulierung Situationen wie Zusammen-
      <br/>
      hänge auf, die vom Rezipienten sprachlich
      <br/>
      erarbeitet werden sollen.
     </p>
     <h1>
      Filme
     </h1>
     <p>
      Filme oder Filmstills werden von Hansik Gebert
      <br/>
      in Inszenierungen, Videoinstallationen benutzt-
      <br/>
      oder als serienprints ( mehrere hundert Fotos
      <br/>
      als Ablauf) in verschiedensten Konstellationen
      <br/>
      innerhalb des Gesamtwerks eingesetzt.
     </p>
     <h1>
      300 Tage
     </h1>
     <p>
      300 Tage- bezeichnet ein Buchprojekt, das über
      <br/>
      dreihundert Seiten fortlaufend, in Form genauer,
      <br/>
      sich aufeinander beziehender Aufzeichnungen das
      <br/>
      Arbeiten und Denken von Hansik Gebert in einen
      <br/>
      Zusammenhang stellt.
      <br/>
      Die drei Bücher sind die Basis für die im zweiten
      <br/>
      Werkblock zusammengefaßten Arbeiten.
     </p>
