    <div class="item" id="e2">
     Programmierung für den Kraftwerksbedarf
    </div>
    <div class="item" id="e1">
     <br/>
     <br/>
     Produktbeschreibung:
     <br/>
     <br/>
     Das Programm "Form3DKessel" unterstützt den Betreiber
     <br/>
     von  Kesselanlagen bei der Dokumentation von Revisionen.
     <br/>
     Über  Jahre hinweg  wird   eine einheitliche Dokumentation
     <br/>
     geschaffen. Revisionsinformationen können auf einfachste
     <br/>
     Art angezeigt und analysiert werden.
     <br/>
     <br/>
     Das Programm "CoalRanking" berechnet den wirtschaftlichen
     <br/>
     Einsatz von Importkohle in Abhängigkeit der Kesselbauart und
     <br/>
     Feuerungssysteme. Außerdem wird die Verschlackungs- und
     <br/>
     Verschmutzungsneigung der Dampferzeuger berechnet.
     <br/>
     <br/>
     Das Programm "VE" berechnet die mögliche Fahrleistung
     <br/>
     von Ionenaustauschermassen in Vollentsalzungsanlagen in
     <br/>
     Abhängigkeit der Wasserqualität, sowie den theoretischen
     <br/>
     HCL- und NaOH Verbrauch. Anhand einfacher pH Wert
     <br/>
     Analysen des Bedienungspersonals ermittelt das Programm
     <br/>
     mögliche Ausfälle der VE Anlage durch Biologie, Kornbruch,
     <br/>
     Schlupf, Desinfektionsmittel u.ä..
    </div>
