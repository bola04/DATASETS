                Tokyo 10.2.3 erschienen
               </a>
              </h2>
              <span class="rdf-meta element-hidden" content="Tokyo 10.2.3 erschienen" property="dc:title">
              </span>
              <span class="rdf-meta element-hidden" content="0" datatype="xsd:integer" property="sioc:num_replies">
              </span>
              <span class="submitted">
               <span content="2018-03-16T14:27:24+01:00" datatype="xsd:dateTime" property="dc:date dc:created" rel="sioc:has_creator">
                Fr, 16/03/2018 - 14:27 —
                <span about="/user/1" class="username" datatype="" property="foaf:name" typeof="sioc:UserAccount" xml:lang="">

                </span>
               </span>
              </span>
              <div class="content clearfix">
               <div class="field field-name-body field-type-text-with-summary field-label-hidden">
                <div class="field-items">
                 <div class="field-item even" property="content:encoded">
                  <p>
                   Die Woche ist das Update zu Tokyo 10.2.3
                   <a href="http://blog.marcocantu.com/blog/2018-march-delphi-10-2-3.html">
                    erschienen
                   </a>
                   . Die Gerüchteküche hat es ja bereits im Vorfeld schon breit getreten: Das Mobile Add On ist nun Bestandteil von Delphi Professional. Kunden, die eine aktive Subscription haben, sollten inzwischen auch die Email erhalten haben, mit der sie einen Key für das AddOn anfordern können.
                   <br/>
                   Im "alten" Embarcaderoform / Newsgroup wurde darauf
                   <a href="https://forums.embarcadero.com/thread.jspa?threadID=291621&amp;tstart=0" target="_blank">
                    aufmerksam gemacht
                   </a>
                   , dass wohl ein neuer Installer für 10.2.3 verwendet wurde: Innosetup! Im Installationsverzeichnis von Tokyo (Embarcadero\Studio\19.0) findet sich tatsächlich eine unins000.dat, die von Innosetup stammt. Glückwunsch an Innosetup! Und natürlich an Embarcadero, dass sie sich für ein gutes Produkt entschieden haben :-)
                  </p>

                   Vom 25. bis 27. April präsentiert die Entwickler Akademie das nächste Delphi Code Camp in Frankfurt am Main. Das hochkarätige Training findet einmal im Jahr statt und bietet drei ganztägige und intensive Workshops für Delphi-Entwickler. Drei der bekannte Delphi-Experten - Bernd Ua, Stefan Glienke und Andrea Magni - vermitteln geballtes Praxiswissen zu wichtigen Delphi-Themen. In diesem Jahr stehen Delphi Internals, mit Delphi in die Azure Cloud und Delphi REST Library im Mittelpunkt des Programms. Alle Workshops können auch einzeln gebucht werden. Alle Infos auf

              <h2>
               <a href="/node/137">
                noch ein Update für Delphi 10.2 in 2018
               </a>
              </h2>
              <span class="rdf-meta element-hidden" content="noch ein Update für Delphi 10.2 in 2018" property="dc:title">
              </span>
              <span class="rdf-meta element-hidden" content="0" datatype="xsd:integer" property="sioc:num_replies">
              </span>
              <span class="submitted">
               <span content="2018-02-13T10:40:36+01:00" datatype="xsd:dateTime" property="dc:date dc:created" rel="sioc:has_creator">
                Di, 13/02/2018 - 10:40 —
                <span about="/user/1" class="username" datatype="" property="foaf:name" typeof="sioc:UserAccount" xml:lang="">
                 Lemmy
                </span>
               </span>
              </span>
              <div class="content clearfix">
               <div class="field field-name-body field-type-text-with-summary field-label-hidden">
                <div class="field-items">
                 <div class="field-item even" property="content:encoded">
                  <p>
                   <a href="https://community.embarcadero.com/view-profile/sarina-d" target="_blank">
                    Sarina Dupont
                   </a>
                   hat ein weiteres Update (10.2.3) für die nächsten Wochen
                   <a href="https://community.embarcadero.com/blogs/entry/exciting-rad-studio-developments-in-2018" target="_blank">
                    angekündigt
                   </a>
                   . Es werden VLC sowie FMX überarbeitet sowie für C++ div. neue Features geben (die eigentlich im 10.2.2er Update enthalten sein sollten lt.
                   <a href="https://community.embarcadero.com/article/16563-rad-studio-roadmap-september-2017" target="_blank">
                    Roadmap
                   </a>
                   ).
                   <br/>
                   Der
                   <a href="https://www.edv-buchversand.de/">
                    EDV-Buchversand
                   </a>
                   hat in einer Werbemail am 7.2.2018 bzgl. der Einstellung der Updateverkäufe übrigens folgendes geschrieben:
                   <br/>
                   [zitat]
                  </p>
                  <p>
                   *Gerüchten zufolge erscheint im März ein neues Release, bei dem das
                   <br/>
                   Mobile AddOn in der Professional Version schon enthalten sein soll.
                   <br/>
                   Mit einer gültigen Update Subscription können Sie dann auf dieses neue
                   <br/>
                   Release zugreifen.*
                  </p>
                  <p>
                   [/zitat]
                   <br/>
                   <br/>
                   Das wäre in der Tat eine interessante Entwicklung. Auf meine Nachfragen hin, ob sie eine Quellenangabe nennen können, kam leider nur zurück, dass das nicht möglich ist...
                   <br/>
                  </p>
                 </div>
                </div>
