       <h2>
        <span>
         <span>
          Willkommen bei Ihren Rechtsanwälten und Notaren in Wiesbaden
         </span>
        </span>
       </h2>
       <p>
        Die traditionsreiche Kanzlei PAULUS.WESTERWELLE zählt zu den alteingesessenen und führenden Rechtsanwalts- und Notariatskanzleien in Wiesbaden und Umgebung. In unserer Sozietät mit über 100jähriger Geschichte arbeiten sieben hoch qualifizierte Rechtsanwälte, von denen fünf gleichzeitig als Notare zugelassen sind.
        <br/>
        <br/>
        Notarielle Dienstleistungen gehören traditionell zu unseren wesentlichen Tätigkeitsschwerpunkten. Unser modernes, kundenorientiertes Anwaltsnotariat zählt zu den größten im Bezirk und wird seit Jahrzehnten überregional geschätzt. Wir entwerfen notarielle Urkunden individuell auf den Gebieten des privaten und gewerblichen Immobilienrechts, des Gesellschaftsrechts und der Nachfolge- und Vorsorgeplanung und zählen hier zu den führenden Anwaltsnotariaten in der Region Wiesbaden.
        <br/>
        <br/>
        Daneben begleiten und vertreten wir als Rechtsanwälte in überwiegend langfristigen Mandatsbeziehungen Familien, Städte und Gemeinden, mittelständige Unternehmen und Konzerne in nahezu allen Fragen des Wirtschaftsrechts. Unserem Rat vertrauen private und institutionelle Investoren, Projektentwickler, Bauträger, Banken und Versicherungen.
       </p>

 function slideShow(speed) {
 // set the opacity of all images to 0
 $('ul.slideshow li').css({opacity: 0.0});
 // get the first image and display it
 $('ul.slideshow li:first').css({opacity: 1.0}).addClass('show');
 // get the caption of the first image from the 'alt' attribute and display it
 var desc = $('ul.slideshow li.show').find('img').attr('alt');
 $('#slideshow-caption p').html(desc);
 // display the caption
 $('#slideshow-caption').css({opacity: 0.6, bottom: 0});
 if (desc.length > 6) {
 $('#slideshow-caption').addClass('dark');
 }
 // call the gallery function to run the slideshow
 var timer = setInterval('gallery()', speed);
 // pause the slideshow on mouse over
 $('ul.slideshow').hover(
 function () {
 clearInterval(timer);
 },
 function () {
 timer = setInterval('gallery()', speed);
 }
 );
 }
 function gallery() {
 $('ul.slideshow li').not('li.show').css('opacity', '0.0');
 //if no images have the show class, grab the first image
 var current = ($('ul.slideshow li.show') ?  $('ul.slideshow li.show') : $('#ul.slideshow li:first'));
 // trying to avoid speed issue
 if (current.queue('fx').length == 0) {
 // get the next image, if it reached the end of the slideshow, rotate it back to the first image
 var next = ((current.next().length) ? ((current.next().attr('id') == 'slideshow-caption') ? $('ul.slideshow li:first') : current.next()) : $('ul.slideshow li:first'));
 // get the next image caption
 var desc = next.find('img').attr('alt');
 // Remove the show class from the current image
 current.removeClass('show');
 // set the fade in effect for the next image, show class has higher z-index
 next.addClass('show').animate({opacity: 1.0}, 1000);
 // hide the caption first, and then set and display the caption
 $('#slideshow-caption').slideToggle(333, function () {
 $('#slideshow-caption p').html(desc);
 if (desc.length > 6) {
 $('#slideshow-caption').addClass('dark');
 } else {
 $('#slideshow-caption').removeClass('dark');
 }
 $('#slideshow-caption').slideToggle(333);
 });
 // hide the current image
 current.animate({opacity: 0.0}, 1000);
 }
 }

 $('#Navi' + current).addClass('naviOpen');

 function hideSubnavi(elm, leave) {

 $('#Navi' + elm).animate({'height': '0px'}, 50);

 }
