     <p>
      Herzlich willkommen auf den Seiten der Rechtsanwaltskanzlei Kreissl Morbach.
      <br/>
      <br/>
      Wir unterstützen Sie engagiert: beratend, als außergerichtliche Interessenver-
      <br/>
      treter oder als Prozessbevollmächtigte in Gerichtsverfahren.
      <br/>
      <br/>
      Für unsere Mandanten suchen wir die zügige, wirtschaftliche und praxisnahe
      <br/>
      Lösung, immer im Hinblick auf die aktuelle Rechtsprechung. Denn ebenso wie
      <br/>
      das Leben werden auch die Gesetze immer komplizierter. Daher ist eine Spezi-
      <br/>
      alisierung sinnvoll. Fragen Sie gerne alle Bereiche des Rechts an, wir sagen
      <br/>
      Ihnen ganz offen, welche Rechtsgebiete wir nicht bearbeiten.
      <br/>
      <br/>
      Unsere Dienste bieten wir Ihnen auch in englisch, französisch
      <br/>
      und italienisch an. Wir vertreten Sie bundesweit vor sämtlichen Amtsgerichten, Landgerichten und Oberlandesgerichten im Zivilrecht; vor sämt-
      <br/>
      lichen Gerichten in Strafsachen; in sämtlichen Instanzen der Arbeits-, Sozial-,
      <br/>
      Familien-, Finanz- und Verwaltungsgerichte. Gerichtstermine nehmen wir
      <br/>
      bundesweit persönlich wahr und bestellen keine Unterbevollmächtigten.
      <br/>
      <br/>
      <br/>
      <strong>
       Fremdsprachen
      </strong>
      <br/>
      <br/>
      Fremdsprachen gibt es für uns nicht. Im Zeitalter der Globalisierung ist weltweite Verständigung zwingend, weshalb wir zu beinahe allen Sprachen auf ein Übersetzungsbüro zurückgreifen (Übersetzungsbüro Behrens in Fulda 0661-51474).
      <br/>
      <br/>
      <br/>
      <strong>
       Mediation – Konflikte anders lösen
      </strong>
      <br/>
      <br/>
      Mediation bezeichnet die außergerichtliche Form, mit Hilfe eines unabhängigen
      <br/>
      Mediators Konflikte zu lösen. Sie beruht auf Freiwilligkeit, Vertraulichkeit,
      <br/>
      Informiertheit und Eigenverantwortlichkeit aller Parteien.
      <br/>
      <br/>
      Rechtsanwältin und Fachanwältin für Familienrecht Gabi Viehmann ist als aus-
      <br/>
      gebildete Mediatorin in unserer Kanzlei tätig. Sie ist Mitglied in der Arbeitsge-
      <br/>
      meinschaft Mediation im Deutschen Anwaltsverein. Auf die dortige vorzügliche
      <br/>
      Internetseite verweisen wir:
      <a href="http://Mediation.Anwaltverein.de" target="_blank">
       Mediation.Anwaltverein.de
      </a>
      <br/>
      <br/>
      Sprechen Sie uns an, wenn Sie Interesse an dieser Form der Konfliktlösung
      <br/>
      besitzen.
      <br/>
     </p>
