      <h1>
       'Eine Krawatte ist ein Kommunikationstrager'
      </h1>
      <div class="links">
       <a class="actionBtn knopMedium" href="/portfolio">
        <span class="text">
         unser Portfolio
        </span>
        <span class="up">
        </span>
        <span class="hover">
        </span>
        <span class="upRight">
        </span>
        <span class="hoverRight">
        </span>
       </a>
      </div>
      <div class="columnLeft">
       <p>
        Ein nicht zu unterschätzender Blickfang, das ist es. Eine Krawatte ist mehr als ein Pflichtteil aus Textil: sie kann eine stillvolle Erscheinung hervorheben oder abbrechen. Kreativität und Qualität sind dann auch unentbehrlich, denn eine Krawatte kommuniziert….
        <br/>
        <br/>
        Mit 40 Jahren Erfahrung und einer Vielfallt an anregenden Krawattenträgern, hat Rooymans Neckwear sich zu einem gut situierten Namen entwickelt. Die Kraft liegt in dem
        <strong>
         kontinuierlichen Fokus auf Qualität
        </strong>
        , in durchgehender Innovation, Antizipieren und in der Suche nach
        <strong>
         kreativen Herausforderungen.
        </strong>
       </p>
      </div>
      <div class="columnRight">
       <p>
        Rooymans Neckwear ist mehr als ein Zwischenhändler. In der Basis sind wir von Zuhause aus Hersteller. Auf Grund einer Firmenansicht kreieren unsere Designer eine grafische Umsetzung in einen frischen, modernen Entwurf. Unsere festen Webereien und Druckereibetriebe stellen aus diesen Mustern bezaubernde Krawatten und Halstücher her. Darüber hinaus haben wir eine umfangreiche
        <strong>
         Lagerauswahl
        </strong>
        , die es ermöglicht, schnell und flexibel zu liefern.
        <br/>
        <br/>
        <em>
         Erfahren Sie mehr über unsere
         <a href="/portfolio" target="_self">
          Einzigartigkeit
         </a>
         , besuchen Sie unsere
         <a href="/kontakt" target="_self">
          Ausstellungsräume
         </a>
         oder fordern Sie direkt einen
         <a href="/angebot-anfragen" target="_self">
          kostenlosen Entwurf
         </a>
         an.
        </em>
       </p>
      </div>
      <div class="columnLeft">
       <h2 class="blauw">
        Rooymans Neckwear plädiert für:
       </h2>
       <ul class="kernwaarden">
        <li>
         <strong>
          Wissen und Erfahrung:
         </strong>
         mehr als 40 Jahre Erfahrung
        </li>
        <li>
         <strong>
          Kreativität und Maßanfertigung:
         </strong>
         eigene Designer, die persönliche Produkte entwickeln.
        </li>
        <li>
         <strong>
          Zügigkeit und Service:
         </strong>
         immer einen kostenlosen Entwurf und Angebot innerhalb einer Woche.
        </li>
       </ul>

    function setCookie(name, value, days) {
         var expires = "";
         if (days) {
             var date = new Date();
             date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
             expires = "; expires=" + date.toUTCString();
         }
         document.cookie = name + "=" + (value || "") + expires + "; path=/";
     }
 
     function getCookie(name) {
         var nameEQ = name + "=";
         var ca = document.cookie.split(';');
         for (var i = 0; i < ca.length; i++) {
             var c = ca[i];
             while (c.charAt(0) == ' ') c = c.substring(1, c.length);
             if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length, c.length);
         }
         return null;
     }
 
     function eraseCookie(name) {
         document.cookie = name + '=; Max-Age=-99999999; path=/';
     }
 
     if (getCookie('cookieMessage')) {
         document.getElementById('cookie-message').classList.add('accepted');
     }
     else {
         document.getElementById('cookie-message').classList.remove('accepted');
     }
 
     function hideCookieMessage() {
