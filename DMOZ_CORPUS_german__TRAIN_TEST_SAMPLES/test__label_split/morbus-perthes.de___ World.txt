    <h2>
     <a name="00000093100a30301">
     </a>
     Warum gibt es diese Initiative
    </h2>
    <div class="sidepicture">
     <a href="./images/rzdmpilogo4c.jpg" target="_blank">
      <img alt="" border="0" height="79" src="./images/rzdmpilogo4c_160.jpg" width="160"/>
     </a>
     <h3>
      Logo Deutsche Morbus Perthes Initiative
     </h3>
     <p>
     </p>
    </div>
    <p style="">
     <b>
      <br/>
      <br/>
      <br/>
      Herzlich willkommen
      <br/>
      bei www.Morbus-Perthes.de
      <br/>
      <br/>
     </b>
     Die
     <b>
      DEUTSCHE MORBUS PERTHES INITIATIVE
     </b>
     wurde 1998 von Wolfgang Strömich, als ehemals selbst Betroffenen in Düsseldorf gegründet.
     <br/>
     <b>
      <br/>
      Wir sind eine Selbsthilfeorganisation der Gesundheitsselbsthilfe auf privater Basis, ohne Gewinnerzielungsabsicht. Wir arbeiten ehrenamtlich für die Krankheit Morbus Perthes.
      <br/>
      <br/>
     </b>
     "Mein
     <b>
      Morbus Perthes
     </b>
     heilte damals, mit viel Glück, völlig aus. Auch Spätfolgen habe ich keine. Ich hatte die Krankheit von 8-10 Jahren und wurde damals mit einer Thomas-Schiene behandelt. Dies macht man heute so nicht mehr.
     <br/>
     <br/>
     Als gutes Beispiel möchte ich meine Erfahrungen an aktuell betroffene Eltern weitergeben", so Wolfgang Strömich.
     <br/>
     <a href="./downloads/hit140917-menschen-im-1005-revier---wolfgang-s.mp3" title="">
      HIER
     </a>
     [2.241 KB]
  hören Sie ein Radio-Interview zur Verleihung des HelferHerzen Preises der dm Drogerie (2014)
     <br/>
     <br/>
     <a href="https://www.youtube.com/watch?v=DitORJlx_o8&amp;t=48s" target="_blank" title="">
      Ich möchte Sie jetzt ermutigen dieses Video an zu schauen.
     </a>
     <br/>
     <b>
      <br/>
     </b>
     Zu der orthopädischen Kinderkrankheit
     <b>
      Morbus Perthes
     </b>
     waren aber damals (1998) noch so gut wie keine Informationen im Internet zu finden. Als bestens ausgeheilter Patient entschloß ich mich, eine eigene Homepage dafür zu erstellen.
     <br/>
     <br/>
     Seither meldeten sich immer mehr betroffene Eltern, aber auch ehemalige Betroffene bei mir. Wir freuen uns über jeden Eintrag in unser
     <a href="http://www.iphpbb.com/board/f-18483519nx100690-1.html" target="_blank" title="">
      Gästebuch
     </a>
     und auch bei
     <a href="http://www.facebook.com/pages/Deutsche-Morbus-Perthes-Initiative/203562023038917" target="_blank" title="">
      Facebook
     </a>
     . Bitte keine kompletten Krankengeschichten in Facebook eingeben! (Wenn Sie zu Facebook gehen möchten, schauen Sie bitte vorher in unser
     <a href="./impressum.html#00000093020d9f436" title="">
      IMPRESSUM
     </a>
     )
     <br/>
     <br/>
     Manchmal schreiben Eltern uns einige anerkennende Zeilen. Herzlichen Dank dafür.
     <br/>
     <a href="./downloads/zitate-zu-mp-initiative-und-mp-tag-.pdf" title="">
      HIER
     </a>
     [51 KB]
  kann man lesen, warum Sie mich anrufen sollten...
     <br/>
     <br/>
     <b>
      Krankengeschichten und Kontaktwünsche schreiben Sie bitte ins
     </b>
     <a href="http://www.perthes-forum.de" target="_blank" title="">
      Forum
     </a>
     <b>
      <br/>
      (Diese Posts können auch über Google gefunden werden. Bitte nur wenn Sie möchten, dass Ihre Posts öffentlich werden könnten.
      <br/>
      Auch über facebook werden Ihre Posts öffentlich. Überlegen Sie genau, ob Sie das wollen).
      <br/>
      <br/>
     </b>
     Mittlerweile erhalten wir nicht nur Anfragen aus Deutschland, sondern auch aus Österreich, der Schweiz, aus Norwegen, Griechenland, Italien, Russland, aus Japan, Brasilien, Usbekistan, Südafrika, Ungarn
     <b>
      <br/>
      <br/>
     </b>
     Seit 2003 veranstaltet die
     <b>
      DEUTSCHE MORBUS PERTHES INITIATIVE
     </b>
     , als einzige Selbsthilfegruppe im deutschsprachigen Raum für den Morbus Perthes, regelmäßig jährlich den
     <a href="./perthestage/index.html#510962985909db001" title="">
      DEUTSCHEN MORBUS PERTHES TAG
     </a>
     , der allein dieser Krankheit gewidmet ist.
     <br/>
     <br/>
     Er findet in jedem Jahr, in Kooperation mit einer Orthopädischen Universitäts-Kinderklinik Ende Mai/Anfang Juni an einem Sonntag statt.
     <br/>
     <br/>
     <b>
      Die Referenten sind international anerkannte Kinderorthopäden.
     </b>
     <br/>
     <br/>
     Dennoch ist, gerade in dieser Situation, in der Sie sind, ein persönlicher Kontakt zu jemandem, der sich als Nichtmediziner schon lange mit der Krankheit Morbus Perthes beschäftigt, außerordentlich wichtig. Denn hier hat sich ein Erfahrungswissen von einigen hundert Fällen angesammelt. Natürlich geben wir keine medizinische Beratung, denn das möchten wir lieber den anerkannten Fachleuten überlassen, die wir kennen und Ihnen gern benennen.
     <br/>
     <br/>
     <br/>
     <br/>
     Anrufe von Vertretern, Verkäufern und sonstige gewerbliche Anrufe sind nicht willkommen.
     <br/>
     Für Medienmitarbeiter gilt das selbstverständlich nicht.
     <br/>

    <h2>
     <a name="267145a29a1467103">
     </a>
     Helfen Sie jetzt einem kranken Kind
    </h2>
    <div class="sidepicture">
     <a href="./images/web-teddy-mit-kruecken.jpg" target="_blank">
      <img alt="" border="0" height="160" src="./images/web-teddy-mit-kruecken_160.jpg" width="160"/>
     </a>
     <h3>
     </h3>
     <p>
      Möchten Sie Tröstebär Sponsor werden? Schenken Sie einem kranken Kind einen Tröstebär zur besseren Genesung.
     </p>
    </div>
    <p style="">
     <br/>
     Jedes Jahr erkranken ca. 2.500 Kinder zwischen 3 und 15 Jahren plötzlich an der orthopädischen Krankheit Morbus Perthes. Zwischen 2-5 Jahren dauert die Ausheilung und die Kinder müssen sich durch den Heilungsprozess quälen.
     <br/>
     <br/>
     Sie können diesen Kindern helfen, in dem Sie Sponsor für Tröstebären werden, den die Kinder im Krankenhaus durch den behandelnden Arzt geschenkt bekommen. Der Tröstebär wird Freund und Vertrauter der Kinder, denn er hatte ja auch diese Krankheit und hat sie überstanden. Der Teddy hilft die Selbstheilungskräfte der Kinder zu stärken.
     <br/>
     <br/>
     Wenn Sie über ein gewisses Maß an Empatie verfügen, können Sie einem kleinen Menschen etwas Gutes tun und ihm solch einen Tröstebär zukommen lassen. Wir organisieren das für Sie. Wir stehen in Kontakt zu medizinischen Spezialisten für diese Krankheit und können den kranken Kindern diesen kleinen Tröstebären überreichen lassen.
     <br/>
     <br/>
     Wenn Sie Kinder in diesem Alter haben, die gesund sind; sie selbst als Kind von der Krankheit betroffen waren, oder einfach nur ein gutes Geschäftsjahr hatten und aus Dankbarkeit kranken Kindern helfen möchten, die temporär in einer schwer überwindlichen Situation sind, dann werden Sie bitte Tröstebär-Sponsor. Das tut Ihnen und den Kindern gut.
     <br/>
     <br/>
     Für nur einmalige 20 €uro können Sie einem kranken Kind einen Benni Tröstebär zukommen lassen.
