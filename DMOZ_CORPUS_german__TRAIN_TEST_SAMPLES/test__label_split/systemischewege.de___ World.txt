       Vorwärts kommen - Bewegung
       <br/>
       <br/>
      </font>
      <font color="#666666" face="Arial, Helvetica, Verdana" size="2">
       <b>
        Vielleicht
       </b>
       <br/>
       haben Sie gerade Probleme oder Sorgen, persönlich oder innerhalb der Familie. Sie wissen im Moment nicht weiter und suchen deshalb nach einer Lösung.
       <br/>
       <br/>
       <br/>
       <b>
        Vielleicht
       </b>
       <br/>
       haben Sie gerade das Gefühl fest zu stecken und wie ohnmächtig der problematischen Situation gegenüber zu stehen.
       <br/>
       <br/>
       <br/>
       <b>
        Sie wünschen sich
       </b>
       ,
       <br/>
       dass sich etwas verändert oder bewegt und  brauchen neue Impulse.
       <br/>
       <br/>
       <br/>
       <b>
        In einer Krise liegt die Chance
       </b>
       ,
       <br/>
       innezuhalten und zu schauen, welchen Weg möchte ich weitergehen?

       Systemische Beratung
       <br/>
       <br/>
      </font>
      <font color="#666666" face="Arial, Helvetica, Verdana" size="2">
       Probleme werden nicht isoliert betrachtet, sondern im Zusammenhang mit Ihren bisherigen Erfahrungen und dem Umfeld, in dem Sie leben.
       <br/>
       <br/>
       <b>
        Ich arbeite lösungsorientiert, das bedeutet:
       </b>

     <td valign="top">
      <font color="#666666" face="Arial, Helvetica, Verdana" size="2">
       <ul>
        <li>
         Konzentration auf die Lösung
        </li>
        <li>
         Gemeinsame Arbeit an den von Ihnen gewünschten Veränderungen
        </li>
        <li>
         Entwicklung neuer Perspektiven und eigener Wege
        </li>
       </ul>
       Systemisch bedeutet auch, die Stärken von Ihnen und/oder Ihrer Familie zu sehen und zu nutzen. 
 		Ich unterstütze Sie dabei, diese Stärken (wieder) zu entdecken und mit dieser Kraft auch neue Wege zu gehen.
       <br/>
       <br/>
       <b>
        Eine kleine Bewegung kann vieles verändern.
       </b>
       <br/>
       Mit den ersten Lösungsschritten können Sie sofort beginnen.
      </font>
      <font color="#990000" face="Arial, Helvetica, Verdana" size="2">
       <br/>
       <br/>
       <br/>
       <br/>
       Systemische Kinder- und Jugendberatung
       <br/>
       <br/>
      </font>
      <font color="#666666" face="Arial, Helvetica, Verdana" size="2">
       In der Erziehung von Kindern -und Jugendlichen stoßen wir als Eltern immer wieder auf Situationen, die uns verunsichern:
       <br/>
       <br/>
      </font>
      <font color="#666666" face="Arial, Helvetica, Verdana" size="2">
       „Warum verhält sich mein Kind so …?”
       <br/>
       „Warum hört es nicht zu …?”
       <br/>
       „Warum macht es nicht das, was ich will …?”
       <br/>
       „Warum ist es so laut, oder auch so gehemmt …?”
       <br/>
       <br/>
       Vielleicht machen Sie sich gerade auch Sorgen um Ihr Kind und erleben Erziehung als anstrengend oder belastend und sind verunsichert, 
         wie Sie Ihrem Kind am besten weiterhelfen können.
       <br/>
       <br/>
       Manchmal ist es gut, sich von außen fachliche Unterstützung zu holen, auch um gegebenenfalls den Kindern individuelle professionelle Hilfe anzubieten.
       <br/>
       <br/>
       Kinder- und Jugendberatung/Therapie ist unter anderem bei folgenden Themen hilfreich:
       <ul>
        <li>
         Unerwünschtes Sozialverhalten
        </li>
        <li>
         Selbstwertschwierigkeiten
        </li>
        <li>
         Ängste, Unsicherheiten
        </li>
        <li>
         Starke Unruhe, Unaufmerksamkeit
        </li>
        <li>
         Schulschwierigkeiten
        </li>
        <li>
         ADHS/ADS
        </li>
        <li>
         Einnässen/Einkoten
        </li>
        <li>
         Probleme in der Familie
        </li>
        <li>
         und natürlich auch bei anderen belastenden Verhaltensweisen
        </li>
       </ul>
       Als Mutter zweier Kinder kenne ich selber viele große und kleine Sorgen innerhalb der Erziehung.
       <br/>
       <br/>
       <b>
        Sie können Sich gerne vertrauensvoll an mich wenden.
       </b>
      </font>
