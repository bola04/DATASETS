       </h1>
       <h2>

       </h2>
      </div>
      <article>
       <p>
        Ankommen, Entspannen und Wohlfühlen im
        <a href="https://www.hotel-heusser.de/4-sterne-hotel-in-rheinland-pfalz" title="4 Sterne Wellnesshotel">
         4 Sterne Hotel
        </a>
        in der Pfalz.
       </p>
       <p>
        Sie suchen ein
        <a href="https://www.hotel-heusser.de/hotel" title="Hotel Rheinland Pfalz das Gartenhotel Heusser">
         Hotel in Rheinland-Pfalz
        </a>
        <strong>
         mit Klimaanlage
        </strong>
        für Ihre persönliche Auszeit und möchten während Ihres
        <a href="https://www.hotel-heusser.de/entspannter-kurzurlaub-im-wellnesshotel-heusser-in-rheinland-pfalz" title="Kurzurlaub Rheinland Pfalz">
         Kurzurlaubs
        </a>
        in Bad Dürkheim die schöne Pfalz mit Weinbergen, Pfälzerwald und Pfälzer Lebensfreude näher kennenlernen?
       </p>
       <p>
        Wir laden Sie herzlich ein unser
        <strong>
         Wellnesshotel in Bad Dürkheim
        </strong>
        zu besuchen. In unser Pfalz Hotel sind Sie in kürzester Zeit aus dem Raum Frankfurt, Mainz, Wiesbaden sowie Karlsruhe und Stuttgart angereist und Ihr Urlaub an der Weinstraße kann beginnen.
       </p>
       <p>
        Das Gartenhotel Heusser liegt im Süden von Bad Dürkheim und ist von allen Hotels in Rheinland-Pfalz wohl das einzige Hotel, welches inmitten einer großen asiatischen Gartenanlage liegt.
       </p>
       <p>
        Ein japanisches Teehaus und Koi-Teiche schaffen eine ganz besondere Wohlfühl-Atmosphäre. In einem unserer 93 Hotelzimmer mit Balkon oder Terrasse, genießen Sie Ihren Aufenthalt mit jeglichem Komfort.
        <a href="https://www.hotel-heusser.de/hotel-mit-klimaanlage-in-rheinland-pfalz" title="Hotelzimmer mit Klimaanlage">
         Alle Hotelzimmer haben eine Klimaanlage
        </a>
        , welche Sie individuell regeln können. Kostenfreies W-LAN im Hotel Heusser, die Nutzung des Hallenbades und der Sauna-Landschaft in der WellnessOase ist für unsere Hotelgäste inklusive.
       </p>
       <p>
        Wellness in der Pfalz erleben Sie in Heussers
        <a href="https://www.hotel-heusser.de/arrangements/wellness-arrangements" title="Wellness Angebote">
         Wellness
        </a>
        Oase. Buchen Sie Ihren Kurzurlaub in die Pfalz oder einen DaySpa in Bad Dürkheim und lassen Sie sich von unserem umfangreichen Angebot für
        <a href="https://www.hotel-heusser.de/kurzurlaub-wellnesshotel-rheinland-pfalz" title="Kurzurlaub im Wellnesshotel">
         Kurzurlaub
        </a>
        und
        <a href="https://www.hotel-heusser.de/wellnessurlaub-pfalz" title="Wellnessurlaub Rheinland Pfalz">
         Wellnessurlaub
        </a>
        in Bad Dürkheim inspirieren.
       </p>
       <p>
        Auch für Ihre Feier, und Ihre Tagung oder Seminar in Bad Dürkheim, finden Sie im Gartenhotel Heusser den passenden Veranstaltungsraum.
       </p>
       <p>
        Gerne sind wir mit unseren Erfahrungen für Ihre Wünsche und Ihren Urlaub in der Pfalz für Sie da.
       </p>
       <p>
        Ob Ihre Reise nach Bad Dürkheim ganz privat oder geschäftlich ist, im Gartenhotel Heusser werden Sie auf jeden Fall immer mit viel Liebe zum Detail verwöhnt.
       </p>
       <p>
        Familie Köhler und Berwing und das gesamte Hotel-Heusser-Team heißen Sie Herzlich Willkommen in Bad Dürkheim an der Weinstraße.

 function getCookie(cname) {
     var name = cname + "=";
     var ca = document.cookie.split(';');
     for(var i=0; i<ca.length; i++) {
         var c = ca[i].trim();
         if (c.indexOf(name) == 0) return c.substring(name.length,c.length);
     }
     return "";
 } 
 
 function _checkCookiesPolicy() {
 	if (getCookie("CookiesPolicyAccepted") != "")  {
 		var cookieDiv = document.getElementById('cookieWarning');
 		if (cookieDiv) {
 			cookieDiv.style.display='none';
 		}
 	}
 }
 
 function _acceptCookiesPolicy() {
 	var exdate=new Date();
 	exdate.setDate(exdate.getDate() + 365);
 	document.cookie='CookiesPolicyAccepted=1; expires='+exdate.toUTCString();
 	document.getElementById('cookieWarning').style.display='none';
 }
