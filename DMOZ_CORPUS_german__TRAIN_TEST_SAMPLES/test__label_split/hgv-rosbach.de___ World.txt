        <br/>
        Der Heimatgeschichtsverein 1984 Rosbach v. d. H. e. V. will die Kenntnis der Heimatgeschichte fördern, pflegen und der Bürgerschaft nahe bringen.
        <br/>
        <br/>
        <strong>
         Zu diesem Zweck
        </strong>
        <br/>
        <br/>
        verwaltet und pflegt der Verein ehrenamtlich die Stadtarchive von Ober- und Nieder-Rosbach,
        <br/>
        <br/>
        gibt jährlich die Rosbacher Geschichtsblätter heraus und veröffentlicht Arbeiten zur Ortsgeschichte,
        <br/>
        <br/>
        organisiert Vorträge,
        <br/>
        <br/>
        veranstaltet Ausstellungen,
        <br/>
        <br/>
        bietet Stadtbesichtigungen an,
        <br/>
        <br/>
        führt die jährliche Blütenwanderungen mit wechselnden Themen zu Heimatgeschichte durch,
        <br/>
        <br/>
        betreut Jugendliche im Rahmen des Ferienspiel-Programms der Stadt Rosbach
        <br/>
        <br/>
        <br/>
        <strong>
         und
        </strong>
        <br/>
        <br/>
        <br/>
        <strong>
         berät alle Bürger unentgeltlich bei Fragen zur Familien-, Orts- und Heimatgeschichte.
        </strong>
