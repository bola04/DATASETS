      <h3>
       Ferienwohnung  Landsberg am Lech für 4-8 Personen
      </h3>
      <br/>
      Die geräumige, stimmungsvolle
      <a class="textlink" href="die_ferienwohnung.htm" title="Ferienwohnung">
       Ferienwohnung
      </a>
      im 1. und 2. Stock des Hauses hält neben  3 Schlafzimmern, Küche,  Esszimmer,  Badezimmer und Wohnzimmer noch 2 Schlafgelegenheiten in den Galerien für Sie bereit.
      <br/>
      <br/>
      <br/>
      <h3>
       Quartier für 1 Person
      </h3>
      <br/>
      Das gemütliche
      <a class="textlink" href="das_quartier.htm" title="Altstadt-Quartier">
       Altstadt-Quartier
      </a>
      im  EG  mit eigenem Eingang, Badezimmer und Kochgelegenheit ist die ideale Unterkunft für eine Person.
      <br/>
      <br/>
      <br/>
      <h3>
       Ferienwohnung + Quartier = Ferienhaus
      </h3>
      <br/>
      Ab 7 Personen empfehlen wir, das
      <a class="textlink" href="beschreibung_haus_kloesterl_57.htm" title="Ferienhaus">
       Ferienhaus
      </a>
      als Ganzes zu mieten. 
 	Sie verfügen dann über 2 Badezimmer bzw. noch mehr Platz und Komfort.
      <br/>
      <br/>
      <br/>
      Nehmen Sie einfach Kontakt  auf unter
      <strong>
       08191 / 92 14 73
      </strong>
      .
      <br/>
      <br/>
      <br/>
      <h3>
       Wer  übernachtet denn hier?
      </h3>
      <br/>
      Ebenso buntgemischt wie  die früheren Bewohner ist das Potpourri  an Gästen heute:
      <strong>
       Fest- und Feriengäste, Arbeiter  und Durchreisende
      </strong>
      genießen Ihren  Aufenthalt im besonderem Ambiente des Ferienhauses.
      <br/>
      <br/>
      Besonders geschätzt wird es
      <strong>
       während des Oktoberfestes
      </strong>
      von  internationalen Gästen (Italien, Spanien, Frankreich und  England).
      <br/>
      <br/>
      Als besonderen Mehrwert  wird es empfunden von Monteuren und
      <strong>
       Mitarbeitern verschiedener Firmen
      </strong>
      aus nah  und fern.
      <br/>
      <br/>
      Sehr gerne wird es   auch  von Landsbergern    genutzt zur
      <strong>
       Unterbringung von Verwandten und Freunden
      </strong>
      bei Familienfeiern, Jubiläen,  Hochzeiten usw. (Ferienwohnung u. Quartier bieten reichlich Gelegenheit, sich  zusammen zu  setzen, und auch Platz genug, sich  mal zurückzuziehen.)
      <br/>
      <br/>
      <br/>
      <h3>
       Wer wohnte hier früher?
      </h3>
      <br/>
      <strong>
       Gesicherte Erkenntnisse über Bewohner
      </strong>
      des Hauses liegen  "erst" seit 1702 vor, als hier  Barthlme Pechler  ansässig war. Darauf folgte eine Reihe von
      <strong>
       Zimmerleuten
      </strong>
      ,  mehrere  Taglöhner und
      <strong>
       Lohnkutscher
      </strong>
      .
      <br/>
      <br/>
      Aus dem Entwässerungsplan der Teil-Sanierung um 1900 geht als Besitzer ein Herr Kaufmann (Privatier) hervor.
 	Der Plan zeigt auch im Erdgeschoss 2 Ställe und eine Remise. Der Erzählung nach hatten die Besitzer bis lange nach dem Krieg Hühner im hinteren EG und bis 1960 noch   ein Schwein, dann wurde die Viehhaltung aufgegeben.
      <br/>
      <br/>
      1922   ging das Haus an den Schreiner  Lorenz Schäffler und danach  an den Förster  Drexl mit Familie, bevor es saniert  und von Architekten, Bautechnikern, Antiquitätenhändlern, Geschäftsleuten und Maklern  bewohnt wurde. Seit 2006 wird es nun als Ferienwohnung genutzt.
      <br/>
      <br/>
      <br/>
      <h3>
       Schauen Sie doch einfach mal rein!
      </h3>
      <br/>
      Sie möchten das Ferienhaus reservieren und  gerne vorher besichtigen?
      <br/>
      Wir zeigen es Ihnen gerne.
      <a class="menuelink" href="http://maps.google.de/maps?f=q&amp;hl=de&amp;q=kl%C3%B6sterl+57&amp;sll=48.047917,10.882476&amp;sspn=0.153774,0.31929&amp;ie=UTF8&amp;om=1&amp;ll=48.047189,10.876422&amp;spn=0.019222,0.039911&amp;z=15" target="_blank">
       <u>
        Klösterl 57
       </u>
      </a>
      <br/>
      <br/>
      Nutzen Sie    die  Gelegenheit zu einem Spaziergang im
      <strong>
       Wildpark
      </strong>
      ,   einer Wanderung zur
      <a class="textlink" href="http://de.wikipedia.org/wiki/Teufelsk%C3%BCche_%28Landsberg_am_Lech%29" target="_blank" title="Teufelsküche Landsberg">
       Teufelsküche
      </a>
      oder    zur Besichtigung der  neu gestalteten Flaniermeile an der
      <strong>
       Lech-Promenade
      </strong>
      .
