      <h2>
       Unsere neuesten Seminare
      </h2>
      <p>
       Regelmäßig nimmt tutego neue Seminare in das Kursprogramm auf. Bleiben Sie mit unseren Seminaren immer auf dem neuesten Stand.
      </p>
      <div class="flexrow">
       <div class="col col1_3">
        <h4>
         Spring Cloud
        </h4>
        <p>
         Microservices sind ein Software- und Organisationsmuster, um Geschäftsprozesse und -werte schnell zu realisieren. Spring Boot und Spring Cloud sind eine mächtige Kombination für moderne Microservice-Architekturen.
         <a href="seminare/java-schulung/spring-cloud-kurs.html">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         Vaadin RIA-Framework
        </h4>
        <p>
         Vaadin ist ein entwicklerfreundliches Web-Framework für browserbasierte Ajax-Anwendungen. Das quelloffene Framework erlaubt die schnelle Umsetzung von Rich Internet Applications (RIA), in dem Entwickler vollständig in Java arbeiten können.
         <a href="http://tutego.de/g/VAADIN/">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         Atlassian JIRA
        </h4>
        <p>
         JIRA ist ein Produkt von Atlassian, was Entwickler und Verantwortliche bei der Planung und Verfolgung der Aufgaben in Projekten unterstützt und alle Informationen zentralisiert.
         <a href="http://tutego.de/g/JIRA/">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         C++11: Neuerungen im C++-Standard
        </h4>
        <p>
         Etliche Elemente kamen in C++11 hinzu, die es zu einer modernen Sprache für Systemprogrammierung und Bibliotheken machen. Teilnehmer bekommen einen umfassenden Einblick in den neuen Standard, angefangen von den Spracheigenschaften über die Anpassungen der C++ Standardbibliothek.
         <a href="seminare/programmiersprachen-schulung/c++11-kurs-cpp11-schulung.html">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         OP-Mapping mit NHibernate
        </h4>
        <p>
         In der .NET Welt steht NHibernate alternativ zum Entity Framework (EF). Der Kurs geht aufühlich auf den flexiblen OR-Mapper ein und gibt Teilnehmern einen tiefen Einblick in die Anpassungsmöglichkeiten.
         <a href="seminare/c-sharp-.net-schulung/NHibernate-Kurs-.NET-ORM-NHibernate-Seminar.html">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         Dependency Injection/Eventing mit CDI
        </h4>
        <p>
         CDI wurde als Teil vom Java EE 6-Standard (JSR-299) entwickelt und wartet mit Dependency-Injection, einem wohldefinierten Lebenszyklus und Ereignisbehandlung auf.
         <a href="http://tutego.de/g/CDIJAVAEE/">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         AngularJS JavaScript-Framework
        </h4>
        <p>
         Google und die Open-Source-Community gestalten mit AngularJS ein JavaScript-Framework zum Aufabau moderner Web-Applikationen unter Berücksichtigung des Model-View-Whatever-Prinzips.
         <a href="http://tutego.de/g/ANGULARJS/">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         Geschäftsprozesse mit Windows Workflow Foundation
        </h4>
        <p>
         Windows Workflow Foundation (WF) ermöglicht den Entwurf aus Ausführung von Geschäftsprozessen in .NET-Applikationen.
         <a href="seminare/c-sharp-.net-schulung/Windows-Workflow-Foundation-Kurs.html">
          <b>
           mehr ›
          </b>
         </a>
        </p>
       </div>
       <div class="col col1_3">
        <h4>
         SOA mit Windows Communication Foundation (WCF)
        </h4>
        <p>
         Microsoft bietet mit der Windows Communication Foundation (WCF) ein umfassendes Framework zum Aufbau verteilter Service-orientierter Anwendungen.
         <a href="seminare/c-sharp-.net-schulung/Windows-Communication-Foundation-WCF-Kurs.html">
          <b>
           mehr ›
          </b>
         </a>
        </p>
