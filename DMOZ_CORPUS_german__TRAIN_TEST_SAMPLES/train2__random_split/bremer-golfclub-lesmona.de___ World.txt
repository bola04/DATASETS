 						function createCookie(name,value,days) {
 							if (days) {
 								var date = new Date();
 								date.setTime(date.getTime()+(days*60*1000));
 								var expires = '; expires='+date.toGMTString();
 							} else var expires = '';
 							document.cookie = name+'='+value+expires+'; path=/';
 						}
 
 						function readCookie(name) {
 							var nameEQ = name + '=';
 							var ca = document.cookie.split(';');
 							for(var i=0;i < ca.length;i++) {
 								var c = ca[i];
 								while (c.charAt(0)==' ') c = c.substring(1,c.length);
 								if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
 							}
 							return null;
 						}

         <h1 class="tiny_p" style="text-align:center">
          Herzlich willkommen...
         </h1>
         <p class="tiny_p">
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
          Wir freuen uns, dass Sie sich auf unserer Homepage über unseren Club, unsere Angebote und unseren Platz informieren und hoffen, Sie demnächst hier begrüßen zu können.
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
         </p>
         <p class="tiny_p" style="margin-bottom:0cm; text-align:center">
          <span style="color:#006400">
           <span style="font-size:20px">
            <strong>
             "Bei uns ist Golf Volksport!"
            </strong>
           </span>
          </span>
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
          „Klein aber Fein, so könnte man die Anlage des Bremer Golfclub Lesmona beschreiben“.
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
          Dieses Fazit hören wir immer wieder von unseren Gästen, wenn sie nach einer Runde auf der Terasse ihr verdientes kühles Getränk genießen und zwanglos mit uns ins Gespräch kommen.
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
          Mit 3324 Metern ist es sicherlich einer der kürzeren 18-Loch Plätze. Doch die engen Spielbahnen, einige Wasserhindernisse und die gut verteidigten kleinen Grüns erfordern ein äußerst präzises Spiel. Eine Herausforderung, die auch bei manchem Single-Handicapper schon Eindruck hinterlassen hat.
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
         </p>
         <p class="tiny_p" style="margin-bottom:0cm">
          Neben dem sportlichen Aspekt des Golfspielens steht die Freundschaft und die Geselligkeit im Mittelpunkt des Clublebens. Quer durch alle Gesellschaftsschichten verbindet uns die Leidenschaft für dieses wunderbare Hobby. Daher freuen wir uns über jeden Gast.
         </p>
