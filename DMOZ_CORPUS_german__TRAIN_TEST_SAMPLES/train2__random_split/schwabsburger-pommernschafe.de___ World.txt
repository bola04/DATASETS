          <br/>
          <h1>
           Willkommen auf unserer Seite
          </h1>
          <br/>
          <p>
           Am Fuße der, aus der Stauferzeit stammenden,
          </p>
          <br/>
          <br/>
          <div align="center">
           <a href="http://www.nierstein.de/Schwabsburg/Schwabsburg.htm" target="blank">
            Schwabsburg
           </a>
          </div>
          <br/>
          <br/>
          <p>
           - die auch unserem Dorf den Namen gab - leben wir.
           <br/>
           <br/>
           Hier haben wir seit November 2002 unsere kleine Schafherde von Rauhwolligen Pommerschen Landschafen. Die Tiere
  helfen uns bei der Sauberhaltung ehemaliger Weinberge und leisten somit einen Beitrag zur Landschaftspflege.
           <br/>
           <br/>
           Nach eingehenden Gesprächen mit anderen Züchtern dieser Rasse haben wir uns 2004 zur Herdbuchzucht entschlossen und wollen unsere
 Herde nach und nach ausbauen.
           <br/>
           <br/>
           Wie kommt man in Rheinhessen zu Rauhwolligen Pommerschen Landschafen?
           <br/>
           <br/>
           Nun, wir haben uns lange mit dem Thema "Schafe" beschäftigt.
           <br/>
           <br/>
          </p>
          <ol>
           <li>
            Es sollte eine Landschafrasse sein.
           </li>
           <br/>
           <br/>
           <li>
            Die Tiere sollten nicht zu groß sein, damit man gut mit ihnen umgehen kann.
           </li>
           <br/>
           <br/>
           <li>
            Sie sollten vor allen Dingen robust und nicht so krankheitsanfällig sein.
           </li>
           <br/>
           <br/>
          </ol>
          <p>
           Bei unseren Recherchen - auch im Internet - sind wir dann auf die Rasse der Rauhwolligen Pommerschen Landschafe gestoßen.
 Beim Besuch eines Züchters in unserer Nähe war es dann Liebe auf den ersten Blick.
           <br/>
           <br/>
           Alles Wissenswerte über diese Schafrasse findet ihr unter
           <br/>
           <br/>
          </p>
          <div align="center">
           <a href="rauhwoller.html">
            Rauhwoller
           </a>
          </div>
          <br/>
          <br/>
          <p>
           und Bilder unserer "Mädels" unter
          </p>
          <br/>
          <div align="center">
           <a href="bilder_2003.html">
            Bilder
           </a>
          </div>
          <p>
           Sollte etwas fehlen oder sollten Fragen bestehen, so geben wir gerne Auskunft.
           <br/>
           <br/>
          </p>
