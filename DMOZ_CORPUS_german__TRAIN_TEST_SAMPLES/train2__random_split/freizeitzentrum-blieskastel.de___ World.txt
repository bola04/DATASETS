              </strong>

             <p>

              <br/>

             </p>
             <p>
              In einer Umfrage durch SAAR TV wurde das Freizeitzentrum Blieskastel  unter die zehn schönsten Schwimmbäder im Saarland bewertet.

           <div class="page-header">
            <h2 class="csc-firstHeader">
             Schwimmen - eine gesunde Sportart
            </h2>
           </div>
          </div>
          <p style="display: block;">
           Beim Schwimmen werden alle Muskelgruppen, insbesondere der Halte- und Stützapparat, aktiviert. Durch den Wasserauftrieb werden Belastungen, die bei Ausdauersportarten an Land auf Knochen und Muskulatur einwirken, weitgehend vermieden. In vielen Studien wurde nachgewiesen, dass körperlich aktive Menschen Krankheiten und Stresssituationen besser bewältigen. Regelmäßiges Schwimmen mindert den Einfluss von Risikofaktoren wie beispielsweise Bluthochdruck oder Übergewicht.
          </p>
          <h2 style="display: block;">
           Wunderwirkungen des Wassers
          </h2>
          <p style="display: block;">
           <strong>
            Abhärtung
           </strong>
           : Der Kältereiz des Wassers trägt zur Abhärtung bei und schützt vor Erkältungskrankheiten.
          </p>
          <p style="display: block;">
           <strong>
            Atmung
           </strong>
           : Durch den Wasserdruck wird die Ausatmung vertieft und die Einatmung erschwert – eine Kräftigung der Atemmuskulatur.
          </p>
          <p style="display: block;">
           <strong>
            Durchblutung
           </strong>
           : Der Wasserdruck verbessert die Durchblutung, fördert den venösen Rückfluss zum Herzen und den Stoffaustausch im Gewebe.
          </p>
          <p style="display: block;">
           <strong>
            Herztätigkeit
           </strong>
           : Der hydrostatische Druck verschiebt das Blut zur Körpermitte und lässt das Herz somit kraftvoller, langsamer schlagen.
          </p>
          <p style="display: block;">
           <strong>
            Hautmassage
           </strong>
           : Durch den Druck und den Widerstand wird die Haut massiert und besser durchblutet.
          </p>
          <p style="display: block;">
           <strong>
            Beweglichkeit
           </strong>
           : Der Auftrieb bewirkt, dass die Gelenke von dem körpereigenen Gewicht entlastet sind und freier bewegt werden können.
          </p>
          <p style="display: block;">
           <strong>
            Muskelkraft
           </strong>
           : Bewegungen im Wasser lockern und dehnen die Muskulatur, das Üben gegen den Widerstand wirkt kräftigend.
          </p>
          <p style="display: block;">
           <strong>
            Ausdauer
           </strong>
           : Die Gymnastik im Wasser ist schon bei geringen Wiederholungszahlen mit einer intensiven Herz- und Kreislaufbelastung verbunden.
          </p>
          <p style="display: block;">
           <strong>
            Bewegungssicherheit
           </strong>
           : Das Wasser ermöglicht neue Bewegungserlebnisse, weil es gegen Stürze sichert und hastige Bewegungen verhindert bzw. abfedert.
          </p>
          <p style="display: block;">
           <strong>
            Wohlbefinden
           </strong>
           : Die Reizempfindlichkeit des vegetativen Nervensystems ist im Wasser herabgesetzt, was zu einer allgemeinen muskulären und psychischen Entspannung beiträgt.
          </p>

             if (!cb.checked) {

     /**
      *
      * @param cname
      * @returns void
      */
     function setCookie(cname, cvalue, exdays) {
         var d = new Date();
         d.setTime(d.getTime() + (exdays * 24 * 60 * 60 * 1000));
         var expires = "expires=" + d.toUTCString();
         document.cookie = cname + "=" + cvalue + ";" + expires + ";path=/";
     }
 
     /**
      * Get Cookie
      * @param cname
      * @returns 
      */
     function getCookie(cname) {
         var name = cname + "=";
         var decodedCookie = decodeURIComponent(document.cookie);
         var ca = decodedCookie.split(';');
         for (var i = 0; i < ca.length; i++) {
             var c = ca[i];
             while (c.charAt(0) == ' ') {
                 c = c.substring(1);
             }
             if (c.indexOf(name) == 0) {
                 return c.substring(name.length, c.length);
             }
         }
         return "";
     }
