        <h2 class="entry-title">
         WC Garnitur
        </h2>
        <div class="entry-content">
         <p>
          <a href="http://www.amazon.de/gp/search?ie=UTF8&amp;keywords=WC%20Garnitur&amp;tag=webung0b-21&amp;index=aps&amp;linkCode=ur2&amp;camp=1638&amp;creative=6742" target="_blank">
           <img alt="" class="alignleft size-full wp-image-74" height="203" src="/wp-content/uploads/guenstig-kaufen.jpg" title="guenstig-kaufen" width="174"/>
          </a>
          Eine WC Garnitur als Standgerät ist eine elegante und zugleich nützliche Sache.
          <br/>
          Nun gibt es so ein Set in verschiedenen Materialien und Ausführung.
          <br/>
          Von WC- Garnitur spricht man, wenn ein Bürstenhalter und Rollenhalter miteinander kombiniert sind.
          <br/>
          Es handelt sich hier um ein Standgerät, wo wie gesagt ein Rollenhalter so angebracht ist, dass Sie sitzend mühelos das Toilettenpapier erreichen.
          <br/>
          Vielfach gibt es auch Modelle, wo zusätzlich ein Reserverollenhalter mit integriert ist. Das hat den Vorteil, dass man die Papierrolle, sollte sie ausgehen, gleich vor Ort nachfüllen kann.
          <br/>
          Bei diesen WC-Garnituren soll der Bürstentopf mit WC-Bürste immer ein geschlossenes System sein. Das verhindert, dass die Feuchtigkeit die ja automatisch im Bürstentopf entsteht, sich nicht auf den Papier Reserverollenhalter überträgt.
          <br/>
          Der Vorteil dieser WC Garnituren ist es, dass keinerlei Wandbohrung notwendig ist.
         </p>

         <p>
          Das Hauptteil einer WC-Garnitur ist der Bürstenhalter mit WC-Bürste.
          <br/>
          Hier wiederum spielt die Auswahl des Materials eine entscheidende Rolle.
          <br/>
          Optisch ist es von Vorteil, wenn die WC Bürste im Bürstenhalter unter Verschluss ist.
          <br/>
          Die Materialien der Bürstenhalter können je nach Preis aus schönem Porzellan mit verchromten Bürstenstil sein oder in einem gesamt verchromten Behälter.
          <br/>
          optimal ist, wenn der Bürstenteil abgeschraubt werden kann und somit immer wieder austauschbar ist.
          <br/>
          Die Abnützung liegt ja in erster Linie an der WC Bürste selbst und daher sind Modelle wo man Ersatzbürsten nachkaufen kann vorteilhaft.
         </p>
         <p>
          Wie gesagt gibt es die Bürstenhalter mit WC-Bürste in verschiedenen Materialien.
          <br/>
          Bei einer WC Garnitur sollte man aber wirklich eher auf die Qualität und Ausführung schauen, als auf den Preis.
          <br/>
          Keramik oder Porzellan ist auf alle Fälle den Kunststoffhaltern vorzuziehen.
          <br/>
          Die Reinigung kann viel hygienischer durchgeführt werden und ein Porzellantopf bekommt auch keine Flecken, wenn man mit Desinfektionsmittel drüber geht.
          <br/>
          Bei den Bürstenköpfen gibt es auch Materialunterschiede und auch Formunterschiede. Das heißt es gibt auch einen Bürstenkopf der unter den Rand gelangt.
         </p>
