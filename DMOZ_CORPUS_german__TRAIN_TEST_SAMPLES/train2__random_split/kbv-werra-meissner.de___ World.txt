                   Sie haben eine landwirtschaftliche Berufsausbildung?
                   <br/>
                   <br/>
                   Sie helfen gern, wenn sich andere in einer Notlage befinden?
                   <br/>
                   <br/>
                   Sie sind neugierig, andere Betriebe und Betriebszweige kennenzulernen?
                   <br/>
                   <br/>
                   Sie sind in der Lage, diese eigenverantwortlich für die Zeit der Einsatzdauer zu führen?
                   <br/>
                   <br/>
                   BetriebshelferInnen springen ein, wenn Not in einem betrieb entsteht, zum Beispiel weil die Betriebsleitung krankheitsbedingt ausfällt und die Bewirtschaftung des Betriebes gefährdet ist.
                   <br/>
                   <br/>
                   Betriebshelfer übernehmen
                   <br/>
                   ... die Versorgung der Tiere
                   <br/>
                   ... die Futterbeschaffung
                   <br/>
                   ... Erntearbeiten
                   <br/>
                   ... Aussaat
                   <br/>
                   ... und alles weitere, was in diesem Moment wichtig ist.
                   <br/>
                   <br/>
                   Die Betriebs- und Haushaltshilfe Hessen GmbH bietet BetriebshelferInnen
                   <br/>
                   ... leistungsgerechte Bezahlung, auch für BerufsanfängerInnen
                   <br/>
                   ... geregelte Arbeitszeiten
                   <br/>
                   ... Möglichkeit der Teilzeitarbeit
                   <br/>
                   ... geregelten Urlaub
                   <br/>
                   ... einen sicheren Arbeitsplatz
                   <br/>
                   <br/>

              <h2 class="art-postheader">
               <span class="art-postheadericon">
                Düngebedarfsermittlung Herbst 2018 Hessen
               </span>
              </h2>
             </div>
             <div class="art-postcontent clearfix">
              <div class="art-article">
               Bei Cross Compliance Kontrollen muss die Düngebedarfsermittlung vom Herbst 2017 sowie vom
               <br/>
               Frühjahr 2018 vorliegen. Nun müssen diese Unterlagen zusätzlich um die Düngebedarfsermittlung Herbst 2018
               <br/>
               ergänzt werden.
               <br/>
               <br/>
               <strong>
                Vor der Ausbringung von organischen Dünger im Herbst muss die Bedarfsermittlung erfolgen!
               </strong>
               <br/>
               <strong>
                Grundsätzlich darf eine Düngung nur bei tatsächlichen Bedarf erfolgen.
                <br/>
                <br/>
               </strong>
               Dieser Bedarf muss nachgewiesen werden. Im Frühjahr galten die aktuellen N
               <sub>
                Min
               </sub>
               -Werte als Referenz.
               <br/>
               Im Herbst bestehen nun zwei Möglichkeiten:
               <br/>
               <br/>
               <ol>
                <li>
                 Es können eigene N Min -Proben (0-30) entnommen werden. Dieser Wert ist Grundlage für die Düngebedarfsermittlung Herbst 2018
                 <br/>
                 <br/>
                 <span style="text-decoration: underline;">
                  <strong>
                   oder
                  </strong>
                 </span>
                 <br/>
                 <br/>
                </li>
                <li>
                 Man verwendet die rechnerische Lösung:
                 <br/>
                 In der Düngebedarfsermittlung vom Frühjahr 2018 wurden erwartete Erträge angegeben. Wurde hier der prognostizierte Ertrag erreicht, besteht der volle Düngebedarf.
                 <br/>
                 Fiel der Ertrag geringer aus, muss die entsprechende Überhangmenge abgezogen werden. Der Bedarf der Folgefrucht reduziert sich um den Überhang.
                 <br/>
                 <br/>
                 Beispiel: Winterweizen, durchschnittlicher Ertrag: 80 dt, Stickstoffbedarfswert: 230 kg N/ha
                 <br/>
                 Tatsächlich geerntet: 60 dt
                 <br/>
                 Differenz: -20 dt
                 <br/>
                 Gemäß der Tabelle „DBE Ackerland“ (Tabelle 1 und Abbildung 1, siehe Anhang) ergibt sich folgende Berechnung:
                 <br/>
                 [(festgestellte Ertragsdifferenz 20 dt/ha) / (Ertragsdifferenz nach DüV 10 dt/ha)] x Mindestabschlag 15 kg
                 <br/>
                 <br/>
                 = 30kg N /ha Abschlag
                 <br/>
                 <br/>
                 Der Stickstoffbedarfswert verringert sich somit auf 200 kg N/ha, da noch 30 kg N vorhanden sind, die der Folgefrucht angerechnet werden müssen.
                </li>
               </ol>
               <br/>
               <strong>
                Organische Düngung im Herbst erfolgt nur nach Bedarf auf Zwischenfrüchte, Raps und Wintergerste (nur bei Stoppelvorfrucht).
               </strong>
               <br/>
               <strong>
                Die maximale Ausbringmenge beträgt 60 kg Gesamt N oder 30 kg Ammonium N.
               </strong>
               <br/>
               <strong>
                Mineralischer Dünger darf nicht gestreut werden.
                <br/>
                <br/>
               </strong>
               Bei Fragen stehen wir gern unter 05651-745660 zur Verfügung.
               <strong>

             <div class="art-postmetadataheader">
              <h2 class="art-postheader">
               <span class="art-postheadericon">
                Mautpflicht für landwirtschaftliche Fahrzeuge?
               </span>
              </h2>
             </div>
             <div class="art-postcontent clearfix">
              <div class="art-article">
               <br/>
               Ab 1. Juli werden alle Bundesstraßen mautpflichtig.
               <br/>
               <br/>
               Landwirtschaftliche Fahrzeuge mit einer bauartbedingten Höchstgeschwindigkeit von maximal 40
               <br/>
               km/h sind grundsätzlich von der Maut befreit, unabhängig davon, ob sie für den eigenen oder einen
               <br/>
               fremden Betrieb unterwegs sind.
               <br/>
               Land- und forstwirtschaftliche Fahrzeuge, die landwirtschaftliche Bedarfsgüter transportieren, sind
               <br/>
               auch dann von der Maut befreit, wenn sie schneller als 40 km/h fahren – sofern sie für eigene
               <br/>
               Zwecke oder im Rahmen eines Maschinenring tätig sind.
               <br/>
               Damit entspricht das Bundesverkehrsministerium den Forderungen des Bauernverbandes.
