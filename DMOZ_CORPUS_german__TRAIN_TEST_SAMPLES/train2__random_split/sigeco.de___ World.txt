      <h1>
       Willkommen
      </h1>
      ...beim Sachverständigenbüro Schmid
      <span class="rightcontentimage">
       <img alt='Bild "Schmid-2.jpg"' class="rightcontentimage" src="kategorien/00_Willkommen/dateien/Schmid-2.jpg"/>
      </span>
      <br/>
      <br/>
      <h2>
       Architekt Dipl.-Ing. (FH) Horst Schmid
      </h2>
      <b>
       <span class="leftcontentimage">
        <img alt='Bild "Brief-CMYK107.jpg"' class="leftcontentimage" src="kategorien/00_Willkommen/dateien/Brief-CMYK107.jpg"/>
       </span>
       <br/>
       Von der Architektenkammer Rheinland-Pfalz öffentlich bestellter und vereidigter Sachverständiger für Bewertung von bebauten und unbebauten Grundstücken; Ermittlung von Mieten und Pachten.
       <br/>
       Von der Industrie und Handelskammer für die Pfalz öffentlich bestellter und vereidigter Sachverständiger für Schäden an Gebäuden.
       <br/>
       REV - recognised european valuer (REV-D/BVS/2019/5)
       <br/>
       Von der Architektenkammer Nordrhein-Westfalen staatlich anerkannter Sachverständiger für Schall- und Wärmeschutz.
      </b>
      <br/>
      <br/>
      <br/>
      Auf meiner Seite biete ich Ihnen Informationen über meine Tätigkeiten:
      <br/>
      <br/>
      <br/>
      <a class="category" href="index.php?cat=20_Bausch-auml~den" title='Zur Kategorie "Bauschäden" wechseln'>
       Bauschäden
      </a>
      :
      <br/>
      Gutachten und Beratung zu Bauschäden, z.B. Abdichtungen, Risse, Schimmelpilz, Wertminderungen infolge Bauschäden, u.v.m. als Entscheidungshilfe für weitere Maßnahmen, Auseinandersetzung der am Bau Beteiligten, als Sanierungsgutachten, zur Beweissicherung, etc.
      <br/>
      <br/>
      Baubegleitende Qualitätsüberprüfung als Präventivmaßnahme zur Vermeidung von Bauschäden.
      <br/>
      <br/>
      <br/>
      <a class="category" href="index.php?cat=10_Immobilienbewertung" title='Zur Kategorie "Immobilienbewertung" wechseln'>
       Immobilienbewertung
      </a>
      :
      <br/>
      Gutachten über den Verkehrswert von Immobilien, Grundstücken, Gebäuden als Entscheidungshilfe bei Kauf oder Verkauf, Ehescheidung, Erbauseinandersetzung, Zwangsversteigerung
      <br/>
      Besuchen Sie hierzu die Seite
      <a class="category" href="index.php?cat=10_Immobilienbewertung" title='Zur Kategorie "Immobilienbewertung" wechseln'>
       Immobilienbewertung
      </a>
      .
      <br/>
      <br/>
      Mieten und Pachten:
      <br/>
      Gutachten über die ortsübliche Vergleichsmiete, z.B. bei Mieterhöhungen
      <br/>
      Besuchen Sie hierzu die Seite
      <a class="category" href="index.php?cat=10_Immobilienbewertung" title='Zur Kategorie "Immobilienbewertung" wechseln'>
       Immobilienbewertung
      </a>
      .
      <br/>
      <br/>
      <a class="category" href="index.php?cat=25_Konfliktl-ouml~sung" title='Zur Kategorie "Konfliktlösung" wechseln'>
       Konfliktlösung
      </a>
      :
      <br/>
      Als meine vornehmste Leistung betrachte ich die Streitbeilegung. Besuchen Sie hierzu die Seite
      <a class="category" href="index.php?cat=25_Konfliktl-ouml~sung" title='Zur Kategorie "Konfliktlösung" wechseln'>
       Konfliktlösung
      </a>
      .
      <br/>
      <br/>
      <br/>
      Ich bin im gesamten Bundesgebiet für Sie tätig, insbesondere in Rheinland-Pfalz (z.B. Kaiserslautern, Landau, Neustadt, Germersheim, Speyer, Wörth, Pirmasens, Zweibrücken, Mainz, Ludwigshafen, Trier, Koblenz, Waldmohr, Kusel), Hessen (z.B. Frankfurt, Darmstadt, Offenbach, Groß-Gerau, Bad Hersfeld, Kassel, Fulda), Baden-Württemberg (z.B. Mannheim, Karlsruhe, Pforzheim, Baden-Baden, Stuttgart, Ulm, Göppingen), Thüringen (z.B. Erfurt, Jena, Gotha, Eisenach), Bayern (z.B. Augsburg, München, Rottach-Egern, Tegernsee, Miesbach, Nürnberg, Würzburg, Bayreuth, Amberg), Saarland (z.B. Homburg, Saarbrücken, Neunkirchen, Merzig, Saarlouis)
