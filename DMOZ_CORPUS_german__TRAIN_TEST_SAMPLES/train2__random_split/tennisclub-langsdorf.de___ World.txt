      <span class="textHeadline">
       Herzlich willkommen beim Tennis Club 1992 Langsdorf e.V.
      </span>
      <br/>
      <br/>
      Seit 1992 gibt es unseren Tennis Club in Lich-Langsdorf.
      <br/>
      Unsere vier Freiluftplätze liegen im herrlichen Grün,
      <br/>
      in unmittelbarer Nähe zum Sportplatz gelegen.
      <br/>
      <br/>
      In unserem schönen Tennissheim „Return" kann man sich im
      <br/>
      Anschluss an das Tennisspiel in gemütlicher Runde erholen.
      <br/>
      <br/>
      Ob zum Tennisspielen oder zum Plaudern, bei einem kühlen Getränk
      <br/>
      und leckeren Gaumenfreuden
        trifft man im Tennisheim immer nette
      <br/>
      Leute.
      Hier werden Kontakte  geknüpft und  Freundschaften geschlossen.
      <br/>
      <br/>
      <strong>
       Sie finden unsere Tennisanlage in:
      </strong>
      <br/>
      <img height="5" src="../abbildungen/design/filler.gif" width="1"/>
      <br/>
      35423 Lich-Langsdorf, Gansahrweg 7, Tel.: 0 64 04 / 66 15 53
      <br/>
      <br/>
      <span class="textHinweis">
       Unser Tennisheim ist geöffnet:
      </span>
      <br/>
      <img height="5" src="../abbildungen/design/filler.gif" width="1"/>
      <br/>
      Mo. bis Do. und Sa. / So. nach Bedarf
      <br/>
      Fr. ab 19:30 Uhr
