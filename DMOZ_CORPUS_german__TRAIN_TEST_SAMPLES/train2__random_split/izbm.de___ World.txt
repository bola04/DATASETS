          <h2 class="">
           Gründen in Berlin: „Rent &amp; Start-up“-Programm zu Top-Konditionen
          </h2>
         </div>
         <h3>
          Die WISTA-Gründerzentren und Innovationsprojekte in Berlin...
         </h3>
         <p>
          ...bieten Start-ups Services und Flächen in einem auf Wirtschaftsförderung ausgerichteten Netzwerk. Ergänzt werden die Leistungen durch spezielle Angebote:
          <a href="https://www.wista.de/projekte/projekte-adlershof/a2-accelerator-berlin/" target="_self">
           Accelerator-Programme
          </a>
          , die
          <a href="https://www.wista.de/gruenderwerkstatt/" target="_self">
           Gründerwerkstatt
          </a>
          , den
          <a href="https://www.adlershof.de/coworking/">
           Coworking Space IM.PULS
          </a>
          und besonderen
          <a href="https://www.wista.de/services/business-support/" target="_self">
           Business Support
          </a>
          für Gründer.
         </p>
         <p>
          Als Berliner Landesgesellschaft ist die WISTA-MANAGEMENT GMBH der Wirtschaftsförderung verpflichtet. Daher können unsere Gründerzentren ein ausgezeichnetes Preis-Leistungsverhältnis bieten. Dabei haben Sie
          <strong>
           volle Flexibilität
          </strong>
          : Wenn Ihr Unternehmen wächst, mieten Sie Räume hinzu und in schwierigen Zeiten geben Sie etwas Fläche ab.
         </p>
        </div>

             Adlershofer Gründerzentren
            </a>

            <a href="https://adlershof.wista.de/gruenderzentrum/projekte-charlottenburg/charlottenburger-innovations-centrum/ueber-uns/">

            </a>

           </p>

        <div class="csc-default" id="c7362">
         <div class="label">
          <h2 class="">
           IM.PULS - der Coworking Space in Berlin
          </h2>
         </div>
         <p>
          Im Souterrain des WISTA-Gebäudes in der Rudower Chaussee 17 stehen etwa 35 Arbeitsplätze auf 550 Quadratmeter Fläche für Gründerwillige, Studierende und Projektgruppen aus Forschungsinstituten in einem offenen, hellen Großraum zur Verfügung. Die Arbeitsplätze werden mindestens für einen Monat vergeben,  neben Schreibtischen, WLAN und Druckern gibt es gemütliche Couchecken, abgeschlossene „Think Tanks“ fürs Telefonieren, einen Konferenzraum und eine voll ausgestattete Küchenzeile.
         </p>
         <p>
          <a href="https://www.adlershof.de/coworking/" target="_self">
           Informieren Sie sich hier über unsere Konditionen...
          </a>
         </p>

         <div class="label">
          <h2 class="">
           Das A² Accelerator-Programm Berlin
          </h2>
         </div>
         <p>
          Das A² Accelerator-Programm Berlin bietet jährlich fünf bis acht Start-ups eine Chance, gemeinsam mit erfahrenen Unternehmern ihre Geschäftsideen umzusetzen und Wachstum zu generieren.
         </p>
         <p>
          Gründer mit hohem Marktpotenzial erhalten in dem fünfmonatigen Accelerator Programm eine maßgeschneiderte Förderung mit professionellem Coaching und kostenlosem Raumangebot, sowie Zugang zu Netzwerken und Kontakten, da ein intensiver Austausch mit Industriepartnern stattfinden wird.
         </p>
         <p>
          Unabdingbar dafür sind ein großes Netzwerk und langjährige Erfahrungen im Umgang mit der Industrie und Jungunternehmern.
         </p>

    window.onload = function() {                class Cookie {
                   /**
                    * Create cookie
                    *
                    * @param name
                    * @param value
                    * @param days - number of days from today until cookie expires
                    */
                   static Create(name, value, days) {
                     let expires = "";
                     if (days) {
                       const date = new Date();
                       date.setTime(date.getTime() + days * 24 * 60 * 60 * 1000);
                       expires = `; expires=${date.toGMTString()}`;
                     }
                     document.cookie = `${name}=${value}${expires}; path=/`;
                   }
 
                   /**
                    * Read cookie
                    *
                    * @param name
                    * @returns {*}
                    */
                   static Read(name) {
                     const nameEQ = `${name}=`;
                     const ca = document.cookie.split(";");
                     for (let i = 0; i < ca.length; i += 1) {
                       let c = ca[i];
                       while (c.charAt(0) === " ") c = c.substring(1, c.length);
                       if (c.indexOf(nameEQ) === 0) return c.substring(nameEQ.length, c.length);
                     }
                     return null;
                   }
 
                   /**
                    * Erase cookie
                    *
                    * @param name
                    */
                   static Erase(name) {
                     this.create(name, "", -1);
                   }
                 }
