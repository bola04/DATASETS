         <h3 class="welcome">
          Herzlich Willkommen
         </h3>
         <p style="text-align: justify;">
          …im modernen und dienstleistungsorientierten
          <strong>
           Notariat Dr. Alexander Grein
          </strong>
          in der Bonner Innenstadt. Das Amt wurde im Jahr 2010 von mir als Amtsnachfolger der Notare Dr. Peter Kemp und Harro Harling übernommen. Auf dieser Seite erwarten Sie interessante Informationen über die Tätigkeitsfelder und Leistungen, die mein Notariat erbringt. Sie haben die Möglichkeit nützliche Formulare und Checklisten herunter zu laden, für die Übermittlung von Kaufvertragsangaben ein von mir erstelltes Webformular zu nutzen, mir einen Terminwunsch oder eine Archivanfrage bzgl. von mir verwahrter Urkunden zuzusenden oder vorab zu erfahren, welche Unterlagen Sie sinnvollerweise zu Ihrem Besprechungstermin mitbringen.
         </p>
         <p style="text-align: justify;">
         </p>
         <p style="text-align: justify;">
          Der Notar ist Träger eines öffentlichen Amtes. Er ist unparteiischer und unabhängiger Betreuer sämtlicher Beteiligter und nicht - wie ein Rechtsanwalt - einseitiger Interessensvertreter. Mit den Beteiligten klärt er zunächst Sachverhalt und Ziele. Sodann zeigt er diejenigen Regelungsinstrumente auf, die der Umsetzung der rechtlichen Interessen der Beteiligten dienen. Soweit erforderlich oder sinnvoll, rät er zu einer Beurkundung und nimmt diese mit den Beteiligten vor. Falls nach der Beurkundung weitere Schritte erforderlich sind (z.B. Eintragungen ins Grundbuch oder ins Handelsregister), stellt er den Vollzug der Urkunde sicher.
         </p>
         <p style="text-align: justify;">
         </p>
         <p style="text-align: justify;">
          Aber nicht jede Tätigkeit des Notars mündet in eine Beurkundung. Der Notar kann auch außerhalb von Beurkundungsvorgängen Beratungsleistungen erbringen, juristische Gutachten erstellen, nicht von ihm erstellte Vertragsentwürfe prüfen oder auf Antrag eigene Wahrnehmungen bescheinigen, z.B. Unterschrifts- und Abschriftsbeglaubigungen, Lebensbescheinigungen, Protokollierung von Verlosungen oder Prioritätsverhandlungen.
         </p>
         <p style="text-align: justify;">
         </p>
         <p style="text-align: justify;">
          Ich nehme mir Zeit für Ihre Anliegen. Sie können von mir und meinem Büro eine zügige Terminvergabe, kompetente und gründliche Beratung, eine schnelle Erledigung anstehender Vollzugsschritte, hohe Einsatzbereitschaft und Flexibilität sowie einen stets freundlichen und menschlichen Umgang erwarten.
         </p>
         <p style="text-align: justify;">
          Dies gilt für das gesamte Spektrum notarieller Tätigkeiten, wie beispielsweise Immobilienkaufverträge, Übertragungsverträge, Testamente, Erbverträge, Vorsorgevollmachten, Patientenverfügungen, Ehe- und Partnerschaftsverträge, Adoptionen, Scheidungsvereinbarungen, Anmeldungen zum Handelsregister oder Vereinsregister, Gesellschaftsgründungen, Gesellschafterversammlungen, Umwandlung von Unternehmen, Unternehmenskauf oder Beglaubigungen.
         </p>
         <p>

    jQuery(document).ready(function(){ 
  if (cookieExists()) { 
 document.getElementById('cookie-alert').style.display = 'none';
  } else {
  fadeIn(document.getElementById('cookie-alert')); } 
 });
 
 function addCookie() {
 	var curDate = new Date();
 	curDate.setTime(curDate.getTime() + (86400000 * 365));
 	document.cookie = 'cookieTermsAccepted=1; expires=' + curDate.toUTCString();
 	fadeOut(document.getElementById('cookie-alert'));
 }
 
 function cookieExists() {
 	var cookies = document.cookie.split(';');
 	for(var i=0; i < cookies.length; i++) {
 		if (cookies[i].trim() == 'cookieTermsAccepted=1') {
 			return true;
 		}
 	}
 	return false;
 }
 
 function fadeOut(element) {
     var op = 1;
     var timer = setInterval(function () {
         if (op <= 0.15){
             clearInterval(timer);
             element.style.display = 'none';
         }
         element.style.opacity = op;
         element.style.filter = 'alpha(opacity=' + op * 100 + ')';
         op -= op * 0.15;
     }, 50);
 }
 
 function fadeIn(element) {
     var op = 0.09;
     element.style.display = 'block';
     var timer = setInterval(function () {
         if (op >= 1){
             clearInterval(timer);
         }
     }, 25);
 }
