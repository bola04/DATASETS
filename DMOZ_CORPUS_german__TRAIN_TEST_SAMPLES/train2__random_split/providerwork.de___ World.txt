     <h1>
      Providerwork
     </h1>
     ProviderWork bildet die zentrale Softwareplattform die Sie bei
     <br/>
     sämtlichen abrechnungsrelevanten Prozesse des Providings unterstützt.
     <br/>
     <br/>
     Die Software is seit 2000 im Einsatz, wurde ständig erweitert und angepasst.
     <br/>
     Es ist
     <b>
      die
     </b>
     Lösung für sämtliche verbrauchs- und gesprächsorientierten automatischen Abrechnungen.
     <br/>
     <br/>
     Das Programm zeichnet sich durch eine leichte Bedienbarkeit und Praxisnähe aus.
     <br/>
     Trotz der umfangreichen Fuktionalität können Sie mit dieser Software auch
     <br/>
     eigene Anpassungen vornehmen, ja sogar Ihre eigene Individuallösung hierauf aufbauen.
     <br/>
     <br/>
     Die wichtigste Funktion des Programmes - die automatische Abrechnung - ist reversibel und fehlertolerant.
     <br/>
     Das erlaubt Ihnen sehr umfangreiche und komplexe Tarifmodele abzubilden.
     <br/>
     Daneben ist hierdurch eine tagesgenaue Zwischenabrechnung (Hotbilling) möglich.
     <br/>
     <br/>
     Auf den folgenden Seiten wird versucht Ihnen einen Eindruck von der Leistungsfähigkeit
     <br/>
     der Software zu vermitteln.
     <br/>
     <br/>
     Sehr gerne erstellen wir Ihnen ein atraktives Angebot,
     <br/>
     das die Einzigartigkeit Ihres Unternehmens wiederspiegelt.
     <br/>
     <br/>
     Denn wir wollen Sie bei Ihrer Arbeit untersützen.
     <br/>
     <br/>
     Wir freuen uns über Ihre
     <a href="mailto:providerwork4@progm.de">
      Email.
