          Cloud Beratung
         </h2>
         <p>
          Seit über 20 Jahren ist das Unternehmen herstellerunabhängiger Berater für individuelle Lösungen in den unterschiedlichsten IT – Umgebungen. Seit 2 Jahren hat sich die ASSISTRA als Cloud Lösungsanbieter auf die nächste IT Revolution konzentriert. Wir, die Spezialisten der ASSISTRA Cloud Services GmbH designen und implementieren individuelle IT Infrastruktur Lösungen. Wir identifizieren die richtigen Lösungen für Ihre Private, Hybrid oder Public Cloud Anforderungen und setzen diese gemeinsam mit Ihnen um.
         </p>

         </p>

         <p>
          Auf dem Weg zur passenden Cloud Strategie stellen sich den Kunden zahlreiche Fragen. Durch individuelle Beratung entwickeln wir gemeinsam mit unseren mittelständischen Kunden die richtige branchenspezifische Cloud Strategie. Nach der Analyse unterstützen wir unsere Kunden bei der Implementierung einer Private, Hybrid oder Public Cloud. Eine effiziente und flexible Nutzung der Cloud sowie die sichere Haltung der Daten bei höchster Verfügbarkeit ist das Ziel der ASSISTRA Cloud Spezialisten. Ein Ansprechpartner für den Zeitraum vor und nach der Implementation macht die Stärke der ASSISTRA Cloud Services GmbH aus.

     <section class="section-lines margin-bottom-25" id="section2">
      <div class="container">
       <div class="row">
        <div class="col-md-10 col-md-offset-1 col-xs-12 col-xs-offset-0">
         <h2 class="content-box-title">
          ASSISTRA Cloud Lösung
         </h2>
         <h4>
          ASSISTRA Cloud Speicher und Backup Lösung
         </h4>
         <p>
          Der ASSISTRA Cloud Speicher ist die ideale Lösung für mittelständische Unternehmen, der eine sichere und flexible Dateiablage realisiert. Besonders bei großen Datenmengen ist die klassische Kommunikation mittels Mail zu unflexibel und langsam. Die ASSISTRA Cloud Lösung stellt eine sichere Methode zur Verfügung, um Daten mit externeren und internen Nutzern zu teilen. Darüber hinaus kann auf diese Daten mit den verschiedenen Endgeräten (Smartphone, Tablet, Laptop) zugegriffen werden.
         </p>
         <p>
          Die Lösung der ASSISTRA bietet Ihnen zahlreiche Vorteile
         </p>
        </div>
        <div class="col-md-5 col-md-offset-1 col-xs-12 col-xs-offset-0 margin-bottom-25">
         <ul>
          <li>
           <h6>
            Sicherer Dateiaustausch mit externen Nutzern
           </h6>
           <ul>
            <li>
             Zugriffsdauer und Zugriffsrechte können verwaltet werden
            </li>
            <li>
             Integrierter Backup
            </li>
           </ul>
          </li>
          <li>
           <h6>
            Sicherer Zugriff von mobilen internen Nutzern auf Daten
           </h6>
           <ul>
            <li>
             Unterstützung aller gängigen mobilen Endgeräte (Smartphone, Tablet, Laptop)
            </li>
            <li>
             Integration Office 365
            </li>
           </ul>
          </li>
         </ul>
        </div>
        <div class="col-md-5 col-md-offset-0 col-xs-12 col-xs-offset-0 margin-bottom-25">
         <ul>
          <li>
           <h6>
            Einfache Integration in das Unternehmensnetzwerk
           </h6>
          </li>
          <ul>
           <li>
            Authentifizieren über Active Directory
           </li>
           <li>
            Lokale Bereitstellung über ein Fileserver Gateway
           </li>
           <li>
            Modernisierung der Speicherarchitektur in Außenstellen
           </li>
          </ul>
          <li>
           <h6>
            Umfangreiche Reporting Möglichkeiten
           </h6>
          </li>
          <li>
           <h6>
            Personalisierung der Web Oberfläche
           </h6>
          </li>
         </ul>
        </div>
        <div class="col-md-10 col-md-offset-1 col-xs-12 col-xs-offset-0 margin-bottom-25">
         <p>
          Der ASSISTRA Cloud Backup ist eine leistungsfähige Lösung für die Sicherung von mobilen Geräten und Cloud Servern. Die Gefährdung durch Ransomware Attacken hat die Notwendigkeit zur Sicherung mobiler Devices erhöht. Die oftmals fehlende Übertragungsgeschwindigkeit verhindert diese Sicherung oft. Bei den Cloud Servern birgt die einfache und oft unzureichende Sicherungsmöglichkeit der Cloud Anbieter ein Gefahrenpotential für die Cloud Infrastruktur.
         </p>
         <h6>
          Die Lösung der ASSISTRA für mobile Endgeräte bietet Ihnen zahlreiche Vorteile
         </h6>
         <ul>
          <li>
           Schnelle Online Bereitstellung der eigenen Backup Plattform in der Cloud
          </li>
          <li>
           Einfache und schnelle Installation der Agenten
           <br/>
           (Mit und ohne den Support unserer Spezialisten)
          </li>
          <li>
           WAN Optimierung durch DeDuplizierung am Endgerät
          </li>
          <li>
           Lokale Sicherung auf Cloud Storage Gateways
          </li>
          <li>
           Intuitives Backup und Restore Interface
          </li>
         </ul>
        </div>
        <div class="col-md-10 col-md-offset-1 col-xs-12 col-xs-offset-0 margin-bottom-25">
         <p>
          Die Lösung für Cloud Server unterstützt die Microsoft Plattform mit den gängigsten Microsoft Applikationen und Linux. Die Sicherung der Server ermöglicht einen granularen Zugriff auf die Daten und erweitert die Funktionalitäten der gängigen Cloud Lösungen, die nur einen „Full Restore“ erlauben. Die Ablage der Sicherungsdaten kann innerhalb der Cloud in z.B. einer anderen Sicherheitszone oder bei einem anderen Cloud Anbieter erfolgen.
         </p>
         <h6>
          Die Vorteile der ASSISTRA Cloud sind:
         </h6>
         <ul>
          <li>
           Verschlüsselte Ablage der Daten in der Cloud
          </li>
          <li>
           Eigene Verwaltung der Verschlüsselungs-Keys
          </li>
          <li>
           Integration mit Antivirus Lösung optional möglich
          </li>
          <li>
           Deutschsprachiger Support
          </li>
          <li>
           Transparente und einfache Abrechnung
          </li>
         </ul>
        </div>
       </div>
      </div>
     </section>
