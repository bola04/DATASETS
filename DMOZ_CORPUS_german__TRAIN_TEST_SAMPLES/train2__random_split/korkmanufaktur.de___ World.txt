        <h2 class="csc-firstHeader">
         Wir lieben Kork!
        </h2>
       </div>
       <p class="bodytext">
        Alles aus
        <b>
         KORK
        </b>
        : Korklaminat, Korkparkett, Klebekork, Korkboden, Wandkork, Korkschrot, Korkplatten, Korkmatten, Rollenkork, Korkschrot, Kopschina-Kork, Korkgranulat und andere Korkprodukte!
        <br/>
        <br/>
        Deshalb informieren wir Sie auch in den Rubriken
       </p>
       <ul>
        <li>
         <a class="internal-link" href="die-natur.html" title="Öffnet internen Link im aktuellen Fenster">
          Die Natur
         </a>
        </li>
        <li>
         <a class="internal-link" href="der-mensch.html" title="Öffnet internen Link im aktuellen Fenster">
          Der Mensch
         </a>
        </li>
        <li>
         <a class="internal-link" href="das-produkt.html" title="Öffnet internen Link im aktuellen Fenster">
          Das Produkt
         </a>
        </li>
        <li>
         <a class="internal-link" href="die-verarbeitung.html" title="Öffnet internen Link im aktuellen Fenster">
          Die Verarbeitung
         </a>
        </li>
        <li>
         <a class="internal-link" href="die-fragen.html" title="Öffnet internen Link im aktuellen Fenster">
          Die häufigsten Fragen
         </a>
         und
        </li>
        <li>
         <a class="internal-link" href="die-farbe.html" title="Öffnet internen Link im aktuellen Fenster">
          Die Farben
         </a>
        </li>
       </ul>
       <p class="bodytext">
        speziell rund um das Naturprodukt
        <b>
         KORK
        </b>
        . Weitere ökologische Bodenbeläge aus Leder oder Linoleum stehen zur Auswahl.
        <br/>
        <br/>
        <br/>
        Hier erhalten Sie alle Informationen über eine fast unendliche Produktvielfalt an Korkparkett, auch als Klebekork bezeichnet, und Korkfertigparkett, welches auch als Korklaminat bekannt ist und mittels UNICLIC-System verbunden wird. Als zusätzliche Trittschalldämmung werden hierfür Rollenkork oder Korkplatten verwendet.
        <br/>
        <br/>
        Der Korkboden kann als Klebekork und als Korkaminat mit
        <a class="internal-link" href="kork-boeden/kork-boeden-massiv.html" title="Öffnet internen Link im aktuellen Fenster">
         massiver
        </a>
        oder
        <a class="internal-link" href="kork-boeden/kork-boeden-furniert.html" title="Öffnet internen Link im aktuellen Fenster">
         furnierter
        </a>
        Korkauflage produziert werden.
        <br/>
        Durch die Verwendung von Fase und Korkfarbe stehen weitere Gestaltungsmöglichkeiten zur Verfügung - ganz individuell und natürlich einzigartig.
        <br/>
        <br/>
        Inzwischen hat sich die Herstellung eines
        <span style="font-size: 12.222222328186px; ">
         Digitaldruck-Korkboden als
        </span>
        <span style="font-size: 12px; ">
        </span>
        <span style="font-size: 12px; ">
         <a class="internal-link" href="print-kork.html" title="Öffnet internen Link im aktuellen Fenster">
          Printkork
         </a>
        </span>
        <span style="font-size: 12px; ">
         etabliert. Aber auch der
        </span>
        <span style="font-size: 12px; ">
         <a class="internal-link" href="print-kork.html" title="Öffnet internen Link im aktuellen Fenster">
          Vinylboden
         </a>
        </span>
        <span style="font-size: 12px; ">
         kann die natürlichen Eigenschaften von Kork sehr gut mit den optischen Eigenschaften von Holzparkett oder auch
        </span>
        <span style="font-size: 12px; ">
         Schiefer
        </span>
        <span style="font-size: 12px; ">
         verbinden.
        </span>
       </p>
       <p class="bodytext">
        <br/>
        Kork gestaltet nicht nur den Boden!
        <a class="internal-link" href="wandkork.html" title="Öffnet internen Link im aktuellen Fenster">
         Wandkork
        </a>
        gibt es in vielen interessanten Varianten; einige haben wir abgebildet. Prinzipiell kann jedoch mit jedem Klebekork eine Korkwand erstellt werden.
        <br/>
        <br/>
        Natürlich können Korkmatten aus
        <a class="internal-link" href="kopschina-kork.html" title="Öffnet internen Link im aktuellen Fenster">
         Kopschina-Kork
        </a>
        Ihr Wohlbefinden verbessern. Für die ökologischen Bauherren gibt es
        <a class="internal-link" href="daemm-kork.html" title="Öffnet internen Link im aktuellen Fenster">
         Korkdämmplatten
        </a>
        in unterschiedlichen Stärken oder auch als expandiertes
        <a class="internal-link" href="kork-granulat.html" title="Öffnet internen Link im aktuellen Fenster">
         Korkgranulat
        </a>
        .
        <br/>
        <br/>
        Aber auch
        <a class="internal-link" href="kork-granulat.html" title="Öffnet internen Link im aktuellen Fenster">
         Korkschrot
        </a>
        wird gern als Schüttung zur Dämmung von Zwischenräumen, zur Herstellung von Dämm-Estrichen oder Leichtlehm-Putzen am Bau verwendet.
        <br/>
        <br/>
        Theater nutzen oft
        <a class="internal-link" href="kork-granulat.html" title="Öffnet internen Link im aktuellen Fenster">
         Kork
        </a>
        zur Gestaltung von Bühnenbildern und die Botanischen Gärten zur Auflockerung des Erdreiches der Pflanzen. Die Korkmatte oder der Korkblock findet für Entspannungsübungen, z.B. Yogakork, Anwendung.
        <br/>
        <br/>
        Hat das Naturprodukt KORK Ihr Interesse geweckt? Dann informieren Sie sich in Ruhe auf unseren Seiten! Wir sind stets bemüht unsere Informationen ständig zu aktualisieren.
        <br/>
        <br/>
        Sie werden künftig die ökologischen Vorteile der Naturprodukte wie
        <a class="internal-link" href="" title="Öffnet internen Link im aktuellen Fenster">
         KORK
        </a>
        , Linoleum, Leder oder Schiefer einem Teppich oder PVC-Bodenbelag vorziehen. Besonders in den Räumen, in denen man sich täglich lange Zeit aufhält.
       </p>
       <p class="bodytext">
        Bei uns können Sie nicht nur Korkboden kaufen, sondern alle Korkprodukte Preis günstig erwerben. Nutzen Sie unsere Kalkulationshilfen und profitieren Sie von individuellen Rabatten!
       </p>

       </p>
