      <h1 align="right">
       PFERDE-ERZIEHUNG
      </h1>
      <p align="right">
       <br/>
       Seit vielen Jahren lebe und arbeite ich
       <br/>
       mit Pferden in Bölkum. Das Verständnis
       <br/>
       für die Zusammenarbeit mit ihnen wurde
       <br/>
       hauptsächlich von amerikanischen
       <br/>
       Pferdemenschen geprägt, lange bevor
       <br/>
       „Pferdeflüsterer“ in Mode kamen.
       <br/>
       <br/>
       <br/>
       <br/>
       <br/>
       <br/>
       <br/>
       <br/>
       <br/>
       <img alt="" border="0" height="47" src="images/pferd_sw.gif" width="50"/>
      </p>
     </td>
     <td align="center" valign="top">
      <img alt="" border="0" height="350" src="images/foto_stuhlmann.jpg" width="255"/>
     </td>
     <td valign="top" width="365">
      <p>
       Gerade die schwierigen Pferde haben mir geholfen,
       <br/>
       einen Weg zu finden, der zuverlässig bei allen Pferden
       <br/>
       (und sogar bei Zebras) zum Erfolg führt - ihr Alter, ihre
       <br/>
       Rasse und Vorgeschichte spielen dabei keine Rolle.
      </p>
      <p>
       Pferde können nur lernen wie Pferde.
       <br/>
       Wir können ihnen nur etwas auf eine Weise beibringen,
       <br/>
       die sie auch verstehen können.
      </p>
