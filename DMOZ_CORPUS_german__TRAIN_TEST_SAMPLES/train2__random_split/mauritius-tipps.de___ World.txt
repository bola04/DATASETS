              <br/>

              <br/>
              wo man im Grunde abgeschottet von Mauritius wohnt.
              <br/>
              Mauritius wirkt auf der einen Seite sehr europäisch, sicher durch die Sprachen,
              <br/>
              die dort gesprochen werden,es ist aber eine andere Welt,in die man außerhalb eines Hotels  etwas eintauchen kann. Dies möchte ich Urlaubern ermöglichen:
              <br/>
              <br/>
              <strong>
               Mauritius abseits eines Hotels kennenzulernen.
              </strong>
              <br/>
              Dabei kann man auch feststellen, das Mauritius auch ein Reiseziel für Gäste sein kann, die nicht zu viel Geld für Ihren Urlaub ausgeben möchten.
              <br/>
              Es klappt vielleicht nicht alles so perfekt, wie in einem teuren Hotel,
              <br/>
              aber man bekommt interessante Einblicke in die Insel Mauritius
              <br/>
              Mauritius bietet eben für jeden das passende, teuren Luxusurlaub,
              <br/>
              preisgünstige Ferien in Mittelklasse Hotels als Pauschalreise und
              <br/>
              private Unterkünfte für den Individual - Reisenden
              <br/>
              von einfach bis zu Luxusunterkünften
              <br/>
              <br/>
              <strong>
               Die Touren über die Insel
              </strong>
              <br/>
              werden alle mit registrierten Taxen und Autos durchgeführt
              <br/>
              mit englisch und französisch sprechenden Taxifahrern oder mit einem
              <br/>
              deutschsprechenden Reiseleiter.
              <br/>
              <br/>
