         <p>
          <big>
           <strong>
            Herzlich Willkommen auf Numispedia
           </strong>
          </big>
          <br/>
          <strong>
           der Numismatischen Online-Enzyklopädie
          </strong>
          <br/>
         </p>
         <p>
          Numispedia ist ein offenes numismatisches Online-Lexikon von Sammler für Sammler, das keinerlei kommerzielle Ziele verfolgt. Die Weiterentwicklung und Verbesserung von Numispedia wird von allen Nutzern auf freiwilliger Basis betrieben. 
 Das Ziel von Numispedia ist eine umfangreiche und sachlich orientierte Sammlung numismatischer und historischer Informationen bereitzustellen, die von seinen Nutzern ständig erweitert und aktualisiert wird.
         </p>

          <p>
           <strong>
            Kurzbeschreibung der Rubriken
           </strong>
           <br/>
          </p>
          <ul>
           <li>
            <strong>
             Lexikon
            </strong>
            : Lexikon zu numismatischen Begriffen
           </li>
           <li>
            <strong>
             Themen
            </strong>
            : größere Beiträge zu einem Thema
            <a class="external" href="http://www.numismatikforum.de/ftopic8877.html" title="http://www.numismatikforum.de/ftopic8877.html">
             mehr im Forum
            </a>
            <span class="urlexpansion">
             (
             <i>
              http://www.numismatikforum.de/ftopic8877.html
             </i>
             )
            </span>
           </li>
           <li>
            <strong>
             Fachliteratur
            </strong>
            : Hier sollte ein Index geschaffen werden der sämtliche auf dem Markt (oder auch nicht mehr auf dem Markt) befindliche Literatur zum Thema umfasst. Mit Kurzbeschreibungen, Meinungen usw. (jeder sollte hier die Möglichkeit haben, den Beitrag um seine Meinung zu ergänzen ... 1. Teil die allgemeine Beschreibung, 2. Teil über die Diskussionsfunktion eure Meinungen
           </li>
           <li>
            <strong>
             Vereine
            </strong>
            : Einfach ein Vereinsindex - mit Hintergrund-Infos wie Homepage, Anschrift usw.
           </li>
           <li>
            <strong>
             Hintergrund
            </strong>
            : ... wäre für geschichtliche Hintergründe, oder nur Tabellen die nicht direkt was mit der Münzgeschichte zu tun haben, aber zum Verstehen eines Themas doch nicht unwichtig sind (Stammbäume, Herrscherlisten usw.)
           </li>
           <li>
            <strong>
             Kategorie
            </strong>
            : Werden vom System weitgehendst selbst verwaltet. Mehrere Lexikoneinträge können einer Kategorie zugeordnet werden (siehe:
            <a href="/Numispedia:Handbuch_-_Kategorien" title="Numispedia:Handbuch - Kategorien">
             Handbuch - Kategorien
            </a>
            )
           </li>
          </ul>
