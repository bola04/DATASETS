      <h1>
       Herzlich willkommen!
      </h1>
      <a class="section_anchor" id="wb_1">
      </a>
      <p>
       "Begeisterung weckt, wer Sinne berührt"...
       <br/>
       <br/>
       Seit 1983 habe ich im Großraum Stuttgart
       <br/>
       als Livemusikerin schon viele Zuhörer mit
       <br/>
       Keyboard, Akkordeon oder Klavier unterhalten.
       <br/>
       <br/>
       Passende Musikrichtung, angenehme
       <br/>
       Lautstärke, spontane Künstlerbegleitung
       <br/>
       und bei Bedarf weitere Unterhaltung durch
       <br/>
       zusätzlich angebotene Programmpunkte
       <br/>
       und waren mir hierbei besonders wichtig.
       <br/>
       <br/>
       Inzwischen verlagert sich mein beruflicher
       <br/>
       Schwerpunkt immer mehr zu meinem
       <br/>
       Onlineshop
       <a href="https://soundwonderland.de" target="_blank">
        Soundwonderland
       </a>
       , über den
       <br/>
       ich Software für elektronische Keyboards
       <br/>
       und eigene Kompositionen verkaufe
       <br/>
       und auch mehrere Lernprojekte für
       <br/>
       Keyboard- und Klavierspieler anbiete.
      </p>
      <p>
       <a href="http://heidruns-musikerseiten.de" target="_blank">
        Heidruns Musikerseiten
       </a>
       sind seit 2001
       <br/>
       den Spielern von Yamaha Keyboards
       <br/>
       durch zahlreiche kostenlose Workshops
       <br/>
       und Downloads eine wichtige Anlaufstelle.
       <br/>
       <br/>
       Hier, auf den Seiten meiner privaten Homepage,
       <br/>
       bekommen Sie einige Informationen zu meiner Person.
      </p>
      <p>
       Ergänzt wird dies durch meine beiden Youtube Video-Kanäle:
       <br/>
       <a href="http://www.youtube.com/user/HeiDol" target="_blank">
        HeiDol
       </a>
       <a href="http://www.youtube.com/user/Soundwonderland" target="_blank">
        Soundwonderland
       </a>
      </p>
      <p>
       <img alt="" src="https://heidrun-dolde.de/media/pics/Titeltasten.png"/>
      </p>
      <h3>
       Hier die Schnellübersicht meiner kommenden öffentlichen Termine:
      </h3>
      <ul>
       <li>
        <span class="bold">
         Mittwoch, 26. September 2018
        </span>
        <span class="bold">
        </span>
        <br/>

        <br/>
        bei Musik City Steinbrecher in Pforzheim
       </li>
       <li>
        <strong>
         Sonntag, 9. und 16. Dezember 2018, nachmittags

       </li>
      </ul>
