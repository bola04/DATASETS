      <h1>
       Weinbergsführungen Kellerweg-Fest 2018
      </h1>
      <div class="imgbox">
       <img alt="me" src="img/content_img.jpg"/>
       <br/>
       <p>
        <br/>
       </p>
      </div>
      <p>
       <strong>
        Informative sowie unterhaltsame
 gefahrene Weinbergsführung
 durch die Weinberge von Guntersblum anlässlich des
 Kellerweg-Festes am 18. und 25. August 2018. Erkunden Sie dabei das
 rheinhessische Weinland auf einem typischen
 "Wingerts-Wägelschen". Machen Sie Rast an drei Stationen und
 genießen Sie den Wein an seinem Ursprung. Eine kleine
 Weinprobe an jeder Station und herrlichste Aus- und Freiblicke auf die
 Flußlandschaft des Rheins sind garantiert.
       </strong>
      </p>
      <p style="width: 608px;">
       Start ist der Marktplatz an der
 Evangelischen Kirche in Guntersblum in der Zeit zwischen 13:00 - 16:30
 Uhr. Ein dauernder Pendelservice von Traktoren mit Wagen bringt Sie auf
 einen Rundkurs von ca. 4 km. Machen Sie dabei Rast an drei Stationen
 direkt in den Weinbergen von Guntersblum. Das besondere daran: jede
 Station steht unter einem Motto, welches Ihnen sehr gerne von den dort
 befindlichen Betreuern vermittelt wird. Verweilen Sie an jeder Station
 nach Lust und Laune und so lange Sie wollen. Steigen Sie dann auf den
 nächsten freien Wagen und fahren zur
 nächsten Station. Natürlich steht eine kleine
 Weinprobe und Wasser zur Erfrischung an jeder Station für Sie
 bereit.
      </p>
      An Station 3 können Sie entscheiden, ob Sie mit dem
 Pendelservice zurückfahren oder
      <br/>
      den Fußweg durch die historische Enggaßhohl von ca.
 5 Minuten direkt zum Weinfest wählen.
      <br/>
      <br/>
      Die Veranstaltung endet gegen 18:30 Uhr - rechtzeitig zum
 Weinfest-Beginn.
      <br/>
