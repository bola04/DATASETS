        <h3>
         Plagiatskontrolle für Hochschulen und Universitäten
        </h3>
        <p>
         PlagAware wurde für die Plagiatsprüfung von wissenschaftlichen Texten wie Masterarbeiten, Bachelorarbeiten, Dissertationen oder Habilitationsschriften entwickelt. Hierfür greift PlagAware auf alle frei im Internet verfügbaren Texte zu, die von Suchmaschinen indexiert wurden. Aber auch eigene Referenzdokumente, wie etwa Arbeiten vergangener Jahrgänge oder lizenzierte wissenschaftliche Artikel, können in einer geschützten Bibliothek hinterlegt werden und stehen damit für die Plagiatsprüfung als weitere Quellen zur Verfügung.
        </p>

     <section class="row">
      <div class="container">
       <div class="col m6 l5 right">
        <img alt="teacher" class="responsive-img z-depth-3" src="/gfx/content/teacher.jpg"/>
       </div>
       <div class="col m6 l7">
        <h3>
         Plagiatsprüfung für Schulen
        </h3>
        <p>
         An Schulen wird das Thema "Plagiatsprüfungen von Seminararbeiten und Hausarbeiten" immer wichtiger. Hier bietet PlagAware individuelle Lösungen an, die speziell auf die Bedürfnisse von Schulen zugeschnitten sind. So können beispielsweise Klassensätze definiert werden, in denen Schüler und Schülerinnen ihre Arbeiten selbstständig übergeben können. Alle abgegebenen Arbeiten werden auf Wunsch Wort für Wort verglichen, um unerlaubte Gruppenarbeiten (Kollusion) aufzudecken.          function initShopDropzone() {
             var previewTemplate = '<div class="dz-preview dz-file-preview">' + '<div class="circle uploadCircle">' + '<strong></strong>' + '</div>' + '<div class="dz-details">' + '<div class="dz-filename"><span data-dz-name></span></div>' + '<div class="dz-size" data-dz-size></div>' + '<div>Hochladen...</div>' + '</div>' + '<div class="error-container">' + '<div>' + '<i class="material-icons">error</i>' + '<span class="error-message"></span>' + '</div>' + '<button class="waves-effect waves-light btn error-clear-btn">Ok</button>' + '</div>' + '<i data-dz-remove class="material-icons cancel-button">close</i></span>' + '</div>';
 
             var shopDropzone = new Dropzone(".shop-upload-form", {
                 url : "/api/index.php?cmd=UploadDropzoneFile", // Set the url
                 previewTemplate : previewTemplate,
                 acceptedFiles : '.txt,.odt,.rtf,.pdf,.doc,.docx,.jpg',
                 clickable : ".shop-upload-form .info-text",
                 maxFiles : 1,
                 maxFilesize : 15,
                 timeout: 300000
             });
 
             shopDropzone.on("addedfile", function(file) {
                 $('.shop-upload-form .info-text').hide();
                 $('.shop-upload-form .error-container').hide();
 
                 $('.uploadCircle').circleProgress({
                     value : 0,
                     animation : true,
                     fill : {
                         color : '#8b8ddf'
                     }
                 });
 
                 $('.uploadCircle strong').text('0').show();
                 $('.uploadCircle').circleProgress('value', 0);
             });
 
             shopDropzone.on("uploadprogress", function(file, progress, bytesSent) {
                 $('.uploadCircle').circleProgress('value', progress / 100);
                 $('.uploadCircle strong').html(progress.toFixed(0) + '%');
             });
 
             shopDropzone.on("canceled", function() {
                 $('.shop-upload-form .info-text').show();
             });
              
              shopDropzone.on("success", function(file) {
                     setTimeout(function() { 
                         document.location = "/de/shop-analysis"; 
                         }, 500);
                     
             });
 
 
             shopDropzone.on('sending', function(file, xhr, formData) {
                 formData.append('token', '018302dfb7130b42d3092dcf8ab9da21');
                 formData.append('timestamp', '1533144669');
             });
             
             shopDropzone.on("error", function(file, message, xhr) {
                 var errorMessage = '';
                 $('.shop-upload-form .info-text').hide();
                 $('.shop-upload-form .uploadCircle').hide();
                 $('.shop-upload-form .dz-details').hide();
                 $('.shop-upload-form .cancel-button').hide();
 
                 if (xhr) {
                     errorMessage = xhr.statusText;
                 }
                 if (!errorMessage) {
                     errorMessage = message;
                 }
                 $('.shop-upload-form .error-message').html(errorMessage || 'error');
                 $('.shop-upload-form .error-container').show();
             });
 
             $('.user-form').on('click', '.error-clear-btn', function(event) {
                 $('.shop-upload-form .error-container').hide();
                 shopDropzone.removeAllFiles(true);
                 $('.shop-upload-form .info-text').show();
             });
         }
