        <div class="csc-header csc-header-n1">
         <h1 class="csc-firstHeader">
          Das Weiterbildungszentrum Ingelheim
         </h1>
        </div>
        <!--  Header: [end] -->
        <!--  Text: [begin] -->
        <p class="bodytext">
         <b>
          Herzlich Willkommen auf den Seiten des Weiterbildungszentrums Ingelheim!
         </b>
         <br/>
         <br/>
         Das WBZ Ingelheim ist eine öffentliche Einrichtung der Jugend- und Erwachsenenbildung. Als gemeinnützige GmbH wird es getragen von der Stadt Ingelheim am Rhein und freundlich unterstützt vom Verein der Freunde des Fridtjof-Nansen-Hauses e.V.
         <br/>
         <br/>
         Die Arbeit unseres Zentrums ist parteipolitisch neutral und konfessionell ungebunden. Gemäß unserem
         <a class="download" href="fileadmin/user_upload/LeitbildWeiterbildungszentrumIngelheim.pdf" target="_blank">
          <b>
           Leitbild
          </b>
         </a>
         bemühen wir uns um ein hochwertiges Programm, das sich durch Kundennähe,  faire Preise, Verlässlichkeit und Kompetenz auszeichnet. In vier  selbsttätig agierenden Fachbereichen bieten wir einem breiten Publikum dabei Angebote aus den folgenden Bereichen:
         <br/>
         <br/>
         • politische Bildung
         <br/>
         • personen- und familienbezogene Bildung
         <br/>
         • sprachliche, berufliche und berufsbezogene Bildung
         <br/>
         • gesundheits- und kreativitätsfördernde Bildung
         <br/>
         • musikalische Bildung
        </p>
        <p class="bodytext">
         Zur schnellen  Programmsuche navigieren Sie bitte über die Reiter der vier Fachbereiche
         <a class="internal-link" href="http://neu.wbz-ingelheim.de/?id=16" target="_top" title="Öffnet einen internen Link im aktuellen Fenster">
          <br/>
          Fridtjof-Nansen-Akademie
         </a>
         (FNA),
         <br/>
         <a class="internal-link" href="volkshochschule/" target="_top" title="?fnet einen internen Link im aktuellen Fenster">
          Volkshochschule
         </a>
         (VHS),
         <a class="internal-link" href="http://neu.wbz-ingelheim.de/?id=6" target="_top" title="Öffnet einen internen Link im aktuellen Fenster">
          <br/>
          Jugendbildungswerk
         </a>
         (JBW) oder
         <a class="internal-link" href="http://neu.wbz-ingelheim.de/?id=5" target="_top" title="Öffnet einen internen Link im aktuellen Fenster">
          <br/>
          Musikschule
         </a>
         (MS).
         <br/>
         <br/>
         Weitere Informationen über das WBZ Ingelheim finden Sie über die Navigationsleiste am oberen Seitenrand.
         <br/>
         <br/>
         <br/>
        </p>
