        <b>
         SmallSQL
        </b>
       </i>
      </p>
      <p align="center" style="font-size: 14pt">
       the ultimate Java Desktop SQL Database Engine with JDBC 3.0 API.
      </p>
      <p align="center">
       SmallSQL is small, fast, embeddable, SQL 99 conformance, platform independent and
       <a href="100-percent-pure-java-dbms.html">
        100% pure Java
       </a>
      </p>
      <hr/>
      <blockquote>
       <p>
        <b>
         Advantages and Disadvantages of the SmallSQL Database
        </b>
       </p>
       <p>
        Why and when should you use 
 the SmallSQL Database and when not?
       </p>
      </blockquote>
      <table width="100%">
       <tr>
        <th width="50%">
         Advatages
        </th>
        <th width="50%">
         Disadvantages
        </th>
       </tr>
       <tr>
        <td>
         <ul>
          <li>
           It has a LGPL license. You can redistribute it with your software for 
 	free.
          </li>
          <li>
           It is
           <a href="100-percent-pure-java-dbms.html">
            100% pure Java
           </a>
           . You can use 
 	it on all platforms with Java version 1.4 or higher.
          </li>
          <li>
           It is open source. You can look in the source code if you want it.
          </li>
          <li>
           It is embeddable in you application. There is no additional setup 
 	required.
          </li>
          <li>
           It is a very small Java library.
          </li>
          <li>
           The quality is tested with
 	JUnit and
           <a href="coverage/index.html" title="Emma Coverage">
            Emma Coverage
           </a>
           .
          </li>
         </ul>
        </td>
        <td>
         <ul>
          <li>
           It does not has a network interface.
          </li>
          <li>
           It required
           <a href="100-percent-pure-java-dbms.html">
            Java
           </a>
           . No other access API is available currently.
          </li>
          <li>
           You can't share the database between different applications at the same time.
          </li>
          <li>
           There are no user management.
          </li>
         </ul>
        </td>
