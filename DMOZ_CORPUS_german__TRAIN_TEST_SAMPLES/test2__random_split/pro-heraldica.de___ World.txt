  	    existingratio = checkRetina(possibleRetina);
     
  	    return item.src.replace(/\.\w+$/, function(m) { return existingratio + m; });
  	    } // function that changes image source
  	  }	  
  		  // other options
  	});
  });
   </script>

          <header>
           <h3>
            Siegelringe
           </h3>
          </header>
          <p>
           Nur eine gelebte Tradition hat Überlebenschancen. Nutzen Sie 18. und runde Geburtstage sowie andere familiäre Anlässe, Schmuckstücke mit Ihrem Familienwappen an Ihre Lieben weiterzugeben. Unser Klassiker ist der Siegelring. Alle unsere Siegelringe sind handgeschmiedet. Sie haben die Auswahl aus mehr als 100 verschiedenen Variationen.
          </p>
         </article>
        </div>

          <header>
           <h3>
            Firmenchronik
           </h3>
          </header>
          <p>
           Die Firmenchronik erzählt die Geschichte Ihres Unternehmens und der Unternehmensgründer, enthält Geschichten aus dem Alltag und historische Dokumente. Sie unterstreicht das Interesse der Firma an Tradition. Mit einer Firmenchronik schaffen Sie eine emotionale Bindung Ihrer Mitarbeiter, Kunden und Partnern an Ihr Unternehmen.
          </p>
         </article>
        </div>

          <header>
           <h3>
            Wappenregistrierung
           </h3>
          </header>
          <p>
           Registrieren Sie jetzt Ihr Wappen in der Allgemeinen Deutschen Wappenrolle (ADW). Ihr Wappen erhält Rechtsschutz für Deutschland. Die Registrierung beinhaltet neben der Veröffentlichung in der alle 1-2 Jahre erscheinenden Allgemeinen Deutschen Wappenrolle auch einen ledergebundenen Wappenbrief.
          </p>
