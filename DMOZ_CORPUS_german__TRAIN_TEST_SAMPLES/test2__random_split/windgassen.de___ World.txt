             <h1 class="title">
              In Einklang mit der Umwelt
             </h1>
            </header>
            <div class="content clearfix">
             <p>
              W
              <img alt="Herzlich Willkommen auf Abwassertechnik Windgassen" border="0" height="182" hspace="6" src="/system/images/stories/abwassertechnik_r15_c10.jpg" style="float: right;" title="Herzlich Willkommen auf Abwassertechnik Windgassen" width="184"/>
              ir sind Hersteller von Fällungsmitteln auf Aktivkohle und Bentonit-Basis für die chemisch-physikalische Reinigung industrieller, technischer und galvanischer Abwässer, welche bisher bei unseren Kunden zu hervorragenden Ergebnissen geführt haben.
              <br/>
              <br/>
              Da wir ein Familienunternehmen mit einer schlanken Unternehmensstruktur sind, garantieren wir Flexilbilität durch kurze Entscheidungswege.
              <br/>
              <br/>
              Die Anlieferung eines Auftrages erfolgt schnellstmöglich mit dem eigenen PKW oder einer Spedition innerhalb von 24-48 Stunden.
              <br/>
              <br/>
             </p>
             <p>
              <img alt="k-Foto1 Nickel" height="108" src="/images/joerns_bilder/k-Foto1_Nickel.JPG" width="145"/>
              <img alt="k-Foto3 Nickel" height="107" src="/images/joerns_bilder/k-Foto3_Nickel.JPG" width="144"/>
              <br/>
              <br/>
             </p>
             <p>
              Durch den Einsatz unserer Fällungsmittel entstehen für Sie folgende Vorteile:
             </p>
             <ul>
              <li>
               <strong>
                Sehr kurze Absetzzeit
               </strong>
              </li>
              <li>
               <strong>
                Sauberes Wasser ohne Anlagen-Investition
               </strong>
              </li>
              <li>
               <strong>
                Verringerung der Deponiekosten
               </strong>
              </li>
              <li>
               <strong>
                Geringer Verbrauch
               </strong>
              </li>
              <li>
               <strong>
                Lange Lebensdauer der Filtertücher
               </strong>
              </li>
              <li>
               <strong>
                Reduzierung des Verbrauchs von Natronlauge oder Kalkmilch bei dem Einsatz von SUT 90-K
               </strong>
              </li>
             </ul>
             <p>
              <br/>
              <strong>
               Wir können unsere Produkte entsprechend auf Ihr Abwasser hin individuell abstimmen.
              </strong>
              <br/>
              <br/>
              Zusätzlich bieten wir fünf Komplexspalter an, die jeweils auf verschiedene Metallkomplexe spezialisiert sind.
              <br/>
              <br/>
              Unsere Produkte zur Abwasserbehandlung finden bei folgenden Branchen Verwendung:
             </p>
             <ul>
              <li>
               Galvanische Betriebe
              </li>
              <li>
               Teilereinigung
              </li>
              <li>
               Leiterplatten-Fertigung
              </li>
              <li>
               Pulverbeschichtung
              </li>
              <li>
               KTL-Beschichtung / Zink-Phosphatierung
              </li>
              <li>
               Gleitschleifereien
              </li>
              <li>
               Druckgussverfahren
              </li>
              <li>
               Aluminium-Profile
              </li>
              <li>
               Produktion Leichtmetallräder
              </li>
              <li>
               Eloxieren von Aluminium
              </li>
              <li>
               Produktion von Dispersionsfarben / Gebindereinigung
              </li>
              <li>
               Mineralölhaltige Abwässer / Emulsionen
              </li>
              <li>
               Glasbearbeitung
              </li>
             </ul>
             <p>
              <br/>
              <strong>
               Wir bieten Ihnen eine ausführliche Laboranalyse Ihres Abwassers mit unserem Spektralphotometer DR 2800 an.
              </strong>
              <br/>
              Folgende Schwermetallionen können zur Zeit mit diesem Photometer gemessen werden:
             </p>
             <p>
              - Aluminium
              <br/>
              - Chrom (III und IV)
              <br/>
              - Fluorid
              <br/>
              - Kupfer
              <br/>
              - Nickel
              <br/>
              - Silber
              <br/>
              - Sulfat
              <br/>
              - Zink
              <br/>
              - Zinn
              <br/>
              - CSB
             </p>
             <p>
              Nähere Informationen entnehmen Sie bitte den weiteren Internetseiten.
              <strong>
               Sind Sie interessiert?
              </strong>
              Dann können Sie mit uns Kontakt aufnehmen, um weitere Informationen zu erhalten. Wir stehen Ihnen auch jederzeit gerne für eine persönliche Vorstellung unserer Produkte zur Verfügung.
              <br/>
              <br/>
              Des Weiteren entnehmen Sie der Anlage (PDF Datei) unsere Teilnahmebescheinigung zum Thema
              <strong>
               Sachkunde Sicherheitsdatenblatt.
               <br/>
               <a href="/images/Teilnahmebescheinigung.pdf">
                Teilnahmebescheinigung BG RCI Sachkunde Sicherheitsdatenblatt
               </a>
               <br/>
              </strong>
             </p>
