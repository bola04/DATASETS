      <h2>
       Allgemeine Informationen
      </h2>
      <h3>
       Öffnungszeiten nach Wiedereröffnung:
      </h3>
      Anfang April bis Ende Oktober,
      <br/>
      sonntags von 14:00 bis 18:00 Uhr.
      <br/>
      Eintrittspreise:
           Erwachsene: 2,00 Euro,
      <br/>
      Freier Eintritt für Kinder und Jugendliche.
      <br/>
      <br/>
      Vereinbarungen für Gruppentermine außerhalb der Öffnungszeiten
                 unter 06345 - 8484 oder
      <a class="email" href="mailto:gruppen@papiermuseum-gleisweiler.de">
       &gt;Email senden
      </a>
      .
      <br/>
      Erwachsenengruppe 20 Euro + 1,50 Euro pro Person,
      <br/>
      Schulklassen werden um eine Spende gebeten.
      <h3>
       Spezielle Angebote zum Papierschöpfen:
      </h3>
      <br/>
      Nach Abstimmung können Sonntags die Besucher unter fachkundiger Anleitung
                 selbst auf traditionelle Weise Papier schöpfen.
      <br/>
      <br/>
      Im September und Oktober ist das Papierschöpfen für Feriengäste
           jeweils mittwochs um 10:30 Uhr geplant.
      <br/>
      Anmeldung erbeten unter 06345 - 8484.
      <h3>
       Ort:
      </h3>
      Kurpfälzischer Zehnthof
      <br/>
      Kellergeschoss
      <br/>
      Zum Sonnenberg 1
      <br/>
      76835 Gleisweiler
      <br/>
      GPS: N 49° 24,075 - E 008° 08,06378
      <br/>
      Kontakt:
      <a class="email" href="mailto:info@papiermuseum-gleisweiler.de">
       &gt;Email senden
      </a>
      .
