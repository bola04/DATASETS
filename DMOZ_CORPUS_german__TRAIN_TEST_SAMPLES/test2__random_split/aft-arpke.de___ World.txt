         <h1>
          Nützliche Tipps zum Waveboard fahren lernen
         </h1>
         <p>
          Ein Waveboard ist ein Funsportgerät, welches einem Skateboard ähnlich verwendet wird. Im Gegensatz zum traditionellen Skateboard (
          <a href="http://street-skate.de/penny-board/">
           Penny Board
          </a>
          ) besteht es jedoch nicht aus einer starren Platte, sondern wird aus zwei Fußplatten gefertigt, welche miteinander verbunden sind und eine gegenläufige Drehung der einzelnen Platten um die Längsachse erlaubt. Durch die höhere Beweglichkeit ist auch eine größere Einsatzmöglichkeit im Vergleich zum Skateboard möglich.
         </p>
         <h2>
          Die unterschiedlichen Arten von Waveboards
         </h2>
         <p>
          Waveboards werden hauptsächlich in 2 Ausführungen angeboten. Bei der Variante 1 sind die beiden Decks, wie die Fußplatten auch genannt werden, auf eine Torsionsstange gesteckt und dadurch miteinander verbunden. Eine gegenläufige Bewegung der beiden Platten während der Fahrt kann so ermöglicht werden. Diese Variante ist jedoch recht schwer. Die neueren Modelle kommen daher ohne jegliche Torsionsstange aus. Möglich wird dies durch das neue flexible und robustere Material, welches heute verwendet wird. Das Waveboard ist leiser und leichter zu manövrieren. Ein Waveboard, welches ein geringeres Gewicht besitzt, erfordert weniger Kraftaufwand bei der Vorwärtsbewegung durch das Verdrehen der Decks. Für Anfänger sind daher die neuen Waveboards ohne Torsionsstange besser geeignet, besonders, wenn sie Tipps zum Waveboard fahren lernen beachten.
         </p>
         <p>
          <iframe allow="encrypted-media" allowfullscreen="" frameborder="0" gesture="media" height="371" src="https://www.youtube.com/embed/pmjBWvChhbM?feature=oembed" width="660">
          </iframe>
         </p>
         <h2>
          7 wertvolle Tipps zum Waveboard fahren lernen
         </h2>
         <p>
          1. Das Stehen auf dem Waveboard
          <br/>
          Genau wie beim Skateboarden müssen Sie vor der ersten Fahrt erst einmal ihren dominanten Fuß ermitteln. Diesen müssen Sie dann auf dem vorderen Deck platzieren. Wenn Sie sich nicht sicher sind, welcher ihr bevorzugter Fuß ist, so überlegen Sie sich, mit welchem Fuß Sie sich abstützen, wenn Sie stolpern. Der Fuß, den Sie automatisch nach vorne setzen, ist Ihr dominanter Fuß und gehört nach vorne.
          <br/>
          2. Der geeignete Untergrund
          <br/>
          Als Anfänger sollten Sie niemals an einem Gefälle die ersten Übungen versuchen. Die Fahrbahn sollte asphaltiert und absolut eben sein, so können Sie in Ruhe die ersten Fahrversuche starten, ohne, dass sich das Waveboard selbstständig macht.
          <br/>
          3. Das Aufsteigen auf das Waveboard
          <br/>
          Für das Fahren selbst muss man nicht studiert haben. Ein wenig Gleichgewichtssinn und Übung sowie die richtige Technik reichen völlig aus, um Waveboard fahren zu lernen. Stellen Sie den dominanten Fuß mittig auf das vordere Deck, der Schwerpunkt sollte auf den Zehen liegen. Balancieren Sie das Board in eine waagerechte Stellung und stoßen Sie sich mit dem anderen Fuß vom Boden ab. Stellen Sie ihn anschließend möglichst mittig auf das hintere Deck. Nun kann die Fahrt losgehen.
          <br/>
          4. Das stilvolle Bremsen
          <br/>
          Zu den Tipps zum Waveboard fahren lernen sollte natürlich die Anleitung zum Bremsen nicht fehlen. Denn wenn Sie gekonnt Waveboard fahren möchten, sollten Sie zuerst vor allem das Bremsen üben. Anfangs natürlich mit einer geringen Geschwindigkeit und auf einem ebenen glatten Untergrund. Das Bremsen ist gegenüber dem Skateboard fahren auch einfacher. Dafür genügt es, während der Fahrt den Druck beider Füße nach vorn auf die Zehen und Ballen zu lenken. Das Wavebord wird langsam und legt sich auf die Seite. Üben Sie nun das timegenaue Absteigen in Fahrtrichtung.
          <br/>
          5. Kurven lernen
          <br/>
          Erst, wenn Sie Ihr Gleichgewicht auf gerader Strecke problemlos halten können, sollten Sie sich an die ersten Kurven wagen. Da das Waveboard entgegen dem Skateboard nur über eine routierende Rolle pro Deck verfügt, kann das Board durch die reine Gewichtsverlagerung erfolgen. Verlagern Sie das Gewicht des dominanten Fußes auf die Ferse, so beschreiben Sie eine Kurve in Richtung Zehen. Auf den hinteren Fuß muss dabei gleichzeitig Druck auf die Zehen ausgeübt werden. Richtungswechsel = entgegengesetzte Gewichtsverlagerung.
         </p>
         <h2>
          Fazit zum “Tipps zum Waveboard fahren lernen”
         </h2>
         <p>
          Waveboard fahren ist einfach und auch für Einsteiger gut zu lernen. Wer sich an die Tipps zum Waveboard fahren lernen aus dem Internet hält und konsequent übt, wird sehr schnell Freude an diesem Funsport finden, hier gibt es mehr
          <a href="https://street-skate.de/">
           Infos zum Thema
          </a>
          .
         </p>
