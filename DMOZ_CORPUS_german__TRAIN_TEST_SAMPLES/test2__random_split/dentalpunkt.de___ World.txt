            <h2>
             Herzlich willkommen bei Ihren Zahnärzten Enders &amp; Hofmann
            </h2>
            <p style="color: #232e69;">
             Ihr strahlendes, gesundes Lächeln voller Selbstvertrauen – das ist unsere Motivation
            </p>
            <p>
             Durch unsere Erfahrung, unsere Philosophie und dem hohen Qualitätsanspruch an unsere eigene Leistung bieten wir erstklassige ZahnMedizin. Bei uns erhalten Sie Leistungen, die auf Ihre Bedürfnisse angepasst sind – von der Prophylaxe bis zur komplexen Zahnsanierung.  			{

 				var c_value=escape(value) + ((exdays==null) ? "" : "; expires="+exdate.toUTCString()) + "; path=/";

 				var nameEQ = name + "=";
 				var jb = document.cookie.split(';');

 					var c = jb[i];
 					while (c.charAt(0)==' ') c = c.substring(1,c.length);
 						if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
 					}
 				return null;
 			}
