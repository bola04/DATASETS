          </div>
          <div class="">
           <p>
            <strong>
             <a href="https://www.caritas-nah-am-naechsten.de/therapeutisches-zentrum-maedchenheim-gauting/cont/29701">

             </strong>

            <br/>
            <br/>
            <br/>
            <br/>
            <br/>
            <strong>
             Das Mädchenheim Gauting ist eine Jugendhilfeeinrichtung unter Trägerschaft des Caritasverbandes der Erzdiözese München und Freising e.V. mit vielfältigen pädagogischen und therapeutischen Angeboten.
            </strong>
            <br/>
            <br/>
            <br/>
            <br/>
            Im
            <strong>
             Mädchenheim
            </strong>
            betreuen wir auf 8 (intensiv)-therapeutischen Gruppen ausschließlich Mädchen ab 13,5 Jahren. Es stehen hierfür sechs individuell geschlossene Wohn-Gruppen sowie zwei offene Wohngruppen zur Verfügung. Insgesamt hat das Mädchenheim 57 Plätze.
            <br/>

            <br/>
            <br/>
            <br/>
            Das
            <strong>

              Caritas Jugendbüro
