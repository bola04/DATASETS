           <h1>
            Wowa
           </h1>
           <p align="left">
            <b>
             Kunstprojekt 2018
            </b>
            <br/>
            <br/>
            Eröffnung: 22.08.2018 (Mittwoch) 17:00 Uhr Magnusplatz
            <br/>
            <br/>
            <br/>
            <b>
             Wowa wird mobil: Der Dorfmaler kommt für sechs Wochen zurück
            </b>
            <br/>
            <br/>
            Der Maler Wladimir Gurski, besser bekannt unter dem Namen Wowa, kommt
            <br/>
            zurück nach Everswinkel. Schon vor mehr als zehn Jahren war der Künstler
            <br/>
            zu Gast in der Vitus-Gemeinde.
            <br/>
            <br/>
            Sein "Atelier"-Container auf dem Magnusplatz war damals Ort für viele Ge-
            <br/>
            spräche und Begegnungen. Wowa tauchte aber auch tief in die örtliche Ge-
            <br/>
            sellschaft ein: Feuerwehr, Kindergarten, Schule, Kirche, Landfrauen. Er sam-
            <br/>
            melte viele Eindrücke und hielt sie in seinen Bildern fest.
            <br/>
            <br/>
            Aufgrund der vielfachen positiven Rückmeldungen hat sich der Kulturkreis
            <br/>
            entschlossen, Wowa nach zwölf Jahren erneut einzuladen. Da die Kultur-
            <br/>
            stiftung der Sparkasse Warendorf als Förderer des damaligen Projektes
            <br/>
            auch jetzt wieder Unterstützung zugessagt hat, steht der Rückkehr des
            <br/>
            Dorfmalers Mitte August für sechs Wochen nichts mehr im Wege.
            <br/>
            <br/>
            Was wird dieses Mal anders? Wowa wird mobil! Sein Atelier wird er in
            <br/>
            einem wunderschönen dunkelroten Bauwagen beziehen. Dieser wird im
            <br/>
            etwa wöchentlichen Rhythmus an einer anderen Stelle stehen - in Evers-
            <br/>
            winkel oder Alverskirchen.
            <br/>
            Sein Hauptaugenmerk wird dieses Mal auf die Menschen, auf besondere
            <br/>
            Charaktere gerichtet sein. Ende September wird es eine erste Ausstellung
            <br/>
            seiner Arbeiten geben.
            <br/>
            <br/>
            Im Frühjahr 2019 wird er dann nochmals zurückkegren und die inzwischen
            <br/>
            in heimatlicher Ruhe gefertigten Werke präsentieren.
            <br/>
            Alle Bürgerinnen und Bürger sind schon jetzt herzlich eingeladen, Wowa
            <br/>
            nach den Ferien kennenzulernen, oder die Bekanntschaft aufzufrischen.
            <br/>
            Am 22. August um 17 Uhr wird er auf dem Magnusplatz begrüßt. Der
            <br/>
            Kulturkreis hat sich einiges ausgedacht, insbesondere für Kinder.
            <br/>
            <br/>
            <br/>
            <br/>

            <br/>
            <br/>
            <br/>
           </p>

           <h1>
            Bühne frei!
           </h1>
           <p align="left">
            <b>
             Kleinkunst in Everswinkel
            </b>
            <br/>
            <br/>
            Gasthof Diepenbrock, 19:30 Uhr
            <br/>
            <br/>
            Einlass ab 19:00 Uhr
            <br/>
            <br/>
            Eintritt frei
            <br/>
            <br/>
            Was ist "Bühne frei!"
            <br/>
            <br/>
            Mit diesem Format soll viermal im Jahr der Kleinkunst im Vitusdorf ein Forum
            <br/>
            geboten werden. Alle Menschen zwischen 8 und 88 sind eingeladen, sich im
            <br/>
            Gasthof Diepenbrock einem wohlwollenden Publikum zu präsentieren.
            <br/>
            <br/>
            Jede Darstellungsform ist willkommen, ob Kabarett oder Comedy,
            <br/>
            Musikalisches oder Literarisches, ob Jonglage oder Zauberei, ob Akrobatik
            <br/>
            oder Tanz  je unterschiedlicher und bunter das Angebot, desto unterhalt-
            <br/>
            samer ist die Show für die Zuschauer. Nach jeweils etwa 15-20 Minuten
            <br/>
            wechseln die Akteure, es gibt eine Pause. Eine Ton- und Lichtanlage steht
            <br/>
            zur Verfügung, auch ein Klavier ist vorhanden. 
 
 Der Auftritt wird im fotogra-
            <br/>
            fisch festgehalten, einige Bilder werden auch auf kulturkreis-everswinkel.de
            <br/>
            und auf Facebook zu sehen sein.
 
 Beste Voraussetzungen
 also, Mut zu fas-
            <br/>
            sen und etwas Erprobtes vorzuführen oder etwas Neues auszuprobieren.
            <br/>
            <br/>
            <br/>
            Bei uns sind Amateure zu Gast, aber auch Semi-Professionelle und echte
            <br/>
            Profis - vielleicht auch einmal Sie? Wir würden uns jedenfalls freuen!
            <br/>
            <br/>
            <br/>
            Kontakt:
            <a href="mailto:buehne.frei@kulturkreis-everswinkel.de">
             buehne.frei@kulturkreis-everswinkel.de
            </a>
            <br/>

            Facebook:
            <a href="http://facebook.com/buehnefrei">
             http://facebook.com/buehnefrei
            </a>
