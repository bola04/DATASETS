               <h1 class="schrift2" style="text-align: justify; padding-left: 30px;">
                Sehr geehrte Damen und Herren,
               </h1>
               <p class="schrift2" style="text-align: justify; padding-left: 30px;">
                herzlich willkommen auf der Homepage der Waldschule Degerloch.
               </p>
               <p class="schrift2" style="text-align: justify; padding-left: 30px;">
                Die Waldschule Degerloch ist eine Schule in freier Trägerschaft. Seit über 140 Jahren ist die Schule eine interessante Alternative zu den staatlichen Schulen. Sie wird von den Eltern der Schüler und den Lehrerinnen und Lehrern selbst getragen. „Schule als Lebensraum“ ist hier nicht nur pädagogische Floskel, denn als Ganztagsschule mit jahrzehntelanger Erfahrung steht die Waldschule für die Werte:
               </p>
               <ul>
                <li>
                 <strong>
                  wertschätzend,
                 </strong>
                </li>
                <li>
                 <strong>
                  zuverlässig,
                 </strong>
                </li>
                <li>
                 <strong>
                  verantwortungsvoll.
                 </strong>
                </li>
               </ul>
               <p class="schrift2" style="text-align: justify; padding-left: 30px;">
                Lernen und Leben findet hier in einer von gegenseitigem Vertrauen getragenen Umgebung statt.
               </p>
               <p class="schrift2" style="text-align: justify; padding-left: 30px;">
                An der Waldschule werden ein zweizügiges allgemeinbildendes Gymnasium (Klassen 5-12) und eine zweizügige Realschule (Klassen 5-10) angeboten. Zwei Schularten befinden sich also unter einem Dach. Als Verbundschule halten wir die Übergänge zwischen den Schularten offen. Der dreijährige Realschulaufsetzer an der Waldschule führt nach der Mittleren Reife zum Abitur. Die Waldschule bietet somit einen achtjährigen (Gymnasium) und einen neunjährigen Weg (Realschule und Realschulaufsetzer) zum Abitur.  Alle Schularten sind staatlich anerkannt und führen zu den jeweiligen Abschlüssen Abitur und Mittlere Reife. Zurzeit wird die Schule von 668 Schülern besucht. Die Waldschule Degerloch ist der Montessori-Pädagogik verpflichtet (Freiarbeit, Individuelles Schülerarbeiten -ISA- im Stundenplan): Der Weg zu den Abschlussprüfungen ist ein Weg zu Selbstständigkeit und Selbstverantwortung.
               </p>
               <p class="schrift2" style="text-align: justify; padding-left: 30px;">
                Im September 2018 startet die neue Grundschule an der Waldschule Degerloch mit zwei ersten Klassen. Die Grundschule wird bis 2021 voll ausgebaut sein.

 function addCookie() {
 	var curDate = new Date();
 	curDate.setTime(curDate.getTime() + (86400000 * 365));
 	document.cookie = 'cookieTermsAccepted=1; expires=' + curDate.toUTCString();
 	fadeOut(document.getElementById('cookie-alert'));
 }
 
 function cookieExists() {
 	var cookies = document.cookie.split(';');
 	for(var i=0; i < cookies.length; i++) {
 		if (cookies[i].trim() == 'cookieTermsAccepted=1') {
 			return true;
 		}
 	}
 	return false;
 }
 
 function fadeOut(element) {
     var op = 1;
     var timer = setInterval(function () {
         if (op <= 0.15){
             clearInterval(timer);
             element.style.display = 'none';
         }
         element.style.opacity = op;
         element.style.filter = 'alpha(opacity=' + op * 100 + ')';
         op -= op * 0.15;
     }, 50);
 }
 
 function fadeIn(element) {
     var op = 0.09;
     element.style.display = 'block';
     var timer = setInterval(function () {
         if (op >= 1){
             clearInterval(timer);
         }
     }, 25);
 }
