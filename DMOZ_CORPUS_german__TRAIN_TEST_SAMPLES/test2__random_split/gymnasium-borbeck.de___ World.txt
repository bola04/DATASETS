         <article class="uk-article" data-permalink="http://gymbo.de/index.php/151-willkommen">
          <div>
           <p>
            <strong>
             Herzlich willkommen am Gymnasium Borbeck!
            </strong>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Wir gehören zu den wenigen Gymnasien in Nordrhein-Westfalen, die sich bewusst für die Möglichkeit, das Abitur nach 9 Jahren abzulegen, entschieden haben. Der
             <a href="/index.php/g9-neu" rel="alternate">
              Bildungsgang G9 neu
             </a>
             im Modellversuch bedeutet für unsere Schülerinnen und Schüler, dass sie genug Zeit zum Lernen und für außerschulische Aktivitäten haben.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Das Gymnasium Borbeck lebt seit vielen Jahren mit großer Selbstverständlichkeit eine Zuwendung zum Internationalen und Multikulturellen.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Der
             <a href="/index.php/biligualitaet" rel="alternate">
              bilinguale Zweig
             </a>
             mit den Sachfächern Erdkunde, Geschichte und Politik in Englisch bietet die Chance, ein hohes Niveau an Sprachkompetenz zu erreichen in einer modernen Welt, in der Englisch als Weltsprache immer wichtiger wird.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-family: verdana,geneva; font-size: 10pt;">
             Austausche, Auslandsbetriebspraktika und Schulprojekte ermöglichen wertvollle interkulturelle Erfahrungen mit unseren Partnern in Frankreich, Großbritannien, den Niederlanden und auch den USA.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Förderung bedeutet an unserer Schule, sowohl die leistungsschwächeren als auch die besonders leistungsstarken Schülerinnen und Schüler zu fördern und zu fordern.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Die Begabtenförderung erfolgt im Rahmen sogenannter
             <a href="/index.php/foerderung/begabtenfoerderung" rel="alternate">
              Drehtürprojekte
             </a>
             mit den Schwerpunkten Englisch und neuerdings auch MINT-Fächern (Mathematik, Informatik und Naturwissenschaften).
             <br/>
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-family: verdana,geneva; font-size: 10pt;">
             Dank eines starken und engagierten
             <a href="/index.php/foerderung/foerderverein" target="_blank">
              Fördervereins
             </a>
             können zusätzliche Anschaffungen getätigt werden oder andere Projekte wie z.B. die Schulhofumgestaltung realisiert werden.
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             <span style="line-height: 115%;">
              Darüber hinaus kann das Gymnasium Borbeck durch die
              <a href="/index.php/foerderung/lothar-boening-stiftung">
               Lothar-Böning Stiftung
              </a>
              auf weitere finanzielle Hilfe bei der Förderung der Schulausbildung aller Schülerinnen und Schüler zählen, zum Beispiel durch das Angebot von subventionierter Nachhilfe.
             </span>
             Der Gedanke der Chancengleichheit ist einer der Leitgedanken der Schule.
             <br/>
            </span>
           </p>
           <p style="text-align: justify;">
            <span style="font-size: 10pt; font-family: verdana,geneva;">
             Gemeinsamer Unterricht für Schülerinnen und Schüler mit besonderem Förderbedarf ist ebenfalls Teil unseres Schullebens.
            </span>
           </p>
           <p>
           </p>

 function addCookie() {
 	var curDate = new Date();
 	curDate.setTime(curDate.getTime() + (86400000 * 365));
 	document.cookie = 'cookieTermsAccepted=1; expires=' + curDate.toUTCString();
 	fadeOut(document.getElementById('cookie-alert'));
 }
 
 function cookieExists() {
 	var cookies = document.cookie.split(';');
 	for(var i=0; i < cookies.length; i++) {
 		if (cookies[i].trim() == 'cookieTermsAccepted=1') {
 			return true;
 		}
 	}
 	return false;
 }
 
 function fadeOut(element) {
     var op = 1;
     var timer = setInterval(function () {
         if (op <= 0.15){
             clearInterval(timer);
             element.style.display = 'none';
         }
         element.style.opacity = op;
         element.style.filter = 'alpha(opacity=' + op * 100 + ')';
         op -= op * 0.15;
     }, 50);
 }
 
 function fadeIn(element) {
     var op = 0.09;
     element.style.display = 'block';
     var timer = setInterval(function () {
         if (op >= 1){
             clearInterval(timer);
         }
     }, 25);
 }
