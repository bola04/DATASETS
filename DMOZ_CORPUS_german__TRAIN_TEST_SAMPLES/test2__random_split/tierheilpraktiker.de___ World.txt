         <img alt="Tierheilpraktikerverband - Verband Deutscher Tierheilpraktiker VDT" height="143" src="/images/stories/tierheilpraktiker/tierheilpraktikerverband-start.jpg" style="margin: 5px; float: left;" width="250"/>
         Herzlich willkommen
        </h2>
        <p>
         beim mitgliederstärksten Berufsverband für Tierheilpraktiker/innen. Sind Sie Tierbesitzer/ Tierfreund, laden wir Sie ganz herzlich ein, sich ein Bild von unserer Arbeit zu machen, sich über die Möglichkeiten zu informieren, Ihren Schützlingen mit  naturgemäßen, nebenwirkungsfreien Methoden der Naturheilkunde zu helfen. Sind Sie selbst Tierheilpraktiker,
         <a href="http://www.paracelsus.de/ausbildung/thp/fr_thp1.asp" title="Ausbildung Tierheilpraktiker">
          Tierheilpraktiker in Ausbildung
         </a>
         oder Tierfreund, der sich aktiv einbringen möchte, laden wir Sie ein, Mitglied in unserer Gemeinschaft zu werden. Wichtigstes Anliegen der Verbandsarbeit ist, dem Berufsstand des Tierheilpraktikers zu dem Stellenwert zu verhelfen, der ihm gebührt. Gerade weil die gesetzliche Normierung fehlt, sollte die Mitgliedschaft im VDT selbstverständliche Pflicht für jeden THP und THP-Anwärter sein!
         <br/>
         <br/>
         Unser Konzept hat bereits
         <a href="https://www.tierheilpraktiker.de/mitgliederbereich/mitglieder-map.html" title="">
          3484
         </a>
         Kolleginnen und Kollegen überzeugt und täglich werden es mehr! Nur 90 € im Jahr (7,50  € monatlich) beträgt der Mitgliedsbeitrag, der bei Fortbildung komplett eingespart werden kann! Werden Sie Mitglied im VDT, arbeiten Sie mit uns gemeinsam an der Konsolidierung unseres Berufsstandes und profitieren Sie von unseren umfangreichen Leistungen. Kostenloser Eintrag in die Therapeutendatenbank, günstige Gruppenversicherungen, viele beitragsfreie Infoseminare, regionale Verbandsstammtische, zwei überregionale THP Kongresse im Jahr, Stempel und andere Praxisausrüstung supergünstig, die Verwendung des Verbandslogos - die Verbandsmitgliedschaft lohnt sich wirklich.
         <br/>
         <br/>
         Für 45 € reduzierten Jahresbeitrag können auch Tierfreunde und Tierhalter als fördernde Mitglieder ohne therapeutischen Hintergrund dem VDT beitreten. Sie erhalten das VDT-Magazin "Mein Tierheilpraktiker" sowie das "Paracelsus" Magazin kostenfrei. Der Besuch auf den halbjährlichen Tierheilpraktiker-Kongressen und den Paracelsus Symposien in den Vorträgen ist für Sie gebührenfrei, die Workshops sind ermäßigt. In Höhe des Mitgliedsbeitrages wird Ihnen ein jährlicher Seminargutschein ausgestellt, der bei den Paracelsus
         <a href="https://www.paracelsus.de" target="_blank" title="Heilpraktikerschule">
          Heilpraktikerschulen
         </a>
         bei Buchung von Seminaren und Fortbildungen angerechnet werden kann.
         <br/>
         <br/>
         Stöbern Sie auf unserer Homepage, stellen Sie uns Ihre Fragen in unseren Chatforen, überzeugen Sie sich von unseren Leistungen und
         <a href="/mitgliederbereich/mitglied-werden.html" title="Tierheilpraktiker Verbandsmitglied werden">
          melden Sie sich als Mitglied in unserem Tierheilpraktikerverband an!
          <br/>
         </a>

     function addPlaceHolderText(fieldId, placeholderText) {
         fieldObj = jQuery("#"+fieldId);
         fieldObj.focus(function() {
             if (jQuery(this).val() == placeholderText)

         fieldObj.blur(function() {
             if (jQuery(this).val() == "")

     function preg_quote( str ) {

         // +   original by: booeyOH

         // +   improved by: Kevin van Zonneveld (http://kevin.vanzonneveld.net)

         // *     returns 1: '\$40'

     }
