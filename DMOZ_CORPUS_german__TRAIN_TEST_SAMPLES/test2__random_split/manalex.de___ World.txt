       <h1>
        Wirtschaftslexikon
       </h1>
       <p>
        Dieses Online-Wirtschaftslexikon eignet sich sowohl für Laien als auch für den akademischen Bereich. Finden Sie jeden Fachausdruck zunächst knapp dargestellt und im Verlauf des Artikels immer ausführlicher. Es sind alle großen Bereiche der Wirtschaftslebens und der Wirtschaftswissenschaften abgedeckt. Aktuell veröffentlichte Artikel finden Sie unten aufgeführt.
       </p>

        </b>
       </p>
       <p>
        Privatdiskonten -  sind -
        <a href="./d/wechsel/wechsel.htm">
         Wechsel
        </a>
        , die im Rahmen von
        <a href="./d/akzeptkredit/akzeptkredit.htm">
         Akzeptkredit
        </a>
        en (
        <a href="./d/akzept/akzept.htm">
         Akzept
        </a>
        ) entstehen. Die zur
        <a href="./d/finanzierung/finanzierung.htm">
         Finanzierung
        </a>
        eines Außenhandelsgeschäftes von einer erstklassigen &gt;&gt;&gt;
        <a href="http://www.wirtschaftslexikon.co/d/privatdiskonten/privatdiskonten.htm">
         Privatdiskonten
        </a>
       </p>
       <p>
        In-House Banking -  Zusätzliche übernahme von Finanzintermediärfunktionen durch das Finanzmanagement von
        <a href="./d/unternehmen/unternehmen.htm">
         Unternehmen
        </a>
        unter vollständiger oder teilweiser Umgehung der traditionellen  Bank . Ziel ist meist die Unternehmenswertsteigerung bzw. die Nutzung der
        <a href="./d/wertschoepfung/wertschoepfung.htm">
         Wertschöpfung
        </a>
        bei Finanzgeschäften, die bisher
        <a href="./d/bei-banken/bei-banken.htm">
         bei Banken
        </a>
        entstanden ist. Voraussetzung für die übernahme von
        <a href="./d/bankgeschaefte/bankgeschaefte.htm">
         Bankgeschäfte
        </a>
        n im eigenen
        <a href="./d/unternehmen/unternehmen.htm">
         Unternehmen
        </a>
        ist eine ausreichende Unternehmensgröße, technisches und personelles Know-how, sowie ein gutes Standing auf dem  Finanzmarkt  (
        <a href="./d/erste-adressen/erste-adressen.htm">
         erste Adressen
        </a>
        , wie z.B. „DaimlerChrysler, Siemens“ oder „Bosch“). Die Tendenz zur Verbriefung von Finanztiteln und die
        <a href="./d/verschiebung/verschiebung.htm">
         Verschiebung
        </a>
        der Finanzierungsmotive (Handel statt
        <a href="./d/spekulation/spekulation.htm">
         Spekulation
        </a>
        ) sowie das fehlende
        <a href="./d/angebot/angebot.htm">
         Angebot
        </a>
        traditioneller Banken für spezielle
        <a href="./d/produkt/produkt.htm">
         Produkt
        </a>
        e ihrer Kunden (
        <a href="./d/flexibilisierung/flexibilisierung.htm">
         Flexibilisierung
        </a>
        , Spezialisierung) führt in den letzten Jahren zu immer mehr
        <a href="./d/gruendung/gruendung.htm">
         Gründung
        </a>
        en von In-house-Banken. Fast jedes Automobilunternehmen und jedes Großunternehmen im Handel hat inzwischen seine eigene Bank, welche die Finanzmarktgeschäfte im
        <a href="./d/konzern/konzern.htm">
         Konzern
        </a>
        und an den Finanzmärkten betreibt. Hauptleistungen einer In-house-Bank sind, neben dem
        <a href="./d/cash-management/cash-management.htm">
         Cash Management
        </a>
        und
        <a href="./d/clearing/clearing.htm">
         Clearing
        </a>
        von Zahlungen im
        <a href="./d/konzern/konzern.htm">
         Konzern
        </a>
        (
        <a href="./d/matching/matching.htm">
         Matching
        </a>
        &gt;&gt;&gt;
        <a href="http://www.wirtschaftslexikon.co/d/in-house-banking/in-house-banking.htm">
         In-House Banking
        </a>
       </p>

 Die
        <a href="./d/arbeiterklasse/arbeiterklasse.htm">

        <a href="./d/ausbeutung/ausbeutung.htm">

        <a href="./d/industrie/industrie.htm">
         Industrie

 Arbeiter,
        <a href="./d/angestellte/angestellte.htm">
         Angestellte
        </a>

        <a href="./d/intelligenz/intelligenz.htm">
         Intelligenz
        </a>

        <a href="./d/landwirtschaft/landwirtschaft.htm">
         Landwirtschaft
        </a>

        <a href="./d/wachstum/wachstum.htm">
