         <h1 class="content">
          Willkommen bei purc.de
         </h1>
         <div class="d-con-box">
          <figure class="home-jigsaw" style="float: right; margin-left: 5px;">
           <img alt="purc.de puzzle" aria-describedby="home-jigsaw-desc" height="96" src="upload/purc.de_jigsaw.jpg" style="border: #eee 1px solid;" width="128"/>
           <figcaption class="hide-me" id="home-jigsaw-desc">
            purc.de Puzzle-Bild (
            <a href="upload/longdesc/homejigsaw.htm" title="Zur Langbeschreibung">
             Langbeschreibung des Puzzle-Bildes
            </a>
            ).
           </figcaption>
          </figure>
          Ich freue mich, Sie auf meiner Website begrüßen zu dürfen. Benötigen Sie Hilfe bei einem konkreten Projekt, oder möchten 
 	Sie sich erstmal informieren, ob und wie ich Ihnen IT-Lösungen bieten kann?
          <br/>
          <br/>
          Dann
          <a accesskey="k" href="kontakt.c-52.html" title="zum Kontaktformular">
           kontaktieren
          </a>
          Sie mich - Ich stehe Ihnen gerne persönlich mit Rat und Tat zur Seite.
          <br/>
          <br class="clear"/>
          <div class="hr">
           <hr/>
          </div>
          <h2>
           Was ist Purc?
          </h2>
          Purc, das ist mein Name und steht auch zugleich für Folgendes:
          <br/>
          <p class="nopm" style="text-align: center;">
           <span class="hdr2">
            P
           </span>
           rogrammierung +
           <span class="hdr2">
            U
           </span>
           msetzung +
           <span class="hdr2">
            R
           </span>
           ealisation +
           <span class="hdr2">
            C
           </span>
           odierung =
           <span class="hdr2">
            PURC
           </span>
          </p>
          <br/>
          Also dreht sich bei purc.de alles um Programmierung - Von Analyse, Design (Softwarearchitektur) bis hin zur Implementierung.
          <div class="hr">
           <hr/>
          </div>
          <h2>
           Service
          </h2>
          <strong>
           Folgenden Service bietet Purc:
          </strong>
          <br/>
          <ul style="margin-top: 5px;">
           <li>
            Erstellung von Internetpräsenzen, Shopsystemen und
            <a href="http://de.wikipedia.org/wiki/Rich_Internet_Application" rel="external" title="Definition von Rich Internet Application bei wikipedia.org (öffnet in einem neuen Fenster)">
             Rich Internet Applications
            </a>
            ,
             mit oder ohne
            <a href="http://de.wikipedia.org/wiki/Content-Management-System" rel="external" title="Definition von Content Management System bei wikipedia.org (öffnet in einem neuen Fenster)">
             Content Management System
            </a>
            .
           </li>
           <li>
            Aktualisierung/Erweiterung von Internet/-Intranetseiten
           </li>
           <li>
            Programmierung von Individualsoftware
           </li>
           <li>
            Design und Erstellung von Datenbanken
           </li>
           <li>
            Professionelle Unterstützung bei IT-Projekten
           </li>
          </ul>
