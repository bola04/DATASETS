      <p class="title">
       生涯スポーツから競技スポーツまで
      </p>
      <p align="left">
       Pikaraスタジアムは、県民のスポーツ活動に対する関心と意欲を一層高め、県民の健康と体力づくりを推進するために、本県のスポーツ施設の中核として平成9年にオープンしました。
       <br/>
       県下唯一の第1種公認陸上競技場  
   であり、陸上競技をはじめ、サッカー・ラグビー等の全国大会や国際大会等の開催が可能な施設設備となっています。
       <br/>
       生涯スポーツから競技スポーツまで、どなたにでも利用できます。
       <br/>
       <a href="info/shibafu.html">
        <br/>
        <em>
         <strong>
          補助競技場芝生張替工事　着工状況
         </strong>
        </em>
       </a>
       <em>
        <strong>
        </strong>
       </em>
       <br/>
       香川県では、スポーツ振興くじ（toto）助成金を活用し、香川県立丸亀競技場（Pikaraスタジアム）のインフィールド内の芝生の張替事業を行っています。　
 
 （スポーツ振興くじ（toto）助成：totoの販売によって得られる資金をもとに、地方公共団体及びスポーツ団体が行うスポーツの振興を目的とする事業に対して助成が行われるものです。）
       <br/>
       <br/>
       <em>
        <strong>
         <a href="info/touteki.html">
          主競技場投てき用囲い設置事業計画
         </a>
        </strong>
       </em>
       <br/>
      </p>
