         <p>
          <strong>
           ご挨拶
          </strong>
         </p>
         <p>
          会長に就任しました武田智子です。
          <br/>
          <img alt="tomoko2018s" class="alignright size-full wp-image-1730" height="269" src="http://www.midwife-chiba.org/wp-content/uploads/tomoko2018s.jpg" width="180"/>
          <br/>
          この度、平成30年度一般社団法人千葉県助産師通常総会において会長に就任いたしました。有床助産所を八千代市で開業しております。前会長より受け継ぎ理事ともども「志」を継続いたしますのでよろしくお願いいたします。
          <br/>
          皆様もご存知の通り本県でも少子化が急速に進行するなか、安心して子育てできる環境づくりの整備が求められています。35歳以上の出産の割合が、平成18年では19.1％であったものが平成28年には29.9％と上昇しており、リスクを伴う出産が増加していることがわかります。また、周産期の医療従事者は全国平均と比べ下回っているため、それぞれの役割に応じた対応と連携のシステム向上が必要です。このような現状から助産師会は、円滑な地域連携システムの強化を図り、妊産褥婦、新生児が安全に安心して過ごすことができる社会環境を作る支援をする努力をしていきたいと考えております。また、分娩を扱う開業助産師の立場として、開業を目指す地域助産師を増やしたい思いがあります。現在勤務助産師として働いている皆様も開業するにはハードルが高いと捉えがちであるのではないでしょうか。「お産に携わっていたい、助産師として専門性をもっと生かすために開業したい」と行動する若手が増えるよう能力の底上げに協力できたらと思います。また、産前産後ケアの重要性が世の中に浸透し、そのケアが家族システム全体のサポートになり、虐待防止、産後うつ予防にもつなげられたらと思います。
          <br/>
          さて、平成30年石川県金沢市にて開催された日本助産師会総会では①的確なリスク判別と安全なケアシステムの確保、②妊娠から産後までの切れ目ない支援、③女性の生涯にわたる健康教育の強化/妊娠時期にかかわるケアの評価の3点を計画として提示されました。本部計画を踏まえながら当会も独自の事業計画を進めていき、千葉県内で実動する皆様が助産師力を発揮できるよう組織体制強化をさらに目指していきます。
          <br/>
          助産師会における役割は多岐に渡っております。それぞれ皆様が得意の分野で活躍できるようにバックアップしていく所存です。皆様と一緒に千葉県助産師会をさらに構築していきたいと思っておりますが、そのためにはご意見もお願いします。
          <br/>
          今年は、千葉県助産師会は創立90周年記念に加えて南北関東地区研修会を成田市で開催いたします。参加した皆様に「勉強になった。楽しかった。」と感想をもって帰ることができる研修会にしたいと思います。「助産師魂」を持っている会員皆様のご協力なくしては成り立ちません　よろしくご協力ほどお願いいたします。
          <br/>
          （社）千葉県助産師会　　　会長　武田智子
         </p>
         <p>
          <strong>
           「賛助会員入会のご案内」
          </strong>
         </p>
         <p>
          千葉県助産師会は、助産師免許を有する者で構成する専門職の職能団体です。
          <br/>
          妊娠、出産、子育て中はもちろん、女性のライフステージに合わせた健康支援を行っています。
          <br/>
          また、公益事業として
          <br/>
          ・助産師による無料電話相談
          <br/>
          ・助産師を広く知っていただくことを目的にした「国際助産師の日」のイベント
          <br/>
          ・命の教育、性の健康教育、など学校等での講演活動
          <br/>
          を行っております。
          <br/>
          是非、賛助会員にご入会いただき、私たちの活動をご支援いただきますようお願い申し上げます。
         </p>
         <p>
          ＊賛助会員入会について
          <br/>
          本会の事業に賛同した個人・団体・企業の方は入会できます。（助産師資格保有者以外）
          <br/>
          ＊会員種類と会費（入会金は無料）
          <br/>
          個人　一口　　10,000円/年
          <br/>
          団体・企業　一口　　20,000円/年
          <br/>
          ＊賛助会員の特典
          <br/>
          ・団体・企業様はご希望に応じて本会ホームページにお名前を掲載いたします。
          <br/>
          ・機関紙「大潮」（年３回発行）をお届けいたします。
          <br/>
          ・研修会を会員価格で受けられます。
         </p>
         <p>
          ＊賛助会員入会方法
         </p>
         <p>
          <img alt="annai" class="alignnone size-full wp-image-1100" height="123" src="http://www.midwife-chiba.org/wp-content/uploads/annai.png" width="191"/>
          <br/>
          本会で申し込みを確認後、会費入金のご案内をさせて頂きます。
         </p>
