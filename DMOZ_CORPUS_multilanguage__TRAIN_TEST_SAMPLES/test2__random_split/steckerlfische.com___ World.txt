       <h1>
        Was sind Steckerlfische
       </h1>
       <div class="content">
        Steckerlfisch ist an einem Stock gegrillter Fisch, eine Spezialität aus dem bayerischen Alpenvorland und Oberösterreich, die vor allem in Biergärten und auf Volksfesten serviert wird. Er hat nichts mit dem getrockneten Stockfisch zu tun.
        <br/>
        <br/>
        Traditionell wird Steckerlfisch aus in den örtlichen Seen und Flüssen gefangenen Renken oder Weißfischen wie Brachsen zubereitet, heute auch aus Zuchtfischen wie Forellen und Saiblingen sowie Makrelen und anderen Seefischen.
        <br/>
        <br/>
        Zur Vorbereitung werden die ganzen, ausgenommenen Fische mit einer kräftigen Marinade aus Öl und Gewürzen eingestrichen und der Länge nach vom Maul her auf etwa 60 cm lange, entrindete Weidenzweige, gewässerte Holzstäbe oder Metallspieße gespießt. Zum Grillen werden die Stöcke  so festgemacht, dass sich die Fische kopfunter schräg über oder neben der Holzkohleglut befinden. Während der Garzeit sollten die Fische gelegentlich mit der Marinade oder Butter bepinselt werden, damit ihre Haut knusprig wird.
        <br/>
        <br/>
        Steckerlfisch wird auf dem Papier gegessen, in das er nach dem Grillen eingewickelt wurde, oder auf einem Teller serviert. Als Beilage isst man zum Beispiel Brezen oder Semmeln.
        <br/>
        <br/>
        <hr/>
        <br/>
        Steckerlfischrezept vom Münchner Oktoberfest:
        <br/>
        <br/>
        Forellen oder Makrelen
        <br/>
        Ausreichend für ca. 10 Fische
        <br/>
        Herstellung einer Gewürzmarinade:
        <br/>
        <br/>
        - 250-500 ml Öl
        <br/>
        - Einige Zweige Rosmarin
        <br/>
        - Einige Zweige Thymian
        <br/>
        - Einige Zweige Salbei
        <br/>
        - 1 Bund Petersilie
        <br/>
        - 1 Bund Dill
        <br/>
        - Estragon
        <br/>
        - Oregano
        <br/>
        - Minze
        <br/>
        - Basilikum
        <br/>
        - Fenchel
        <br/>
        - 1 Chili, getrocknet, entkernt
        <br/>
        - 1/2 Bund Frühlingszwiebeln, in grobe Ringe geschnitten
        <br/>
        - Knoblauchzehen, zerdrückt
        <br/>
        - Salz
        <br/>
        - Zitronensaft
        <br/>
        <br/>
        <br/>
        <a class="rezept" href="marinade.pdf" target="_self" title="download Rezept für Marinde">
         Rezept herunterladen
        </a>
        <br/>
        <br/>
        <br/>
        <a class="adobe" href="http://get.adobe.com/de/reader" target="_blank">

         <br/>

         <br/>
        </a>
        Fertig marinierte rohe Steckerlfische in eine lange flache Schüssel legen. Schüssel mit Klarsichtfolie oder einem passenden Deckel verschließen.
        <br/>
        Danach ruhen die Steckerlfische bei 4-6°C im Kühlschrank für mind 12 Stunden. Bevor der Fisch nach Bedarf auf den Grill kommt, aus dem Kühlschrank nehmen und 1 Stunde Zimmertemperatur annehmen lassen.
