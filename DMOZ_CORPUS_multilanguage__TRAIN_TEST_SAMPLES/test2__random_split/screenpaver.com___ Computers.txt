        <h1>
         Features
        </h1>
        <p>
         <strong>
          Screen Paver
         </strong>
         is an easy to use photo slideshow screen saver that uses your own digital picture collections.
 								The following features are available:
        </p>
        <ul>
         <li>
          Windows 7, Windows 8, and Windows 10 are supported.
         </li>
         <li>
          Support for JPEG, PNG, and bitmap picture formats (as well as shortcuts to such images).
         </li>
         <li>
          Use an unlimited number of pictures from local and/or network directories.
         </li>
         <li>
          Adjustable picture display time.
         </li>
         <li>
          Display pictures in alphabetical or random order, or by file date.
         </li>
         <li>
          Accounts for digital camera Exif orientation.
         </li>
         <li>
          Remember last picture in alphabetical or by-date mode.
         </li>
         <li>
          35 user-selectable transition effects.
         </li>
         <li>
          Adjustable transition speed.
         </li>
         <li>
          Choose from a variety of different options for stretching/shrinking/tiling the pictures.
         </li>
         <li>
          Preview pictures as thumbnails during directory selection.
         </li>
         <li>
          Enable/Disable different picture sets with a mouse click.
         </li>
         <li>
          Favorite pictures can be saved (as shortcuts) on the fly into a favorites directory.
         </li>
         <li>
          Automatically select all subdirectories.
         </li>
         <li>
          Long filename captions with optional date can be displayed with the pictures.
         </li>
         <li>
          Wallpaper can be easily created while previewing thumbnails.
         </li>
         <li>
          Automatic wallpaper changer (last picture displayed while running can be wallpapered).
         </li>
         <li>
          Screen saver can be paused and continued during normal operation.
         </li>
         <li>
          Screen saver can operate in manual slideshow mode.
         </li>
         <li>
          pictures can be advanced by hand in either normal or manual modes.
         </li>
         <li>
          A user message can be placed over the pictures.
         </li>
         <li>
          Select CDs or audio files to play during the slide show.
         </li>
         <li>
          Multiple monitors are supported.
         </li>
         <li>
          Selectable background color for pictures that do not fill the screen.
         </li>

          Password protection is supported.
