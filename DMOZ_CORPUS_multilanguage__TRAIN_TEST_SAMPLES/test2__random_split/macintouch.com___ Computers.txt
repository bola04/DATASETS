    <h4 class="section" id="products.2018.07.30">
     Products/Security
    </h4>
    <div id="prod.2018.07.30.cirrus">
     <p>
      <a href="https://eclecticlight.co/2018/05/19/cirrus-version-1-0-release-the-icloud-tool-is-ready-for-general-use/">
       Cirrus
      </a>
      is a Mac utility from Howard Oakley that offers more control over iCloud Drives than Apple does:
     </p>
     <p class="q2">
      • Upload a small test file to kickstart sync.
      <br/>
      • Browse all the folders held in iCloud Drive, not just those the Finder thinks you should know about.
      <br/>
      • Evict unwanted locally-stored files. If one or more of your Macs has Documents &amp; Desktop Folders enabled, do you really want all those files copied locally to your other Macs? The Finder provides no way to evict them back into the cloud, only Cirrus does.
      <br/>
      • Diagnose problems with iCloud syncing using its special colour-coded log browser.
      <br/>
      • Assess iCloud status of files without downloading them automatically.
      <br/>
      • Obtain a detailed listing of any folder stored on iCloud (excluding databases).
      <br/>
     </p>
     <p>
      <a href="https://eclecticlight.co/2018/05/19/cirrus-version-1-0-release-the-icloud-tool-is-ready-for-general-use/">
       Cirrus
      </a>
      1.1  
 is a free
      <a href="https://eclecticlight.co/downloads/" target="_blank">
       download
      </a>
      for OS X 10.11 and up.
      <br/>
      <a href="https://eclecticlight.co/2018/06/12/bailiff-1-0-now-available-take-control-of-icloud-drive/" target="_blank">
       Bailiff
      </a>
      1.1,  
 a Mac menubar app for evicting iCloud files and folders from local storage and retrieving them from iCloud, 
 is a free
      <a href="https://eclecticlight.co/downloads/" target="_blank">
       download
      </a>
      for OS X 10.11 and up.
     </p>
    </div>
    <div id="prod.2018.07.30.angelshark">
     <p>
      <a href="http://amfeltec.com/products/mac-pro-late-2013-carrier-board-for-m-2-pcie-ssd-modules/" target="_blank">
       AngelShark
      </a>
      is an internal adapter kit from Amfeltec Corp. for Apple's 2013 Mac Pro models, designed to accomodate one or two standard M.2 PCIe SSD modules in addition to the original Apple SSD. 
 Features include support for standard M.2 PCIe modules (M-key 2260, 2280, and 22110), 
 "easy plug-and-play installation", heat sinks, and LED status lights. 
 (See product manual (
      <a href="http://amfeltec.com/pdf/AngelShark_SKU-088-01_hwmanual_v1.0.pdf" target="_blank">
       PDF
      </a>
      ) for details.)
      <br/>
      <a href="http://amfeltec.com/products/mac-pro-late-2013-carrier-board-for-m-2-pcie-ssd-modules/" target="_blank">
       AngelShark
      </a>
      price and availability information should be available from the company. 
 (M.2 SSDs, such as the
      <a href="http://www.amazon.com/dp/B07DFJ3YQR/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Samsung 970 PRO NVMe
      </a>
      , 
 are readily available from Amazon and other vendors.)
     </p>
    </div>
    <div id="prod.2018.07.30.optane">
     <p>
      <a href="https://www.intel.com/content/www/us/en/products/memory-storage/solid-state-drives/gaming-enthusiast-ssds/optane-905p-series.html" target="_blank">
       Optane SSD 905P
      </a>
      is an extremely fast storage device from Intel Corp. in the form of a PCIe NVMe card that uses radically new storage technology (
      <a href="https://en.wikipedia.org/wiki/Optane" target="_blank">
       3D XPoint
      </a>
      ), which offers much higher performance and durability vs. typical NAND flash/SSD, operating in a completely different way. 
 While sustained sequential read/write performance is "only" up to 2600/2200 MB/s, random 4KB read/write is an amazing 575,000/550,000 IOPs, and read/write latency is under 10 μs, with tremendous durability of 10 Drive Writes per Day. 
 The PCIe 3.0 x4 card is half height, half length in size and features extra LED lighting effects.
      <br/>
      <a href="http://www.amazon.com/dp/B07CVNS851/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Optane SSD 905P
      </a>
      is sold through third-parties, including Amazon ($1299 for 960GB when checked) 
 for macOS 10.13 and up and systems equipped with PCIe slots (e.g. Thunderbolt 3 expansion boxes, such as Akitio's
      <a href="http://www.amazon.com/dp/B074HGR7QN/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Node Lite
      </a>
      ,
      <a href="http://www.amazon.com/dp/B06XKKSNTS/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Node
      </a>
      and
      <a href="http://www.amazon.com/dp/B077YCR5LC/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Node Pro
      </a>
      ).
     </p>
     <p class="q2">
      "Intel Optane technology is an entirely new technology which, while not as fast as DRAM, is fast enough to act as an extended memory pool and more responsive than even the highest performing NAND SSDs. This means that, for many applications, Intel Optane technology can replicate in-memory performance and therefore provide a much better high-performance solution for warm or hot data than disk, and a much cheaper solution than DRAM. Effectively, Intel Optane technology combines the attributes of memory and storage with low latency, high endurance, outstanding quality of service and high throughput."
     </p>
     <p>
      <a href="https://www.intel.com/content/www/us/en/products/memory-storage/solid-state-drives/gaming-enthusiast-ssds/750-series/750-1-2tb-aic-20nm.html" target="_blank">
       Intel SSD 750
      </a>
      is a very-high-performance PCIe storage card based on NAND technology, rather than Optane, but still offering sequential read/write up to 2400/1200 MB/s, random read/write up to 440000/290000 IOPS, 20 µs latency, and 70GB per day endurance. 
 It's sold via third-parties, including
      <a href="http://www.amazon.com/dp/B013QP8XUE/ref=nosim/macintouchcom-20?&amp;tag=macintouchcom-20" target="_blank">
       Amazon
      </a>
      ($539.99 for 800GB when checked).
     </p>
    </div>
    <div id="prod.sec.2018.07.30.keka">
     <p>
      <a href="http://www.kekaosx.com/en/" target="_blank">
       Keka
      </a>
      is a Mac app from Jorge Garcia Armero for creating and extracting archive files (including password-protected archives) via drag-and-drop, including compressing into 7z, Zip, Tar, Gzip, Bzip2, DMG and ISO, and extracting from RAR, 7z, Lzma, xz, Zip, Tar, Gzip, Bzip2, ISO, EXE, CAB, PAX and other formats. 
 Features include
      <a href="https://github.com/aonez/Keka/wiki/Compressing-with-Keka#password-protected-and-encrypted-files" target="_blank">
       encryption
      </a>
      via 7z, which can be helpful for securely sharing files with Windows systems.
 (See
      <a href="https://github.com/aonez/Keka/wiki" target="_blank">
       Keka Wiki
      </a>
      for more details.)
      <br/>
      <a href="http://www.kekaosx.com/en/" target="_blank">
       Keka
      </a>
      1.1.2  
 is a free download for OS X 10.9 and later (or
      <a href="https://itunes.apple.com/us/app/id470158793?mt=12" target="_blank">
       $1.99
      </a>
      at the Mac App Store). Legacy versions support Macs back to Mac OS X 10.4. 
 The
      <a href="https://changelog.keka.io/" target="_blank">
       latest
      </a>
      release includes fixes for a macOS 10.14 issue and a few other glitches.
     </p>
    </div>
    <div id="sec.prod.2018.07.30.buttercup">
     <p>
      <a href="https://buttercup.pw/" target="_blank">
       Buttercup
      </a>
      is a cross-platform, open-source password manager from Perry Mitchell and Sallar Kaboli,
      <i>
       "written in JavaScript for NodeJS and the browser."
      </i>
      <br/>
      Features include a simple user interface, 
 grouping of entries, 
 custom fields, 
 a password generator (and strength indicator), auto-lock (after 10 minutes), 
 support for multiple archives, 
 fingerprint/FaceID unlock, 
 import from 1Password, KeePass and LastPass, 
 and support for Dropbox, OwnCloud, Nextcloud and WebDAV cloud services.
      <br/>
      <a href="https://buttercup.pw/" target="_blank">
       Buttercup
      </a>
      1.9.0 
 is free,
      <a href="https://github.com/buttercup" target="_blank">
       open-source
      </a>
      software for 
 OS X 10.9 and up, Linux and Windows.
      <br/>
      <a href="https://itunes.apple.com/us/app/buttercup-password-manager/id1294001514?ls=0&amp;mt=8" target="_blank">
       Buttercup
      </a>
      1.4.0 
 is a free download for iOS 10 and up (also available for Android), 
 and there are browser extensions/add-ons for
      <a href="https://chrome.google.com/webstore/detail/buttercup/heflipieckodmcppbnembejjmabajjjj?hl=en" target="_blank">
       Chrome
      </a>
      and
      <a href="https://addons.mozilla.org/en-US/firefox/addon/buttercup-pw/" target="_blank">
       Firefox
      </a>
      browsers, as well.
     </p>
    </div>
    <div id="sec.prod.2018.07.30.chromium">
     <p>

      </a>
      is an open-source web browser project on which Google Chrome is built, but it doesn't include Google's proprietary

      and omits the Adobe/Pepper Flash plug-in, and Flash's dangerous security flaws. 
 The
      <a href="http://www.freesmug.org/chromium#toc0" target="_blank">
       latest version
      </a>
      includes high-priority
      <a href="https://chromereleases.googleblog.com/2018/07/stable-channel-update-for-desktop.html" target="_blank">
       security patches
      </a>
      and other security updates, 
 among other
      <a href="https://chromium.googlesource.com/chromium/src/+log/67.0.3396.99..68.0.3440.75?pretty=fuller&amp;n=10000" target="_blank">
       changes
      </a>
      (sans readable documentation).
      <br/>
      Chromium 68.0.3440.75  
 is a free download for OS X 10.10 and up, available
      <a href="http://sourceforge.net/projects/osxportableapps/files/Chromium/" target="_blank">
       via SourceForge
      </a>
      and
      <a href="http://www.freesmug.org/chromium">
       via SMUG

    <h4 class="section" id="apple.2018.07.30">
     Apple
    </h4>
    <div id="apple.2018.07.30.configurator">
     <p>
      <a href="https://itunes.apple.com/us/app/apple-configurator-2/id1037126344?mt=12&amp;ls=0" target="_blank">
       Apple Configurator
      </a>
      is Apple's utility for configuring and deploying iOS devices en masse via USB
      <i style="color:gray;">
       "with the settings, apps, and data you specify for your students, employees, or customers."
      </i>
      <br/>
      Features include integration with Apple's Device Enrollment Program and Volume Purchase Program; a configuration profile editor and configuration "blueprints" (custom templates); iCloud Drive sync; AppleScript and Automator support and a command-line program. 
 Some people are experimenting with the app as an alternative to Apple's iTunes app, after iTunes 12.7 abandoned support  for app management.
      <br/>
      <a href="https://itunes.apple.com/us/app/apple-configurator-2/id1037126344?mt=12&amp;ls=0" target="_blank">
       Apple Configurator
      </a>
      2.7.1   
 is a free download for macOS 10.12 and up. 
 (See also
      <a href="http://imazing.com" target="_blank">
       iMazing
      </a>
      , a third-party alternative for some tasks.)
      <br/>
      The latest Configurator release brings the following changes:
     </p>
     <p class="q2" style="color:gray;">
      • Preparing supervised devices for management by Configurator now automatically allows USB accessory connections while device is locked.
      <br/>
      • Configure new supervised-only restriction for iOS 11.4.1: Allow USB accessories while device is locked.
     </p>
