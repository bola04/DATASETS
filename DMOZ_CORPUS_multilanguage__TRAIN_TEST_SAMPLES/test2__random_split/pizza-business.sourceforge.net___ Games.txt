     <div class="newsHeader">
      The forums are now powered by phpBB
     </div>
     <div class="newsDate">
      -- Posted by
      <a href="mailto:ares32585@hotmail.com">
       ares32585
      </a>
      on Sunday, May 11 2003
     </div>
     <p>
      We have replaced the forums powered by yaBB with forums powered by phpBB (www.phpBB.com).  There are two main reasons for this switch:  phpBB provides us with more flexibility, more options, more customizability and more cool features.  Also, phpBB, since it uses a database to store information, is faster, and can scale better with more users.  So please register and start some discussions in the
      <a href="http://pizza-business.sourceforge.net/phpBB2/">
       new forums!
      </a>
     </p>

     <div class="newsHeader">
      Pizza Business 1.0-beta8
     </div>
     <div class="newsDate">
      -- Posted by
      <a href="mailto:ares32585@hotmail.com">
       ares32585
      </a>
      on Thursday, March 13 2003
     </div>
     <p>
      Pizza Business 1.0-beta 8 has been released.  New in this version:
     </p>
     <br/>
     <ul>
      <li>
       <p>
        Updated prices and customer simulation algorithms.
       </p>
      </li>
      <li>
       <p>
        Auto-pricing is no longer available because it is no longer useful with the changed in the simulation.
       </p>
      </li>
      <li>
       <p>
        There has also been some cleanup of code hacks in various parts of the code.
       </p>
      </li>
      <li>
       <p>
        Addition, removal and editing of your own pizza recipes has been implemented
       </p>
      </li>
      <li>
       <p>
        New in-game background picture added
       </p>
      </li>
      <li>
       <p>
        crc.txt was added to let people check for possible corruption or changes in source files using md5sum
       </p>
      </li>
      <br/>
     </ul>
     <p>
      Please give us some feedback in our
      <a href="http://pizza-business.sourceforge.net/cgi-bin/yabb/YaBB.cgi">
       new, easier-to-use, and nicer-looking forums!
      </a>
     </p>
     <p>
      <b>
       Update:  The Windows binary is now available.
      </b>
     </p>

     <div class="newsHeader">
      Pizza Business 1.0-beta7 Released
     </div>
     <div class="newsDate">
      -- Posted by
      <a href="mailto:ares32585@hotmail.com">
       ares32585
      </a>
      on Friday, February 7 2003
     </div>
     <p>
      New in this version are: a pie chart and line graph added to the stats dialog;  fairly comprehensive saving/loading game functionality added; the source code is now compatible with wxWindows 2.4.  Also, a statically linked linux exectuable is available for this release, rather than the dynamically linked one available for previous releases.  This means you don't have to have any libraries installed to run the game on Linux.  Make sure to let us know what you think.
     </p>
