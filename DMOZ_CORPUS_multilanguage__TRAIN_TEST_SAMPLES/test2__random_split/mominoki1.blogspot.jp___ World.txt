               ぼかし作りを始めた。6月15日
               <br/>
               夏なのでたぶん1週間かかるかな？
               <br/>
               とりあえず　毎日の切り返しを1週間頑張る。
               <br/>
               <br/>
               出来たぼかしは家庭菜園で追肥として使う。
               <br/>
               今　自然農法とか不耕起栽培とか無農薬栽培とか有機栽培とかいろんな野菜作りがあるけれど、
               <br/>
               <br/>
               化学肥料をつかうよりは、ぼかし肥料が良いかなと今は思っている。
               <br/>
               <br/>
               何がよくて　何がいけないのかよくわからないけれど、とりあえずやってみる。
               <br/>
               <br/>
