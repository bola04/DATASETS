        <h2>
         About Quintano
        </h2>
        <p>
         In 2004, Simonds Farsons Cisk plc acquired the assets of Law. Quintano &amp; Co. Ltd and the business started operating under the newly set up company Quintano Foods Ltd.
         <br/>
         The company has a strong brand portfolio that include, amongst others, Danone, Tropicana, Quaker, Walkers, Bayernland, Ribena, Ocean Spray, Calvè, Cooperlat, and Mevgal.
         <br/>
         Quintano Foods Ltd seeks to further consolidate its leading market position by identifying, anticipating and satisfying consumer demands through constant brand development whilst maintaining a flexible company structure to accommodate for change in a dynamic marketplace.
        </p>
       </section>

        function DoValidation(validationGroup) {
         var isValid = Page_ClientValidate(validationGroup);
         if (!isValid) {

             setTimeout(function () {
                 $('input[type="text"]').removeClass('error');
                 $('textarea').removeClass('error');
 
                 $('.error-message ul').hide();
                 var errMsg = '';
 
                 $('.error-message').children('ul').children('li').each(function () {
 
                     if ($(this).text().toLowerCase().indexOf("name") >= 0) {
                         $('#txtName').addClass('error');
                     }
 
                     if ($(this).text().toLowerCase().indexOf("e-mail") >= 0) {
                         $('#txtEmail').addClass('error');
                     }
 
                     if ($(this).text().toLowerCase().indexOf("message") >= 0) {
                         $('#txtComments').addClass('error');
                     }
 
                     if (errMsg.length > 0) {
                         errMsg = errMsg + ', ' + $(this).text();
                     } else {
                         errMsg = $(this).text();
                     }
 
                 });
 
                 errMsg = 'Please enter your ' + errMsg;
                 $('.error-message').text(errMsg);
 
             }, 200);
         }
