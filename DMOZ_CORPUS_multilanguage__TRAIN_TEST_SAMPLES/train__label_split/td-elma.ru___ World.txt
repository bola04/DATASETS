      <div class="col-text left">
       <h2>
        <a href="http://td-elma.ru/catalog/99/" target="_blank">
         Надёжные немецкие химические насосы
        </a>
       </h2>
       <p>
        ТД Элма предлагает широкий спектр химических насосов своим клиентам - это и вертикальные насосы, и центробежные насосы, и мембранные насосы – но основной специализацией компания являются бочковые насосы или насосы для бочек. Бочковой насос прост и удобен в эксплуатации, идеально подходят для перекачки как нейтральных, так и агрессивных жидкостей из бочек, контейнеров, ванн и других ёмкостей. Состоят бочковые насосы из погружной трубки, которая заканчивается лопастной системой, ведущей к основному валу. Бочковые насосы снабжаются электродвигателем, защищенным от самостоятельных включений при перебоях питания. Существует модели с пневмоприводом.
       </p>
       <h2>
        <a href="http://td-elma.ru/catalog/92/" target="_blank">
         Насосы для гальванических производств
        </a>
       </h2>
       <p>
        В ассортимент компании Элма входят насосы без уплотнений, роторные насосы, насосы с регулятором производительности (регулировкой оборотов двигателя), а так же насосные комплектующие и аксессуары: незаменимые шланги для кислоты (рукава для кислот), надёжные и безопасные нагреватели для агрессивных жидкостей. Плоские Тефлоновые нагреватели могут быть установлены на дно или на боковую часть ванны и использоваться как нагреватели для гальваники. По своей надёжности, такие нагреватели оптимально подходят для гальванических производств (гальванических процессов). Компания Элма так же предлагает целый ряд химических насосов для агрессивных жидкостей (сред), отвечающих всем нуждам гальванических производств, насосов способных эффективно слить электролит, откачать ванну, добавить кислоту, опорожнить ёмкости с агрессивными жидкостями.
       </p>
      </div>
      <div class="col-text">
       <h2>
        <a href="http://td-elma.ru/catalog/96/" target="_blank">
         Насосы для агрессивных жидкостей
        </a>
       </h2>
       <p>
        Среди высококачественного и лёгкого в эксплуатации немецкого насосного оборудования, поставляемого компанией Элма, представлены как насосы для кислот (насосы для серной кислоты, фосфорной кислоты, азотной кислоты и соляной кислоты), насосы для перекачки электролита, насосы для щелочи, насосы для хрома, насосы для гипохлорита натрия, насосы для аммиака, так и насосы для вязких жидкостей (насосы для перекачки нефтепродуктов: дизельного топлива, нефти, масла, солярки, бензина и ацетона), насосы для спирта, насосы для фруктового сока и насосы для молока.
       </p>
       <p>
        Торговый дом «Элма» поможет подобрать и поставит в кратчайшие сроки - центробежные химические насосы, мембранные химические насосы (диафрагменные насосы) – самовсасывающие насосы, идеальные для перекачки тяжелых жидкостей с механическими включениями (с твёрдыми частицами); и вертикальные центробежные насосы – отличающиеся высокой производительностью и подходящие для перекачки промышленных объёмов агрессивных жидкостей.
       </p>
       <h2>
        <a href="http://td-elma.ru/catalog/105/" target="_blank">
         Бочковые насосы и пластиковые насосы для бочек
        </a>
       </h2>
       <p>
        Компания Элма поможет подобрать оптимальный пластиковый бочковый насос для Вашего производства, будь то насос из фторопласта или насос из полипропилена. Пластиковые химические насосы изготавливаются полностью из химостойких полимеров и потому идеально подходят как для перекачки нейтральных жидкостей, так для работы в агрессивных и даже коррозийных средах, а так же подачи реагентов из различных емкостей.
       </p>
