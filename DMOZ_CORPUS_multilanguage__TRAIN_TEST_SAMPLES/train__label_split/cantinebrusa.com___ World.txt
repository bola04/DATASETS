       <h1>
        A steady partner
       </h1>
       <div class="content">
        <p>
         Da piccola cantina imbottigliatrice di vino fondata nel lontano 1888, Cantine Brusa SpA si presenta oggi come un’ industria leader nella trasformazione di mosti d’uva e
         <strong>
          punto di riferimento
         </strong>
         per cantine vinicole ed aziende alimentari in Italia e nel mondo.
        </p>
        <p>
         La nostra gamma di prodotti offre succhi d’uva e mosti freschi, succhi e mosti concentrati, mosti concentrati per aceto balsamico di Modena IGP, succo concentrato deionizzato, mosto concentrato rettificato e vini. Tutto prodotto con perizia e professionalità, utilizzando mosti d’uva selezionati e provenienti dai migliori vigneti nazionali ed europei.
        </p>
        <p>
         I nostri principali clienti sono le
         <strong>
          cantine vinicole imbottigliatrici
         </strong>
         , i produttori di aceto balsamico e succhi di frutta, i distributori di ingredienti alimentari, soft drinks, flavours, aromi o confetture di frutta.
        </p>
        <p>
         Le soluzioni di packaging offerte da Cantine Brusa sono molteplici, così come le
         <strong>
          collaborazioni con importanti aziende di trasporto
         </strong>
         , tutto questo ci permette di fare fronte a richieste di clienti da ogni angolo del pianeta.
        </p>
        <p>
         Il nostro laboratorio di analisi interno garantisce un costante controllo della qualità dei prodotti nell’arco di tutta la stagione vinicola, nonché un supporto tecnico ed analitico ai clienti, mentre il nostro staff commerciale fornisce assistenza qualificata pre e post vendita, e
         <strong>
          facilita le esportazioni anche più complesse
         </strong>
         .
        </p>
        <p>
         La nostra forza è la capacità di far coesistere le principali qualità della grande industria (ovvero l’enorme capacità produttiva, l’impiantistica all’avanguardia ed una moderna logistica) con la flessibilità e il dialogo diretto verso i clienti e i produttori: caratteristiche tipiche dell’azienda familiare.
        </p>

    jQuery(function ($) {
         function setCookie(c_name, value, exdays) {
             var exdate = new Date();
             exdate.setDate(exdate.getDate() + exdays);
             var c_value = escape(value) + ((exdays == null) ? "" : "; expires=" + exdate.toUTCString()) + "; path=/";
             document.cookie = c_name + "=" + c_value;
         }
 
         function getCookie(cname) {
             var name = cname + "=";
             var ca = document.cookie.split(';');
             for (var i = 0; i < ca.length; i++) {
                 var c = ca[i].trim();
                 if (c.indexOf(name) == 0)
                     return c.substring(name.length, c.length);
             }
             return "";
         }
 
         if (getCookie("cookie-message-readed") != 1) {
             $("#cookie-wrapper").show();
         }
 
         $("#cookie-close").on("click", function () {
             setCookie("cookie-message-readed", 1, 365);
             $("#cookie-wrapper").hide();
             $('footer').css('margin-bottom', '0px');
         });
     });
