                        <div class="headline">
                         <h1 class="title">
                          <a href="/Level-2/PolyScript/polyscript.html">
                           PolyScript - Telnet and SSH Client
                          </a>
                         </h1>
                        </div>
                        <p class="articleinfo">
                        </p>
                        <h6>
                         <div>
                          <img alt="Windows10" height="130" src="/images/stories/probit/Windows10.png" style="margin-left: 10px; float: right;" width="100"/>
                         </div>
                        </h6>
                        <h6 style="text-align: justify;">
                         <strong>
                          <a name="Telnet-Client">
                          </a>
                          Polyscript
                         </strong>
                         is an advanced,  user friendly, secure flexible terminal client with Telnet, SSH, SSH2 connectivity in a new tabbed multi-session interface. It provides an  easy way to organize devices in a tree-like library, allowing you perform operations on a many devices simultanously.  Polyscript includes a revolutionary scripting engine with no need to  write code ! a simple drag-n-drop system allowing technicians,  administrators and developers to create automated operations in a  minute.
                        </h6>
                        <h6 style="text-align: justify;">
                         There are many features  and advanced options available  within Polyscript which are being added with the release of each new  version.
                        </h6>
                        <div style="text-align: justify;">
                        </div>
                        <div style="text-align: justify;">
                        </div>
                        <h3>
                         <a href="http://www.soft82.com/download/windows/polyscript/">
                          <img alt="soft82_award_88x88" height="88" src="/images/stories/soft82_award_88x88.gif" style="margin: 10px; float: right;" width="88"/>
                         </a>
                         <a name="SSH-Client">
                         </a>
                         Tabbed Connection
                        </h3>
                        <ul>
                         <li>
                          <h6>
                           Use one copy of Polyscript to open as many Telnet and SSH connections as you need.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Session windows can be arranged.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Run scripts simultaneously.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Type text simultaneously.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Switch easily between sessions with tabbed browsing and the CTRL-TAB hotkey.
                          </h6>
                         </li>
                        </ul>
                        <h3>
                         <a name="Telnet-Scripting">
                         </a>
                         Easy Organization
                        </h3>
                        <ul>
                         <li>
                          <h6>
                           Organise the hosts you use hierarchically.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Run scripts on a group of hosts
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Connect and login multiple devices with one click
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Manage favorite scripts for a group/host.
                          </h6>
                         </li>
                        </ul>
                        <h3>
                         Scripting
                        </h3>
                        <ul>
                         <li>
                          <h6>
                           No more programming knowledge !
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Simple drag-n-drop interface
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Customize the scripts with user input.
                          </h6>
                         </li>
                         <li>
                          <h6>
                           Run script on multiple hosts at the same time.
                          </h6>
                         </li>
                        </ul>
