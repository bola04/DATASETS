         <h2>

         </h2>
         <div style="border:1px solid gray; background:#fafafa; padding:5px;">
          DNA Dragon Contig Assembler assembles sequences (FASTA, GenBank etc.), trace data (ABI, SCF, AB1), and Illumina and Roche 454 flowgrams into contigs.
     It is a very fast and accurate DNA sequence assembly software for MS Windows (c) operating systems.
     The DNA sequences are assembled into contigs and a direct comparision of trace date with nucleotide data is possible. It also allows for proofreading and base editing.
          <ol>
           <li>
            <b>
             Fast and robust assembly machine
            </b>
            handles more than 100000 samples from sequence formats (FASTA, GenBank, EMBL, etc.), trace file formats (SCF, ABI, AB1) and next generation sequencing file formats (Illumina, SFF, GFF, ...).
           </li>
           <li>
            <b>
             Easy to use interface
            </b>
            which accelerates the proofreading and the comparison of nucleotides with trace peaks.
           </li>
           <li>
            <b>
             XML-conform project files
            </b>
            containing all necessary data.
