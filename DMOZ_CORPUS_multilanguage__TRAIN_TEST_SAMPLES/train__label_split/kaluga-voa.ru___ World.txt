      <h1>
       <span>
        ВОА
       </span>
      </h1>
      <!-- <div class="breadcrumb"><a href="/">Главная</a> &raquo; <span>ВОА</span></div> -->
      <div style="text-align: center;">
       <strong>
        <br/>
        <span style="color: rgb(0, 51, 0);">
         ВНИМАНИЕ!!! ВНИМАНИЕ!!! ВНИМАНИЕ!!!
         <br/>
         КАЖДУЮ ПЯТНИЦУ  НА УЧЕБНОЙ ПЛОЩАДКЕ НАШЕЙ АВТОШКОЛЫ ПРОВОДИТСЯ ПРИЕМ ЭКЗАМЕНОВ ПО ПРАКТИЧЕСКОМУ ВОЖДЕНИЮ.
         <br/>
         ЗАПИСАТЬСЯ НА СДАЧУ (ПЕРЕСДАЧУ) МОЖНО ПО ПОНЕДЕЛЬНИКАМ И ВТОРНИКАМ ПО ТЕЛЕФОНУ: 57-93-22
         <br/>
        </span>
        <span style="color: rgb(128, 0, 0);">
         <br/>
         СТОИМОСТЬ ОБУЧЕНИЯ
         <span style="font-size: x-large;">
          от
         </span>
         <span style="font-size: xx-large;">
          14 000
         </span>
         РУБЛЕЙ!!!
         <br/>
         УСПЕЙ ЗАПИСАТЬСЯ!!!
        </span>
        <br/>
        <br/>
        <br/>
        <br/>
        КАЛУЖСКОЕ ОБЛАСТНОЕ ОТДЕЛЕНИЕ ОБЩЕСТВЕННОЙ ОРГАНИЗАЦИИ "ВСЕРОССИЙСКОЕ ОБЩЕСТВО АВТОМОБИЛИСТОВ" (КОО ОО ВОА)
       </strong>
       <br/>
      </div>
      <div style="text-align: justify;">
       23 февраля 1973 г. Совет Министров РСФСР по инициатиые Министерства автомобильного транспорта РСФСР принял Постановление № 89 "Об организации Всероссийского добровольного общества автомотолюбителей" (ВДОАМ). Создание Общества было необходимо для решения вопросов, связанных с растущими темпами автомобилизации для организации хранения и безаварийной эксплуатации автотранспорта, принадлежащего индивидуальным владельцам,  защиту их прав и законных интересов. главной целью Общества стало повышение безопасного дорожного движения, профилактика аварийности и дорожно-транспортного травматизма.
       <br/>
       Решением Калужского облисполкома № 169 от 20.03.1973 г. было организовано Калужское областное отделение ВДОАМ, члены которого при государственной поддержке создали соответствующую учебно-производственнцю базу, на которой осуществлялся21 вид деятельности, в том числе:
      </div>
      <div style="text-align: left;">
       - подготовка водителей,
       <br/>
       - автоэкспертиза,
       <br/>
       - инструментальный контроль,
       <br/>
       - хранение автотранспорта,
       <br/>
       - страхование,
       <br/>
       - юридическая помощь,
       <br/>
       - спортивные туристические клубы,
       <br/>
       - юношеские автошколы
      </div>
      <div style="text-align: center;">
       За 41 год деятельности ВОА (ВДОАМ) было подготовлено более 50 тыс. водителей.
       <br/>
       На основании решения внеочередной конференции Калужского областного отделения ВОА от 10.09.2015 года Председателем КОО ОО ВОА является
       <strong>
        МИСИКОВ ВЛАДИМИР АЛЕКСАНДРОВИЧ
       </strong>
       .
       <br/>
      </div>
      <div style="text-align: justify;">
       В настоящее время КОО ОО ВОА проводит обучение курсантов по программам профессиональной подготовки водителей транспортных средств категории "В" и переподготовки водителей транспортных среств с категории "С" на категорию "В".
       <br/>
       <br/>
       Калужская автошкола ВОА успешно прошла проверку УМБ, в результате получено
       <strong>
        ЗАКЛЮЧЕНИЕ УГИБДД УМВД России по Калужской области № 6/90 от 18.05.2016 г.
       </strong>
       о соответсвии учебно-материальной базы установленным требованиям.
