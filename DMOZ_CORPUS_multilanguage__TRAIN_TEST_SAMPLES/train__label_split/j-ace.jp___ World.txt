      <p class="unit08 desc3">
       <span class="tx-danger-pw">
        緊急声明！
       </span>
       <br/>
       <br/>
       日本最長洞穴　安家洞（24,000ｍ）が開口する岩泉町安家で川が氾濫し大きな被害が発生しております。
       <br/>
       この地域は東日本大震災では直接的な被害を受けなかった地域ですが、この度の台風10号では家や道路が流され床上浸水等の被害が発生しました。
       <br/>
       <br/>
       このため当会ではボランティア隊を編成し現地に派遣することになりました。
       <br/>
       多くの皆様のご協力をお願い致します。
       <br/>
       <br/>
       第2次安家川流域救援ボランティア隊：9月16日～19日
       <br/>
       <br/>
       問合せ申込み先：
