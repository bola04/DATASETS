      <h1 align="center" style="font-family: serif">
       Xinc, an XSL-FO Processor
      </h1>
      <p>
       Xinc is an XSL-FO processor which was designed to be fast and memory efficient. A Java Swing viewer allows you to view and print XSL-FO files as well as generate PDF files with the click of a button. Xinc can be used as a server component via its Java API. Xinc can also be used in a Microsoft server environment by using its COM interface.
      </p>
      <p>
       <b>
        News: 06/05/2005 - Xinc 2.02 released
       </b>
      </p>
      <p>
       Xinc 2.02 fixes several problems with table-layout="auto" and adds support for Saxon 8.4 . For more details on these and
 other fixes, please see the Release notes.
      </p>
      <p>
       <b>
        News: 04/05/2005 - Xinc 2.0 released
       </b>
      </p>
      <p>
       Xinc 2.0 adds support for auto table layout. This feature causes Xinc to choose the optimum width for each column in a table based on
 it's content. Support for TIFF, PBM, PGM and PPM graphics formats have been added. Solaris and other Unix platforms are now supported.
 New Zip and Tar packages allow Xinc to be setup and used on any Java enabled platform. Documents may now be rendered directly to a
 printer using the java API or the command line tool.
 
 A variety of block-container and table bugs are fixed. For more details on these and
 other fixes, please see the Release notes.
      </p>
      <p>
       An OEM license is now available.
      </p>
      <p>
       <b>
        News: 07/23/2004 - Xinc 1.2 released

       Xinc 1.2 now includes a full featured COM API and new sample code. Please see the Release notes for
 more details.
      </p>
      <p>
       <b>
        News: 07/23/2004 - Xinc 1.1.2 released

 and the font-family property. Please see the Release notes for
 a complete list of bug fixes.
      </p>
      <p>
       <b>
        News: 06/29/2004 - Xinc 1.1 released
       </b>
      </p>
      <p>
       Lunasil Ltd. is pleased to announce the release of Xinc 1.1. This release features an improved high level Java API which supports the formatting of documents via SAX InputSource and Source classes. Also included is a new
 low-level Java API which can be used by applications which need access to a SAX ContentHandler. Sample code showing the use of the API's is included. Please see the Release notes for
 a complete list of features and bug fixes.
      </p>
      <p>
       <b>
        News: 05/20/2004 - Xinc 1.0 released
       </b>
      </p>
      <p>
       Lunasil Ltd. is pleased to announce the release of Xinc 1.0. Xinc is a fast, memory efficient XSL-FO processor written in Java. Xinc is quick to install and easy to use. 
 
 A Swing based XSL-FO viewer allows you to view and print XSL-FO files as well as generate PDF files with the click of a button.
 
 Xinc can be used as a server component via its Java API. Xinc can also be used in a Microsoft server environment by using a simple COM interface. A command line interface is included for use in batch processes and build environments.
      </p>
