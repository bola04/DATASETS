             <br/>
             <br/>
             Hele godset udgør ca. 3.700 ha, hvoraf landbruget udgør ca. 1850 ha og skovbruget ca. 1850 ha.
             <br/>
             <br/>
             <a href="./landbrug/">
              Landbruget
             </a>
             er fordelt på 7 hovedgårde hvor 5 ligger omkring Sakskøbing og 2 ved Nakskov.
             <br/>
             <br/>
             <a href="./skovbrug/">
              Skovdistriktet
             </a>
             er fordelt på 8 skove, som ligger omkring Sakskøbing. Ca. 1.500 ha. er bevokset.
             <br/>
             <br/>
             <a href="./jagt/">
              Jagten
             </a>
             er udlejet til 13 konsortier der alle er mere eller mindre lokalt tilknyttede.
             <br/>
             <br/>
             Godset administerer også ca. 70 lejeboliger og nogle erhvervslejemål.
             <br/>
             <br/>
             <a href="./bryggeri/">

             </a>
