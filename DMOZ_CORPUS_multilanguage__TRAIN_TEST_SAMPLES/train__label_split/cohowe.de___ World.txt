           <h1>
            Anspruchsvolles Webdesign in Saarbrücken
            <br/>
            für ihren Internetauftritt
           </h1>
           <h2 class="clear">
            Ein professioneller Internetauftritt ist ein wichtiges Instrument für ein erfolgreiches Unternehmen.
           </h2>
           <div class="buttonstart">
            <a href="http://www.cohowe.de/de/webdesign-service">
             <img alt="Internetabentur in Saarbrücken" src="/sites/default/files/service.png" style="width: 80px; height: 66px;"/>
             <br/>
             SERVICE
             <br/>
             <span>
              Ihr Online Auftritt steht im Mittelpunkt!
             </span>
            </a>
           </div>
           <div class="buttonstart">
            <a href="http://www.cohowe.de/de/referenzen/cms">
             <img alt="Website in Saarbrücken - Saarland - Pfalz" src="/sites/default/files/referenzen-icon.png" style="width: 80px; height: 66px;"/>
             <br/>
             REFERENZEN
             <br/>
             <span>
              In guter Gesellschaft!
             </span>
            </a>
           </div>
           <div class="buttonstart">
            <a href="/node/80">
             <img alt="Responsive Webdesin" src="/sites/default/files/responsive-web-mobil.png" style="width: 80px; height: 66px;"/>
             <br/>
             WEB MOBIL
             <br/>
             <span>
              Ist Ihre Website schon Mobil?
              <br/>
              Hier testen!
             </span>
            </a>
           </div>
           <p class="clear">
            <br/>
            Das Internet hat sich zu einer grundlegenden Informationsquelle etabliert. Wer eine ansprechende Webseite findet, der bleibt.
           </p>
           <p>
            Genau hier möchte ich Sie unterstützen. Ich entwickle für Sie eine gut strukturierte, übersichtliche speziell für Sie und Ihr Unternehmen angepasste Website. Damit ihre Seite ein gutes Fundament hat, berate ich sie individuell.
           </p>
           <p>
            Benötigen sie eine dynamische
            <a href="http://cohowe-design.de/de/service/webdesign-statisch">
             » (CMS)
            </a>
            oder eine klassische
            <a href="http://cohowe-design.de/de/service/webdesign-statisch">
             » (HTML5)
            </a>
            Website.
            <br/>
            Sie erhalten, eine speziell auf Ihr Unternehmen und Sie angepasstes Design mit einer klaren Navigation / Benutzerführung.
           </p>
           <p>
            Besonders großen Wert lege ich auf eine standartkonforme CSS &amp; HTML validierte Programmierung Ihrer Website. Eine saubere programmierte Seite ist
            <a href="http://cohowe-design.de/de/infos/suchmaschine">
             suchmaschinenfreundlich
            </a>
            und
            <a href="http://cohowe-design.de/de/infos/barrierefreiheit">
             barrierearm
            </a>
            , das ist mir wichtig!
           </p>
           <p>
            <a href="http://www.cohowe.de/de/info-thema/responsive-webdesign">
             Responsive Webdesign
            </a>
            ist selbstverständlich und unverzichtbar. Die mobile Optimierung ihrer Website ermöglicht eine geräteübergreifende Darstellung ihres Online-Auftritts, sowohl auf mobilen Smartphone, als auch auf Tablets und Desktops.
           </p>
           <p>
            Die technische Umsetzung, sowie Beratung und die Hilfestellung bei der Auswahl eines Internetproviders oder den Umzug zu einem neuen Provider sind selbstverständlich, sowie die Domainanmeldung.
           </p>
           <p>
            Hier finden Sie Informationen zu
            <a href="http://www.cohowe.de/de/webdesign-service">
             Webdesign - Service - Saarbrücken
            </a>
           </p>
          </div>
