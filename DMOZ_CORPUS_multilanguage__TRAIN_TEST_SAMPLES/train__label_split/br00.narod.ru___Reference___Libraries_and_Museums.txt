    <h1>
     <b>
      <font color="#993300">
       Иосиф Александрович БРОДСКИЙ
       <br/>
       <br/>
       (1940-1996)
       <br/>
       <br/>
       Поэт, переводчик, критик, эссеист, драматург,
       <br/>
       <br/>
       лауреат Нобелевской премии по литературе 1987 года
      </font>
     </b>
    </h1>
   </center>
   <br/>
   <br/>
   <center>
    <img border="0" height="568" src="55_022.jpg" vspace="15" width="697"/>
   </center>
   <br/>
   <br/>
   <center>
    <p>
     <font color="#ff0033">
      "Наше время - планетарное время - выразило Бродскому свое признание с необычайной щедростью.
      <br/>
      Такого множества высочайших наград и почетных званий при жизни
      <br/>
      не получал ни один поэт; кажется, и близкого к такому не было.
      <br/>
      Среди этих знаков признания есть и такой:
      <br/>
      в минувшем году Флоренция избрала Бродского своим почетным гражданином -
      <br/>
      как бы расплачиваясь с российским изгнанником
      <br/>
      за выдворенного ею некогда флорентийского гражданина Данте,
      <br/>
      который и по смерти не пожелал вернуться в родной город, - и тем самым говоря: Бродский - наш Данте.
      <br/>
      <br/>
      Время, в лице своих авторитетных институций - комитетов по международным премиям, академий,
      <br/>
      университетов, муниципалитетов древних городов,
      <br/>
      безоговорочно признало в Бродском своего центрального поэта.
      <br/>
      И конечно, это же сделала читающая русская публика, не успевшая каким-то формальным,
      <br/>
      официальным образом короновать Бродского при жизни.
      <br/>
      Российских наград и российских званий Бродский не получил:
      <br/>
      и в России - это самая высокая честь, как известно.
      <br/>
      Быть отмеченным государством, которое ни с одним из своих поэтов по-человечески не обошлось, -
      <br/>
      это бросает на награжденного некоторую тень штрейкбрехерства;
      <br/>
      да и вообще, всерьез Россия любит только поэтов-страдальцев.
      <br/>
      И Бродский, не навестивший родину в те годы, когда он был бы здесь самым желанным гостем,
      <br/>
      подтвердил этим неотменимость своего страдания: в определенном смысле - смертельность своего изгнания.
      <br/>
      Но еще сильнее, чем признание во всем мире и государственное гонение на родине, центральное положение
      <br/>
      Бродского выражает отношение множества читателей к нему: особенно интимное, находящее в его сочинениях
      <br/>
      собственный опыт, который без Бродского остался бы невыраженным,
      <br/>
      не нашедшим формы, не вошедшим в искусство и в историю.
      <br/>
      "Он выразил наше поколение", "он пишет именно о том, что я думаю и чувствую, -
      <br/>
      и выражает это единственно возможным образом," -
      <br/>
      такие слова часто можно услышать от читателей Бродского".
     </font>
     <br/>
     <i>
      Ольга Седакова
     </i>
    </p>
