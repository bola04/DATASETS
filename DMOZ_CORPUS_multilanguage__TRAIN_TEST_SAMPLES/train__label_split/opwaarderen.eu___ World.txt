       <h2 class="bb-seosubheader">
        Beltegoed opwaarderen
       </h2>
       Beltegoed opwaarderen kan zeer makkelijk online. Volg bovenstaande stappen om je prepaid beltegoed online te kopen. Voor het opwaarderen gebruik je de code die je na betaling ontvangt. In principe kun je het vergelijken met de beltegoed-code die je in de winkel op een bonnetje ontvangt.
       <br/>
       <br/>
       <h2 class="bb-seosubheader">
        Opwaardeer instructies
       </h2>
       Wanneer je de beltegoed bestelling hebt voltooid ontvang je de beltegoed code. Middels de opwaardeer instructies kun je de beltegoed code invoeren voor het opwaarderen van je beltegoed. Elke provider kent zijn eigen opwaardeer instructie met bijbehorend opwaardeer nummer dat je moet bellen.

       <a class="bb-route" href="/opwaardeer-instructies">

       <h2 class="bb-seosubheader">
        Beltegoed opvragen
       </h2>
       Voor het prepaid opwaarderen van je beltegoed wil je weten hoeveel saldo aan beltegoed nog op je telefoon staat. Het prepaid saldo is vaak eenvoudig op te vragen via het servicenummer of je toetst een code in op je telefoon. Vervolgens hoor je of zie je op je scherm de hoogte van het tegoed.
       <br/>
       Elke provider heeft een unieke instructie, per provider vind je hier hoe je het beltegoed
