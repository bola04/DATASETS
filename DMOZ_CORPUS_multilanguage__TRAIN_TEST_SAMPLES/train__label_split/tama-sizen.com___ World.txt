       <h2 class="pb-4">
        多摩市立八ヶ岳自然の家は、県内外どなたでも宿泊できる施設です。
        <br/>
        アウトドア初心者の方も、安心して自然を存分にお楽しみください。
       </h2>
       <p class="mb-0">
        多摩市立八ヶ岳少年自然の家は、青少年が良好な自然環境の中でのびのびとした野外活動や集団生活の中、育成と交流を深めるための施設として、また、自然が好きな地元町民、県内外、年齢を問わず、2名様以上でどなたでもご利用いただけます。
        <br/>
        食事や宿泊施設は完備されているので、アウトドア初心者の方も安心です。なお、当日のオリエンテーションへ必ず参加してください。
       </p>

      <h2 class="text-center mb-8">
       <small class="d-block font-size-base pb-4">
        HOW TO ENJOY AND HOW TO SPEND
       </small>
       <small class="d-block font-size-lg text-dark">
        自然に触れて季節を感じて、また帰って来たくなる八ヶ岳

       <div class="col-md-6 images pr-lg-0">

       </div>
       <div class="col-md-6">
        <h2 class="text-center pt-3 pb-4">
         <small class="d-block font-size-base">
          CLIMBING / TREKKING
         </small>
         登山・トレッキング
        </h2>
        <p>
         八ヶ岳周辺には、高原の涼やかな空気と自然が織りなす風景を楽しみながら、登山初心者の方も楽しめるコースがたくさんあります。また、自然の家から車で20分の富士見パノラマリゾートでは、ゴンドラに乗って入笠山へ。四季折々の山野草が咲き誇る湿原を抜けて、360°の大パノラマを満喫することができます。
        </p>
        <a class="btn btn-link" href="http://www.fujimipanorama.com/" target="_blank">
         富士見パノラマリゾートウェブサイト
         <i class="icon-angle-right02">
         </i>
        </a>
       </div>
      </div>
      <div class="row flex-row-reverse pb-6">

       </div>
       <div class="col-md-6">
        <h2 class="text-center pt-3 pb-4">
         <small class="d-block font-size-base">
          ENJOY WINTER SPORTS
         </small>
         ウィンタースポーツを楽しむ
        </h2>
        <p>
         富士見町には、富士見高原スキー場と富士見パノラマリゾートの2つのスキー場があります。どちらもアクセス良好。パウダースノーの降り積もる景色も見事です。スキー教室やスノーシュー体験も行うことができます。たくさん遊んだ後は、床暖房の効いた自然の家のお部屋でくつろぎタイム。
        </p>
        <a class="btn btn-link" href="http://fujimikogen-resort.jp/" target="_blank">
         富士見高原リゾートのウェブサイト
         <i class="icon-angle-right02">
         </i>
        </a>
       </div>

      <div class="row pb-6">

       </div>
       <div class="col-md-6">
        <h2 class="text-center pt-3 pb-4">
         <small class="d-block font-size-base">
          EXPERIENCE THE FOREST
         </small>
         森を体験
        </h2>
        <p>
         敷地内にある森では、散策路を歩きながらゆったりと自然観察をすることができます。鳥のさえずりを聞きながら、のんびり森林浴を行えば、ココロも体もリフレッシュ。少し足を延ばして立場川キャンプ場（徒歩40分）へ。夏でもひんやり冷たい清流で川遊びが楽しめます。
        </p>
       </div>
      </div>

       <div class="col-md-6">
        <h2 class="text-center pt-3 pb-4">
         <small class="d-block font-size-base">
          AUTUMN LEAVES
         </small>
         秋は紅葉散策
        </h2>
        <p>
         10月になると木々が少しづつ色づき始め、赤、橙、黄色が重なり合うこの時期にしか見ることのできない見事な風景が広がります。
         <br/>
         自然の家から車で20分の富士見パノラマリゾートでは、紅葉ゴンドラも運行し、入笠山の秋の山野草とともに散策を楽しめます。
        </p>
       </div>

        </h2>
        <p>

         <br/>

        </p>

        <h2 class="text-center pt-3 pb-4">
         <small class="d-block font-size-base">
          OBSERVATION OF THE STARS
         </small>
         星空観察・ナイトハイク
        </h2>
        <p>
         標高の高い八ヶ岳の麓は、空気は澄み星の光をさえぎる光源がないため、空いっぱいに広がる満天の星空を眺めることができます。
         <br/>
         また、40〜50分で自然の家周辺をナイトハイクするコースがあります。真っ暗な森の中を散策し、月の灯下森の息吹を感じるのも楽しみのひとつ。

     //スクロール禁止用関数
     function no_scroll(){
         //PC用
         var scroll_event = 'onwheel' in document ? 'wheel' : 'onmousewheel' in document ? 'mousewheel' : 'DOMMouseScroll';
         $(document).on(scroll_event,function(e){e.preventDefault();});
         //SP用
         $(document).on('touchmove.noScroll', function(e) {e.preventDefault();});
     }
 
     //スクロール復活用関数
     function return_scroll(){
         //PC用
         var scroll_event = 'onwheel' in document ? 'wheel' : 'onmousewheel' in document ? 'mousewheel' : 'DOMMouseScroll';
         $(document).off(scroll_event);
         //SP用
         $(document).off('.noScroll');
     }
 
     //

     // For Toggle menu
     $('.navbar-collapse').on('show.bs.collapse', function () {
         $('header').addClass("toggle-on");
         no_scroll();
     });
     $('.navbar-collapse').on('hide.bs.collapse', function () {
         $('header').removeClass("toggle-on");
         return_scroll();
     });

     $(".header-fontsize-switch").find("i").on('click',function(){
         $getFontSize = $(this).attr('data-fontsize');
         $(".header-fontsize-switch").find("i").removeClass("active");
         $(this).addClass("active");
         $('html').removeClass(function(index, className) {
             return (className.match(/\bfont-\S+/g) || []).join(' ');
         });
         $('html').addClass($getFontSize);
         $.removeCookie("fontSize", { path: "/" });
         $.cookie('fontSize', $getFontSize, { path: "/" });
     });
 
     //Background
     $(".header-bg-switch").find("i").on('click',function(){
         $getBgColor = $(this).attr('data-bgsize');
         $(".header-bg-switch").find("i").removeClass("active");
         $(this).addClass("active");
         $('html').removeClass(function(index, className) {
             return (className.match(/\bbg-\S+/g) || []).join(' ');
         });
         $('html').addClass($getBgColor);
         $.removeCookie("bgColor", { path: "/" });
         $.cookie('bgColor', $getBgColor, { path: "/" });
     });
 
     //Cookie
     $htmlFontSize = $.cookie('fontSize');
     $htmlBgColor = $.cookie('bgColor');

     $("html").addClass($htmlFontSize);
     $("html").addClass($htmlBgColor);
 
     $(".header-fontsize-switch").find("i").each(function () {
