               <h2>
                What is MemTest86
               </h2>
               <p>
                MemTest86 is the original, free, stand alone memory testing software for x86 computers. MemTest86 boots from a USB flash drive or CD and tests the RAM 
 														in your computer for faults using a series of comprehensive algorithms and test patterns.
               </p>
               <h2>
                Main Features
               </h2>
               <p>
                The latest version of MemTest86 supports all the current technologies, including
               </p>
               <ul>
                <li>
                 13 different RAM testing algorithms
                </li>
                <li>
                 DDR4 RAM (and DDR2 &amp; DDR3) support
                </li>
                <li>
                 XMP - high performance memory profiles
                </li>
                <li>
                 UEFI - The new graphical standard for BIOS
                </li>
                <li>
                 64bit - From version 5, MemTest86 is native 64bit code
                </li>
                <li>
                 ECC RAM - support for error-correcting code RAM
                </li>
                <li>
                 Secure boot - With MemTest86 being code signed by Microsoft
                </li>
                <li>
                 Graphical interface, mouse support and logging of results to disk
                </li>
                <li>
                 Foreign language support (Chinese, German &amp; more)
                </li>
                <li>
                 Self booting off USB or CD, without needing DOS, Linux nor Windows
                </li>
                <li>
                 Network (PXE) boot - Scalable, disk-less provisioning from a single PXE server
                </li>
                <li>
                 Dual booting - Will boot to V4 in BIOS and V7 in UEFI from a single flash drive
                </li>
               </ul>
               <p align="left">
                <a href="technical.htm">
                 Click here to find out more »
                </a>
               </p>
               <h2>
                History
               </h2>
               <p>
                The test algorithms have been developed for over 20 years, with the origin of MemTest86 going back to 1994.
 														See this page for a more detailed
                <a href="history.htm">
                 history of MemTest86
                </a>
               </p>
               <h2>
                Why use MemTest86
               </h2>
               <p>
                Unreliable RAM can cause a multitude of problems. Corrupted data, crashes and unexplained behaviour. Bad RAM is one of the most
 														frustrating computer problems to have as symptoms are often random and hard to pin down. MemTest86 can help diagnose faulty RAM 
 														(or rule it out as a cause of system instability). As such it is often used by system builders, PC repair stores, overclockers
 														&amp; PC manufacturers.
               </p>
               <h2>
                MemTest naming
               </h2>
               <p>
                The number 86 following the MemTest part of the name, refers to the original
                <a href="http://en.wikipedia.org/wiki/X86" target="_blank">
                 x86 CPU instruction set
                </a>
                that
 													 all of today's PCs use. This covers both 32bit and 64bit machines. (So there is no need for a future product called MemTest64).
               </p>
               <h2>
                Free, Professional or Site Edition
               </h2>
               <p>
                Since MemTest86 v5, the software is offered as a Free edition, or as a paid for Pro and Site edition. The Pro edition offers
 														a number of
                <a href="features.htm">
                 additional features
                </a>
                such as customizable reports &amp; automation via a configuration file. The
                                                         Site edition includes all
                <a href="features.htm">
                 features
                </a>
                in the Pro Edition but also supports scalable deployment of MemTest86 
                                                         across LAN via PXE boot
               </p>
