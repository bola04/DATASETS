            <div class="comop">
             米原市商工会、東浅井商工会、びわ商工会、長浜北商工会では、小規模事業者の持続的発展支援ための５年計画「経営発達支援計画」に基づいて、地域経済動向に関する情報等を収集するための「道の駅等交通量調査事業」を実施しました。
             <br/>
             つきましては、当該調査結果を含む地域経済動向に関する情報等について、インターネット上に公表いたします。
             <br/>
             ぜひ、湖北地域における事業計画策定や販路開拓等の経営戦略立案や実現の際にご活用ください。

            <div class="comop">
             <br/>
             米原市商工会では、米原市の受託事業として、インターネットショップを開設・運営することとなりました。インターネットを通じて米原市内の特産品・工芸品・農林水産物等の新たな販路を開拓する事業として、現在準備を進めています。
             <br/>
             <br/>
             商工会として初めて取り組む分野でもあり、米原市商工観光課、運営支援者とともに毎週月曜日の夜に会議を開き、ああだこうだと議論を重ねてきました。
             <br/>
             <br/>
             そして、３月１日、「orite（オリテ）米原市特選品市場」がオープンしました。「orite（オリテ）」とは、もっともっとたくさんの人に米原を通過せずに、「降りて」もらいたいとの願いをこめたポータルサイト名です。
             <br/>
             <br/>
             商工会らしくない斬新な企画とデザインまた行動力をフルに発揮しながら、メイドイン米原の商品の良さを日本全国に発信し伝えます。右のロゴが目印です。
             <br/>
             <br/>
             水引をモチーフに交流のまち米原を表現しています。ワクワクするような企画や情報の発信も行いますので、ぜひ、アクセスいただき、ご愛顧いただきますようお願いいたします。
