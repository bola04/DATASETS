     <br/>

    【３】　佐々木事務所の業務範囲
    <br/>
    <br/>
    １
    <font color="#000000">
     株式会社
    </font>
    の設立登記手続きの内、佐々木事務所が代行・代理を行う業務は、
    <br/>
    次の【４】の手続きのうち、
    <font color="#0000ff">
     青字の部分
    </font>
    です。
    <br/>
    ２　「黒字の部分」の手続きにつきましては、十分ご説明いたします。
    <br/>
    ３　登記所への
    <font color="#000000">
     株式会社
    </font>
    の設立登記の代理申請は、一般に、登記の専門家である
    <br/>
    司法書士が行います。
    <br/>
    ４　佐々木事務所は、「司法書士・行政書士・税理士事務所」ですので、会社設立を、
    <br/>
    安心して、ご依頼いただけます。
    <br/>
    ５　会社設立直後における、税務署・都税事務所・市役所・社会保険事務所・
    <br/>
    職業安定所・労働基準監督署などへの届け出なども、ご支援できます。
    <br/>
    ６　「12　設立時の税務署など税務官庁への届け出」は、別途、お引き受け
    <br/>
    しております。「報酬」は、１万円（８％税込み１０，８００円）です。
    <br/>
    ７　事業開始後における、会計経理処理・税務申告なども、ご支援できます。
    <br/>
   </p>
   <p class="Class30">
    【４】　会社設立の手続きの流れと所用期間
    <br/>
    <br/>
    １
    <font color="#000000">
     株式会社
    </font>
    の設立の登記が完了するまでには、順調にいっても、
    <br/>
    ３−４週間はかかります。
    <br/>
    ２　４週間のうち、登記所に申請するまでに、２週間、その後の登記所での事務作業に、
    <br/>
    １−２週間かかります。
    <br/>
    ３　会社設立のためには、次の事務手続きが必要になります。
    <br/>
    １　「商号（会社名）」などの「定款記載事項」の決定
    <br/>
    ２　出資者・役員の「印鑑証明書」の入手
    <br/>
    <font color="#000000">
    </font>
    <font color="#0000ff">
     ３　登記事項の事前確認
    </font>
    <font color="#000000">
     （司法書士業務）
    </font>
    <br/>
    <font color="#0000ff">
     ４　定款等の設立書類の作成
    </font>
    （司法書士・行政書士業務）
    <br/>
    <font color="#0000ff">
     ５　公証人による定款の認証手続き
    </font>
    （司法書士・行政書士業務）
    <br/>
    ６　「出資金」の払い込み
    <br/>
    <font color="#0000ff">
     ７　登記所への設立登記の申請
    </font>
    （司法書士業務）
    <br/>
    ←【この日が、会社設立日になります。】
    <br/>
    【 この間、登記所での事務処理に１−２週間掛かります。 】
    <br/>
    ８
    <font color="#0000ff">
     登記所での登記簿謄本・印鑑証明書の入手
    </font>
    <font color="#000000">
     （司法書士業務）
    </font>
    <br/>
    【これで、会社設立が完了したことになります。】
    <br/>
    <font color="#0000ff">
     ９　設立時の税務署など税務官庁への届け出
    </font>
    （税理士業務）
    <br/>
    <br/>
   </p>
   <p class="Class30">
    【５】
    <font color="#000000">
     株式会社の
    </font>
    設立に必要なもの
    <br/>
    <br/>
    １　株式会社は、「出資金（資本金）」として、最低１円以上が必要です。
    <br/>
    金銭以外の財産を「現物出資」するときには、資本金として、現金を用意する
    <br/>
    必要はありません。
    <br/>
    ２　会社には、役員として、取締役が１名以上必要です。監査役は、置くことも
    <br/>
    できますが、通常は置きません。
    <br/>
    ３　出資者・役員の印鑑証明書が必要です。印鑑証明書をご用意できない方は、
    <br/>
    出資者・役員になれません。
    <br/>
    ４　「出資金（資本金）」は、銀行に預ける必要はありません。
    <br/>
    「出資金（資本金）」は、出資者（発起人）の個人の口座に振り込みます。
    <br/>
    ５　株式会社を設立するには、「法務局（登記所）」に「設立登記」が必要です。
    <br/>
    「設立登記申請書」を提出した日が、会社の設立日になります。
    <br/>
    登記所への設立登記の代理申請ができるのは、司法書士及び弁護士
    <br/>
    （法人を含む。）のみです。
    <br/>
    ６　会社設立には、既に述べたとおりの費用が掛かります。
