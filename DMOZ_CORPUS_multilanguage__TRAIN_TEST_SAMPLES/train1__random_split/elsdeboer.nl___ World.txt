        <div class="page-content entry-content" itemprop="articleBody">
         <p style="text-align: left;">
          <span style="color: #000000;">
           Welkom op de website van Core &amp; Pip; de naam waaronder ik (Els de Boer-Brethouwer) mijn werk publiceer.
          </span>
          <br/>
          <span style="color: #000000;">
           De naam Core &amp; Pip heb ik bedacht tijdens het werken aan mijn kunstproject waarin een plant centraal staat: de Amphorafonia domestica ofwel Ruiskelk.
          </span>
          <br/>
          <span style="color: #000000;">
           Core &amp; Pip staat voor een fictief bedrijfje dat Ruiskelken kweekt en aanverwante artikelen verkoopt.
          </span>
          <br/>
          <span style="color: #000000;">
           Het gebruik van deze naam voor het publiceren van al mijn werk is een praktische en logische stap, al mijn werk draait nu immers om de Ruiskelk.
          </span>
         </p>
         <hr/>
         <p style="text-align: left;">
          <span style="color: #008000;">
           Willkommen auf der Website von Core &amp; Pip; unter diesem Namen veröffentliche ich (Els de Boer-Brethouwer) meine Arbeit.
          </span>
          <br/>
          <span style="color: #008000;">
           Der Name Core und Pip ist entstanden während der Arbeit an mein Kunstprojekt in dem eine Pflanze zentral steht: der Amphorafonia domestica, auch bekannt als Raschelkelch.
          </span>
          <br/>
          <span style="color: #008000;">
           Core und Pip ist der Betriebsname von einem fiktiven Unternehmen in diesem Projekt, das Raschelkelche züchtet und verwandte Artikel verkauft.
          </span>
          <br/>
          <span style="color: #008000;">
           Es ist eine praktische Entscheidung meinerseits um dieser Name jetzt zu verwenden für al meine Aktivitäten und Veröffentlichungen auf Kunstgebiet.
