       <p>
        Het begon allemaal met een zeilvakantie in de zomer van 1993. Aan het einde van hun eerste jaar gingen Marleen, 
 Louise (toen nog met een e), Saskia en Femke een weekje zeilen op de Friese wateren. Deze week was om 
 meerdere redenen een groot succes. Vrolijk begonnen in bootje nummer 1, moest na 	één dag deze boot worden 
 ingeruild voor bootje nummer 2. Reden: een enorme botsing met een 70 meter lange tanker. Met behulp van een 
 colaatje van de aardige schipper bijgekomen van de schrik en koud van het natte pak, werden we afgevoerd naar 
 het plaatselijke politiebureau, waar we de grootste attractie "ever" waren. Wij zijn de beroerdste niet: hadden die mannen 
 ook eens een leuke dag. De rest van de week werden we overal herkend, we hadden met onze actie zelfs de 
 plaatselijke kranten gehaald! Een ander succes was dat we bedachten om een damesdispuut (met de naam STER) 
 op te richten. Helaas besloot Louise na dat eerste jaar Enschede te verruilen voor AmSTERdam, waardoor zij de echte 
 oprichting van STER niet "live" mee kon maken. Op 11 oktober 1993 was deze oprichting een feit. Zeven dames (Ilse,
 Mariska, Maud, Saskia, Marleen, Fiona en Femke) kwamen bij Femke thuis bij elkaar om eerst gezellig met elkaar te 
 gaan eten en vervolgens na te denken over hoe ze het dispuut wilden vormgeven. Zeven eigenwijze vrouwen, dus dat 
 had nog wel wat voeten in de aarde.
        <br/>
        <br/>
        In de eerste maanden volgden vele etentjes en vergaderingen waar ideeën voor activiteiten, ideeën voor een
 kledingstuk en een vaste borrel uit voort kwamen. In eerste instantie werd de borrel nog in café van Eijck gehouden 
 en was er nog niet echt sprake van een wekelijks terugkerend fenomeen. Na enige tijd werd de maandagavond 
 uitverkozen tot de vaste borrelavond. We besloten de borrel te verplaatsen naar gezellig (bier)café de Geus, omdat 
 daar op de maandagavond het meest te beleven viel.
        <br/>
        <br/>
        De identiteit van STER -inmiddels overal bekend- werd nog eens vergroot door de enige echte STER-trui. Via de 
 huisgenoot van Marleen werden de stoere schipperstruien aangeschaft. Met deze truien togen Marleen, Saskia en 
 Femke (nauwelijks bekomen van de avond daarvoor in "t Gat) naar het dorpje Houten in de provincie Utrecht. Daar 
 op het station aangekomen, bleek de enige manier om bij het verafgelegen borduurbedrijf te komen, de fiets te zijn. 
 We huurden drie fietsen op het station en fietsten als drie echte Hollandse meiden, met een enorme vaart en een pak
 melk in de hand, tussen de weilanden door naar het bedrijfje. Al snel werden ons alle mogelijkheden van het 
 borduren duidelijk, waarop wij kozen voor een strak maar gedistingeerd logo, namelijk de letters STER.
        <br/>
        <br/>
        Het dispuut STER stond nu in Enschede -en omstreken- stevig op de kaart. De eerste lichting had hiermee de 
 fundamenten gelegd voor het leukste, mooiste en gezelligste damesdispuut dat Enschede rijk is: damesdispuut STER!

    var slideIndex = 1;
 showSlides(slideIndex);
 
 function plusSlides(n) {
   showSlides(slideIndex += n);
 }
 
 function currentSlide(n) {
   showSlides(slideIndex = n);
 }
 
 function showSlides(n) {
   var i;
   var slides = document.getElementsByClassName("mySlides");
   var dots = document.getElementsByClassName("dot");
   if (n > slides.length) {slideIndex = 1} 
   if (n < 1) {slideIndex = slides.length}
   for (i = 0; i < slides.length; i++) {
       slides[i].style.display = "none"; 
   }
   for (i = 0; i < dots.length; i++) {
       dots[i].className = dots[i].className.replace(" active", "");
   }
   slides[slideIndex-1].style.display = "block"; 
   dots[slideIndex-1].className += " active";
 }
