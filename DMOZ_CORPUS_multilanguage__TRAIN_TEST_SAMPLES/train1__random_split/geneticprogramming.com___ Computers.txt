         <h2>
          About Genetic Programming
         </h2>
         <p>
          <img alt="Genetic Programming functions" height="109" src="http://geneticprogramming.com/wp-content/uploads/2016/05/screenshot_trees-1100x109.png" width="1100"/>
         </p>
         <p>
          Genetic Programming (GP) is a type of Evolutionary Algorithm (EA), a subset of machine learning. EAs are used to discover solutions to problems humans do not know how to solve, directly. Free of human preconceptions or biases, the adaptive nature of EAs can generate solutions that are comparable to, and often better than the best human efforts.
          <a href="http://watchmaker.uncommons.org/manual/ch01s02.html" target="_blank">
           *
          </a>
         </p>
         <p>
          Inspired by biological evolution and its fundamental mechanisms, GP software systems implement an algorithm that uses random mutation, crossover, a fitness function, and multiple generations of evolution to resolve a user-defined task. GP can be used to discover a functional relationship between features in data (symbolic regression), to group data into categories (classification), and to assist in the design of electrical circuits, antennae, and quantum algorithms. GP is applied to software engineering through code synthesis, genetic improvement, automatic bug-fixing, and in developing game-playing strategies, … and more.
         </p>
         <p>
          Types of GP include:
         </p>
         <ul>
          <li>
           <a href="http://geneticprogramming.com/about-gp/tree-based-gp/">
            Tree-based Genetic Programming
           </a>
          </li>
          <li>
           <a href="http://geneticprogramming.com/about-gp/stack-based-gp/">
            Stack-based Genetic Programming
           </a>
          </li>
          <li>
           <a href="https://en.m.wikipedia.org/wiki/Linear_genetic_programming" target="_blank">
            Linear Genetic Programming
           </a>
           (LGP)
          </li>
          <li>
           <a href="https://en.wikipedia.org/wiki/Grammatical_evolution" target="_blank">
            Grammatical Evolution
           </a>
          </li>
          <li>
           Extended Compact Genetic Programming (ECGP)
          </li>
          <li>
           <a href="http://www.cartesiangp.co.uk/" target="_blank">
            Cartesian Genetic Programming
           </a>
           (CGP)
          </li>
          <li>
           Probabilistic Incremental Program Evolution (PIPE)
          </li>
          <li>
           Strongly Typed Genetic Programming (STGP)
          </li>
          <li>
           <a href="http://www0.cs.ucl.ac.uk/staff/ucacbbl/gismo/" rel="noopener" target="_blank">
            Genetic Improvement of Software for Multiple Objectives
           </a>
           (GISMO)
          </li>
         </ul>
         <p>

           Generational GP Algorithm
