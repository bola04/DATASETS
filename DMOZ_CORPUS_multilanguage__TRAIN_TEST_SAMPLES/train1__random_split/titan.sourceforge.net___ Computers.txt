          <h2>
           Titan 2.0
          </h2>
          <small>
           <b>
            Most recent version - Titan 2.0 (22/07/2004)
            <br/>
           </b>
          </small>
          <br/>
          <br/>
         </center>
         &amp;nbsp &amp;nbsp Welcome to the Titan website. Titan is an image handler for your graphics applications.
         <br/>
         <br/>
         &amp;nbsp &amp;nbsp For still images, Titan can load/save most varieties from .TGA, .BMP, .PCX, .PNG and .JPG.
         <br/>
         &amp;nbsp &amp;nbsp Files can be loaded from from disk, from memory, or from anywhere else you like, for use in a virtual file system. Future releases are going to reintroduce the old animation system and add sound and 3d mesh handlers.
         <br/>
         <br/>
         &amp;nbsp &amp;nbsp The following platforms/compilers are currently supported -
         <br/>
         &amp;nbsp &amp;nbsp &amp;nbsp &amp;nbsp Windows/Microsoft Visual C++ 6
         <br/>
         With Linux and Windows Gnu versions coming as soon as I create some makefiles
         <br/>
         <br/>
         &amp;nbsp &amp;nbsp I am keen to hear your thoughts about Titan, what it should or shouldn't do or how it
 can be improved. I especially want to hear of any bugs you may find.
         <br/>
         <br/>
         &amp;nbsp &amp;nbsp Titan is released under the
         <a href="http://www.gzip.org/zlib/zlib_license.html">
          Zlib license
         </a>
         . This basically means that you can link Titan to your application free of charge. Any modifications to the source you make MUST be made available to the general public and may be used to further Titans development.
