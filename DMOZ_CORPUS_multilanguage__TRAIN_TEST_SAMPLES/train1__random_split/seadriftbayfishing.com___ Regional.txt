           </span>
          </h1>
          <p>
           <span style="line-height:1.3em;">
            Welcome to Castaway Lodge in the heart of Seadrift, Texas where you'll find a rich heritage of local watermen and professional fishing and hunting guides. Whether you're after that one-of-a-kind Seadrift fishing and hunting experience or taking a quick Texas staycation, we'll get you right in the action! From heart pounding fishing trips to hot and fast wingshooting for waterfowl and dove, or the excitement of deer and hog hunting, we offer numerous full-day combination trips; night time activities; and, the best Seadrift fishing lodge backdrop for relaxing with friends and family; customer and employee appreciation fishing trips; or, networking and team building with associates.
           </span>
          </p>
          <p>
           <span style="line-height:1.3em;">
            "From concept to planning and construction, our family of local watermen have worked hard to bring one of the last special places to life".  We are owner operators wearing a dozen hats and striving for a great experience for our guests. We are local hunting and fishing guides, we are "Seadrifters"!
           </span>

                         var coords = "50% " + yPos + "px";
                         $scroll.css({ backgroundPosition: coords });
                     });
                 }
             });
 
 
             //Anchor scroll
             $("a[href*=\\#]:not([href=\\#], [data-toggle=collapse])").click(function () {
                 if (location.pathname.replace(/^\//, "") == this.pathname.replace(/^\//, "") && location.hostname == this.hostname) {
                     var target = $(this.hash);
                     target = target.length ? target : $("[name=" + this.hash.slice(1) + "]");
                     if (target.length) {
                         $("html,body").animate({
                             scrollTop: target.offset().top - 90
                         }, 1000);
                         return false;
                     }
                 }
             });
 
 
             // Back to Top
             var offset = 220;
             var duration = 500;
             $(window).scroll(function () {
                 if ($(this).scrollTop() > offset) {
                     $('.back-to-top').fadeIn(duration);
                 } else {
                     $('.back-to-top').fadeOut(duration);
                 }
             });
 
             $('.back-to-top').click(function (event) {
                 event.preventDefault();
                 $('html, body').animate({ scrollTop: 0 }, duration);
                 return false;
             })
 
 
             // Loadscroll
 
             function loadScroll() {
                 inf = $.cookie("loadScroll");
                 if (!inf) { return }
                 var ar = inf.split("_");
                 if (ar.length == 2) {
                     window.scrollTo(parseInt(ar[0]), parseInt(ar[1]));
                 }
                 $.removeCookie("loadScroll");
             }

                 var myWindow = window.open("");
                 myWindow.location = url;

                 });
             });
