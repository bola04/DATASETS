     <h2>
      Behandelingen
     </h2>
     <div class="intro">
      De juiste techniek afgestemd op uw ogen. Lasik Centrum oogkliniek biedt naast algemene oogzorg een totaaloplossing van innovatieve en geavanceerde behandelingstechnieken.
      <br/>
      <br/>
      <div class="center">
       <a class="cta" href="/behandelingen/overzicht">
        Hier vindt u alle mogelijkheden
        <br/>
        in één oogopslag
       </a>
      </div>
     </div>
     <div class="listing equalheight" id="behandelingen">
      <article>
       <img alt="" height="134" src="/images/made/48867310f8441c83/lasik_centrum_140_134_home_behandeling_icon_ooglaseren_V2_140_134_s.png" width="140"/>
       <h3>
        Ooglaseren
       </h3>
       <h4>
        Leeftijd 18-55
       </h4>
       <div class="content">
        Bijziendheid tot -10 dioptrie.
        <br/>
        Verziendheid tot +6 dioptrie.
        <br/>
        Cylindersterkte tot -6 dioptrie.
        <br/>
        <br/>
        3 behandelmethoden:
        <ul>
         <li>
          Femto Second LASIK methode
         </li>
         <li>
          Trans PRK methode
         </li>
         <li>
          Lasek methode
         </li>
        </ul>
       </div>
       <a class="cta wrap" href="/behandelingen/ooglaseren/">

       <h3>
        Extra lenzen
       </h3>
       <h4>
        Leeftijd 20-55
       </h4>
       <div class="content">
        Dunner hoornvlies.
        <br/>
        Hogere sterktes vanaf
        <br/>
        -8 en +6 dioptrie.
        <br/>
        Leesproblemen (40+).
        <br/>
        <br/>
        Lens mogelijkheden:
        <ul>
         <li>
          ICL extra lenzen
          <br/>
          (enkelvoudig of torisch)
         </li>
         <li>
          Raindrop extra lens
         </li>
        </ul>
