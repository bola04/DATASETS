         <h2 class="midashi">
          ホームページ制作をご検討の方へ
         </h2>
         <div class="col2Box-L">
          <h3>
           栃木、群馬を中心に全国サポート
          </h3>
          <strong>
           栃木県、群馬県など近郊地域のWeb制作
          </strong>
          <p>
           宇都宮、小山、足利、高崎、前橋、館林、太田など近郊地域は、お伺いミーティングによりホームページ制作が可能です。地の利を活かした細やかなサポートで、効果の高いWebサイト制作をご提供します。
          </p>
          <em>
           遠方地域のお客さまのWeb制作
          </em>
          <p>
           遠方のお客さまのHP制作でも、オンラインでのデザイン提案、テストサイト、お電話、Eメール、スカイプなどにより、円滑なホームページ制作が可能です。遠方でも多くの制作実績ありますので、安心してご相談ください。
          </p>
         </div>
         <div class="col2Box-R">
          <h3>
           SEO×Webプロモーション
          </h3>
          <em>
           集客に強いSEO対策
          </em>
          <p>
           全国検索１位、栃木県内検索１位など検索上位実績多数。新規ユーザーの獲得や、リターンユーザーの増加まで、さまざまな内部SEO対策、外部SEO対策により、ウェブサイトを成功へと導きます。
          </p>
          <em>
           総合的なWebプロモーションをサポート
          </em>
          <p>
           twitter、Facebook、Instagram、LINEなどSNS活用や、動画を利用した動画プロモーション、販促効果の高いキャンペーン、ランディングページ制作など、ホームページ作成だけでなく総合的なWebプロモーションをサポート。
          </p>
         </div>

       // アンカーの値取得

       // 移動先を取得

       // 移動先を数値で取得
       var position = target.offset().top;

       return false;
    });
 });

   <script>
    function smartRollover() {
 	if(document.getElementsByTagName) {
 		var images = document.getElementsByTagName("img");
 
 		for(var i=0; i < images.length; i++) {
 			if(images[i].getAttribute("src").match("_off."))
 			{
 				images[i].onmouseover = function() {
 					this.setAttribute("src", this.getAttribute("src").replace("_off.", "_on."));
 				}
 				images[i].onmouseout = function() {
 					this.setAttribute("src", this.getAttribute("src").replace("_on.", "_off."));
 				}
 			}
 		}
 	}
 }
 
 if(window.addEventListener) {
 	window.addEventListener("load", smartRollover, false);
 }
 else if(window.attachEvent) {
 	window.attachEvent("onload", smartRollover);
 }
   </script>
