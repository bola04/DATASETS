          <div align="Center">
           The pam_cuecat module reads decoded bar codes from
  /dev/scanners/cuecat, looks up the username:barcode_type:barcode line in
  /etc/cuecatpasswd, and returns whether or not this user has been authenticated.
           <br/>
           <br/>
           <i>
            "Making the CueCat useful"
           </i>
           <br/>
           <br/>
           This module by itself cannot communicate with a CueCat.  You will need to
           <a href="http://www.google.com/search?q=cuecat-0.2.1.tar.gz">
            download the CueCat Kernel Driver (version 0.2.0 or 0.2.1)
           </a>
           and, if you want the optional async reads, patch cuecat_driver.c with a file
 included in /usr/doc/pam_cuecat-0.04.
