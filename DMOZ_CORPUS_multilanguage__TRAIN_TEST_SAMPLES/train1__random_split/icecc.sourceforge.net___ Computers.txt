   <h4 align="center">
    About version number
   </h4>
   <p>
    The version number of binary package contains three part: xx.yy.zz. Where xx is the
 major changes in code of the IceWM's Control Center, yy is the minor one and zz is
 the changes in Control Center's tools. Thus the IceWM's Control Center 2.4.x contains
 the code of Control Center shell 2.4 and other my tools.
   </p>
