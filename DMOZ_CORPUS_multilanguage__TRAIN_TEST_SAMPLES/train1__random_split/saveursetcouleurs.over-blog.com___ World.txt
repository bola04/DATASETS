            <a class="titreArticle" href="http://saveursetcouleurs.over-blog.com/article-riz-noir-aux-cepes-et-creme-de-parmesan-facon-risotto-46935236.html" title="RIZ NOIR AUX CEPES ET CREME DE PARMESAN FACON RISOTTO">
             RIZ NOIR AUX CEPES ET CREME DE PARMESAN FACON RISOTTO
            </a>
           </h2>
          </div>
          <div class="contenuArticle">
           <div class="ob-sections">
            <div class="ob-section ob-section-html">
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-size: 12pt;">
                  <em>
                   <img alt="Risotto-rouge3.jpg" class="CtreTexte" height="225" src="http://img.over-blog.com/300x225/1/92/85/67/Risotto-rouge3.jpg" width="300"/>
                   <br/>
                   <br/>
                   Préparation : 15 min
                   <br/>
                  </em>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="font-size: 12pt;">
                    <em>
                     Cuisson : 1h
                    </em>
                   </span>
                   <br/>
                   <br/>
                   <br/>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                    <em>
                     <span style="font-size: 12pt;">
                      Ingrédients (pour 2 personnes) :
                     </span>
                    </em>
                    <br/>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                    - 200 g de
                    <a href="http://www.edelices.com/epicerie-italienne/riz/riz-venere-nero-gli-aironi.html">
                     <span style="color: #00ccff;">
                      riz Venere Nero GLI AIRONI
                     </span>
                    </a>
                    de chez
                    <a href="http://www.edelices.com/index.php">
                     <span style="color: #00ccff;">
                      e.delices.com
                      <br/>
                     </span>
                    </a>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                    - 2 oignons
                    <br/>
                    - 30 g de beurre
                    <br/>
                    - 20 cl de vin blanc sec
                    <br/>
                    - 1L de bouillon de volaille
                    <br/>
                    - 40 g de cèpes déshydratés
                    <br/>
                    - 100 g de parmesan râpé
                    <br/>
                    - 2 cuillères à soupe de crème fraîche épaisse
                    <br/>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-family: verdana,geneva;">
              <strong>
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                    - Sel
                    <br/>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </strong>
             </span>
             <span style="font-size: 10pt;">
              <span style="color: #666699;">
               <span style="font-family: verdana,geneva;">
                <span style="font-size: 10pt;">
                 <span style="color: #666699;">
                  <strong>
                   - Poivre du moulin
                   <br/>
                  </strong>
                  <br/>
                  <br/>
                 </span>
                </span>
               </span>
              </span>
             </span>
             <span style="font-size: 10pt;">
              <span style="color: #666699;">
               <span style="font-family: verdana,geneva;">
                <span style="font-size: 10pt;">
                 <span style="color: #666699;">
                  <strong>
                   <span style="font-size: 12pt;">
                    <em>
                     Préparation :
                    </em>
                   </span>
                   <br/>
                   - Mettre les cèpes à tremper dans un grand volume d'eau.
                   <br/>
                   - Dans une grande casserole, faire suer les
                  </strong>
                 </span>
                </span>
               </span>
              </span>
             </span>
             <a href="http://www.linternaute.com/femmes/cuisine/encyclopedie/fiche_composant/34/oignon.shtml" target="_blank">
              <span style="color: #666699; text-decoration: none; text-underline: none;">
               <span style="font-family: verdana,geneva;">
                <span style="font-size: 10pt;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="color: #666699;">
                    <strong>
                     oignon
                    </strong>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </span>
             </a>
             <span style="font-family: verdana,geneva;">
              <span style="font-size: 10pt;">
               <span style="color: #666699;">
                <span style="font-family: verdana,geneva;">
                 <span style="font-size: 10pt;">
                  <span style="color: #666699;">
                   <strong>
                    s finement hachés avec le beurre.
                    <br/>
                    - Ajouter le riz et remuer jusqu'à ce qu'il brille (environ 5min). Incorporer le vin blanc et
                   </strong>
                  </span>
                 </span>
                </span>
               </span>
              </span>
             </span>
             <a href="http://www.linternaute.com/femmes/cuisine/definition/88/monter.shtml" target="_blank">
              <span style="color: #666699; text-decoration: none; text-underline: none;">
               <span style="font-family: verdana,geneva;">
                <span style="font-size: 10pt;">
                 <span style="color: #666699;">
                  <span style="font-family: verdana,geneva;">
                   <span style="font-size: 10pt;">
                    <span style="color: #666699;">
                     <strong>
                      monter
                     </strong>
                    </span>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </span>
             </a>
             <span style="font-size: 10pt;">
              <span style="color: #666699;">
               <span style="font-family: verdana,geneva;">
                <span style="font-size: 10pt;">
                 <span style="color: #666699;">
                  <strong>
                   un peu le feu et laisser évaporer les vapeurs d'alcool.
                   <br/>
                  </strong>
                 </span>
                </span>
               </span>
              </span>
             </span>
             <span style="color: #666699;">
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <span style="color: #666699;">
                 <strong>
                  - Préparer le bouillon et mouiller le riz en plusieurs fois. Remuer régulièrement jusqu’à absorption du liquide puis ajouter à nouveau du bouillon... ainsi de suite jusqu'à cuisson parfaite. Prévoir un peu plus de bouillon au cas où car la cuisson est beaucoup plus longue que le risotto classique !
                  <br/>
                  - Pendant ce temps, égoutter les cèpes, les émincer finement et les faire dorer une dizaine de minutes à la poêle avec un filet d'huile d'olive, sel et poivre. Réserver.
                  <br/>
                  - Une fois le riz cuit, ajouter le parmesan, la crème fraîche et les champignons et laisser chauffer l'ensemble 2min. Rectifier l'assaisonnement et servir immédiatement.
                  <br/>
                  <br/>
                  <em>
                   Pour certain cette recette sera une hérésie mais c'est quand même original non !
                   <br/>
                   Le côté croquant du riz se marie à merveille avec le fondant des cèpes et le crémeux du fromage !
                  </em>

          <div class="contenuArticle">
           <div class="ob-sections">
            <div class="ob-section ob-section-html">
             <p style="MARGIN: 0cm 0cm 0pt; mso-layout-grid-align: none">
              <em>
               <span style="font-size: 12pt;">
                <span style="font-size: 12pt;">
                 <span style="color: #666699; font-family: Verdana;">
                  <span style="font-size: 12pt;">
                   <span style="color: #666699; font-family: Verdana;">
                    <strong>
                     <img alt="Bugnes.jpg" class="CtreTexte" height="225" src="http://img.over-blog.com/300x225/1/92/85/67/Bugnes.jpg" width="300"/>
                     <br/>
                     <br/>
                     Préparation : 1h15
                     <br/>
                    </strong>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
               <span style="font-size: 12pt;">
                <span style="font-size: 12pt;">

                    <span style="color: #666699; font-family: Verdana;">
                     Cuisson : 4 X 30 secondes

                   </span>
                   <br/>
                   <br/>
                   <br/>
                   Ingrédients :
                   <br/>
                  </strong>
                 </span>
                </span>
               </span>
              </em>
              <strong>
               <span style="font-size: 10pt;">
                <span style="font-family: verdana,geneva;">
                 <span style="color: #666699;">
                 </span>
                </span>
               </span>
              </strong>
              <span style="font-family: verdana,geneva;">
               <strong>
                <span style="font-size: 10pt;">
                 <span style="color: #666699;">
                  -
                 </span>
                </span>
                <span style="font-size: 10pt;">
                 <span style="font-family: verdana,geneva;">
                  <span style="color: #666699;">
                   500 gr de farine
                   <br/>
                  </span>
                 </span>
                </span>
               </strong>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="color: #666699;">
                   - 3 œufs
                   <br/>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="font-family: verdana,geneva;">
                    <span style="color: #666699;">
                     - 1 sachet de levure
                     <br/>
                    </span>
                   </span>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="font-family: verdana,geneva;">
                    <span style="color: #666699;">
                     - 125 gr de beurre pommade
                     <br/>
                    </span>
                   </span>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="font-family: verdana,geneva;">
                    <span style="color: #666699;">
                     - 1 cuillère à soupe de sucre
                     <br/>
                     - 1 pincée de sel
                     <br/>
                    </span>
                   </span>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 10pt;">
                   <span style="font-size: 10pt;">
                    <span style="font-family: verdana,geneva;">
                     <span style="color: #666699;">
                      - 1 cuillère à soupe de rhum
                     </span>
                    </span>
                   </span>
                   <br/>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-size: 12pt;">
               <span style="font-size: 12pt;">
                <br/>
                <br/>
                <strong>
                 <em>
                  <span style="font-family: verdana,geneva;">
                   <span style="color: #666699;">
                    Préparation :
                   </span>
                  </span>
                  <br/>
                 </em>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <strong>
                <span style="color: #666699;">
                 <span style="font-size: 10pt;">
                  - Mélanger un par un les ingrédients jusqu’à obtenir une belle boule. Si votre pâte est un peu trop sèche, ajouter un peu d'eau pour lier le tout.
                  <br/>
                 </span>
                </span>
               </strong>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="color: #666699;">
                   <span style="font-family: verdana,geneva;">
                    - Laisser reposer 30min.
                    <br/>
                    - A l’aide d’un rouleau à pâtisserie, étaler assez finement un morceau de pâte, puis à l’aide d’un couteau pointu, tracer des bandes dans un sens puis dans l’autre afin d’obtenir des losanges d’environ 6cm par 6cm. Fendre le centre de chaque losange.
                    <br/>
                    - Finaliser vos bugnes une par une en prenant un coin du losange, le passer dans le trou et tirer.
                    <br/>
                    - Dans une sauteuse d’huile bouillante, plonger les bugnes jusqu’à ce qu’elles dorent légèrement (environ 30 secondes). Je vous conseiller de les retourner délicatement en cour de la cuisson à l’aide d’une écumoire pour les faire dorer uniformément.
                    <br/>
                   </span>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-family: verdana,geneva;">
               <span style="font-size: 10pt;">
                <strong>
                 <span style="font-family: verdana,geneva;">
                  <span style="color: #666699;">
                   <span style="font-family: verdana,geneva;">
                    <span style="font-size: 10pt;">
                     - Retirer les bugnes avec l’écumoire, déposer sur une feuille de papier absorbant et saupoudrer généreusement de sucre glace.
                     <br/>
                    </span>
                   </span>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
              <span style="font-size: 10pt;">
               <strong>
                <span style="font-family: verdana,geneva;">
                 <span style="color: #666699;">
                  <span style="font-family: verdana,geneva;">
                   <span style="font-size: 10pt;">
                    - Renouveler l’opération avec la pâte restante.
                    <br/>
                   </span>
                  </span>
                 </span>
                </span>
               </strong>
              </span>
              <span style="font-size: 10pt;">
               <strong>
                <span style="font-family: verdana,geneva;">
                 <span style="color: #666699;">
                  <span style="font-family: verdana,geneva;">
                   <span style="font-size: 10pt;">
                    - Laisser refroidir et déguster.
                   </span>
                  </span>
                 </span>
                </span>
                <br/>
               </strong>
              </span>
              <br/>
              <span style="font-family: verdana,geneva;">

               <span style="color: #666699;">
                <span style="color: #666699;">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 12pt;">
                   <strong>
                    <em>
                     <span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: 'Times New Roman'; mso-ansi-language: FR; mso-fareast-language: FR; mso-bidi-language: AR-SA;">
                      <img alt="Veloute-Petit-Pois-copie-3.jpg" class="CtreTexte" height="225" src="http://img.over-blog.com/300x225/1/92/85/67/Veloute-Petit-Pois-copie-3.jpg" width="300"/>
                      <br/>
                      <br/>
                      Cuisson : 35 min
                      <br/>
                     </span>
                     <span style="font-family: verdana,geneva;">
                      Préparation : 20 min
                     </span>
                     <br/>
                     <br/>
                     <br/>
                     Ingrédients : (pour 4 personnes)
                     <br/>
                    </em>
                   </strong>
                  </span>
                 </span>
                </span>
                <strong style="FONT-FAMILY:">
                 <span style="font-family: verdana,geneva;">
                  - 200 g de lardons fumés
                  <br/>
                  - 1 gros oignon
                  <br/>
                  - 2 pommes de terre
                  <br/>
                  - 600 g de petits pois
                  <br/>
                  - 2 cuillères à soupe de ciboulette
                  <br/>
                  - 75 cl de bouillon Kub-Or
                  <br/>
                  - Sel
                  <br/>
                  - Poivre du moulin
                  <br/>
                  Pour les ravioles
                  <br/>
                  - 100 g de St-Môret
                  <br/>
                  - 50 g de fromage râpé
                  <br/>
                  - Poivre
                  <br/>
                  - 16 carrés de pâte à ravioles (épicerie asiatique)
                 </span>
                </strong>
               </span>
              </span>
             </p>
             <em style="mso-bidi-font-style: normal;">
              <span style="color: #666699; font-family: Verdana;">
               <span style="font-size: 12pt;">
                <strong style="FONT-FAMILY:">
                 <span style="font-family: verdana,geneva;">
                  <span style="font-size: 12pt;">
                   <br/>
                   <br/>
                   Préparation :
                   <br/>
                  </span>
                 </span>
                </strong>
               </span>
              </span>
             </em>
             <span style="font-size: 10pt;">
              <span style="color: #666699;">
               <strong style="FONT-FAMILY:">
                <span style="font-family: verdana,geneva;">
                 - Dans une grande cocotte, faire suer l’oignon émincé avec les lardons en veillant à ne pas les faire dorer.
                 <br/>
                 - Ajouter les pommes de terre coupées en dés et les petits pois. Mélanger le tout et laisser saisir 5min.
                 <br/>
                 - Mouiller avec le bouillon et laisser cuire l’ensemble 20min à feu moyen.
                 <br/>
                 - Pendant ce temps vous pouvez préparer les ravioles.
                 <br/>
                 - Dans un bol, mélanger les deux fromages et poivrer (avec ces quantités vous pouvez préparer 8 ravioles, soit 2 pièces par personnes).
                 <br/>
                 - Disposer 8 carrés de pâte à ravioles sur votre plan de travail et les mouiller délicatement avec un peu d’eau tiède à l’aide d’un pinceau.
                 <br/>
                 - Déposer une cuillère à café de préparation au fromage au centre et disposer un nouveau carré de pâte à raviole par-dessus afin de le refermer. Veiller à bien évacuer l’air et à bien souder les 2 carrés de pâte. Pour obtenir de beaux ravioles, vous pouvez les découper avec un emporte-pièce. Réserver.
                 <br/>
                 - Une fois les légumes cuits, passer au blender puis au chinois afin d’obtenir un velouté très onctueux (cela permettra de retirer l’excès de peau des petits pois et les "déchets" des lardons) ! Rectifier l’assaisonnement et réserver au chaud.
                 <br/>
                 - Avant de servir, porter une grande casserole d’eau à ébullition et y plonger les ravioles 2min (veiller à réduite l’ébullition lorsque vous plonger vos ravioles afin de ne pas les percer).
                 <br/>
                 - Déposer deux ravioles dans chaque verrine, recouvrir de velouté et servir immédiatement.
                 <br/>
                 <br/>
                 <em>
                  C'est tout simplement bluffant... limite raffiné !
                  <br/>
                  Et avez-vous remarqué ces jolis

                    verres

                  sur lesquels j'ai craqué sur

               <span style="font-family: Verdana;">
                <span style="font-size: 10pt;">

                     <em>
                      <img alt="Creme-O-Chocolat.jpg" class="noAlign" height="225" src="http://img.over-blog.com/300x225/1/92/85/67/Creme-O-Chocolat.jpg" width="300"/>
                      <br/>
                      <br/>
                      <br/>
                      Préparation : 15 min
                      <br/>
                     </em>
                    </strong>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </span>
              <span style="font-size: 10pt; color: #575757; font-family: Verdana;">
               <span style="font-family: verdana,geneva;">
                <span style="color: #666699;">
                 <strong>
                  <span style="font-size: 12pt;">
                   <span class="stdtxt101">
                    <em>
                     Cuisson : 15 min
                     <br/>
                    </em>
                   </span>
                  </span>
                  <span class="stdtxt101">
                   <br/>
                   <br/>
                   <em>
                    <span style="font-size: 12pt;">
                     Ingrédients (pour 4 personnes) :
                    </span>
                   </em>
                   <br/>
                  </span>
                  <span class="stdtxt101">
                   - 16 g de farine
                   <br/>
                  </span>
                  <span class="stdtxt101">
                   - 8 g de maïzena
                   <br/>
                  </span>
                  <span class="stdtxt101">
                   - 50 cl de lait ½ écrémé
                   <br/>
                  </span>
                 </strong>
                </span>
               </span>
              </span>
              <span class="stdtxt101">
               <span style="font-size: 10pt; font-family: Verdana;">
                <span style="color: #575757;">
                 <span style="font-size: 10pt;">
                  <span style="font-family: verdana,geneva;">
                   <span style="color: #666699;">
                    <strong>
                     - 2 jaunes d’œufs
                     <br/>
                    </strong>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </span>
              <span class="stdtxt101">
               <span style="font-size: 10pt; font-family: Verdana;">
                <span style="color: #575757;">
                 <span style="font-size: 10pt;">
                  <span style="font-family: verdana,geneva;">
                   <span style="color: #666699;">
                    <strong>
                     - ½ fève tonka
                     <br/>
                    </strong>
                   </span>
                  </span>
                 </span>
                </span>
               </span>
              </span>
              <span class="stdtxt101">
               <span style="font-family: Verdana;">
                <span style="font-size: 10pt;">
                 <span style="font-family: verdana,geneva;">
                  <span style="color: #666699;">
                   <strong>
                    - 10 cl de crème fraîche à 3%
                    <br/>
                   </strong>
                  </span>
                 </span>
                </span>
               </span>
              </span>
              <span style="font-size: 10pt; color: #575757; font-family: Verdana;">
               <span style="font-size: 10pt;">
                <span style="font-family: verdana,geneva;">
                 <span style="color: #666699;">
                  <strong>
                   <span class="stdtxt101">
                    - 50 g de sucre en poudre
                    <br/>
                   </span>
                   <span class="stdtxt101">
                    - 100 g de chocolat noir
                    <br/>
                   </span>
                  </strong>
                 </span>
                </span>
               </span>
              </span>
              <span style="font-size: 10pt; color: #575757; font-family: Verdana;">
               <span style="font-size: 10pt;">
                <span style="font-family: verdana,geneva;">
                 <span style="color: #666699;">
                  <strong>
                   <span class="stdtxt101">
                    <em>
                     <span style="font-size: 12pt;">
                      <br/>
                      <br/>
                      Préparation :
                     </span>
                    </em>
                    <br/>
                   </span>
                   <span class="stdtxt101">
                    - Mélanger dans un saladier la farine, la maïzena et la tonka râpée, délayer petit à petit avec 40cl de lait et ajouter la crème fraîche. Incorporer ensuite les jaunes d'œufs.
                    <br/>
                   </span>
                   - Dans une grande casserole, mettre à chauffer à feu moyen le lait restant avec le
                   <span class="stdtxt101">
                    sucre et le chocolat jusqu'à ce que mélange soit bien onctueux.
                    <br/>
                   </span>
                   <span class="stdtxt101">
                    - Incorporer à cette préparation le mélange farine/maïzena/lait et remettre à chauffer toujours sur feu moyen jusqu'à épaississement.
                    <br/>
                   </span>
                  </strong>
                 </span>
                </span>
               </span>
              </span>
              <span style="font-size: 10pt;">
               <span style="font-family: verdana,geneva;">
                <span style="color: #666699;">
                 <strong>
                  <span class="stdtxt101">
                   - Verser immédiatement dans vos ramequins, laisser refroidir et mettre au réfrigérateur.
                   <br/>
                  </span>
                 </strong>
                </span>
               </span>
              </span>
              <span class="stdtxt101">
               <span style="font-size: 10pt; font-family: Verdana;">

           <h2>
            <a class="titreArticle" href="http://saveursetcouleurs.over-blog.com/article-curry-de-poisson-43181672.html" title="CURRY DE POISSON">
             CURRY DE POISSON
            </a>
           </h2>
          </div>
          <div class="contenuArticle">
           <div class="ob-sections">
            <div class="ob-section ob-section-html">
             <p style="margin: 0cm 0cm 0pt;">
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   <img alt="Soupe-de-Poissons" class="noAlign" src="http://img.over-blog.com/300x225/1/92/85/67/Soupe-de-Poissons.jpg"/>
                   <br/>
                   <br/>
                   <br/>
                   Préparation : 15 min
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   Cuisson : 35 min
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   <br/>
                   <br/>
                   Ingrédients : (pour 4 personnes)
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 8 pavés de poisson blanc (colin, cabillaud…)
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 2 oignons
                <br/>
                - 2 bâtons de citronnelle
                <br/>
                - 1L de bouillon de volaille
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 2 cuillères à soupe rases de pâte de piment au basilic
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 4 cuillères à soupe de Nuoc Mam (sauce poisson)
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 2 cuillères à soupe de gingembre râpé
                <br/>
                - 2 cuillères à soupe rases d’huile de sésame
                <br/>
                - 8 feuilles de citronnier
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 50 cl de lait de coco
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Sel
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <br/>
                  <br/>
                  Préparation :
                  <br/>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Préparer le bouillon de volaille et porter à ébullition.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Pendant ce temps, mélanger dans un saladier les oignons et la citronnelle finement émincés, l’huile de sésame, le Nuoc Mam, le gingembre et la pâte de piment.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Une fois le bouillon… bouillant, ajouter la préparation et laisser cuire 5min.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                Ajouter le lait de coco, les feuilles de citronnier entières et porter à nouveau à ébullition. Laisser mijoter une dizaine de minutes.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Découper vos pavés de poisson en dés de 2cmX2cm et plonger dans la préparation.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Laisser cuire le tout une quinzaine de minutes à feu moyen et servir immédiatement accompagné d’un ramequin de riz thaï.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                 <br/>
                 Coucou les coquinous… me revoilà !
                 <br/>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                 Et en plus c’est une victoire ! Je viens enfin de réussir à 100% mon premier curry… Les autres n’étaient pas mal mais trop piquants, trop sucrés, trop…
                 <br/>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                 Et en plus cette préparation est idéale avec du poisson (comme ici), avec des crevettes et même avec du poulet !
                 <br/>
                </span>
               </em>
              </b>
              <b>
               <span style="font-size: 10pt; color: #666699; font-family: Verdana; mso-bidi-font-family: Arial;">
                <br/>
                <br/>
                <a href="http://saveursetcouleurs.over-blog.com/article-28535362.html">
                 <em>
                  <span style="color: #cc99ff; font-family: Verdana; text-decoration: none; mso-bidi-font-family: Arial; text-underline: none;">
                   "N'oubliez pas de vous inscrire à ma Newsletter, il y a de superbes livres de cuisine à gagner... et un gagnant tous les 25 inscrits !"
                  </span>
                 </em>
                </a>
               </span>
              </b>
             </p>

             <p style="margin: 0cm 0cm 0pt;">
              <strong>
               <span style="font-size: 10pt; color: #666699; font-family: Verdana; mso-bidi-font-family: Arial;">
                Chèrs vous tous, chères vous toutes,
                <br/>
               </span>
              </strong>
              <span style="color: #666699; font-family: Verdana; mso-bidi-font-family: Arial;">
               <span style="color: #666699;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana; mso-bidi-font-family: Arial;">

                    <br/>
                    <br/>
                    L'occasion aussi, de rassurer ceux d'entres vous qui se demandent : "mais qu'est ce qu'il fabrique ? Plus de recettes, plus de signe de vie, rien"...
                    <br/>
                    Et bien soyez tranquille, tout va bien, mais le temps et l'inspiration me manquent cruellement en ce moment... même si je suis certain que les choses vont vite rentrer dans l'ordre !
                    <br/>
                    Je pense même vous proposer des recettes un peu différentes... et plus légère! Affaire à suivre.
                    <br/>
                    <br/>
                    Quoiqu'il en soit, je ne vous remercierais jamais assez de votre fidélité et de votre sympathie.
                    <br/>
                    A très vite pour de nouvelles aventures culinaires... et avec un peu d'avance, très bonne année 2010 !
                    <br/>
                    Amicalement.
                    <br/>
                    <br/>
                    Yann

             VELOUTE DE COURGETTES ET FROMAGE FRAIS AU PIMENT D'ESPELETTE
            </a>
           </h2>
          </div>
          <div class="contenuArticle">
           <div class="ob-sections">
            <div class="ob-section ob-section-html">
             <p style="margin: 0cm 0cm 0pt;">
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   <img class="CtreTexte" height="225" src="http://idata.over-blog.com/1/92/85/67//Velout--Courgettes.jpg" width="300"/>
                   <br/>
                   <br/>
                   Préparation : 15 min
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   Cuisson : 10 min
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   <br/>
                   <br/>
                   Ingrédients : (pour 4 personnes)
                   <br/>
                  </span>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 1,5 kg de courgettes
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 3 grosses échalotes
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 300 g de pommes de terre
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 1 L de bouillon cube
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - 200 g de fromage frais type St Môret
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Piment d’Espelette
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Sel
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="color: #666699; font-family: Verdana;">
                 <span style="font-size: 12pt;">
                  <span style="font-size: 12pt;">
                   <br/>
                   <br/>
                   Préparation :
                  </span>
                  <br/>
                 </span>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Eplucher et découper grossièrement les courgettes et les pommes de terre puis émincer les échalotes.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Mettre le tout dans une grande cocotte et faire cuire une vingtaine de minutes avec un filet d’huile d’olive et le piment d’Espelette.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Une fois les légumes bien colorés, ajouter le bouillon et prolonger la cuisson encore 20min.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Verser enfin le tout dans le bol du blender, ajouter le fromage frais et mixer.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                - Rectifier l’assaisonnement et servir avec des petits croutons.
                <br/>
               </span>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                 <br/>
                 Avec l’arrivée du froid, rien de tel qu’une bonne petite "souplette".
                 <br/>
                </span>
               </em>
              </b>
              <b style="mso-bidi-font-weight: normal;">
               <em style="mso-bidi-font-style: normal;">
                <span style="font-size: 10pt; color: #666699; font-family: Verdana;">
                 Bien entendu, si le mélange est trop épais n’hésitez pas à allonger votre velouté avec un peu de bouillon supplémentaire.
                 <br/>
                 <b>
                  <span style="font-size: 10pt; color: #cc99ff; font-family: Verdana;">
                   <br/>
                   <br/>
                   "
                   <a href="http://saveursetcouleurs.over-blog.com/article-28535362.html">
                    <em>
                     <span style="color: #cc99ff; font-family: Verdana; text-decoration: none; text-underline: none;">
                      N'oubliez pas de vous inscrire à ma Newsletter, il y a de superbes livres de cuisine à gagner... et un gagnant tous les 25 inscrits !"
                     </span>
