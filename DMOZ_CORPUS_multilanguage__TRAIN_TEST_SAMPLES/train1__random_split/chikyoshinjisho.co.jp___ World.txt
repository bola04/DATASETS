     <p>
      <span class="red">
       地域密着30数年、幕張本郷を中心に幕張・新検見川・花見川区周辺地域・津田沼周辺
      </span>
      地域にて
      <span class="red">
       「外断熱・泡断熱工法」
      </span>
      による
      <span class="red">
       「魔法瓶の家」
      </span>
      を建売分譲・注文住宅で提供して来ました。賃貸・売買・設計・施工・管理まですべて自社・協力会社で行なう総合不動産ショップです。
      <span class="red">
       経験豊富なスタッフが土地探しから建物ま
      </span>
      で、皆様のご
      <span class="red">
       希望に合わせて提案
      </span>
      させて頂きます。必ず各人に「少しでも役立つ事を一つでもお持ち帰り頂きます」のでお気軽にご来店下さい（不在の時もありますので、連絡後の来社をお勧めします）。
     </p>
     <p>
      <span class="red">
       不動産は縁物
      </span>
      と昔から良く言われております。特に戸建は人間同士の
      <span class="red">
       結婚
      </span>
      と
      <span class="red">
       共通
      </span>
      する事も
      <span class="red">
       多い
      </span>
      です。どんなに良い人と人とのめぐり会いでも時期が早過ぎたり、遅過ぎたりすると縁が無い時が多いです。お客様の物件探しでも、
      <span class="red">
       最初に良い物件
      </span>
      （希望の8割以上クリア物件）
      <span class="red">
       とめぐり会ったとしても
      </span>
      、その時点で
      <span class="red">
       決断出来る状況に有る
      </span>
      か
      <span class="red">
       無い
      </span>
      かで、その物件との
      <span class="red">
       縁の有無が決まります。
      </span>
      接触する業者、担当者、見た物件に対する専門的な詳しい説明による「安心」「信頼できる」等の事が決断条件になると思われます。一生の内1〜2回位しか取引が無い事ですので不安も多いと思いますが、
      <span class="red">
       素直に本音で
      </span>
      お客様の
      <span class="red">
       希望と現況を伝えて頂く事が
      </span>
      希望物件との
      <span class="red">
       良縁に繋がる早道です。
      </span>
     </p>
     <p>
      ちなみに、ずっと家賃を払い続けて行きますと…
      <br/>
     </p>
     <div id="red_back">
      仮に月7万円と8万円の賃貸住宅に入居すると、10年で約970万円と約1100万円、35年で約3420万円と3860万円（2年に1回更新と火災保険加入、駐車場は月8000円で計算した場合）を家賃で支払う事になります。
     </div>
     <p>
      本当に
      <span class="red">
       家族（妻・子供）の事を思うなら
      </span>
      、家賃を払うか住宅ローンを払うかの違いですので、
      <span class="red">
       住宅ローンの時は万一の時
      </span>
      、又は8大疾病にかかった時には
      <span class="red">
       生命保険で住宅ローンがOになる（医師の診断書が出た後は生死にかかわらず）
      </span>
      ので、
      <span class="red">
       本人と家族は住宅ローンなしで住宅に住み続けられます
      </span>
      。賃貸の場合は、残された家族はそのまま住んでも他に移っても、家賃を払い続けなければなりません。現況と将来的な事を考えて、
      <span class="red">
       無理の無い支払い範囲内
      </span>
      であれば、購入の時は気が付いたら住宅口ーンが終わり、
      <span class="red">
       老後は資産として残ります。
      </span>
      その頃には子供も巣立っている時が多いと思われますので、老後の生活をエンジョイする方法も幾つか考えられます。金融機関評価の50〜60%は借入れて（死亡時清算）、海外旅行に行くも良し、趣味に使うも良し、ある時期が来たら処分して有料老人ホームに入るも良しと選択の幅が広がります。もちろん、死ぬまで住み続ける事も全て自分の考えで出来ます。
     </p>
