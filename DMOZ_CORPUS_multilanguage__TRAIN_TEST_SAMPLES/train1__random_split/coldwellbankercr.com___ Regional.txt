     // Tells if an element is completely visible. if the 2nd parm is true, it only looks at the top.
 
 	utils.isScrolledIntoView = function (elem, topOnly, container) {
 
 		var useWindow = false, docViewTop, docViewBottom, elemTop, elemBottom;
 
 		if (!container) {
 
 			container = window;
 
 			useWindow = true;
 
 		}
 
 		if (useWindow) {
 
 		   docViewTop = jQuery(container).scrollTop();
 
 		   elemTop = elem.offset().top;
 
 		} else {
 
 		   docViewTop = 0;
 
 		   elemTop = elem.position().top;
 
 		}
 
 		docViewBottom = docViewTop + jQuery(container).height();
 
 		elemBottom = elemTop + elem.height();
 
 		
 
 		if (topOnly) {
 
 		   return elemTop >= docViewTop && elemTop <= docViewBottom;
 
 		} else {
 
 		   return ((elemBottom >= docViewTop) && (elemTop <= docViewBottom));
 
 		}
 
     };

        <p>

 Despite its size and small number of inhabitants, Costa Rica is a considered medium to high- developed country.  It is among the 40% of countries designated with the highest development rate.
         <br/>
         The Happy Planet Index recently rated Costa Rica as the Greenest and Happiest country on the planet. Anyone who has travelled to Costa Rica can attest to the warm and friendly people also known as “Ticos”.          </p>

          Playas Del Coco serves all these communities offering all possible services: 4 super markets, 3 pharmacies, 4 banks, 3 medical clinics, dental clinics, private ambulance service, laboratory, appliance stores, home center, police dept., post office, port captain, artisan shops, casino, mechanics, international and local cuisine, artisan shops and most important a beach front park.  The Coco Bay is calm making it possible to enjoy kayaking, paddle boarding, snorkeling, swimming or bench sitting watching a basketball or volleyball game or sipping a cold drink in one of the beach front colorful bars, restaurants.  Though Coco has all these services to offer, “she” hasn’t lost her ambience as a charming fishing village.
