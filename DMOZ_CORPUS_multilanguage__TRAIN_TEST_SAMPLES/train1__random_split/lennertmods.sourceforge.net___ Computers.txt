           <h2 class="title" style="clear: both">
            <a name="d4e30">
            </a>
            Enhancing and Extending phpBB!
           </h2>
          </div>
         </div>
         <div>
         </div>
        </div>
        <p>
         Welcome to Lennert's phpBB Modifications, an Open Source project dedicated to enhancing and
             extending the
         <a href="http://www.phpbb.com" target="_top">
          phpBB
         </a>
         forum software.  This project aims to
             serve several audiences in the phpBB community: administrators, developers, and end users.  For
             phpBB administrators and developers, we offer configurable and easy to install mods that extend
             phpBB's capabilities, functionality, and appearance.  For end users, we hope to
             enhance the already exceptional user experience created by phpBB.
        </p>
        <p>
         Be sure to visit the
         <a href="news.html">
          News
         </a>
         page for updates concerning mod progress and new features being added to existing mods.
        </p>
