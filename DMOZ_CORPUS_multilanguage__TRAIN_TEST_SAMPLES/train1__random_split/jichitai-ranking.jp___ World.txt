       <h1>
        自治体ランキング
       </h1>
       <p class="topTEXT">
        このサイトは、
        <a href="http://www.soumu.go.jp/">
         総務省
        </a>
        が公表している、
        <a href="http://www.soumu.go.jp/iken/zaisei/card.html">
         『決算カード』
        </a>
        （＊１）をデータ化し、 全国の市区町村の決算状況をランキングしています。
        <br/>
        財政分析を行うため『比較する』という事を行っています。
        <br/>
       </p>
       <p class="topTEXT">
        （※１）決算カードとは、総務省が実施している地方財政状況調査（決算統計）の集計結果に基づいて、
 		各都道府県・市区町村の各種決算状況を１枚のカードにまとめたものです。
       </p>
       <p class="topTEXT">
        具体的には、
       </p>
       <ul>
        <li>
         『経年的に比較する』・・・
         <a href="chart.html">
          決算額推移
         </a>
         で比較
        </li>
        <li>
         『近隣の他団体と比較する』・・・
         <a href="menue.html">
          決算ランキング
         </a>
         で、地図を使って比較
        </li>
        <li>
         『似ている団体と比較する』・・・
         <a href="menue.html">
          決算ランキング
         </a>
         で、
         <a href="st_shichouson_ruikei.html">
          市町村類型
         </a>
         を使用して比較
        </li>
       </ul>
       という内容で、比較を行います。
       <br/>
       <br/>
       <p class="topTEXT">
        財政再建団体になった夕張市、日毎に増えていく生活保護費、人口の推移など地方自治体の現状の財政状況がどのような状況か、自分の住んでいる街の状況を少しでも分かって頂ければと思います。
        <br/>
        ※データの検証は行なっていますが、内容の正確性は保証できません。データを使用する場合は、自己責任でお願いします。
       </p>
