          </h6>

          </h5>
          <p>

          </p>
          <p>
           Entrando en el siglo XXI Jobesa ha realizado importantes mejoras en su enseñanza, adaptándose a los requisitos de la comunidad Europea. El centro está totalmente preparado y acondicionado para impartir enseñanzas a un alto nivel tanto técnico como humano, equiparable a cualquier país de la comunidad Europea.

         <h3>
          TÉCNICO SUPERIOR EN HIGIENE BUCODENTAL EN JOBESA GRANADA
         </h3>
         <div class="inicio-img">

         <p>
          En
          <strong style="color:#d50c00">
           Jobesa Granada
          </strong>
          , ponemos a su disposición el Ciclo Formativo de
          <strong style="color:#d50c00">
           Técnico Superior en Higiene Bucodental
          </strong>
          . Con instalaciones y maquinarias de última generación, algo fundamental para el futuro desarrollo del Higienista Dental. Somos un centro Homologado por la Junta de Andalucía con más de 30 años formando a Profesionales.
          <br/>
          El Curso incluye todo el material de prácticas.
          <br/>
          Todos los alumnos/as recibirán una docencia ágil y basada en unas prácticas desde un principio con Pacientes Reales, ya que contamos con Gabinetes Dentales Propios para poder facilitar las prácticas a todos los alumnos/as de este Ciclo Formativo y apostando por la futura inserción laboral una vez terminado este.

 function pe2_setup_photoswipe(collection){
 	// check to make sure our collection has records
 	if(collection.length == 0){
 		// nothing to do
 		return;
 	}
 
 	// otherwise, setup photoswipe
 	var collection_counter = 0;
 	var myPhotoSwipe = collection.photoSwipe({
 		// enable settings of photoswipe
 		enableMouseWheel: true,
 		enableKeyboard: true,
 		captionAndToolbarAutoHideDelay: 0,
 		imageScaleMethod: 'fitNoUpscale',
 		// set the caption from the A tag's title attribute
 		getImageCaption: function(item){
 			// increment our image counter
 			collection_counter++;
 
 			// create the caption
 			var caption = document.createElement('span');
 			caption.appendChild(document.createTextNode(jQuery(item).attr('title')));
 
 			// append a br
 			caption.appendChild(document.createElement('br'));
 
 			// define our second row separator
 			var second_row_separator = false;
 
 			// create the second row container
 			var second_row = document.createElement('span');
 			second_row.setAttribute('style', 'color: #BBB;');
 
 			// create the "Image X of X"
 			var num = document.createElement('span');
 			num.appendChild(document.createTextNode('Image ' + collection_counter + ' of ' + collection.length));
 			num.setAttribute('style', 'margin-right: 10px; margin-left: 10px;');
 			second_row.appendChild(num);
 			second_row_separator = true;
 			// check to see if we need to output the separator
 			if(second_row_separator){
