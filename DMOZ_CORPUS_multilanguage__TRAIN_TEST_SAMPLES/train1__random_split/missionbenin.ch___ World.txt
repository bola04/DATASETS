          <h1 class="big-title">
           Mot de l'Ambassadeur
          </h1>
          <div class="space" style="clear:both; height:50px;">
          </div>
          <p>
           <img class="alignleft wp-image-804 size-full" sizes="(max-width: 300px) 100vw, 300px" src="http://missionbenin.ch/wp-content/uploads/2016/04/S-E-M-Eloi-LAOUROU.jpg" srcset="http://missionbenin.ch/wp-content/uploads/2016/04/S-E-M-Eloi-LAOUROU.jpg 300w, http://missionbenin.ch/wp-content/uploads/2016/04/S-E-M-Eloi-LAOUROU-251x300.jpg 251w, http://missionbenin.ch/wp-content/uploads/2016/04/S-E-M-Eloi-LAOUROU-225x269.jpg 225w" width="300"/>
           Chers compatriotes, chers visiteurs,
          </p>
          <p>
           Il me plaît, au nom de tout le personnel et en mon nom propre, de vous souhaiter la bienvenue sur le site internet de l’Ambassade/Mission Permanente du Bénin auprès de l’Office des Nations Unies et des autres Organisations Internationales basées à Genève et à Vienne.
          </p>
          <p>
           Les activités de l’Ambassade/Mission Permanente, ouverte en novembre 2000, concernent aussi bien la coopération bilatérale que multilatérale.
          </p>
          <p>
           Au plan bilatéral, l’Ambassade du Bénin à Genève a juridiction sur les six pays ci-après : la Suisse, l’Autriche, la Bulgarie, la Croatie, la Hongrie, le Liechtenstein, et la Slovénie, et a pour missions la promotion et le renforcement des relations entre le Bénin et ces pays, d’une part, ainsi que la protection des intérêts du Bénin et de ses ressortissants résidant dans ces pays, d’autre part.
          </p>
          <p>
           Au plan multilatéral, la Mission Permanente du Bénin auprès de l’Office des Nations Unies et des autres Organisations internationales basées à Genève et à Vienne constitue l’instrument d’animation de la coopération multilatérale avec les Organisations et Institutions spécialisées du Système des Nations Unies ainsi qu’avec les Institutions du système commercial multilatéral, notamment l’Organisation Mondiale du Commerce (OMC), la Conférence des Nations Unies sur le Commerce et le Développement (CNUCED), le Centre du Commerce International (CCI).
          </p>
          <p>
           Le présent site a pour ambitions de faciliter la communication et les relations entre l’Ambassade/Mission Permanente du Bénin à Genève et ses usagers et de rendre visibles ses activités.
          </p>
          <p>
           Il se veut une fenêtre sur le Bénin, pays épris de paix et de justice, pionnier des conférences nationales en Afrique dont les filles et fils s’emploient, chaque jour davantage, à la consolidation des acquis démocratiques, à la promotion de la paix, du dialogue, et à l’édification d’une Nation émergente par la création de richesses aux fins d’une prospérité partagée.
          </p>
          <p>
           Plusieurs rubriques sont proposées sur ce site, notamment:
          </p>
          <ul class="list-disq">
           <li>
            les informations utiles sur l’Ambassade/Mission Permanente : localisation, heures d’ouverture, pays de juridiction, etc.
           </li>
           <li>
            les prestations consulaires offertes aux usagers ;
           </li>
           <li>
            les attractions touristiques du Bénin, pays ayant une histoire et une culture multiséculaires.
           </li>
          </ul>
          <div class="space" style="clear:both; height:30px;">
          </div>
          <p>
           Cordialement
          </p>
          <p>
           Eloi LAOUROU
           <br/>
           Ambassadeur / Réprésentant Permanent
          </p>

    var recaptchaWidgets = [];
 var recaptchaCallback = function() {
 	var forms = document.getElementsByTagName( 'form' );
 	var pattern = /(^|\s)g-recaptcha(\s|$)/;
 
 	for ( var i = 0; i < forms.length; i++ ) {
 		var divs = forms[ i ].getElementsByTagName( 'div' );
 
 		for ( var j = 0; j < divs.length; j++ ) {
 			var sitekey = divs[ j ].getAttribute( 'data-sitekey' );
 
 			if ( divs[ j ].className && divs[ j ].className.match( pattern ) && sitekey ) {
 				var params = {
 					'sitekey': sitekey,
 					'type': divs[ j ].getAttribute( 'data-type' ),
 					'size': divs[ j ].getAttribute( 'data-size' ),
 					'theme': divs[ j ].getAttribute( 'data-theme' ),
 					'badge': divs[ j ].getAttribute( 'data-badge' ),
 					'tabindex': divs[ j ].getAttribute( 'data-tabindex' )
 				};
 
 				var callback = divs[ j ].getAttribute( 'data-callback' );
 
 				if ( callback && 'function' == typeof window[ callback ] ) {
 					params[ 'callback' ] = window[ callback ];
 				}
 
 				var expired_callback = divs[ j ].getAttribute( 'data-expired-callback' );
 
 				if ( expired_callback && 'function' == typeof window[ expired_callback ] ) {
 					params[ 'expired-callback' ] = window[ expired_callback ];
 				}
 
 				var widget_id = grecaptcha.render( divs[ j ], params );
 				recaptchaWidgets.push( widget_id );
 				break;
 			}
 		}
 	}
 };
 
 document.addEventListener( 'wpcf7submit', function( event ) {
 	switch ( event.detail.status ) {
 		case 'spam':
 		case 'mail_sent':
 		case 'mail_failed':
 			for ( var i = 0; i < recaptchaWidgets.length; i++ ) {
 				grecaptcha.reset( recaptchaWidgets[ i ] );
 			}
 	}
 }, false );
   </script>
