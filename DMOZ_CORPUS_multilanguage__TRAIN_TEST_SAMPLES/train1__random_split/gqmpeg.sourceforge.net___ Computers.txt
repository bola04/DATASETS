      <p>
       Welcome!
      </p>
      <p>
       This is the homepage of GQmpeg and GQradio. GQmpeg is a front-end to various audio players, including mpg123. It includes playlists, skins (themes), and many other options. Please see the
       <a href="mpeg-over.html">
        Overview
       </a>
       page for details. GQradio is an easy to use FM tuner application for radio cards supported by the video4linux interface.
      </p>
      <br/>
      Current stable release:
      <blockquote>
       <a href="mpeg-down.html">
        GQmpeg 0.20.0
       </a>
       - February 2, 2003
       <br/>
       <a href="radio.html">
        GQradio 1.0
       </a>
       - March 17, 2004
      </blockquote>
      Current
      <i>
       development
      </i>
      release:
      <blockquote>
       <a href="mpeg-down.html">
        GQmpeg 0.91.1
       </a>
       - April 7, 2004
       <br/>
       <a href="radio.html">
        GQradio 1.9.2
       </a>
       - February 24, 2005
       <br/>
      </blockquote>
      No longer maintained at this time:
      <blockquote>
       <a href="gqoob.html">
        GQoob 0.3.0
       </a>
       - September 4, 2002
      </blockquote>

        <td bgcolor="#D2E8FF" valign="top">
         <b>
          GQradio 1.9.2
         </b>
         (devel)
         <p>
          GQradio now requires GTK+ 2.4.0. This release adds a volume boost option for tuner drivers that initialize the radio card with a low initial volume. It also includes many updates, and fixes text size and window placement bugs. The source package and a Fedora Core 3 RPM are available for download.
         </p>
         <p>
          <a href="radio.html">
           Download GQradio 1.9.2
          </a>
         </p>
         <br/>
         <p>
          In addition, slik 0.92.0 was released. If anyone is interested in the skin engine of GQmpeg and GQradio the slik package is available on the
          <a href="mpeg-down.html">
           GQmpeg download
          </a>
          page.
         </p>

         <b>
          GQmpeg 0.91.0 released
         </b>
         (devel)
         <p>
          This release includes more updates for GTK 2, and fixes slow playlist loading with the playlist window open. Please refer to the
          <a href="mpeg-hist.html">
           History page
          </a>
          for the complete list of enhancements. Note that the color shift option has changed, upgraders that use this option will notice that they may need to adjust the RGB values. The SLIK skin engine has also been updated to 0.91.0.
