           <td>
            This is the sourceforge site for Infobot, a daemon module in Perl.
     It is currently the official home for Infobot. The
            <a href="http://infobot.org">
             old home
            </a>
            only has the old releases.
     Here you will find the current
            <a href="http://sourceforge.net/svn/?group_id=2241">
             Subversion
            </a>
            Tree
     which you can
            <a href="http://infobot.svn.sourceforge.net/viewvc/infobot/">
             Browse
            </a>
            .
     You might want to browse the
            <a href="http://infobot.wiki.sourceforge.net/">
             wiki
            </a>
            to see if anything comes of it.
     Check the
            <a href="http://infobot.sourceforge.net/wiki/Special:Recentchanges">
             RecentChanges
            </a>
            for example.
