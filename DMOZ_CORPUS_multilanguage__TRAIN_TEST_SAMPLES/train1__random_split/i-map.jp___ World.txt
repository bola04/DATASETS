         var x1 = x2 = x3 = 0;
         var y1 = y2 = y3 = 0;
         if (document.documentElement) {
           x1 = document.documentElement.scrollLeft || 0;
           y1 = document.documentElement.scrollTop || 0;
         }
         if (document.body) {
           x2 = document.body.scrollLeft || 0;
           y2 = document.body.scrollTop || 0;
         }
         x3 = window.scrollX || 0;
         y3 = window.scrollY || 0;
         var x = Math.max(x1, Math.max(x2, x3));
         var y = Math.max(y1, Math.max(y2, y3));

    </h3>
    <ul>
     <li>
      選択肢は「○」：整備あり、「×」：整備なし、「整備不要・代替物有」、「非該当・無回答」となっています。
      <br/>

     </li>
     <li>
      垂直移動（エレベーター等）
      <br/>
      →１階にある施設は、エレベーター等の設備を整備する必要がないので「整備不要・代替物有」としました。
     </li>
     <li>
      駐車場
      <br/>
      →駐車場がない施設における「係員の有無」は「非該当」としました。（設備がないため）
     </li>
     <li>
      トイレ
      <br/>
      →トイレがない施設における「オストメイト対応」、「ベビーベッド」、「出入口の段差解消」等の小項目は「非該当」としました。（設備がないため）
     </li>
     <li>
      バリアフリー協力店
      <br/>
      →施設名称の後ろに「（バリアフリー協力店）」と記載しています。
      <br/>
      →「バリアフリー協力店」とは、高齢者も障害者も、車椅子で出かけたり、手話で会話をしたりと、その人にあった生活を工夫していることを理解し、お客様への気配りや対応がやさしい誰もが安心して利用できるお店のことです。
      <br/>
      →一定要件を満たす区内の店舗から申請をいただき、要件に該当する店舗に「バリアフリー協力店」のステッカーを配布し、店頭などへ貼付します。まちなかで、このステッカーを貼ったお店を見かけたときは、安心してご利用ください。
      <br/>
     </li>
    </ul>
