           <p>
            <strong>
             This is how the Web works!
            </strong>
            Be on the web frontiers with Smalltalk, try Aida/Web!
           </p>
           <p>
            Aida is a Smalltalk web application framework for building complex web applications with latest seamlesly integrated HTML5 technologies for today's real-time web. MVC, REST, Mobile, WebSocket,  you name it!
           </p>
           <p>
            Start with
            <a href="http://www.youtube.com/user/AidaWebSmalltalk">
             our video tutorials
            </a>
            , this is a first one:

             <span>
              nov15
             </span>
             <strong>
              Aida on Squeak 5
             </strong>
             Squeak 5 together with a
             <a href="http://squeak.org">
              shiny new website
             </a>
             was released this summer so it was a time to prepare Aida for this release as well. On
             <a href="http://www.aidaweb.si/download">
              Download page
             </a>
             you can find a link to All-in-One package to run Aida 6.6 on latest Squeak 5 for Linux, Mac and Win.
            </li>
            <li>
             <span>
              sep14
             </span>
             <strong>
              Aida 6.7 released
             </strong>
             Mostly a maintenance release, on VisualWorks only, with minor improvements and quite some paches, Bootstrap CSS framework upgraded, ..
             <p class="more">
              <a href="http://www.aidaweb.si/release-notes-6.7">
               see release notes
              </a>
             </p>
            </li>
            <li>
             <span>
              aug13
             </span>
             <strong>
              Aida 6.6 released
             </strong>
             Optimizations on HTTP level: HTML streaming, GZIP compression. CSS sprites, internal IP detection for access control, improved Geolocation support,
 JS anc CSS libraries like jQuery and Bootstrap upgraded, ..
             <p class="more">
              <a href="http://www.aidaweb.si/release-notes-6.6">
               see release notes
              </a>
             </p>
            </li>
            <li>
             <span>
              feb13
             </span>
             <strong>
              Aida now twice speedier
             </strong>
             Due to the HTTP level optimizations like gzip compression and streaming of HTML content this Aida based website achieves below 250ms to start rendering time
             <p class="more">
              <a href="http://forum.world.st/HTTP-optimizations-streaming-everything-gzip-compression-tt4667752.html">
               more
              </a>
             </p>
            </li>
            <li>
             <span>
              nov12
             </span>
             <strong>
              Aida 6.5 interim with Realtime (WebSocket based) support released
             </strong>
             It can be safely claimed that Aida/Web is a first Smalltalk web
 framework, even more, one of the first frameworks at all to integrate HTML5 WebSockets for so called realtime web apps.
             <p class="more">
              <a href="http://forum.world.st/Aida-with-Realtime-WebSockets-support-released-tt4653510.html">
               see announcement
              </a>
             </p>
            </li>
            <li>
             <span>

             <p class="more">
              <a href="http://forum.world.st/ANN-Aida-6-5-Summer-released-more-Mobile-more-Cloud-more-REST-tt4645407.html">
               see announcement
              </a>
             </p>
