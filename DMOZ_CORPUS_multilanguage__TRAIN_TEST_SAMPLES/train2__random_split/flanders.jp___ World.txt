                     <h3 class="post-title entry-title" itemprop="name">
                      <a href="http://www.flanders.jp/2018/04/eu-film-days-2018.html">
                       EU Film Days 2018!　東京・京都
                      </a>
                     </h3>
                     <div class="post-header">
                      <div class="post-header-line-1">
                      </div>
                     </div>
                     <div class="post-body entry-content" id="post-body-7553238584151330354" itemprop="articleBody">
                      <div class="separator" style="clear: both; text-align: center;">
                      </div>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="https://2.bp.blogspot.com/-E9gfNiD4ihw/WuAAt2n5vFI/AAAAAAAAAHM/k_7Kq3eqq_IdwAeqbeFXKn-JUR9Tdr3wwCLcBGAs/s1600/BELGICA_01_%25C2%25A9Thomas_Dhanens.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" data-original-height="1067" data-original-width="1600" height="213" src="https://2.bp.blogspot.com/-E9gfNiD4ihw/WuAAt2n5vFI/AAAAAAAAAHM/k_7Kq3eqq_IdwAeqbeFXKn-JUR9Tdr3wwCLcBGAs/s320/BELGICA_01_%25C2%25A9Thomas_Dhanens.jpg" width="320"/>
                       </a>
                      </div>
                      <div class="separator" style="clear: both; text-align: center;">
                       <span style="font-size: x-small;">
                        ©Thomas Dhanens
                       </span>
                      </div>
                      <br/>
                      <span style="font-size: large;">
                       EU Flim Days 2018が東京・京都・広島3カ所で開催されます！
                      </span>
                      <br/>
                      <br/>
                      欧州連合（EU）加盟国が選りすぐりの作品を紹介する映画祭。
                      <br/>
                      スクリーンの中で多様な文化を体験し、新たな才能を発見する。
                      <br/>
                      ユニークで新しいヨーロッパ映画旅行へ出発！
                      <br/>
                      <br/>
                      ベルギー・フランダースからは「Belgica （ベルヒカ）」をお楽しみいただきます。
                      <br/>
                      皆様のご来場を心よりお待ち申し上げます。
                      <br/>
                      <br/>
                      「Belgica （ベルヒカ）」上映スケジュール
                      <br/>
                      （東京）会場：
                      <a href="http://www.nfaj.go.jp/visit/information_map/">
                       国立映画アーカイブ（旧、東京国立近代美術館フィルムセンター）
                      </a>
                      <br/>
                      5月27日（日）13：15
                      <br/>
                      5月31日（木）19：00
                      <br/>
                      <br/>
                      （京都）会場：
                      <a href="http://www.bunpaku.or.jp/info/access/">
                       京都府京都文化博物館
                      </a>
                      <br/>
                      6月14日（木）13：30
                      <br/>
                      6月16日（土）17：00
                      <br/>
                      <br/>
                      （詳細）
                      <br/>
                      監督：フェリックス・ヴァン・ヒュルーニンゲン
                      <br/>
                      上映時間：126分
                      <br/>
                      制作：ベルギー
                      <br/>
                      言語：オランダ語（日本語・英語字幕）
                      <br/>
                      <br/>
                      <a href="https://eufilmdays.jp/films/2018/belgica/">
                       「Belgica（ベルヒカ）」
                      </a>
                      ←詳細は左記をクリック！
                      <br/>
                      <span style="background-color: white; font-family: Lato, sans-serif; font-size: 14px;">
                       カフェを改装しナイトクラブをオープンさせた兄弟。やがて町の一番の人気ス ポットとなったそのナイトクラブの存在が、2人の絆に思わぬ影響を落とすことに。『あきれた日常』（2009）、『オーバー・ザ・ブルースカイ』（2012）（いずれもEUフィルムデーズで上映）に続くヴァン・ヒュルーニンゲン監督作品。サンダンス映画祭ワールド・シネマ・ドラマ部門監督賞受賞。
                      </span>
                      <br/>
                      <span style="background-color: white; font-family: Lato, sans-serif; font-size: 14px;">
                       <br/>
                      </span>
                      <span style="background-color: white; font-family: Lato, sans-serif; font-size: 14px;">
                       その他EU各国から出品された作品詳細・スケジュールについては下記リンクをご覧ください！
                      </span>
                      <br/>
                      <span style="font-family: Lato, sans-serif;">
                       <span style="background-color: white; font-size: 14px;">
                        <a href="https://eufilmdays.jp/">
                         EU Flim Days 2018
                        </a>
                       </span>
                      </span>

                     </a>
                     <h3 class="post-title entry-title" itemprop="name">
                      <a href="http://www.flanders.jp/2018/04/in2018.html">
                       ツール・ド・フランダースin大阪2018への ご参加、　　　　ご支援ありがとうございました！
                      </a>
                     </h3>
                     <div class="post-header">
                      <div class="post-header-line-1">
                      </div>
                     </div>
                     <div class="post-body entry-content" id="post-body-1610219152517499367" itemprop="articleBody">
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="https://3.bp.blogspot.com/-uvSOAayBwC8/WtQ_z-US3RI/AAAAAAAAAG4/vaZ7d6O1mHw1Zk0WoCAtYb4hg3XMfUT3wCLcBGAs/s1600/004.JPG" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" data-original-height="1067" data-original-width="1600" height="213" src="https://3.bp.blogspot.com/-uvSOAayBwC8/WtQ_z-US3RI/AAAAAAAAAG4/vaZ7d6O1mHw1Zk0WoCAtYb4hg3XMfUT3wCLcBGAs/s320/004.JPG" width="320"/>
                       </a>
                      </div>
                      <br/>
                      <br/>
                      <div class="MsoNormal">
                       <span style="font-family: Arial, Helvetica, sans-serif;">
                        <span lang="EN-US">
                         2018
                        </span>
                        <span style="mso-ansi-language: JA;">
                         年
                        </span>
                        <span lang="EN-US">
                         3
                        </span>
                        <span style="mso-ansi-language: JA;">
                         月
                        </span>
                        <span lang="EN-US">
                         25
                        </span>
                        <span style="mso-ansi-language: JA;">
                         日（日）、「ツール・ド・フランダース
                        </span>
                        <span lang="EN-US">
                         in
                        </span>
                        <span style="mso-ansi-language: JA;">
                         大阪
                        </span>
                        <span lang="EN-US">
                         2018
                        </span>
                        <span style="mso-ansi-language: JA;">
                         」を無事に開催することができました。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <br/>
                      </div>
                      <div class="MsoNormal">
                       <span style="font-family: Arial, Helvetica, sans-serif;">
                        <span style="mso-ansi-language: JA;">
                         当日はお天気にも恵まれ、またこの日に合わせるように例年よりも早い桜の開花という絶好のタイミングで
                        </span>
                        <span lang="EN-US">
                         125
                        </span>
                        <span style="mso-ansi-language: JA;">
                         名の方にご参加いただく事ができました。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <br/>
                      </div>
                      <div class="MsoNormal">
                       <span style="font-family: Arial, Helvetica, sans-serif;">
                        <span style="mso-ansi-language: JA;">
                         自慢の愛車で大阪の名所をのんびりと巡っていただきながら歴史や文化の香りを感じ、人とのふれあいを通して大人から子どもまでサイクリングを楽しむ
                        </span>
                        <span lang="EN-US">
                         5km
                        </span>
                        <span style="mso-ansi-language: JA;">
                         のファミリーコースに加え、今大会は大阪市天王寺区と和歌山県九度山を往復する
                        </span>
                        <span lang="EN-US">
                         130km
                        </span>
                        <span style="mso-ansi-language: JA;">
                         のアドバンストコースを設け、自転車を愛するより多くの方々との出会いの場を実現することができました。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <br/>
                      </div>
                      <div class="MsoNormal">
                       <span style="mso-ansi-language: JA;">
                        <span style="font-family: Arial, Helvetica, sans-serif;">
                         単なる移動手段ではなく、「生活をより豊かに楽しむための自転車」をコンセプトとするこのイベントに共感いただき、そして格別のご支援とご協力をくださいましたこと、誠にありがたく、ここに心からの感謝の気持ちをお伝え申し上げます。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <br/>
                      </div>
                      <div class="MsoNormal">
                       <span style="mso-ansi-language: JA;">
                        <span style="font-family: Arial, Helvetica, sans-serif;">
                         「ツール・ド・フランダースin大阪」はこれからも大阪の春の名物イベントとなることを目指し、来年へのペダルをこぎ出そうと思っております。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <span style="mso-ansi-language: JA;">
                        <span style="font-family: Arial, Helvetica, sans-serif;">
                         今後とも一層の応援をよろしくお願い申し上げます。
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <br/>
                      </div>
                      <div class="MsoNormal">
                       <span style="mso-ansi-language: JA;">
                        <span style="font-family: Arial, Helvetica, sans-serif;">
                         来年もまたお会いできることを楽しみにしています！！
                         <o:p>
                         </o:p>
                        </span>
                       </span>
                      </div>
                      <div class="MsoNormal">
                       <span style="font-family: Arial, Helvetica, sans-serif;">

                     <h3 class="post-title entry-title" itemprop="name">
                      <a href="http://www.flanders.jp/2018/04/2018421.html">
                       第２回たかまつ国際古楽祭　2018年4月21日（土）
                      </a>
                     </h3>
                     <div class="post-header">
                      <div class="post-header-line-1">
                      </div>
                     </div>
                     <div class="post-body entry-content" id="post-body-4315853193816375348" itemprop="articleBody">
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="https://3.bp.blogspot.com/-44KYO1qRLOc/WsxLKephURI/AAAAAAAAAGg/-XfLLGZtG5kx6hDc_BUj4GUWlPsfVPYkwCLcBGAs/s1600/MAT2018-F-A4-check-20180322-page-001.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                        <img border="0" data-original-height="1600" data-original-width="1132" height="320" src="https://3.bp.blogspot.com/-44KYO1qRLOc/WsxLKephURI/AAAAAAAAAGg/-XfLLGZtG5kx6hDc_BUj4GUWlPsfVPYkwCLcBGAs/s320/MAT2018-F-A4-check-20180322-page-001.jpg" width="226"/>
                       </a>
                      </div>
                      ４月21日（土）第２回たかまつ国際古楽祭が開催されます！
                      <br/>
                      クラシックの一大ジャンルである「古楽」を高松や瀬戸内の皆さんにご紹介し、ヨーロッパの著名な演奏家達をゲストとし、芸術活動を通し文化交流を図るイベントとして企画されました。また、海外ゲストには高松市や瀬戸内の文化に触れていただく、高松・瀬戸内と世界を繋ぐイベントとして活動されています。
                      <br/>
                      <br/>
                      今回のゲストは世界を代表する古楽アンサンブル
                      <br/>
                      「イル・ガルデリーノ」。
                      <br/>
                      古楽とは当時の楽器を使いその音楽が書かれた当時の
                      <br/>
                      演奏法を理解し演奏するもので、欧米でクラシックのジャンルとして注目を浴びています。
                      <br/>
                      ベルギーを代表する素晴らしい古楽演奏者達の演奏を
                      <br/>
                      お楽しみください。
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="https://4.bp.blogspot.com/-LwB6RkaWag4/WsxLKXrEnzI/AAAAAAAAAGs/cg7Zypo0_DAyHxnjqMLOsnuwiwDsu4MCACEwYBhgL/s1600/MAT2018-F-A4-check-20180322-page-002.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                        <img border="0" data-original-height="1600" data-original-width="1132" height="320" src="https://4.bp.blogspot.com/-LwB6RkaWag4/WsxLKXrEnzI/AAAAAAAAAGs/cg7Zypo0_DAyHxnjqMLOsnuwiwDsu4MCACEwYBhgL/s320/MAT2018-F-A4-check-20180322-page-002.jpg" width="226"/>
                       </a>
                       <a href="https://4.bp.blogspot.com/-LwB6RkaWag4/WsxLKXrEnzI/AAAAAAAAAGs/cg7Zypo0_DAyHxnjqMLOsnuwiwDsu4MCACEwYBhgL/s1600/MAT2018-F-A4-check-20180322-page-002.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                        <br/>
                       </a>
                       <a href="https://4.bp.blogspot.com/-LwB6RkaWag4/WsxLKXrEnzI/AAAAAAAAAGs/cg7Zypo0_DAyHxnjqMLOsnuwiwDsu4MCACEwYBhgL/s1600/MAT2018-F-A4-check-20180322-page-002.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                        <br/>
                       </a>
                      </div>
                      日時：2018年4月21日（土）
                      <br/>
                      プログラムA　14：00開演
                      <br/>
                      プログラムB　17：00開演
                      <br/>
                      <br/>
                      チケット：一日通し券・プログラムA・プログラムB
                      <br/>
                      <br/>
                      会場：
                      <a href="http://www.sunport-hall.jp/access/">
                       サンポートホール高松
                      </a>
                      <br/>
                      香川県高松市サンポート2-1
                      <br/>
                      <br/>
                      お申し込み・詳細は下記HPをご覧ください。
                      <br/>
                      <a href="https://www.mafestivaltakamatsu.com/#home">
                       たかまつ国際古楽祭実行委員会
                      </a>
                      <br/>
                      電話：080-5665-7050
