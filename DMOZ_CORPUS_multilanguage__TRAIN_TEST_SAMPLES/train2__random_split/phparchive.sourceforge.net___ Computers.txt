            <p>
             phparchive was born around October 2001 when I begun to write my own
 					code to build a simple intranet application. Some month after I changed
 					my target and the new one was to create an application to manage contents
 					and relations between them, that use only open source technologies.
 					This is now the result of my free time spent to follow this killer
 					application.
            </p>

 					to read CREDITS file to know from where comes the things. The TODO file

            </p>

 					comes without any warranty and without technical support, so please if you
