               <h5>
                <br/>
                <br/>
                千城台動物病院は、千葉県千葉市若葉区千城台北にて、主に犬・猫を診療している動物病院です。
                <br/>
                <br/>
                <br/>
                ご連絡をいただいて、ペットの症状により、電話予約にて往診もしています。ペットは言葉が話せないため、飼い主が日常を観察し、管理してあげなければいけません。その為、ある程度のしつけや、ペットの病気など最低限の知識を知っておかないと、あとあとペットたちがかわいそうなおもいをする事があります。
                <br/>
                <br/>
                <br/>
                千城台動物病院のホームページでは、ペットの病気や予防についてのページが設けられていますので、事前に気づいてあげられる事、予防という意味でも、このホームページを多いに活用してください。他にも、当動物病院患者さん達の協力により、動物病院ならではの、みみよりなペットにまつわる情報など、お楽しみも盛りだくさんです。是非、この千城台動物病院のページを御覧いただき少しでも皆さんのペット達の為になれば、と思っています。
                <br/>
                <br/>
                <br/>
                今後も随時ホームページの内容を充実させていく予定ですので、たくさんの人に見ていただき、お気軽にご意見ご感想をいただきたいと思っています。
               </h5>
