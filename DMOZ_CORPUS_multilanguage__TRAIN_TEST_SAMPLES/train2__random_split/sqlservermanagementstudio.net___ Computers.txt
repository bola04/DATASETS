                       Yes, you hear it right, you can easily setup ads.txt file in your blogger.com blog or blogspot blog.
                       <br/>
                       <b>
                        QandA about ads.txt?
                       </b>
                       <br/>
                       If you come across with error in your adsense account with error message below and  if you are  monetizing on you blog using adsense.
                       <br/>
                       <div style="text-align: left;">
                        <b>
                         1)
                         <span style="white-space: pre;">
                         </span>
                         <span style="background-color: #999999;">

                         </span>
                        </b>
                       </div>
                       <div style="text-align: left;">
                        <span style="background-color: #999999;">
                         <b>
                          2) How to add ads.txt file on blogger.com
                         </b>
                        </span>
                       </div>
                       <div style="text-align: left;">
                        <br/>
                       </div>
                       <h2 style="text-align: left;">
                        Adding ads.txt file on blogger.com:
                       </h2>
                       It would be very difficult to find answer of the question above about adsense monetize publisher id error - if you visit many forums so you can't find proper answer for this:
                       <br/>
                       you may find such bogus answers adsense does not require publisher to use this id on site, actually this information not complete just half information for the answer you are looking for.
                       <br/>
                       somebody would say add ads.txt file in your domain hosting area.
                       <br/>
                       somebody would say you are using custom domain so this is not possible in blogger platform.
                       <br/>
                       <br/>
                       Normally if you have taken domain and space then you can easily add this file on your domain hosted platform, but if this is not the case if you are using blogger.com blog or blogspot blog.
                       <br/>
                       So when problems come, if you have  hosted custom domain and using space on blogger.com’s blog or blogspot blog.
                       <br/>
                       So here you will learn how you can fix and allow blogspot to use ads.txt file which contains your publisher ID so adsense team can recognize you as authorized partner and advertize on your blog and and you can safe  your earning from adsense and if you are unable to fix this issue ASAP then your revenue will be zero otherwise adsense ID may have some serious issues.
                       <br/>
                       <br/>
                       <h2 style="text-align: left;">
                        Creating ads.txt on blogger.com:
                       </h2>
                       Point your location to
                       <b>
                        blogger.com
                       </b>
                       <br/>
                       1) login with your
                       <b>
                        credential
                       </b>
                       where your blog is hosted with custom domain.
                       <br/>
                       2) From left navigation click on
                       <b>
                        settings
                       </b>
                       and
                       <b>
                        search preferences
                       </b>
                       <br/>
                       3)
                       <b>
                        Click on enable custom ads.txt content on monetization section.
                       </b>
                       <br/>
                       3) and enter your publisher id with the following format:
                       <br/>
                       <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">
