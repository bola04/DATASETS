         <b>
          Witamy w DOMU ŻEGLARZA!!!
         </b>
         <br/>
         <br/>
         <b>
          Swornegacie
         </b>
         to miejscowość położona nad rzeką Brdą, pomiędzy jeziorami Karsińskim i Witoczno, w Zaborskim Parku Krajobrazowym stanowiącym otulinę Parku Narodowego Bory Tucholskie, 25 km od Chojnic.
         <b>
          Swornegacie
         </b>
         znane są m.in. z malowniczych tras rowerowych
 oraz - przede wszystkim - kajakowych (spływy rzekami Brdą, Zbrzycą i Chociną).
         <br/>
         <br/>
         <b>
          Oferujemy
         </b>
         pokoje 2, 3 i 4 osobowe (w sumie 20 miejsc). Wszystkie z własnym węzłem sanitarnym (natrysk, umywalka, WC).
         <br/>
         <br/>
         <b>
          Udostępniamy
         </b>
         kuchnię wyposażoną w podstawowy sprzęt AGD (bez możliwości gotowania obiadów), grill, plac zabaw dla dzieci oraz miejsca parkingowe na terenie ogrodzonym.
         <br/>
         Dla gości dostępna jest również sauna (za dodatkową opłatą).
         <br/>
         <br/>
         <b>
          Pomagamy
         </b>
         w organizacji spływów kajakowych.
         <br/>
         <br/>
         <b>
          Cena
         </b>
         za dobę wynosi od 40 zł od osoby (bez wyżywienia).
         <br/>
         <br/>
         <br/>
         <b>
          W pobliżu
         </b>
         :
         <br/>
         •	wypożyczalnia sprzętu pływającego (żaglówki, kajaki, rowery wodne, łodzie wiosłowe),
         <br/>
         •	wypożyczalnia rowerów i quadów,
         <br/>
         •	sklepy spożywcze,
         <br/>
         •	możliwość wykupienia całodziennych posiłków,
         <br/>
         •	możliwość wykupienia zezwoleń wędkarskich.
         <br/>
         <br/>
         Oferujemy nie tylko wygodne, atrakcyjne pokoje, ale również wypoczynek w miłej atmosferze. Gościmy klientów indywidualnych jak również grupy zorganizowane.
         <br/>
         Klimat
         <b>
          DOMU ŻEGLARZA
         </b>
         sprzyja integracji i dobrej zabawie, a otoczenie oferuje wiele możliwości aktywnego (lub nie) wypoczynku.
         <br/>
         <br/>
         <i>
          <b>
           Serdecznie zapraszamy
          </b>
         </i>

          </b>
