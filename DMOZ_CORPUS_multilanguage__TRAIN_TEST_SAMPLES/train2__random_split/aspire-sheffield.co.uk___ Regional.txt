                </p>
                <p>
                 Although our work was supported in part by public and private funders, aided by a dedicated team of volunteers and contributors, we recognised the need for us to become self supporting through business activities.  We continue to be self sustaining thanks to generous donations of computer recycling stock and other WEEE along with the continued assistance of our amazing volunteers.
                </p>

                 We have also developed a range of commercially viable activities that support our core costs and importantly, provide a wealth of opportunities for the people whom we support.
                </p>

                </p>

                  “Aspiring People”

                </p>

         if (url.indexOf('#') > 0) {
         var cl = url.indexOf('#');
         urlhash = url.substring(url.indexOf('#'), url.length);
         } else {
         urlhash = '';
         cl = url.length;
         }
         sourceUrl = url.substring(0, cl);
         var urlParts = sourceUrl.split("?");
         var newQueryString = "";
         if (urlParts.length > 1) {
         var parameters = urlParts[1].split("&");
         for (var i = 0; (i < parameters.length); i++) {
         var parameterParts = parameters[i].split("=");
         if (!(replaceDuplicates && parameterParts[0] == parameterName)) {

         newQueryString += "&";
         newQueryString += parameterParts[0] + "=" + (parameterParts[1] ? parameterParts[1] : '');
