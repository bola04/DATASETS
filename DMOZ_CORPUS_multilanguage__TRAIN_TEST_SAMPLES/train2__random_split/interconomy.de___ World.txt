         <h1>
          Freiberufliche Mitarbeiter finden und buchen!
         </h1>
         <p>
          <b>
           Seit 1999 helfen wir unseren Kunden bei der Projektbesetzung. Bisher haben wir über 6.000 externe Mitarbeiter für mehr als 3.000 Projekte bei rund 350 Kunden gefunden.
          </b>
         </p>
         Auch für Ihr Projekt finden wir schnell und zuverlässig den richtigen Freelancer.
         <br/>
         <p>
          <iframe allowfullscreen="" alt="So helfen wir Ihnen" frameborder="0" height="284" src="http://www.youtube-nocookie.com/embed/rSVjlnKOYkM?rel=0&amp;autoplay=0&amp;modestbranding=1&amp;showinfo=0&amp;theme=light" width="505">
          </iframe>
          <br/>
         </p>
         <br/>
         <h2>
          Schwerpunkte unserer Engineering und IT Freelancer:
         </h2>
         <p>
          Mit rund 10.240 handverlesenen registrierten Engineering und IT Spezialisten verfügen wir über ein erstklassiges Netzwerk und sind in der Lage Ihnen kurzfristig erfahrene Freiberufler für viele Engineering und IT Themen zur Verfügung zu stellen. Unser Pool umfasst derzeit:
         </p>
         <ul>
          <li>
           4.409
           <a href="/node/44972 ">
            C / C++ Programmierer
           </a>
          </li>
          <li>
           3.575
           <a href="/node/44969 ">
            Java Programmierer
           </a>
          </li>
          <li>
           2.232
           <a href="/node/44978 ">
            .NET Programmierer
           </a>
          </li>
          <li>
           540
           <a href="/node/30 ">
            SPS Programmierer und Inbetriebnehmer
           </a>
          </li>
          <li>
           261
           <a href="/node/30 ">
            Siemens SIMATIC S7 / Step7 / TIA Portal SPS Programmierer
           </a>
          </li>
          <li>
           128
           <a href="/node/30 ">
            WinCC Programmierer
           </a>
          </li>
          <li>
           41
           <a href="/node/30 ">
            PCS7 Programmierer
           </a>
          </li>
          <li>
           27
           <a href="/node/30 ">
            Beckhoff / TwinCAT SPS Programmierer
           </a>
          </li>
         </ul>
         <br/>
         <h2>
          Warum Freelancer von uns?
         </h2>
         <ul>
          <li>
           Unsere
           <a href="/node/19 ">
            letzte Kundenbefragung
           </a>
           ergab: 100% Bereitschaft zur Weiterempfehlung.
          </li>
          <li>
           Wir bieten Rechtssicherheit beim Freiberuflereinsatz und hohe Qualität zum fairen Preis.
          </li>
          <li>
           Unsere Versicherung haftet bei Personen-, Sach- und Vermögensschäden bis 5 Mio. EUR.
          </li>
         </ul>
         <h3>
          Sie erreichen uns deutschlandweit kostenlos unter 0800 68 11 700 per Telefon.
         </h3>
         <br/>
         <h2>
          <br/>
          Formular ausfüllen und kostenlos Mitarbeitervorschläge erhalten:
          <br/>
          Mit dem Speichern dieses Formulars erklären Sie sich mit unserer
          <a href="/Datenschutzerklaerung" target="_blank">
           Datenschutzerklärung
          </a>
          einverstanden.
          <br/>
         </h2>
