        <div id="oshirase">
         2014.01.25
         <img alt="NEW" border="0" height="13" src="top/new.gif" width="25"/>
         <br/>
         ２０１３年５月９日、２７７０グラムの元気な女の子が生まれました！久しぶりの赤ちゃんにメロメロになりながら、少しずついろいろと発信していきたいと思います。
         <br/>
         <hr/>
         2012.03.21
         <br/>
         莱夢が空に帰って３年経ちました。莱夢はシアワセな子です。彼女を偲んでメールやお花が届きました。みなさん、ありがとう。私たち家族はみんな元気に仲良く過ごしてますよ。莱夢、安心して見守っていてね。
         <br/>
         <hr/>
         2012.02.01
         <br/>
         いろいろ始動です。壊れてしまったパソコンから、莱夢が空に還った当時のメモを抽出でしました。ようやく、当時のことを書き綴ることができそうです。
         <br/>
         サイトイメージを変更しました。きれいに表示できない場合は、ブラウザの「更新」ボタンを押してください。
         <br/>
         <hr/>
         2011.03.21
         <br/>
         莱夢が空に還って、今日で２年経ちました。彼女を偲んでメールをくださる方もいて、改めて彼女が私に残してくれた数々の縁に感謝です。
         <br/>
         あれからもう２年経ったんですね。大好きな莱夢が天国で心配しないように、地上にいる私たち家族は仲良く元気に暮らしていきたいと思ってます。いろいろと不安も多いですが、頑張っていきましょう。被災された方々のためにも、節電しながらしっかり経済活動して日本を支えていきたいです。
         <br/>
         <hr/>
         2011.02.08
         <br/>
         莱夢が空に還って、もう２年近く経ちます。そろそろ当時の日記を、記録として書かなければと思うのです。でも、彼女が生きた最後の数ヶ月は、やはりまだ文章に書くことができません・・・。私の胸の中に決して癒えない傷があり、ともすれば簡単に血を流すのです。魂の深い分部に、悲しみの記憶が刻印されています。
         <br/>
         彼女が自力で立てなくなり、やがて薬も注射も効果がなくなり、とうとう食べ物も水も受け付けなくなったとき、ずっと恐れていた「いつか」という瞬間が迫っていると感じました。生きていることの不思議、死んでいくことの不思議を感じながら、決して後悔しないよう、生命の選択を迫られる毎日でした。莱夢は莱夢らしく、天から与えられた地上での時間をしっかりと生きたのだと思います。でも、どうしてこんなに短かったのでしょう・・・。神様は意地悪です。あんなにも私たち家族に愛され、必要とされていた莱夢を、こんなにも早く連れて行っちゃうなんて・・・。
         <br/>
         莱夢は最期、私の膝の上に頭を乗せて、私の腕に抱かれながら眠るように呼吸を止め、家族に見守られて潔く旅立っていきました。「ありがとう莱夢。もう逝っていいよ。いっぱい頑張ったよ。貴女がいてとてもシアワセだった。ありがとう。ありがとう。」私は泣きながらそんな言葉を掛けたと思います。
         <br/>
         あれから２年近く経ち、彼女がいないことが日常になりつつあります。それでも、ふとした瞬間に彼女の姿を探してしまう自分がいます。私の記憶の中で、いつも物静かに微笑んでいる莱夢です。
         <hr/>
         2010.09.11
         <br/>
         ときどき、思わぬ方から励ましのメールをいただき、莱夢が残してくれた「縁」の不思議を感じます。みなさまとても心配してくださり、こうして再開したウェブサイトを見て安心してくださったようで、もっと早く再開すればよかったと反省しきりです。ご心配をおかけし、すみませんでした。また、莱夢のことを覚えていてくださり、本当に嬉しいです。ありがとうございます。
         <br/>
         <hr/>
         2010.07.21
         <br/>
         アラスカンマラミュートの莱夢が空に還ってから、もう一年以上経ちます。育児と家事、そして仕事と、日常に追い立てられながら過ぎ去った毎日でした。
         <br/>
         このサイトをどのように運営していくか、ずいぶん考えました。彼女が逝った今、更新できる情報などほとんどないのですから・・・。ありきたりな「育児日記」にしたくない。かといって、彼女が生きた証として、さまざまな記録は取っておきたいし、知ってもらいたい・・・。ならば、このサイトはこのまま続けていこう・・・。あの素晴らしくシアワセだった毎日を綴っていこう、そう決めました。
         <br/>
         今回、リニューアルオープンするにあたり、すべてのページを見直しました。以前はこのサイトを見るだけで、どこか心が痛んだ私でした。でも、今回の作業は本当に楽しかった・・・。「あんなこともあった」「こんなこともあった」と、ただただ懐かしく、莱夢への感謝の気持ちばかり募りました。きっと、私の中で、機が熟したのでしょうね。
         <br/>
         ただ、彼女の体調がどんどん悪くなっり、「どうやったら元気にしてあげられるか」から、「どんな死を迎えさせてあげればよいか」へと変化していった、２００９年１月から３月２１日までの日記は、やはりまだちゃんと向き合って書くことができないようです。当時のメモから日記を書こうとすると、簡単に当時の気持ちに戻ってしまい、涙が止まらないのです・・・。だから、どうかもう少しお待ちください。
         <br/>
         サイトデザインは大きく変わりましたが、コンテンツは変わってません。追加もされてません。今後は、お伝えしていなかった莱夢のことなど、少しずつ更新していきたいと思います。
         <br/>
         <hr/>
         2006.06.20
         <br/>
         ６月１５日放映の「奇跡体験！アンビリバボー」に出演予定だった莱夢でしたが、いざ当日になってみるとカットされていました。楽しみにしていた方、録画予約していた方、本当に申し訳ありませんでした。私自身もプロダクションも、まさかカットされてるとは知らず、とても楽しみにしていたのですが、番組制作会社の都合で突然カットになってしまいました。
         <br/>
         今回は他のVTRやスタジオトークとのからみで、時間が合わなくなってしまったとの事なのです。（きっと、み○もんたのせいね〜！）ＶＴＲ自体がボツという事ではないので、また「気づかなかった」ネタの時にオンエアされるそうです。その際はまたお知らせいたしますので、懲りずに見てやってくださいね。どうぞよろしくお願いします。
         <br/>
         <hr/>
         2006.06.05

         2003.07.01
         <br/>
         長い間おやすみをしてしまい、ご心配をおかけしました。サイト内に「伽凛」の姿がないこと、もうお気づきですね？
         <br/>
         私たちの身勝手な都合で、伽凛とは離れて暮らすことになってしまいました。臆病で繊細な伽凛だから、住み慣れた家でパパと一緒に静かに暮らすほうがいいよね？もう莱夢に追いかけられたり、北斗にしつこくされて逃げなくていいんだもの、ママがいなくても平気だよね？彼の「伽凛だけは残していってくれ」という願いもあって、迷っいに迷った末の選択した。今でもその選択が正しかったとは思えません。時々、「無理にでも連れきてしまえばよかった」と、後悔して泣けてしまいます。でも、莱夢と北斗と一緒に、新しい環境で新しい暮らしをさせることが正しかったとも、やっぱり思えないのです。
         <br/>
         気持ちの整理が付かず、サイト内の伽凛のコーナーをどのようにしたらよいか悩んでいます。もう別の家の飼い猫になってしまった伽凛なので、このサイトで紹介するのは心苦しいし、かといって、まったく消してしまうのは淋しいし・・・・。いずれ何らかの形で伽凛のコーナーは設けるつもりですが、今はしばらくこのスタイルで再スタートを切りたいと思います。
