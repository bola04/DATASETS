             Design a Funkce
            </h2>
            <p>
             Stylový nábytek vyráběný z masivního smrkového a bukového dřeva vás překvapí nejen moderním designem severského stylu, ale také svou variabilitou, funkčností a dlouhou životností. Naše výrobky vnesou do vašeho domova nádech přírody.
            </p>
            <p>
             Značka Gazel je zárukou kvality, stabilního sortimentu a zdravotní nezávadnosti, proto je k dispozici stovkám spokojených zákazníků hotelů, penzionů a rekreačních chat po celé Evropě. Nábytek Gazel byl také použit při zařizování několika Dětských domovů, léčeben a ozdravoven. Pro své interiéry volily nábytek z masivu také maminky S.O.S. vesniček.
            </p>
            <h2>
             Ekologie
            </h2>
            <p>
             Surovinou používanou při výrobě nábytku je kvalitní tuzemské dřevo, které neobsahuje žádné zdraví škodlivé substance, jako jsou formaldehydy, dioxiny nebo těžké kovy. Nábytek je ošetřen kvalitními, ekologicky šetrnými laky, které zvýrazní kresbu dřeva a umocní pocit z krásy přírody. Toto je ideální kombinace zaručující velmi příjemné, teplé, zdravé a pryskyřicí provoněné bydlení. Ekologické aspekty, zdraví a bezpečnost zákazníka jsou prioritou při výrobě našeho nábytku.
            </p>
            <h2>
             Zdravotní nezávadnost
            </h2>
            <p>
             Veškerý sortiment je certifikován ve Zkušebním ústavu lehkého průmyslu a v Mendelově zemědělské a lesnické univerzitě v Brně. Komponenty a materiály používané k výrobě nábytku odpovídají požadovaným normám a jsou opatřeny atesty. Materiály používané pro výrobu nábytku jsou těženy v lokalitách s minimem negativních vlivů člověka. Komponenty a materiály před použitím procházejí náročným testem, ve kterém je hodnocena jak jejich kvalita, tak i vzhled. Důležitým ukazatelem kvality je spokojený zákazník
            </p>
            <h2>
             Rychlost dodání
            </h2>
            <p>
             Díky dostatečným skladovým zásobám a flexibilitě výroby máme krátké dodací lhůty, ve většině případů jsme schopni uspokojit zákazníka ihned. Na zakázku vyráběný nábytek atypických rozměrů je dodáván většinou do 4 týdnů od objednání.
            </p>
            <h2>
             Nábytek pro celou rodinu
            </h2>
            <p>
             Sortiment obsahuje širokou škálu výrobků, kterými lze zařídit téměř všechny pokoje ve vašem bytě. Nepřetržité sledování trhu a vlastní účast na tématických veletrzích zaručuje přizpůsobování se požadavkům zákazníků či nejnovějším, neustále se měnícím trendům.
            </p>
            <h2>
             Chytré balení

             Většina našeho zboží je dodávána v demontovaném stavu a prakticky balena. Obaly z několikavrstvé lepenky zaručují ochranu proti poškození jak při skladování, tak při vlastní přepravě či manipulaci.

         function newsletter_subscribe(subscribe) {
             var subscribeProgress = $("#subscribe-loading-progress");
             subscribeProgress.show();
             var postData = {
                 subscribe: subscribe,
                 email: $("#newsletter-email").val()
             };
             $.ajax({
                 cache: false,
                 type: "POST",
                 url: "/subscribenewsletter",
                 data: postData,
                 success: function(data) {
                     subscribeProgress.hide();
                     $("#newsletter-result-block").html(data.Result);
                     if (data.Success) {
                         $('#newsletter-subscribe-block').hide();
                         $('#newsletter-result-block').show();
                     } else {
                         $('#newsletter-result-block').fadeIn("slow").delay(2000).fadeOut("slow");
                     }
                 },
                 error: function(xhr, ajaxOptions, thrownError) {
                     alert('Failed to subscribe.');
                     subscribeProgress.hide();
                 }
             });
         }
 
         $(document).ready(function () {
             $('#newsletter-subscribe-button').click(function () {
                     newsletter_subscribe('true');
             });
             $("#newsletter-email").keydown(function (event) {
                 if (event.keyCode == 13) {
                     $("#newsletter-subscribe-button").click();
                     return false;
                 }
             });
         });
