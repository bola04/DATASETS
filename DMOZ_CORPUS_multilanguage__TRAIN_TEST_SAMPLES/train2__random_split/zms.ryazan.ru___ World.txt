                         <h1 class="page-title">
                          АО "Завод металлов и сплавов"- производство припоя, баббита, олова, свинца, порошков
                         </h1>
                         <img border="1" class="leftimg" src="images/zda.jpg" title="Завод металлов и сплавов"/>
                         Акционерное общество "Завод металлов и сплавов" было образовано в 1998 году на базе опытно-металлургического цеха лидера российской цветной металлургии - завода "Рязцветмет".
                         <br/>
                         <br/>
                         Основными задачами цеха были совершенствование существующих и разработка новых технологий, создание и отладка оборудования и аппаратуры для производства цветных металлов. Разработки цеха были защищены 13 авторскими свидетельствами. В 90-е годы коллективом цеха освоен выпуск проволоки, трубки с канифолью, ленты, прутка, анодов из припоев, баббитов, олова и свинца. Для этого производства был создан высокопроизводительный автоматизированный комплекс прессового оборудования. 
 АО "Завод металлов и сплавов" в полной мере располагает этими технологиями, оборудованием, опытом их разработок.
                         <br/>
                         <br/>
                         <img border="1" class="rightimg" src="images/pripoi-POS-61.jpg" title="Производство припоев и сплавов"/>
                         В настоящее время АО "Завод металлов и сплавов" выпускает:
                         <br/>
                         <br/>
                         <p>
                          <strong>
                           <ul>
                            <li class="pripoi">
                             <a href="solder_tin">
                              припой оловянно-свинцовый (ПОС, ПОССу, ПОСК, ПОСу)
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             <a href="pripoi-katushki">
                              припой оловянно-свинцовый в мелкой фасовке
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             <a href="babbit_tin">
                              баббиты оловянно-свинцовые (Б-16, Б-83, Б-88) и изделия из них
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             <a href="babbit_calcium">
                              баббиты кальциевые (БК2, БК2Ш, БКА)
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             <a href="powder_tin">
                              порошок оловянный (ПО1, ПО2, ПО3, ПО4, ПОЭ)
                             </a>
                             ,
                             <a href="powder_lead">
                              порошок свинцовый (ПС1, ПС2, ПСА)
                             </a>
                             ,
                             <a href="powder_solder">
                              припойный порошок
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             изделия из чистых металлов:
                             <a href="tin">
                              олова
                             </a>
                             ,
                             <a href="lead">
                              свинца
                             </a>
                             , висмута, кадмия, цинка;
                            </li>
                            <li class="pripoi">
                             <a href="solder_cooper">
                              припой медно-фосфористый (ПМФОЦр, ПМФ-9)
                             </a>
                             ;
                            </li>
                            <li class="pripoi">
                             флюсы, канифоль сосновую.
                            </li>
                           </ul>
                          </strong>
                         </p>
                         <br/>
                         <br/>
                         Производство продукции осуществляется по практически безотходной технологии - полученные отходы производства перерабатываются восстановительной плавкой в электропечи.
                         <br/>
                         <br/>
                         Утилизация отходов сочетается с надежной системой газопылеулавливания, которая очищает отходящие от металлургического оборудования газы. Это обеспечивает экологическую безопасность производства.
                         <br/>
                         АО "Завод металлов и сплавов" постоянно работает над совершенствованием существующих технологий, разработкой и внедрением новых перспективных, оригинальных технологий производства продукции и управления производством. Основу коллектива завода составляют опытные, квалифицированные специалисты, на заводе практически нет текучести кадров.
                         <br/>
                         <br/>
                         Управление заводом осуществляют молодые, энергичные руководители, имеющие отвечающее современным требованиям образование по экономике, менеджменту, маркетингу.
                         <img alt="Производство припоев и сплавов" border="1" class="leftimg" src="images/babbit.jpg" title="Припой, олово, свинец, баббитовая проволока"/>
                         Указанные факторы гарантированно обеспечивают выпуск продукции высшего качества. Контроль качества осуществляется с помощью как химических, так и спектральных методов анализа в аттестованной центральной заводской лаборатории, имеющей все необходимое оборудование и ПЭВМ.
                         <br/>
                         <br/>
                         <b>
                          Система менеджмента качества (СМК) АО "Завод металлов и сплавов" соответствует требованиям ГОСТ ISO 9001-2015
                         </b>
                         .
                         <br/>
                         <br/>
                         В результате АО "Завод металлов и сплавов" имеет стабильное хорошее состояние, постоянно наращивает объемы производства, его продукция пользуется высоким спросом.
