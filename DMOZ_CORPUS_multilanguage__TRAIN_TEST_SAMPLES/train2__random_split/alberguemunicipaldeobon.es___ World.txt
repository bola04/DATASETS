       <div align="left" class="bodyText">
        <p>
         La Villa de Obón pertenece 
           a la comarca turolense de
         <a href="http://www.comarcacuencasmineras.es/" target="_blank">
          Cuencas
           Mineras
         </a>
         ; es uno de los ocho pueblos incluidos en el
         <a class="soloazul" href="http://www.parqueriomartin.com/" target="_blank">
          Parque Cultural del Río Martín
         </a>
         (uno de los 5
         <a href="http://www.aragob.es/edycul/patrimo/parques/parques.htm" target="_blank">
          Parques
             Culturales de Aragón
         </a>
         ), río
             en cuyos cañones
             se encuentra un conjunto de pinturas prehistóricas
         declarado Patrimonio Mundial por la UNESCO. Desde nuestro Albergue hay
         mucho que pasear y ver.
        </p>
        <blockquote>
         <p>
          <b>
           Sitios de Interés:
          </b>
          <br/>
          - Iglesia de la Asunción
          <br/>
          - Riberas del Río Martín
          <br/>
          - Río Cabra (llamado aquí Río La Torre,
          <a href="http://www.parqueriomartin.com/ruta_descenso_rio_cabra.htm" target="_blank">
           barranquismo
          </a>
          7 h.)
          <br/>
          - Barranco del Regallo (Posibilidad de barranquismo)
          <br/>
          - Cerro de las Olivas (Yacimientos Edad Hierro)
          <br/>
          -
          <a href="http://www.parqueriomartin.com/ruta_chornas_y_cerrao.htm" target="_blank">
           Ruta de Chornas-El Cerrao
          </a>
          (Duración 3
           horas, documentación del Parque Cultural) (
          <a href="fotos_Chornas_Cerrao.html" target="_blank">
           fotos
          </a>
          )
          <br/>
          - Paseo hasta la Fuente del Batán (merendero, 1 h. ida)
          <br/>
          - Ruta de la Rallera (unas 2,30 horas) (
          <a href="fotos_Rallera.html">
           fotos
          </a>
          )
          <br/>
          - Ruta de la Coquinera (Duración
           2 horas, o
          <a href="http://www.parqueriomartin.com/ruta_canones_rio_martin.htm" target="_blank">
           4 hasta Alcaine
          </a>
          , íd.) (
          <a href="fotos_Coquinera_Alcaine.html" target="_blank">
           fotos
          </a>
          )
          <br/>
          - Ruta desde o
          <a href="http://www.parqueriomartin.com/ruta_penas_royas.htm" target="_blank">
           hacia Peñarroyas
          </a>
          (4,30 horas, bajando o subiendo por el río).
          <br/>
          - Ruta desde o
          <a href="http://www.parqueriomartin.com/ruta_canones_rio_martin.htm" target="_blank">
           hacia Alcaine
          </a>
          (3+1 horas). (
          <a href="http://www.youtube.com/watch?v=BNQzL4mKih0" target="_blank">
           Vídeo
          </a>
          programa Chino-chano)
         </p>
        </blockquote>
