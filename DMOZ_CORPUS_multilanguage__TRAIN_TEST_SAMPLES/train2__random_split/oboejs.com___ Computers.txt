     <div class="col col2">
      <main class="marked">
       <p>
        Oboe.js is an
        <a href="LICENCE">
         open source
        </a>
        Javascript library
 for loading JSON using streaming, combining the convenience of DOM with
 the speed and fluidity of SAX.
       </p>
       <p>
        It can parse any JSON as a stream, is small enough to be a
        <a href="http://microjs.com/#">
         micro-library
        </a>
        ,
 doesn’t have dependencies, and doesn’t care which other libraries you need it to speak to.
       </p>
       <p>
        We can load trees
        <a href="examples#loading-json-trees-larger-than-the-available-ram">
         larger than the available memory
        </a>
        .
 Or we can
        <a href="examples#demarshalling-json-to-an-oop-model">
         instantiate a classical OOP models from JSON
        </a>
        ,
 or
        <a href="examples#transforming-json-while-it-is-streaming">
         completely transform your JSON
        </a>
        while it is being read.
       </p>
