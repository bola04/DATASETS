       <h2 class="head-set">
        WEB-ENGINEERING
       </h2>
       <p>
        Unter Web-Engineering verstehen wir die Erstellung serverbasierender Anwendungen, die als "Clienten" die vorhandenen Webbrowser benötigen. Für den Anwender liegt der Vorteil damit in der Unabhängigkeit seines Systems - er benötigt lediglich ein internetfähiges "Gerät" wie einen Computer, ein Tablet oder Smartphone. Für die Realisierung der Anwendungen verwenden wir vorzugsweise PHP und Java als Programmiersprachen und MySQL als Datenbank.
        <br/>
        <br/>
        <strong>
         Je nach Anforderung greifen wir auf bewährte Open-Source Lösungen zurück und modifizieren diese entsprechend oder entwickeln komplette Anwendungen neu:
        </strong>
       </p>
       <br/>
       <ul>
        <li>
         Internetshop-Realisierung auf Basis modified eCommerce mit Anbindung an Warenwirtschaftssysteme
        </li>
        <br/>
        <li>
         Content-Management-Syteme auf Basis Joomla!
        </li>
        <br/>
        <li>
         Neuentwicklung und Erweiterung von Webanwendungen
        </li>
        <br/>
       </ul>
       <br/>
       <strong>
        Diese von uns entwickelten Webanwendungen sind bei unseren Kunden im Einsatz:
       </strong>
       <br/>
       <br/>
       <ul>
        <li>
         Dokumentation von Anlagen- und Maschinenprüfungen, Prüffristen und Archivierung von Prüfprotokollen
        </li>
        <br/>
        <li>
         Planung und Steuerung von Instandhaltungen, Inspektionen und Wartungen
        </li>
        <br/>
        <li>
         Auswertung und Archivierung von Energiedaten
        </li>
        <br/>
        <li>
         Rezeptdatenbanken
        </li>
        <br/>
        <li>
         Export von freien Wohnungen aus Hausverwaltungssoftware und deren Veröffentlichung im Internet
        </li>
        <br/>
        <li>
         Datenerfassung und Auswertung von digital Signage Stationen (in Kooperation mit
         <a href="https://heico-media.de/ " target="_blank">
          heico-media.de)
         </a>
         <a>
         </a>
