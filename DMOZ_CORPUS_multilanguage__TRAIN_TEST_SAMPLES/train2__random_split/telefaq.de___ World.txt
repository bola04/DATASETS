    - Erreichbarkeit gehend, kommend, umlenken
   </h2>
   <div class="inhalt">

    <dl>
     <dt style="background-image:url(img/logo-verbindungen.gif)">

      - Verwaltung einzelner und mehrerer Rufe
     </dt>
     <dd>
      • Besetzt, Anklopfen, Besetzt bei Besetzt
      <br/>
      • Rückruf bei Besetzt, Rückruf bei Nichtmelden
      <br/>
      • Halten, Rückfragen/Makeln, Parken, Konferenz, Vermitteln
      <br/>
      • Tarifinformation, Einheiten-Übermittlung
     </dd>
    </dl>
    <dl>
     <dt style="background-image:url(img/logo-umleitungen.gif)">
      <a href="umleitungen.html">
       <strong>
        Umleitungen
       </strong>
      </a>
      - Weiterschaltung von Anrufen
     </dt>
     <dd>
      • Sofort, bei Besetzt, Verzögert, Wartezeit, Fernsteuerung
      <br/>
      • Selektive Anrufweiterschaltung bestimmter Anrufer
      <br/>
      • Während der Rufphase, nebenstellenindividuell, im Störungsfall
      <br/>
      • Parallelruf, Switch&amp;Profit vom Handy zum Telekom-Festnetz
     </dd>
    </dl>

     <dt style="background-image:url(img/logo-waehlsperren.gif)">

     </dt>
     <dd>

      <br/>

      <br/>
      • Rufnummern-Sperre mit individueller Negativ-/Positivliste
      <br/>
      • 010-Sperre zur Verhinderung von Call-by-Call-Auswahl

    </dl>

       </strong>
      </a>

     </dt>
     <dd>

      <br/>
      • Annahme erwünschter Anrufer mit Positivliste
      <br/>
      • Abweisen von Belästigern über Sperrliste, Sicherheitspaket
      <br/>
      • Abweisen unbekannter Anrufer, Identifizieren/Fangen
     </dd>
