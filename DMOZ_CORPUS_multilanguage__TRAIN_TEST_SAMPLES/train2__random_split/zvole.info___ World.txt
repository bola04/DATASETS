        Adam Máj – úspěšný závodník v motokrosu
       </h3>
       <p>
        Třináctiletý kluk ze Zvole předvádí v letošní motokrosové sezoně skvělou formu. Po polovině seriálu a po vyhranných závodech v Netolicích, foto
        <a class="file" href="e_download.php?file=data/editor/538cs_5.JPG&amp;original=m%C3%A1j%201.JPG" target="_blank" title="Soubor: máj 1.JPG, Typ: JPEG Image, Velikost: 184.44 kB">

        <a class="file" href="e_download.php?file=data/editor/538cs_1.PNG&amp;original=m%C3%A1j%20tabulka.PNG" target="_blank" title="Soubor: máj tabulka.PNG, Typ: Portable Network Graphics (PNG), Velikost: 715.77 kB">
         Zde
        </a>

        <a href="http://www.fairplay-motorsport.cz/pk_reward/adam-maj/">

       <h2>
        Zpráva pro občany
       </h2>
       <p>
        Dnes v podvečer dorazí do Zvole specializovaná odchytová služba z Karlových Varů, kterou kontaktoval majitel ZOO. Bude mu vydáno povolení k činnosti na katastru obce a poskytneme mu jakoukoli součinnost, které jsme schopni. Budou vybaveni přístroji pro noční vidění a puškami s narkotizačními střelami. Od večera budou kontrolovat předpokládané místo výskytu šelmy.
       </p>
       <p>
        Žádáme všechny občany, aby se zdrželi jakýchkoli pokusů přilákat pumu a pokusit se jí zneškodnit. Situace není pro podobné pokusy vhodná. Prosím o omezení pohybu v lesích tak, aby se puma zklidnila a neutekla mnohem dál, než se dá její výskyt předpokládat. To by věc s odchytem značně zkomplikovalo.
       </p>
       <p>
        Děkuji   Mirek Stoklasa, starosta obce
        <em>
         <span style="font-size: 10px;">
          30.6.2018
         </span>

       <h2>
        Zpráva pro občany
       </h2>
       <p>
        Puma je stále ve volné přírodě. Ošetřovateli se ji nepodařilo odchytit. Rozšiřuji varování na pozornost, kterou mají věnovat rodiče dětem. Vzhledem k jejich výšce by hladová puma mohla dítě považovat za snadnou kořist. Týká se i dětí v kočárcích, které by neměly zůstat bez dozoru, a to ani v uzavřených zahradách. Plot není pro kočkovitou šelmu překážkou. Prosím o zvýšenou opatrnost. Podobně se může chovat puma i k malým psům. Vyzval jsem chovatele, aby jako vstřícný a preventivní krok přesunul všechny své kočkovité šelmy mimo katastr Zvole do odpovídajícího zařízení.

Jedná se o rekonstrukci administrativní budovy u bývalého vepřína. Na tuto akci vypisujeme výběrové řízení na zhotovitele. Předpokládané finanční náklady jsou okolo šesti milionů, dotace bude krýt 95 % uznatelných nákladů. Dokončení rekonstrukce by mělo být do konce letošního roku. Na obě dotace jsme čekali řadu měsíců, to že přišly v jeden den je příjemná shoda náhod. Oba projekty byly podány na základě výzev MAS Dolnobřežansko a především při administrování žádosti na nízkoprahové zařízená nám pracovníci této organizace výrazně pomáhali. Děkujeme.

       </h3>
       <p>
        Milí přátelé,
       </p>
       <p style="text-align: justify;">
        výsledek Tříkrálové sbírky 2018 ve Zvoli a Černíkách činí úctyhodných 71.042 Kč. Částka vybraná v celé zbraslavské farnosti (kam územně patří i Zvole) je 114.613 Kč. Celková částka vybraná v ČR pak představuje nový rekord ve výši téměř 115 miliónů korun. Tyto peníze půjdou na podporu lidí v nouzi u nás i v zahraničí.
       </p>
       <p style="text-align: justify;">
        Jak jistě víte, část peněz vybraných ve zbraslavské farnosti je určena na nákup polohovacího zdravotního křesla pro Petra Škobise ze Zvole. Petr je v dlouhodobé rekonvalescenci po rozsáhlé mozkové příhodě. Momentálně je v rehabilitačním zařízení. Po návratu do domácího prostředí může zdravotní křeslo ihned začít užívat.
       </p>
       <p style="text-align: justify;">
        Milí přátelé, rádi bychom vám ze srdce poděkovali za váš velmi pozitivní přístup k Tříkrálové sbírce. Děkujeme všem malým i velkým koledníkům, kteří překonali pohodlí a ostych, a také děkujeme vám, kteří jste jakoukoli částkou přispěli do sbírky. Snad nikdo z koledníků se nesetkal s nezájmem nebo odmítnutím. Právě naopak. Prokázali jste svá otevřená srdce a přispěli nejen do pokladniček, ale také jste obdarovali malé koledníky sladkostmi a ty velké něčím ostřejším pro zahřátí. Výsledek je to velmi povzbudivý. Velké poděkování vám všem.

        function kaPrevMonth()
 {
 	var m = kaGetSelectedMonth.apply(this);
 	var y = kaGetSelectedYear.apply(this);
 	var minY = Math.min(getSelectOptionValues(this.find('#year')));
 
 	if(m === 1){
 		m = 12;
 		y -= (y === minY ? 0 : 1);
 	}else{
 		m -= 1;
 	}
 
 	kaRedrawCalendar.apply(this, [y, m]);
 }
 
 function kaNextMonth()
 {
 	var m = kaGetSelectedMonth.apply(this);
 	var y = kaGetSelectedYear.apply(this);
 	var maxY = Math.max(getSelectOptionValues(this.find('#year')));
 
 	if(m === 12){
 		m = 1;
 		y += (y === maxY ? 0 : 1);
 	}else{
 		m += 1;
 	}
 
 	kaRedrawCalendar.apply(this, [y, m]);
 }
 
 function getSelectOptionValues($select){
 	var values = [];
 	$select.find('opiton').each(function(){
 		values.push(this.value);
 	});
 
 	return values;
 }
 
 var sShow=false;
 var bShowDetail=false;
 function ka_showdetail(id)
 {
 	oEl=document.getElementById(id);
 	if (oEl)
 	{
 		if (sShow!=false)
 		{
 			cls.add(document.getElementById(sShow),'hide');
 		}
 		sShow=id;
 		cls.remove(oEl,'hide');
 		oEl.style.zIndex=9999;
 	}
 	else
 		sShow=false;
 }
 
 function ka_hidedetail(id)
 {
 	oEl=document.getElementById(id);
 	if (oEl && !bShowDetail)
 	{
 		cls.add(oEl,'hide');
 		sShow=false;
 		bShowDetail=false;
 	}
 }
 
 function ka_hidedetails()
 {
 	if (sShow!=false)

 		oEl=document.getElementById(sShow);
 		if (oEl)
 		{
 			cls.add(oEl,'hide');
 			sShow=false;
 			bShowDetail=false;
 		}
 	}
 }
 
 function kaRedrawCalendar(year, month){
 	var $form = this;
 	if(year !== undefined){
 		$form.find('#year option[value="' + year + '"]').prop('selected', true);
 	}
 	if(month !== undefined){
 		$form.find('#month option[value="' + month + '"]').prop('selected', true);
 	}
 
 	jQuery.ajax({
 		url: 'modules/kalendar/ax_kalendar.php'
 		, data: $form.serialize()
 		, type: 'POST'
 		, beforeSend: function(xhr, settings){}
 		, success: function(data){
 			if(data.length)
 				$form.find('#kal_table table').replaceWith(data);
 		}
 		, complete: function(){}
 	});
 }
 
 function kaGetSelectedMonth(){
 	return parseInt(this.find('#month option:selected').val(), 10);
 }
 function kaGetSelectedYear(){
 	return parseInt(this.find('#year option:selected').val(), 10);
 }
 
 function getParentForm(el) {
 	return jQuery(el).closest('form');
 }
 
 jQuery(document).ready(function(){
