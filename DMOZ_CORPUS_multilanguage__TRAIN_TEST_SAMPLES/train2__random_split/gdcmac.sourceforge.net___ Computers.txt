   <h1>
    gdcmac - GCC D Compiler for Mac OS X (Apple GCC)
   </h1>
   <p>
    <a href="http://sourceforge.net">
     <img alt="SourceForge.net" height="31" src="http://sourceforge.net/sflogo.php?group_id=135857&amp;type=1" width="88"/>

      http://sourceforge.net/projects/gdcmac/
     </tt>
    </a>
   </p>
   <table>
    <tr>

     <td>
      <p>
       <b>
        <span class="red">
         D
        </span>
        is a general purpose systems and applications programming language.
       </b>
      </p>
      <p>
       <i>
        The D programming language is a higher level language than C++, but retains the ability to
 write high performance code and interface directly with the operating
 system API's and with hardware. D is not a scripting language, nor an interpreted language.
       </i>
      </p>
      <p>
       <b>
        The Digital Mars D Compiler (DMD), is also available for various systems
       </b>
      </p>
      <p>
       DMD by Digital Mars, see
       <a href="http://www.digitalmars.com/d/1.0/dmd-osx.html">
        http://www.digitalmars.com/d/1.0/
       </a>
      </p>
      <p>
       <i>
        The official D "reference" compiler, DMD, is based on the same back-end as the Digital Mars C/C++ compiler for Windows.
 The GDC compiler was made by adapting the DMD frontend source code, to work with the regular GCC backend.
       </i>
      </p>
      <p>
       <b>
        The GCC D Compiler (GDC) is based on the
        <a href="http://gcc.gnu.org/">
         GNU Compiler Collection
        </a>
        (GCC)
       </b>
      </p>
      <p>
       GDC by David Friedman, see
       <a href="http://dgcc.sourceforge.net/">
        http://dgcc.sourceforge.net/
       </a>
      </p>
      <p>
       <i>
        You can run programs created with GDC on any
        <a href="http://www.apple.com/macosx/">
         Mac OS X
        </a>
        machine (no special run-time libraries required)
 It is also possible to link with programs created by the Apple versions of the GCC C Compiler (
       </i>
       <tt>
        gcc
       </tt>
       <i>
        ) and the GCC C++ Compiler (
       </i>
       <tt>
        g++
       </tt>
       <i>
        ).
       </i>
      </p>
     </td>
