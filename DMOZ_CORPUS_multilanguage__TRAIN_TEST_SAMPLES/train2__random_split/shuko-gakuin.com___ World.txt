      <td class="maintext" valign="top" width="452">
       埼玉県鴻巣市吹上のそろばん教室「珠光学院（しゅこうがくいん）」では、そろばんをはじめ暗算、基礎計算（算数）を学ぶことができます。
       <br/>
       教室は
       <a href="/nyujuku/index.html#class">
        「吹上富士見教室」「北鴻巣教室」
       </a>
       の選べる2カ所。
       <br/>
       各種競技会にもチャレンジできるほか、バスレクリエーション、夕涼み会、
       <br/>
       クリスマス会など楽しいイベントもいっぱいです。
       <br/>
       イベント写真は
       <a href="/gallery/gallery.php">
        こちら
       </a>
       からご覧いただけます。

       <h2 class="title780">
        <span style="font-weight:bolder;">
         そろばんの良いところ
        </span>
       </h2>
      </td>
     </tr>
     <tr>
      <td class="maintext" valign="top">
       そろばんや暗算を学ぶと「集中力」「記憶力」「根気力」「想像力」などの能力がアップされます。
       <br/>
       これらの能力は学力よりはるかに大切な「気持ちの能力」です。
       <br/>
       「気持ちの能力」は生まれつきの能力ではありません。自分で出来たという達成感や、人から認められた体験を積み重ねることによって育って行きます。
      </td>

        </span>
       </h2>
      </td>
     </tr>
     <tr>
      <td class="maintext" valign="top" width="452">
       幼児、小学生時期は「読み・書き・計算」の基礎学力養成の大事な時期です。計算のプロセスや数の仕組みをしっかり学ぶことが重要です。
       <br/>
       珠光学院では一人ひとりの習熟度に合わせ指導します。また、学齢の枠がない進級法をとっていますので、学習意欲をどんどん向上させていくことができます。
