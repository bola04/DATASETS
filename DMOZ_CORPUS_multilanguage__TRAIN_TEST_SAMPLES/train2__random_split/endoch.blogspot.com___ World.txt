           <a class="timestamp-link" href="http://endoch.blogspot.com/2015/02/cara-membuka-file-tar-di-ubuntu-linux.html" rel="bookmark" title="permanent link">
            <abbr class="published" itemprop="datePublished" title="2015-02-11T15:39:00+07:00">
             2/11/2015 03:39:00 PM
            </abbr>
           </a>
          </h4>
         </div>
         <div class="post-body" id="post-body-2899244548412489546" itemprop="description articleBody">
          <section role="main">
           <div class="cap page columnize">
            <span style="font-family: Trebuchet MS, sans-serif;">
             sebagai pengguna Linux pemula, saya sedang membiasakan menggunakan command line karena lebih ngirit resource. maklum, pc linux ini saya gunakan sebagai database server, sekaligus torrentbox. Biasanya, ketika nemu file dengan ekstensi .tar.gz atau .tar, tinggal di klik kanan saja di Thunar lalu ekstrak. Lalu, bagaimana caranya apabila kita mau membuka file tersebut melalui command line?
            </span>
            <br/>
            <b>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </b>
            <b>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Membuka atau Untar file ".tar.gz"
             </span>
            </b>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Langkah di bawah ini berasumsi bahwa nama file yang akan dibuka adalah
             <b>
              namafile.tar.gz,
             </b>
             silahkan ganti
             <b>
              namafile
             </b>
             dengan nama file asli yang ada di komputer/laptop anda.
            </span>
            <br/>
            <br/>
            <ol>
             <li>
              <span style="font-family: Trebuchet MS, sans-serif;">
               Dari terminal, pindah ke direktori di mana
               <b>
                namafile.tar.gz
               </b>
               berada
              </span>
             </li>
             <li>
              <span style="font-family: Trebuchet MS, sans-serif;">
               ketik perintah
               <b>
                tar -zxvf namafile.tar.gz
               </b>
               untuk mengekstrak pada direktori saat ini kita berada. Apabila ingin mengekstrak file tersebut ke direktori lain, gunakan parameter
               <b>
                -C
               </b>
               . Contoh:
               <b>
                tar /namafolder -zxvf namafile.tar.gz
               </b>
              </span>
             </li>
            </ol>
            <div>
             <b>
              <span style="font-family: Trebuchet MS, sans-serif;">
               Membuka atau Untar file ".tar"
              </span>
             </b>
            </div>
            <div>
             <ol>
              <li>
               <span style="font-family: Trebuchet MS, sans-serif;">
                Dari terminal, pindah ke direktori di mana
                <b>
                 namafile.tar
                </b>
                berada
               </span>
              </li>
              <li>
               <span style="font-family: Trebuchet MS, sans-serif;">
                Ketik perintah
                <b>
                 tar -xvf namafile.tar
                </b>
                untuk mengekstrak pada direktori saat ini kita berada. Apabila ingin mengekstrak file tersebut ke direktori lain, bisa menggunakan parameter
                <b>
                 -C
                </b>
                seperti contoh di atas, yaitu
                <b>
                 tar /namafolder -xvf namafile.tar
                </b>
               </span>
              </li>
             </ol>
             <div>
              <span style="font-family: Trebuchet MS, sans-serif;">
               Cara diatas bisa juga digunakan tidak hanya untuk distro Ubuntu, tapi bisa juga untuk distro Linux lain dan Unix.
              </span>
             </div>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              sumber:

               Pendrive Linux
              </span>
             </a>
            </div>

             </li>
             <li>
              <span style="font-family: Trebuchet MS, sans-serif;">
               menyimpan file di Cloud
              </span>
             </li>
             <li>
              <span style="font-family: Trebuchet MS, sans-serif;">
               menyediakan
               <a href="https://core.telegram.org/api" target="_blank">
                API
               </a>
               dan
               <a href="https://core.telegram.org/mtproto" target="_blank">
                Protoko
               </a>
               l gratis, sehingga memungkinkan kita membuat aplikasi Telegram kita sendiri
              </span>
             </li>
             <li>
              <span style="font-family: Trebuchet MS, sans-serif;">
               Bisa untuk mengirim file
               <i>
                sampai dengan
               </i>
               1,5 GB via chatting
              </span>
             </li>
            </ul>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Nah, saya tertarik dan baru saja menguji fitur yang terakhir saya sebutkan, ketika
              <a href="https://telegram.org/blog/shared-files" target="_blank">
               dirilis
              </a>
              tanggal 1 Februari kemarin.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Perangkat yang saya gunakan:
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              - Android Picophone X1 dengan OS Jelly Bean 4.1.2
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              - Telegram versi 2.4.0 yang bisa di download via Google Play Store.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              ketika mencoba mengirim film sebesar 1,1 GB ternyata muncul pesan dari Telegram bahwa besar file tidak boleh melebihi 1GB. Akhirnya saya mencoba mengirim file sebesar 864,9 MB ke teman saya si Gendonk.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              File tersebut saya kirim pukul 22:34 WIB, dan baru diterima oleh Gendonk pada pukul 05:10 WIB (membutuhkan waktu 6 jam 36 menit untuk mengirim file sebesar 860 Mega Byte).
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Oh iya, koneksi yang saya gunakan adalah Speedy dengan upstream 0,22 Mbps. Jadi, apakah waktu yang dibutuhkan tersebut termasuk lama? mari kita hitung.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              satuan file di atas adalah 860 Mega Byte, artinya 860 x 8 Mega bit = 6.880 Mega bit = 6.880.000.000 bit
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              kecepatan upload saya adalah 0,22 Mbps = 0,22 Mega bit per detik = 0,22 x 1.000.000 bit = 220.000 bit per detik
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              dengan membagi 6.880.000.000 bit dengan 220.000 bit per detik didapatkan 31.272 detik, alias 521,2 menit atau 8,6 jam.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              ternyata setelah dihitung dengan overhead 10% hasilnya memang kurang lebih segitu. Kesimpulan: fitur baru Telegram untuk mengirim file memang masih tergantung koneksi internet, tapi bisa menjadi salah satu alternatif mengirim file yang ukurannya cukup besar.
             </span>
            </div>
           </div>
          </section>

            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">

            <span style="font-family: Trebuchet MS, sans-serif;">

            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">

            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">

             <a href="http://bit.ly/1u8Ld2u" target="_blank">
              MySql

             sudo dpkg -i mysql-workbench-gpl-5.2.47-1ubu1204-i386.deb pada terminal,
            </span>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             hasilnya: pesan error yang saya dapat, package libzip2 dan libctemplate0 tidak ditemukan.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">

            </span>

             <br/>

          <h1 class="post-title">
           <a href="http://endoch.blogspot.com/2014/10/utorrent-di-ubuntu-1204.html">
            uTorrent di Ubuntu 12.04
           </a>
          </h1>
          <h3>
           <span class="fn" itemprop="author" itemscope="itemscope" itemtype="http://schema.org/Person">
            <a class="g-profile" href="https://www.blogger.com/profile/03411457852896816173" rel="author" target="_blank" title="author profile">
             <span itemprop="name">
              endoch
             </span>
            </a>
           </span>
          </h3>
          <h4>
           <meta content="http://endoch.blogspot.com/2014/10/utorrent-di-ubuntu-1204.html" itemprop="url"/>
           <a class="timestamp-link" href="http://endoch.blogspot.com/2014/10/utorrent-di-ubuntu-1204.html" rel="bookmark" title="permanent link">
            <abbr class="published" itemprop="datePublished" title="2014-10-06T19:50:00+07:00">
             10/06/2014 07:50:00 PM
            </abbr>
           </a>
          </h4>
         </div>
         <div class="post-body" id="post-body-1562741855537694880" itemprop="description articleBody">
          <section role="main">
           <div class="cap page columnize">
            <span style="font-family: Trebuchet MS, sans-serif;">
             Setelah kecewa dengan
             <a href="http://deluge-torrent.org/" target="_blank">
              Deluge
             </a>
             dan
             <a href="http://ugetdm.com/" target="_blank">
              uGet
             </a>
             di ubuntu 12.04, saya akhirnya berpindah ke uTorrent. Penyebabnya cukup sederhana, waktu
             <a href="http://yify-torrents.com/" target="_blank">
              Yify
             </a>
             merilis Xmen dan Transformers, kok proses download nya nggak jalan-jalan, padahal semua settingan sudah betul, untuk download file lain pun tidak ada masalah. Sebel kan ya, ini Xmen dan Transformers lho, dua filem yang hampir tidak mungkin nggak ada seeder nya.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Setelah puasa 40 hari 40 malam dan membaca tutorial
             <a href="http://www.ubuntututorials.com/install-utorrent-ubuntu-12-04/" target="_blank">
              ini
             </a>
             , akhirnya saya memutuskan untuk mencoba menginstall uTorrent. langkah demi langkah di bawah ini adalah sepenuhnya hasil translate dari tutorial yang saya baca. Hihihi.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             1. Kunjungi situs resmi
             <a href="http://www.utorrent.com/" target="_blank">
              uTorrent
             </a>
             , lalu klik Get Utorrent untuk mendownload installernya.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             2. buka terminal (maksudnya bukan terminal giwangan, tapi di ubuntu 12.04 bisa pake shortcut
             <b>
              Ctrl+Alt+t
             </b>
             ) lalu ketikkan perintah di bawah ini untuk mengekstrak (bukan kulit manggis) file instalasi tadi:
            </span>
            <br/>
            <span style="background-color: #f9cb9c; font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             sudo cp ~/Downloads/utorrent-server-alpha-v3_3 /opt/
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             cd /opt
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             sudo tar -xzf utorrent-server-alpha-v3_3.tar.gz
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             sudo ln -s /opt/utorrent-server-alpha-v3_3/utserver /usr/sbin/utserver
            </span>
            <br/>
            <span style="background-color: #cccccc;">
             <span style="font-family: Trebuchet MS, sans-serif;">
              sudo rm -f
             </span>
             <span style="font-family: 'Trebuchet MS', sans-serif;">
              utorrent-server-alpha-v3_3.tar.gz
             </span>
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             3. Install libssl0.9.8 dengan perintah berikut ini:
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             sudo apt-get install libssl0.9.8
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <span style="background-color: white;">
              4. Nyalakan utserver dengan perintah berikut:
             </span>
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <span style="background-color: white;">
              <br/>
             </span>
            </span>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             utserver -settingspath /opt/utorrent-server-alpha-v3_3/
            </span>
            <br/>
            <span style="background-color: #cccccc; font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="background-color: white; font-family: Trebuchet MS, sans-serif;">
             5. Akses utorrent client melalui web browser (Firefox atau Chrome)
            </span>
            <br/>
            <span style="background-color: white; font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <span style="background-color: #cccccc;">
              localhost:8080/gui
             </span>
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <span style="background-color: #cccccc;">
              <br/>
             </span>
            </span>
            <span style="background-color: white; font-family: Trebuchet MS, sans-serif;">
             username nya adalah admin, tanpa password.
            </span>
           </div>
          </section>

           <div class="cap page columnize">
            <span style="font-family: Trebuchet MS, sans-serif;">
             Saat ini, sistem operasi windows terbaru adalah 8.1 update 1, namun saya masih saja setia menggunakan windows xp sp 3. iya, saya memang orang yang setia.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Sejak desember 2013, saya akhirnya beralih menggunakan hp android (Axioo Picopad 4 GDK yang dilengkapi dengan os Jelly Bean), delapan bulan kemudian, saya baru memiliki kebutuhan untuk terhubung ke internet melalui hp (tether) android saya. Maklum, habis pindahan, dan di tempat baru belum ada koneksi internetnya.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Biasanya, saya tether hp saya itu ke laptop (os ubuntu 12.04), tinggal colok kabel usb nya, aktifkan menu tether di android, beres. Lain halnya ketika saya mau tether ke pc saya yang masih windows xp sp 3 tadi. Windows hanya bisa mengenali android sebagai media penyimpanan, memang sih, file-file yang berada di media card hp, bisa di browse dengan normal melalui windows explorer.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Tapi tidak begitu untuk urusan tethering. ketika saya hubungkan android dan windows xp sp 3, si windows meminta saya untuk menginstall driver. Seperti yang sudah diduga, fitur mencari driver otomatis di windows xp memang rada sedikit terbelakang, hasilnya: nihil. Kemudian saya berusaha mencari dus hp axioo. Ketemu. Harap-harap cemas ada dvd mungil isi driver, ternyata nihil juga. #sedang #apess.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             setelah semedi mencari wangsit, saya menemukan kasus yang serupa tapi tak
             <a font-family:="" href="http://karceh.blogspot.com/2012/05/how-to-usb-tether-android-phones-with.html" ms="" rebuchet="" sans-serif="" target="_blank">
              sama
             </a>
             . Bedanya, penulis tersebut menggunakan hp Samsung Galaxy S2. Memang tidak disebutkan, os android dan windows nya versi berapa, tapi berdasarkan

              hotfix
             </a>
             dari microsoft, penulis tersebut mungkin menggunakan xp sp2 atau sp3. Setelah mencoba memahami sedikit apa itu
             <a href="http://www.microsoft.com/whdc/device/network/NDIS/rmNDIS.mspx" target="_blank">
              RNDIS
             </a>
             , saya mengambil kesimpulan, teknologi tersebut memungkinkan berbagai macam hp bisa menggunakan satu driver saja untuk tethering ke windows xp.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Konfigurasi awal yang saya coba cukup seadanya. artinya, pas kebetulan ngeliat ada kabel usb blackberry, ya saya pake kabel itu. step berikutnya, di depan mata ada usb hub sederhana (dari satu usb port, jadi 2 interface usb dan 1 micro usb), saya hubungkan kabel usb blackberry ke usb hub, kemudian ke hp android saya.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Semua step saya ikuti dengan perlahan dan seksama untuk menginstall driver thetherxp.inf, hasilnya gagal. Saya baca ulang referensi yang tadi, sambil mencari referensi lain. Akhirnya ada statement bahwa saya harus menggunakan kabel usb original bawaan dari hp.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Sayapun mengubah konfigurasi, kabel usb original saya hubungkan langsung dengan port usb motherboard (tanpa melalui usb hub), baru ke hp android. Semua step untuk menginstall driver saya ulang lagi, gagal. Saya ulang sekali lagi, dan baru berhasil.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>

            <span style="font-family: Trebuchet MS, sans-serif;">

            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">

            </span>
            <br/>

           <a href="http://endoch.blogspot.com/2014/06/politik-dan-kegemukan.html">
            Politik Dan Kegemukan
           </a>
          </h1>
          <h3>
           <span class="fn" itemprop="author" itemscope="itemscope" itemtype="http://schema.org/Person">
            <a class="g-profile" href="https://www.blogger.com/profile/03411457852896816173" rel="author" target="_blank" title="author profile">
             <span itemprop="name">
              endoch
             </span>
            </a>
           </span>
          </h3>
          <h4>
           <meta content="http://endoch.blogspot.com/2014/06/politik-dan-kegemukan.html" itemprop="url"/>
           <a class="timestamp-link" href="http://endoch.blogspot.com/2014/06/politik-dan-kegemukan.html" rel="bookmark" title="permanent link">
            <abbr class="published" itemprop="datePublished" title="2014-06-05T11:36:00+07:00">
             6/05/2014 11:36:00 AM
            </abbr>
           </a>
          </h4>
         </div>
         <div class="post-body" id="post-body-9091038886137077926" itemprop="description articleBody">
          <section role="main">
           <div class="cap page columnize">
            <span style="font-family: Trebuchet MS, sans-serif;">
             Menjelang Pilpres 2014, semua orang berbicara tentang politik. Sayangnya, belum ditemukan alat untuk mengukur berapa banyak energi (kalori) yang dibutuhkan ketika kita sedang berbicara (baca: berdebat) tentang siapa calon terbaik yang layak menjadi Presiden Indonesia. Meski begitu, kita tidak bisa memungkiri bahwa telah terjadi banyak gesekan energi, baik itu di dunia maya maupun nyata.
            </span>
            <br/>
            <span style="font-family: Trebuchet MS, sans-serif;">
             <br/>
            </span>
            <span style="font-family: Trebuchet MS, sans-serif;">
             Apakah disaat banyak gesekan energi yang terjadi, lantas penjualan alat musik gesek juga naik omsetnya? Saya tidak tau.
            </span>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             Satu hal yang saya tau, sekarang saya tambah gemuk, dan nampaknya selama 30th belakangan ini, kegemukan (obesitas) menjadi masalah di dunia.
            </span>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             <br/>
            </span>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             Kemudian polusi pertikaian pilpres semakin sering terjadi. Tadinya teman, lantas jadi perang dingin. Tadinya tidak kenal, lantas jadi musuh. Akhirnya saya lebih tertarik mencari tau penyebab perut saya yang semakin gendut.
            </span>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             <br/>
            </span>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             Jawaban itu saya temukan dari video presentasi Robert H. Lustig, profesor pediatrik di UCSF (University Of California, San Fransisco). Presentasi beliau yang berdurasi 1 jam 29 menit tersebut membuat saya beberapa kali mengalami momen WOW. (videonya bisa ditonton di
            </span>
            <span style="background-color: white; color: #006621; font-family: arial, sans-serif; font-size: 13px; line-height: 16px; white-space: nowrap;">
             www.
            </span>
            <b style="background-color: white; color: #006621; font-family: arial, sans-serif; font-size: 13px; line-height: 16px; white-space: nowrap;">
             youtube
            </b>
            <span style="background-color: white; color: #006621; font-family: arial, sans-serif; font-size: 13px; line-height: 16px; white-space: nowrap;">
             .com/watch?v=dBnniua6-oM
            </span>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             )
            </span>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             <br/>
            </span>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             Jebule, wabah kegemukan yang melanda di dunia, tidak lepas dari kisah politik dan pilpres. Dr Lustig memaparkan 3 kejadian politik yang turut berperan, yaitu:
            </span>
            <br/>
            <br/>
            <ol>
             <li>
              <span style="font-family: 'Trebuchet MS', sans-serif;">
               Pada tahun 1972, harga pangan di amerika sangat bergejolak, sampai-sampai Richard Nixon (pada saat itu adalah presiden AS ke 37, periode 1969 -1974) kawatir posisinya sebagai presiden bisa LENGSER. Nixon lalu memerintahkan Earl Butz (mentri pertanian pada saat itu) untuk membuat harga pangan menjadi murah, sehingga harga bahan pangan tidak boleh menjadi masalah pada saat pilpres.
              </span>
             </li>
             <li>
              <span style="font-family: 'Trebuchet MS', sans-serif;">
               Gula jagung (High Fructose Corn Syrup) yang ditemukan di Jepang pada tahun 1966 mulai memasuki pasar amerika pada tahun 1975. Dengan adanya gula jagung, harga gula biasa yang tadinya bergejolak bisa diredam karena harga gula jagung sangat murah sekali. Sejak saat itu, gula jagung digunakan pada hampir semua makanan olahan (pabrikan). Kenapa gula jagung dari jepang berkaitan dengan politik amerika? Perlu kita ingat bahwa jepang mengebom pearl harbor pada 7 Desember 1941. Kemudian amerika membalasnya pada tanggal 6 &amp; 9 agustus 1945, hiroshima dan nagasaki luluh lantak oleh "Little Boy" dan "Fat Man". Setelah jepang menyerah, posisi amerika menjadi penting dalam perekonomian jepang. Menurut analisa saya, masuknya gula jagung ke amerika adalah salah satu pertanda mereka baikan setelah bom boman gitu.
              </span>
             </li>
             <li>
              <span style="font-family: 'Trebuchet MS', sans-serif;">
               Kebijakan diet rendah lemak yang digagas oleh USDA (US Department of Agriculture), AMA (American Medical Association), dan AHA (American Heart Association) ternyata memiliki kesalahan silogisme. Untuk poin ketiga ini, saya agak kesulitan memahaminya karena berkaitan dengan multi linear regression analysis. mungkin bisa dipelajari lebih lanjut di videonya saja ya.
              </span>
             </li>
            </ol>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Nyebelin juga ya, mau menghindari topik politik, lha kok ternyata masalah kegemukan juga ada kaitannya sama politik.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Oke, intinya, penyebab epidemi kegemukan adalah kelebihan konsumsi fruktosa dan kekurangan makanan berserat.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              yang menarik bagi saya, Fruktosa diolah oleh tubuh sama persis dengan etanol (biasa didapatkan dari minuman beralkohol). Bedanya, fruktosa tidak memiliki efek memabukkan. Pada video tersebut, Dr Lustig memaparkan bagaimana etanol dan fruktosa diolah oleh tubuh, dan apa hubungannya ketika kita banyak mengkonsumsi fruktosa dan etanol bisa berakibat pada sakit diabetes, tekanan darah tinggi, obesitas, dll.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              Anehnya, konsumsi fruktosa tidak ada regulasinya. ini berbahaya. jadi saya berkesimpulan, mengkonsumsi minuman beralkohol, jauh lebih aman ketimbang mengkonsumsi HFCS yang bisa ditemukan pada banyak makanan dan minuman pabrikan.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              kesimpulannya apa? daripada berantem tentang politik, mending tonton saja deh videonya ya.
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <div>
             <span style="font-family: Trebuchet MS, sans-serif;">
              <br/>
             </span>
            </div>
            <br/>
            <span style="font-family: 'Trebuchet MS', sans-serif;">
             <br/>

             <br/>

          <h1 class="post-title">
           <a href="http://endoch.blogspot.com/2013/07/kesulitan-berdiri-tegak.html">
            Kesulitan Berdiri Tegak
           </a>
          </h1>
          <h3>
           <span class="fn" itemprop="author" itemscope="itemscope" itemtype="http://schema.org/Person">
            <a class="g-profile" href="https://www.blogger.com/profile/03411457852896816173" rel="author" target="_blank" title="author profile">
             <span itemprop="name">
              endoch
             </span>
            </a>
           </span>
          </h3>
          <h4>
           <meta content="http://endoch.blogspot.com/2013/07/kesulitan-berdiri-tegak.html" itemprop="url"/>
           <a class="timestamp-link" href="http://endoch.blogspot.com/2013/07/kesulitan-berdiri-tegak.html" rel="bookmark" title="permanent link">
            <abbr class="published" itemprop="datePublished" title="2013-07-13T04:28:00+07:00">
             7/13/2013 04:28:00 AM
            </abbr>
           </a>
          </h4>
         </div>
         <div class="post-body" id="post-body-2126072945000710370" itemprop="description articleBody">
          <section role="main">
           <div class="cap page columnize">
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Seandainya ada yang kesulitan untuk berdiri tegak seperti saya, ada baiknya periksa kondisi tulang belakang. Iya, saya divonis bungkuk oleh pak Santo, ahli pijat tulang yang cukup terkenal di jogja. Persisnya, vonis ini terjadi hari kamis, dua minggu yang lalu.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Pikiran saya lalu kembali ke jaman SMP. Waktu itu, bapak saya sering sekali mengingatkan perihal posisi berdiri saya yang bungkuk. Malahan, beliau sempat bilang,
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              "apa perlu dipaku ke kayu biar ndegeg?"
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              sambil kemudian dua tangannya menarik pundak saya kebelakang. Pernah juga, tulang punggung saya sekalian disapa oleh dengkul beliau. Kebayang kan posisinya? Nah, posisi dan kalimat yang beliau ucapkan itu masih sering terbayang-bayang dalam pikiran saya sampai sekarang.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Pernah juga, semasa di jakarta, saya sampai diajarkan berdiri tegak oleh salah satu rekan kerja. kaki dibuka selebar pundak, membentuk posisi segitiga sama sisi, begitu resep yang diturunkannya. Memang, posisi berdiri saya jarang sekali posisi sempurna. Posisi kebangsaan saya adalah kaki kanan lurus selebar pundak, sementara kaki kiri 'dibuang' agak jauh. Mirip posisi segitiga siku-siku, dengan kaki kiri saya menjadi sisi miringnya. Selalu begitu.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Empat minggu sebelumnya, kapalan di jempol telapak kaki kiri saya mulai sakit. Saya tau, normalnya, jempol kaki adalah pusat keseimbangan manusia ketika berdiri atau berjalan. Bertahun-tahun sebelumnya, saya cuma heran, kenapa yang kapalan cuma jempol kaki kiri saja sementara yang kanan tidak? Bukankah kulit yang kapalan seharusnya lebih tahan banting dibandingkan yang tidak? Kenyataanya, dipegang biasa saja, saya merasakan sakit, sehingga untuk berjalan saya harus secara sadar memindahkan titik berat tubuh ke telapak kaki yang berhubungan langsung dengan syaraf paru-paru.
             </span>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Sampai akhirnya hari itu tiba. hari kamis dua minggu yang lalu. Rasa sakit di pinggang kiri bawah saya tak tertahankan lagi. Sakit yang membuat saya tak nyaman untuk beraktivitas apapun. Saya memutuskan untuk periksa ke pak Santo.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Saya cuma diminta untuk telungkup, dipegang bagian tulang belakang sebentar, lalu keluarlah vonis itu, ditambah lagi kaki saya dibilang panjang sebelah. Pemandangan kaki saya yang panjang sebelah itu dinikmati oleh teman-teman saya. Sementara pikiran saya masih asik, bahwa akhirnya saya menemukan penyebab kapalan di jempol kaki kiri dan penyebab pinggang kiri saya yang sakit sekali.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Proses saya dipijat, tak sampai satu menit. biayanya tujuh puluh ribu rupiah. Setelah dipijat pertama kali, posisi tulang saya mengalami perbaikan. kaki tak lagi panjang sebelah. Saya dianjurkan untuk datang ke pak Santo sebanyak tiga puluh kali.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Perlahan-lahan, kapalan di jempol kaki kiri saya berangsur normal. Mau tak mau, saya jadi lebih memperhatikan posisi berdiri, duduk, dan tidur. Semua demi pertahanan terbaik dari sakit pinggang yang aduhai rasanya itu.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Perihal miringnya tulang belakang saya ini seolah menjawab beberapa pertanyaan saya. Salah satunya, kenapa kalo saat duduk jegang, saya lebih nyaman dengan kaki kiri yang diatas. Posisinya, dengkul kanan menopang mata kaki kiri. Posisi lain, paha kiri bagian belakang ditopang dengkul kanan. Iya, selalu merasa nyaman begitu, tapi tidak sebaliknya.
             </span>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Belakangan, saya merasa kurang puas dengan foto-foto dimana ada saya didalamnya, seringkali tampak bagian kepala dan leher saya nggak fotogenic. Kepala saya selalu maju, seolah kamera menjadi magnet bagi kepala saya ini. Setelah itu, saya selalu merasa leher saya kok gendut. Ternyata, penyebab dua hal ini ya karena tulang belakang itu tadi.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Mendengar cerita dari kawan lain yg kebetulan punya masalah sama dengan tulang belakangnya, tapi terapi dengan orang lain, akibat dari tulang belakang yang tak sempurna ini efeknya bisa kemana-mana. Contohnya, pencernaan. Kawan saya itu, setelah makan biasanya sekitar ulu hati terasa sakit, ternyata ya memang masuk akal, karena posisi tulang belakang yg bungkuk, menekan ulu hati. Kemudian, ketika dia membuka mulutnya, akan muncul bunyi dari arah rahang bagian belakang (kurang lebih sekitar geraham yg paling belakang), hal ini juga efek dari tak sempurnanya tulang belakang.
             </span>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Tulang belakang yang tak sempurna itupun membuat otot menjadi bekerja lebih keras, karena seolah ditarik terus menerus, akibatnya tubuh menjadi butuh waktu istirahat yang lebih lama (baca: bagi saya waktu tidur yang lebih lama, minimal delapan jam setiap hari)
             </span>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Pelajaran yang saya dapat, umur tiga puluhan kita mulai harus bisa menerima keterbatasan badan. Masalah tulang belakang ini hanya salah satunya. Sementara umur empat puluhan, kita harus bisa menerima keterbatasan jiwa.
             </span>
             <br/>
             <br/>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Ah, sudahlah, saya mau tidur saja. Otot-otot saya sudah minta istirahat ...
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <i>
               ps: ketika menulis posting ini, jadwal pak Santo praktek adalah setiap hari kamis malam di kantor notaris depan gereja kranggan (tugu ke utara), sementara setiap selasa pagi beliau praktek di depan wirogunan. Satulagi tempat prakteknya ada di dekat prambanan, tapi saya tak tahu detailnya.
              </i>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>

             <span style='font-family: "Trebuchet MS",sans-serif;'>

             </span>

            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS",sans-serif;'>
              <br/>
              446. Bank Mandiri
              <br/>
              461. Bank Rakyat Indonesia
              <br/>
              613. Bank Central Asia
              <br/>
              685. Telkom Indonesia
              <br/>
              922. Bank Negara Indonesia
              <br/>
              1188. Perusahaan Gas Negara
              <br/>
              1378. Gudang Garam
              <br/>
              1425. Semen Indonesia
              <br/>
              1453. Bank Danamon Indonesia
              <br/>
              <br/>
              Jadi, dari 2000 perusahaan publik terbesar sedunia, ada 9 dari Indonesia. 5 bergerak di bidang perbankan, sisanya bergerak pada bidang telekomunikasi, pertambangan, rokok, dan industri semen. Kemudian, saya melongok website BEI**, dan menemukan ada 467 perusahaan publik yang terdaftar. Artinya, dari jumalah emiten sebanyak itu, hanya kurang dari dua persen yang masuk Global 2000. Anehnya lagi, saat ini jumlah pengusaha di Indonesia juga di angka dua persen. Ada hubungannya?
              <br/>
              <br/>
              Tadinya saya ingin membuat tulisan yang agak panjang, tapi perhatian saya tersita pada sendok makan nara yang baru, ujung sendoknya berubah putih (semula berwarna biru muda) ketika terkena air panas. Sendok makan baru ini memang berfungsi untuk menjaga agar makanan bayi tidak terlalu panas ketika masuk mulut. Untuk membuktikan fitur ini, saya menuang air panas segelas kecil, mencelupkan dua buah sendok, dan memperhatikan perubahan warna yang terjadi.
              <br/>
              <br/>
              iya, sore ini, berita tentang Perusahaan Indonesia yang masuk Global 2000 masih kalah oleh sendok makan Nara yang baru.

            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Pada hari yang istimewa, kita bisa membuat serangkaian kata-kata manis, hadiah yang mahal harganya, makan malam mewah, atau berbagai macam bentuk perayaan lainnya. Bagi saya, dan tentunya masih yang saya sukai sampai sekarang, adalah menyempatkan waktu untuk berdoa bersama.
             </span>
            </div>
            <br/>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Kemaren, 20 april 2013, ibu suri genap berusia 51 tahun. Setelah bangun, saya mengucapkan "selamat ulang tahun tante", dan ibu suri hanya tertawa. Kami berpelukan, lalu cium pipi kanan kiri. Pagi itu, dia sedang masak sop sosis kesukaan kami bertiga. Saya memanggilnya tante, karena adik saya sedang mengajak pacarnya, si Tata, untuk liburan di jogja. Meski tidak mendengar secara langsung bagaimana Tata menyebut ibu suri ketika mengucapkan selamat ulang tahun, Hihihi.. saya tergoda menyindirnya dengan panggilan Tante.
             </span>
            </div>
            <br/>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Saya tau, sebutan tante itu mungkin tak lama lagi akan segera berubah menjadi Mama atau Mamah. Senang aja rasanya bahwa saudara kandung saya akhirnya mengenalkan pada keluarga kami, seseorang yang spesial di hatinya. Memang, adik saya belum membuka obrolan sama sekali perihal dia akan segera menikah. Tapi, saya akan menyampaikan langsung ke adik saya, bahwa saya tidak keberatan seandainya saya akan dilangkahi.
             </span>
            </div>
            <br/>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Kembali lagi perihal ulang tahun ibu suri, siangnya, saya tanya umur dia sekarang berapa. Meski saya tahu, jawabannya adalah 51. Setelah menjawab, ibu suri cerita bahwa ibunya meninggal di umur 50 atau 51, setelah sakit stroke 8 tahun. Ketika saya bilang bahwa ibu suri sudah berhasil mematahkan karma penyakit keluarga, dia cuma tersenyum, dan bilang semoga. Iya, itu maksud pertanyaan saya. Cerita dibalik angka umur ibu suri.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Malamnya, kami bersebelas kumpul untuk makan malam keluarga. Nara tentunya juga ikut. Ditambah 4 kakak ibu suri yang khusus datang dari banjarnegara. Setelah selesai makan malam, Nara sekeluarga langsung pulang ke ceper. Om dan tante saya, mampir ke rumah sebentar, lalu langsung pulang ke banjarnegara.
             </span>

              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Setelah itu, saya mengajak ibu suri untuk berdoa bersama. Adik saya kaget, tapi lalu dia bergabung. Saya ingat betul, ini adalah pengalaman doa bersama kedua di bulan April.
             </span>
            </div>
            <span style='font-family: "Trebuchet MS", sans-serif;'>
             <br/>
            </span>
            <span style='font-family: "Trebuchet MS", sans-serif;'>
             <i>
              ps: wahai ibu kita Kartini, selamat hari minggu ya, jangan lupa minum-minumnya...

          </h1>
          <h3>
           <span class="fn" itemprop="author" itemscope="itemscope" itemtype="http://schema.org/Person">
            <a class="g-profile" href="https://www.blogger.com/profile/03411457852896816173" rel="author" target="_blank" title="author profile">

         <div class="post-body" id="post-body-1527730049035358172" itemprop="description articleBody">
          <section role="main">
           <div class="cap page columnize">
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Pernah ketemu orang yang
              <i>
               physically
              </i>
              ,
              <i>
               emotionally
              </i>
              , dan
              <i>
               spiritually compatible
              </i>
              ?
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Saya sedang merasa menemukan itu. Untungnya dalam wujud lawan jenis sehingga saya tidak harus berhadapan dengan ketakutan saya untuk memiliki orientasi seksual (maaf) sejenis. Kali ini, saya ingin berbagi perihal kompatibilitas yang ketiga, yaitu secara spiritual
             </span>
             .
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Tunggu, jangan berpikir bahwa spiritualitas diantara kami adalah hal-hal yang suci, justru kebalikannya. Kami tengil, menggunakan bakat untuk menebak nomer togel. Togel?? iya, togel. Permainan semacam lotere dimana kita memasang angka tertentu dan berharap mendapatkan hadiah berkali-kali lipat. Mungkin untuk tata cara permainan togel, bisa di googling aja kali ya? :D
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Kejadian ini baru saja terjadi, tepatnya tanggal 16 maret 2013. Sebut saja namanya si tukang label. Biasanya, si tukang label selalu tidur dan sering mandi tengah malam, biar sakti, begitu ungkapnya. Hanya saja, malam itu dia tidur lebih awal, si tukang label tidur jam 12 malam, lalu terbangun jam 2, 5, dan 6. Obrolan kami pagi itu masih seputar bakat. Si tukang label, beberapa tahun belakangan ini sedang berjuang mengerem bakatnya, sementara saya belum lama menggunakan bakat saya yang menurutnya adalah hal tidak penting untuk dilakukan. Singkat cerita, saya tergelitik untuk membuktikan kemampuan kami berdua. Akhirnya kami sepakat untuk memilih 1256 untuk diuji coba di togel. Saya langsung teringat pak Nunuk (karyawan @fellicitasemtc) yang memiliki kebiasaan untuk memasang nomer togel setiap hari. Oh iya, pak Nunuk juga pandai bermain catur, saya pernah main sekali dan langsung sadar kalau kami berdua memiliki level kemampuan bermain catur yang jauh sekali berbeda. Maksudnya, saya kalahan gitu.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              16 Maret, siang hari, saya BBM ke @fellicitasemtc, saya bilang titip pesen untuk pak Nunuk, nomernya 1256, monggo dikombinasi sesuai isi hati. Si
             </span>
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              @fellicitasemtc yang temen saya semenjak tk itu kemudian berusaha memperjelas perihal angka 1256 harus dipasang posisinya bagaimana, jawaban saya sama, monggo dikombinasi atau dipilih sesuai isi hati yang masang.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Malam harinya, saya dikabari kalau angka yang keluar adalah 1278. Saya senyum-senyum sendiri, kemudian ngabarin si tukang label. Saya sebenarnya tidak terlalu kaget dengan hasil itu, senang sih iya, tapi bukan kaget. Bisa 'menebak' 2 angka dari 4, sementara dua angka sisanya cuman beda tipis, antara 56 dan 78. Saya sendiri tidak bisa menebak pasti ekspresi si tukang label waktu tau angka togel yang keluar malam itu, iya seringkali memang dia pandai memutarbalikkan fakta sehingga saya sering bingung yang benar yang mana. Malam itu kami sepakat untuk tidak bermain-main lagi dengan bakat kami.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Hari-hari berikutnya, perihal menebak angka togel ini hanyalah menjadi bahan bercandaan kami berdua. Sampai akhirnya 21 maret 2013, sore hari, kebetulan @fellicitasemtc menghubungi saya lewat telpon, bertanya tentang berapa lama proses transfer dari bank BRI ke BCA. Di akhir pembicaraan tersebut, saya menanyakan kemaren hasil pak Nunuk bagaimana, dia menegaskan kalau angka aslinya di hongkong ada 6 dan togel di indonesia hanya mengambil 4 angka belakangnya saja. Pesan terakhirnya sebelum menutup telpon adalah: seandainya ada feeling angka berapa lagi yang akan keluar, dia minta untuk dikabari.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Malam harinya, saya rada iseng karena penasaran, kemudian membuka webiste www.hongkongpoools.com, saya foto hasilnya, kemudian saya kirimkan ke si tukang label. Angka yang keluar tanggal 16 maret 2013 adalah 561278. "Holy Crap" adalah komentar si tukang label yang saya ingat pada saat itu. Beberapa menit kemudian saya baru sadar bahwa 2 angka di depannya adalah angka 56, angka yang kami berdua tebak. Saya terdiam dan tak mampu berkata-kata.
             </span>
            </div>
            <div style="text-align: justify;">

Padahal waktu itu kami berdua sama sekali tidak serius dan tidak konsentrasi. Tapi tenang, kami berdua sepakat untuk tidak bermain togel dan tidak menebak angka togel lagi.
             </span>
            </div>

            </div>
            <div style="text-align: justify;">

              Ihihihi... apakah kejadian diatas sekedar kebetulan? entahlah, alam semesta memang memiliki caranya sendiri, sementara kami berdua hanyalah bagian kecil yang ada di dalamnya.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>

Sampai saya menuliskan postingan ini, jumlah CCIE di dunia tidak sampai 40 ribu orang. Jangan tanya lah ya, di Indonesia ada berapa banyak, selain jumlahnya yang masih sedikit, saya juga tidak memiliki data yang akurat :D

             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Akhirnya saya berdamai dengan diri saya. Impian itu tidak terukir di atas batu. Impian itu bisa berubah dan kita tidak perlu merasa buruk karena memiliki impian yang berubah. Memiliki impian itu penting, tapi tanggung jawab juga tidak kalah pentingnya. Oh, percayalah, saya perlu waktu lama untuk berdamai dengan diri sendiri perihal impian ini.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Kenapa bisa begitu? mari, coba kita ingat kehidupan orang tua kita. Dalam kasus saya, kedua orang tua saya memang kurang beruntung perihal impiannya. Saat menuliskan ini, saya sedang tidak ingat apa impian bapak saya. Impian ibu saya waktu kecil adalah tidak ingin buka toko, lha kok sekarang malah buka toko. Kenapa bisa begitu?
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Bapak dan ibu saya menikah di usia yang cukup muda. Mungkin, itulah sebabnya mereka harus mengesampingkan impian mereka, untuk kemudian lebih mementingkan tanggung jawab sebagai orang tua, membesarkan kami bertiga. Bagi saya, orang tua adalah manusia biasa tapi dengan tanggung jawab lebih. Aneh ya, pandangan tentang orang tua, sementara saya sendiri belum menjadi orang tua.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Lalu saya berpikir, rejeki generasi sebelum saya, nampaknya memiliki pilihan impian yang terbatas. Saya sempat iri dengan generasi jaman penjajahan. Dalam imajinasi saya, impian generasi muda saat itu adalah berjuang mengusir penjajah, kalo perlu, berangkat perang. Simple sekaligus bikin saya iri. Iya, iri karena nggak perlu lah ya mikir mau mencapai apa dalam hidup, yang penting berjuang untuk negara, berangkat perang saat yang lain juga berangkat perang.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Bingung? Sudahlah, intinya saya cuman mau bilang, saya merasa saat ini saya harus lebih bertanggung jawab membalas budi ke orang tua. Wujudnya apa? nanti ya, bagusnya kalo sudah terwujud baru saya cerita.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <br/>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Jadi saya akan menyimpan impian itu, sama seperti para orang tua yang menyimpan impiannya untuk membesarkan anak-anaknya, dan apabila anak-anaknya sudah dewasa, bisa hidup mandiri, ternyata impian itu masih ada, maka saat itulah impian itu harus dikejar lagi. Dalam konteks saya, berlaku kebalikannya.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Sebetulnya, ketika saya bertanya lagi kenapa saya harus CCIE, saya merasa untuk ukuran orang dengan kecerdasan pas-pas an, saya ingin menjadi teladan bagi orang lain, bahwa mereka juga bisa jadi CCIE kalau mereka mau. Menginspirasi banyak orang. berguna bagi orang lain. Kali ini, saya ingin melakukannya dengan cara menuliskan hal-hal teknis yang saya bumbui dengan pengalaman saya selama menjadi network engineer. Tak banyak memang, bila dibandingkan dengan pengalaman network engineer lain, tak apalah, setidaknya saya tidak banyak berdiam diri, bergerak meski pelan dan tak berhenti. Jauh lebih masuk akal untuk saya saat ini ketimbang harus kembali menjadi network engineer setiap hari.
             </span>
            </div>
            <div style="text-align: justify;">
             <br/>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              Kesimpulannya, ada yang menemukan dirinya dalam pekerjaan, ada yang berkarya bagi orang banyak, ada yang menemukan dirinya melalui kisah cintanya. Perjalanan setiap orang pastinya berbeda-beda. Yang asik adalah ketika dalam perjalanan kita itu, kita menemukan orang lain yang tau tentang bakat kita, dan selalu mendukung kita untuk tetap berada pada jalur bakat tersebut, terlebih ketika kita sendiri sedang tidak yakin dengan diri kita. itulah the happiest thing in life.
             </span>
            </div>
            <div style="text-align: justify;">
             <span style="font-size: xx-small;">
              <i>
               <span style='font-family: "Trebuchet MS", sans-serif;'>
                <br/>
               </span>
              </i>
             </span>
            </div>
            <div style="text-align: justify;">
             <span style='font-family: "Trebuchet MS", sans-serif;'>
              <span style="font-size: xx-small;">
               <i>
                ps: kamu, iya kamu. kombinasi antara the saddest thing in life dan the happiest thing in life. Trimakasih ya...
               </i>
              </span>
             </span>
            </div>
            <div style="text-align: justify;">
            </div>
