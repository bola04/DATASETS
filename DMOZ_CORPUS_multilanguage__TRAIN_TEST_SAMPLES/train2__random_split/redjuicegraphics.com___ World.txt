          久しぶりの更新です。
          <br/>
          <br/>
          お仕事情報も溜まってってるようで溜まってないようで、
          <br/>
          色々告知できる事はあるんですが、とりあえず一つ自分の人生の中でも｢事件｣と言えるお仕事を頂けたので報告です。
          <br/>
          <br/>
          <a href="http://www.beams.co.jp/news/detail/4378" target="_blank">
           BOOM BOOM SATELLITES × redjuice Tシャツリリース
          </a>
          (BEAMS)
          <br/>
          <br/>
          <br/>
          <a href="http://www.bbs-net.com/" target="_blank">
           BOOM BOOM SATELLITES
          </a>
          のニューアルバム
          <br/>
          『SHINE LIKE A BILLION SUNS』
          <br/>
          に収録されている曲、
          <br/>
          <br/>
          <a href="https://www.youtube.com/watch?v=jlqTeCQehOs" target="_blank">
           『A HUNDRED SUNS』
          </a>
          (MV)
          <br/>
          <br/>
          をイメージしたイラストが、BEAMSとのコラボでTシャツになりました。
          <br/>
          ご好評いただき、BEAMS通販の方では即 SOLD OUT になってしまいましたが、
          <br/>
          2/9 18:00〜 からオフィシャルサイトの方で追加生産分の通販がスタートするとのことです。
          <br/>
          http://shop.mu-mo.net/a/list1/?categ_id=229803   (2/10追記：即日完売したそうです・・・)
          <br/>
          <br/>
          Tシャツのデザインは、僕の画集の装丁もしてくださった有馬トモユキさん (
          <a href="http://www.tatsdesign.com/" target="_blank">
           TATSDESIGN
          </a>
          )です。『SHINE LIKE A BILLION SUNS』のパッケージデザインも手がけられています。
          <br/>
          ご購入されるCDショップによっては、このイラストがアナザージャケットとして付属するなどの特典もありますので、
          <a href="http://www.bbs-net.com/news/" target="_blank">
           オフィシャルサイトのNEWS
          </a>
          でチェックしてみてください。
          <br/>
          <br/>
          自分がどのぐらいBOOM BOOM SATELLITESが好きか、という事を話そうとすると・・・どうも言葉が安くなってしまって多くは語れないのですが、2ndアルバム『UMBRA』(2001年)で衝撃を受け、約14年にわたってずっと追いかけてきたバンドです。14年前と言えば、まだ自分に絵心があったかどうか・・・という時代でもあってまさかこんなチャンスがあるなんて当時は冗談でも思いつきませんでした。
          <br/>
          <br/>
          川島さん、中野さん、関係者の方々、本当にありがとうございました。
          <br/>
          <br/>
          アルバムの内容については、僕の語彙では語りきれない、素晴らしく、特別な1枚になっています。
          <br/>
          これまでブンブンサテライツを聞いたことが無い、という人にも、是非聞いて欲しいです。
          <br/>
          <br/>
          <br/>
