 function obtenerCookie(c){
 	var existe = false
 	var galleta = document.cookie
 	var galletas = galleta.split(";")
 	for(var m=0; m<galletas.length;m++){
 		var gal=galletas[m].split("=")
 		var nom=gal[0]
 		var valor = gal[1]
 		var esp = ' ';
 		var re = new RegExp(esp, 'g');
 		nom = nom.replace(re, '');
 		if(nom === c){ 
 			existe = valor
 		}
 	}
 		return existe; 
 }

          </h3>
          <div class="cuadro_perso_texto">
           Hola amigos, desde la protectora baix vinalopó, queremos pedir ayuda a todos, os necesitamos!
 
 Actualmente la protectora pasa por un momento complicado el cual nos lleva a pedir ayuda, tanto de particulares como de empresas. Tenemos que abandonar las instalaciones en las que refugiamos gran parte de nuestros acogidos en breve y buscamos un terreno para poder construir un albergue donde seguir adelante con la labor que llevamos realizando ya más de 20 años. Necesitamos ayuda económica si alguien puede ayudar, para comprar un terreno, también localizar un terreno que supere a poder ser los 3000 metros cuadrados, que sea asequible y cercano a elche. 
 
 También una vez encontremos el terreno y la forma se pagarlo, habrá que construir de 0 unas instalaciones adecuadas para todos esos animales desamparados que buscan una oportunidad de de ser felices. 
 
 Por lo tanto os agradecemos cada ayuda, antes, aconsejandonos terrenos, ayudando económicamente o llegado el momento, donando materiales y mano de obra.  Más que nunca os necesitamos amigos. Gracias de antemano
 
 Si quieres darnos material , mano de obra o algo que crees que puede ser de utilidad escríbenos. Info@protectorabaix.org
