         <p align="left">
          <font face="Arial" size="2">
           Rezeptideen bis ans Lebensende ! Wozu noch
         Kochbücher wälzen, wenn im Internet jedes nur denkbare
         Rezept aus aller Herren Länder gratis bereitsteht ?
         Kalorio verwaltet Ihre Rezepte, die Ihrer Freunde und
         Bekannten und unterstützt den Austausch über das
         weltweite Internet. Alles, was Sie dazu benötigen, ist
         Kalorio und Ihre Küche.
          </font>
         </p>
         <p align="center">
          <font face="Arial" size="2">
           Kalorio ist
         für den Laien geschrieben. Wer ein Kochbuch lesen kann,
         wird auch mit diesem Programm keine Schwierigkeiten
         haben. Alle Angaben zu einem Rezept, die Suchergebnisse,
         Ihre Einkaufsliste usw. auf einen Blick. Testen Sie
         Kalorio! Es ist ohne jede Funktionseinschränkung hier
         gratis für einen Testzeitraum von 30 Tagen frei
         verfügbar.
          </font>
         </p>
         <p align="center">
          <font face="Arial" size="2">
           Lesen Sie die Meinungen
           unserer Kundschaft! Darunter finden Sie eine Zusammenfassung der
           herausragendsten Punkte. Oben im Menü können Sie gezielt in die
           Schwerpunkte von Kalorio Einsicht erhalten.

          <font face="Arial" size="3">
           <strong>
            <br/>
            <br/>
            <br/>
            Ein Auszug aus den Möglichkeiten von Kalorio:
           </strong>
          </font>
         </p>
         <ul>
          <li>
           <font face="Arial" size="2">
            Alles
                 in einem Fenster
            <br/>
            Kein ewiges Geklicke mehr mit der Maus, um das
                 Rezept endlich komplett anzuschauen. Dabei kann wählen, wie die
               Rezepte angezeigt werden sollen: Als klassischer Windows-Dialog,
               als selbst definierbares HTML, als normaler Text oder als
               Druckvorschau. Ausserdem lassen sich jederzeit nützliche Dialog
               jederzeit zublenden: Lexikon, Zutaten der Saison, Aufnahme von
               gegessenen Rezepten pro Tag, ein Wiochenplaner, die Einkaufsliste
               usw.
            <br/>
            <br/>
           </font>
          </li>
          <li>
           <a href="ffappklist.html" target="haupt">
            <font face="Arial" size="2">
             Die Erstellung von
                 Einkaufszetteln
            </font>
           </a>
           <font face="Arial" size="2">
            wird zum Kinderspiel
            <br/>
            Einfach Rezepte per Drag &amp; Drop
                 zusammensuchen und schon kann die Einkaufsliste
                 gedruckt werden
            <br/>
            Oder eine Einkaufliste für die ganze Woche direkt aus dem Wochenplaner? Kein Problem.
            <br/>
            Kalorio lernt sogar bei jedem Einkauf dazu und sortiert Ihre
               Einkaufsliste nach Händlern und Theken!
            <br/>
            <br/>
           </font>
          </li>
          <li>
           <font face="Arial" size="2">
            <a href="ffappcooler.html">
             Gefriergutverwaltung
            </a>
            : Ab jetzt müssen
               keine Lebensmittel in dem Mülleimer geworfen werden weil man sie
               in der Tiefkühltrühe vergessen hat.
            <br/>
            Verwalten Sie beliebig viele Gefrierschränke, definieren Sie die
               Anzahl und Bezeichnung ihrer Schubladen und verwalten Sie Inhalt
               und Haltbarkeitsdatum
            <br/>
            Lassen Sie Kalorio dazu lernen und vermeiden Sie dadurch unnötige
               Eingabe-Arbeit.
           </font>
           <br/>
           <br/>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Planen Sie ganze
               Menüs mit Kalorio! Mit der zuschaltbaren
             <a href="ffappmenu.html">
              Menüplanung
             </a>
             können Sie Rezepte und Getränke zusammenstellen, in Gänge
               aufteilen, auf die Personenanzahl automatisch umrechnen lassen und
               natürlich einen ansprechenden und anpassbaren Menüplan oder
               Speisekarte ausgeben lassen
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <a href="ffappexport.html" target="haupt">
             <font face="Arial" size="2">
              Import / Export
             </font>
            </a>
            <font face="Arial" size="2">
             über das neue XML-Standard-Format
             <a href="http://www.kalorio.de/cml/cookml.html">
              CookML
             </a>
             ,
               das die Übertragung aller Rezeptdaten inklusiver Bildern
               ermöglicht.
             <br/>
             MealMaster (TM) und RezKonv werden selbstverständlich auch unterstützt ohne
               daß irgendwlche Tools dazu installiert werden müssen..
             <br/>
             Dadurch weltweiter (und lokaler :-) Austausch
                 aller Rezepte möglich
             <br/>
             Import von mehreren Dateien gleichzeitig, auch
                 mit unterschiedlichen Formaten
             <br/>
             Quick-Report: Mit einem Mausklick Rezepte aus dem Clipboard importieren.
             <br/>
             Auto-Import: Automatischer Import aus allen Dateien, die in ein definierbares Verzeichnis kopiert werden. Keine weiteren Import-Aktionen notwendig.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <a href="ffapplex.html" target="haupt">
             <font face="Arial" size="2">
              Automatisches Lexikon
             </font>
            </a>
            <font face="Arial" size="2">
             : Ein erweiterbares Kochlexikon schaltet
                 sich bei Bedarf zu und zeigt online die im Lexikon vorhandenen
                 Fachbegriffe zum aktuellen Rezept an. Die Lexikon-Eintragungen
                 lassen sich in verschiedene Oberbegriffe aufteilen und werden
 		vom Autor von Version zu Version erweitert
             <br/>
             Wem das noch nicht reicht, der kann die Zutat oder die Kochtechnik
               mit einem Mausklick im weltweiten Online-Lexikon Wikipedia
               aufrufen zu dem Kalorio eine direkte Schnittstelle anbietet.
            </font>
           </p>
          </li>
          <li>
           <a href="ffappexport.html" target="haupt">
            <font face="Arial" size="2">
             HTML-Export
            </font>
           </a>
           <font face="Arial" size="2">
            Ihrer Rezepte in Ihre
                 eigene Homepage inklusive Linkliste problemlos
                 möglich. Auf Wunsch mit Frame-Technik. Eigene Anpassungen durch Kalorio-HTML-Erweiterungen sind möglich.
               Eine HTML-Ausgabe Ihres Rezeptes erfolgt mit einem Mausklick! Sie
               definieren, was und wie ihre Rezepte aussehen sollen.
            <br/>
           </font>
          </li>
          <li>
           <a href="ffappinput.html" target="haupt">
            <font face="Arial" size="2">
             3
               Editmodi
            </font>
           </a>
           <font face="Arial" size="2">
            : Ausführlicher Modus mit
                 genauer Zutateneingabe oder flexible Texteingabe
                 für Importe von gescannten oder HTML-Rezepten. Dabei wird ein
               unformatierter Text in Echtzeit untersucht und Kalorio zeigt ihnen
               während der Eingabe, wie er das rezept importieren würde und
               unterstützt Sie mit weiteren Angaben.
            <br/>
           </font>
          </li>
          <li>
           <p align="left">
            <a href="ffappsearch.html" target="haupt">
             <font face="Arial" size="2">
              Powersuche
             </font>
            </a>
            <font face="Arial" size="2">
             : Suche nach
                 Textausschnitten, Zutaten, Gattungen, Bilderm in einem
                 oder über alle Kochbücher gleichzeitig.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             <a href="faappcalc.html" target="haupt">
              Nährwertberechnung
             </a>
             : Berechnung der Kalorien pro Rezept und Portion auf 
 		Grundlage der erweiterbaren Zutaten-Datenbank. Mit Direktaufnahme von Kalorienangaben
 		und der Angabe von Ersatz-Zutaten für die Berechnung. Alternativ kann
               eine erweiterte Nährwertberechnung mit Hilde der
               Inhaltsstoffangaben aus dem Bundeslebensmittelschlüssel
               aufgerufen werden.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             <a href="faappnahr.html" target="haupt">
              Ernährungs-
               und Gewichtsberatung
             </a>
             : In Kalorio kann man beliebig viele
               Personen anlegen. Für jede Person kann man das Gewicht, Fettanteil,
               Blutzucker usw. graphisch verfolgen. Auch kann man unter Angabe ihrer
               Daten den Grundumsatz, den Body-Mass-Index bis hin zur
               Körperoberfläche berechnen lassen. Kalorio berechnet auch den
               Verbrauch von diversen Inhaltsstoffen aus. Jede Person kann einen
               eigenen Kalender anlegen, in dem er festhält, welche Rezepte und
               Lebensmittel er gegessen hat. Darauf aufbauend kann man sich alle
               berechnen lassen vom der Vitaminaufnahme bis hin zu den
               Broteinheiten.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             <a href="faappsaison.html" target="haupt">
              Saisonkalender
               / Frischekalender
             </a>
             : Was ist diese Woche wirklich frisch. Es
               werden alle Zutaten zum aktuellen Rezept in einer übersichtlichen
               Grafik dargestellt, für die gerade Haupt- oder Nebensaison ist.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <a href="ffappstatis.html" target="haupt">
             <font face="Arial" size="2">
              Statistik
             </font>
            </a>
            <font face="Arial" size="2">
             -Funktion
             <br/>
             Sie bleiben immer informiert, wieviele Rezepte,
                 Kochbücher, Gattungen, Zutaten, Einheiten usw.
                 verwaltet werden
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <a href="ffappwplan.html" target="haupt">
             <font face="Arial" size="2">
              Wochenplaner
             </font>
            </a>
            <font face="Arial" size="2">
             -Funktion
             <br/>
             Planen Sie Ihre Menüs und Aktionen auf Monate im Vorraus. Kalorio
 		unterstützt Sie dabei und erstellt Einkausfslisten kompletter Menüs oder
 		ganzer Wochen.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             <a href="faappbls.html" target="haupt">
              <b>
               Bundeslebensmittelschlüssel
              </b>
              :
             </a>
             Kalorio integriert den Bundeslebensmittelschlüssel vollständig.
               Er ist von der Zutatenliste bis zur Nährwertberechnung bei Bedarf
               ständig präsent und ermöglicht Ihnen Zugang zu allen möglichen
               Inhaltsstoffen Ihrer Rezepte und Zutaten, von der Kalorie über
               Cholesterinwerte bis hin zu Broteinheiten.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Rezeptanzeige mit
               Rezeptbildern, Zutatenbildern und Gattungssymbolen. Sofern
               gewünscht untersucht Kalorio automatisch ihr gerade angezeigtes
               Rezept und stellt die Bilder der darin enthaltenen Zutaten neben
               oder im Rezept dar. Ebenso für die Gattungen, die dem Rezept
               zugewiesen woren sind.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Massendatenänderungen;
               Wollen Sie nchträglich Gattungen oder Stichworte einem ganzen
               Kochbuch oder Teilen davon zuweisen oder gar entfernen? Oder
               Quellenangaben korrigieren? Geht alles mit einem Mausklick mit
               Hilfe der neuen Massendatenänderung.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Suche nach doppelten Rezepten
             <br/>
             Suchen Sie doppelte Rezepte mit einstellbaren Suchbedingungen. Sie können z.B. Rezepte suchen, die
                 mindestens 60% gleiche Zutaten besitzen, vom gleichen Autor geschrieben wurden usw.
               Doppelte Rezepte können entweder zum Vergleich in einer Liste
               dargestellt oder automatisch gelöscht werden (Priorität ist
               dabei pro Kochbuch einstellbar)
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Schnittstelle zu RezKonv
             <br/>
             In Kalorio ist die Möglichkeit für den Im.- und Export von
               RezKonv-Rezepten D´direkt eingebaut. Es müssen keine weiiteren
               Tools installiert werden.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Eigene
                 Datenbankentwicklung speziell für
                 Kalorio-Rezeptverwaltung
             <br/>
             Es wird keine Monsterdatenbank auf Ihrem Rechner
                 installiert. Kalorio bringt eine eigens
                 optimierte und schnelle Datenbank mit
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Verschieben
                 / Kopieren von Rezepten in andere Kochbücher mit
                 einem Mausklick zur sauberen Verwaltung Ihrer
                 Sammlung
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Kleine
                 Assistenten geben Hinweise bei nicht ganz so
                 offensichtlichen Möglichkeiten.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Fenster
                 frei vergrößerbar. Alle Anzeige- und
                 Auswahlfenster sind über Gridliners beliebig
                 verstellbar
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Automatische
                 einheitenbezogene Auswahl Ihrer Zutaten
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Automatische Ergänzung
 		der Einheiten und Zutaten bei der Rezepteingabe.
            </font>
           </p>
          </li>
          <li>
           <p align="left">
            <font face="Arial" size="2">
             Unterstützung
                 von Bildern Ihrer Kochkünste. Sogar die Zutatenbilder können
               eingepflegt und rezeptspezifisch ausgegeben werden. (
             <a href="appexpdemo.html" target="_blank">
              Beispiel
               hier
             </a>
             )
            </font>
           </p>

            <font face="Arial" size="2">
             <a href="faapplangua.html" target="haupt">
              Mehrsprachigkeit
             </a>
             :
               Kalorio gibt es mit deutscher oder englischer Oberfläche. Bei

               übersetzen.
            </font>
           </p>
          </li>
          <li>
           <p align="left">

                 möglich
            </font>
