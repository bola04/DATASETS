            <p>
             Καλώς ήρθατε στο ξενοδοχείο «
             <b>
              Summerland
             </b>
             » στο Καστράκι της Νάξου
             <br/>
             <br/>
             Ιδανικός προορισμός για τις καλοκαιρινές σας διακοπές στις Κυκλάδες
             <br/>
             Στο Καστράκι της Νάξου σας περιμένουν ατέλειωτες παραλίες με αμμόλοφους και κρυστάλλινα γαλαζοπράσινα νερά που εκτείνονται για χιλιόμετρα, ίσως από τις πιο μαγευτικές της Μεσογείου.
             <br/>
             <br/>
             Είναι ιδανική περιοχή για όσους αναζητούν ήρεμες, ξεκούραστες διακοπές, απολαμβάνοντας τον ήλιο και τη φύση, σ' ένα ήσυχο μέρος μακριά από τα θορυβώδη καλοκαιρινά θέρετρα.
             <br/>
             <br/>
             Δίπλα σε μια απέραντη παραλία με άσπρη άμμο, σας περιμένει το ξενοδοχείο
             <b>
              Summerland
             </b>
             και με το προσωπικό του υπόσχεται να σας προσφέρει τις καλύτερες υπηρεσίες, ώστε να κάνουν τη διαμονή σας όσο το δυνατόν πιο άνετη.
             <br/>
             <br/>
             Βρίσκεται μόλις 250 μέτρα από μια όμορφη μοναχική παραλία στο Καστράκι της Νάξου και περιλαμβάνει όμορφα και ευρύχωρα δωμάτια, στούντιο &amp; διαμερίσματα, όλα με a/c και ιδιωτικό μπαλκόνι, που θα σας χαλαρώσουν ακόμα και κατά την περίοδο του καλοκαιριού με τη μεγαλύτερη τουριστική κίνηση.
             <br/>
             <br/>
             Στην κεντρική περιοχή του ξενοδοχείου
             <b>
              Summerland
             </b>
             , κήποι, 2 πισίνες, υδρομασάζ, σνακ μπαρ, εστιατόριο, μπάρμπεκιου, πινγκ πονγκ, παιδική χαρά, εγκαταστάσεις υψηλών προδιαγραφών που θα κάνουν τη διαμονή σας αξέχαστη.
             <br/>
             <br/>
             Παράλληλα, έχοντας ως βάση το ξενοδοχείο
             <b>
              Summerland
             </b>
             , μπορείτε να περιηγηθείτε σε ολόκληρη τη Νάξο, να κολυμπήσετε σε μαγευτικές παραλίες, να επισκεφθείτε τα πολυάριθμα χωριά του νησιού και να θαυμάσετε τα πλήθος αξιοθέατα.
             <br/>
             <br/>
             Αν όμως είστε φίλοι των σπορ, υπάρχει ένα πλήθος δραστηριότητες με τις οποίες μπορείτε να ασχοληθείτε.
             <br/>
             <br/>
             Σε κάθε περίπτωση, η διαμονή σας στο ξενοδοχείο
             <b>
              Summerland
             </b>
             θα σας μείνει αξέχαστη.
