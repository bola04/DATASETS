      <h2>

       </a>
      </h2>
      <p class="mainboxp">
       <a href="04.html">
        <img border="0" class="mainboximg" height="180" src="img/centerbox/hn02.jpg" width="180"/>
       </a>
       ホテルデザインに携わって２８年。
       <br/>
       ホテルを知り尽くしたデザイナー達が ホテルインテリアをデザイン。
       <br/>
       世界のトレンドをリードし、 省エネで使いやすく人が集まる 魅力あるデザインをご提案いたします。
       <br/>
       <br/>
       実績
       <br/>
       <span>
        ホテルニューオータニ東京
        <br/>
        エグゼクティブハウス「ZEN」
        <br/>
        北京長冨宮飯店
        <br/>
        ハワイカイマナビーチホテル他
       </span>
      </p>
     </div>
     <div class="mainbox01">
      <h2>
       <a href="04.html">

       </a>
      </h2>
      <p class="mainboxp">
       <a href="04.html">
        <img border="0" class="mainboximg" height="180" src="img/centerbox/sekisui5.jpg" width="180"/>
       </a>
       住宅・マンション・別荘インテリア及びリニューアルのホテルライクなデザインを提案。
 パーソナルカラー・テーマカラーを分析し、スペースに適したカラーコーディネートし、さらに風水理論の元、健康空間・癒し空間をデザイン・コーディネートいたします。
       <br/>
       <br/>
       実績
       <br/>
       積水ハウス京都モデルルーム
       <br/>
       住友不動産神戸マンション
       <br/>
       モデルルーム他
      </p>
