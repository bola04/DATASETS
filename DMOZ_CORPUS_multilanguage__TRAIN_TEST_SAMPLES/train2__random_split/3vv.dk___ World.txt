                 <div class="content">
                  <div class="rtecenter">
                   <strong>
                    Vi er 3. Vesterbro Vildgæssene, Spejderne på Azaleavej
                   </strong>
                  </div>
                  <ul>
                   <li>
                    Vi bygger sauna og hjælper hinanden med at finde ting der er blevet væk
                   </li>
                   <li>
                    Vi skråler højt vores ynglingssange og fisker krabber
                   </li>
                   <li>
                    Vi sætter telte op både i Danmark og udlandet og holder jul i fremtiden
                   </li>
                   <li>
                    Vi finder vej i mørket og spiser grøn havregrød og lyserøde morgenboller
                   </li>
                   <li>
                    Vi deltager i verdens længste sæbekasseløb med Hogwarts Ekspressen
                   </li>
                   <li>
                    <strong>
                     <a href="blivspejder">
                      Vi vil altid gerne have nye spejdervenner
                     </a>
                    </strong>
                   </li>
                  </ul>
                  Spejderarbejdet handler om at skabe ansvarlige unge mennesker og give dem masser af positive oplevelser, som de kan få gavn af hele livet.  Kammeratskab og sjov er de vigtigste nøgleord i vores spejderarbejde, da vi gennem dette lærer børnene at tage ansvar for sig selv og hinanden. At være spejder handler ikke om at vinde eller være bedst, men om at kunne være sig selv. I en spejderpatrulje er der brug for alle slags personligheder og færdigheder, så ingen er for store eller små til at være med i fællesskabet. Rammen for alt dette er friluftslivet, som giver børnene gode og sunde udfordringer og masser af frisk luft.
                  <br/>
                  <br/>
                  Vores hytte ligger på
                  <a href="her-bor-3vv">
                   Azaleavej 26
                  </a>
                  - lige på grænsen mellem Valby og Frederiksberg, 5 minuter fra Zoologisk Have og Søndermarken. Vil du få vide mer om vores gruppe kan du
                  <a href="kontakt">
                   kontakte
                  </a>
                  vores gruppeleder
                  <strong>
                   Maria
                  </strong>
                  på
                  <strong>
                   gl
                   <img alt="" height="12" src="/userfiles/image/snabela.png" width="12"/>
                   3vv.dk
                  </strong>
                  <br/>
                  <br/>
                  Du kan også følge os på Facebook -
                  <a href="http://www.facebook.com/3.VesterbroVildgaessene/" title="www.facebook.com/3.VesterbroVildgaessene/">
                   www.facebook.com/3.VesterbroVildgaessene/
                  </a>
