          <br/>
          И, прежде всего, самого себя,
          <br/>
          Для человека – есть ориентир:
          <br/>
          Создатель, как Великое «Я»!
          <br/>
          <br/>
          Совершенствуясь всегда сам,
          <br/>
          Человек совершенствует Бога!
          <br/>
          Прекращается тогда бедлам,
          <br/>
          Порядок, как на Небесах, в итоге!
          <br/>
          <br/>
          Человек должен понимать Правила,
          <br/>
          Установленные Создателем!
          <br/>
          Каждая частица поступает правильно,
          <br/>
          Если творит, а не падает в обывателя!
          <br/>
          <br/>
          Расписана в Сценарии роль человека!
          <br/>
          Не на проявление, а навсегда!
          <br/>
          На человека не влияет окончание века,
          <br/>
          Важнее эволюции череда!
          <br/>
          <br/>
          Каждый шаг – есть шаг эволюции!
          <br/>
          Совершенство в каждом движении!
          <br/>
          Никаких не может быть революций,
          <br/>
          Любых, и без исключения!
          <br/>
          <br/>
          Матрица – есть человек в Боге!
          <br/>
          Неразрывная часть Создателя!
          <br/>
          Каноны важны для человека в итоге,
          <br/>
          Человек сейчас – себя открыватель!
          <br/>
          <br/>
          Только в Боге и никаких отклонений!
          <br/>
          Тьма не может закрыть Свет!
          <br/>
          Человек – это лучшее Бога явление!
          <br/>
          Подарившее, Миру Завет!
          <br/>
          <br/>
          Завет или Каноны для жизни!
          <br/>
          Равенство – прежде всего!
          <br/>
          Без Равенства невозможно жить,
          <br/>
          Людям не достичь ничего!
          <br/>
          <br/>
          Аминь.
          <br/>
          Отец Абсолют
          <br/>
          29.07.18
          <br/>

          И это – основное Условие!
          <br/>
          <br/>
          Важнейшая деталь – целеполагание!
          <br/>
          Цель – есть Одна и на всех!
          <br/>
          Все люди – Бога слагаемые,
          <br/>
          Движение людей только вверх!
          <br/>
          <br/>
          Главное условие – Согласие!
          <br/>
          Согласие на Вечное Равенство!
          <br/>
          Все решения принимаются Гласно:
          <br/>
          Единодушие или неравенство!
          <br/>
          <br/>
          Все решения по Стратегии,
          <br/>
          Обсуждает Народ на Вече!
          <br/>
          Важно, что люди к этому тяготели,
          <br/>
          Чтобы горели на Душе свечи!
          <br/>
          <br/>
          Выбранные отвечают больше!
          <br/>
          Больше дано – больший спрос!
          <br/>
          Жить люди станут гораздо дольше,
          <br/>
          Сотни лет – уже не вопрос!
          <br/>
          <br/>
          Каждый чувствует своё Участие!
          <br/>
          Придёт понимание Мироздания,
          <br/>
          При Единении придёт и Счастье,
          <br/>
          И единое Судьбы понимание!
          <br/>
          <br/>
          Шестая раса – уже раса Богов!
          <br/>
          Люди поднимутся на Олимп!
          <br/>
          Никто не предвидел таких итогов,
          <br/>
          Небеса споют людям Гимн!
          <br/>
          <br/>
          Но, это произойдёт только «Завтра»!
          <br/>
          Сегодня – переходный период!
          <br/>
          Очень скоро услышите Правду,
          <br/>
          Поговорить о Судьбе – есть повод!
          <br/>
          <br/>
          Переходный срок – очень короткий!
          <br/>
          Повторяю: всего три года!
          <br/>
          Примут люди решение Бога безропотно,
          <br/>
          Изменится в Душах погода!
          <br/>
          <br/>
          Маятник Истории покачнулся!
          <br/>
          Впереди долгожданная Встреча!
          <br/>
          Каждый, кто Бога коснётся,
          <br/>
          Поймёт – Мир Любви вечен!
          <br/>
          <br/>
          Аминь.
          <br/>
          Отец Абсолют
          <br/>
          27.07.18

          <br/>

         <p>
          Нужно видеть года, а лучше столетия!
          <br/>
          Человеку неверия – то не дано,
          <br/>
          Зрение проявляется в годы лихолетья,
          <br/>
          Нужно видеть во всём зерно!
          <br/>
          <br/>
          Зерно по-обычному или фрактал!
          <br/>
          Великое видится на расстоянии!
          <br/>
          Человек для Вечности тоже кристалл,
          <br/>
          Точнее, Создателя достояние!
          <br/>
          <br/>
          Человек человеку – рознь!
          <br/>
          Кто-то видит на уровне носа!
          <br/>
          Чаще живут и видят порознь,
          <br/>
          Единицы знают ответ на вопрос!
          <br/>
          <br/>
          Богочеловек видит Вселенную!
          <br/>
          Он живёт Её горизонтами!
          <br/>
          Человеку простому мешает лень,
          <br/>
          Жизнь его, как под зонтиками!
          <br/>
          <br/>
          Он закрывается ото всего,
          <br/>
          И от Бога в первую очередь!
          <br/>
          Не хочется ему поменять чело,
          <br/>
          Ни на минуту, тем более, навек!
          <br/>
          <br/>
          Кажется, что такой, как все!
          <br/>
          Живет по законам Плотного плана!
          <br/>
          Покорился своей Судьбе,
          <br/>
          Жизнь похожа его на заклание!
          <br/>
          <br/>
          Даже в России – Надежде Бога!
          <br/>
          Обыватели тормозят развитие!
          <br/>
          Никто не хочет знать, что в итоге,
          <br/>
          Бог определится с его обителью!
          <br/>
          <br/>
          Выбор человека – вопрос сложный!
          <br/>
          До Весны ещё далеко,
          <br/>
          Чаще человек видит ответ ложный,
          <br/>
          Поэтому непонятно, куда влекло!
          <br/>
          <br/>
          Кажется, что влекло в никуда!
          <br/>
          Жизнь – в голове обывателя!
          <br/>
          Изменилась человека Судьба,
          <br/>
          Изменилась она основательно!
          <br/>
          <br/>
          Теперь очень важно зрение!
          <br/>
          Зрение на дальние расстояния!
          <br/>
          Оно определяет человека мнение,
          <br/>
          И, конечно, Души состояние!
          <br/>
          <br/>
          Единицы, но – это важно,
          <br/>
          Видят горизонты событий,
          <br/>
          Шаг их сейчас отважен,
          <br/>
          Для изменения людей бытия!
          <br/>
          <br/>
          Сознание всех – помогает каждому!
          <br/>
          Идущему – нужна поддержка!
          <br/>
          Кто выжидает – для него важно,
          <br/>
          Что нет никакой издержки!
          <br/>
          <br/>
          Повторяю, смотрите за горизонты!
          <br/>
          Под ногами не видно дороги!
          <br/>
          Не спасут никакие зонты,
          <br/>
          Если на глазах поволока!
          <br/>
          <br/>
          Нужно видеть и себя в дали!
          <br/>
          Тогда понятна и цель движения!
          <br/>
          Без компаса не бывает пути,
          <br/>
          Невозможно и достижение!
          <br/>
          <br/>
          Посмотрите на года вперёд,
          <br/>
          Подсказка: хотя бы на три!
          <br/>
          Фантазии смелой полёт,
          <br/>
          Обернётся пониманием, как творить!
          <br/>
          <br/>
          Постепенно, фрактал за фракталом!
          <br/>
          Собирается цепь событий!
          <br/>
          Так, без шума или кагала,
          <br/>
          Меняется и путь добытый!
          <br/>
          <br/>
          Сегодня взвешенный шаг!
          <br/>
          Начало Великой дороги!
          <br/>
          Вы зрите вперёд и в такт,
          <br/>
          Поверьте, вы у Порога!
          <br/>
          <br/>
          Аминь.
          <br/>
          Отец Абсолют
          <br/>
          24.07.18

          <br/>

          <br/>
          Люди, как перевёртыши такие,
          <br/>
          Жизнь словно набор подделок!
          <br/>
          <br/>
          Нельзя говорить одно – делать другое!
          <br/>
          Это искривляет Пространство!
          <br/>

          <br/>

          <br/>
          <br/>

          <br/>

          Люди видят, не образ, а пересказ!
          <br/>
          <br/>

          <br/>

          <br/>

          <br/>

          <br/>

          <br/>

          <br/>

          <br/>

          <br/>
          Правда осталась лишь в имени!
          <br/>
          <br/>
          Ложь в действиях, как и в мыслях!
          <br/>
          Полное несоответствие!
          <br/>
          Жизнь во лжи теряет смысл!
          <br/>
          Всё имеет обратное действие!
          <br/>
          <br/>
          Ложь процветает в Мире!
          <br/>
          Лжёт власть, а в ответ – люди!
          <br/>
          Дуальность не обеспечивает перемирия,
          <br/>
          Ложь – это большой беды прелюдия!
          <br/>
          <br/>

          <br/>

          <br/>

          <br/>
          Затоптанное в пыли!
          <br/>
          <br/>
          Нет уже ничего из прошлого!
          <br/>
          Как будто не было и рождения!
          <br/>
          Переврали всё, что возможно!
          <br/>
          От вранья жизнь теперь – отторжение!
          <br/>
          <br/>
          Так продолжаться не может!
          <br/>
          Люди не похожи уже на людей!
          <br/>
          Государствам ничто не поможет,
          <br/>
          Природа сделает Мир светлей!
          <br/>
          <br/>
          Надежда была на Русь!
          <br/>
          На Неё была возложена Миссия!
          <br/>
          Но, сковала национальная грусть,
          <br/>
          Не прошла Она сессию!
          <br/>
          <br/>
          Грусть – национальное качество!
          <br/>
          Печалиться поздно – делать пора!
          <br/>
          Не делается ничего, только плачется,
          <br/>
          И, по-прежнему, в кустах голова!
          <br/>
          <br/>
          Страусиная на Руси позиция!
          <br/>
          Чем наглее власть, тем – страшней!
          <br/>
          Мало ей своей инквизиции,
          <br/>
          Страх на Руси – даже жизни важней!

          <br/>
          Только бунт на Руси жестокий!
          <br/>
          Разнесёт всё и вся!
          <br/>
          Но, соединятся на Руси потоки,
          <br/>
          Это единственная для Неё стезя!

          <br/>
          Сохранилось в Душе оно!
          <br/>
          <br/>
          Россия стоит уже в капитале,
          <br/>
          Ненавистном, для большинства населения!
          <br/>
          Но, 37 год витает,
          <br/>
          Готовятся и поселения!
          <br/>
          <br/>
          У руля страны – мамона!
          <br/>
          Новоиспечённые капиталисты,

          Для России они – статисты!

          Они точно знают, как жить,
          <br/>
          Забыт социализм и Совесть!

          <br/>

          <br/>
          Внешний Мир против России!

          <br/>
          Забыли, что у каждого по жизни – рок!
          <br/>
          <br/>

          <br/>
          У Большинства внутри сущность!

          <br/>
          От Великой страны остался придаток!
          <br/>
          Нужно, чтобы порог прогнулся,
          <br/>
          И в Ад попали предатели!
          <br/>
          <br/>
          Не вернуть прошлый порядок,
          <br/>
          Власть не желает Равенства!
          <br/>
          Она формирует свои отряды,
          <br/>
          Усиливающие неравенство!
          <br/>
          <br/>
          Кризис достиг апогея,
          <br/>
          Россия с человечеством придёт в ноль!
          <br/>
          Политики несут ахинею,
          <br/>
          Им кажется, что мамона – пароль!
          <br/>
          <br/>
          Но, пароль – это не деньги!
          <br/>
          Деньги – часто самообман!
          <br/>
          Жизнь не печальная нега,
          <br/>
          Жизнь для многих – туман!
          <br/>
          <br/>
          В тумане царит дипломатия!
          <br/>
          По принципу: «не пойман – не вор»!
          <br/>
          Нет Знаний – одна хиромантия,
          <br/>
          Люди попрятались за забор!
          <br/>
          <br/>
          Только видно всё с Неба!
          <br/>
          Видно, кто обманывал и воровал!
          <br/>
          Жизнь для них покажется небылью,
          <br/>
          На власть их рогатый короновал!
          <br/>
          <br/>
          Хаос в Душах и на Планете!
          <br/>
          Непонятно для людей грядущее,
          <br/>
          Кто-то держит в кармане валета,
          <br/>
          А кто-то туза на будущее!
          <br/>
          <br/>
          Нет жизни в законе и по понятиям!
          <br/>
          Власть перечеркнула Правила!
          <br/>
          Всё у Народа давно изъято!
          <br/>
          Ей кажется, что всё правильно!
          <br/>
          <br/>
          Мрак сгущается – нет Света!
          <br/>
          В сумерках «все кошки серые»!
          <br/>
          Есть десять Бога Заветов,
          <br/>
          Главное для России принять Веру!
          <br/>
          <br/>
          Аминь.
          <br/>
          Отец Абсолют
          <br/>
          22.07.18
          <br/>
          <br/>
          <noindex>
           <a href="http://blagayavest.info/poems/22.07.18.html" rel="nofollow">
            http://blagayavest.info/poems/22.07.18.html
           </a>
          </noindex>
         </p>

         <a href="http://esoterix.ru/blog/5854.html">
          21.07.18 Катрен “Трудное время”
         </a>
        </h2>
        <ul class="action">
         <li class="user">

        <div class="content">
         <p>
          Мир разорвал Пространство!
          <br/>
          Теперь нет нигде Мира!
          <br/>
          Сбились в кучу все государства!
          <br/>
          Мамона управляет Миром!
          <br/>
          <br/>
          Все ждали Великого перемирия!
          <br/>
          Оно наступить не сможет!
          <br/>
          Государства превратились в частицы Мира!
          <br/>
          Соединить их теперь только Бог сможет!
          <br/>
          <br/>
          Но, в людях нет Веры Создателю,
          <br/>
          Вера Богу заменена религиями,
          <br/>
          Религии превратили Богов в обыватели,
          <br/>
          На верящих наложили вериги!
          <br/>
          <br/>
          Последняя встреча – говорит поздно!
          <br/>
          Не соединить несоединимое!
          <br/>
          В бизнесе, конечно, возможно,
          <br/>
          «Прилавки» все обозримые!
          <br/>
          <br/>
          Невозможно соединить в Духе!
          <br/>
          Нет Веры и Духа в людях,
          <br/>
          Чувствуется в головах «разруха»,
          <br/>
          Великого Конца – прелюдия!
          <br/>
          <br/>
          Исторический подходит Момент!
          <br/>
          Две Державы стали врагами,
          <br/>
          Малейший боевой инцидент,
          <br/>
          Обернётся «борьбой на татами»!
          <br/>
          <br/>
          Мировой рынок – это не План!
          <br/>
          Котировки прыгают поминутно!
          <br/>
          Жирует теневой клан,
          <br/>
          У человечества в голове мутно!
          <br/>
          <br/>
          Нет стабильности в государствах!
          <br/>
          Для мамоны важнее – рынок!
          <br/>
          Границы перечеркнуты в Пространстве,
          <br/>
          Идеологический в Мире упадок!
          <br/>
          <br/>
          Америка – взрывоопасна, тоже – Россия!
          <br/>
          Европа подожжена мусульманами,
          <br/>
          Россия готовится к своей Миссии,
          <br/>
          Мир на Планете – Её планы!
          <br/>
          <br/>
          На Небе свёрстаны Её планы!
          <br/>
          Преображение начнётся с частиц,
          <br/>
          В России нет мировых кланов,
          <br/>
          И Народ ещё не падал ниц!
          <br/>
          <br/>
          На фоне развала Европы,
          <br/>
          Россия сохранила потенциал!
          <br/>
          Мамоны пытается одеть оковы,
          <br/>
          Но, в Душах остался интернационал!
          <br/>
          <br/>
          Необходимо остановить разграбление!
          <br/>
          Предатели не вне, а внутри!
          <br/>
          В Конституции страны есть определение,
          <br/>
          Ресурсы Народу не будут сохранены!
          <br/>
          <br/>
          Ресурсы России уже проданы!
          <br/>
          Народу не известны хозяева,
          <br/>
          Они – капитала подданные,
          <br/>
          Скоро всё будет явлено!
          <br/>
          <br/>
          Народ вдруг увидит предателей!
          <br/>
          Сколько лет в подчинении были!
          <br/>
          Оказалось, что свои грабители,
          <br/>
          Членами КПСС слыли!
          <br/>
          <br/>
          Идеология партии страну ограбила!
          <br/>
          Не говорю о Боге, только о Совести!
          <br/>
          По карманам всё разложила,
          <br/>
          Но, время уже другой Вести!
          <br/>
          <br/>
          Разобранная на части – соединяется!
          <br/>
          Идеология Равенства от Бога!
          <br/>
          Волнения в Народе легко объясняются!
          <br/>
          Восстановится эволюция в итоге!
          <br/>
          <br/>
          Пример – Европа или Америка,
          <br/>
          Разорванные экономики тлеют,
          <br/>
          Простые люди в истерике,
          <br/>
          Богатые только жиреют!
          <br/>
          <br/>
          Угрожающий ждёт результат!
          <br/>
          Предначертан конец человечеству,
          <br/>
          Человечеству Тьмой поставлен мат,
          <br/>
          «Завтра» только в России отечество!
          <br/>
          <br/>
          Каждый день – уже как последний!
          <br/>
          Рыночные экономики – беспорядок!
          <br/>
          Научные говорят исследования,
          <br/>
          Нет страны, если в Душе упадок!
          <br/>
          <br/>
          Трудное время сделали люди!
          <br/>
          Переход состоится в любом случае!
          <br/>
          Сегодня – это только прелюдия,
          <br/>
          Пусть Совесть людей помучает!
          <br/>
          <br/>
          Аминь.
          <br/>
          Отец Абсолют
          <br/>
          21.07.18
          <br/>
          <br/>

            http://blagayavest.info/poems/21.07.18.html
           </a>
          </noindex>
         </p>
