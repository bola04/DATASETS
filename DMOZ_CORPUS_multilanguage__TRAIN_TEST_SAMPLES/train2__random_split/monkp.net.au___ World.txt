                     <h3 class="post-title entry-title" itemprop="name">
                      <a href="https://www.monkp.net.au/2018/03/choose-one-thought-over-another.html">
                       轉念 choose one thought over another
                      </a>
                     </h3>
                     <div class="post-header">
                      <div class="post-header-line-1">
                      </div>
                     </div>
                     <div class="post-body entry-content" id="post-body-5422400972043410966" itemprop="articleBody">
                      同一天發兩次牢騷好像是第一次?
                      <br/>
                      <br/>
                      昨天和 x 吃完飯後
                      <br/>
                      心情都一直很沉重
                      <br/>
                      <br/>
                      直到早上的牢騷後
                      <br/>
                      接下來去上課
                      <br/>
                      似乎好像有比較釋懷了
                      <br/>
                      <br/>
                      真的
                      <br/>
                      <br/>
                      有時候不用想的太遠
                      <br/>
                      該吃飯的時候就吃飯
                      <br/>
                      該睡覺的時候就睡覺
                      <br/>
                      反正和我暫時沾不上邊
                      <br/>
                      為什麼要掛在心中呢?
                      <br/>
                      <br/>
                      剛好這個時候
                      <br/>
                      認識不久的好朋友傳 LINE
                      <br/>
                      和她哈拉了一陣子後
                      <br/>
                      也稍微爆料了自己最近的煩惱
                      <br/>
                      非常感謝她的建議
                      <br/>
                      <br/>
                      與其自己在原地畫圈
                      <br/>
                      倒不如就坦然的接受
                      <br/>
                      <br/>
                      反正我也沒損失
                      <br/>
                      沒有人會有損失
                      <br/>
                      <br/>
                      直話直說
                      <br/>
                      即使不是預期的
                      <br/>
                      那也只能那樣囉
                      <br/>
                      <br/>
                      ps. 要能在這個社會生存好像要懂得厚臉皮是吧? XD
