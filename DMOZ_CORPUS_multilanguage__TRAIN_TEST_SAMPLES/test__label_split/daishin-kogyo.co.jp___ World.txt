      <h2>
       <img alt="メッセージ" height="90" src="web_images/title01.gif" width="550"/>
      </h2>
      <p>
       弊社は1949年の創業以来、建築資材・自動車・電気製品部品等の
       <br/>
       様々な分野へ向けた樹脂押出成形部品の製造を行っております。
       <br/>
       1970年にはイタリア、カラープラスト社と硬質塩化ビニール樹脂の押出の技術提携をし、
       <br/>
       日本初の樹脂製窓の製造を開始しました。
       <br/>
       住環境の厳しい極寒地域の北海道を対象として開発した
       <br/>
       弊社独自の窓システム「内窓プラスト」は、
       <br/>
       多くの方々にご支持を頂き、快適な住環境の創造に務めてまいりました。
       <br/>
       近年ではビル・施設の断熱改修用の「内窓プラスト 〜インスリード〜」を開発し、
       <br/>
       住宅のみならず非住宅の建築物の断熱改修・省エネルギー化に取り組んでおります。
       <br/>
       企業として地球環境の保全が社会責任として重要な役割を担う昨今、
       <br/>
       弊社は40年以上建築物の断熱化に取り組んできた
       <br/>
       「断熱のパイオニア」として日本の建築物の断熱化を推進し、
       <br/>
       低炭素社会の実現へ向け、顧客と社会に貢献できる様努めてまいります。
       <br/>
       また、樹脂押出成形部門においても、急速に変化をする時代の中で
       <br/>
       求められる製品を確かな品質で顧客にお届けすると共に、
       <br/>
       新たな製品作りへ向けた開発活動を行い、
       <br/>
       新しい社会のニーズに答えられる企業を目指し、邁進してまいります。
       <img alt="サイン" height="157" src="web_images/sign.gif" width="230"/>
      </p>
