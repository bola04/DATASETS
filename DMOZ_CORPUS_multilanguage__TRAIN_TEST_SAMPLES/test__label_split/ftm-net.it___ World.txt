                 <p style="text-align: justify;">
                  La nostra azienda opera nel settore dello stampaggio e della saldatura dei metalli sin dal 1990 quando, sulla base dell'esperienza ventennale del fondatore nella realizzazione di stampi per tranciatura e deformazione della lamiera, fu insediata nell'attuale stabilimento.
                 </p>
                 <p style="text-align: justify;">
                  Il know-how accumulato nella costruzione di stampi, oggi ci consente di conoscere a fondo tutte le problematiche legate alla lavorazione della lamiera e, quindi, di prevederle ed affrontarle nel modo migliore.

                 </p>
                 <p style="text-align: justify;">
                  Attualmente la produzione è costituita da particolari metallici di vario genere, ottenuti attraverso diverse tipologie di presse ed assemblati per mezzo di saldatura MIG-MAG con l'ausilio di robots dell'ultima generazione. Contemporaneamente all'interno dell'azienda è presente un reparto di attrezzeria dedicato alla realizzazione degli stampi e delle attrezzature necessarie alla produzione e che ci permette di garantire la corretta manutenzione delle stesse.

 	function setCookie(c_name,value,exdays)
 	{
 		var exdate=new Date();
 		exdate.setDate(exdate.getDate() + exdays);
 		var c_value=escape(value) + ((exdays==null) ? "" : "; expires="+exdate.toUTCString()) + "; path=/";
 		document.cookie=c_name + "=" + c_value;
 	}
 	
 	function readCookie(name) {
 		var nameEQ = name + "=";
 		var ca = document.cookie.split(';');
 		for(var i=0;i < ca.length;i++) {
 			var c = ca[i];
 			while (c.charAt(0)==' ') c = c.substring(1,c.length);
 			if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
 			}
 		return null;
 	}
