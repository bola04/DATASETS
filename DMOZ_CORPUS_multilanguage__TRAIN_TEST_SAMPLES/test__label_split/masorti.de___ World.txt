        <b>
         Gründung der Masorti Grundschule
        </b>
        <br/>
        <br/>
        <img border="1" height="147" id="Image" name="Image" src="images/Infoabend_Masorti_Grundschule.jpg" width="147"/>
        <p style="position:relative; left:4px;">
         Masorti plant die Eröffnung einer Grundschule zum Schuljahr 2018/19.
         <br/>
         <br/>
         Die Masorti Grundschule wird sich auszeichnen durch:
         <br/>
         *ihren reformpädagogischen Ansatz mit freien Lernformen
         <br/>
         *doppelte Bilingualität deutsch-hebräisch und deutsch-
         <br/>
         englisch (in unmittelbarer Kontinuität der Masorti Kitas)
         <br/>
         *innerjüdischem Pluralismus, der als Bereicherung
         <br/>
         erfahrbar wird
         <br/>
         *Offenheit für vielfältige Familienformen und
        </p>
        *Gendergerechtigkeit gerade auch in der jüdisch religiösen Praxis.
        <br/>
        <br/>
        Bei Interesse schicken Sie bitte eine email an
        <a class="intext" href="mailto:schule@masorti.de" target="_new">
         schule(at)masorti.de
        </a>
        .

        <b>
         "Machon le Giur"
        </b>
        <br/>
        <br/>
        Willkommen zum
        <b>
         3. Seminar von Machon le Giur
        </b>
        in Berlin vom 15.-18. März 2018!
        <br/>
        <br/>
        <img border="1" height="152" id="Image" name="Image" src="images/Machon_Seminar_Marz.jpg" width="200"/>
        <p style="position:relative; left:4px;">
         In diesem neuen, von Masorti e.V. initiierten und vom Zentralrat der Juden in Deutschland geförderten Projekt
 werden bis zu 20 meist russischsprachige Personen, die einen jüdischen Vater und eine nicht-jüdischen Mutter haben, zu einem Giur 
 durch die Allgemeine Rabbinerkonferenz Deutschland begleitet.
         <br/>
         <br/>
         Das Lernen findet in Gruppen in den Heimatgemeinden und bei fünf Schabbatonim in
        </p>
        Berlin in Kooperation mit der 
 Synagoge Oranienburger Strasse statt.
        <br/>
        Mehr Informationen dazu finden Sie im
        <a class="intext" href="pdf/189_Flyer_Giur_DE.pdf" target="_new">
         deutschen
        </a>
        oder im
        <a class="intext" href="pdf/190_Flyer_Giur_RU.pdf" target="_new">
         russischen
        </a>
        Projektflyer.
 Bei Interesse schicken Sie uns bitte
        <a class="intext" href="mailto:machonlegiur@gmail.com" target="_new">
         eine Email
        </a>
