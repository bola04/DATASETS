              <h3 class="post-title entry-title">
               <a href="http://etickykodexbloggera.blogspot.com/2008/08/etick-kdex-bloggera.html">
                Etický kódex bloggera
               </a>
              </h3>
              <div class="post-header-line-1">
              </div>
              <div class="post-body entry-content">
               <a href="http://2.bp.blogspot.com/_Sc94S4MCLwA/SZxoqFiHkHI/AAAAAAAABE0/Z6ihmx03raA/s1600-h/dopis2.jpg" onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}">
                <img alt="Etický kódex bloggera" border="0" id="BLOGGER_PHOTO_ID_5304229533384675442" src="http://2.bp.blogspot.com/_Sc94S4MCLwA/SZxoqFiHkHI/AAAAAAAABE0/Z6ihmx03raA/s400/dopis2.jpg" style="margin: 0px auto 10px; display: block; text-align: center; cursor: pointer; width: 150px; height: 200px;"/>
               </a>
               <br/>
               <div style="text-align: left;">
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 1.Autorský zákon
                </span>
                <br/>
                Kopírovanie akéhokoľvek materiálu z iných stránok (okrem Youtube videí)  je porušením autorského zákona, ak to nie je autorom povolené. Ak uvádzate obsah z inej stránky, uveďte zdroj. Celé znenie autorského zákona si môžete stiahnuť z
                <a href="http://sk.wikipedia.org/wiki/Redaktor:Helix84/Autorsk%C3%BD_z%C3%A1kon" target="_blank">
                 Wikipédie
                </a>
                .
                <br/>
                <br/>
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 2. Slobodná vôľa
                </span>
                <br/>
                Akékoľvek nútenie návštevníka písať komentáre alebo vykonávať iné aktivity, z ktorých plynie prospech iba vám alebo vášmu blogu,  je neetické. To isté platí aj pri výmene odkazov (spriatelenie blogov) medzi stránkami. Podmienky výmeny odkazov sú len na základe vašej dohody,  mali by však byť rovnaké pre obe strany a nemali by sa vymykať pravidlám EKB.
                <br/>
                <br/>
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 3. Obsah
                </span>
                <br/>
                Obsah nevhodný do 18 rokov cenzurujte alebo naň upozornite návštevníka vopred. Nepropagujte násilie, extrémizmus a rasizmus. To platí aj pre reklamy.
                <br/>
                <br/>
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 4. Ankety, súťaže
                </span>
                <br/>
                Vždy zabezpečte, aby súťaže boli férové, pravé a aby sa  nedali zmanipulovať.
                <span dir="ltr" id=":128">
                 V prípade zmanipulovania súťaže by ste mali výsledky označiť za nedôveryhodné, prípadne súťaž zrušiť.
                </span>
                <br/>
                Blogger by po spustení ankety nemal akokoľvek manipulovať s výsledkami a priebehom ankety.
                <br/>
                <br/>
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 5. Komunikácia
                </span>
                <br/>
                Nemažte príspevky návštevníkov v chatoch, komentároch, návštevných knihách a pod. Výnimku tvoria vulgarizmy, spam a nemorálne príspevky.
                <br/>
                <br/>
                <span style="font-weight: bold; color: rgb(255, 102, 0);">
                 6. Informácie
                </span>
                <br/>
                Blogger by mal publikovať len overené informácie. Ak nie sú overené, mal by na túto skutočnosť upozorniť.
                <br/>
                <br/>
                Blogger sa podpísaním tohto kódexu zaväzuje spĺňať všetky jeho body. Čo nie je napísané v EKB, je povolené. Všeobecne by sa však mal riadiť morálnymi a etickými zásadami.
                <br/>
               </div>
