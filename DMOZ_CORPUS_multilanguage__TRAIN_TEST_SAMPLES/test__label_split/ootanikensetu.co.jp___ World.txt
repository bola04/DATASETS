     <div class="contentsp">
      昭和の横浜の地で産声を上げた当社は、平成の今日５０有余年の歴史を有しています。
      <br/>
      ご承知の通り、この５０年は日本がかつて経験した事がない激動の時代でもありました。
      <br/>
      高度成長時代、オイルショック、バブル時代、そしてその崩壊を生で体験する中でも
      <br/>
      私どもは自分を見失わないことを第一主義でやって参りました。
      <br/>
      それは自分たちに何が出来るか、また自分たちの使命は何なのかを考え
      <br/>
      職務に精進する５０年でもありました。
      <br/>
      <br/>
      バプル崩壊後、多くのピジョン無き「利益優先型企業」が崩壊していくなか、
      <br/>
      小さいけれど熱狂的なご支援を、数多く賜った事は、その証であると自負しております。
      <br/>
      <br/>
      そして我社は、次の５０年も応援して下さるみなさまの立場になり、共に汗を流し、
      <br/>
      夢創りのお手伝いをしていきたいと考えます。
      <br/>
      <br/>
      <span class="fp18">
       私たちは「未来」を創る会社です。
