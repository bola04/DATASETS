       <h1 class="midasi1">
        いちご狩りの３つの方式
       </h1>
       <img align="right" height="145" src="stb/p_b/itigo3.jpg" width="208"/>
       <span class="f-red">
        ●
       </span>
       <strong>
        食べ放題
       </strong>
       ：いちご狩りといったら、一般的には食べ放題のことをいいます。制限時間は30分のところが多いようですが、15分くらいでお腹いっぱいになる人が多いようです。なかには時間制限がないイチゴ園もあります。多くの園では、コンデンスミルクが付いてきますが、イチゴの味にこだわっている園では、コンデンスミルクが付かないことがあります。
       <br/>
       <span class="f-red">
        ●
       </span>
       <strong>
        量り売り
       </strong>
       ：「
       <strong>
        イチゴ摘み
       </strong>
       」と書いてある場合は、食べ放題ではなくて、量り売りの場合があります。自分で摘み取ったいちごをパックにつめたり、重さによって、料金が決まる方式です。
       <br/>
       <span class="f-red">
        ●
       </span>
       <strong>
        株売り
       </strong>
       ：区画ごとイチゴを100苗とか50苗とか買って、期間内はいつでも訪れて食べられる方式です。
      </div>
      <h2 class="midasi1">
       いちごの栽培によるイチゴ狩りの違い
      </h2>
      <table border="0">
       <tbody>
        <tr>
         <td>
          <img alt="高設栽培のいちご狩り" height="101" src="stb/p_b/kamakura.jpg" width="203"/>
         </td>
         <td>
          <div class="fs2">
           <span class="f-red">
            ●
           </span>
           <b>
            高設栽培
           </b>
           （こうせつさいばい）
           <br/>
           地面より高い位置でイチゴを栽培することです。いちご狩りが、最も楽にできます。イチゴは空中に宙吊りになっているので、衛生的です。イチゴ狩りができる農園は、この高設栽培が多くなっています。
          </div>
         </td>
        </tr>
        <tr>
         <td align="center" valign="top">
          <img alt="土耕栽培のいちご狩り" height="89" src="stb/p_b/dokou.jpg" width="130"/>
         </td>
         <td>
          <div class="fs2">
           <span class="f-red">
            ●
           </span>
           <b>
            土耕栽培
           </b>
           （どこうさいばい）
           <br/>
           地面に畝（うね：盛り土）を作り、黒いビニールシートで土を覆って、イチゴの苗を植える方法です。黒いビニールにイチゴが接するので、衛生的に気になる人もいます。
          </div>
         </td>
        </tr>
        <tr>
         <td align="center" valign="top">
          <img alt="石垣いちご狩り" height="89" src="stb/p_b/ishigaki_itigo.jpg" width="131"/>
         </td>
         <td>
          <div class="fs2">
           <span class="f-red">
            ●
           </span>
           <b>
            石垣栽培
           </b>
           （いしがきさいばい）
           <br/>
           南向きの斜面にコンクリートブロックを積み（昔は左の写真のように石でした）、その石の間にイチゴの苗を植える方法です。珍しいので観光にむきます。静岡県久能山周辺が有名です。
          </div>
         </td>
        </tr>
       </tbody>
      </table>
      <div class="fs2">
       いちご狩り観光農園は、ほとんどがビニールハウスなどの施設栽培ですが、雪が多い地域では
       <b>
        露地栽培
       </b>
       （野外）の農園も多数あります。
      </div>
