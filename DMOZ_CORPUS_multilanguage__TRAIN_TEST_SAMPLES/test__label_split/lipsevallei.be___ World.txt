              <h2 itemprop="name">
               <a href="/joomla/index.php?option=com_content&amp;view=article&amp;id=430:kelderactiviteit-wijn-en-bier-boerenmarkt-veltem-en-degustatie-van-slovaakse-wijnen&amp;catid=80&amp;Itemid=63" itemprop="url">
                Kelderactiviteit wijn en bier, boerenmarkt Veltem en degustatie van Slovaakse wijnen
               </a>
              </h2>
             </div>
             <p style="text-align: justify;">
              <strong>
               O
              </strong>
              p de kelderactiviteit van vrijdag
              <strong>
               4 mei
              </strong>
              hebben de aanwezige
              <img alt="1" height="271" src="/joomla/images/stories/Articles/KelderactiviteitwijnenbierboerenmarktVeltemendegustatievanSlovaaksewijnen/1.PNG" style="float: right;" width="250"/>
              leden en bestuursleden 65 liter Johanniter gebotteld, werd de bierleder op orde  gezet en er het laatste brouwsel opgeborgen. Er waren spijtig genoeg weinig helpende handen aanwezig.
              <br/>
              <br/>
             </p>
             <table style="height: 310px; width: 822px;">
              <tbody>
               <tr>
                <td>
                 <img alt="2" height="285" src="/joomla/images/stories/Articles/KelderactiviteitwijnenbierboerenmarktVeltemendegustatievanSlovaaksewijnen/2.PNG" style="vertical-align: bottom;" width="250"/>
                </td>
                <td>
                 <img alt="3" height="171" src="/joomla/images/stories/Articles/KelderactiviteitwijnenbierboerenmarktVeltemendegustatievanSlovaaksewijnen/3.PNG" width="250"/>
                </td>
                <td>
                 <img alt="4" height="138" src="/joomla/images/stories/Articles/KelderactiviteitwijnenbierboerenmarktVeltemendegustatievanSlovaaksewijnen/4.PNG" width="250"/>
                </td>
               </tr>
              </tbody>
             </table>
             <p style="text-align: justify;">
              <br/>
              Hierbij een klein beeldverslag:
              <br/>
              <strong style="background-color: initial;">
               <br/>
               O
              </strong>
              <span style="background-color: initial;">
               nze gilde was op zaterdag
              </span>
              <strong style="background-color: initial;">
               12 mei
              </strong>
              <span style="background-color: initial;">
               aanwezig met een info- en degustatiestand op de boerenmarkt in Veltem. De zonnige zaterdag zorgde voor behoorlijk wat volk. Onze Gildestand genoot mee en ontving heel wat belangstellende dorpsgenoten voor een gezellige babbel en andere passanten, uit Oekraïne bijvoorbeeld, die wel Belgisch bier kenden en ontdekten dat hier ook goede wijnen worden gemaakt.
              </span>
             </p>
             <p style="text-align: justify;">
              <img alt="5" height="262" src="/joomla/images/stories/Articles/KelderactiviteitwijnenbierboerenmarktVeltemendegustatievanSlovaaksewijnen/5.PNG" width="500"/>
             </p>
             <p style="text-align: justify;">
              <strong>
               O
              </strong>
              p de ledenvergadering van dinsdag
              <strong>
               29 mei
              </strong>
              gaf Patrik Van Den Bossche van SLOVAKWINES ons met zijn presentatie een boeiende rondleiding door dit prachtige land en liet ons een aantal typische wijnen proeven uit dit voor ons onbekende wijnland. We degusteerden twee mousserende, drie droge witte, een rosé, twee rode en om af te sluiten een halfdroge witte. Dikwijls gemaakt met unieke plaatselijke druivensoorten, moet de kwaliteit van deze wijnen zeker niet onderdoen voor veel bekendere internationale kwaliteitswijnen: goed gemaakt met een eigen stijl en een prijs/kwaliteitsverhouding die meer dan correct is, sommige staan zelfs in de wereld top-100 lijst! Het bestuur bood ons tijdens de degustatie, als kers op de taart, een rijkelijk kaas- en patébuffet aan. En of het een geslaagde avond was? Ge moogt gerust zijn.
             </p>

 	function setCookie(c_name,value,exdays)
 	{
 		var exdate=new Date();
 		exdate.setDate(exdate.getDate() + exdays);
 		var c_value=escape(value) + ((exdays==null) ? "" : "; expires="+exdate.toUTCString()) + "; path=/";
 		document.cookie=c_name + "=" + c_value;
 	}
 	
 	function readCookie(name) {
 		var nameEQ = name + "=";
 		var ca = document.cookie.split(';');
 		for(var i=0;i < ca.length;i++) {
 			var c = ca[i];
 			while (c.charAt(0)==' ') c = c.substring(1,c.length);
 			if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
 			}
 		return null;
 	}
