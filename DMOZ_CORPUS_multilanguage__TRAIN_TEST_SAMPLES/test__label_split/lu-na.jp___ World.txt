             <span>

             <p class="mb10">
              こんにちは。
              <br/>
              ルナレディースクリニック・不妊・更年期センター院長の根来良材です。
              <br/>
              <br/>
              近年、
              <strong>
               「不妊症」
              </strong>
              という言葉をニュースでもよく耳にするようになっておりますが、これは
              <strong>
               不妊治療が高度になってきた証拠
              </strong>
              でもあります。
              <br/>
              かつてはとうてい妊娠できなかったようなご夫婦も、現在では十分に妊娠を期待できるほど医療技術は向上してきました。
              <br/>
              <br/>
              当院は、街中のクリニックではありますが、
              <strong>
               タイミング法から人工授精、高度不妊治療（体外受精・顕微授精）まで対応
              </strong>
              しています。
              <br/>
              <span style="font-size:140%;">
               <strong>
                <span style="color: rgb(254, 156, 0);">
                 10年間で約1,300件の妊娠のサポートをしてまいりました。
                </span>
               </strong>
              </span>
              <br/>
              <br/>
              一人ひとりの患者さまには、
              <strong>
               丁寧で繊細な治療
              </strong>
              を心掛けています。
              <br/>
              不妊ということから、身体だけでなく心に不安を抱えた方がほとんどです。
              <br/>
              そういう現場で働いているのだということを忘れずに、
              <strong>
               ひとりでも多くの命を授かる手助け
              </strong>
              を、これからも続けていきたいと考えております。
              <br/>
              <br/>
              不妊症だけでなく、
              <strong>
               更年期障害、婦人病などにも対応
              </strong>
              いたしております。
              <br/>
              できるだけ気軽に来ていただければ、と思います。スタッフ一同、お待ちしております。
              <br/>
              <br/>
              院長　　根来 良材
