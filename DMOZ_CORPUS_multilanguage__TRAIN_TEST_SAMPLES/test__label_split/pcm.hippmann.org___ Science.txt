    <h2>
     Polygonal Contact Model
    </h2>
   </center>
   <h4>
    Introduction
   </h4>
   <p>
    The Polygonal Contact Model (PCM) is a contact algorithm for multibody dynamics. It uses polygonal body surfaces and implements the elastic foundation contact model.
   </p>

   <h4>
    References
   </h4>
   <p>
    PCM is based on
    <a href="./doc/hippmann_dissertation.pdf">
     Gerhard Hippmann's Dissertation
    </a>
    . In July 2003 a
    <a href="./doc/eccomas03_hippmann.pdf">
     paper about PCM
    </a>
    was
    <a href="./doc/eccomas03_presentation_hippmann.pdf">
     presented
    </a>
    at the
    <a href="http://www.dem.ist.utl.pt/Multibody2003/" target="_blank">
     ECCOMAS Multibody Dynamics conference
    </a>
    .
    <a href="mailto:Gongjun11@hotmail.com">
     Gongjun Hong
    </a>
    wrote his
    <a href="./doc/hong_diploma.pdf">
     diploma thesis
    </a>
    about PCM's contact element generation based on
    <a href="http://www.cc.gatech.edu/gvu/geometry/implicit/" target="_blank">
     variational implicit surfaces
    </a>
    at the
    <a href="http://www.amm.mw.tu-muenchen.de/index_e.html" target="_blank">
     Institute for Applied Mechanics of the TUM
    </a>
    . Refer to the books
    <a href="http://books.cambridge.org/0521347963.htm" target="_blank">
     Contact mechanics
    </a>
    by K.L. Johnson and
    <!--http://www.amazon.com/exec/obidos/tg/detail/-/0792307127/qid=1051695104/sr=8-1/ref=sr_8_1/103-3528956-2432652?v=glance&s=books&n=507846-->
    <a href="http://www.wkap.nl/prod/b/0-7923-0712-7" target="_blank">
     Three-dimensional elastic bodies in rolling contact
    </a>
    by J.J. Kalker about the elastic foundation model.
   </p>
   <h4>
    Supporters
   </h4>
   <ul>
    <li>
     PCM uses a collision detection algorithm based on hierarchical bounding volumes described by
     <a href="http://web.informatik.uni-bonn.de/II/ag-klein/people/zach/" target="_blank">
      Gabriel Zachmann
     </a>
     's online publications
    </li>
    <li>
     <a href="http://www.3dcafe.com/asp/meshes.asp" target="_blank">
      3D Cafe
     </a>
     and
     <a href="http://www.exchange3d.com/" target="_blank">
      Meta3D
     </a>
     provided amazing polygonal surfaces for free
    </li>
    <li>
     The 3D-modelling shareware
     <a href="http://www.ac3d.org/" target="_blank">
      AC3D
     </a>
     was very helpful to manipulate polygonal surfaces
    </li>
    <li>
     The excelent free editor
     <a href="http://www.nedit.org/" target="_blank">
      NEdit
     </a>
     was the most important tool to develop and document PCM
    </li>
    <li>
     <a href="http://www.mathematik.uni-halle.de/institute/numerik/mitarbeiter/arnold.html" target="_blank">
      Martin Arnold
     </a>
     and collegues of the
     <a href="http://www.ae.op.dlr.de/" target="_blank">
      vehicle system dynamics group at the DLR
     </a>
     contributed to PCM by many discussions
    </li>
   </ul>

   <h4>
    Source Code
   </h4>
   <p>
    PCM was implemented in C by
    <a href="mailto:pcm@hippmann.org">
     Gerhard Hippmann
    </a>
    and
    <a href="mailto:Gongjun11@hotmail.com">
     Gongjun Hong
    </a>
    . It compiles on Linux, HP-UX, IRIX and Windoze and was tested as a force element user routine within
    <a href="http://www.simpack.com/" target="_blank">
     SIMPACK
    </a>
    8.514 - 9.2. However it should be easy to make it run on other platforms and MBS-codes.
   </p>
