   <div id="concept">
    <div id="copy">
     <h2>
      <img alt="cozy. concept" height="13" src="top/title_concept.gif" width="91"/>
     </h2>
     <p>
      世の中にはたくさんのものが溢れ、それぞれの価値観にしたがって
      <br/>
      わたしたちは取捨選択をしながら暮らしています。
      <br/>
      例えば、毎日のように着るT シャツを選ぶとき、
      <br/>
      『普段着だから、毎日着ていればすぐにくたびれてしまうだろう。
      <br/>
      だから気兼ねなく着られるモノで構わない。』 そう考えるひともいれば、
      <br/>
      『毎日着るものだから、着心地がよくて見た目だってこだわりたい。
      <br/>
      だから気分よく着られるモノを選びたい。』 と考えるひともいるでしょう。
     </p>
     <p>
      cozy. は、どちらかといえば
      <br/>
      『気分よく』の価値観を持つひとに向けて発信するブランドです。
      <br/>
      もちろん気分よくが長くつづくよう、上質で丈夫なものづくりが信条です。
      <br/>
      それぞれのひとが、使いたいように使える、
      <br/>
      そしてそのひとが必要なものを、必要なかたちで応える、
      <br/>
      シンプル（簡素）で、 スタンダード（普通）で、 フレキシブル（柔軟）な
      <br/>
      家具や暮らしを提案していきます。
     </p>
