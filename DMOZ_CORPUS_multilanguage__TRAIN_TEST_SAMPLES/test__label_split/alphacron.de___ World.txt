      <h2>
       Home
      </h2>
      <div class="entry">
       <p>
        AlphaCron Datensysteme ist Ihr starker Partner in den Bereichen:
       </p>
       <ul>
        <li>
         Webhosting, Webdesign, Domains, Zertifikate
        </li>
        <li>
         Serverhousing im eigenen Rechenzentrum
        </li>
        <li>
         DSL Zugänge mit festen IP-Adressen
        </li>
        <li>
         Standleitungen (sowohl zur Internetanbindung als auch direkt)
        </li>
        <li>
         Standortvernetzung (VPN)
        </li>
        <li>
         Internet-Telefonie (Voice over IP / VoIP)
        </li>
        <li>
         Planung und Installation von Firmennetzwerken
        </li>
        <li>
         Analyse und Überwachung von komplexen Netzwerkstrukturen
        </li>
        <li>
         Linux basierte Systeme
        </li>
        <li>
         Windows basierte Systeme (im Active-Directory Umfeld, Exchange, SQL)
        </li>
        <li>
         gesamte Systembetreuung im Automotive-Umfeld (Volvo, VW, Audi, Skoda, Seat, Nissan, Renault, Dacia)
        </li>
        <li>
         Apple/Mac basierte Systeme
        </li>
        <li>
         Einrichtung und Betrieb von WLAN Infrastrukturen
        </li>
        <li>
         netzwerkbasierte Videoüberwachung
        </li>
        <li>
         LWL / Glasfaser Spleißarbeiten jeglicher Art (Multimode, Singlemode)
        </li>
        <li>
         Installation und Wartung von Satellitentechnik
        </li>
        <li>
         digitale Raumbeschilderungssysteme
        </li>
        <li>
         Funktechnik
        </li>
        <li>
         EPROM-Programmierung (für Netzwerkkarten, etc.)
        </li>
        <li>
         Großformatdruck (bis zu einer Größe von A0)
        </li>
       </ul>
