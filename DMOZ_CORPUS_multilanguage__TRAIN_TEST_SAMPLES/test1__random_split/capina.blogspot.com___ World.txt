           <h3 class="post-title">
            <a href="http://capina.blogspot.com/2007/07/blog-post.html">
             פודקאסטים מומלצים בנושא אנימציה
            </a>
           </h3>
           <div class="post-header-line-1">
           </div>
           <div class="post-body">
            <p>
            </p>
            <p>
             עכשיו
             <a href="http://www.ynet.co.il/articles/0,7340,L-3272931,00.html" target="_blank">
              כשהבלוגים חילחלו לטלויזיה
             </a>
             ולכל נערה בת 15 יש בלוג, ובא לנו להקיא רק מלשמוע את המילה הזאת, זה הזמן לדבר על פודקאסטים.
             <br/>
             פודקאסט (podcast) למי שאינו יודע הוא כמו תוכנית רדיו מוקלטת שניתן להוריד מהרשת בצורת קובץ קול או להקשיב לה בסטרימינג, תוכנם של הפודקאסטים נע בין ראיונות מרתקים עם אנימטורים, הדרכות, חדשות, סקירות וביקורות קולנוע רציניות לנונסנס מוחלט. על אותו משקל ישנו גם את הוודקאסט (vodcast) שמקביל לתוכנית טלויזיה מוקלטת, וניתן להורדה כקובץ וידאו.
             <br/>
             היתרון של הפודקאסט על-פני הוודקאסט והבלוג, הוא שניתן להוריד את קובץ הקול ל-mp3 player החביב עליכם (כפי שמרמז שמו שניגזר מהאיי-פוד) ולהקשיב לתוכניות תוך כדי שטיפת כלים או טיול אופניים למשל.
             <br/>
             <br/>
             <em>
              <strong>
               איך מוצאים פודקאסטים?
              </strong>
             </em>
             <br/>
             <br/>
             ישנם המון אתרי אינדקס ברשת לחיפוש פודקאסטים כמו
             <a href="http://www.podcastalley.com/index.php" target="_blank">
              PodcastAlley
             </a>
             או
             <a href="http://www.zencast.com/" target="_blank">
              ZENcast
             </a>
             חלקם מאפשרים לגולשים להמליץ על הפודקאסט החביב עליהם כמו
             <a href="http://www.mefeedia.com/" target="_blank">
              Mefeedia
             </a>
             ואפשר גם ללכת ישירות לאתרי שיתוף (איחסון) פודקאסטים כמו:
             <a href="http://www.blip.tv/shows/?page=1" target="_blank">
              blip.tv
             </a>
             ,
             <a href="http://www.libsyn.com/index.php" target="_blank">
              libsyn
             </a>
             ,
             <a href="http://odeo.com/" target="_blank">
              Odeo
             </a>
             למי שמשתוקק לפודקאסטים בעיברית, ישנו את
             <a href="http://www.icast.co.il/" target="_blank">
              אתר הפודקאסט הישראלי
             </a>
             <br/>
             <br/>
             <strong>
              <em>
               איך מקשיבים לפודקאסטים?
              </em>
             </strong>
             <br/>
             <br/>
             כאמור אפשר לנגן בסטרימינג / להוריד ישירות מהאתר, למכורי ה-RSS יש כמובן את האפשרות להשתמש בתוכנה לפודקאסטים. התוכנה הותיקה ביותר היא
             <a href="http://juicereceiver.sourceforge.net/download/index.php" target="_blank">
              Juice
             </a>
             אשר מתאימה לכל הפלטפורמות, אלא שלטעמי היא פשוטה מידי.
             <br/>
             אני ממליצה על
             <a href="http://www.ziepod.com/" target="_blank">
              Ziepod
             </a>
             , תוכנה חינמית שמגדירה מחדש את המושג ממשק אסתטי, מכילה המון אפשרויות: כמו אפשרות להמשיך הורדות שניפסקו, נגן פנימי לאודיו (מי שירצה גם נגן וידאו פנימי יצטרך לקנות את Ziepod+), אפשרות סינכרון עם איי-פוד וחיפוש פנימי של פודקאסטים חדשים באינדקס פודקאסטים.
             <br/>
             למי שמתעקש גם על נגן וידאו פנימי יכול להשתמש בתוכנה החינמית
             <a href="http://www.noopod.com/index.php?page=download" target="_blank">
              Noopod
             </a>
             שבאה עם ה- VLC Media Player מובנה בתוכה וכן דפדפן פנימי, גם לה יש אפשרות סינכרון עם איי-פוד, אבל אין לה אפשרות להמשיך הורדות שניפסקו.
             <br/>
             אה, ואפשר להשתמש גם ב-
             <a href="http://www.apple.com/itunes/store/podcasts.html" target="_blank">
              iTunes
             </a>
             כמובן... אך קחו בחשבון שהאפשרות הזאת מגבילה אתכם רק לפידים שמותאמים ל-iTunes ולא כל הפידים הם כאלה.

          <div class="post">

           <h3 class="post-title">
            <a href="http://capina.blogspot.com/2007/05/art-work.html">
             art-work במוסיקה / פני ברסימנטוב
            </a>
           </h3>
           <div class="post-header-line-1">
           </div>
           <div class="post-body">
            <p>
             <em>
              כזכור בעבר כתבתי טור אורח
              <a href="http://www.kaseta.net/" target="_blank">
               לקסטה
              </a>
              בנושא
              <a href="http://kaseta.blogspot.com/2006/10/blog-post.html" target="_blank">
               אפקטים קוליים
              </a>
              , במסגרת חילופי משלחות קבלו טור אורח של פני ברסימנטוב לכבוד יום הולדת שנה לבלוג שלה שחל היום
             </em>
             <br/>
             <br/>
             <br/>
             לאחרונה נתקלתי בפעם הראשונה בהתייחסות אקדמית לעניין אומנות במוסיקה.
             <br/>
             <a href="http://www.artofrecordproduction.com/" target="_blank">
              arp
             </a>
             - art of music production הוא כתב עת יוקרתי שעוסק בכל ההיבטים של הפקת מוסיקה. בכנס arp שיתקיים בדצמבר 2007 באוסטרליה אחד הנושאים המפורשים שידונו בו יהיה "the art surrounding the art". עד יוני ישנה הזמנה
             <a href="http://www.artofrecordproduction.com/content/view/56/95/" target="_blank">
              להגיש הצעות למאמרים
             </a>
             (=הרצאות) שיעסקו באינטראקציה ובהשפעה של קליפים, עיצוב אתרים ועטיפות אלבומים. בהקשר של הבלוג הזה, אדבר על אנימציה בקליפים ואיור לעטיפות אלבומים:
             <br/>
             <br/>
             <strong>
              אנימציה
             </strong>
             <br/>
             מסוף שנות ה-80 כנראה שאין קליפ שלא נעשה בו שימוש מסוים באפקטים ויזואליים או שילוב של אנימציה ממוחשבת עם וידאו. עם כל הכבוד לפיקסאר ולסכומים שהושקעו בסרט ספיידרמן, מי שבאמת מאז ומתמיד קידם את התחום היתה המוסיקה. כשקוראים את
             <a href="http://en.wikipedia.org/wiki/Music_video" target="_blank">
              ההיסטוריה של ה- music video
             </a>
             (להלן MV) מגלים שאנימציה למוסיקה החלה כבר ב-1910. חידושים וקפיצות מדרגה הגיעו מאנימציה שנעשתה עבור
             <strong>
              מוסיקה קיימת
             </strong>
             : המבוגרים יספרו לכם איך נשמטה לצופים הלסת בסרט "פנטזיה" של וולט דיסני (1940) או "החומה" של פינק פלויד (1980).
             <br/>
             <br/>
             אבל אני רוצה לדבר על MV שיש בהם אנימציה טהורה. שימו לב
             <a href="http://en.wikipedia.org/wiki/List_of_music_videos_using_animation" target="_blank">
              לרשימה שלהם בויקיפדיה
             </a>
             : נכון להיום רשומים 193 כאלה (לא כולל
             <a href="http://e.walla.co.il/?w=/214/1105989" target="_blank">
              קליפים בפסטיבל האנימציה הבינ"ל ביוני הקרוב
             </a>
             !), ולא חסרים אמנים/להקות שבחרו באנימציה יותר מפעם אחת. על יד כל שיר רשומה הטכניקה העיקרית ויש ייצוג מכובד לתלת-ממד. הופתעתי שטכניקת
             <a href="http://en.wikipedia.org/wiki/Rotoscope" target="_blank">
              Rotoscope
             </a>
             מופיעה מעט, כנראה שמאז
             <a href="http://www.youtube.com/watch?v=u0R7dvLAiP8" target="_blank">
              Take On Me
             </a>
             הראשון והאלמותי של
             <em>
              א-הא
             </em>
             אף אחד לא רצה להסתכן ולהיות מושווה אליהם או אולי זה בגלל שהיום מחשבים עושים זאת בקלילות?
             <br/>
             <br/>
             בגדול יש הסכמה ש-MV פורץ דרך בהיסטוריה היה
             <a href="http://www.youtube.com/watch?v=VArQABogDL4" target="_blank">
              Sledgehammer
             </a>
             של פיטר גבריאל שהופק ע"י חברת
             <a href="http://www.aardman.com/" target="_blank" title="Aardman Animations">
              Aardman Animations
             </a>
             (וואלס וגרומיט). ב-1986 לא היו המחשבים והתוכנות של היום (וגם לא של לפני 4 שנים...) תארו לכם מה היתה המשמעות של הפקת הקליפ שבאמת טרם נראה כמותו. בין השאר, גבריאל היה צריך לשכב 16 שעות מתחת לזכוכית שדרכה צולם, בקליפ שולבו גם סטופ-מושן, גם claymation וגם
             <a href="http://en.wikipedia.org/wiki/Pixilation" target="_blank">
              pixilation
             </a>
             ועד היום לא נשבר שיא פרסי MTV לקליפ.
             <br/>
             <br/>
             ולמרות כל זאת, אני מציעה –
             <br/>
             א. להתאפק. נכון שיש כישרון וטכנולוגיות מדהימות, אבל זה לא אומר שתמיד מתאים ללכת על אנימציה כדי לעטוף ולהגיש שיר. לדעתי MV צריך לעשות שירות ולתת ערך מוסף לשיר ולא להפך. אנימציה יכולה לא להתאים לרוח השיר או לתוכנו (דוגמא צורמת שקופצת לראש היא
             <a href="http://www.youtube.com/watch?v=r8asQsfn82E" target="_blank" title="Paranoid Android">
              Paranoid Android
             </a>
             של רדיוהד), או כל כך מוטרפת וגאונית (קליפים של הבמאי
             <a href="http://www.michelgondry.com/index.php/Clips" target="_blank">
              מישל גונדרי
             </a>
             ) שהיא גונבת את ההצגה מהשיר ו"בולעת" אותו.
             <br/>
             ב. לא צריך תקציבי ענק. מוסיקאים יכולים לעשות את זה בעצמם פשוט ויפה, עם פוטושופ בלבד.
             <a href="http://capina.blogspot.com/2007/01/cities-are-moving.html" target="_blank">
              כפינה כתבה על זה
             </a>
             .
             <br/>
             <br/>
             <strong>
              איור ועיצוב גרפי
              <br/>
             </strong>
             כולכם יודעים שעטיפות תקליטים ודיסקים (והחוברות בתוכם) היו פי אלף יותר דומיננטיות לפני שהתחלנו לקנות ולהוריד שירים ואלבומים ברשת. זה מצער כי דווקא עכשיו, כשיש תוכנות איור וגרפיקה מעולות, התחום היה יכול לשגשג יותר.
             <br/>
             אבל אולי חסרונן של התוכנות בעשורים הקודמים גורמים לנו להעריך יותר עטיפות מרהיבות כמו אלה שהיו בשנות ה-80 ותחילת ה-90 לחברת
             <a href="http://www.4ad.com/" target="_blank">
              4AD
             </a>
             ,
             <a href="http://www.blender.com/guide/articles.aspx?id=1946" target="_blank">
              עטיפות המטאל
             </a>
             על סוגיו (אפילו
             <a href="http://www.amazon.co.uk/Heavy-Metal-Thunder-Covers-Rocked/dp/1845331958" target="_blank">
              כתבו על זה ספר
             </a>
             !), וכמובן הפרוגרסיב-רוק שנתנו לנו כמה עטיפות מצויירות שיישארו יפות לנצח (לחצו להגדלה):
             <br/>
             <br/>
             <a href="http://www.studentradion.no/rev/uploads/Image/Programmer/Revers/King_Crimson_-_In_The_Court_Of_The_Crimson_King-front.jpg" target="_blank">
              <img alt="The Court of The Crimson King - King Crimson" border="0" id="BLOGGER_PHOTO_ID_5066900484078229522" src="http://1.bp.blogspot.com/_HCWbv2lwKm4/RlE_JQm82BI/AAAAAAAAAFw/UDwYKoUGNe4/s320/kings.jpg" style="MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
             </a>
             <a href="http://perso.orange.fr/religionnaire/artistes/jethro_tull/art/aqualung.jpg" target="_blank">
              <img alt="Aqualung – Jethro Tull" border="0" id="BLOGGER_PHOTO_ID_5066900088941238274" src="http://1.bp.blogspot.com/_HCWbv2lwKm4/RlE-yQm82AI/AAAAAAAAAFo/Qs6bcFJ29zE/s320/aqualungs.jpg" style="MARGIN: 0px 10px 10px 0px; CURSOR: hand"/>
             </a>
             <br/>
             <br/>
             מי עושה היום art-work למוסיקה?
             <br/>
             סוגי יוצרים שאני מכירה (אתם מוזמנים להשלים את החסר):
             <br/>
             מאיירים, צלמים ומעצבים גרפים שעושים עבודות בכל מיני תחומים, גם למוסיקה. לדוגמא
             <a href="http://blog.dootdootgarden.com/2007/05/05/friend-and-foe/" target="_blank">
              אמן הקומיקס Craig Thompson
             </a>
             ;
             <br/>
             כאלה שמתפרנסים כמעט בלעדית מתעשיית המוסיקה (פוסטרים, עטיפות אלבומים).
             <a href="http://www.thesmallstakes.com/" target="_blank">
              לדוגמא Jason Munn
             </a>
             ;
             <br/>
             מוסיקאים שעושים זאת בעצמם.
             <a href="http://www.shanykedar.com/portfolio/index.html" target="_blank">
              לדוגמא שני קדר
             </a>
             שעיצבה את הדיסק שלה, את הלוגו של הלייבל שלה ועוד.
             <br/>
             <br/>
             לסיום כמה קישורים מומלצים שאספתי להמשך הדרך –
             <br/>
             <br/>
             1.
             <a href="http://www.albumcoverfinder.com/" target="_blank">
              album cover finder
             </a>
             - תוכנה שהיא מנוע חיפוש לעטיפות אלבומים
             <br/>
             <br/>
             2. חפשו בפליקר. הנה דוגמא לאוסף עטיפות תקליטים בכלל:
             <br/>
             <a href="http://www.flickr.com/groups/64513560@N00/pool" target="_blank">
              http://www.flickr.com/groups/64513560@N00/pool
             </a>
             <br/>
             והנה דוגמה לאוסף עטיפות תקליטים מצוירות בפרט:
             <br/>
             <a href="http://www.flickr.com/photos/leifpeng/sets/72057594054394206" target="_blank">
              http://www.flickr.com/photos/leifpeng/sets/72057594054394206
             </a>
             <br/>
             <br/>
             3.
             <a href="http://musicisart.ws/" target="_blank">
              music is art
             </a>
             - בלוג מיוחד שמדגים את המקום של הצד האסתטי במוסיקה, מעטיפות תקליטים ועד צילום אומנים. כדאי.
             <br/>
             <br/>
             4. מאגר עטיפות אלבומים שיש בהן איכויות אומנותיות בלבד:
             <br/>
             <a href="http://tralfaz-archives.com/coverart/coverart.html" target="_blank">
              http://tralfaz-archives.com/coverart/coverart.html
             </a>
             <br/>
             (שימו לב
             <a href="http://tralfaz-archives.com/coverart/coverartlinks.html" target="_blank">
              לעמוד הקישורים
             </a>
             שיש באתר למאגרים נוספים)
             <br/>
