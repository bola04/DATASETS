         <h2>
          <img height="16" src="images/blt_01.jpg" width="14"/>
          マカとはどんなもの？
         </h2>
         <h3>
          マカは南米ペルーの高山地帯を原産とする、アブラナ科の多年生植物です。
          <br/>
          日本でもサプリメントなどとして市販されており、滋養強壮、精力増強などに効果があるとされています。
          <br/>
          <a href="http://www.hwi.org/" target="_blank">
           マカの効能について
          </a>
          はこちらをご覧ください。
          <br/>
          本場ペルーでは古くから食用にされてきた植物で、紀元前頃にはすでに民間で用いられており、インカ帝国時代にも食物として重宝されてきました。
          <br/>
          <br/>
          マカは高山地帯特有の苛酷な自然環境で育つ特性を持ち、紫外線や雨の少ない土壌、かつ温度差の激しい気候でも育つ非常に屈強な植物です。
          <br/>
          厳しい環境で育つためか、土壌の栄養素をまんべんなく吸収して育ち、マカを収穫したあとの土壌は栄養分などが十分でなく、しばらくの間他の植物が栽培できなくなるとまで言われています。
          <br/>
          そんな
          <a href="http://www.abbeyroadgrill.com/" target="_blank">
           マカに秘められた高い栄養素
          </a>
          についてはこちらをご覧ください。
          <br/>
          <br/>
          マカは根から葉まで食べられるオールマイティな植物ですが、薬用の製品として用いる際は生食ではなく天日乾燥させた粉末を使用します。
          <br/>
          錠剤よりも粉末状のマカが良いとされていますが、有機栽培された原料を使っているものであれば、あまり形にはこだわらなくても良いでしょう。
          <br/>
          ちなみに、根の部分は乾燥させておくと数年間「保存食」として口にすることができるそうです。
          <br/>
          <br/>
          マカに含まれる栄養素は非常に豊富で、カルシウム・タンパク質などのほか、亜鉛や鉄分など貧血症状に効果的な成分も含み、さらに必須アミノ酸も多量に含んでいます。
          <br/>
          これらの栄養素が血流や体調を改善させ、それと同時に更年期障害やストレスなどにも一定の効果があることが判明しています。
          <br/>
          マウスを使った実験以外にも、馬や羊の飼料として用いられることもあるようで、マカを与えられた動物は疲れにくく、かつ多産になるという報告もあります。
          <br/>
          <br/>
          このように、マカには疲労や体力の低下に効果的な作用があり、ストレスだらけで生活も不規則になりがちな現代人にぴったりの植物といえます。
          <br/>
          肥満に対しても一定の改善作用があると報告されていますが、実際の効果についてはまだまだ未知数な部分も多く、今後さらに注目されそうな植物といえるでしょう。
          <br/>
          マカには育毛効果もあります。

          </a>
          を読んで参考になさってください。

         </h3>
