        </font>
        <font size="2">
         <br/>
        </font>
        <font size="2">
         <font color="#800000">

        </font>

         </font>

         </font>
        </li>

         <br/>
        </font>

        </font>
       </p>

          <br/>
         </font>
        </li>

          nach einem glücklichen, gesunden Leben?
          <br/>
         </font>
        </li>

         <font size="2">

         </font>
        </li>
       </ul>
       <p class="bodytext">
        <font size="2">
        </font>
        <font size="2">
         <br/>
         <font color="#800000">
          <strong>
           Eine ganzheitliche Betrachtungsweise zum Thema Krankheit und Gesundheit
          </strong>
         </font>

        </font>
        <font size="2">
         <br/>
         Allgemeine Erklärung:
         <br/>

        <font size="2">

        </font>

          körperlichen Ebene
          <br/>
         </font>
        </li>
        <li>

         </font>
        </li>

         </font>

         </font>

         <br/>
        </font>
        <font size="2">
         <br/>
        </font>
        <font size="2">
        </font>
        <font color="#800000" size="3">
         <br/>
         Ein Beispiel aus unserer Praxis am Symptom Nierenleiden / Blasenleiden:
         <br/>
        </font>
        <br/>
        Körperliche Ebene: Nieren- /Blasenerkrankung
        <br/>
        Emotionale Ebene: Ängste / Traurigkeit (Tränen/Trauer = Wasser)
        <br/>
        Mentale Ebene: Gedanken um die Angst verlassen zu werden, kreisen unkontrolliert.
        <br/>
        Spirituelle Ebene: fehlendes Urvertrauen
        <br/>

        <br/>
        <br/>
        Es ist also notwendig, das unliebsame Thema auf allen dieser vier Ebenen zu lösen. Geschieht dies nur auf einer Ebene verschiebt es sich, oft auch in anderer Form und Ausdruck, auf eine andere Ebene.
        <br/>
        <br/>
        <font color="#800000" size="3">
         Wie ein Muster bzw. Programm entsteht:
        </font>
        <strong>
         <br/>
        </strong>
        <br/>
        Ein Muster entsteht aus Ereignissen und Erfahrungen, in der Vergangenheit. Alles Erlebte wird in unserem Unterbewusstsein, sowie auch in unserem Energiefeld (Aura) abgespeichert und beeinflußt unser Leben. Es stellt so zu sagen eine Blockade dar. Der Grad der Schwere kann unterschiedlich ausgeprägt sein.
        <br/>
        <br/>
        Es kann in unserem Beispiel entstanden sein durch:
        <br/>
        <br/>
       </p>
       <ul>
        <li>
         Verlust seines Zwillinggeschwisters im Mutterleib
        </li>
        <li>
         Längere Trennung nach der Geburt von der Mutter (Brutkasten, spez. Notmaßnahme, ...)
        </li>
        <li>
         Krankenhausaufenthalt im Kindesalter
        </li>
        <li>
         Verlust/Trennung eines Elternteils
        </li>
        <li>
         Keine Mutter- und /oder Vater-Liebe
        </li>
        <li>
         Findel- oder Waisen-Kind
        </li>
        <li>
         Leben im Kinderheim
        </li>
        <li>
         Abwesenheit der Eltern
        </li>
        <li>
         Todesfälle im Familienfeld
         <br/>
        </li>
       </ul>
       <p class="bodytext">
        Ein solches, für das Kind meist traumatisches Erlebnis, wird wie bei einem Computer als Programm auf der Festplatte (Seele) abgespeichert.
        <br/>
        Mit einem Grafikprogramm kann man keine Rechen-Optionen durchführen! Das heißt, mit einem Verlassenheits-Konflikt (-Programm), kann man keine gesunde, erfüllte und glückliche Beziehung (Ehepartner, Vorgesetzten, Geschwister, Freunde, ) leben.
        <br/>
        Man hat in der Regel keine Wahl, als immer wieder die ursprüngliche Situation des Verlassenwerdens zu wiederholen.
        <br/>
        Dabei wird unser Schmerz und die Verzweiflung mit jeder Enttäuschung größer und die Blockade und Symptome werden ausgeprägter.
        <br/>
        <br/>
        <font color="#800000" size="3">
         Was ein Muster/-Programm der Verlassenheit bewirken kann?
        </font>
        <br/>
        <br/>
        Aus dem Erlebten, oft unbewussten Verlassensein, können sich Verhltensmuster wie diese ergeben:
        <br/>
        <br/>
       </p>
       <ul>
        <li>
         Eifersucht
        </li>
        <li>
         Hysterie
        </li>
        <li>
         Fremdgehen des Partners
        </li>
        <li>
         Untertauchen / Ableben des Partners
        </li>
        <li>
         Trennung/Scheidung
        </li>
        <li>
         Ortswechsel (beruflich) des Partners (weitweg, unerreichbar)
        </li>
        <li>
         Sich nur als zweite Wahl fühlen
        </li>
        <li>
         Einseitiges Verliebtsein - es kommt keine Partnerschaft zu Stande
        </li>
        <li>
         Ständige Gedanken: "Wie komme ich aus dieser Nummer wieder heraus?!"
         <br/>
        </li>
       </ul>
       <p class="bodytext">
        Kurzum unser geliebtes Gegenüber, verhält sich gemäß unseres Programmes (abgespeicherte Erfahrung).
