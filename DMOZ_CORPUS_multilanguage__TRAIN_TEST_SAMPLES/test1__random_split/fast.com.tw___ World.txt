        <h1 class="stats">
         (2) 資訊服務方案
        </h1>
        <p>
         <b>
          <font color="#2C9AB7">
           A. BI (Business Intelligence)商業智慧系統
          </font>
         </b>
         <br/>
         <br/>
         企業在日常營運過程中，往往會產生與作業、交易相關的資料。
         <br/>
         隨著導入企業資源規劃(ERP)、供應鏈管理(SCM)與客戶關係管理(CRM)等應用系統後，企業所積累的資料量極大。
         <br/>
         如何處理龐雜且量大的「資料」，並設法將其轉變成為有用的「資訊」，進而從中發現「趨勢」、形成「洞見」與「觀點」，
         <br/>
         以對企業之策略與行動方案提供相當程度的貢獻與助益，此即為商業智慧(Business Intelligence, BI)解決方案。
         <br/>
         企業可藉由BI的導入，將所蒐集之不同來源、大量的資料進行彙整與分析，將結果以報表或圖表方式呈現，不僅高階管理者得以運用，
         <br/>
         甚至可擴展至企業內部有資訊分析需求的一般員工也可使用，依據個人工作職務需求取得相關資訊及使用分析工具，與企業流程更加緊密結合，
         <br/>
         使員工對客戶需求或外界變化能夠快速反應。
         <br/>
         <br/>
         <b>
          <font color="#2C9AB7">
           B. 資訊安全與智慧機房資產管理解決方案
          </font>
         </b>
         <br/>
         <br/>
         資訊安全已成為各機構不可忽視的安全防護重點。根據專業機構Gartner Group的研究，59% 的意外網路停機時間發生在物理層，
         <br/>
         LAN Technology 的報告更指出，70%的網絡故障原因來自於實體線路。
         <br/>
         但目前一般資安解決方案，多半側重於軟體層面的防護，對於資訊實體線路安全的防護十分缺乏。
         <br/>
         我們的解決方案可以強化這塊資訊安全最脆弱的部分，並一次解決實體層網路設施自動監控與設備資產管理兩大需求。
         <br/>
         以TE Connectivity專利纜線晶片所發展出的這套解決方案，已應用到美國國防軍事、金融與醫療等最重視資安防護的領域，並通過美國國防部資安產品認證測試。
         <br/>
         在線路出現問題時可立即顯示問題發生的節點，節省大量尋找故障節點的人力與時間。機房線路在非授權狀態下被變更會發生警示，並留下紀錄。
         <br/>
         同時在資訊設備變更時，可自動建置網路鍊路圖，無須再花費大量時間以人工進行線路繪製。
         <br/>
         <br/>
         <b>

           C. 資訊軟硬體服務

         <br/>
