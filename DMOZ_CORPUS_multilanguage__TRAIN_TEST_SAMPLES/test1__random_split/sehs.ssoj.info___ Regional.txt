          新SEHS Next Tide Workshop（NTW）の報告者募集について
          <br/>
          <br/>
          2011年度から実施されてきたSEHS Next Tide Workshop（社会経済史学会次世代研究者育成ワークショップ）について、本年度は新たな試みとして、添付の要領で開催します。これは、『社会経済史学』への投稿を念頭において論文を執筆している研究者に対して、執筆途中の論文の内容を発表して学術的・実践的なコメントを受ける機会を提供するとともに、論文執筆の「コツ」を参加者の間で共有することを目的とするものです。つきましては、以下の要領で、報告を公募いたします。
          <br/>
          <br/>
          ■期日：2018年9月15日（土）
          <br/>
          ■会場：早稲田大学　早稲田キャンパス（東京都新宿区）
          <br/>
          ■申込期限：2018年6月30日（土）
          <br/>
          ■申込方法：郵便にてお送りください。

          社会経済史学会第87回全国大会における託児室の設置について
          <br/>
          <br/>
          社会経済史学会員の皆様
          <br/>
          <br/>
          第87回全国大会実行委員会（大阪大学）では、大会期間中、以下の要領で会場内にベビーシッター会社の運営委託による託児室の設置を計画しております。ご利用希望の方は、4月30日（月）までに、sehs@econ.osaka-u.ac.jp宛にご連絡ください。とりまとめの上、5月初旬に折り返し申し込み方法の詳細をお伝えいたします。
          <br/>
          <br/>
          託児時間：両日とも9:00〜17:00（予定）
          <br/>
          託児定員：10名程度
          <br/>
          対象年齢：生後3ヶ月程度〜小学6年生
          <br/>
          利用料金：1時間あたり1,000円
          <br/>
          委託業者：（株）ポピンズ（
          <a href="https://www.poppins.co.jp/" target="_blank">
           https://www.poppins.co.jp/
          </a>
          ）
          <br/>
          申込方法：2018年4月30日（月）までに、ご利用希望のご意向をメールにてお知らせください（sehs@econ.osaka-u.ac.jp宛）。
