                       <td>
                        <img align="left" src="images/icons/comu.gif"/>
                        Como parece que este ańo hay algunos usuarios que se están animando para el concurso de mapas, este ańo volveremos a retomar lo que era una tradición en Duke Espańa: el
                        <span style="font-weight: bold">
                         concurso anual de mapas de Duke Nukem 3D
                        </span>
                        .
                        <br/>
                        <br/>
                        Os recordamos de las
                        <a class="postlink" href="http://www.dukespana.com/concursos_build_normas.php" target="_blank">
                         normas del concurso de mapas
                        </a>
                        las más importantes:
                        <br/>
                        <br/>
                        <ul>
                         <br/>
                         <li>
                          El mapa debe ser
                          <span style="font-weight: bold">
                           original
                          </span>
                          y tuyo al 100%. Esto implica que no se aceptan mapas ya presentados en anteriores concursos ni de otros usuarios.
                          <br/>
                         </li>
                         <li>
                          El mapa puede ser para JFDuke o para Eduke32. Esto incluye los límites ampliados de Eduke32 (en las normas tenéis una referencia a estos).
                          <br/>
                         </li>
                         <li>
                          El mapa NO debe contener tiles nuevas, ficheros scripts CON o cualquier otro ańadido con la excepción de un
                          <span style="font-weight: bold">
                           fichero de música
                          </span>
                          para la ambientación. Como novedad empezaremos a admitir ficheros ogg como música ya que prácticamente todo el mundo juega con Eduke32.
                          <br/>
                         </li>
                         <li>
                          Tampoco se admitirán mapas que no sean jugables con el renderizador de Duke3D Clásico implementado en Eduke32. Esto implica que no admitiremos ni mapas para Polymost ni mapas para Polymer.
                          <br/>
                         </li>
                        </ul>
                        <br/>
                        <br/>
                        Las fechas para este ańo son:
                        <br/>
                        <ul>
                         <br/>
                         <li>
                          Se admitirán los mapas enviados durante el mes de
                          <span style="font-weight: bold">
                           enero del ańo 2017
                          </span>
                          , cerrándose el plazo de envío el día 31 de enero del 2017.
                          <br/>
                         </li>
                         <li>
                          El
                          <span style="font-weight: bold">
                           proceso de votaciones
                          </span>
                          durará durante todo el mes de
                          <span style="font-weight: bold">
                           febrero del 2017
                          </span>
                          . El día 28 de febrero del 2017 se cerrará el proceso de votaciones.
                          <br/>
                         </li>
                         <li>
                          Un par de días después, a partir del 1 de marzo del 2017, publicaremos los resultados.
                          <br/>
                         </li>
                        </ul>
                        <br/>
                        <br/>
                        <br/>
                        ĄEsperamos vuestros mapas!
                       </td>
