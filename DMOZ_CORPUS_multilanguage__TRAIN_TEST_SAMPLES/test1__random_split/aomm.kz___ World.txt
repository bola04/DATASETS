                       <span style="font-family: tahoma,arial,helvetica,sans-serif;">
                        <span style="color: #ff9d00;">

                        </span>
                       </span>
                      </h1>
                      <p>
                       <span style="font-family: tahoma, arial, helvetica, sans-serif; color: #000000;">
                        Наше предприятие АО «Мунаймаш» образовалось в 1941 году на базе
                        <br/>
                        эвакуированного Таганрогского завода «Красный гидропресс» и во время
                        <br/>
                        Великой Отечественной войны работало на оборону. После войны завод
                        <br/>
                        выпускал оборудование для народного хозяйства, а затем вошёл в Министерство
                        <br/>
                        судостроения СССР и был переориентирован на выпуск специальной продукции,
                        <br/>
                        в частности высокоточных механических приборов и электроники.
                        <br/>
                        На предприятии АО «Мунаймаш» реализован инвестиционный проект
                        <br/>
                        « Обновление технологической база АО «Мунаймаш», который позволяет как
                        <br/>
                        увеличить конкурентоспособность выпускаемой продукции, так и изготавливать
                        <br/>
                        новые образцы нефтегазопромыслового оборудования, АО «Мунаймаш» в
                        <br/>
                        настоящее время выпускает нефтегазовое оборудование:
                        <br/>
                        - Скважинные штанговые насосы;
                        <br/>
                        - Инструмент КРС, ПРС;
                        <br/>
                        - Запчасти к нефтегазовому оборудованию.
                        <br/>
                        Основным видом выпускаемой продукции – являются СШН (скважинные
                        <br/>
                        штанговые насосы) и дополнительные защитные устройства к ним. Продукция
                        <br/>
                        производиться по оригинальной технологии и обладает высокой точностью,
                        <br/>
                        антикоррозионной устойчивостью к агрессивным жидким и газовым средам,
                        <br/>
                        повышенной износостойкостью. По эксплуатационно-техническим
                        <br/>
                        характеристикам насосы находятся на уровне лучших мировых образцов.
                        <br/>
                        Предназначенная к производству продукция, разрабатывается с учетом
                        <br/>
                        требований нефтедобывающих предприятий, выработанных в процессе
                        <br/>
                        эксплуатации, аналоговых изделий и может быть доработано с учетом
                        <br/>
                        требований заказчика.
                        <br/>
                        Продукция выпускается в соответствии с ГОСТ Р 51896-2002 и стандартом
                        <br/>
                        API spec. 11AX.
                       </span>
                      </p>
                      <p>
                       <span style="font-family: tahoma, arial, helvetica, sans-serif; color: #000000;">
                        <br/>
                        <strong>
                         НАША МИССИЯ
                        </strong>
                        - быть одним из лидирующих промышленных предприятий
                        <br/>
                        Казахстана.
                       </span>
                      </p>
                      <p>
                       <span style="font-family: tahoma, arial, helvetica, sans-serif; color: #000000;">
                        <br/>
                        <strong>
                         НАШИ ЦЕЛИ
                        </strong>
                        - производство и обеспечение рынка продукцией, применяемой
                        <br/>
                        нефтедобывающей и нефтеперерабатывающей отрасли Казахстана и СНГ, на
                        <br/>
                        основе анализа существующих и будущих потребностей заинтересованных
                        <br/>
                        сторон и постоянного улучшения процессов. Понятие “качество” распространить
                        <br/>
                        на все виды деятельности предприятия. Удовлетворять запросы потребителей
                        <br/>
                        настоящие и будущие. Поддерживать высокую рыночную репутацию на
                        <br/>
                        освоенных и расширяемых сегментах рынка. Устанавливать и развивать
                        <br/>
                        взаимовыгодное и долгосрочное сотрудничество с потребителями и
                        <br/>
                        поставщиками. Совершенствовать технологию производства, путём
                        <br/>
                        взаимодействия с другими предприятиями Казахстана и СНГ.

                        <br/>

                        <br/>
                        ресурсов. Эффективно управлять производственным оборудованием, внедрять
                        <br/>
                        новые технологии, процессы и материалы.
