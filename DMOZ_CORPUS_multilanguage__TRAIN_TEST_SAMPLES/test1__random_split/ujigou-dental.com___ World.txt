       <h1>
        笑顔、健康、そして白い歯。
       </h1>
       倉敷のうじごう歯科医院では、痛みや不安を持って来院された患者さんをスタッフ全員の明るい笑顔でお迎えし、十分なコミュニケーション、納得の説明、そして適切な歯科治療で、痛みと不安を解消させることにより、お帰りの際には自然に笑顔がこぼれるような歯科医院づくりを目指しています。
       <br/>
       <br/>
       まずは、主訴の改善を速やかに行います。
       <br/>
       その後は、十分な検査、診断を行い、治療方針について患者さんと納得行くまで話し合います。
       <br/>
       <br/>
       治療に入りますと、最大限に痛くない治療を心がけ、虫歯や歯周病によって破壊された口腔内の機能回復に、スタッフ一丸となって努力します。
       <br/>
       <br/>
       当歯科医院の治療方針は、従来の「治療を中心とした歯科医療」から ｢予防を中心とした歯科医療｣ へと大きく転換し、虫歯や歯周病の治療のために来られた患者さんの治療を予防的観点から行うとともに、治療後の健康維持のためのメインテナンス、さらに一歩進んで健康な人たちにも通ってもらえる歯科医院づくりに取り組んでいます。
       <br/>
       <br/>
       健康な方が、定期的に美容院へいくような感覚で歯科医院へ通い、口腔内の健康づくりを自ら進んで行い、一生涯自分の歯でおいしく食べるという大きな目標に向かって一緒に頑張ってみませんか?
       <br/>
       <br/>
       倉敷の地で歯科を開院して20年余り、
       <br/>
       患者さんが笑顔で通える歯科医院づくりを目指して、我々スタッフ一同は毎日努力しています。
       <br/>
       そして健康の象徴とも言える白い歯の獲得と皆様の幸せにつながる歯科治療をお約束します。
