      <div align="justify" style="margin:30px;">
       <br/>
       <div align="center" class="texte_grand">
        Bienvenue sur notre portail consacré à lélectronique
       </div>
       <br/>
       Créé en l'an 2000, abcelectronique.com est devenu au fil des années une référence en matière de communauté pour les professionnels et les amateurs dans le domaine de l'électronique et des circuits intégrés qu'ils soient débutants ou confirmés.
       <br/>
       <br/>
       Toute léquipe vous souhaite bonne visite et espère que ce site vous apportera satisfaction.
       <br/>
       <br/>
       Première visite, voici une brève description des rubriques principales qui vous permettront de découvrir rapidement les fonctionnalités du site.
       <br/>
       <br/>
       »
       <strong>
        Info composants
       </strong>
       : Rubrique qui référence les circuits intégrés et les composants discrets : numéro de série "part number", fabricant, fonction, description et fiche technique "datasheet". La recherche dun circuit donné peut être effectuée soit par son numéro de série, soit par sa fonction. Dautres utilitaires sont aussi fournis comme les équivalences, les types de boîtiers, les broches... . Pour pouvoir consulter cette rubrique, une inscription au préalable est demandée. Linscription est gratuite.
       <br/>
       <br/>
