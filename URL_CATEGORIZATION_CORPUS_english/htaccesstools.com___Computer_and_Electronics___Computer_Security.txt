        <h1>
         Welcome to .htaccess Tools
        </h1>
       </div>
       <div class="entry">
        <p>
         One this site you will find various generators to create .htaccess files for ie hotlink protection, password protection, or blocking of hitbots.
        </p>
        <h3>
         Help other users discover Htaccess Tools
        </h3>
        <p>
         This site is created to help other webmasters learn htaccess. You can +1 this site or any article to acknowledge its usefulness.
        </p>
        <div>
        </div>
        <h3>
         iPhone detection and redirection
        </h3>
        <p>
         I have written an article on how to use htaccess to
         <a href="/articles/detect-and-redirect-iphone/" title="Learn how to use iPhone detection and redirection to target iPhone users.">
          detect and redirect iPhone
         </a>
         users, by examining the User-Agent header iPhone sends.
        </p>
        <h3>
         Htaccess redirection
        </h3>
        <p>
         Htaccess can be used to create various forms of redirection. Read the
         <a href="/articles/htaccess-redirect/">
          htaccess redirection
         </a>
         article and learn how to redirect from old domain to new domain, or redirect to www.
        </p>
        <h3>
         Htpasswd Generator – Create a htpasswd password
        </h3>
        <p>
         Use the
         <a href="/htpasswd-generator/">
          htpasswd generator
         </a>
         to create passwords for htpasswd files. Just enter username and password and an entry for a htpasswd file is generated.
        </p>
        <h3>
         How to create a password for a .htpasswd file using PHP
        </h3>
        <p>
         A user emailed me and asked how to
         <a href="/articles/create-password-for-htpasswd-file-using-php/">
          create a password for a .htpasswd file using PHP
         </a>
         . Take a look at the article.
        </p>
        <h3>
         .htaccess FAQ
        </h3>
        <p>
         I have made a
         <a href="/htaccess-faq/">
          htaccess FAQ
         </a>
         with answers to the most frequently asked questions about .htaccess and mod_rewrite. I will continuously update the FAQ with more questions and answers.
        </p>
