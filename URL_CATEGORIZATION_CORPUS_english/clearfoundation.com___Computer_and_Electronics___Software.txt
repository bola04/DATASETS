        <h3>
         What Are CLEAR Tokens And What Is ClearNODE?
        </h3>
       </div>
       <div class="partallcontent-set margn-top20">
        <p>
         CLEAR Token is a socially responsible utility token from ClearFoundation that is designed to help build, promote, and defend blockchain-based technologies while enabling a platform and marketplace to make blockchain apps secure and reliable.
         <br/>
         <br/>
         50% of the total proceeds from CLEAR Token will fund a Stability Endowment for ClearFoundation and its mission while the remaining 50% will fund the Utility Platform.
         <br/>
         <br/>
         CLEAR Token can also be used to purchase ClearCenter’s ClearNODE hardware, software, and services.
         <br/>
         <br/>
         ClearNODE is a secure backbone infrastructure, platform, and marketplace for blockchains, digital currencies, and tokens. ClearNODE is a combination of HPE ProLiant server hardware, ClearOS operating system, and ClearCenter software and services via the ClearOS Marketplace.
         <br/>
         <br/>
         You will be able to use ClearNODE as a secure decentralized storage device, a hybrid hardware/software wallet, to mine CLEAR Tokens, and more.
        </p>

        <div class="partallcontent-set margn-top20">
         <p>
          The ClearFoundation is a New Zealand company which was created in 2009 with the purpose of developing and maintaining ClearOS under a Simple, Secure, and Affordable open source model. The foundation exists to ensure that every small business and distributed IT environment on the globe can have access to enterprise-grade IT security, privacy, filtration, and management tools.
         </p>
         <p>
          ClearOS began life as ClarkConnect in May 2000 as an innovative and easy-to-use Linux-based operating system with a simple web-based GUI featuring automatic updates via a Service Delivery Network. The original developers of ClarkConnect (Peter Baldwin and Ben Chambers) continue to maintain and grow ClearOS today in addition to the thousands of Linux and open source developers and community members who maintain its core functionality from around the globe.
         </p>

         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse2">
            When will the ClearNODE software and hardware backbone infrastructure be available?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse2">
          <div class="panel-body">
           The current roadmap plans for availability in Q2 2018.
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse3">
            Do I need to own CLEAR Tokens to buy and use the ClearNODE software and hardware backbone infrastructure?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse3">
          <div class="panel-body">
           Yes
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse4">
            How can I earn free CLEAR Tokens?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse4">
          <div class="panel-body">
           Purchase the ClearNODE infrastructure then start using the Distributed Storage app and Mining app.
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse5">
            Can I run the ClearNODE software and hardware backbone infrastructure on my own hardware?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse5">
          <div class="panel-body">
           Yes. However, keep in mind you may not be able to take advantage of one key feature — Silicon Root of Trust on select HPE ProLiant servers . The ClearNODE infrastructure software is designed for bundled use on pre-configured HPE ProLiant servers. This allows for us to secure all four critical layers — Server, Chip, Disk and Apps.
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse6">
            Can the ClearNODE infrastructure be used for other blockchain and/or mining of other digital currency?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse6">
          <div class="panel-body">
           Yes, as long as there is an app for your desired service inside of the ClearOS Marketplace.
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse7">
            What is ClearOS?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse7">
          <div class="panel-body">
           ClearOS is the only Simple, Secure, and Affordable way to manage IT resources. ClearOS is a Linux-based open source operating system for managing Cloud, Server, Network and Gateway layers. Thanks to its web-based GUI and the ClearOS Marketplace, you can fully utilize the power of Linux services without becoming a Linux expert or knowing how to use the Linux command line interface.
          </div>
         </div>
        </div>
        <div class="panel panel-default">
         <div class="panel-heading">
          <h4 class="panel-title">
           <a data-parent="#accordion" data-toggle="collapse" href="#collapse8">
            Who is ClearFoundation?
           </a>
          </h4>
         </div>
         <div class="panel-collapse collapse" id="collapse8">
          <div class="panel-body">
           ClearFoundation develops and maintains Clear products including ClearOS for ClearCenter and the broader open source community. The foundation exists to ensure that every small business and distributed IT  environment on the globe can have access to enterprise-grade IT security, privacy, filtration, and management tools.

    $(document).ready(function(){
      //$('.homebanner-content').addClass('stick');
     $(window).scroll(function(){
        //console.log($('#token').offset().top);
       
       if (  $(window).scrollTop() > $('#hometp').offset().top-150 ) {
           $('#token,#vision,#resource, #contact,#history,#team,#products,#overview').removeClass('stick');
          $('#hometp').addClass('stick');
       }
       if (  $(window).scrollTop() > $('#overview').offset().top-150 ) {
           $('#token,#vision,#hometp').removeClass('stick');
          $('#overview').addClass('stick');
       }
        if (  $(window).scrollTop() > $('#vision').offset().top-150 ) {
           $('#overview,#products,#hometp').removeClass('stick');
          $('#vision').addClass('stick');
       }
        if (  $(window).scrollTop() > $('#products').offset().top-150 ) {
           $('#vision,#team,#hometp').removeClass('stick');
          $('#products').addClass('stick');
       }
        if (  $(window).scrollTop() > $('#team').offset().top-150 ) {
           $('#products,#history,#hometp').removeClass('stick');
          $('#team').addClass('stick');
       }
        if (  $(window).scrollTop() > $('#history').offset().top-150 ) {
           $('#team,#resource,#hometp').removeClass('stick');
          $('#history').addClass('stick');
       }
        if (  $(window).scrollTop() > $('#resource').offset().top-150 ) {
           $('#history,#contact,#token,#hometp').removeClass('stick');
          $('#resource').addClass('stick');
       }
       if ( $(window).scrollTop() > $('#token').offset().top-150 ) {
           $('#overview, #hometp, #history,#resource,#contact').removeClass('stick');
           $('#token').addClass('stick');
       }
       if (  $(window).scrollTop() > $('#contact').offset().top-150 ) {
           $('#resource,#hometp,#token').removeClass('stick');
          $('#contact').addClass('stick');
       }    
       });
 
 
   });
   </script>
