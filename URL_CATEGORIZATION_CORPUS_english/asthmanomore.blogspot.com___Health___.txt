                         FREKUENSI TENAGA
                         <br/>
                         MENGANDUNGI 13 JENIS MINERAL SEMULAJADI DIGABUNGKAN MENGGUNAKAN TEKNIK “ NANO FUSION” KUASA HABA YG TINGGI KE DALAM KEPINGAN “DISC”.
                         <br/>
                         KOMBINASI PENGGABUNGAN MINERAL SEMULAJADI (FUSION TECHNIQUES) MENJADI AJEN PEMANGKIN TENAGA,
                         <br/>
                         “ SCALAR ENERGY”.
                         <br/>
                         <br/>
                         EFEKTIF UNTUK;-
                         <br/>
                         1. Meningkatkan IMUN (Daya Tahan Badan) untuk melawan penyakit secara SEMULAJADI seperti: Asma, Kanser, Resdung, Demam, Batuk, Keletihan, Gout, Lumpuh, Lupus, Darah Tinggi, Sengal Sendi, Lemah Jantung, Gastrik, Sendi, Ketumbuhan, Migrain, Susah Tidur, Diabetis, Batu Karang, Buasir, Senggugut, Stress Menambah Keupayaan Seksual dan BANYAK LAGI.
                         <br/>
                         (TELAH TERBUKTI RAMAI YANG MENGGUNA)
                         <br/>
                         2. Menjimatkan Petrol, Diesel, Gas dan Bahan Api.
                         <br/>
                         3. Awet Muda/Kecantikan , Menambah Kualiti alat solek
                         <br/>
                         4. Tenaga / Ketangkasan (untuk Sukan / Kerja Seharian)
                         <br/>
                         5. Meningkatkan I.Q untuk Belajar @ Kerja
                         <br/>
                         6. Pro MultiMineral
                         <br/>
                         7. Mengaktifkan Sel, Menambah Oksigen
                         <br/>
                         &amp; BANYAK LAGI, INSYA ALLAH.
                         <br/>
                         <br/>
                         KEAJAIBAN DUNIA KESIHATAN
                         <br/>
                         <br/>
                         PENYEMBUHAN SEMULAJADI
                         <br/>
                         MERAWAT PENYAKIT TANPA UBATAN &amp; PEMBEDAHAN
                         <br/>
                         <br/>
                         KEISTIMEWAAN BIODISC:
                         <br/>
                         <br/>
                         **Kaedah Rawatan tanpa Ubat &amp; Pembedahan
                         <br/>
                         **Merawat Pelbagai jenis penyakit secara semulajadi.
                         <br/>
                         **Tidak perlu mengulangi pembelian ubat,setiap minggu,bulan,tahun
                         <br/>
                         **Hanya menggunakan Air bertenaga Biodisc &amp; cahaya FIR+Biodisc
                         <br/>
                         **Jangkahayat Biodisc sehingga 20 tahun atau lebih
                         <br/>
                         **Rawatan Mudah: Lakukan sendiri di rumah
                         <br/>
                         **Amat Menakjubkan
