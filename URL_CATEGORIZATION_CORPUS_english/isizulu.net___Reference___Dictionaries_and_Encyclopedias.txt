      <h1 class="center" style="font-size:larger">
       isiZulu.net: Bilingual Zulu-English dictionary
      </h1>
      <p>
       isiZulu.net is a hobby project attempting to build a reasonably useful,
 modern Zulu-English online dictionary that anyone can
       <a href="forum/">
        contribute
       </a>
       to. Besides the usual bidirectional lookups it offers automatic
 morphological decomposition, conjugation and phonetic spelling of Zulu words,
 translation of simple Zulu and English phrases, spell checking, and grammar and
 pronunciation basics.
      </p>
      <hr style="margin-top:1em"/>
      <h4>
       <em>
        PINNED:
       </em>
       A side note on papers
      </h4>
      <p>
       If you mention the dictionary in a paper, lecture etc. I'd be happy to
 receive an advance copy along with your comments.
      </p>

       </em>
       isiZulu.net presented at AFRILEX 2012
      </h4>
      <p>
       isiZulu.net is used as a reference work in the evaluation of a
 Lëtzebuergesch-German online dictionary at the
       <a href="http://afrilex.africanlanguages.com/afrilex2012.pdf">
        AFRILEX Conference
 2012
       </a>
       .
      </p>
      <h4>
       <em>
        2012-02-08:
       </em>
       Browser search plug-in
      </h4>
      <p>
       You can add isiZulu.net as a search engine to browsers supporting OpenSearch
 (e.g. Firefox).
      </p>
      <h4>
       <em>
        2011-07-16:

      </h4>
      <p>
       The dictionary offers a
       <a href="//api.isizulu.net/doc/">
        REST interface
       </a>
       that returns JSON data. To use it you need an API key that you can request
       <a href="contact/">
        by e-mail
       </a>
       .
      </p>
