          <h2>
           Tip of the Day
          </h2>
          <h3>
           Add your name to the status bar
          </h3>
          <p>
           <img alt="modemacro" class="right" height="171" src="/site/tips/images/modemacro.gif" width="334"/>
           Have you ever wanted to see your name or your company's name displayed on the AutoCAD status bar? If so, you need to know about the
           <span class="command">
            modemacro
           </span>
           variable. Whatever value you set this variable to will appear on the status bar, just to the left of the dynamic co-ordinate display. To do this, enter "modemacro" at the command prompt and then enter the text you'd like to display.
          </p>
          <p>
           The assigned value of modemacro remains set only for the current drawing session. If you's like to see your text displayed every time you start AutoCAD, simply add the following line to acad.lsp:
           <br/>
           <span class="code">
            (setvar "modemacro" "text to display")
           </span>
          </p>
