              Linking to a database search in Mobialia Chess
             </a>
            </h2>
           </header>
           <section class="entry-summary">
            <p>
             Now you can link from external websites to a specific database query in the Mobialia Chess web app with Urls like:
            </p>
            <ul>
             <li>
              <a href="https://chess.mobialia.com/?db_query=Carlsen+Karjakin" target="_blank">
               https://chess.mobialia.com/?db_query=Carlsen+Karjakin
              </a>
             </li>
             <li>
              <a href="https://chess.mobialia.com/?db_query=Deep+Blue">
               https://chess.mobialia.com/?db_query=Deep+Blue
              </a>
             </li>
            </ul>
            <p>
             You must pass the “
             <em>
              ?db_query=
             </em>
             ” parameter with a Url encoded value. Multiple values must be separated by “+”. This encoding can be done with a online Url encoder like
             <a href="http://www.url-encode-decode.com/">
              http://www.url-encode-decode.com/
             </a>
             .
            </p>
            <p>
             The link opens the Mobialia chess web app in the database game list screen. The passed value is searched in the player names, events, sites and dates.
            </p>

            <h3>
             Analysis in online games
            </h3>
            <p>
             Playing online it will be only available while observing or examining a game. It does not allow to analyze a game while playing because it could be used for cheating.
            </p>
            <p>
             When a played game is finished, you can examine it with the “Examine” menu option and then start the analysis. You can also browse your played games in the “History” section to analyze them.
            </p>
            <h3>
             Openings names and ECO codes
            </h3>
            <p>
             The analysis window also shows the opening name and the ECO code (
             <a href="https://en.wikipedia.org/wiki/Encyclopaedia_of_Chess_Openings">
              https://en.wikipedia.org/wiki/Encyclopaedia_of_Chess_Openings
             </a>
             ). The opening names are available in 5 languages and they also include the name of the variants.
            </p>
            <p>
             The openings database was very big for being embedded with the app, so it is done querying an online service in the Mobialia servers.
            </p>
            <h3>
             Updated engine
            </h3>
            <p>
             This new version includes the last Carballo Chess Engine 1.5 with a great strength increase over the previous version, we worked a lot the last months in our opensource chess engine: (
             <a href="https://github.com/albertoruibal/carballo/">
              https://github.com/albertoruibal/carballo/
             </a>
             ). In the analysis mode the engine works always at full strength. Playing against the engine, we keep the ELO levels 500-2100, but probably now it is a bit stronger.
            </p>
