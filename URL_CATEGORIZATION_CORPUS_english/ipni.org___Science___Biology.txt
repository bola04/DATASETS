      <h2>
       <a name="top">
       </a>
       About IPNI
      </h2>
      <p>
       The International Plant Names Index (IPNI) is a database of the names 
         and associated basic bibliographical details of seed plants, ferns and lycophytes. Its goal is to eliminate the need for repeated reference 
         to primary sources for basic bibliographic information about plant names. 
         The data are freely available and are gradually being standardized and 
         checked. IPNI is a dynamic resource, depending on direct contributions 
         by all members of the botanical community.
      </p>
      <p>
       IPNI is the product of a collaboration between
       <a href="http://www.kew.org/" target="_blank">
        The Royal 
         Botanic Gardens, Kew
       </a>
       ,
       <a href="http://www.huh.harvard.edu/" target="_blank">
        The Harvard 
         University Herbaria
       </a>
       , and the
       <a href="http://www.anbg.gov.au/cpbr/herbarium/" target="_blank">
        Australian National Herbarium
       </a>
       .
      </p>
      <p>
      </p>
      <h2>
       NEWS
      </h2>
      <ul>
       <li>
        <p>
         <a href="http://www.iapt-taxon.org/nomen/main.php">
          Melbourne Code
         </a>
         available online.
        </p>
       </li>
       <li>
        <p>
         From 1 January 2012 electronic material published online in Portable Document Format (
         <a href="http://en.wikipedia.org/wiki/PDF">
          PDF
         </a>
         ) with an International Standard Serial Number  (
         <a href="http://www.issn.org/2-22635-What-is-an-ISSN.php">
          ISSN
         </a>
         ) or an International Standard Book Number (
         <a href="http://www.isbn-international.org/">
          ISBN
         </a>
         ) constitutes effective publication.
         <br/>
         <br/>
         We want to make sure that IPNI captures all relevant nomenclatural novelties, whether published online or in printed form.
         <br/>
         <br/>
         IPNI users can help.
         <u>
          <a href="mailto:ipnieditors@ipni.org">
           Let us know
          </a>
         </u>
         if you publish or find names in an online  publication by sending us a link to the relevant site.
        </p>
       </li>
       <li>
        <p>
         To help you to quickly find information on authors, many new links have been created from author records to
         <a href="http://www.sil.si.edu/digitalcollections/tl-2/index.cfm">
          TL-2
         </a>
         page in
         <a href="http://www.biodiversitylibrary.org/">
          BHL
         </a>
         . Try it out. Click on the link in
         <a href="http://www.ipni.org/ipni/idAuthorSearch.do?id=6787-1&amp;back_page=/ipni/editAdvPlantNameSearch.do?find_infragenus=&amp;find_isAPNIRecord=true&amp;find_geoUnit=&amp;find_includePublicationAuthors=true&amp;find_addedSince=&amp;find_family=&amp;find_genus=chry*&amp;find_sortByFamily=true&amp;find_isGCIRecord=true&amp;find_infrafamily=&amp;find_rankToReturn=all&amp;find_publicationTitle=&amp;find_authorAbbrev=&amp;find_infraspecies=&amp;find_includeBasionymAuthors=true&amp;find_modifiedSince=&amp;find_isIKRecord=true&amp;find_species=par*&amp;output_format=normal">
          this record.
         </a>
        </p>
       </li>
       <li>
        <p>
         Some records now have a link to the protologue page in
         <a href="http://www.biodiversitylibrary.org/">
          BHL
         </a>
         . See an example
         <a href="http://www.ipni.org/ipni/idPlantNameSearch.do?id=227935-2&amp;back_page=/ipni/editAdvPlantNameSearch.do?find_infragenus=&amp;find_isAPNIRecord=true&amp;find_geoUnit=&amp;find_includePublicationAuthors=true&amp;find_addedSince=&amp;find_family=lamiaceae&amp;find_genus=&amp;find_sortByFamily=true&amp;find_isGCIRecord=true&amp;find_infrafamily=&amp;find_rankToReturn=all&amp;find_publicationTitle=Ann*miss*bot*gard*&amp;find_authorAbbrev=&amp;find_infraspecies=&amp;find_includeBasionymAuthors=true&amp;find_modifiedSince=&amp;find_isIKRecord=true&amp;find_species=&amp;output_format=normal">
          here
         </a>
         .
        </p>
       </li>
       <li>
        <p>
         <a href="missing_protologues.html">
          Missing protologues
         </a>
         - can you help us find some of these?
        </p>
       </li>
      </ul>
      <h2>
      </h2>
      <h2>
       TIPS
      </h2>
      <ul>
       <li>
        <p>
         IMPORTANT: IPNI does not have information on what are currently accepted names and what are taxonomic (i.e. heterotypic) synonyms. Find this information in floras, monographs, checklists, revisions etc.
        </p>
       </li>
       <li>
        <p>
         See new names just added by
         <a href="http://www.ipni.org/ipni/editAdvPlantNameSearch.do?show_extras=true&amp;find_includeBasionymAuthors=true&amp;find_includePublicationAuthors=true&amp;find_isAPNIRecord=true&amp;find_isIKRecord=true&amp;find_isGCIRecord=true&amp;find_sortByFamily=true">
          searching by additional terms
         </a>
         . Search must be limited by at least one term, like family, genus or author.
        </p>
       </li>
      </ul>
      <ul>
       <li>
        <p>
         A hyperlink on author name or publication title in the detailed search results indicates standardized format.
        </p>
        <br/>
       </li>
      </ul>
