             <h1>
              Student Online Assessment Resource
             </h1>
             <h3>
              <u>
               Enduring Understanding
              </u>
              :
              <br/>
              Effective instructional  practice focuses on the use of assessment data.
             </h3>
             <h3>
              <u>
               Essential question
              </u>
              :
              <br/>
              How does SOAR support educators  as they work to: locate assessment data, analyze assessment results, make  informed instructional decisions, and focus on student achievement?
              <img align="right" alt="" height="222" name="" src="Images/teacher.png" style="background-color: #FFFFFF" width="331"/>
              <br/>
             </h3>
             <h3>
              SOAR Fundamentals:
             </h3>
             <ul>
              <li>
               Everything in one location with a single login
              </li>
              <li>
               Easy to use
              </li>
              <li>
               Saves time and allows educators to focus on  instructional practice
              </li>
              <li>
               Flexibility to accommodate any assessment
              </li>
              <li>
               Immediate feedback
              </li>
              <li>
               Flexibility to create various student groups
              </li>
              <li>
               Customized reports
              </li>
              <li>
               Constant upgrades/improvements
              </li>
             </ul>
             <h3>
              SOAR Modules (present):
             </h3>
             <ul>
              <li>
               Classroom Level Assessment Displays (group and  individual charts)
              </li>
              <li>
               Administration   Level Assessment Displays (sorting and district/school groups)
              </li>
              <li>
               Score entry with immediate feedback in displays
              </li>
              <li>
               Student Learning Plans (includes Title/LAP/ELL  documentation)
              </li>
             </ul>
             <h3>
              Modules (under development)
             </h3>
             <ul>
              <li>
               Student/Family Access (upcoming module)
              </li>
              <li>
               Graduation Requirements (upcoming module)
              </li>
              <li>
               Teacher Professional Development Log and  Professional Growth Planning (upcoming module)
              </li>
              <li>
               School Improvement Plan Integration (upcoming  module)
              </li>
              <li>
               IEP/Early Learning Documentation Integration  (upcoming module)
              </li>
              <li>
               Always more to come…
              </li>
             </ul>
             <p>
              If  you are interested in learning more about this system for your district please  contact the Burlington-Edison Technology Department at 360-757-3344 or e-mail
              <a href="mailto:soar@be.wednet.edu">
               soar@be.wednet.edu
              </a>
