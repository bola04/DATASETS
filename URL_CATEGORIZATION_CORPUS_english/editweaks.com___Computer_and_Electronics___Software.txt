 			sensitivity: 7,
 			interval: 100,

 		};
 		// override configuration options with user supplied object

 		// pX, pY = previous X and Y position of mouse, set by mouseover and polling interval
 		var cX, cY, pX, pY;
 
 		// A private function for getting mouse position
 		var track = function(ev) {

 			cY = ev.pageY;
 		};

 			}

                 <div id="summary4656255795793978237">
                  Looking at the latest features that has been introduced on Instagram story, you probably know you can add stickers to your stories.
                  <br/>
                  These stickers are interactive but, like real stickers, they don't move ... until now. In a Jan 23 update to the app, more action can be added to your stories with animated GIF stickers.
                  <br/>
                  <br/>
                  <br/>
                  <div class="separator" style="clear: both; text-align: center;">
                   <a href="https://2.bp.blogspot.com/-qgxdDabi3YI/WyKqn9xVNHI/AAAAAAAAIGI/Gmo9L3ztZ8EIP8oqicvyXebnOpHkShjSACLcBGAs/s1600/instagram-101-2.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;">
                    <img border="0" data-original-height="1600" data-original-width="900" height="320" src="https://2.bp.blogspot.com/-qgxdDabi3YI/WyKqn9xVNHI/AAAAAAAAIGI/Gmo9L3ztZ8EIP8oqicvyXebnOpHkShjSACLcBGAs/s320/instagram-101-2.jpg" width="180"/>
                   </a>
                   <a href="https://1.bp.blogspot.com/-DK8x7gQb0Yw/WyKqYePpVuI/AAAAAAAAIGE/a5qjb9PmwYMH4oCaUL5-H9FjEuvT97RAQCLcBGAs/s1600/instagram-101.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                    <img border="0" data-original-height="1600" data-original-width="900" height="320" src="https://1.bp.blogspot.com/-DK8x7gQb0Yw/WyKqYePpVuI/AAAAAAAAIGE/a5qjb9PmwYMH4oCaUL5-H9FjEuvT97RAQCLcBGAs/s320/instagram-101.jpg" width="180"/>
                   </a>
                  </div>
                  <br/>
                  <br/>
                  <div class="separator" style="clear: both; text-align: center;">
                   <br/>
                  </div>
                  <br/>
                  Instagram teamed up with GIPHY to provide a high-quality, transparent collection of GIFs to all Instagram users.
                  <br/>
                  While the photo app cut ties with the service due to a controversial, racist GIF, GIPHY support is currently reinstated. To sift through such a large volume of GIFs, Instagram built a search feature into GIF stickers, which gives users access to countless options that match whatever location, mood, or event they want to capture. Snapchat's going to be playing catch up on this one — GIF stickers are pure, goofy fun.
                  <br/>
                  <br/>
                  <h3>
                   <b>
                    Searching For &amp; Adding Live Stickers to Stories
                   </b>
                  </h3>
                  <div class="separator" style="clear: both; text-align: center;">
                   <a href="https://4.bp.blogspot.com/--i2RQPd24zg/WyKqpqMR4DI/AAAAAAAAIGQ/_ouZ4SLjvRcDAP8L3wf9k8PbZ16P3vB9gCLcBGAs/s1600/giphy2.gif" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                    <img border="0" data-original-height="480" data-original-width="270" height="320" src="https://4.bp.blogspot.com/--i2RQPd24zg/WyKqpqMR4DI/AAAAAAAAIGQ/_ouZ4SLjvRcDAP8L3wf9k8PbZ16P3vB9gCLcBGAs/s320/giphy2.gif" width="180"/>
                   </a>
                  </div>
                  <br/>
                  <i>
                   Adding GIFs to stories sounds more complicated than adding stickers, but it's pretty much the same process.
                  </i>
                  <br/>
                  <br/>
                  To get started, swipe right or tap your icon under "Stories" to access the story camera as you normally would. Once you take or choose a photo or video, you just tap the smiley-sticker icon in the top-right corner of the display. Among the stickers, you should find the GIF search option on the second sticker page.
                  <br/>
                  <br/>
                  You can simply browse the GIF stickers on the screen or you can search for them by name in the handy "Search" field. Just tap there, enter your query, then tap the GIF you want. You can place and resize the GIF on your story like you would any other sticker.

                <div class="post-header-line-1">

                   Google Photos is now available as a Progressive Web App (PWA) on Android devices, so you no longer need to download and install the application on your Android smartphone or tablet before using it. The PWA is also available on desktop through the Google Chrome browser.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://3.bp.blogspot.com/-Cpc0dil0ua8/WxXxVzPSfoI/AAAAAAAADT0/70-DB3dNrkY2YQPkQsvv97BDTmd8cUCAACPcBGAYYCw/s1600/Google%2BPhotos%2BDesktop%2BPWA.PNG" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="768" data-original-width="1366" height="179" src="https://3.bp.blogspot.com/-Cpc0dil0ua8/WxXxVzPSfoI/AAAAAAAADT0/70-DB3dNrkY2YQPkQsvv97BDTmd8cUCAACPcBGAYYCw/s320/Google%2BPhotos%2BDesktop%2BPWA.PNG" width="320"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   A Progressive Web App lives in your browser and allows you to enjoy the features of a native application without actually installing it on your device, which is a huge plus for users of smartphones with low storage capacities.
                   <br/>
                   <br/>
                   With this latest development, Google Photos joins other apps like Twitter and Google Maps Go in the growing list of native applications that have PWA versions. PWAs don't have the same adoption rates as native applications yet, but with big players in the smartphone and app industry including support for them by the day, PWAs just might overtake native apps someday, especially on mobile devices.
                   <br/>
                   <br/>
                   For now, though, PWAs are still limited in their applications, and they don't offer the full range of functionalities provided by their native counterparts. Google Photos PWA currently doesn't have an offline mode, so you can only use it when you're connected to the internet. You also don't get push notifications yet on Android, but the notifications are available on the desktop version. Other features unavailable on the PWA right now are automatic photo uploads and
                   <a href="https://paul.kinlan.me/web-share-target-api/" target="_blank">
                    Share Target
                   </a>
                   support.
                   <br/>
                   <br/>
                   <h2 style="text-align: left;">
                    How to get Google Photos PWA on Android &amp; Desktop
                   </h2>
                   <div>
                    For both platforms (Android and Desktop), you have to be using the latest Google Chrome browser in order to use Progressive Web Apps. Simply update the browser to version 67 and above, and you're good to go.
                   </div>

                   </h3>
                   <div>
                    <ol style="text-align: left;">
                     <li>
                      Visit
                      <a href="http://photos.google.com/">
                       photos.google.com
                      </a>
                      in the Google Chrome browser.
                     </li>
                     <li>
                      Click on the three dots at the top right corner of the browser.
                      <br/>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="https://2.bp.blogspot.com/-QiME_LVyToc/WxXxUXeRPWI/AAAAAAAADTw/Mn42B_W0mtczFWbDoHRWUDcVZAA-bjQbwCEwYBhgL/s1600/Add%2BGoogle%2BPhotos%2BPWA%2Bto%2BHome%2Bscreen.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" data-original-height="1600" data-original-width="900" height="320" src="https://2.bp.blogspot.com/-QiME_LVyToc/WxXxUXeRPWI/AAAAAAAADTw/Mn42B_W0mtczFWbDoHRWUDcVZAA-bjQbwCEwYBhgL/s320/Add%2BGoogle%2BPhotos%2BPWA%2Bto%2BHome%2Bscreen.png" width="180"/>
                       </a>
                      </div>

                     <li>
                      Select "
                      <b>
                       Add to Home screen
                      </b>
                      ".
                      <br/>
                      <br/>

                       </a>
                      </div>
                     </li>
                     <li>
                      Click on "
                      <b>
                       ADD
                      </b>
                      " on the dialog that appears, and wait for a few seconds for the Google Photos PWA to be added to your home screen.

                       <a href="https://3.bp.blogspot.com/-cNlJ7Oa7SCU/WxXyLiLkW-I/AAAAAAAADUA/EWAqTbdDE5M13_494ihbe0IZxFyw9Mo3QCEwYBhgL/s1600/Installing%2BGoogle%2BPhotos%2BDesktop%2BPWA.PNG" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">

                       </a>
                      </div>

                <h3 class="post-title entry-title" itemprop="name">
                 <a href="https://www.editweaks.com/2018/06/how-to-get-iphone-x-like-gesture-android.html">
                  How to get iPhone X-like gesture navigation on any Android smartphone
                 </a>
                </h3>
                <div class="post-header-line-1">
                </div>
                <div class="postmeta-primary">
                 <span class="meta_categories">
                  <a href="https://www.editweaks.com/search/label/How%20To" rel="tag">
                   How To
                  </a>
                 </span>
                 <span class="meta_comments">
                  <a href="https://www.editweaks.com/2018/06/how-to-get-iphone-x-like-gesture-android.html#comment-form" onclick="">
                   No comments
                  </a>
                 </span>
                 <span class="item-control blog-admin pid-1596312082">
                  <a href="https://www.blogger.com/post-edit.g?blogID=2166018400461921071&amp;postID=7781475033522665719&amp;from=pencil" title="Edit Post">
                   <img alt="" class="icon-action" height="18" src="//img2.blogblog.com/img/icon18_edit_allbkg.gif" width="18"/>
                  </a>
                 </span>
                </div>
                <div class="post-body entry-content" id="post-body-7781475033522665719" itemprop="articleBody">
                 <div id="summary7781475033522665719">
                  <div dir="ltr" style="text-align: left;" trbidi="on">
                   Gesture navigation is not a new thing on smartphones, Android and iPhones alike, but with the iPhone X came native support and custom gesture navigations. Google is just adding native support for gestures on the upcoming Android P which currently runs on a very small percentage of smartphones, and you have to either be using a custom ROM pre-loaded with the feature or install a third-party application to enable gesture navigation on your device. One of such apps, probably the best of them all, is
                   <b>
                    Navigation Gestures
                   </b>
                   by XDA Developers.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://4.bp.blogspot.com/-bsDLS4KpOi4/WxDmp9fa2aI/AAAAAAAAC4c/IRUUAliEf4EpPUVUwObOC2hP2fI4E6SkQCEwYBhgL/s1600/XDA%2BNavigation%2BGestures.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="1600" data-original-width="800" height="320" src="https://4.bp.blogspot.com/-bsDLS4KpOi4/WxDmp9fa2aI/AAAAAAAAC4c/IRUUAliEf4EpPUVUwObOC2hP2fI4E6SkQCEwYBhgL/s320/XDA%2BNavigation%2BGestures.png" width="160"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   Navigation Gestures does everything almost every other gesture-control app will do for you, but it is unique in that it is the only gesture control app that allows you to completely hide your phone's default navigation bar without requiring root access. If you're not a root user, you'd have to use the navigation bar in conjunction with gestures when using other apps, which doesn't seem so exciting.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://2.bp.blogspot.com/-rxXTzoBduRM/WxDmqUBgrgI/AAAAAAAAC4U/FNXwc05k9Mc1xwn3p5K7M8dMGcRO6UHsgCEwYBhgL/s1600/XDA%2BNavigation%2BGestures%2BActions.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="1600" data-original-width="827" height="320" src="https://2.bp.blogspot.com/-rxXTzoBduRM/WxDmqUBgrgI/AAAAAAAAC4U/FNXwc05k9Mc1xwn3p5K7M8dMGcRO6UHsgCEwYBhgL/s320/XDA%2BNavigation%2BGestures%2BActions.png" width="165"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   Another interesting thing about Navigation Gestures is that it supports a wide range of Android devices, indeed almost any Android smartphone running Android 8 Oreo down to JellyBean (4.2). It, however, doesn't support Android P and higher, which shouldn't be a cause for concern since native support for gestures is being added to Android from that version upwards anyway.
                   <br/>
                   <br/>
                   <h2 style="text-align: left;">
                    How to get gesture control on any Android smartphone
                   </h2>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://3.bp.blogspot.com/-A8ke66_27l4/WxDmqP-UTtI/AAAAAAAAC4c/3-oVxpV3SE0eE3UWQD8Bqr05LvzhuJ1HwCEwYBhgL/s1600/Navigation%2BGestures.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="1600" data-original-width="822" height="320" src="https://3.bp.blogspot.com/-A8ke66_27l4/WxDmqP-UTtI/AAAAAAAAC4c/3-oVxpV3SE0eE3UWQD8Bqr05LvzhuJ1HwCEwYBhgL/s320/Navigation%2BGestures.png" width="164"/>
                    </a>
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    All you need to do is to head to the
                    <a href="https://play.google.com/store/apps/details?id=com.xda.nobar" target="_blank">
                     Google Play Store
                    </a>
                    and install the Navigation Gestures application. Most gesture controls are enabled by default on the app, but to get some additional gestures for actions like pulling down the notification shade, pulling down quick settings, opening the power menu and media controls, you'd have to install a premium addon for just
                    <b>
                     $0.99
                    </b>
                    , also from the
                    <a href="https://play.google.com/store/apps/details?id=com.xda.nobar.premium" target="_blank">
                     Play Store
                    </a>
                    .
                   </div>

                  <div dir="ltr" style="text-align: left;" trbidi="on">
                   The Nokia 3.1 is the successor of 2017's Nokia 3 smartphone, and it comes with beefed-up hardware and the latest software version. It was launched alongside the successors of the other high-end and budget Nokia smartphones launched in 2017, the
                   <a href="http://www.editweaks.com/2018/05/nokia-21-review-quick-specs-and-price.html">
                    Nokia 2.1
                   </a>
                   and Nokia 5.1.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://4.bp.blogspot.com/-UAXYnayDXHs/Ww3iUJqSAAI/AAAAAAAACxY/pwlDmap00gwm8i31CmeILZTuouc7SHIPgCEwYBhgL/s1600/Nokia%2B3.1.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img alt="Nokia 3.1 " border="0" data-original-height="884" data-original-width="1024" height="345" src="https://4.bp.blogspot.com/-UAXYnayDXHs/Ww3iUJqSAAI/AAAAAAAACxY/pwlDmap00gwm8i31CmeILZTuouc7SHIPgCEwYBhgL/s400/Nokia%2B3.1.jpg" title="Nokia 3.1" width="400"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   The Nokia 3.1 is slightly larger than its predecessor, comes with a faster processor with double the number of cores, double the memory, better cameras, a stronger battery and the latest software updates. Considering that the Nokia 3 was the company's most successful smartphone in 2017, the number of upgrades on the Nokia 3.1 aren't surprising.
                   <br/>
                   <br/>
                   <h2 style="text-align: left;">
                    Nokia 3 Specifications Review
                   </h2>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://1.bp.blogspot.com/-UQv691WYONM/Ww3iT7U5IdI/AAAAAAAACxU/wrzb0rVQgZQO6-pxihiUL6leLxpYsUBlQCEwYBhgL/s1600/Nokia%2B3.1%2BSpecs.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img alt="Nokia 3.1 Specs" border="0" data-original-height="1024" data-original-width="966" height="400" src="https://1.bp.blogspot.com/-UQv691WYONM/Ww3iT7U5IdI/AAAAAAAACxU/wrzb0rVQgZQO6-pxihiUL6leLxpYsUBlQCEwYBhgL/s400/Nokia%2B3.1%2BSpecs.jpg" title="Nokia 3.1 Specs" width="376"/>
                    </a>
                   </div>
                   <div>
                    <br/>
                   </div>
                   <h3 style="text-align: left;">
                    Design &amp; Display
                   </h3>
                   The Nokia 3.1 comes with a 5.2-inch IPS LCD display which has an HD+ (720 * 1,440) pixel resolution. It's covered with 2.5D curved glass and protected with Corning Gorilla Glass 3, like the Nokia 3.
                   <br/>
                   <br/>
                   The smartphone keeps the looks of the Nokia 3, featuring a polycarbonate back and metallic (aluminium) frame. It has a thickness of 8.7 millimetres and weighs just 138.3 grammes.
                   <br/>
                   <h3 style="text-align: left;">
                    Storage &amp; Performance
                   </h3>
                   The Nokia 3.1 comes in two memory variants, the first with 2GB of RAM and 16GB of internal memory, and the second with 3GB of RAM and 32GB of internal memory. Both can take an external microSD card with a size of up to 128GB to complement the default storage.
                   <br/>
                   <br/>
                   The smartphone is powered by an octa-core 1.5GHz MediaTek MT6750N processor which promises a 50% increase in performance over its predecessor. Graphics rendering will be better too, thanks to an improved Mali T860 GPU.
                   <br/>
                   <h3 style="text-align: left;">
                    Battery &amp; Camera
                   </h3>
                   The Nokia 3.1 packs a 2,990mAh battery, an upgrade from the 2,600mAh unit powering the Nokia 3. It comes with a 13MP camera on the rear (with a LED flash) and an 8MP camera with 86.4-degree wide-angle lens on the front for taking selfies.
                   <br/>
                   <h3 style="text-align: left;">
                    Connectivity &amp; Software
                   </h3>
                   The Nokia 3.1 comes with a dual SIM slot and supports 4G LTE connections, including those with Voice-over-LTE (VoLTE) functionality. It runs on Android 8.0 Oreo and will be upgraded to Android 9.0 P when the final build is released later in the year.
                   <br/>
                   <h2 style="text-align: left;">
                    Nokia 3.1 Price and Availability
                   </h2>
                   The Nokia 3.1 comes in Blue with Copper, Black with Chrome, and White with Iron colour options and will be available for sale starting from June. The 2GB RAM version costs
                   <b>
                    $160 (N58,000)
                   </b>
                   while the 3GB RAM version costs
                   <b>
                    $195 (N72,000
                   </b>
                   ).

                   The Nokia 2.1 is the 2018 version of the Nokia 2, one of the new Android smartphones launched by HMD Global in the last year. The Nokia 2.1 was launched alongside the Nokia 3.1 and Nokia 5.1, also refreshes of the Nokia lineup in 2017.
                   <br/>
                   <br/>
                   Some of the specifications are similar to those of the Nokia 2, but there are upgrades in some departments, especially in the hardware. The phone comes with a bigger display and a more powerful processor, and everything is topped with the latest software and promises of an update to the next version of Android.
                   <br/>
                   <h2 style="text-align: left;">
                    Nokia 2.1 Review
                   </h2>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://4.bp.blogspot.com/-rz8IfxzbrYw/Ww2tzRLzomI/AAAAAAAACw8/DSt5Uz7BN_8oKVbG6OuwkRLdze5EO4_vgCLcBGAs/s1600/Nokia%2B2.1.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">

                   <h3 style="text-align: left;">
                    Design &amp; Display
                   </h3>
                   <div>
                    The Nokia 2.1 comes with a 5.5-inch display which has an HD (720 * 1,280) pixel resolution and anti-fingerprint coating. This is an upgrade from the 5-inch display present on the Nokia 2, although it appears that the Corning Gorilla Glass 3 protection has been sacrificed in the process.
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    In terms of design, the Nokia 2.1 looks pretty much like its predecessor. It has a polycarbonate back with a metallic frame, but the speaker count has been increased to two, both front-facing. The phone is still lacking a fingerprint sensor.
                   </div>
                   <h3 style="text-align: left;">
                    Storage &amp; Performance
                   </h3>
                   <div>
                    The Nokia 2.1 is powered by a quad-core Snapdragon 425 Mobile Platform processor with a clock speed of 1.4GHz and an Adreno 308 GPU handling its graphics. The RAM size remains 1GB, and there's still 8GB of internal storage which can be expanded up to 128GB via microSD.
                   </div>
                   <h3>
                    Battery &amp; Camera
                   </h3>
                   <div>
                    Not much has changed in these sections; the smartphone sports an 8MP camera with autofocus and a LED flash on the rear, while a 5MP camera handles selfie photography at the front.
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    The Nokia 2.1 packs a 4,000mAh battery, which is a little downgrade from the 4,100mAh unit powering the Nokia 2. There's no sign of fast-charging yet, though.
                   </div>
                   <h3 style="text-align: left;">
                    Connectivity &amp; Software
                   </h3>
                   <div>
                    The Nokia 2.1 comes with a dual SIM slot and supports 4G VoLTE network connectivity. It runs on Android 8.1 Oreo (Android Go edition) out of the box and will be upgraded to Android P (Android 9.0) when the final build is released by Google later this year.

                  Google Assistant Screen Search not working? Here's a fix

                  <b>
                   Here's A Quick Fix to Google Assistant Screen Search not working -
                  </b>
                  Google Assistant has many uses related to searching for and finding out information, one of which is to find things on the internet related to what is currently displayed on your phone screen. This feature, called Google Assistant Screen Search, was first introduced as
                  <i>
                   Now on Tap
                  </i>
                  in the days of Google Now. The feature hasn't been working properly of recent, with many users (including me) getting a
                  <i>
                   Nothing found on screen
                  </i>
                  result. In this article, I'll show you how to use the Google Assistant Screen Search despite this bug using a workaround.
                  <br/>
                  <br/>
                  To use the Google Assistant Screen Search feature, you have to start Google Assistant (by long-pressing your phone's home button or manually launching the application) and tapping the
                  <i>
                   What's on my screen?
                  </i>
                  button. However, when I tap the button immediately after launching Google Assistant, I get the following result:
                  <br/>
                  <br/>
                  <div class="separator" style="clear: both; text-align: center;">
                   <a href="https://2.bp.blogspot.com/-ddQfNcqZn6c/Ww2NQ8C6fWI/AAAAAAAACwk/lic1DaBsH907wpAAgeeVXD7sOdsPWnDngCPcBGAYYCw/s1600/videotogif_2018.05.29_18.21.47.gif" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                    <img border="0" data-original-height="450" data-original-width="253" height="320" src="https://2.bp.blogspot.com/-ddQfNcqZn6c/Ww2NQ8C6fWI/AAAAAAAACwk/lic1DaBsH907wpAAgeeVXD7sOdsPWnDngCPcBGAYYCw/s320/videotogif_2018.05.29_18.21.47.gif" width="179"/>
                   </a>
                  </div>
                  <br/>
                  <br/>
                  This is not a one-time glitch. It happens over and over again and is probably a bug in the latest update to the Assistant app. There's currently no
                  <i>
                   fix
                  </i>
                  for it, but there's a workaround, and it's not by clearing the cache and data for your Google app, uninstalling and reinstalling the Assistant application, or installing the beta version of the app. It takes a simple thing: patience.
                  <br/>
                  <br/>
                  When you launch Assistant, wait for around 3 seconds before tapping the
                  <i>
                   What's on my screen
                  </i>
                  button. If this doesn't work for you, wait until the four dots at the bottom of the screen turn into a microphone before tapping the phone, and the feature should work properly.
                  <br/>
                  <br/>
                  <div class="separator" style="clear: both; text-align: center;">
                   <a href="https://4.bp.blogspot.com/-CVgTzywKs0A/Ww2NP-bUeTI/AAAAAAAACwg/63Zcq28B-vQmhrKu7IISPC5csEhqnP0zQCPcBGAYYCw/s1600/videotogif_2018.05.29_18.23.51.gif" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">

                   </a>
                  </div>
                  <br/>
                  <br/>
                  We hope Google will fix this soon in an update to the Assistant application. Till then, you're welcome.

                  How to Send message to Contacts on WhatsApp without saving their number
                 </a>
                </h3>
                <div class="post-header-line-1">
                </div>
                <div class="postmeta-primary">
                 <span class="meta_categories">
                  <a href="https://www.editweaks.com/search/label/How%20To" rel="tag">

                 <span class="item-control blog-admin pid-1596312082">
                  <a href="https://www.blogger.com/post-edit.g?blogID=2166018400461921071&amp;postID=7527149474100797888&amp;from=pencil" title="Edit Post">
                   <img alt="" class="icon-action" height="18" src="//img2.blogblog.com/img/icon18_edit_allbkg.gif" width="18"/>

                <div class="post-body entry-content" id="post-body-7527149474100797888" itemprop="articleBody">
                 <div id="summary7527149474100797888">
                  <div dir="ltr" style="text-align: left;" trbidi="on">
                   Looking for a way to send message(s) to a phone number without having it saved as contact in your WhatsApp? Here's how, works for both Android and iOS devices.
                   <br/>
                   <br/>
                   From inception till now, WhatsApp has been a peer-to-peer messaging platform, which means that you have to personally know someone, or at least have their primary phone contact before you can message them. In fact, you have to save the contact of the individual you wish to message via WhatsApp on your phone, with or without a name...or do you?
                   <br/>
                   <br/>
                   In this article, I'll show you how to send messages to a contact on WhatsApp without saving their number.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://2.bp.blogspot.com/-MEQVxzFo2g8/Ww17oLh9bII/AAAAAAAACwI/f7RfwfXo9tUH50kVagfbzUfjf7258HjYwCEwYBhgL/s1600/WhatsApp%2BMessenger.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="431" data-original-width="768" height="358" src="https://2.bp.blogspot.com/-MEQVxzFo2g8/Ww17oLh9bII/AAAAAAAACwI/f7RfwfXo9tUH50kVagfbzUfjf7258HjYwCEwYBhgL/s640/WhatsApp%2BMessenger.jpg" width="640"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   Without much ado, there are two ways to send messages to an unsaved phone number on WhatsApp, and one is easier than the other (takes fewer steps). These are explained below:
                   <br/>
                   <h2 style="text-align: left;">
                    Method 1
                   </h2>
                   <h3 style="text-align: left;">
                    Unsaved number with one or more groups in common
                   </h3>
                   <div>
                    If the contact you want to send a message to without saving is on the same WhatsApp group chat with you, you can message them easily. Simply open the WhatsApp group, find any message sent by that contact to the group, and click on the contact's number displayed on top of the message.
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    In the menu that appears, select
                    <i>
                     Message [contact's number]
                    </i>
                    and a new chat will be opened for you and that contact. You can then proceed to send messages to the contact without saving their phone number.
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    Another way you can do this, if you can't readily find a message sent by the contact to the group (or don't want to), is to open the group info page (from the overflow menu or by clicking the group name at the top of the group chat) and scroll through the list of members until you find the particular contact's number.
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    Then click on the number and select
                    <i>
                     Message [contact's number]
                    </i>
                    and a new chat will be opened for you and the contact.
                   </div>
                   <h2 style="text-align: left;">
                    Method 2
                   </h2>
                   <h3 style="text-align: left;">
                    WhatsApp Click to Chat
                   </h3>
                   <div>
                    WhatsApp has a feature called Click to Chat with which you can send messages to numbers not saved in your phone's address book directly. To do this, you'll need to create a special URL with the contact's phone number in international format. When the URL is clicked or visited in a web browser, a new chat will be opened between you and the contact on WhatsApp (on your phone or on WhatsApp Web).
                   </div>
                   <div>
                    <br/>
                   </div>
                   <div>
                    To create the special link, add the contact's phone number at the end of this link
                    <code>
                     https://api.whatsapp.com/send?phone=
                    </code>
                    . To get the international format of a phone number, remove the 0 at the beginning and add the country code without the + (and the leading zeros if the country code starts with zeros).
                    <br/>
                    <br/>
                    <div class="separator" style="clear: both; text-align: center;">
                     <a href="https://2.bp.blogspot.com/-fOzg5laDLkw/Ww17ht_QNNI/AAAAAAAACwE/P1mdomcEqTQXWQpcsxkJCmakM_2eV7rhACPcBGAYYCw/s1600/PicsArt_05-29-05.10.19.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                      <img border="0" data-original-height="1600" data-original-width="900" height="400" src="https://2.bp.blogspot.com/-fOzg5laDLkw/Ww17ht_QNNI/AAAAAAAACwE/P1mdomcEqTQXWQpcsxkJCmakM_2eV7rhACPcBGAYYCw/s400/PicsArt_05-29-05.10.19.jpg" width="225"/>
                     </a>
                    </div>
                    <br/>
                    <br/>
                    For example,
                    <br/>
                    <br/>
                    If the phone number is 08012345678 and the country code is +234, the international format for WhatsApp Click to Chat will be 2348012345678 and the link will be
                    <code>
                     https://api.whatsapp.com/send?phone=2348012345678
                    </code>
                    .
                    <br/>
                    <br/>
                    Again, if the phone number is +001-(555)1234567, the link will be
                    <code>
                     https://api.whatsapp.com/send?phone=15551234567
                    </code>
                    .
                   </div>
                  </div>

                  List of devices with Qualcomm Quick Charge 4 and 4+ Fast Charging
                 </a>
                </h3>
                <div class="post-header-line-1">
                </div>
                <div class="postmeta-primary">
                 <span class="meta_categories">
                  <a href="https://www.editweaks.com/search/label/Android" rel="tag">

                  <a href="https://www.blogger.com/post-edit.g?blogID=2166018400461921071&amp;postID=5276851151705301222&amp;from=pencil" title="Edit Post">

                <div class="post-body entry-content" id="post-body-5276851151705301222" itemprop="articleBody">
                 <div id="summary5276851151705301222">
                  <div dir="ltr" style="text-align: left;" trbidi="on">
                   Fast charging used to be associated with the most premium of devices, but competition in the technology industry has seen this flagship feature trickle down to high-end and mid-range smartphones, though it's more likely to be in a device in the former category than the latter. There are different fast charging technologies available from different OEMs and across different devices, but there's no doubt that Qualcomm's Quick Charge is the most popular of them. If you're looking for devices that support the latest Qualcomm Quick Charge 4 /4+ fast charging tech, this article provides a list of them.
                   <br/>
                   <br/>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://2.bp.blogspot.com/-uBo5u1DMjQ0/WwyCfY7OOmI/AAAAAAAACvw/O80LoobEHBsKCSqS0ORX6b6CU2HG-GNvACEwYBhgL/s1600/Qualcomm%2BQuick%2BCharge%2B4%2BPlus.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img border="0" data-original-height="329" data-original-width="585" height="179" src="https://2.bp.blogspot.com/-uBo5u1DMjQ0/WwyCfY7OOmI/AAAAAAAACvw/O80LoobEHBsKCSqS0ORX6b6CU2HG-GNvACEwYBhgL/s320/Qualcomm%2BQuick%2BCharge%2B4%2BPlus.jpg" width="320"/>
                    </a>
                   </div>
                   <br/>
                   <br/>
                   Quick Charge 4 and 4+ are the latest versions of the quick charging tech from popular chip maker, Qualcomm, and they were launched early in 2017. But despite the fact that they've been in existence for more than a year now, the number of smartphones that support and use them are relatively low.
                   <br/>
                   <br/>
                   Many OEMs that use Qualcomm have decided to stick with Quick Charge 3.0, and that's largely due to the fact that there's not much difference between the two quick charge versions, and many OEMs are coming up with such technologies of their own for use on their devices. Oppo has VOOC, OnePlus has Dash Charge, MediaTek has Pump Express, and even Google's Pixel devices use USB Power Delivery (USB-PD).
                   <br/>
                   <br/>
                   However, some other OEMs have seen some light in the dark tunnels that are QC4 and QC4+, and have taken the plunge. These are listed below:
                   <br/>
                   <ul>
                    <li style="box-sizing: border-box; text-size-adjust: auto;">
                     BQ Aquaris X2.
                    </li>
                    <li style="box-sizing: border-box; text-size-adjust: auto;">

                  Nokia 1 Review, Quick Specs and Price in Nigeria
                 </a>
                </h3>
                <div class="post-header-line-1">
                </div>
                <div class="postmeta-primary">

                 <div id="summary8155729846830633077">
                  <div dir="ltr" style="text-align: left;" trbidi="on">
                   The Nokia 1 is on the bottom of the chart containing the new Android smartphones launched with the Nokia brand name by HMD Global since the resurgence of the brand last year, and as expected, it's a basic smartphone with low-end features and a budget price. But it's not all low-end as we are used to.
                   <br/>
                   <br/>
                   The Nokia 1 comes with Android Go, a build of the latest version of the Android operating system, Android 8.1 Oreo that's designed to run efficiently on low-end and entry-level smartphones like the Nokia One. That means you get the latest software features available in the Android ecosystem on a phone that costs less than a hundred dollars. Fair bargain, if you ask me.
                   <br/>
                   <br/>
                   <a href="https://3.bp.blogspot.com/-ys2EC4uIIlQ/Wws1p43Q9NI/AAAAAAAACis/ax3jz285xf05VNztd-ZEzYq6KNPg3ccBQCEwYBhgL/s1600/Nokia%2B1.jpg" imageanchor="1" style="clear: right; margin-bottom: 1em; margin-left: 1em;">
                    <img alt="Nokia 1" border="0" data-original-height="350" data-original-width="600" height="372" src="https://3.bp.blogspot.com/-ys2EC4uIIlQ/Wws1p43Q9NI/AAAAAAAACis/ax3jz285xf05VNztd-ZEzYq6KNPg3ccBQCEwYBhgL/s640/Nokia%2B1.jpg" title="Nokia 1" width="640"/>
                   </a>
                   <br/>
                   <br/>
                   The Nokia 1 also brings something back from the old Nokia we all knew -- Nokia Xpress-on covers that make it as easy as a case swap to change the look of your smartphones. This is made possible by the design of the device, which has the front covered in black bezels and a silver frame, and the entire back and part of the frame covered by the Xpress-on cover. That way, you can simply use an Xpress-on cover with another colour of your choice, and you get a fresh-looking device.
                   <br/>
                   <br/>
                   Other features of the phone are pretty basic, but they're not disappointing. The display is covered with a 4.5-inch IPS LCD FWVGA screen and the phone is 9.5 millimetres thick, even though the battery has a capacity of only 2,150mAh.
                   <br/>
                   <br/>
                   The Nokia 1 is powered by a 1.1GHz Quad-core MediaTek MT6737M processor paired with a Mali-T720 GPU. It has 1GB of RAM for multi-tasking and 8GB of internal storage that can be expanded up to 128GB via its external hybrid micro-SD/Nano-SIM slot.
                   <br/>
                   <br/>
                   Its 5MP main camera takes decent shots that are brightened by its accompanying LED flash, but you can't expect much from the 2MP camera on the front. For a budget smartphone, the Nokia One supports a wide range of (global) 4G LTE bands, even more than you would find on a far more expensive high-end smartphone.
                   <br/>

                    Nokia 1 Specs
                   </h2>
                   <div class="separator" style="clear: both; text-align: center;">
                    <a href="https://2.bp.blogspot.com/-ogLZCJ_M0QM/Wws1qR6k_HI/AAAAAAAACiw/w0GsekflGg4fkk1uLvtpVBVVCa9R6GiLACEwYBhgL/s1600/Nokia%2BOne.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                     <img alt="Nokia One" border="0" data-original-height="647" data-original-width="970" height="266" src="https://2.bp.blogspot.com/-ogLZCJ_M0QM/Wws1qR6k_HI/AAAAAAAACiw/w0GsekflGg4fkk1uLvtpVBVVCa9R6GiLACEwYBhgL/s400/Nokia%2BOne.jpg" title="Nokia One" width="400"/>
                    </a>
                   </div>
                   <div class="separator" style="clear: both; text-align: center;">
                    <br/>
                   </div>

                   </div>

                      4.5-inch IPS LCD display with FWVGA (480 * 854) pixel resolution.
                     </li>
                     <li>
                      Dimensions: 133.6 * 67.8 * 9.5 mm.
                     </li>
                     <li>
                      1.1GHz Quad-core MediaTek MT6737M Processor.
                     </li>

                     </li>
