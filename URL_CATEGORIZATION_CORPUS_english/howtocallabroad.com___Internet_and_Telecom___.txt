      <h2>
       Basic 
         international dialing instructions
      </h2>
      <h3>
       From the USA/Canada:
      </h3>
      <ul>
       <li>
        For calls to most countries dial the following:
        <em>
         011 
           + country code + phone number
        </em>
       </li>
       <li>
        For calls to countries within the North American 
           Numbering Plan dial:
        <em>
         1 + area code + 
           7 digit local number
        </em>
        , same as dialing a US state to state call (note that you will incur international charges)
       </li>
      </ul>
      <h3>
       From any country:
      </h3>
      <ul>
       <li>
        dial:
        <em>
         your country's exit code + destination 
           country code + phone number
        </em>
       </li>
      </ul>
      <h3>
       Handy:
      </h3>
      <ul>
       <li>
        <a href="codes.html">
         List of country codes/exit codes
        </a>
       </li>
       <li>
        <a href="nanp.html">
         Countries using the North American Numbering Plan
        </a>
       </li>
      </ul>
      <h2>
       Helpful tips for calling abroad
      </h2>
      <p>
       <b>
        Calls to cell phones
       </b>
       registered in another country 
         are often charged much higher rates than calls to fixed lines. Our 
         advice: make sure you know what type of number you are calling - this website shows you what mobile phone numbers look like in most countries.
      </p>
      <p>
       <b>
        If the international number you wish to call 
         starts with a 0 (zero)
       </b>
       , you must drop this digit when 
         dialing the number. The 0 (zero) is a so called  "trunk 
         code" used for domestic calls within many countries.
      </p>
      <p>
       <b>
        The plus sign (+)
       </b>
       used before an international telephone number can either be dialed or replaced with your country's  exit code. For example, if you are in the US and you see this UK number
       <em>
        +44 7000 000 000
       </em>
       you can dial it as is or replace the plus sign with 011.
      </p>
      <p>
       <b>
        Be aware of extra charges
       </b>
       incurred when using a service such as a calling card to place international calls. Often they advertise low rates, but will charge additional connection or maintenance fees.
      </p>
