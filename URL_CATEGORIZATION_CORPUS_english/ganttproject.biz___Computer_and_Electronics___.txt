         <div class="feature-content" id="feature-gantt-chart">
          <h3>
           Gantt chart
          </h3>
          <ul class="features">
           <li>
            <strong>
             Create tasks and milestones
            </strong>
            . Aside from the start date and duration, every task may have
             priority, cost, color and fill pattern, text notes and user-defined
             custom fields.
           </li>
           <li>
            <strong>
             Organize tasks in a work breakdown structure
            </strong>
            . Hierarchical tree where progress, dates or costs of
             lower level tasks is summarized on the higher levels. Summary tasks can be collapsed to hide tasks which are not important
             at the moment.
           </li>
           <li>
            <strong>
             Draw dependency constraints
            </strong>
            between tasks, like
            <i>
             "start X when Y finishes"
            </i>
            and GanttProject will take care
             of enforcing these constraints. You can add a lag or use other types of constraints.
           </li>
           <li>
            <strong>
             Create baselines
            </strong>
            to be able to compare current project state with previous plans..
           </li>
           <li>
            <strong>
             PERT chart
            </strong>
            for read-only view can be generated from the Gantt chart.
           </li>
          </ul>
         </div>
         <div class="feature-content" id="feature-resource-chart">
          <h3>
           Resource chart
          </h3>
          <ul class="features">
           <li>
            <strong>
             Create human resources
            </strong>
            with the basic contact information, payment rate
            <a class="label label-warning" href="/ostrava">
             New in 2.7
            </a>
            and role.
           </li>
           <li>
            <strong>
             Assign resources
            </strong>
            to work on tasks with different roles and assignment units.
           </li>
           <li>
            <strong>
             Monitor task assignments
            </strong>
            and see when some resource gets overloaded or is sitting without work.
           </li>
          </ul>
         </div>
         <div class="feature-content" id="feature-export">
          <h3>
           Export
          </h3>
          <ul class="features">
           <li>
            <strong>

            </strong>
            with a summary, required task and resource information and vector chart images.
           </li>
           <li>
            <strong>
             PNG/JPEG images
            </strong>
            can be generated from individual charts and printed
           </li>
           <li>
            <strong>
             Export to CSV
