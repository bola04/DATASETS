     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C0">
       00 Gauge Card Kits
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C1">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C2">
       Simple construction for all modellers
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C1">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       All Wordsworth kits come in a compressed format for which “jZip” or “7-Zip” are both
     good recommended program, or you can use any program that can read .RAR format.
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       Once the file is unpacked you are presented with a standard PDF which “Adobe Reader”
     is recommended, but you may wish to use another program.
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       You are now ready to print your kit, print each kit at 100% using your home printer.
     The ideal paper weight for this is 80gsm, once all the pages have been printed the
     ideal mounting media is onto 300gsm card.
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       Glue each sheet onto the card using a Jumbo Pritt stick, once again you may wish
     to use other glue.
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       <br/>
      </span>
     </h1>
     <h1 class="Wp-Heading-1-P">
      <span class="Placeholder-C-C3">
       Let the glue dry then you simply cut out each part, score along the fold lines which
     makes constructing the kits easier, hold in place with Pritt PVA glue.
      </span>
     </h1>
