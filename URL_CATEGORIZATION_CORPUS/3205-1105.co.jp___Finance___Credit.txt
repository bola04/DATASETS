     <h2 class="close mb15" id="newinfo_hdr2">
      学生ローンご利用までの流れ
     </h2>
     <dl id="newinfo2">
      <dt>
       <font color="#0000ff">
        <b>
         ●ご用意頂くもの
        </b>
       </font>
      </dt>
      <dd>
       ①学生証　②免許証か保険証　③キャッシュカード　④※公共料金の領収書
       <br/>
       ※ご来店の場合、公共料金の領収書は不要です。
       <br/>
       ※公共料金の領収書としてお取り扱い可能な書類につきましては、
       <a href="inet2.htm" target="_blank">
        こちらのページ
       </a>
       の一番下の「公共料金の領収書・または請求書一覧」をご覧下さい。
       <br/>
      </dd>
      <dt>
       <b>
        <font color="#0000ff">
         ●学生ローンご利用までの手順
        </font>
       </b>
      </dt>
      <dd>
       ①お振込みご希望の場合、
       <a href="form.htm">
        学生ローンのお申込みページ
       </a>
       よりお申込み下さい。
       <br/>
       ②当社営業時間に電話連絡を致します。(どちらから連絡するかは、お申込みページでご選択頂きます)
       <br/>
       ③電話連絡で
       <font color="#ff0000">
        審査が通った後
       </font>
       、必要書類(上記「ご用意頂くもの」)をメールで送信して下さい。
       <br/>
       必要書類の送信は、
       <a href="inet2.htm" target="_blank">
        こちらのページ
       </a>
       から送信できます。(電話連絡で審査が通った後で結構です)
       <br/>
       必要書類確認後、お客様の銀行口座にお振込みでご融資させて頂きます。
       <br/>
       ①~③が午後2時位までに完了すれば当日中のお振込みが可能です。
       <br/>
       お急ぎの方は、必要書類をご用意の上、午後1時までにお申込み下さい。
       <br/>
       (※審査により、ご希望に添えない場合もございます)
       <br/>
       <br/>
       ●後日、当社から書類を郵送させて頂きます。
       <br/>
       ご返済方法等の説明書類や、契約書が同封されておりますので、契約書を記入の上、返送して頂きます。
       <br/>
       ご家族様同居の場合などで、個人名希望の場合はお申し出下さい。
       <br/>
      </dd>

     <h2 class="close mb15" id="newinfo_hdr4">
      学生ローンと大手の違い
     </h2>
     <dl id="newinfo4">
      <dt>
       <font color="#0000ff">
        <b>
         ●学生ローンとは
        </b>
       </font>
      </dt>
      <dd>
       学生ローンは1970年代初頭、学生の街で知られる東京・高田馬場で日本で初めて誕生しました。
       <br/>
       奨学金制度のローンとよく勘違いされますが、一般のローン会社の学生さん専門とお考え頂ければ、わかりやすいかと思います。
       <br/>
       金利面などでよく比較されますが、平成28年6月24日現在の大手消費者金融の金利は、下記の通りとなっております。
       <br/>
       大手3社の金利は約18.0%、学生ローン専門店は、全国でも10社程度ですが、平均的な金利は実質年率17.0%となっており、大手ローン会社と比較してもお安くなっているのが特徴です。
       <br/>
       当社は、もちろん実質年率17.0%とさせて頂いております。
       <br/>
      </dd>
      <dt>
       <b>
        <font color="#0000ff">
         ●大手との金利比較
        </font>
       </b>
      </dt>
      <dd>
       アコム・・実質年率18.0%
       <br/>
       レイク・・実質年率18.0%
       <br/>
       プロミス・・実質年率17.80%
       <br/>
