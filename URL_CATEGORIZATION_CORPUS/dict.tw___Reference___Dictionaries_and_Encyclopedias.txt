  <ul>
   <b>
    查詢字彙:
   </b>
   <li>
    請輸入中文、日文、或英文字彙，可查詢英文、或中文解釋。
   </li>
   <li>
    輸入單一字中文可查詢注音、中文輸入法拆碼。
   </li>
  </ul>
  <div class="db_info">
   <span>
    進階查詢
   </span>
  </div>
  <ul>
   切換至顯示細目 (請按
   <small>
    【顯示細目】
   </small>
   /
   <small>
    【隱藏細目】
   </small>
   切換。)
   <br/>
   <br/>
   <b>
    查詢方式:
   </b>
   <li>
    Return Definitions - 查詢單字解釋 (預設)
   </li>
   <li>
    Match headwords exactly - 僅查詢完全符合的單字
   </li>
   <li>
    Match prefixes - 查詢字首相符的所有單字
   </li>
   <li>
    Match substring occurring anywhere in a headword - 查詢包含本字串的所有單字
   </li>
   <li>
    Match suffixes - 查詢字尾相符的所有單字
   </li>
   <li>
    POSIX 1003.2 (modern) regular expressions - 《正規表示式》查詢
   </li>
   <br/>
   <b>
    字典選擇:
   </b>
   <li>
    Any - 查詢所有資料庫 (預設)
   </li>
   <li>
    First match - 查詢第一個有收錄的資料庫
   </li>
   <li>
    ... - 選擇各別的資料庫
   </li>
  </ul>

      ^

     <td>
      必須位於檢索條件的開頭
     </td>
    </tr>
    <tr>
     <th>
      $
     </th>
     <td>
      必須位於檢索條件的結尾
     </td>
    </tr>
    <tr>
     <th>
      .
     </th>
     <td>
      在檢索條件中，表示任意一個字元
     </td>
    </tr>
    <tr>
     <th>
      *
     </th>
     <td>
      在星號前的字元可以重複出現零次以上
     </td>
    </tr>
   </tbody>
  </table>

    範例
    <br/>
    問：想要找單字，只知道 di 開頭、 ry 結尾 (也就是 di??????ry 的單字)，要如何查詢？
    <br/>
    答：使用《正規表示式》查詢方式，查詢
    <tt>
     ^di.*ry$
    </tt>
    即可。
