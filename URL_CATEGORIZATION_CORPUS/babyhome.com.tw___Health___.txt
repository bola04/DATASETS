義大利麵&amp;玉米濃湯一直以來都是我家呈寶貝的最愛，媽咪我自己本身也很喜歡香濃的白醬料理，在這次懷二寶的孕期中，自己也幾乎天天都會喝上一杯【福樂鐵、葉酸低脂牛乳】補充孕期營養，比喝媽媽奶粉還方便。


自從呈寶貝出生6個月開始吃副食品後，為了他可以吃的健康無負擔，很多食品都要自己製作比較安心，除了舉凡很常在家製作的歐姆蛋、鮮奶麵包、法式吐司、鮮奶布丁… …外，加上老公自己本身很愛喝鮮奶茶，他說福樂鈣多多系列調製奶茶最香濃，所以福樂一番鮮&amp;鈣多多系列一直以來幾乎是我們家中常備的鮮乳，這次可以幸運得到BabyHome的試喝體驗，真的是太開心了。

今天就來分享一道呈寶貝很常吃簡單又清爽的菇類義大利麵吧！ 另外這份食譜直接使用福樂鈣多多一樣可以烹調出道地可口的好滋味，味道甚至比鮮奶版本的更香濃，同時兼備美味長高高。

【培根菇菇義大利麵】


【材料】：義大利麵、洋蔥適量、蒜頭2-3瓣、杏鮑菇、鴻喜菇、雪白菇、毛豆(可改成紅蘿蔔)適量、培根少許、福樂鈣多多300cc
【調味料】：鹽1/2茶匙，黑胡椒粉1/4茶匙，起士粉少許



【作法】：
1. 毛豆和各種菇類洗淨切好後川燙後備用。

2.洋蔥、蒜頭切細末，義大利麵事先煮好備用。

3.平底鍋放入培根炒香，放入切好的洋蔥、蒜頭爆香、川燙好的菇類拌炒。

4. 加入義大利麵，煮稍微拌炒均勻即可。

5.加入福樂鈣多多混合熬煮。

6. 酌量以鹽巴、黑胡椒等調味盛盤後灑上少許起士粉，毛豆裝飾擺盤。

【補充說明】：青菜配料可以依照寶寶喜歡去做變化，麵粉使用中高低筋皆可，另外也可以利用悶燒罐煮義大利麵會更快速方便，媽咪輕鬆同時也省瓦斯開銷。


真的超級簡單又美味，輕輕鬆鬆美味可口的長高高義大利麵就可以熱騰騰上桌囉，超級推薦給家中有寶寶的媽咪們，這道零失敗的長高高料理，相信寶貝們都會愛上！

/* 延遲載入大圖 */
function lazyload_img(){
    $(document).find('img.lazy_load_img').each(function(){
        var img_obj = $(this),
            org_img_url = img_obj.attr('src'),
            big_img_url = img_obj.attr('data-src');

        if(
            typeof(org_img_url) !== 'undefined' && typeof(big_img_url) !== 'undefined' && 
            big_img_url != '' && org_img_url != big_img_url
        ){
            var img = new Image();
            img.src = big_img_url;
            img.onload = function(){
                img_obj.hide().removeClass('lazy_load_img').attr('src', big_img_url).fadeIn(400);
            };
        }
    });
}
