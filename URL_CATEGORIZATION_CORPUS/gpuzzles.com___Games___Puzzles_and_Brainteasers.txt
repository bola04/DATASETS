        <a href="https://gpuzzles.com/mind-teasers/donald-duck-math-puzzle/">
         #19 - Donald Duck Math Puzzle
        </a>
       </h1>
       <p>
        <!-- Diff Level / Pop Ques-->
        <span>
         <span class="pr_diff">
          Difficulty
          <img src="/images/diff1.gif" style="width:10px;height:10px"/>

         </span>
         <span class="pr_diff" style="margin-left:12px;">
          Popularity
          <img src="/images/pop1.gif" style="width:10px;height:10px"/>
          <img src="/images/pop1.gif" style="width:10px;height:10px"/>
          <img src="/images/pop1.gif" style="width:10px;height:10px"/>
          <img src="/images/pop0.gif" style="width:10px;height:10px;"/>
          <img src="/images/pop0.gif" style="width:10px;height:10px;"/>
         </span>
        </span>
        <!-- Diff Level / Pop Ques-->
       </p>
       <div class="mt15 lr-fn">
        <!-- Puzzle -->
        Donald duck can can swim his pool downstream(with the pool current helping time) in exact 40 seconds and upstream(against the pool current) in a pool in exact 60 seconds. The length of pool is 2 kilometers.
        <br/>
        <br/>
        How long Donald duck can cover distance of one side at a still pool (with no current).
        <!-- Puzzle -->
        <div class="mt15">
         <a class="lr-answer lnkcol" href="javascript: void 0;" id="id19">
          View Answer
         </a>
         <a class="lr-discuss lnkcol" href="https://gpuzzles.com/mind-teasers/donald-duck-math-puzzle/" target="_blank">
          Discuss
         </a>
        </div>
        <!-- Soln -->
        <div class="lr-bix-div-answer" id="ansid19" style="display:none;">
         <div class="lr-title-bar" id="sid19" style="cursor: pointer;">
          <a>
           Answer &amp; Explanation
          </a>
         </div>
         <div class="lr-div-spacer">
          <p>
           <span class="lr-ib-green">
            <b>
             Solution:
            </b>
           </span>
          </p>
          <p style="color:black;">
           48 minutes.
           <br/>
           <br/>
           Donald duck's speed = X km/seconds
           <br/>
           pool current speed  = Y km/seconds
           <br/>
           <br/>
           2/(X+Y) = 40
           <br/>
           2/(X-Y) = 60
           <br/>
           <br/>
           Solving the simultaneous equations gives X = 1/24
           <br/>
           <br/>
           Therefore, to cover 2km will take 2/X = 48 seconds
          </p>
