        <h2>
         Популярная запись в блогах
        </h2>
        <p>
         <strong>
          Использование динамического SQL - легко и понятно (часть первая EXECUTE IMMEDIATE)
         </strong>
         <br/>
         <a href="/blogs/oracleandsql">
          Oracle SQL
         </a>
        </p>
        <a href="http://landingsql.orasource.ru/" target="_blank">
         Учебный тренинг. SQL от новичка до профессионала.
        </a>
        <br/>
        <a href="https://www.youtube.com/channel/UCCMANdkq1T47prGmK9-A3fA/videos" target="_blank">
         Бесплатные авторские видеокурсы SQL, PLSQL, JAVA
        </a>
        <br/>
        <br/>
        <br/>
        <font size="1">
         Динамический SQL. Часть первая
        </font>
        <br/>
        В некоторых задачах требуется использование динамических запросов. Это случается , когда мы заранее не знаем какой нам нужен SQL запрос, какие обьекты будут использоваться для получения данных.
        <br/>
        <br/>
