                        multe detalii despre plasarea, utilizarea, administrarea si
                        dezactivarea “cookie”-urilor utilizate de Site-ul presto-pizza.ro "Site".
       <br/>
       In cazul in care aveti nevoie de mai multe informatii, si ele nu se regasesc mai jos, ne puteti

       <br/>

       <br/>
       <br/>
       <b>
        2. Scopul utilizarii cookie-urilor de catre Site
       </b>
       <br/>
       <br/>
       Acest Site foloseste cookie-uri din urmatoarele considerente:
       <br/>
       • pentru a furniza vizitatorilor o experienta mult mai buna de navigare si servicii adaptate
                        nevoilor si interesului fiecaruia.
       <br/>
       • pentru a genera statistici despre traficul pe site
       <br/>
       <br/>
       <b>
        3. Definirea “cookie-urilor de pe Site.
       </b>
       <br/>
       <br/>
       Pe site se folosesc cookie-uri HTTPS. Un cookie HTTPS sau un modul cookie este un text special,
                        deseori codificat, trimis de un web-server unui web browser și
                        apoi trimis înapoi (nemodificat) de către web browser, de fiecare dată când accesează acel
                        server. Un site web care foloseste HTTPS are cookies care impiedica
                        transmiterea lor catre o pagina non-HTTPS chiar daca aceasta este localizata la acelasi URL. Un
                        cookie HTTPS nu este accesibil prin mijloace/ metode non http
                        (care include si HTTPS), cum ar fi Java Script neputand fi tinta unor atacuri de tip cross –site
                        scripting. Un cookie este un fisier de mici dimensiuni, format
                        din litere si numere, care va fi stocat pe computerul, terminalul mobil sau alte echipamente ale
                        unui utilizator depe care se acceseaza Site-ul.
       <br/>
       Cookie-ul este complet “pasiv” nu contine programe software, virusi sau spyware si nu poate
                        accesa informatiile de pe hard drive-ul utilizatorului.Un cookie
                        este format din doua parti: numele si continutul sau valoarea cookie-ului. Mai mult, durata de
                        existenta a unui cookie este determinata; tehnic, doar
                        webserverul care a trimis cookie-ul il poate accesa din nou in momentul in care un utilizator se
                        intoarce pe website-ul asociat webserverului respectiv.
       <br/>
       Cookie-urile de pe acest site in sine nu solicita informatii cu caracter personal pentru a putea
                        fi utilizate si nici nu asigura accesul la date cu caracter
                        personal.
       <br/>
       <br/>
       <b>
        4. Rolul cookie-urilor de pe Site
       </b>
       <br/>
       <br/>
       In cadrul Site-ul nostru, “cookie”-urile HTTPS joaca un rol important in facilitarea accesului
                        si livrarii serviciilor de care utilizatorul se bucura cum ar fi:
       <br/>
       • oferirea de feedback asupra modului cum este utilizat Site-ul, astfel incat acesta sa poata fi
                        facut si mai eficient si mai accesibil pentru utilizatori;
                        feedback-ul consta in date statistice anonimizate (fara a permite individualizarea
                        utilizatorului).
       <br/>
       • informatii despre traficul pe acest site: tipul browserului folosit, timpul petrecut pe site,
                        numarul de pagini accesate, date despre furnizorul de internet
                        al utilizatorului (nume, tara si judet/regiune).
       <br/>
       <br/>
       <b>
        5. Cum sunt folosite cookie-urile de catre acest Site?
       </b>
       <br/>
       <br/>
       O vizita pe acest site (https://presto-pizza.ro) poate plasa urmatoarele tipuri cookie-uri:
       <br/>
       • Cookie-uri de analiza (_ga)
       <br/>
       • Cookie-uri de sesiune (PHPSESSID)
       <br/>
       <br/>
       <b>
        Cookie-uri de analiza
       </b>
       <br/>
       Site-ul “https://presto-pizza.ro" implementeaza o solutie Google Analytics pentru a colecta
                        informatii despre traficul pe site. De fiecare data cand un
                        utilizator viziteaza acest site, soft-ul de Google Analytics genereaza cookie-uri de analiza.
                        Browserul ne va spune daca aveti acest cookie-uri, iar daca nu,
                        vor fi generate. Aceste cookie-uri nu sunt folosite de site-ul https://presto-pizza.ro pentru a
                        identifica persoanele fizice, ele sunt folosite doar in scop
                        statistic.
       <br/>
       Pe baza acestor cookie-uri se pot colecta urmatoarele informatii:
       <br/>
       • Tipul de browser folosit si informatii cu privire la furnizorul de internet (ISP) al
                        utilizatorului precum denumire, tara si judet. Nu se colecteaza /
                        acceseaza date cu privire la adresa IP a terminalului utilizatorului.
       <br/>
       • Performanta site-ului
       <br/>
       • Ce pagini au fost accesate
       <br/>
       • Timpul petrecut pe o pagina sau pe mai multe pagini
       <br/>
       Mai multe informatii despre Google Analytics puteti sa gasiti aici.
       <br/>
       <br/>
       Cookie-ul de analiza se numeste ”_ga”. Cookie-ul _ga este creat si folosit de libraria
                        JavaScript a Google Analytics. La prima accesare a website-ului, libraria
                        Javascript Google Analytics creeaza acest cookie cu valoare random unica care reprezinta client
                        ID. Acest Client ID este folosit ulterior de catre serverul
                        Google Analytics pentru a identifica un utilizator unic.
       <br/>
       <br/>
       <b>
        Cookie-uri de sesiune
       </b>
       <br/>
       Cookie-ul de sesiune este utilizat pentru identificarea unei vizite pe site-ul
                        https://presto-pizza.ro. Acesta expira in momentul in care inchideti browserul
                        dvs. Web sau dupa 1.440 secunde de inactivitate.
       <br/>
       <br/>
       Cookie-ul de sesiune se numeste “PHPSESSID”. La orice vizita pe website Cookie-ul PHPSESSID este
                        creat de webserverul website-ului cu o valoare unica. Acest
                        cookie este folosit pentru a determina o sesiune a unui user. Spre exemplu, daca un user se
                        autentifica cu succes pe website, toate request-urile browser-ului
                        catre webserver sunt atribuite aceluiasi user cat timp cookie-ul nu este expirat sau sters. Daca
                        acest cookie se sterge sau expira, userul trebuie sa se
                        autentifice din nou. Atat cookie-urile de analiza, cat si cele de sesiune de pe Site sunt
                        plasate de domeniul https://presto-pizza.ro accesat de catre
                        utilizator, a carui adresa apare in bara de adrese a browser-ului.
