        Rocket Internet, kraamkamer van Duitse internetbedrijven
       </h1>
       <p>
        Berlijn was een belangrijk centrum voor industriële bedrijvigheid na de Duitse eenwording van 1871. In de Duitse hoofdstad zetelden grote bedrijven als Siemens, AEG, Allianz, Deutsche Bank en Lufthansa. Maar na de Tweede Wereldoorlog kwam Berlijn afgelegen te liggen. De doodsteek kwam met de Koude Oorlog. Westerse bedrijven en hun hoofdkantoren hadden niets meer te zoeken in het eilandje in het afgesloten Oost-Duitsland. Nu bloeit de stad weer op, en daarbij lopen internetbedrijven als Rocket Internet voorop.

        function BeursNewsController($http, $scope, $interval, $sce) {
        var beursNewsPageBaseUrl = "/beurs/beursnieuws";
        $scope.beursNewsPageBaseUrl = beursNewsPageBaseUrl;

        var loadNewsItems = function() {
            $http.get('/Umbraco/Api/PublicBeursNews/GetBeursNews').success(function (data) {
                if (beursNewsPageBaseUrl) {
                    angular.forEach(data, function (item) {
                        item.detailsPageUrl = beursNewsPageBaseUrl + '?pagina=1#' + item.id;
                        item.title = $sce.trustAsHtml(item.title);
                    });
                }
                $scope.BeursNewsItems = data;
            });
        };

        loadNewsItems();
        $interval(loadNewsItems, 30000);
    }
