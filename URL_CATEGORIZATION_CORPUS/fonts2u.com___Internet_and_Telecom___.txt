      <div class="titleblue">
       How to install a font
      </div>
      <div class="pad10">
       <strong>
        Unzip file(s) downloaded.
       </strong>
       <br/>
       <br/>
       Depending on the platform:
       <ul>
        <li>
         <strong>
          Windows 7 / Vista
         </strong>
         - right-click the font file and select
         <u>
          INSTALL
         </u>
        </li>
        <li>
         <strong>
          Windows XP
         </strong>
         - copy or move the font file(s) to
         <u>
          C:\WINDOWS\FONTS
         </u>
        </li>
        <li>
         <strong>
          Mac OS X
         </strong>
         - double-click the font file and select
         <u>
          INSTALL FONT
         </u>
        </li>
        <li>
         <strong>
          Linux
         </strong>
         - copy the font file(s) to
         <u>
          /USR/SHARE/FONTS
         </u>
        </li>
       </ul>
