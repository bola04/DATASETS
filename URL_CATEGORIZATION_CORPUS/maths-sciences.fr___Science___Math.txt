      <br/>
      <br/>
      <br/>
      Ce site s'adresse avant tout aux sections des lycées professionnels (troisième Prépa Pro, seconde, première, terminale pro et CAP)
      <br/>
      <br/>
      Il met en ligne des documents pour l'apprentissage des mathématiques et des sciences physiques.
Une page est dédiée aux
      <a href="/ccf.php" title="lien vers la page des sujets de CCF">
       sujets de CCF
      </a>
      .
      <br/>
      Tous les anciens documents sont encore disponibles en consultant l'onglet :
      <a href="/archives-maths-sciences.php" title="archives du site">
       les archives du site.
      </a>
      <br/>
      <br/>
      Notamment les
      <a href="/tp-sciences-bac-pro.php" title="lien vers les TP de sciences fmb">
       TP de sciences
      </a>
      des sessions antérieures à celles du bac pro 2012.
      <br/>
      <br/>
      La page des
      <a href="/calculatrice-tice.php" title="lien vers la page des calculatrices">
       calculatrices
      </a>
      (Casio et Texas instruments) permet de rassembler toutes les activités qui y sont consacrées.
      <br/>
      Certains chapitres ne font apparaître aucun document. Ils seront complétés au fur et à mesure.
      <br/>
      <br/>
      Les corrigés des exercices mis en ligne n'existent pas.
      <br/>
      <br/>
      Les mises à jour du site sont informées sur
      <a href="http://www.facebook.com/pages/maths-sciencesfr/155291951217742" target="_blank">
       facebook
      </a>
      ,
      <a href="http://fr.twitter.com/#!/maths_sciences" target="_blank">
       twitter
      </a>
      et
      <a href="https://plus.google.com/b/106023655902469999174/+mathssciences/posts" target="_blank">
       google+
      </a>
      .
      <br/>
      <br/>
      Prenez le temps, si vous le pouvez, de me signaler toute erreur ou dysfonctionnement.
      <br/>
      <br/>
      Merci.
      <br/>
      <br/>
      Luis LOPEZ
      <br/>
