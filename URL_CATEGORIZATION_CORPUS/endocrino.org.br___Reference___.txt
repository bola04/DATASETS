       <h3>
        AE&amp;M Volume 62 Nº3
       </h3>
       <p>
        A SBEM disponibiliza o volume 62, número 3, dos Archives of Endocrinology and Metabolism. Os AE&amp;M, que tem como editor-chefe o Dr. Marcello D. Bronstein, é o órgão oficial de divulgação científica da SBEM, afiliada à Associação Médica Brasileira. A revista é, também, um espaço de colaboração científica de Sociedades Médicas da área de Endocrinologia e Metabologia.
       </p>

       <h3>
        10 Coisas Sobre Tireoide
       </h3>
       <p>
        <a href="../../../../10-coisas-que-voce-precisa-saber-sobre-tireoide/">
         A glândula tireoide é localizada na parte anterior do pescoço, logo abaixo do Pomo de Adão. Reguladora da função de importantes órgãos do corpo. Quando ela não funciona de maneira correta, pode liberar hormônios em quantidade insuficiente, causando o hipotireoidismo, ou em excesso, ocasionando o hipertireoidismo.
        </a>
       </p>

    function loadEvents(index = 0) {
        var label = ["jan", "fev", "mar", "abr", "mai", "jun", "jul", "ago", "set", "out", "nov", "dez"];
        $.getJSON("/agenda/eventos.json", function( data ) {
            $.each(data, function( idx, item) {
                if (new Date() < new Date(item.ano, item.mes-1, item.dia ) && index === 0) {
                    index = idx;
                }
            });
            var items = data.slice(index, index+4);
            $('.eventos').attr('data-index', index);
            $.each(items, function( idx, item ) {
                var $evento = $(".eventos .item").slice(idx)
                $('h4', $evento).html(item.titulo);
                $('.dia', $evento).html(item.dia +'/');
                $('.mes', $evento).html(label[item.mes-1]);
                $('.infos', $evento).attr('href', 'http://beta.endocrino.org.br/' + item.ano + '/' + item.mes + '/' + item.dia + '/' );
            });
        });
    }
    loadEvents();

    $('.eventos .prev').click(function(event) {
        event.preventDefault();
        loadEvents(parseInt($('.eventos').attr('data-index'))-4);
    });
    $('.eventos .next').click(function(event) {
        event.preventDefault();
        loadEvents(parseInt($('.eventos').attr('data-index'))+4);
    });
