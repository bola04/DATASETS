                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://www.morelikehome.net/2018/06/diy-shiplap-fireplace.html">
                      DIY Shiplap Fireplace
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-9301223726444401" itemprop="description articleBody">
                     Sometimes I just need a little kick in the pants to get a project done. Or a little bun in 
the oven. I've wanted to update our fireplace ever since we moved in
                     <i>
                      (two years ago)
                     </i>
                     but it wasn't until this spring when I was in full pregnancy nesting 
mode that I finally decided "I AM DOING THIS. RIGHT NOW." Naturally the 
"I" translated to "husband" because we were in the late-pregnancy stage 
of nesting. And he isn't one to let his round wife balance on a chair 
with a nail gun. He's a keeper.
                     <br/>
                     <br/>
                     And so, with a little help, we managed to check it off my nesting to-do list just in time before our newest guy was born.
                     <i>
                      (you can see my entire living room nesting to-do list
                      <a href="http://www.morelikehome.net/2018/02/nesting-questions-living-room-edition.html" target="_blank">
                       here
                      </a>
                      )
                     </i>
                     One of my favorite details is the faux shiplap we installed above the fireplace.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/shiplap.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="443" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/shiplap.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     I
 love the texture and charm it adds to the living room. I've always 
dreamed of a casual, cozy space for our family to gather, and I feel 
like this was a step in the right direction.
                     <br/>
                     <br/>
                     <div style="text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0793.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="534" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0793.jpg" width="266"/>
                      </a>
                     </div>
                     <br/>
                     Here's a little "before" action. Just drywall painted the same gray as our walls.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_4430.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="450" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_4430.jpg" width="300"/>
                      </a>
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_4431.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="450" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_4431.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     Now
 it's even lighter, brighter, and totally beautiful. And I really need a
 full-fireplace photo. Not gonna lie, I probably shot close up because 
the floor was a hot mess. But I don't really remember. Because newborns 
don't sleep. It's all a blur.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0777.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="705" data-original-width="800" height="352" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0777.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     The
 best part is the whole project only cost about $15 for wood! We also 
had to buy nails for our nail gun (I lost that price in the 
newborn-blur). But we saved a few dollars on paint by using the leftover from our cabinets... 
which doesn't
                     <i>
                      quite
                     </i>
                     match exactly, but that's a problem for another pregnancy.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0787.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="534" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0787.jpg" width="266"/>
                      </a>
                     </div>
                     <br/>
                     Want to see how we did it? We
 started by cutting a $15 sheet of thin plywood into 7" strips. Then the
 process is pretty simple - just nail them to the wall - but as we know,
 nothing ever actually fits the "simple" mold. In this old house, 
nothing is level. Nothing is square. So our next step was cutting the 
top edge of the top board at an angle for a snug fit against the crooked
 crown molding, so the rest of the boards will sit level below it.
                     <i>
                      We started with the front of the fireplace since it's the most important and we wanted to make sure it was right.
                     </i>
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5354.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5354.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     I wanted the front boards to overhang the side boards so the seams would't
 show on the front, so we cut the front boards 1/4" longer than the 
fireplace so theyÂ  would overhang the side boards. We used small scraps 
of wood (you can see it in the pic below) to make sure our front board 
was centered with enough overhang on either side. Then we attached the 
board with a nail gun.
                     <br/>
                     <br/>
                     We also used the old nickel 
trick and pinched a couple nickels between the boards to create the gaps
 that give it a "shiplap" look. I wouldn't go any smaller than a nickel.
 That gap is what gives it the charm!
                     <br/>
                     <br/>
                     <div style="text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5357.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="800" height="300" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5357.jpg" width="400"/>
                      </a>
                     </div>
                     <div style="text-align: center;">
                      <i>
                       See the nickels?
                      </i>
                     </div>
                     <br/>
                     So
 we worked our way down the front with nickels and boards until we 
reached the mantle. This time "we" means Husband and my BIL. I
 very helpfully watched from the couch. Go team.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5356.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5356.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     Then "we" started on the sides.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5358.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5358.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     This
 is when we learned just how wonky the walls are. We're talking all kinds of wonky. Crooked, wavy, slanted, it was all of those and more. So
 we had to measure the top and bottom of each space and cut each board at a different angle to fit. Then up they went with nickels and the nail gun.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5361.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5361.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     Almost done!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5363.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5363.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     We
 did run in to one little road bump with the nail gun. Turns out there 
is brick behind the plaster (duh, it's a chimney. but we totally didn't think of that.) 
and our nails were too long. They hit the brick stuck out like this:
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_53641.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_53641.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     So I
                     <i>
                      (this is an actual "I." See? I totes helped with this project.)
                     </i>
                     ran to the hardware store to pick up shorter nails. And just like that, the boards were up!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5366.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5366.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     Then I taped around the edges...
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5387.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5387.jpg" width="300"/>
                      </a>
                     </div>
                     <br/>
                     ...and
 gave it a coat of white paint. Should I have primed it? Probably. But 
our plywood was "pre-primed" (you can see a few boards are backwards in 
the earlier pics) and I was using one of those "paint and primer in 
one!" deals. PLUS I was basically a ticking time bomb. So it is what it 
is and we'll see how it holds up. So far it looks great!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5388.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="600" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5388.jpg" width="300"/>
                      </a>
                     </div>
                     <div style="text-align: center;">
                      <i>
                       This was the first of three coats.
                      </i>
                     </div>
                     <br/>
                     If
 you want a sharper finish, you can fill the nail holes and corner 
seams with caulk before you paint. But I liked the more rustic feel of 
the visible nail holes and imperfections. And the less-work of not 
filling the holes. Double-win!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0795.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="539" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0795.jpg" width="268"/>
                      </a>
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0791.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="534" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0791.jpg" width="266"/>
                      </a>
                     </div>
                     <br/>
                     As
 soon as the paint was dry, we got our antique wooden star back up. I 
found this star at an antique store with my SIL for like $6 to use with 
our Christmas decor. When I got home I walked in the door, thought "I 
don't want to take this to the basement right now" and hung it on a nail
 that had been in the wall since we bought the house. And now it's been there for a year and a half. Turns out we love it!
                     <br/>
                     <br/>
                     <div style="text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0775.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="534" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0775.jpg" width="265"/>
                      </a>
                     </div>
                     <br/>
                     I've changed up the rest of the accessories about five times. Just because mantles are fun.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0778.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="604" data-original-width="800" height="301" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0778.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     There are so many options for mantle decor, but I'll tell you what I know about decorating with boys in the house - 
don't use anything you can't afford to lose. They will most likely break
 everything. It's just a matter of time. But, since we've already learned that lesson, we can plan accordingly and avoid a few disasters! That's why I love using things like books,
 wooden accents, and plants (in $0.50 pots, so it's no sweat if it takes
 a fall). And this adorable little globe I found on our local yard sale 
page.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0790.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="534" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_0790.jpg" width="266"/>
                      </a>
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: left;">
                      All in all, this shiplap was a pretty quick weekend project with a big impact! Just cut boards, nail them up, and
 add a few coats of paint. Now the whole fireplace has a strong presence and sense of style.
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5568.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="800" data-original-width="800" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5568.jpg" width="400"/>
                      </a>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <br/>
                     </div>
                     <div class="separator" style="clear: both; text-align: left;">
                      Want to make a big impact somewhere else? You
 can even use this same technique for bigger projects like a feature wall! 
It would be great behind a headboard, in the dining room, or in any 
little nook that could use some interest and texture. We even helped my 
SIL install some in her breakfast room. And it's amazing. You can check 
out the "after" on her Instagram at
                      <a href="https://www.instagram.com/p/BSMn4wQB8pS/?taken-by=clayanddust" target="_blank">
                       Clay + Dust
                      </a>
                      .
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <br/>
                     </div>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/20160423_183605.jpg" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="450" data-original-width="800" height="225" src="http://i1106.photobucket.com/albums/h364/morelikehome/2018/20160423_183605.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     I know shiplap is super trendy at the moment, but it also has a timeless, cozy quality. It's a great way to add a little charm and give your home a more comfortable, lived-in vibe. You know my dream is a casual bring-your-friends-over home for our boys, and shiplap was a perfect accent to fit that goal and still make momma happy that it's looking pretty in here.

                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2018/IMG_5568.jpg" style="margin-left: 1em; margin-right: 1em;">

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://www.morelikehome.net/2018/06/diy-travel-map-pinboard.html">
                      DIY Travel Map Pinboard
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-8623453169596921010" itemprop="description articleBody">
                     This little project is
                     <i>
                      almost
                     </i>
                     too simple to even mention. But it's also so easy and so fun that I don't want guys to miss it! It's a little framed map pinboard to mark all your travels. It's easy dress it up or down to match your decor by switching out the frame or changing the map style, plus it's a great family keepsake!
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_0312b.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="473" src="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_0312b.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     We only have a handful of pins so far. My first thought was to only mark the places we'd all been together, as a family of five. But now that already doesn't make sense because we just added a new baby who hasn't been any of these places! So we might as well go back and add everywhere we've been since we were married. Maybe we'll use a different color pin to show how big our family was for each trip.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_0315.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="400" data-original-width="600" height="266" src="http://i1106.photobucket.com/albums/h364/morelikehome/IMG_0315.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     This map was a great fit for the new adventure gallery in our dining room. You can see it up on the far left above the mountain painting.

                     <a href="http://www.morelikehome.net/2018/05/diy-house-painting-tips-how-we-painted.html">
                      DIY House Painting Tips {how we painted our huge house and saved thousands}
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-5812690486650184758" itemprop="description articleBody">
                     Last fall we took on our biggest project to date. It was a doozy. But a very necessary doozy that would have cost thousands of dollars to hire out. We painted our house! It was an enormous job but we're here to tell you that it is actually totally doable, and it can save you a big chunk of change.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0061b.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0061b.jpg" width="350"/>
                      </a>
                     </div>
                     <br/>
                     We knew when we bought our old yellow farmhouse that it came with 90-year-old wood siding. Which was pretty cool. But in our new house excitement we maybe overlooked just what condition that siding was in. After our first winter, we started to notice that it was looking a little worse for the wear. Then we found out from a neighbor that
                     <i>
                      it hadn't been painted it about 15 years
                     </i>
                     . And it showed.
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2016/3%20july-sep/move-in.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="716" data-original-width="800" height="357" src="http://i1106.photobucket.com/albums/h364/morelikehome/2016/3%20july-sep/move-in.jpg" width="400"/>
                      </a>
                     </div>
                     <div style="text-align: left;">
                      <br/>
                     </div>
                     <div style="text-align: left;">
                      We spent the summer trying to decide if we should paint it now or hold 
off until next year. And then we had a hail storm that knocked off huge 
amounts of paint (so I guess it did some of the work for us!). Now it 
was painfully obvious that we needed to do something before things got 
any worse.
                     </div>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0060.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="400" data-original-width="600" height="266" src="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0060.jpg" width="400"/>
                      </a>
                     </div>
                     <br/>
                     Fortunately I had spent those summer months thinking about 
what color paint to buy. Which is good because I used every minute right
 up until my husband called and said "I'm on my way home and we are 
buying paint tonight.
                     <i>
                      You have to decide
                     </i>
                     ." Seriously, it took me months. (You can see all
                     <a href="http://www.morelikehome.net/2017/03/exterior-paint-debate.html" target="_blank">
                      the top contenders here
                     </a>
                     . Including the one Husband hated the most. Just do me a favor and don't tell me if I was wrong. M'kay? We are
                     <i>
                      not
                     </i>
                     repainting.)
                     <br/>
                     <br/>
                     <div class="separator" style="clear: both; text-align: center;">
                      <a href="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0062.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                       <img border="0" data-original-height="600" data-original-width="400" height="400" src="http://i1106.photobucket.com/albums/h364/morelikehome/2017/IMG_0062.jpg" width="266"/>
                      </a>
                     </div>
                     <div style="text-align: center;">
                      <i>
                       It was so pretty in the fall!
                      </i>
                     </div>

        if (type === 'checkboxes' || type === 'radio_buttons') {
          this.checkboxAndRadioValidation(field);
        } else {
          this.textAndDropdownValidation(field, type);
        }
      }
    },

    fieldType: function(field) {
      var type = field.querySelectorAll('.field_type');

      if (type.length) {
        return type[0].getAttribute('data-field-type');

        return 'checkboxes';
      } else {
        return 'text_field';
      }

    checkboxAndRadioValidation: function(field) {
      var inputs   = field.getElementsByTagName('input'),
          selected = false;

      for (var i = 0; i < inputs.length; ++i) {
        var input = inputs[i];
        if((input.type === 'checkbox' || input.type === 'radio') && input.checked) {
          selected = true;
        }
      }

      if (selected) {
        field.className = field.className.replace(/ invalid/g, '');
      } else {
        if (field.className.indexOf('invalid') === -1) {
          field.className += ' invalid';
        }

        this.isValid = false;
      }
    },

    textAndDropdownValidation: function(field, type) {

            this.dropdownValidation(field, input);
