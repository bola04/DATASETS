     <h2>
      Kolarstwo maraton czy półmaraton?
     </h2>
     Sporty, szczególnie takie jak maratony od lat cieszą się ogromną popularnością wśród sportowców i społeczeństwa. Są okazją do wspólnej, międzypokoleniowej zabawy, a niejednokrotnie wyniki maratonu to dla wielu powód do dumy. Podobnie półmaratony - podobne formą i oprawą, lecz bardziej osiągalne dla przeciętnych biegaczy - pasjonatów. Wyniki półmaratonu to również osiągnięcie, którym warto się pochwalić.
		Zawody sportowe zyskują coraz większą rzeszę fanów i uczestników ; na tematycznych portalach dzielą się swoimi osiągnięciami i wynikami. Wystarczy w wyszukiwarkę wpisać słowa triathlon wyniki lub kolarstwo wyniki, a natychmiast zobaczymy, co przedstawiały ostatnie np. zawody triathlonowe. Nie tylko imprezy triathlonowe stają się coraz popularniejsze. Wszystkie sporty biegowe notują wzrost zainteresowania zarówno hobbystów, jak i mediów, inwestorów. Niegdyś traktowane po macoszemu, obecnie są szeroko komentowane. Wyniki triathlonu cieszą się już porównywalnym zainteresowaniem do kiedyś najpopularniejszych wyszukiwanych haseł, jak maraton wyniki lub półmaraton wyniki.
		Prowadzący serwisy poświęcone imprezom dla biegaczy notują też niesłabnące zainteresowanie takimi frazami jak ogólne biegi wyniki lub bieganie wyniki, choć i ironman wyniki pojawia się już coraz częściej.
		Jakie wnioski można z tego wyciągnąć? Sport wraca do łask ; nie tylko profesjonaliści, lecz i amatorzy, pasjonaci trenują, mierzą swoje wyniki i dzielą się nimi ze światem. Motywują się wzajemnie i wspierają w kolejnych sportowych wyzwaniach.

	 /* resize background images */
	 function backgroundResize(){
	     var windowH = $(window).height();
	     $(".background").each(function(i){
		 var path = $(this);
		 // variables
		 var contW = path.width();
		 var contH = path.height();
		 var imgW = path.attr("data-img-width");
		 var imgH = path.attr("data-img-height");
		 var ratio = imgW / imgH;
		 // overflowing difference
		 var diff = parseFloat(path.attr("data-diff"));
		 diff = diff ? diff : 0;
		 // remaining height to have fullscreen image only on parallax
		 var remainingH = 0;
		 if(path.hasClass("parallax")){
		     var maxH = contH > windowH ? contH : windowH;
		     remainingH = windowH - contH;
		 }
		 // set img values depending on cont
		 imgH = contH + remainingH + diff;
		 imgW = imgH * ratio;
		 // fix when too large
		 if(contW > imgW){
		     imgW = contW;
		     imgH = imgW / ratio;
		 }
		 //
		 path.data("resized-imgW", imgW);
		 path.data("resized-imgH", imgH);
		 path.css("background-size", imgW + "px " + imgH + "px");
	     });
	 }
	 $(window).resize(backgroundResize);
	 $(window).focus(backgroundResize);
	 backgroundResize();
