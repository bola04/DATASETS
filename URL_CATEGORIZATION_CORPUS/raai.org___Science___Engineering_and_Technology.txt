           Уважаемые коллеги!
           <br/>
           <br/>
           <b>
            19 июня 2018 года
           </b>
           (вторник) в 14.00 (
           <b>
            7.00 мск
           </b>
           ) в
           <b>
            Институте автоматики и процессов управления
           </b>
           (к. 606) состоится заседание семинара Владивостокского Регионального отделения РАИИ.
           <br/>
           <br/>
           Повестка: обсуждение диссертационной работы инженера-программиста ИПМ ДВО РАН Соловьёва Игоря Владимировича
           <b>
            "Реконструкция трехмерной модели городской обстановки по топографическому плану"
           </b>
           , представляемой на соискание ученой степени кандидата технических наук по специальности 05.13.11 "Математическое и программное обеспечение вычислительных машин, комплексов и компьютерных сетей".
           <br/>
           Научный руководитель: к.т.н. Кудряшов Алексей Павлович.
           <br/>
           <br/>
           Для
           <b>
            дистанционного участия
           </b>
           в семинаре через skype желательно заранее сообщить об этом секретарю (
           <a class="page" href="mailto:seminar@iacp.dvo.ru" target="_blank">
            seminar@iacp.dvo.ru
           </a>
           ) и прислать ник в скайпе (наш - int.sys.lab).
           <br/>

            <div align="right">
             РАИИ

           </b>
           . Начало в 16:00.
           <br/>
           Место проведения: НИЯУ МИФИ, аудитория К-716. Адрес: Каширское шоссе, д. 31.
           <br/>

           Тема:
           <b>
            "Toward a conceptual neuropsychological framework of social trust"
           </b>
           (доклад на английском языке), которую представляет
           <a class="page" href="http://frank-krueger.socialpsychology.org" target="_blank">

            </b>
           </a>
           , Ph.D., Associate Professor of Systems Social Neuroscience at the School of Systems Biology at George Mason University, Fairfax, Virginia, USA.
           <br/>
           <br/>
           Для заказа пропусков заранее обращайтесь к
           <a class="page" href="mailto:vikhapalov@mephi.ru" target="_blank">
            Хапалову Виктору Игоревичу
           </a>
           . В письме обязательно указывайте ФИО полностью, а также место работы (включая город).

           Очередное заседание 
	Общемосковского научного семинара
           <b>
            "Проблемы искусственного интеллекта"
           </b>
           состоится
           <b>
            23 мая
           </b>
           (среда) в Институте системного анализа РАН (Москва, проспект 60-летия Октября, 9), в конференц-зале на 1 этаже. Начало в
           <b>
            18.30
           </b>
           .
           <br/>
           <br/>
           Доклад:
           <br/>
           <b>
            Люксембург А.А.
           </b>
           "Автоматизированные построения в математике: история, современность и перспективы".
           <br/>
           <br/>
           <a class="page" href="/news/pii/annot/2018/pii1805_Luksemburg.pdf" target="_blank">
            Аннотация доклада
           </a>
           <br/>
           <br/>
           Вход свободный, без регистрации. Приглашаются все желающие.
           <br/>

           Уважаемые коллеги!
           <br/>
           <br/>
           <b>
            Изменены сроки подачи материалов на конференцию КИИ-2018:
           </b>
           <ul>
            <li>
             <b>
              15 мая
             </b>
             – последний срок представления докладов (на русском языке и английском языках).
            </li>
            <li>
             5 июня – извещение авторов о решениях программного комитета.
            </li>
            <li>
             1 августа – получение текстов принятых докладов, подготовленных к публикации (camera-ready).
            </li>
           </ul>
           На конференцию принимаются работы по всем основным направлениям искусственного интеллекта:
           <br/>
           - автоматизация рассуждений и интеллектуальный анализ данных;
           <br/>
           - мультиагентные системы, интеллектуальные роботы и поведение;
           <br/>
           - анализ естественного языка и понимание текстов;
           <br/>
           - интеллектуальные системы и приложения методов ИИ.
           <br/>
           <br/>
           Подача статей осуществляется через систему Easychair. Правила оформления докладов и инструкция по загрузке работ представлены на
           <a class="page" href="http://2018.rncai.ru/ru/submission/" target="_blank">
            сайте конференции
           </a>
           .
           <br/>
           <br/>
           В рамках КИИ-2018 будет организован открытый хакатон и конкурс работ молодых ученых в области искусственного интеллекта по следующим номинациям:
           <br/>
           - фундаментальные исследования;
           <br/>
           - прикладные исследования.
           <br/>
           <br/>
           Подробности во
           <a class="page" href="/news/arch_news/2018/04/CAI-2018_2_inf_let.pdf" target="_blank">
            втором информационном письме
           </a>
           и на

            сайте конференции
