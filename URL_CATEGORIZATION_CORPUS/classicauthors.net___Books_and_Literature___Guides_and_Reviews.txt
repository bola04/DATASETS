        健康に良いと注目を集めているプロポリスですが、実は年間で計算するとわずか200gほどしか採取ができず、とても希少価値の高い物質です。
        <br/>
        高品質なプロポリスにはフラボノイドやビタミン、ミネラルなどが豊富に含まれており、人々からの需要は高いものの、やはり値段も高いです。
        <br/>
        ランクの高いプロポリスは安いものと比べると、値段が倍以上になります。
        <br/>
        <br/>
        一般的にプロポリス液は、精製水にプロポリスを加えた商品で高濃度に含有させている商品はあるものの、やはり内容量としては少なく、値段も高いです。
        <br/>
        安い商品の場合は、1,500円程度であるものの、内容量は精製水含め30ml程度。
        <br/>
        高額な商品は濃度が20%程度にも関わらず1万円する商品もあります。
        <br/>
        <br/>
        やはり費用対効果を考えると、濃度が高い商品を選ぶべきです。
        <br/>
        濃度が一番高い商品で30w/v%含有している商品があり、値段は4,000円程度です。
        <br/>
        <br/>
        プロポリス液の定番の使い方といえば、やはりうがいでしょうか。
        <br/>
        風邪の引き始めに水に液を数滴たらしてうがいをすると治る、と言われています。
        <br/>
        飲むこともできるので、風邪予防にもなりますしジュースに数滴入れて飲むこともできます。
        <br/>
        <br/>
        品質が悪い物と、品質が良い物を見分ける方法などもありますが、やはり選ぶ際には原産国と濃度、そして値段を比較して選ぶ方法が一番良いといえます。
        <br/>
        <a href="http://xn--nckgz3j4a3efw2db.com/" target="_blank">
         プロポリスのランキング
        </a>
        や口コミを見ると、良品かどうかはすぐにわかりますし、どういった商品を選ぶべきかもよくわかります。
        <br/>
        値段が高いから良い商品だ、とは限りません。
        <br/>
        値段は高くてもほとんどが水で、濃度が低くまったく効果を実感できない商品もあります。
        <br/>
        健康のために使用するものなので、値段に左右されず品質や安全性を重視して選びましょう。
        <br/>
        <br/>
        このように、値段にはバラつきがあるプロポリス液でしたが、原産国によっても値段は異なります。
        <br/>
        高品質かどうか、初めて選ぶ際にはわかりづらいかもしれません。
