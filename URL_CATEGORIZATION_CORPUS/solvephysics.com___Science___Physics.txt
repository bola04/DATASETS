            <h1 align="center" class="style25 style31 style40">
             Physics Problems
            </h1>
           </th>
          </tr>
          <tr>
           <td valign="top">
            <p align="justify">
             <strong>
              Physics Problems
             </strong>
             have a special place in Physics Learning.
            </p>
            <p align="justify">
             We understand Physics – it means that we can solve physics problems.
            </p>
            <p align="justify">
             At the same time to understand physics we need to solve as many physics problems as possible.
            </p>
            <p align="justify">
            </p>
            <p align="justify" class="style40">
             <strong>
              Only by solving physics problems  can we understand Physics Laws and  how to apply them.
             </strong>
            </p>
            <p align="justify">
            </p>
            <p align="justify">
             There are a few general rules we need to follow when we solve
             <strong>
              Physics Problems.
             </strong>
             These rules are
            </p>
            <ul>
             <li class="style17">
              <div align="justify">
               <strong>
                drawing the picture of the problem
               </strong>
               <span class="style48">
                (very important !!!)
               </span>
               - the correct picture of the physics problem is more then 80 % of success.
              </div>
             </li>
            </ul>
            <ul>
             <li class="style17">
              <div align="justify">
               <strong>
                having the same
                <a href="units.htm">
                 units
                </a>
                for all variables in the problem;
               </strong>
              </div>
             </li>
            </ul>
            <div align="justify">
             <ul>
              <li class="style17">
               <strong>
                checking the dimensionality of analytical expressions,
               </strong>
              </li>
             </ul>
             <span class="style17">
              and so on.
             </span>
            </div>
            <h2 align="center" class="style31 style32">
             <em>
              Solutions of Physics Problems:
             </em>
            </h2>
            <p align="justify">
             <strong>
              Another important fact about Physics Problems is knowing
              <span class="style28">
               how to read the Solution to a Physics Problem:
              </span>
             </strong>
            </p>
            <p align="justify">
             <strong>
             </strong>
             It is very important to understand the solution of the problem when you read  it in the book.
            </p>
            <p align="justify">
             You read the solution of the problem and it looks very simple, so you think you understand it. But you could be wrong.
            </p>
            <p align="justify">
            </p>
            <p align="justify">
             To find out if you understand the solution of the problem or not
             <strong>
              you need to close the book and try to solve the problem by yourself.
             </strong>
            </p>
            <p align="justify">
             <strong>
              If you can solve the problem then
             </strong>
             <strong>
              you understand the solution.
             </strong>
            </p>
            <p align="justify">
             <span class="style30">
              If not
             </span>
             then you need to open the book and  read the solution again, then close the book and try to solve the problem again.
            </p>
            <p align="justify" class="style30">
             <em>
              It is very important that you are able to solve physics problems without looking at the solutions.
             </em>
            </p>
