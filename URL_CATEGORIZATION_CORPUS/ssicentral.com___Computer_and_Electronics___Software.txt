             <p class="plaintext" span="">
              We provide software tools for statistical 
                    data analysis, useful for work in a variety of fields such 
                    as:
             </p>
             <ul>
              <li>
               Statistics (including survey research and informatics)
              </li>
              <li>
               Behavioral and social sciences (such as psychology, sociology, 
                      psychiatry, criminal science, family studies, political 
                      science, developmental research, anthropology, or social 
                      work)
              </li>
              <li>
               Medical research (including nursing, pharmacy, epidemiology, 
                      gerontology, kinesiology, sport science, and other fields)
              </li>
              <li>
               Education (in administration, policy studies, test analysis, 
                      counseling, and more)
              </li>
              <li>
               Business research (marketing, management, economics, 
                      organization)
              </li>
              <li>
               Environmental science (including resource administration 
                      and longitudinal research)
              </li>
              <li>
               Other diverse research areas, such as language studies, 
                      engineering, law, chemistry, or urban planning
              </li>
             </ul>
             <p align="justify" class="plaintext">
              Our best-known products include:
             </p>
             <ul>
              <li>
               <a href="/lisrel/LISREL_9.1_Release_Notes.pdf">
                <span class="paragraph_headers">
                 LISREL 
                        9
                </span>
               </a>
               The pioneering software for structural equation 
                      modeling now includes statistical methods for complex survey data. Create path diagrams 
                      in an easy to use interface and generate syntax directly 
                      from the diagram. The latest LISREL for Windows includes 
                      the following statistical applications.
               <ul>
                <li>
                 <a href="lisrel/index.html#lisrel">
                  <b>
                   LISREL
                  </b>
                 </a>
                 for structural equation modeling.
                </li>
                <li>
                 <a href="lisrel/index.html#prelis">
                  <b>
                   PRELIS
                  </b>
                 </a>
                 for data manipulations and basic statistical analyses.
                </li>
                <li>
                 <a href="lisrel/index.html#multilev">
                  <b>
                   MULTILEV
                  </b>
                 </a>
                 for hierarchical linear and non-linear modeling.
                </li>
                <li>
                 <a href="lisrel/index.html#sglim">
                  <b>
                   SURVEYGLIM
                  </b>
                 </a>
                 for generalized linear modeling.
                </li>
                <li>
                 <a href="lisrel/index.html#mapglim">
                  <b>
                   MAPGLIM
                  </b>
                 </a>
                 for generalized linear modeling for multilevel data.
                </li>
               </ul>
              </li>
              <li>
               <a class="paragraph_headers" href="hlm/index.html">
                HLM 
                      7
               </a>
               Hierarchical data modeling with power. The latest 
                      version includes an improved interface, HTML output, and 
                      new statistical features such as cross-classified random 
                      effects for linear models for three-level data, and linear 
                      and nonlinear models for four-level data.
              </li>
              <li>
               <a class="paragraph_headers" href="supermix/index.html">
                SuperMix 
                      1
               </a>
               <font color="#CC0033">
                <font size="-2">
                </font>
               </font>
               Mixed-effect 
                      program that combines MIXREG, MIXOR, MIXNO and MIXPREG; 
                      allows two- and three-level models. Easy to use graphical 
                      user interface.
              </li>
              <li>
               <a class="paragraph_headers" href="irt/index.html">
                Item 
                      Response Theory
               </a>
               Our group of item analysis programs 
                      (BILOG-MG, IRTPRO, and PARSCALE) now feature friendly, new 
                      Windows graphical user interfaces. They are all the tools 
                      you will need to perform a wide variety of analyses on many 
                      different types of tests and items.
              </li>
             </ul>
