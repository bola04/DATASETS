      <div class="entry clearfix">
       <h1 class="post-title entry-title">
        Коледж радіоелектроніки
       </h1>
       <ul class="post-meta">
        <li class="print">
         <a href="javascript:print();" title="Print this Сторінка">
          <i class="fa fa-print">
          </i>
         </a>
        </li>
       </ul>
       <div class="entry-content clearfix">
        <h3 style="text-align: center;">
         <iframe height="350" src="//www.youtube.com/embed/4984wDrCBL0" width="425">
         </iframe>
        </h3>
        <p style="text-align: center;">
         <span style="font-size: 14pt;">
          <strong>
           Вітаємо Вас на сайті коледжу радіоелектроніки
           <br/>
          </strong>
         </span>
        </p>
        <p>
         <span style="font-size: 12pt;">
          Коледж радіоелектроніки ‒ вищий навчальний заклад І рівня акредитації заснований у 1962 році. За роки свого існування коледж підготував більше 16 тисяч фахівців за освітньо-кваліфікаційним рівнем молодшого спеціаліста.
         </span>
        </p>
        <p>
         <span style="font-size: 12pt;">
          Коледж має унікальну навчальну матеріально-технічну базу з сучасними кабінетами та лабораторіями, які обладнані сучасною комп’ютерною технікою, електрорадіовимірювальними приладами, радіотелевізійною апаратурою та системами електрозв’язку.
         </span>
        </p>
        <p>
         <span style="font-size: 12pt;">
          Навички вивчення програмного забезпечення та комп’ютерного проектування в навчальному  процесі виводять наших студентів на сучасний рівень інноваційних технологій виробництва.
         </span>
        </p>
        <p>
         <span style="font-size: 12pt;">
          У коледжі активно працюють гуртки технічної творчості, художньої самодіяльності та спортивні секції.
         </span>
        </p>
        <p>
         <span style="font-size: 12pt;">
          Після закінчення коледжу випускники отримують диплом молодшого спеціаліста. Він надає можливість вступу у вищі навчальні заклади ІІІ-ІV рівня акредитації відповідного напрямку за скороченим терміном навчання.
         </span>
        </p>
       </div>

function createCookie(name,value,days) {
  if (days) {
    var date = new Date();
    date.setTime(date.getTime()+(days*24*60*60*1000));
    var expires = "; expires="+date.toGMTString();
  } else {
    var expires = "";
  }
  document.cookie = name+"="+value+expires+"; path=/";
}

function readCookie(name) {
  var nameEQ = name + "=";
  var ca = document.cookie.split(';');
  for(var i=0;i < ca.length;i++) {
    var c = ca[i];
    while (c.charAt(0)==' ') {
      c = c.substring(1,c.length);
    }
    if (c.indexOf(nameEQ) == 0) {
      return c.substring(nameEQ.length,c.length);
    }
  }
  return null;
}

function eraseCookie(name) {
  createCookie(name,"",-1);
}
