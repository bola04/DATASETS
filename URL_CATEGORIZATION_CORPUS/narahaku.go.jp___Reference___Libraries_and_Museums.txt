       <div class="list">
        <h3>
         お知らせ
        </h3>
        <p>
         平成30年6月18日（月）に発生した大阪府北部を中心とする地震においてお亡くなりになられた方々の御冥福をお祈りするとともに、被災された皆様に心からお見舞い申し上げます。なお、当館では館内の点検を行った結果、異常はありませんでした。本日より通常通り、なら仏像館・青銅器館は開館しております。[2018.06.19]
         <br/>
        </p>
       </div>
       <div class="list2">
        <h3>
         平成30年度夏季仏像供養
        </h3>
        <p>
         7月31日（火）、なら仏像館第6室において夏季仏像供養を行います。
         <a href="https://www.narahaku.go.jp/events/2018event/kaki_kuyo.html">
          詳しくはこちらへ
         </a>
         [2018.06.08]
         <br/>
        </p>
       </div>
       <div class="list2">
        <h3>
         第47回 奈良国立博物館 夏季講座 「素材から探る仏像のひみつ」
        </h3>
        <p>
         夏季講座の募集は終了いたしました。受入の可否は、6月15日（金）までに返信用はがきまたはメールにてご連絡いたします。
         <a href="https://www.narahaku.go.jp/events/2018event/2018kakikouza.html">
          詳しくはこちらへ
         </a>
         [2018.06.04]
         <br/>
        </p>
       </div>
       <div class="list2">
        <h3>
         「ガラス乾板データベース」の公開
        </h3>
        <p>
         仏教美術資料研究センターにて、標記データベースの公開を開始します。明治28年の開館以降、昭和40年代までのガラス乾板約1万3千枚の画像をご覧頂けます。どうぞご活用下さい。
        [2018.06.01]
         <br/>
        </p>
       </div>
       <div class="list2">
        <h3>

        </h3>

         <a href="https://www.narahaku.go.jp/news/05.html">
          詳しくはこちらへ
         </a>
         [2017.10.24]
         <br/>
        </p>
       </div>
       <div class="list2">
        <h3>
         室生寺　釈迦如来坐像・弥勒菩薩立像の公開について
        </h3>
        <p>
         10月3日（火）より、名品展「珠玉の仏たち」（なら仏像館）において、奈良　室生寺の国宝・釈迦如来坐像、重要文化財・弥勒菩薩立像を特別公開しております。
         <a href="https://www.narahaku.go.jp/exhibition/2017mei/2017mei_10_chokoku.html">
          詳しくはこちらへ
         </a>
         [2017.10.03]
         <br/>
