        <h1 class="style-2" style="margin-bottom: 0; padding-bottom: 4px;">
         Word of the Day: Trouble
        </h1>
        <div style="border-top: 1px solid #d9d9d9; padding-bottom: 8px;">
        </div>
        <div style="font-size: 0.9em;">
         TROU-ble / trəb(ə)l
         <br/>
         <br/>
         noun
         <br/>
         <br/>
         a problem; difficulty
         <br/>
         <br/>
         Expect
         <strong>
          trouble
         </strong>
         as an inevitable part of life and repeat to yourself the most comforting words of all: this, too, shall pass.
         <br/>
         <br/>
         Ann Landers, 1918-2002
         <br/>
         <br/>
         <br/>
         <br/>
         a dangerous or concerning situation
         <br/>
         <br/>
         Opinion has caused more
         <strong>
          trouble
         </strong>
         on this little earth than plagues or earthquakes.
         <br/>
         <br/>
         Voltaire, 1694-1778
         <br/>
         <br/>
         <br/>
         <br/>
         civil conflict or disorder
         <br/>
         <br/>
         If there must be
         <strong>
          trouble
         </strong>
         , let it be in my day, that my child may have peace.
         <br/>
         <br/>
         Thomas Paine, 1737-1809
         <br/>
         <br/>
         <br/>
         <br/>
         physical ailment
         <br/>
         <br/>
         If you haven’t got any charity in your heart, you have the worst kind of heart
         <strong>
          trouble
         </strong>
         .
         <br/>
         <br/>
         Bob Hope, 1903-2003
         <br/>
         <br/>
         <br/>
         <br/>
         mental anxiety; distress
         <br/>
         <br/>
         A contented mind is the best source for
         <strong>
          trouble
         </strong>
         .
         <br/>
         <br/>
         Plautus, c. 254 BC-185 BC
         <br/>
         <br/>
         <br/>
         <br/>
         bothersome exertion
         <br/>
         <br/>
         To be good is noble; but to show others how to be good is nobler and no
         <strong>
          trouble
         </strong>
         .
         <br/>
         <br/>
         Mark Twain, 1835-1910
         <br/>
         <br/>
         <br/>
         <br/>
         verb
         <br/>
         <br/>
         to cause or be an annoyance or bother
         <br/>
         <br/>
         One could not pluck a flower without
         <strong>
          troubling
         </strong>
         star.
         <br/>
         <br/>
         Loren Eiseley, 1907-1977
         <br/>
         <br/>
         <br/>
         <br/>
         to cause emotional unrest
         <br/>
         <br/>
         It never
