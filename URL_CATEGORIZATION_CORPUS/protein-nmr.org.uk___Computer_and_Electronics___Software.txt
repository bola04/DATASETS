       <h2 class="page-title entry-title">
        <a href="http://www.protein-nmr.org.uk/" rel="bookmark" title="Introduction">
         Introduction
        </a>
       </h2>
       <div class="entry-content entry">
        <div>
         <p>
          Most books on Protein NMR focus on theoretical aspects and pulse sequences with only little space devoted to resonance assignment and structure calculations. At the same time many software manuals provide detailed information on how to use the software, but assume prior knowledge of the concepts of assignment and structure calculation. This has produced a gap in this area which these webpages aim to bridge by describing the concepts of assignment in detail with the help of many illustrations. Much space and discussion is devoted to practical aspects.
         </p>
         <p>
          The implementation of protein NMR assignment is described using the program
          <a href="https://www.ccpn.ac.uk/v2-software/software" title="CCPNmr Analysis">
           CCPNmr Analysis
          </a>
          . This program has been developed by
          <a href="http://www.ccpn.ac.uk/" title="CCPN">
           CCPN
          </a>
          and actively seeks input from the NMR community. CCPNmr Analysis is based on the detailed and well thought-out
          <a href="http://www.ccpn.ac.uk/software/extras/datamodelfolder/datamodel" title="CCPN Data Model">
           CCPN Data Model
          </a>
          which has the advantage (a) that it feeds directly into the CCPN Format Converter thus simplifying the import from and export to other programs, and (b) that as more and more NMR-related programs adopt the CCPN Data Model it is likely to take on a key role in NMR data management – in a similar way to
          <a href="http://www.ccp4.ac.uk" title="CCP4">
           CCP4
          </a>
          for protein X-ray crystallography. CCPNmr Analysis is already one of the best assignment programs availabe while still being developed and provides excellent support via the
          <a href="http://www.jiscmail.ac.uk/lists/CCPNMR.html" title="CCPN Mailing List">
           CCPN Mailing List
          </a>
          (a
          <a href="http://sites.google.com/site/ccpnwiki/Home/documentation/ccpnmr-analysis" title="CCPNmr Analysis Wiki Manual">
           manual
          </a>
          is also available).
         </p>
        </div>
        <p>
        </p>
        <div>
         <p>
          Webpages include:
         </p>
         <ul>
          <li>
           description of several
           <a href="http://www.protein-nmr.org.uk/solution-nmr/assignment-theory/" title="Assignment Theory">
            resonance assignment strategies
           </a>
          </li>
          <li>
           simple descriptions and discussions of many multidimensional
           <a href="http://www.protein-nmr.org.uk/solution-nmr/spectrum-descriptions/" title="Spectrum Descriptions">
            NMR experiments
           </a>
           commonly used in protein NMR
          </li>
          <li>
           basic usage of CCPNmr Analysis (versions
           <a href="http://www.protein-nmr.org.uk/ccpnmr-analysis/version-1-0-basics/" title="Version 1.0 Basics">
            1
           </a>
           and
           <a href="http://www.protein-nmr.org.uk/ccpnmr-analysis/version-2-x-basics/" title="Version 2.x Basics">
            2
           </a>
           )
          </li>
          <li>
           how to make
           <a href="http://www.protein-nmr.org.uk/ccpnmr-analysis/figures-with-ccpnmr-analysis/">
            publication quality figures
           </a>
           using CCPNmr Analysis
          </li>
          <li>
           advice for using
           <a href="http://www.protein-nmr.org.uk/solid-state-mas-nmr/ccpnmr-analysis-solid-state-tips/" title="CCPNmr Analysis Solid-state Tips">
            CCPNmr Analysis with solid-state MAS NMR
           </a>
           data
          </li>
          <li>
           <a href="http://www.protein-nmr.org.uk/solid-state-mas-nmr/solid-state-assignment-tutorial/" title="Solid-state Assignment Tutorial">
            tutorial
           </a>
           on protein assignment using solid-state MAS NMR data
          </li>
          <li>
           description of isotopic
           <a href="http://www.protein-nmr.org.uk/general/isotopic-labelling/" title="Isotopic Labelling">
            labelling strategies
           </a>
           commonly used in protein NMR
          </li>
          <li>
           links to a large number of protein
           <a href="http://www.protein-nmr.org.uk/general/software/" title="Software">
            NMR software packages
           </a>
          </li>
          <li>
           suggested
           <a href="http://www.protein-nmr.org.uk/general/literature/" title="Literature">
            literature
           </a>
           for further reading
          </li>
          <li>
           <a href="http://www.protein-nmr.org.uk/general/nmr-links/" title="NMR Links">
            links
           </a>
           to other useful NMR webpages
          </li>
          <li>
           <a href="http://www.protein-nmr.org.uk/general/glossary-abbreviations/" title="Glossary / Abbreviations">
            Glossary
           </a>
          </li>
         </ul>
        </div>
        <p>
         <strong>
          <span style="color: #ff6600;">
           NEW!
          </span>
         </strong>
         <br/>
         Solid-state NMR spectroscopists may also be interested in my recent
         <a href="https://www.sciencedirect.com/science/article/pii/S0079656518300116">
          article on ‘Solid-state MAS NMR resonance assignment methods for proteins’
         </a>
         published in Progress in NMR Spectroscopy (free download until 27 July 2018 with
         <a href="https://authors.elsevier.com/a/1XB13I79wUZ7K">
          this link
         </a>
         ).
        </p>
       </div>
