         <br/>
         <em>
          DMM電子書籍：
         </em>
         <a href="https://book.dmm.com/detail/b213cnhbg01923/furanskin-001" target="_blank">
          これ喰ってシメ！ (1)
         </a>
        </p>
       </div>
       <p style="clear: both;">
        「孤独のグルメ」で大人気の久住昌之さんが原作を手掛け、
        <br/>
        「世界の果てで愛ましょう」「グレイプニル」の武田すんさんが作画し、
        <br/>
        ゴラクエッグというWeb媒体で連載している漫画が「これ喰ってシメ！」です。
        <br/>
        原作が久住昌之な上にこのタイトルですからね。
        <br/>
        もちろん、グルメ漫画です。
       </p>
       <p>
        出版社に務める三十路後半のやり手編集者の神保マチ子。
        <br/>
        タイトなスケジュールをこなし、漫画家にやる気を起こさせ、
        <br/>
        後輩の面倒も見る彼女が仕事とかをシメる時に、
        <br/>
        ちょっとオヤジ臭いけど、とても美味そうなメシを食べるわけです。
       </p>
       <p>
        <img alt="" border="1" class="alignnone size-full wp-image-35289" height="363" sizes="(max-width: 350px) 100vw, 350px" src="http://www.furanskin.net/wp-content/uploads/korekutte01_01.png" srcset="http://www.furanskin.net/wp-content/uploads/korekutte01_01.png 350w, http://www.furanskin.net/wp-content/uploads/korekutte01_01-289x300.png 289w" width="350"/>
       </p>
       <p>
        マチ子さん、本当に美味そうに食べるよなぁ…
        <br/>
        仕事中の昼日中の蕎麦と冷酒のシメに蕎麦湯、
        <br/>
        回転寿司のシメにウニ、そして小料理屋でお茶漬け…
        <br/>
        どれもが美味そうで胃袋を刺激してくるのです。
        <br/>
        これは深夜に読んじゃダメなヤツだ…
       </p>
       <p>
        それにしても久住昌之さんと武田すんさんの組み合わせって割と異色ですよね。
        <br/>
        久住昌之さんはこう言ったらなんですが、おっさん向けな人なのに、
        <br/>
        武田すんさんはどちらかというと萌え漫画系な人だし。
        <br/>
        でもこの漫画は違和感なく面白いので不思議です。
       </p>
       <p>
        ザックリ言うと基本的に「孤独のグルメ」のノリを萌え系の絵で、
        <br/>
        女性主人公とその後輩がやっている漫画です。
        <br/>
        そういった漫画が好きな人には読んで欲しいですね。
       </p>

      <div class="storycontent">
       <div class="amazon">
        <p>
         <a href="http://www.amazon.co.jp/exec/obidos/ASIN/4088910583/furanskin-22/ref=nosim/" target="_blank">
          <img alt="すんどめ!! ミルキーウェイ 4巻" border="0" class="icon_left" src="http://www.furanskin.net/wp-content/uploads/sundome04.png"/>
         </a>

        </p>
       </div>
       <p style="clear: both;">
        スポポン星人のルルネに種付けするには童貞でなくてはならない。
        <br/>
        でも付き合うことになった同僚の由井園ともさっさとエッチしたい。
        <br/>
        そんな男の欲望に忠実な義武はルルネの恥じらいを克服させて種付けした後、
        <br/>
        それから由井園とセックスしようという、外道な二股を画策するけれど…
       </p>
       <p>
        思いの外早くバレましたね！
        <br/>
        まぁ、そっちの方がテンポ良いですし、
        <br/>
        バレないように二股するハラハラを楽しむきまぐれなオレンジロードとか、
        <br/>
        この漫画には求めていないので問題ナッシングです！
       </p>
       <p>
        では何を求めるのかというと、それはもちろん
        <strong>
         エロ
        </strong>
        でしょう。
        <br/>
        ふなつかずきさんが描くエロい女体を見たいに決まってるじゃないですか！
        <br/>
        ルルネや由井園の恥じらいながらも感じるエロを見たいに決まってるじゃないですか！
        <br/>
        だから全力で二股したい宣言をする義武は嫌いになれないじゃないですか！
       </p>
       <p>
        <img alt="" border="1" class="alignnone size-full wp-image-35281" height="331" sizes="(max-width: 350px) 100vw, 350px" src="http://www.furanskin.net/wp-content/uploads/sundome04_01.png" srcset="http://www.furanskin.net/wp-content/uploads/sundome04_01.png 350w, http://www.furanskin.net/wp-content/uploads/sundome04_01-300x284.png 300w" width="350"/>
       </p>
       <p>
        公衆の面前で大声で自分の欲望を素直に叫ぶ…！
        <br/>
        義武さんマジアホだわー
        <br/>
        でもちょっと憎めないわー
        <br/>
        なんか発言から親から虐待かネグレクトされて育ったように思えるんだけど、
        <br/>
        それにしたってここまで性欲に素直になるとは…
        <br/>
        本人の資質に依るところが大きそうです。
       </p>
       <p>
        それとエロはこの二人だけではありません。
        <br/>
        ゲストキャラ的な扱いで新しいエロが供給されました。
        <br/>
        会社でカタブツと言われている高梨さんです！
        <br/>
        義武も面と向かって叱責するほどにおカタい高梨さんですが、
        <br/>
        一緒にエレベーターに閉じ込められた上で尿意を我慢できなくなり、
        <br/>
        それを指摘されたら流石に恥じらうみたいですね！
       </p>
       <p>
        <img alt="" border="1" class="alignnone size-full wp-image-35280" height="230" sizes="(max-width: 350px) 100vw, 350px" src="http://www.furanskin.net/wp-content/uploads/sundome04_02.png" srcset="http://www.furanskin.net/wp-content/uploads/sundome04_02.png 350w, http://www.furanskin.net/wp-content/uploads/sundome04_02-300x197.png 300w" width="350"/>
       </p>
       <p>
        いやー、良いですねぇ…
        <br/>
        普段はカタブツなのに生理的欲求には負けて恥じらうところ…！
        <br/>
        それだけでも充分美味しいというのに、
        <br/>
        義武のフリーダムなまでの漢っぷりのお蔭で、
        <br/>
        まさかの飲尿展開を見せてくれるとか素晴らしい…！
       </p>
       <p>
        かなりマニアックな性癖だとは思いますが、
        <br/>
        ぢたま某先生に鍛えられた私はこれくらい余裕ですので、
        <br/>
        大変に堪能させていただきましたよ。
       </p>
       <p>
        できれば高梨さんには再登場して欲しいのですが、
        <br/>
        流石に難しいかなー…
        <br/>
        さらなる多角関係になるのは物語が複雑になるだけだと思いますが、
        <br/>
        それでも高梨さんは魅力的なのでもう一度出て欲しいですね。
        <br/>
        別に尿な展開じゃなくてもいいので。（笑
       </p>

       <p style="clear: both;">
        私が昔から同人誌を読んでいる野中友さんの商業新作です。
        <br/>
        同人誌では友達以上百合未満な創作が多かった印象がありますが、
        <br/>
        この作品も百合姫掲載ということからわかるようにほんのり百合作品です。
        <br/>
        可愛いJCたちの日常に癒やされます。
       </p>
       <p>
        ちさきとほのりは3年ぶりに再会した女子中学生。
       </p>
       <p>
        <img alt="" border="1" class="alignnone size-full wp-image-35271" height="304" sizes="(max-width: 350px) 100vw, 350px" src="http://www.furanskin.net/wp-content/uploads/miraino01_01.png" srcset="http://www.furanskin.net/wp-content/uploads/miraino01_01.png 350w, http://www.furanskin.net/wp-content/uploads/miraino01_01-300x261.png 300w" width="350"/>

       <p>
        背伸びしたいお年頃のちさきちゃんと、
        <br/>
        大人になりかけのほのりちゃん。
        <br/>
        幼馴染みなんだけど三年会ってなかったので、
        <br/>
        昔の思い出と今の姿にギャップを感じて軽く喧嘩しつつ、
        <br/>
        仲良く生活している姿にほんわかします。
       </p>
       <p>
        ちさきちゃんは子供に見られたくないんだけど、子供っぽくて可愛く、
        <br/>
        ほのりちゃんはそんなちさきちゃんのことが好きだけど、
        <br/>
        それを言えない葛藤があるみたいで、ちょっと大人っぽい。
        <br/>
        そんな二人がとても愛おしくなる漫画です。
       </p>
       <p>
        1話あたりが短いのでサクサク読める上に、
        <br/>
        ウェットな感じではなく、かといってドライでもない、
        <br/>
        良い塩梅の百合具合なので日常モノとしても楽しめます。
        <br/>
        二人の可愛いJCを愛でるだけでも充分面白いと思うので、
        <br/>
        百合好きにもそうでない人にも読んで欲しいです。
       </p>

        <br/>
        使い魔のポチに心配されて外に出てみれば、
        <br/>
