                       Некоторые люди умирают не сразу. Часто они оказываются в ситуации, в которой им длительное время ещё можно помочь.
                      </b>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <b style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       У вас займёт всего одну минуту прочесть следующее ...
                      </b>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <b style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       Нейрохирурги говорят, что если они в течении 3 часов успевают к жертве инсульта, то последствия приступа могут быть устранены. Трюк состоит в том, чтобы распознать и диагностировать инсульт и приступить к лечению в первые 3 часа – что, конечно, не просто.
                      </b>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <b style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       Существуют 4 шага к распознанию инсульта:
                      </b>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <b style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       1. Попросите человека улыбнуться (он не сможет этого сделать) 2. Попросите сказать простое предложение (напр. «Сегодня хорошая погода») 3. Попросите поднять обе руки (не сможет или только частично сможет поднять) 4. Попросите высунуть язык (если язык искривлён, повёрнут – это тоже признак)
                      </b>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <span style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       Если проблемы возникнут даже с одним из этих заданий – звоните в неотложку и описывайте симптомы по телефону!
                      </span>
                      <span style="background-color: white; color: #333333; font-family: Arial, Helvetica, sans-serif; font-size: 15px; line-height: 22.5px; white-space: pre-wrap;">
                      </span>
                      <b style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 18.4799995422363px;">
                       Один кардиолог сказал, что, разослав это сообщение как минимум 10 адресатам, можно быть уверенным, что чья-нибудь жизнь – может быть и наша – будет спасена. Мы шлём каждый день столько «мусора» по свету, что может стоит разок пустить по проводам действительно что-то полезное и нужное. ------------------------------ Поделитесь этой информацией с друзьями!
                      </b>

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://tolkointeresnoe.blogspot.com/2015/04/5.html">
                      СОЧНОЕ И ПИКАНТНОЕ ЭКСПРЕСС-МЯСО ЗА 5 МИНУТ!Причем по этому рецепту получается бесподобно вкусно любое мясо!
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-8075332946596277380" itemprop="description articleBody">
                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       Очень вкусное мясо получается если попробовать необычную пикантную заливку, способную промариновать мяско за 5 минут. Причем по этому рецепту получается бесподобно вкусно любое мясо!
                      </span>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://1.bp.blogspot.com/-2AAmpQygI_s/VT-2AARnp0I/AAAAAAAACT8/J31QK2skqtc/s1600/timthumb.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" height="253" src="http://1.bp.blogspot.com/-2AAmpQygI_s/VT-2AARnp0I/AAAAAAAACT8/J31QK2skqtc/s1600/timthumb.jpg" width="400"/>
                       </a>
                      </div>
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       <br/>
                      </span>
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       <br/>
                      </span>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Итак, на 0,5 кг мяса потребуются следующие ингредиенты для заливки:
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Горчица -1 ст.л.,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Смесь перев (черный или душистый, или душистый и белый и т.д.),
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Куриные яйца – 2 шт. средних,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Кунжут – 2 ч.л.,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Растительное масло – 1 ст.л.,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Крахмал – 1 ст.л.,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Соль – 1 ч.л,
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Красный перец чили –по вкусу и желанию для любителей остренького.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Как приготовить экспресс-заливку для мяса:
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       1. Соединить вместе горчицу, соль, крахмал, яйца куриные и перцы.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       2. Попробовать на вкус и можно добавить еще свои любимые приправы и специи или оставить так.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       3. Мясо нужно нарезать кусочками среднего или мелкого размера. Залить мясо заливкой в кастрюле так, чтобы заливка полностью покрыла все мясо. Убрать в холодильник и через 5 минут мясо можно жарить или запекать.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       4. Если убрать в холодильник на ночь мясо в заливке получится более пряным и пикантным. Мясо в такой заливке может хранится в прохладном месте несколько суток, а перед подачей на стол его можно достать и быстро приготовить.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       5. Очень вкусно готовить такое мясо жареным. Мясо выкладываем на сковородку и посыпаем кунжутом. Затем тоже самое делаем со второй стороной.
                      </div>

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://tolkointeresnoe.blogspot.com/2015/04/blog-post_21.html">
                      Характер на кончиках пальцев. Что скажут ваши ногти? Проверьте себя и друзей, так ли это?
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-4818634687985565336" itemprop="description articleBody">
                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <b>
                       <br/>
                       <br/>
                       Столетиями гадалки предсказывали судьбу по рукам. Возможно в этом все-таки что-то есть? И снова ученые решили проверить информацию дошедшую до нас из глубины веков и подтвердить, или опровергнуть народные приметы. На этот раз под исследование попала форма ногтей пальцев рук. Хотите проверить себя? Приступим!
                       <br/>
                       <br/>
                       <a href="http://interesbook.ru/wp-content/uploads/2015/04/nails.jpg">
                        <img height="226" src="http://interesbook.ru/wp-content/uploads/2015/04/nails.jpg" width="400"/>
                       </a>
                       <br/>
                       1. Длинный прямой ноготь
                       <br/>
                       <br/>
                       Такие люди обычно имеют мягкий, покладистый характер. Они талантливы и часто занимаются творческой деятельностью. Легко сходятся с другими людьми.
                       <br/>
                       2. Широкие ногти
                       <br/>
                       <br/>
                       Эти люди откровенны с другими. Не склонны делать глупости. Однако их легко вывести из себя. Они всегда готовы резать правду-матку и давать советы другим.
                       <br/>
                       3 и 4. Круглые или яйцеобразные ногти
                       <br/>
                       <br/>
                       <br/>
                       Они обычно всегда расслаблены, спокойны и живут по какому-то своему ритму жизни. Их круг общения часто ограничен старыми-добрыми друзьями с которыми комфортно и спокойно.
                       <br/>
                       5. Ногти квадратной, заостренной формы
                       <br/>
                       <br/>
                       Это лидеры. Сильные, храбрые и отважные. Эти черты характера ярко выражены и окружающие признают за ними лидерство. Но в каждом из них есть слабость, о которой известно только близким людям. Всем нам иногда нужно кому-то поплакаться, не так ли?
                       <br/>
                       6 и 7. Ногти треугольной формы
                       <br/>
                       <br/>
                       Умные, склонные к перфекционизму, новаторы и генераторы новых идей. Среди таких людей часто встречаются гении, которые находят самые необычные решения и вызывают восхищение у окружающих.
                       <br/>
                       8. Миндалевидные ногти
                       <br/>
                       <br/>
                       Преданные друзья, открытые и внешне привлекательные. Эти люди приятны в общении, но могут отстаивать свою точку зрения когда это нужно. Они легко вливаются в компанию и вызывают восхищение своими необычными методами решения проблем.
                       <br/>
                       9. Ромбовидные ногти
                       <br/>
                       <br/>
                       Это вдохновители и идейные организаторы. Они нацелены на перспективу и видят общую картину гораздо лучше остальных. Такие люди поднимают настроение, дают веру в будущее и вдохновляют окружающих на развитие и рост.
                       <br/>
                       <br/>
                       Нажмите «Поделиться» — протестируйте друзей!
                      </b>

                    <h3 class="post-title entry-title" itemprop="name">
                     <a href="http://tolkointeresnoe.blogspot.com/2015/04/blog-post_20.html">
                      Узнайте характер по длине пальцев! Кто бы мог подумать что узнать секреты характера можно всего лишь взглянув на руку?
                     </a>
                    </h3>
                    <div class="post-header">
                     <div class="post-header-line-1">
                     </div>
                    </div>
                    <div class="post-body entry-content" id="post-body-2991869984513747399" itemprop="description articleBody">
                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       Ученые из университета Кембриджа (и снова эти Британские Ученые) провели статистические исследования и выяснили интересные факты.
                      </span>
                      <br style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; margin: 0px; outline: none; padding: 0px;"/>
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       Посмотрите на свою руку и сравните длину указательного и безымянного пальцев.
                      </span>
                      <br/>
                      <div class="separator" style="clear: both; text-align: center;">
                       <a href="http://1.bp.blogspot.com/-1rw6TehvULs/VTVzPGFdA4I/AAAAAAAACSs/qqdCQMHbCwQ/s1600/142933889484gkn-520x318.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;">
                        <img border="0" height="266" src="http://1.bp.blogspot.com/-1rw6TehvULs/VTVzPGFdA4I/AAAAAAAACSs/qqdCQMHbCwQ/s1600/142933889484gkn-520x318.jpg" width="400"/>
                       </a>
                      </div>
                      <div class="separator" style="clear: both; text-align: center;">
                       <br/>
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       A) Указательный палец короче чем безымянный.
                       <br style="border: 0px none; list-style: none; margin: 0px; outline: none; padding: 0px;"/>
                       Такие люди чаще всего внешне весьма привлекательны. Они приятны в общении и харизматичны. Эти люди более решительны и склонны к рискованным поступкам. Они умны и логичны в рассуждениях. Не удивляйтесь если они хорошо справляются с кроссвордами. Еще один немаловажный статистический факт — люди с более длинным безымянным пальцем зарабатывают больше остальных. Девушкам на заметку — если на безымянном пальце нет кольца и он длинный, то…
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       B) Указательный палец длиннее безымянного
                       <br style="border: 0px none; list-style: none; margin: 0px; outline: none; padding: 0px;"/>
                       Люди у которых указательный палец длиннее безымянного часто высокомерны, самодостаточны и заносчивы. Они наслаждаются своим обществом и очень не любят когда их беспокоят по пустякам. Они не из тех людей которые сделают первый шаг навстречу. Будь то новый бизнес или отношения. Однако они ценят внимание и благосклонно воспринимают похвалу.
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       C) Указательный и безымянный пальцы одинаковой длины
                       <br style="border: 0px none; list-style: none; margin: 0px; outline: none; padding: 0px;"/>
                       Люди с такими пальцами миролюбивы, добродушны и очень не любят конфликтные ситуации. Они очень организованы по жизни и ладят со всеми. Такие люди верны в отношениях, преданы своему делу и компании в которой работают. Ну хоть к ране прикладывай
                      </div>
                      <div style="background-color: white; border: 0px none; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px; list-style: none; outline: none; padding: 1em 0px;">
                       Спеши поделиться  с друзьями, не лишай их возможности узнать это!
                      </div>
                      <span style="background-color: white; color: #333333; font-family: Georgia, serif; font-size: 18px; line-height: 27px;">
                       <br/>
                      </span>

                     <a href="http://tolkointeresnoe.blogspot.com/2015/04/14.html">
                      14 Страшных Вещей Из Прошлого, От Которых Бросает В Дрожь

                    <div class="post-body entry-content" id="post-body-8895501642957745040" itemprop="description articleBody">
                     <div dir="ltr" style="text-align: left;" trbidi="on">
                      <b>
                       <br/>
                       <br/>
                       Представляю вам подборку из 18 исторических вещей, до которых дошла наука и которые активно использовались в прошлом, но сейчас только при их виде бегут мурашки по коже, настолько уж они жуткие.
                       <br/>
                       <br/>
                       1. Мотоцикл-катафалк
                       <br/>
                       <br/>
                       <br/>
                       Выпустила такое чудо компания Norton.

                       <br/>
                       <br/>
                       <br/>
                       2. Стул с шипами, 18 век
                       <br/>
                       <br/>
                       <br/>
                       На этом стуле казнили ведьм.
                       <br/>
                       <br/>
                       <a href="http://interesbook.ru/wp-content/uploads/2015/04/%D0%BE%D1%80%D0%BF2.jpg">

                       <br/>
                       <br/>
                       <a href="http://interesbook.ru/wp-content/uploads/2015/04/%D0%BE%D1%80%D0%BF3.jpg">
                        <img height="400" src="http://interesbook.ru/wp-content/uploads/2015/04/%D0%BE%D1%80%D0%BF3.jpg" width="326"/>
                       </a>
                       <br/>
                       <br/>
                       4. Пояс верности, 16 век
                       <br/>
                       <br/>
                       <br/>
                       Вот в таких нарядах мужья вынуждены были в одиночестве проводить вечера
                       <img src="http://i1.wp.com/www.pure-t.ru/wp-includes/images/smilies/icon_smile.gif?w=750"/>

                       6. Медицинские пилы
                       <br/>
                       <br/>

                       <br/>

Так вот для того, чтобы с вашим родственником не сотворили такое ужасное надругательство, вы могли заказать ему на похороны ограждение «Mortsafe».
