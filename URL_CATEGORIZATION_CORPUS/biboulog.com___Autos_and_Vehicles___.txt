     <h2 class="entry-title" id="a005962">

       旧江戸川の、川面を滑走するボート
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年7月11日（木曜日） 19:44
      </dt>

      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E3%82%AF%E3%83%AB%E3%83%BC%E3%82%BA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="クルーズ [54]">

      </dt>

     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        大変ご無沙汰しています。
        <br/>
        なんか、こう、日々のモロモロをTwitterなんかでチョコチョコとガス抜きなんてしちゃっておりますと、まあ、ブログの更新が滞っちゃいますな。
       </p>
       <p>
        そんなTwitterだって、最初はミステリ系の情報だったはずが、フト気がつくともはやなんでもありのカオスな状態。
        <br/>
        ああ、それは当初はミステリ読書系だったはずのこのブログと状況は似てますね！
        <br/>
        ……というか、一緒ですね！
        <br/>
        かたじけない。
       </p>
       <p>
        ということで、久々の更新です。
        <br/>
        旧江戸川の妙見島にあるニューポート江戸川でボートを借りて、隅田川までお散歩クルーズしてきました。
        <br/>
        （まるで昨日のことのように書いてますが、実はもう5月下旬の話なんですが）
       </p>
       <p>
        この日は風もなく、いい天気の絶好のクルーズ日和、他に行き交う船もないとなれば、川面も鏡のように穏やかなんです。
        <br/>
        そんな中をボートで走れば、まあ、まるで教科書のように見事な滑走状態が楽しめました。
        <br/>
        これがメチャクチャ気持ちいいのなんの。
        <br/>
        アドレナリンが出まくる滑走状態を、動画に撮ってみました。
       </p>
       <p>
        いつもは早回しなんてしちゃうのですが、今回は通常速度です。
        <br/>
        ディズニーランド横の河口から、京葉線の鉄橋を超えたあたりで「気持ちいい！」と撮り始めています。
        <br/>
        そこから妙見島手前の船だまりあたりでスピードを落とすまでの3分弱のトリップ体験。
        <br/>
        なるべくフルスクリーンで、一緒にボートに乗ってるような気持ちでどうぞ。
        <br/>
        <iframe frameborder="0" height="360" src="//www.youtube.com/embed/dipgCILcdE4" title="ニューポート江戸川を目指して、旧江戸川をさかのぼる " width="640">
         ニューポート江戸川を目指して、旧江戸川をさかのぼる
        </iframe>
       </p>

     <h2 class="entry-title" id="a005961">
      <a href="http://biboulog.com/archives/2013/04/17-232029.php" rel="bookmark">
       供養に参加した大手町ビルヂング屋上の観音さま
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年4月17日（水曜日） 23:20
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_27.php" title="主宰人の日常的などーでもいい話題のあれこれ">
        日常
       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E5%A4%A7%E6%89%8B%E7%94%BA%E3%83%BB%E4%B8%B8%E3%81%AE%E5%86%85&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="大手町・丸の内 [41]">
        大手町・丸の内
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E5%BB%BA%E7%89%A9&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="建物 [18]">
        建物
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        毎年、この時期になるとソワソワしてしまう、大手町ビルヂングの屋上開放デー。
        <br/>
        なんでしょうね、屋上というともう昭和のデパートの遊園地の印象があるからか、なんだかワクワクしてしまうのですね。
       </p>
       <p>
        ……って、いえいえ、そんな浮かれポンチなお遊びではないのです。
        <br/>
        大手町ビルヂングの屋上に安置されていて、毎年4月17日（土日の場合はその前後）のみご開帳される観音さまがあるのですよ。
        <br/>
        その観音さまに、今年もお参りをしてきました。
       </p>
       <p>
        この観音さまについては、もうすでに何回も毎年書いてますが、またしても今年も書いちゃいますと……。
       </p>
       <p>
        大手町ビルが竣工したのは昭和33年なのですが、その頃はまだ戦後10年が過ぎたあたりで、まだ丸の内も寂れていたそうです。
        <br/>
        そこに、3万3千坪という当時としては最大規模のビルヂングを建てることで、ビジネス街を復興させようとした当時の三菱地所社長の思いに、同郷の平櫛田中が同感、5年の歳月をかけて昭和42年に観音さまを完成させたのだそうです。
       </p>
       <p>
        そんな毎年行われる供養行事ですが、通常は先に関係者だけで供養をとり行い、その後に一般の参詣が行われるのですね。
        <br/>
        それを、今年はビル管理会社にダメ元でお願いしてみたら……えへへ、やったね。
        <br/>
        なんと関係者に混じって参加してもよろしい、とのお許しをいただいちゃいました。
       </p>
       <p>
        そんな訳で供養がとり行われる時間のちょっと前、屋上に上がってみることにしました。
        <br/>
        ビル最上階のフロアに上がっていくと、普段は決して行くことのできない屋上への階段に、今年も案内板が出ていますよ。
        <br/>
        <img alt="屋上へのいざない" height="400" src="http://biboulog.com/archives/images/2013/04/20130417-001.jpg" width="600"/>
       </p>
       <p>
        普段は使われていない階段を上って屋上に出てみると……おおう。
        <br/>
        まだ設営の準備中だったのでした。
        <br/>
        強風が吹き荒れていて、なかなかうまく設営ができないようです。
        <br/>
        <img alt="まだ会場は設営の準備中でした" height="400" src="http://biboulog.com/archives/images/2013/04/20130417-002.jpg" width="600"/>
       </p>
       <p>
        その間、思う存分屋上を堪能させてもらっていたのでした。
        <br/>
        「昼休みに屋上でバレーをするOL」を妄想してみたり、普段は囲いがあって見ることができない工事中のお隣のビルを上から覗いてみたり。
        <br/>
        そして撮った写真をミニチュア写真風に加工してみたり（我ながら、よくぞ毎年飽きずにやってるな……）。
        <br/>
        <img alt="屋上から、工事中の隣のビルを覗いてみたところ" height="600" src="http://biboulog.com/archives/images/2013/04/20130417-003.jpg" width="400"/>
       </p>
       <p>
        ここから丸の内方面を眺めてみると、面白いことに気付かされます。
        <br/>
        仲通り沿いには再開発された高層ビルが続々と建っているのですが、三菱地所のビルだけが“二段重ね”のような形になっているのですね。
        <br/>
        それも、どのビルも同じ高さなんです。
        <br/>
        <img alt="丸の内の仲通り沿いに建ち並ぶ「百尺規制」時代の名残のあるビル" height="600" src="http://biboulog.com/archives/images/2013/04/20130417-004.jpg" width="400"/>
        <br/>
        これは、大昔に「百尺規制」というルールがあって、31m以上の高さを超えるビルは建ててはいけなかったことがあったんです。
        <br/>
        そのため、かつてはどのビルも、この高さに揃っていたそうなんですよ。
        <br/>
        実は大手町ビル、古き良き時代のこの高さのままなので、必然的に「百尺規制の名残」が目の前に揃っている感じに見えるのですね。
       </p>
       <p>
        そして、フト足元を見てみると、そこにあるのはバッテン印。
        <br/>
        これはいったい何なのでしょうか……。
        <br/>
        <img alt="謎のバッテン" height="600" src="http://biboulog.com/archives/images/2013/04/20130417-005.jpg" width="400"/>
        <br/>
        場所的には、ちょうど丸の内仲通りを延長してきた線上にあたる場所のようなのですが……なんか怖いよ。
       </p>
       <p>
        そうこうしているうちに強風のなか、設営も完了し、15分遅れで供養が始まりました。
        <br/>
        <img alt="" height="" src="http://biboulog.com/archives/images/2013/04/20130417-006.jpg" width=""/>
        <br/>
        なーむー……とお経を読むお坊さん、供養が終わってからのお話の際に築地本願寺から来られたことをおっしゃってました。
        <br/>
        観音菩薩ということで、菩薩とはなんぞやというありがたいお話に、フムフム。
       </p>
       <p>
        その後、出席者によってお参りをして解散となります。
        <br/>
        屋上に向かう階段を見てみると、一般参詣の待ち行列がズラリと並んでいるのでした。
        <br/>
        <img alt="" height="400" src="http://biboulog.com/archives/images/2013/04/20130417-007.jpg" width="600"/>
        <br/>
        声の反響具合からすると、屋上からかなり下のフロアにまで行列は伸びているようです。
        <br/>
        どうも去年辺りから、なぜかお参りする人が増えてきたような気がします。
        <br/>
        大手町観音の存在が、有名になってきたのかしら。
       </p>
       <p>
        そして！ またしても！ やったね！
        <br/>
        今年もまた、榮太郎本舗の紅白まんじゅうをいただきました！
        <br/>
        大切にオフィスまで持ち帰ってきて、身も心も大満足の供養だったのでした。
        <br/>
        <img alt="榮太郎本舗の紅白まんじゅう" height="400" src="http://biboulog.com/archives/images/2013/04/20130417-008.jpg" width="600"/>
       </p>

     <h2 class="entry-title" id="a005959">
      <a href="http://biboulog.com/archives/2013/02/26-221154.php" rel="bookmark">
       多摩川から眺める羽田飛行機見学クルーズ
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年2月26日（火曜日） 22:11
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_42.php" title="船に乗ってのなんやかんやの話題あれこれ">
        クルーズ
       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E3%82%AF%E3%83%AB%E3%83%BC%E3%82%BA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="クルーズ [54]">
        クルーズ
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E7%A4%BE%E4%BC%9A%E8%A6%8B%E5%AD%A6&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="社会見学 [17]">
        社会見学
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E7%A9%BA%E6%B8%AF&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="空港 [8]">
        空港
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        海から飛行機を見るため、羽田空港沖へ船で行こうとした場合、
       </p>
       <ol>
        <li>
         海上を進む「表道」のルート
        </li>
        <li>
         京浜運河を進む「裏道」のルート
        </li>
       </ol>
       <p>
        の2通りがあるのです。
       </p>
       <p>
        この2番の裏道ルート「京浜運河」は、その名のとおり、本来なら東京から横浜までつながる長ーい運河なんですね。
        <br/>
        ところがこの運河、羽田空港あたりに大きく広がる浅瀬に阻まれて、なかなかの難所になってしまっているんです。
        <br/>
        （レンタルボートだと航行禁止エリアになっているほど）
        <br/>
        でも「行ってはいけません」と言われると、余計に気になって仕方ありません。
       </p>
       <p>
        そんな訳でずっと「どんなところなんだろう」という思いでいっぱいだったのですが、やったね。
        <br/>
        ついにその疑問を解き明かせるときがやってきたのでした。
       </p>
       <p>
        なんと
        <a href="http://www.zeal.ne.jp/plan/134.html" target="_blank">
         「多摩川から眺める羽田飛行機見学クルーズ」
        </a>
        というクルーズが開催されたのです。
        <br/>
        羽田空港沖に飛行機を見に行くクルーズはたくさんあるのですが、このクルーズの目玉は、“「裏道」ルート”の先にある浅瀬エリアという、プロでも嫌がる難所を敢えて行くというものなんですね。
        <br/>
        通常ではありえないコースなんですよ！
        <br/>
        これはもう、願ってもない大チャンスなんですよ！
        <br/>
        募集されていると知るやいなや、即、申し込みしちゃいましたよ！
       </p>
       <p>
        というわけで、2月24日の日曜日、天王洲から歩いて5分のところにある桟橋にやってきました。
        <br/>
        今日のクルーズはこの船で行くのです！
        <br/>
        <img alt="今日のクルーズはこちらの船で行きます" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-001.jpg" width="600"/>
       </p>
       <p>
        40人乗りということで、いったい何人の参加者がいるのかとドキドキしていたら……おおう。
        <br/>
        ぼくを含めても5人だけなんですよ！
        <br/>
        ほぼ貸切状態で、自由に撮影で船の上をウロウロとさせて頂きました。
       </p>
       <p>
        京浜運河を進むこと20分、いつも入ったことがない禁断の羽田空港裏のエリアに入ると……ああ、こりゃスゴイわ。
        <br/>
        広大な水辺が広がっているようなんですが、真ん中辺りに水鳥がたくさんかたまっているところがあって、よくよく見ると、浅瀬が見えているんです。
        <br/>
        <img alt="羽田空港の裏にある危険な浅瀬エリア" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-002.jpg" width="600"/>
        <br/>
        広さからすると埋立地がもう1つ入りそうなほど広大な水辺なのですが、真ん中のほとんどが浅瀬で、実際に航行できるのは隅っこだけという何だか非常にもったいない場所なんですね。
       </p>
       <p>
        広大な浅瀬のある難所を過ぎると……おお、見えてきました。
        <br/>
        羽田可動橋です。
        <br/>
        <img alt="海側から見る羽田可動橋" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-003.jpg" width="600"/>
        <br/>
        首都高速の入口として作られたものの、わずか4年しか使用されていない、幻の建築物ですよ。
        <br/>
        ただ、またいつでも使えるようにという考えがあるのか（例えば非常時の際とか）、決して廃棄されたという印象はなく、ピカピカに保存されていて、ちゃんと整備されているようなのです。
       </p>
       <p>
        <a href="http://biboulog.com/archives/2010/11/14-005745.php">
         以前、陸からこの可動橋を見に行ったとき
        </a>
        は、橋がバラバラの方角を向いているような感じだったのですが、こうして海側から眺めてみると……おおう。
        <br/>
        ちゃんとホラ、こちらに向かって橋が開いているのですよね。
        <br/>
        <img alt="海側から見ると、ちゃんとこちらに向かって開いているように見えます" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-004.jpg" width="600"/>
       </p>
       <p>
        そして、ここからは京浜運河ではなく「海老取川」に入っていくことになります。
        <br/>
        しばらくは京浜運河並みに穏やかな広々とした川が続いているのですが、それは突然にやってきます。
        <br/>
        「いきなり川幅が狭っ！」「そして、橋が低っ！」
        <br/>
        <img alt="川幅が狭いだけでなく、橋も低い" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-005.jpg" width="600"/>
       </p>
       <p>
        まあ漁船やプレジャーボートも多く係留されているので、行けないことはないはずなんですが、それでもどこに浅瀬があるのかよく判らない恐怖感が、もうヒシヒシと。
        <br/>
        これはもう完全に「どこに浅瀬があるか」土地勘がある人でないと、入ってはいけないデインジャラスゾーンなんですね。
       </p>
       <p>
        「浅瀬があるなら、満潮時に行けばいいじゃない」と思われそうですが、いえいえ。
橋が低すぎて、特に大潮のときは満潮になると橋をくぐることができないそうなんですよ。
        <br/>
        ということは、このあたりに船を係留している人たちは、時間配分を間違えると戻れなくなっちゃうこともあるのかと……。
        <br/>
        やっぱりデインジャラス～。
       </p>
       <p>
        天空橋駅前には新しい桟橋ができていましたが、こんな狭くて浅くて低い難所で観光船の拠点として活用するには、ちょっと面倒かも。
        <br/>
        <img alt="新しくできた天空橋桟橋" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-006.jpg" width="600"/>
       </p>
       <p>
        いよいよ海老取川を超え、多摩川に入るあたりで緊張感MAXの最高潮。
        <br/>
        目の前に広々とした多摩川が見えることで、ホッと気を緩めてしまいそうになるのですが……いえいえ。
        <br/>
        それでなくても狭くなっているカーブの内側に、黒々と浅瀬が姿を見せていて
        <strong>
         <big>
          キャー、コワイ。
         </big>
        </strong>
        <br/>
        <img alt="海老取川の最後の難所。その隣には羽田の大鳥居" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-007.jpg" width="600"/>
        <br/>
        ちなみに、この海老取川を抜けるあたりの羽田空港側には、長年移設できず駐車場の真ん中にぽつんと鎮座していた大鳥居があります。
       </p>
       <p>
        こんな恐ろしい海老取川を抜けて、ようやく広々とした多摩川に出たとしても、まだ油断はできません。
        <br/>
        というか、多摩川こそが魔のルートなんですね。
        <br/>
        <img alt="多摩川も浅すぎて、航路がきっちり定められています" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-008.jpg" width="600"/>
        <br/>
        水面から伸びているさお竹があるのですが、実はこれが航路を指しているんです。
        <br/>
        この外側は航行不可能な浅瀬が広がっていて、ちょっとでも油断してさお竹の外に出てしまうと、あっという間に浅瀬に乗り上げてしまいます。
        <br/>
        最悪、座礁して身動きが取れなくなっちゃったり。
        <br/>
        なので多摩川に出ても、しばらくは目を凝らして、さお竹に沿って進まなければなりません。
        <br/>
        この難所を思えば、旧江戸川の河口に広がる三枚州なんて、目じゃないですよね。
       </p>
       <p>
        多摩川に出ると、あとはひたすら羽田空港を目指します。
        <br/>
        途中で、こんな川なのに警察の船が走ってきてるなー、お巡りさんが船から身を乗り出して見回りしてるなーと思っていたら……その理由は、あとで判ることになります。
        <br/>
        <img alt="お巡りさんが身を乗り出して、警察船からパトロール中" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-009.jpg" width="600"/>
       </p>
       <p>
        多摩川の河口付近まで下ってくると、ちょうど「A滑走路」がよく見えてくるポイントに出てきます。
        <br/>
        <img alt="A滑走路に着陸する飛行機" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-010.jpg" width="600"/>
       </p>
       <p>
        「A滑走路」への着陸は、ちょうど多摩川を横切る形で行われるので、川に向かって伸びている誘導灯の先にいると、ちょうど着陸する飛行機が頭上を通過していくポイントになります。
        <br/>
        <img alt="頭上を通り過ぎていく飛行機" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-011.jpg" width="600"/>
       </p>
       <p>
        時間帯がよかったのか、それとも羽田空港が過密ラッシュだからなのか、頭上を次から次へとブンブンゴーゴーと各社の飛行機が通過して行きます。
        <br/>
        あと、「D滑走路」から離陸をする準備をしているする飛行機の姿もよく見えますね。
        <br/>
        ポケモンジェットも、まだあったんですね！（←失礼なヤツ）
        <br/>
        <img alt="ポケモンジェットが離陸準備中" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-012.jpg" width="600"/>
       </p>
       <p>
        こうして河口付近でゆったりしていると、船長さんが「政府専用機ですよ！」。
        <br/>
        「え？ どこどこ」と指差す方向を見てみると……おおう。
尾翼の赤丸が一瞬、JALの鶴丸マークにも見えるのですが、いえいえ、ちゃんと日の丸なんですよ。
        <br/>
        <img alt="アメリカから帰国する安倍総理を乗せた政府専用機" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-013.jpg" width="600"/>
        <br/>
        多摩川側とは逆の海側にある「C滑走路」に着陸しようとしているのか、やや遠いのですが、確かに尾翼の日の丸がよく見えます。
        <br/>
        アップにすると、機体に描かれた「日本国」の文字もちゃんと見えていますね。
        <br/>
        <img alt="政府専用機の機体には、ちゃんと「日本国」の文字が" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-014.jpg" width="600"/>
       </p>
       <p>
        安倍総理がアメリカに行っているとニュースでも報じられているので、ちょうど帰国しているところに出くわしたのでしょうか。
        <br/>
        <big>
         <strong>
          ああ、なるほど！
         </strong>
        </big>
        <br/>
        先ほどの警察の船は、安倍総理が空港を利用するということで、テロ対策の一環によるパトロールだったのですね！
        <br/>
        <img alt="安倍総理が帰国するということでのテロ対策の見周り？" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-009.jpg" width="600"/>
       </p>
       <p>
        そして船は新しくできた「D滑走路」の傍にも寄ってくれます。
        <br/>
        <img alt="D滑走路の下に注目" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-015.jpg" width="600"/>
        このD滑走路は、海を埋め立てたわけではなく、海上に多数の柱を立てて、その上に設置されたものなんですよね。
        <br/>
        その柱が、これ。
        <br/>
        <img alt="D滑走路の下には芸術的に並んでいる柱、柱、柱" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-016.jpg" width="600"/>
        <br/>
        もうね、なんて芸術的なのかと、ため息が出るばかりなんですよね。
        <br/>
        何しろ、どの方向から見ても、柱がまっすぐに奥まで続いているんです。
        <br/>
        真正面から見てももちろん、斜めから見ても、どの角度から見ても、見る方向に柱がまっすぐ……。
        <br/>
        もうこの柱が並んでいるところを見ているだけでも、丼にご飯が3杯はいけます（←ムリです）。
       </p>
       <p>
        ここで来た道を引き返すわけですが、なぜか帰り道、カモメがしばらく一緒に並んで、お見送りしてくれたのでした。
        <br/>
        かわいいのう。
        <br/>
        <img alt="しばらく船と並走していたカモメ" height="400" src="http://biboulog.com/archives/images/2013/02/20130224-017.jpg" width="600"/>
       </p>
       <p>
        ということで、今回の「多摩川から眺める羽田飛行機見学クルーズ」、2時間30分のルートは、このようなものになりました。
        <br/>
        <img alt="多摩川から眺める羽田飛行機見学クルーズ" height="600" src="http://biboulog.com/archives/images/2013/02/20130224-018.jpg" width="600"/>
       </p>
       <p>
        また、その他の写真はいつものようにこちらにアップしています。
        <br/>
        お気軽にご覧ください。
       </p>

      <a href="http://biboulog.com/archives/2013/01/28-120747.php" rel="bookmark">
       久しぶりに横浜港で極私的「工場クルーズ」
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月28日（月曜日） 12:07
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_41.php" title="小型船舶に関する話題のあれこれ">
        ボート
       </a>
       ,
       <a href="http://biboulog.com/archives/cat_42.php" title="船に乗ってのなんやかんやの話題あれこれ">

       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E3%82%AF%E3%83%AB%E3%83%BC%E3%82%BA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="クルーズ [54]">
        クルーズ
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E5%B7%A5%E5%A0%B4&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="工場 [18]">
        工場
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E6%A8%AA%E6%B5%9C&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="横浜 [71]">
        横浜
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        昼頃、モゾモゾと目を覚ますと、なんだか窓の外が眩しいのです。
        <br/>
        カーテンの向こうを見てみたら……おおう。
        <br/>
        なんといういい天気。絶好のお外日和なんですよ。
        <br/>
        しかも、風もまったく吹いてないのか、横浜港の風車がまったく回っていません。
        <br/>
        海の様子も……凪っぽい感じ？
       </p>
       <p>
        これはチャンス。
       </p>
       <p>
        久々にボートに乗ってみたい衝動に駆られちゃいました。
        <br/>
        いつもお世話になっている東神奈川のUYフラッグさんに電話して訊いてみます。
        <br/>
        すると！ なんと！ やったね！
        <br/>
        今日はどのボートも空いてるからウェルカム、というではありませんか。
       </p>
       <p>
        ということで！
        <br/>
        久々にやってきました、横浜港！極私的工場クルーズ！ヒャッホウ！
        <br/>
        お伺いした時間が遅かったので、借りられる時間は2時間しかありませんが、行けるとこまで行ってみましょう！
        <br/>
        （こんなことならもうちょっと早くに起きるべきでした……）
       </p>
       <p>
        瑞穂埠頭の根元にあるUYフラッグさんを出航し、まず見えてくるのはこれ、バンザイクレーンなんですね。
        <br/>
        <img alt="瑞穂埠頭の根元にある小さなバンザイクレーン" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-001.jpg" width="600"/>
        <br/>
        「バンザーイ」というよりも、両手を振り上げて「いってらっしゃーい！」とお見送りされているようなものでしょうかねえ。
        <br/>
        いってきまーす！
       </p>
       <p>
        その先にあるのが、ぼくが大好きなバナナ1号2号。
        <br/>
        まるで「ロンドンブーツ1号2号」みたいなこの倉庫は、名前どおり、バナナ倉庫なんだそうです。
        <br/>
        たまにバナナの甘い匂いも漂ってくることもあるそうなんですよ。
        <br/>
        <img alt="バナナ2号はブルーです。バナナ色じゃないのが残念" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-002.jpg" width="600"/>
        <br/>
        1号の方が、見事にキュートなバナナ色の倉庫なのですが、残念ながら船が留まっていてよく見えません。
        <br/>
        仕方ないので、なぜかブルーに染め上げられた2号のみ見て行きましょう。
       </p>
       <p>
        意外と知られてない昭和電工のボーキサイト置き場。
        <br/>
        <img alt="ボーキサイト置き場にある双子のバケットクレーン" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-003.jpg" width="600"/>
        <br/>
        この工場では長らく、アルミの原料となる「アルミナ」というのを造っているそうですが、環境問題から2015年までには撤退するそうです。
        <br/>
        そうなると、この光景も見られなくなると思うのですね。
        <br/>
        この双子のバケットクレーンたちも、いずれは撤去されてしまうのでしょうか。
        <br/>
        こうして見られる今のうちに、ガンガン見ておかないといけませんね。
        <br/>
        ぼくの大好きな東亜石油の京浜製油所扇町工場も撤退しちゃったし、京浜工業地帯で見どころがドンドンなくなるのは寂しい限りです。
       </p>
       <p>
        ここから一度京浜運河に出てみます。
        <br/>
        鶴見川の河口には、東京電力の火力発電所があるのですね。
        <br/>
        <img alt="双子の煙突が特徴の、東京電力火力発電所" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-009.jpg" width="600"/>
        <br/>
        例の原発問題で、今はフル稼働しているところだと思います。
        <br/>
        奥に双子煙突があるのですが、震災前はこの煙突がとてもカラフルにライトアップされていたものです。
        <br/>
        それが地震以降、すっかり暗くなってしまって、横浜の夜景が寂しいものになってしまっています。
        <br/>
        （震災の影響を考えると、そんなことも言ってられないとは思うので仕方ないといえば仕方ないのですが）
       </p>
       <p>
        鶴見川の向こうには、かなりの規模の造船所「ジャパンマリンユナイテッド」があります。
        <br/>
        しかしこの会社、ちょっと前まではユニバーサル造船という名前だったはずなのに、いつの間にか変わってるんですね。
        <br/>
        そのさらに前には「日本鋼管」（と日立造船の合併）だったし、もう覚えきれません。
       </p>
       <p>
        このジャパンマリンユナイテッドでは、民間の船よりも海上保安庁の巡視艇や海上自衛隊の護衛艦がメンテナンスされているところをよく見かけます。
        <br/>
        海上保安庁の巡視艇が3隻、停泊中のところ。
        <br/>
        <img alt="造船所には、海上保安庁の巡視艇が3隻停泊中" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-004.jpg" width="600"/>
       </p>
       <p>
        この船なんて、かなり艦橋や甲板がモノモノしいことになっています。
ひょっとして新造船？……と調べてみたら「補給艦（艦名ときわ）」というものだそうです。
        <br/>
        船首に番号が付いていると、調べやすくていいですね。
        <br/>
        <img alt="何やら艦橋が物々しい状態" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-005.jpg" width="600"/>
       </p>
       <p>
        またこの造船所では浮きドックが海上に設置してあるんですね。
        <br/>
        このなかを覗き込むのも楽しみのひとつなんです。
        <br/>
        <img alt="浮きドックのなかでは、まさにメンテナンスの真っ最中" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-006.jpg" width="600"/>
       </p>
       <p>
        浮きドックのなかでは、まさに掃海艇の「のとじま」がメンテナンスの真っ最中でした。
        <br/>
        お仕事、ご苦労さまです！
        <br/>
        <img alt="メンテナンス真っ最中の掃海艇「のとじま」" height="600" src="http://biboulog.com/archives/images/2013/01/20130127-007.jpg" width="400"/>
       </p>
       <p>
        このジャパンマリンユナイテッド（旧ユニバーサル造船）と運河を挟んだ向かいには、旭硝子の工場もあるんですよね。
        <br/>
        このような看板を掲げて、とてもウェルカムしてくれてます……が、何をする課なのか、もひとつ判ってません。
        <br/>
        <img alt="非常にウェルカムな課のようですが……何をする部署なのか、もひとつ判りません" height="600" src="http://biboulog.com/archives/images/2013/01/20130127-008.jpg" width="400"/>
       </p>
       <p>
        ふたたび京浜運河に出て、しばらく行くと見えてくるのは……カッコイイ！
        <br/>
        日清製粉のバンザイクレーンです。
        <br/>
        <img alt="日清製粉の巨大なバンザイクレーン" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-013.jpg" width="600"/>
        <br/>
        さっきのバンザイクレーンとは違って、こちらは倉庫の規模が大きいからか、バンザイする腕もかなりデカイ。
        <br/>
        なんというか、さっきの「いってらっしゃーい」みたいなカワイイものではないのですね。
        <br/>
        全米アームレスリング大会で、優勝した選手が「I'm the Champioooooon!」などと雄叫びを上げているかのような、そんな迫力が感じられます。
       </p>
       <p>
        そのバンザイクレーンの向こうにあるのが田辺運河ですね。
        <br/>
        そこに入ろうとした所で「おお！」。
        <br/>
        全長が200m以上はあろうかと思われるメチャクチャ巨大な貨物船が、運河の角のところにある三井埠頭にちょうど接岸しようとしているところだったんですよ！
        <br/>
        “すでに停泊している巨大な船”というのはいつでも見ることができるのですが、こうして接岸するところなんて、なかなか見られるものではないですからねー。
        <br/>
        もうずっと貨物船のお尻あたりに陣取って、眺めていたんです。
       </p>
       <p>
        そうしたら、ひとつの発見が。
        <br/>
        こうした巨大船でも、停泊しているときはちゃんと岸にロープを張って係留しているのですが、あのロープって、どうやって繋ぐのかなーと思っていたんです。
        <br/>
        そうしたら、その瞬間を見ることができましたよ！ やったね！
        <br/>
        こうして小さな船でロープを船から受け取って、それを岸壁まで運んでいたのですね！
        <br/>
        <img alt="係留ロープは小型船で岸壁に運びます" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-011.jpg" width="600"/>
        <br/>
        わずか1隻の船が接岸するだけなのに、もうそれだけでも何十人もの人が働く現場、何だかステキです。
       </p>
       <p>
        田辺運河の入口に2機あるイグアナクレーンも、日曜日だというのに1機が稼働中です。
        <br/>
        <img alt="イグアナクレーン、おシゴト中" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-010.jpg" width="600"/>
        <br/>
        仕事していない“頭を上げているとき”は確かにイグアナなんですが、おシゴトするために頭を下げたら……ああ、やっぱりイグアナだ。
       </p>
       <p>
        この運河を奥に入っていくと、そこにそびえ立つのが、これ。
        <br/>
        昭和電工の要塞のようなプラントですね。
        <br/>
        <img alt="ラスボス感満載の風格、昭和電工の要塞のようなプラント" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-012.jpg" width="600"/>
        <br/>
        どうですか、この重厚感。
        <br/>
        背後からもスチームがもうもうと立ち込めていて、ラストボスの風格がありすぎです。
        <br/>
        これを見るだけでも、工場クルーズの価値があるというものですね。
       </p>
       <p>
        本当はこのさらに向こうにある東亜石油のタワーや、東燃化学の煙突から吹き出すフレアスタックも見に行きたかったのですが……あうあう、風が強くなってきたので断念、引き返すことにしました。
       </p>
       <p>
        といっても、まだ時間もあるので、ちょっとだけ横浜港内をクルージング。
        <br/>
        すると、何やら見慣れないものがあるんです。
        <br/>
        緑色の帆船みたいなもの……？アレはいったい何だろう？と近寄ってみると……おお！
        <br/>
        <img alt="マストと舳先に作業用ネットが張られて、何やら帆船みたいに見えた氷川丸" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-014.jpg" width="600"/>
        <br/>
        なんと、氷川丸なのでした！
        <br/>
        ちょうど海の方に向かっている舳先を、これは塗料の塗り直しでもしているのでしょうかねえ。
        <br/>
        マストにも緑色の作業用ネットが掛けられているその姿が、沖合から見たら見慣れない緑色の帆船のように見えたのでした。
       </p>
       <p>
        こうして無事に瑞穂埠頭のUYフラッグさんまで戻ってきたのですが、ここでまたしばらく長っ尻しちゃいました。
        <br/>
        おかみさんの入れてくれるお茶をいただきながらベラベラ話しているうちに、外はもう夕暮れの景色に。
        <br/>
        昭和初期に建てられた三井倉庫のカッコイイクレーンもこんなシルエットに染まっていたのでした。
        <br/>
        <img alt="UYフラッグ裏にある三井倉庫の建物も夕暮れの中に" height="400" src="http://biboulog.com/archives/images/2013/01/20130127-015.jpg" width="600"/>
       </p>
       <p>
        ということで今回、2時間で約25kmの航跡と、各所で撮りまくった写真は以下でご覧いただけます。
        <br/>
        お気軽にどうぞ～。
        <br/>

        </iframe>
        <br/>
        <small>
         より大きな地図で
         <a href="https://maps.google.co.jp/maps/ms?msid=213661552277195641245.0004d45013f37600a41a1&amp;msa=0&amp;brcurrent=3,0x60185defe614082f:0xffe94e0a864405f3,0&amp;ie=UTF8&amp;t=m&amp;ll=35.476608,139.679832&amp;spn=0.069896,0.102825&amp;z=13&amp;source=embed" style="color:#0000FF;text-align:left">

     <h2 class="entry-title" id="a005957">
      <a href="http://biboulog.com/archives/2013/01/15-235837.php" rel="bookmark">
       大雪の宴のあと
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月15日（火曜日） 23:58
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_27.php" title="主宰人の日常的などーでもいい話題のあれこれ">
        日常
       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E9%9B%AA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="雪 [6]">
        雪
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        雪が積もった日はテンション高く「やったぜ、やったぜ、ヤッホー、ヤッホー」とはしゃぎまくっていたのに……。
        <br/>
        その次の日が大変だということを忘れていました。
        <br/>
        考えてみたら、前回横浜でも大雪が降ったのはもう7年も前とのこと。
そりゃすっかり後の日の大変なことなんて、忘れてしまってますよ。
       </p>
       <p>
        というわけで、今回の大雪……というには多すぎるドカ雪騒動。
        <br/>
        大はしゃぎだった祝日の月曜日から一夜明けた次の日の朝のことです。
        <br/>
        案の定、道路上には溶けかけた雪がすっかり凍結していて、ガッチンゴッチンになっていましたよ。
        <br/>
        <img alt="大雪から一夜明けた道路は凍結していてガッチンゴッチンに" height="600" src="http://biboulog.com/archives/images/2013/01/20130115-001.jpg" width="450"/>
       </p>
       <p>
        ガッチンゴッチンに凍結していて、まるであざけり笑うかのように朝日をテカテカ照り返す、ツルッツルの坂道や階段も目前にして「どうしよう……こんなところ歩けるかよ……」と、途方に暮れていたんですが、そうも言ってられません。
        <br/>
        サラリーマン、OL、学生たちが恐る恐る歩いている姿があちこちで見られました。
        <br/>
        何しろ横浜なんてちょっとした住宅街も丘の上にあったりして、坂の街ですから、もう大変です。
       </p>
       <p>
        ぼくも意を決して、果敢にも一歩を踏み出したのですが……おおっっと！。
        <br/>
        傾斜となっている道路上で、見事に足を滑らせてしまい、転倒、お尻から着地するというハプニングをやっちましました。
       </p>
       <p>
        が、しかし。痛くないんです。
        <br/>
        キレイに足を前に揃えて、マンガみたいにお尻から着地したため、お肉がエアバックのような役割を果たしたのでしょうね。
        <br/>
        我がお尻とは生まれて依頼の長年の付き合いがあるのですが、全然気づいていませんでした。「お尻には、ちゃんと優れた機能性がある」のですね。
       </p>
       <p>
        そんな大騒ぎだった休み明けの火曜日だったのですが、お昼には、昨日の大雪がまるでウソのように太陽がガンガン照りつけるいいお天気になりました。
        <br/>
        これはもう雪もすっかり溶けて、帰りは楽チンだろうな、と思っていたのですが……甘すぎました。
        <br/>
        中途半端に溶けた分が、夜の気温低下でまた凍結してしまい、場所によっては相変わらずガチンゴチンのままなんです。
        <br/>
        <img alt="夜になるとまた凍結して、さらに強度が増したようです" height="450" src="http://biboulog.com/archives/images/2013/01/20130115-002.jpg" width="600"/>
        <br/>
        こうなってしまうと、また明日の朝が大変そう……。
       </p>
       <p>
        そして、真の悲劇はウチのあるマンションの管理人さんに襲いかかっていたのでした。
        <br/>
        エントランスには、このような管理人さんからの魂の叫びが……。
        <br/>
        <img alt="大雪のため、ゴミ回収が来ませんでした" height="600" src="http://biboulog.com/archives/images/2013/01/20130115-003.jpg" width="450"/>
        <br/>
        月曜日のゴミ回収が大雪で来られなかったのは判るのですが、なんと火曜日も回収されずに残ったままなのだそうです。
        <br/>
        「火曜日
        <strong>
         “も”
        </strong>
        来ていません」というあたり、管理人さんの悲哀を感じさせられます。
        <br/>
        ゴミ集積場が満杯ということで、これはもう非常事態宣言ものなんですよ……。
        <br/>
        横浜市、危うし。
       </p>

     <h2 class="entry-title" id="a005956">
      <a href="http://biboulog.com/archives/2013/01/14-224259.php" rel="bookmark">
       足首まで埋もれる大雪でした
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月14日（月曜日） 22:42
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_27.php" title="主宰人の日常的などーでもいい話題のあれこれ">
        日常
       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E9%9B%AA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="雪 [6]">
        雪
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        もともと天気予報では、今日は大荒れの天気でしょう、とは言われていたんです。
        <br/>
        だから何の予定も入れず、昼過ぎまでゆっくり寝ていたんです。
       </p>
       <p>
        で、起きてときに「なんだか今日は寒いなー」と窓の外を見てみたら、あらビックリ。
        <br/>
        もう空から地面から、目の前一面が真っ白なんです。
        <br/>
        <img alt="昼過ぎに起きたら、目の前一面が真っ白なことに" height="400" src="http://biboulog.com/archives/images/2013/01/20130114-010.jpg" width="600"/>
       </p>
       <p>
        これはもう家で寝てる場合じゃないよ！ 外に出ないとせっかくの雪の話題に乗り遅れちゃうよ！
        <br/>
        そう思って、ご飯も食べず外に出ようとしたら……うわお。メチャクチャ危ないところでした。
       </p>
       <p>
        ベランダから見たマンション南側は、雪がシンシンと降り積もるだけの静かな銀世界だったのですよね。
        <br/>
        ところが、玄関のある北側はもうエライことになっているんです。
        <br/>
        暴風が吹き荒れていて、雪が横殴りで叩きつけてきて、かなりのブリザード。
        <br/>
        マンションの窓や壁にも雪がびっしりとこびりついていて、もうどこの雪山なのかと。
        <br/>
        <img alt="北側は暴風雪となっていて、下手すると遭難するような事態に" height="400" src="http://biboulog.com/archives/images/2013/01/20130114-011.jpg" width="600"/>
        <br/>
        これはアカン雪や、外に出たら遭難する雪や……と慌てて引き返したのですが、うーん……。
        <br/>
        やっぱし外に出たほうが楽しそうなんですよ。
        <br/>
        もうね、意を決して、厳重に暖かい格好で身を包んで、出て行きましたよ。
       </p>
       <p>
        マンションの出入り口では管理人さんが雪かきしてくれています。
        <br/>
        「こんにちは。エライことになっちゃってますねー」と挨拶すると、管理人さん「雪でゴミの回収がまだ来てないんですよ」。
        <br/>
        嘆いてます。なんでも、ゴミの回収が来ないことには帰ることができないんだとか。
       </p>
       <p>
        まあ今日の回収は、生モノもある家庭ゴミの日じゃないから、まだ不幸中の幸いといったところでしょうか。
        <br/>
        そう思いながら道路に出てみると、おおう。
        <br/>
        ゴミ回収車が立ち往生してました。
        <br/>
        <img alt="道路ではいろんな車が立ち往生中" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-001.jpg" width="600"/>
        <br/>
        ゴミ回収車の前方でも、トラックがチェーンの取り付け真っ最中のようです。
        <br/>
        一時は、いろんなクルマが、この道路上で立ち往生していました。
       </p>
       <blockquote class="read">
        <strong>
         【追記】
        </strong>
        <br/>
        実は、これを書いている夜になっても、ゴミ回収車は道路上に駐車されたままになっているようです。
        <br/>
        きっと、この雪でどうにも動かすことができず、お手上げ状態で引き上げていったということなのでしょうか……。
       </blockquote>
       <p>
        住宅街のなかを歩いていたら、あちらこちらからスコップで道路を擦るシャリシャリという音が響いてくるんですよね。
        <br/>
        どこのおたくでも雪かきをしている、その音のようでした。
        <br/>
        <img alt="横浜で、雪かき" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-002.jpg" width="600"/>
        <br/>
        しかし、まさか横浜で雪かきをするようなことがあるとは、夢にも思っていませんでした。
       </p>
       <p>
        そして、住宅街の真ん中でもクルマが立ち往生していました。
        <br/>
        踏み込んでも踏み込んで空転するタイヤに、近所の人たちが集まってきて力を合わせて押しているんです。
        <br/>
        <img alt="立ち往生したクルマを、皆で力を合わせて動かす" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-003.jpg" width="600"/>
        <br/>
        （ギャー、カメラに雪が降り積もってきた！）
        <br/>
        それでもどうしようもない状況に、お父さんみたいな方が、「おい、家にある毛布持ってこい」……。
        <br/>
        なんというか、雪が降った世界では、積もった雪の美しさだけじゃなくて、こうしたヒトの優しさも目にする事ができるようです。
        <br/>
        恐るべし、雪景色。
       </p>
       <p>
        街なかに出てみると……おおう、なんですか、これは。
        <br/>
        もうね、auのお店では携帯をどうしようというのかと。
        <br/>
        いや、考えようによっては防水性やタフネス性ををアピールできる、絶好の機会なのかもしれませんね！
        <br/>
        ……って、あれ？
        <br/>
        そのアピールができるのは、カシオのスマホぐらいものじゃないのでしたっけ。
        <br/>
        <img alt="auショップの店先では、携帯がすっかり雪に埋もれてしまいました" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-004.jpg" width="600"/>
       </p>
       <p>
        しかし、この大雪でやっぱり大喜びなのは子どもたちなんでしょうね。
        <br/>
        小さい子にはまだちょっと歩きにくそうですが、それでも空き地などがあると立ち止まって雪遊びをしていました。
        <br/>
        <img alt="お母さんと一緒に歩く雪道" height="600" src="http://biboulog.com/archives/images/2013/01/20130114-005.jpg" width="450"/>
       </p>
       <p>
        一方、この子にとってはまだ、雪は未知の異質物なのかもしれませんね。
        <br/>
        <img alt="" height="400" src="http://biboulog.com/archives/images/2013/01/20130114-012.jpg" width="600"/>
        <br/>
        おそるおそる歩きながら、それでも時折雪をすくっては散らしてみたりして、とても興味津々のようなのでした。
        <br/>
        ああそうか、雪の上を歩く感触って、まるで片栗粉を踏みしめるような違和感があるから、この子はきっとその感触に慣れてないだけなのかもしれません。
       </p>
       <p>
        家族が揃ったら雪合戦ですな。
        <br/>
        こちらのお宅では主に、お母さん VS 男の子という様相を示していました。
        <br/>
        男の子の方が先に飽きちゃって家の中に入って行っちゃいましたが……。
        <br/>
        <img alt="" height="400" src="http://biboulog.com/archives/images/2013/01/20130114-013.jpg" width="600"/>
       </p>
       <p>
        こちらでは、女の子たちが雪だるまづくりに精を出していました。
        <br/>
        頭と胴体を手分けしてつくっているので、これはきっと、大きな雪だるまができるでしょうね。
        <br/>
        <img alt="" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-006.jpg" width="600"/>
       </p>
       <p>
        その向こう側では、男子たちが坂道を滑り降りる競争で大はしゃぎしてました。
        <br/>
        ここの道は普段、結構クルマが通るのですが、さすがに今日はなかなか通るクルマもないようです。
        <br/>
        安心して遊んでいるのでしょうか。
        <br/>
        <img alt="" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-007.jpg" width="600"/>
       </p>
       <p>
        しかしこの雪景色では、子どもたちだけじゃなくて、親も楽しんでいたんです。
        <br/>
        娘と一緒にそりを持ってきて遊んでいたお父さん、「そり楽しいよ！お父さんもどう？」と娘に言われて、ヒャッホー状態。
        <br/>
        娘よりも大きなリアクション、見せてくれてました。
        <br/>
        <img alt="" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-008.jpg" width="600"/>
       </p>
       <p>
        そして。
        <br/>
        やっぱりいちばん大はしゃぎするのは、やっぱりワンコでしょう。
        <br/>
        「イヌは喜び、庭駆けまわり」状態で、お散歩途中でもはしゃいでる、はしゃいでる……。
        <br/>
        ワンコって脚の裏が冷たくないのかなーとか思っていましたが、どうやら彼らにとって雪は、冷たいとか、寒いもの、という概念はなくて、ただ単に楽しい“おもちゃ”なんでしょうね。
       </p>
       <p>
        ……ってキミは、完全に雪のなかに埋もれちゃってしまってるじゃないですか！
        <br/>
        <img alt="" height="450" src="http://biboulog.com/archives/images/2013/01/20130114-009.jpg" width="600"/>
       </p>

       大手町まで箱根駅伝（復路）を見てきました
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月3日（木曜日） 23:02
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_26.php" title="観に行った！ 参加した！ そんなイベントの話題のあれこれ">

       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E5%A4%A7%E6%89%8B%E7%94%BA%E3%83%BB%E4%B8%B8%E3%81%AE%E5%86%85&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="大手町・丸の内 [41]">
        大手町・丸の内
       </a>
       ,
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E9%A7%85%E4%BC%9D&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="駅伝 [2]">

      <div class="entry-content clearfix">
       <p>
        昨日、家の近所を通る箱根駅伝の往路を見たことで、何なんでしょうね、変な炎が燃え上がったようなんです。
        <br/>
        往路も見たんだから、復路も見ないと！と張り切ってしまったのです。
        <br/>
        ただ、家の近所に見に行ったとしても、結局はブログには昨日とまったく同じことしか書けないわけですよね。
        <br/>
        それじゃあ、まったく面白くないんです。
       </p>
       <p>
        もうちょっと変わったこと書かなくちゃ……と思っていたら、おお、そうだ。
        <br/>
        もういっそのこと、大手町までゴールの様子を見に行ってみてはどうかしら。
        <br/>
        そう思いついてしまっては、居ても立ってもいられません。
        <br/>
        電車に乗っていそいそと、駅伝での一番のクライマックス、ゴールとなる大手町まで行ってきたのでした。
       </p>
       <p>
        ゴールは13時半ごろということで、余裕を持って12時半に行ったのですが……なんですか、これは。
        <br/>
        とにかく人の数がスゴイのなんの。
        <br/>
        ゴール前どころか、そこに向かう歩道からして、ずっとヒト、ヒト、ヒト、ヒト……で埋め尽くされているんです。
        <br/>
        <img alt="歩道を埋め尽くすヒト、ヒト、ヒト、ヒト……" height="600" src="http://biboulog.com/archives/images/2013/01/20130103-001.jpg" width="400"/>
       </p>
       <p>
        歩道には柵が設置されているのですが、人の波に押されて柵が倒れそうになったりして、もう大変。
        <br/>
        ライブ会場でのモッシュかという勢いなんです。
        <br/>
        下手すりゃ将棋倒し寸前のデインジャラスさなんですよ。
        <br/>
        警備員も「押さないでください」「後ろの通行場所を空けてください」とハンドマイクでガンガン注意しています。
        <br/>
        雰囲気、ピリピリしてます。
        <br/>
        <img alt="設置された柵が倒れそうになったりしていて、危険な状態に" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-002.jpg" width="600"/>
       </p>
       <p>
        そんななか、ゴール地点のスタッフたちは淡々と準備を進めています。
        <br/>
        全然知らなかったのですが、どうやら「ゴールテープの持ち方」はかなり重要なことのようなのです。
        <br/>
        テープにはスポンサーのロゴマークも入っているので、真正面のテレビカメラにちゃんと映らなければなりません。
        <br/>
        選手が美しくテープを切ることができるように、高さも大事です。
        <br/>
        <img alt="ゴールテープの持ち方は、スポンサーのロゴを美しく見せることも必要なんです" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-003.jpg" width="600"/>
       </p>
       <p>
        そのような理由からでしょうか。
        <br/>
        テープを両側で持つ係員に対して、ずっと高さの確認や、ロゴの位置の調整をしていたのでした。
        <br/>
        テレビで映るのはわずか一瞬なんですが、それでもここまでパーフェクトに調整をして万事整えているのですね。
        <br/>
        <img alt="ゴールテープの持ち方、練習中" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-004.jpg" width="600"/>
       </p>
       <p>
        向かいの工事現場でも、事務所から人がゾクゾクと姿を見せては様子を見ています。
        <br/>
        写真左上、工事現場の鉄骨の上には、ヘルメットをかぶったオジサンがカメラまで構えています。
        <br/>
        事務所には「ラジオ日本」の旗が出てますね。
        <br/>
        臨時のスタジオか何かなのかしら。
        <br/>
        <img alt="工事現場事務所からもたくさんの人が見ています" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-005.jpg" width="600"/>
       </p>
       <p>
        そして。
        <br/>
        時間は13時半ちょっと過ぎ、日本橋を経由して選手がゾクゾクと戻ってきています。
        <br/>
        まずは、箱根をスタートしてからずっとトップを独走してきたという、日本体育大学が安定したゴールインです。
        <br/>
        このさすがの安定性には、観客も安心して声援を投げ掛けていました。
        <br/>
        <img alt="" height="600" src="http://biboulog.com/archives/images/2013/01/20130103-006.jpg" width="400"/>
       </p>
       <p>
        しかしやはり観客としては筋書きのない、ドキドキ感を求めているのでしょう。
        <br/>
        この日、もっとも観客が沸いたのは、ゴール直前まで展開していた帝京大学と早稲田大学の接戦でした。
        <br/>
        最後の最後まで、いったいどちらが抜け出すのかまったく読めず、2人の選手がもつれ込むようにゴールした瞬間、歓声と悲鳴のどよめきが、地響きのように沸き起こったんです。
        <br/>
        <img alt="ゴール前までもつれ込んだ帝京大学と早稲田大学の接戦" height="600" src="http://biboulog.com/archives/images/2013/01/20130103-007.jpg" width="400"/>
        <br/>
        いやー、ぼくだって、この光景にはもうずっとドキドキしっぱなしだったんですよね。
       </p>
       <p>
        このあとにゴールしたのは明治大学の選手です。
        <br/>
        彼はゴールテープを切った瞬間に気力を使い果たしたのか、倒れこんでしまいました。
        <br/>
        <img alt="ゴール直後に倒れこむ明治大学の選手" height="600" src="http://biboulog.com/archives/images/2013/01/20130103-008.jpg" width="400"/>
       </p>
       <p>
        すぐさまチームメイトが駆けつけたのですが、順位を落としてしまったことを悔やむような表情が印象的でした。
        <br/>
        <img alt="順位を落としてしまったことを悔やむような表情が印象的だった明治大学" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-009.jpg" width="600"/>
       </p>
       <p>
        笑顔やガッツポーズで選手がゴールインする姿もいいですが、このように、ゴール後でしか見られないドラマを目の当たりにできるのも、直接見に行くことの醍醐味なんですね。
        <br/>
        この明治大学の選手のあとは、ずっとゴールした後の姿ばかり見てしまうようになってしまいました。
        <br/>
        きっと、選手にとっては競技そのものを見てもらいたいのでしょうが……申し訳ありません。
       </p>
       <p>
        笑顔でチームメイト称える法政大学。
        <br/>
        <img alt="笑顔でチームメイト称える法政大学" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-010.jpg" width="600"/>
       </p>
       <p>
        往路で棄権したために「オープン参加」となった城西大学のこの表情。
        <br/>
        <img alt="往路で棄権したために「オープン参加」となった城西大学の表情" height="600" src="http://biboulog.com/archives/images/2013/01/20130103-011.jpg" width="400"/>
       </p>
       <p>
        倒れこむ選手をサポートする東京農業大学。
        <br/>
        <img alt="倒れこむ選手をサポートする東京農業大学" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-012.jpg" width="600"/>
       </p>
       <p>
        選手を笑顔で称えながらサポートする日本大学。
        <br/>
        <img alt="" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-013.jpg" width="600"/>
       </p>
       <p>
        いやあ、このゴール後のこんな光景にはサブイボが出て止まりませんって。
        <br/>
        来年は、ゴールする姿よりも、その後を追っていってみようかしら……とか思ってしまうほどでした。
       </p>
       <p>
        こうして、すべての選手がゴールインしたと思ったら……早っ！
        <br/>
        みるみるうちに撤収をはじめていくのでした。
        <br/>
        <img alt="みるみるうちに撤収するゴール地点" height="400" src="http://biboulog.com/archives/images/2013/01/20130103-014.jpg" width="600"/>
        <br/>
        そりゃそうか。
        <br/>
        このゴールがある道路はずっと閉鎖しているので、早く通行できるようにしないといけないのですね。
        <br/>
        なかなかテレビでは見ることができない駅伝の舞台裏を、最後の最後まで楽しませてもらったのでした。
       </p>
       <p>
        来年もまた、見に行こうっと。
        <br/>
        いや、ちゃんと起きることができたならの話、なんですが。
       </p>

       家の近所で箱根駅伝（往路）を見てきました
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月2日（水曜日） 23:57
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_26.php" title="観に行った！ 参加した！ そんなイベントの話題のあれこれ">

       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E9%A7%85%E4%BC%9D&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="駅伝 [2]">
        駅伝
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        毎年、お正月に開催されている箱根駅伝なんですが、実はこれ、ルートが家の近所を通っているんですね。
        <br/>
        テレビでも何度か「おお、近所を走ってるぜ」と見ているんです。
        <br/>
        なので沿道まで見に行くこともできるのですが、実はこれがなかなか難しい。
        <br/>
        休みの日は宵っ張りになるため、通過している時間帯は、ちょうど寝ているときなんですよね。
        <br/>
        なので起きた頃にはもうすっかり通り過ぎている、とか、ひどいときにはもうゴールしたあと、ということもよくあるんです。
       </p>
       <p>
        いや、実際に1回だけ沿道まで見に行ったことはあるんですね。
        <br/>
        といっても、もう7年も前、
        <a href="http://shoko.jpn.org/read2006/read0601a.html#BM060103" target="_blank">
         2006年のこと
        </a>
        なんですよ。
        <br/>
        しかもこれ、午前中に通過する往路ではなく、昼頃に通過する復路のときなんですよね。
        <br/>
        （どうも前日は寝過ごしたらしく、「リベンジ」とか言ってるし）
       </p>
       <p>
        でもやっぱり見られるものは見に行ってみたい。
        <br/>
        だったら寝なかったらいいんじゃないか、その思いだけで、今年は本当に寝ずに待っていることにしました。
        <br/>
        フフフフ、その甲斐ありましたよ。
        <br/>
        テレビで状況を見ながら余裕を持って沿道に行くことができました。
        <br/>
        そのため、まだ沿道に人が少ないうちから、見やすいいい場所を確保することができたんです。
        <br/>
        沿道にいても、現在の状況は携帯ワンセグでバッチリとテレビ中継を見ることができるので、あとどれぐらいで通過することになるのか、心の準備まで万端に整えられるんですね。
        <br/>
        いやー、便利な世の中だ。
       </p>
       <p>
        ところで、テレビで見ていると、沿道で皆がお揃いの「新聞社の旗」を振っていますよね。
        <br/>
        あれはいったいどうやって手に入れるのでしょうか。
       </p>
       <p>
        それは選手が来る前に、こうして新聞社のワゴン車が配ってるんです。
        <br/>
        <img alt="応援旗（という名の新聞社の旗）を配って回るワゴン車" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-001.jpg" width="600"/>
        <br/>
        といっても、係員が降りてきて手渡しで配るわけじゃないのです。
        <br/>
        ワゴン車のスピーカーから「応援旗をお配りしています。沿道の皆さんでおまわしください」とアナウンスすると同時に、旗をまとめてドバッと下ろしていくだけなんですね。
        <br/>
        あとは沿道の観衆同士で助けあってなんとかしてくれ、と。
        <br/>
        ひょっとしてこの調子で、大手町から箱根まで旗を配り続けていくのかしら。
       </p>
       <p>
        また、広報車として、先日発表されたばかりの新型クラウンが使われていました。
        <br/>
        しかも、そのうちの1台は、なんと、記者発表でも度肝を抜かれた“ピンクのクラウン”なんですよ！
        <br/>
        これです、これ！
        <br/>
        <img alt="新型クラウンが広報車に使われていましたが……この色はないわー" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-002.jpg" width="600"/>
        <br/>
        うわー、うわー、うわー、うわー。
        <br/>
        実際に目で見たら、このあまりの常軌を逸した色彩感覚にクラクラきちゃいます。
        <br/>
        ええっと……、やっぱりクラウンにこの色は……、売れないと思います……。
       </p>
       <p>
        そうこうしているうちに白バイが！
        <br/>
        <img alt="お調子者がいないか、沿道の観衆をチェックする白バイ隊員" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-003.jpg" width="600"/>
        <br/>
        パトカーが！
        <br/>
        <img alt="お調子者がいないか、沿道の観衆をチェックするパトカー隊員" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-004.jpg" width="600"/>
        <br/>
        沿道の観衆をギロリと眺めながら通り過ぎていきます。
        <br/>
        車道にはみ出しているお調子者がいないかどうか、まずは偵察しているようなんですね。
        <br/>
        ちょっとでもお行儀が悪い人がいると、「車道に降りないでください。歩道に上がってください」と注意されています。
       </p>
       <p>
        そして……ああ、来ました！
        <br/>
        まずは中継車です。
        <br/>
        <img alt="テレビ中継の要となる中継車" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-005.jpg" width="600"/>
        <br/>
        そして、その後ろには、先頭のランナーを映すカメラを積んだトラック。
        <br/>
        <img alt="先頭走者を映しているテレビカメラ車" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-006.jpg" width="600"/>
       </p>
       <p>
        これら車両と、先導する白バイが通りすぎていくと……現れました！
        <br/>
        トップを独走している東洋大学の選手が……あっという間に通り過ぎて行きました。
        <br/>
        <img alt="トップは独走状態の東洋大学でした" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-007.jpg" width="600"/>
       </p>
       <p>
        しかし、あまりに安定した独走ぶりに、中継はどちらかというとその後ろがメインのようです。
        <br/>
        2位だった明治大学を、後ろから一気に追い上げてきた駒澤大学が抜かしているところのようです。
        <br/>
        <img alt="2位争いの接戦だった、駒澤大学と明治大学" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-008.jpg" width="600"/>
       </p>
       <p>
        その後も団子状態で、次々と選手が目の前をあっという間に駆け抜けていってしまうのですね。
        <br/>
        こんな感じが、もう何回も続くのです。
        <br/>
        <img alt="団子状態の選手たち" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-009.jpg" width="600"/>
        <br/>
        なので沿道で見ていると、テレビを見ているのと違って、いったい今はどこの大学が通り過ぎていって、それがいったい何位なのやら、よく判らないのですね。
        <br/>
        やっぱりテレビの見せ方はうまいものです。
       </p>
       <p>
        そして、最後の選手が通り過ぎていったところで、最後尾のパトカーが「交通規制は間もなく解除します。係員の指示に従ってください」とアナウンスして行き、これで終わりです。
       </p>
       <p>
        しかし“交通規制”といっても、道路を閉鎖するわけではないのですね。
        <br/>
        2車線のうちの歩道側を駅伝用に使用するだけであって、中央線側の車線は普通にクルマが走っています。
        <br/>
        ただし、ランナーのいる辺りはカメラ車やペース車、白バイなどによるブロックがあるため、自由にブンブン通ることもできずちょっとした渋滞となってしまっているのでした。
       </p>
       <p>
        こんな感じで白バイや大会関係車がブロックしているんです。
        <br/>
        <img alt="選手の後続には、通行をブロックする白バイや大会関係車" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-010.jpg" width="600"/>
       </p>
       <p>
        そのため、後ろはちょっとした渋滞となってしまっています。
        <br/>
        <img alt="駅伝通過直後のちょっとした渋滞" height="450" src="http://biboulog.com/archives/images/2013/01/20130102-011.jpg" width="600"/>
       </p>
       <p>
        やっぱり沿道まで見に来たら、テレビ中継を見ているだけでは判らない、臨場感や手作り感、運営の緊張感などが伝わってきて、なんというか、面白いものです。
        <br/>
        ああ、なんだかクセになりそうです。
       </p>

     <h2 class="entry-title" id="a005953">
      <a href="http://biboulog.com/archives/2013/01/02-020247.php" rel="bookmark">
       2013年、初日の出と初日の入り
      </a>
     </h2>
     <dl class="entry-header">
      <dt class="entry-date">
       2013年1月2日（水曜日） 02:02
      </dt>
      <dt class="entry-category">
       Category :
       <a href="http://biboulog.com/archives/cat_27.php" title="主宰人の日常的などーでもいい話題のあれこれ">
        日常
       </a>
      </dt>
      <dt class="entry-tags">
       Tags:
       <a href="http://biboulog.com/MT/mt-search.cgi?blog_id=1&amp;tag=%E5%88%9D%E6%97%A5%E3%81%AE%E5%87%BA&amp;limit=20&amp;IncludeBlogs=1" rel="tag" title="初日の出 [4]">
        初日の出
       </a>
      </dt>
     </dl>
     <div class="entry">
      <div class="entry-content clearfix">
       <p>
        やっぱりお正月は「1年の計は元旦にあり」ですよね。
        <br/>
        そして、さらに「初日の出」は、ニッポン人としての一大イベントじゃないでしょうか。
        <br/>
        ということで、除夜の汽笛に引き続いて、このイベントもクリアしようと、日の出を待つことにしたんです。
        <br/>
        寝ることをやめて、テレビをダラダラ観ながら夜更かしするなんて、本末転倒な気もしますが……いや、いいんです！
       </p>
       <p>
        ちなみに、暦で確認すると、横浜での1月1日の日の出時間は6時50分なんだそうです。
        <br/>
        ちょっと早いのですが、念のため6時40分頃には、マンション屋上に行ってスタンバイすることにしました。
       </p>
       <p>
        ……が、寒い。
        <br/>
        風がないのがまだ救いだったのですが、とにかく寒いのです。寒い。メチャ寒。
        <br/>
        明け方の空気がしんしんとカラダを蝕んで、10分と言えども、日の出時間前に屋上に来たのが悔やまれます。
       </p>
       <p>
        いやいや、こんな寒さも初日の出のご来光を拝んだら、吹き飛んじゃうさ！
        <br/>
        そう自分を励まし、また誤魔化しながら、いざその瞬間を待っていたのですが……オオウ。
        <br/>
        いざ、日の出時間となったそのとき、いったい何ということでしょう。
        <br/>
        <big>
         <strong>
          こんなことになってしまっていたのでした。
         </strong>
        </big>
        <br/>
        <img alt="残念な結果となってしまった初日の出" height="400" src="http://biboulog.com/archives/images/2013/01/20130101-001.jpg" width="600"/>
       </p>
       <p>
        空全体に雲はあまりなく、絶好の日和なのに、よりによって東の方角には雲が出ているんです（それも地表近くは厚く）。
        <br/>
        上っているらしい太陽は、見事にこの厚い雲に遮られてしまって、ご来光は僅かな雲の隙間からチョロ見せという、実に残念な初日の出タイムとなってしまったのでした。
       </p>
       <p>
        しかし、まあ、考えてみると、去年の初日の出は今年以上に残念だったんですね。
        <br/>
        何しろ、
        <a href="http://biboulog.com/archives/2012/01/02-013731.php">
         銚子の沖合まで行って船に乗って見に行った
        </a>
        のに、曇天でまったく見られなかったんですから……。
        <br/>
        そう考えると、まだご来光をチョロ見できた今年のほうがマシだったかもしれません。
       </p>
       <p>
        去年から今年に掛けて、初日の出の見え方が徐々によくなってきていると考えると、来年こそは完璧に美しいご来光を眺めることができそうです！
        <br/>
        よーし、来年は頑張るぞ～。
       </p>
       <p>
        ……とは言うものの、やっぱりなんだかシックリこないのですね。
       </p>
       <p>
        ああ、そうだ。
        <br/>
        初日の出はダメだったけど、その代わり、“初日の入り”が見られたらいいのじゃないでしょうかね。
        <br/>
        そもそも初日の出は「新しい年の新しい1日の幕開け」と考えると、“初日の入り”は「新しい年の新しい夜の幕開け」ですよね？
        <br/>
        だったら、日の入りを眺めて、新しい年の新しい夜の幕開け迎え入れるとともに、新しい年の1日を無事に過ごせたことへの感謝を捧げたらいいと思うのですよ。
       </p>
       <p>
        ……もう自分でも何を言っているのか判らなくなってきました（おい）。
        <br/>
        が、そこはそれ、とにかく“初日の入り”を無事に見ることはできたのでした。
        <br/>
        これで2013年はよしとしておきましょう。
        <br/>
        <img alt="残念な結果となってしまった初日の出" height="400" src="http://biboulog.com/archives/images/2013/01/20130101-002.jpg" width="600"/>
       </p>
