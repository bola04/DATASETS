     <h2>
      HTML Color Codes Theory
     </h2>
     <p>
      So you are wondering "Does this weird combination of letters and numbers have any meaning?" Well the answer is "Yes" and this is how it goes:)
      <br/>
      <br/>
      <b>
       HTML Codes format:
      </b>
      <br/>
      Each HTML code contains symbol "#" and 6 letters or numbers. These numbers are in hexadecimal numeral system. For example "FF" in hexadecimal represents number 255 in Decimal.
      <br/>
      <br/>
      <b>
       Meaning of symbols:
      </b>
      <br/>
      The first two symbols in HTML color code represents the intensity of
      <b>
       red
      </b>
      color. 00 is the least and FF is the most intense.
The third and fourth represents intensity of
      <strong>
       green
      </strong>
      and fifth and sixth represents the intensity of
      <strong>
       blue
      </strong>
      .
So with combining the intensity of red, green and blue we can mix almost any color that our heart desire;)
      <br/>
      <br/>
      <b>
       Examples:
      </b>
      <br/>
      <b style="color:#FF0000">
       #FF0000
      </b>
      - With this HTML code we tell browser to show maximum of red and no green and no blue.
The result is of course pure red color:
      <span style="background:#FF0000">
      </span>
      <br/>
      <br/>
      <b style="color:#00FF00">
       #00FF00
      </b>
      - This HTML code shows just green and no red and blue.
The result is:
      <span style="background:#00FF00">
      </span>
      <br/>
      <br/>
      <b style="color:#0000FF">
       #0000FF
      </b>
      - This HTML code shows just blue and no red and green.
The result is:
      <span style="background:#0000FF">
      </span>
      <br/>
      <br/>
      <b style="color:#FFFF00; background:#000000">
       #FFFF00
      </b>
      - Combination of red and green color gives us yellow:
      <span style="background:#FFFF00">
      </span>
      <br/>
      <br/>
      <b style="color:#cceeff; background:#000000">
       #CCEEFF
      </b>
      - Take some red a bit more of green and maximum of blue to get color of sky:
      <span style="background:#cceeff">
      </span>
      <br/>
     </p>
