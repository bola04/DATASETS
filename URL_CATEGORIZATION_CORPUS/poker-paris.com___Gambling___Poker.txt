           <strong>
            Impression plot : le flocage et la flexographie
           </strong>
           <br/>
           <br/>
           L'impression plot est un procédé d'impression de grande qualité qui offre d'excellents résultatss. L'imprimé résiste longtemps et les couleurs ne s'affadissent pas.
           <br/>
           <br/>
           Il existe 2 types d'impression plot : le flocage et la flexographie.
           <br/>
           <br/>
           Le flocage offre un toucher similaire au velours et le motif présente un léger relief.
           <br/>
           <br/>
           La flexographie offre quant à elle un fini lisse, très légèrement brillant, comme du vinyle. Les articles dont les motifs sont imprimés avec ces deux types d'impression sont d'un grand confort.
           <br/>
           <br/>
           <br/>
           <strong>
            Le transfert numérique :
           </strong>
           <br/>
           <br/>
           Le transfert numérique offre de très bons résultats d'impression: le choix des couleurs pour les motifs n'est pas limité et est diversifié et ses couleurs sont distinctes.
           <br/>
           <br/>
           Il est lisse et agréable à porter. Ce type d'impression vous offre une grande flexibilité
           <br/>
           <br/>
           Malheureusement, l'imprimé ne peut évidemment pas durer éternellement. Les couleurs partent peu à peu au lavage.
           <br/>
           <br/>
           <br/>
