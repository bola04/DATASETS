       </a>

	and MS/MS database searches. A selection of popular sequence databases

        <a href="./search_form_select.html">
         free service
        </a>
        is ideal for evaluation and searching smaller data sets. 
	For large scale and routine work, you'll want to

       <a href="./distiller.html">
        <h3 class="mascotDistiller">
         Mascot Distiller
        </h3>
       </a>
       <p>
        Mascot Distiller offers a single, intuitive interface 
	to native (binary) data files from Agilent, AB Sciex, 
	Bruker, Shimadzu, Thermo and Waters. Raw data can be 
	processed into high quality, de-isotoped peak lists. 
	Optional toolboxes support Mascot search result review, 
	de novo sequencing, batch processing and quantitation. 
	If not registered, Mascot Distiller can still be used 
	as a free project viewer, ideal for distributing search 
	and quantitation results to colleagues.
       </p>
