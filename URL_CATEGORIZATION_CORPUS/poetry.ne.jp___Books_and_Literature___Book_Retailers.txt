                 <td width="353">
                  <p>
                   <img align="BOTTOM" border="0" height="46" src="platinum/platinum2.gif" width="66"/>
                   <br/>
                   <br/>
                   <br/>
                   <br/>
                  </p>
                  <div>
                   石の影　　　木村ユウ
                   <br/>
                   <br/>
                   <br/>
                   空だけはつながっていて
                   <br/>
                   そして雲は
                   <br/>
                   流れていくのね
                   <br/>
                   <br/>
                   夕暮れの光線の中、短い草を手で触って
                   <br/>
                   小さな石を眺めていた
                   <br/>
                   その影は長く長く
                   <br/>
                   染まった草は冷たく
                   <br/>
                   君のことを考えていた
                   <br/>
                   <br/>
                   こうやって
                   <br/>
                   君の言葉を反芻しているうちに
                   <br/>
                   枝を通ってくる光も、消えていくだろう
                   <br/>
                   君を連れて
                   <br/>
                   <br/>
                   僕は
                   <br/>
                   突然胸が詰まり
                   <br/>
                   君の名前を何度も何度も
                   <br/>
                   <br/>
                   何度も
                   <br/>
                   <br/>
                   おやすみなさい、って
                   <br/>
                   なんだか哀しい気がするの
                   <br/>
                   眠ってしまったら
                   <br/>
                   二度と目が覚めないんじゃないかと怖いの
                   <br/>
                   だからずっと起きていました
                   <br/>
                   <br/>
                   落ちてきそうな空
                   <br/>
                   君の使う言葉
                   <br/>
                   答える声
                   <br/>
                   空が回って
                   <br/>
                   <br/>
                   なんとか僕は
                   <br/>
                   自分自身を押しとどめる
                   <br/>
                   無意識に手を伸ばす
                   <br/>
                   草が触れる
                   <br/>
                   強く閉じていた目を開くと
                   <br/>
                   茜に染まった丘
                   <br/>
                   <br/>
                   僕は草を撫で
                   <br/>
                   おさえていた涙落ちる
                   <br/>
                   ごめん
                   <br/>
                   <br/>
                   ごめんね
                   <br/>
                   もう、
                   <br/>
                   届かないよね
                   <br/>
                   <br/>
                   <br/>
                   <br/>
                   <p>
                   </p>
                   <p>
                    <br/>
                    トパァズ　　　木村ユウ
                    <br/>
                    <br/>
                    <br/>
                    色づく石の宿めは
                    <br/>
                    消えることがないの
                   </p>
                   <p>
                    決められた指に
                    <br/>
                    よりそう肌
                    <br/>
                    <br/>
                    甘みを帯びていくのがわかる
                    <br/>
                    熱くなる
                    <br/>
                    <br/>
                    空気がほしい
                    <br/>
                    ほしくない
                    <br/>
                    明るい空なんていらない
                    <br/>
                    歌なんてもともと聞こえていなかった
                    <br/>
                    <br/>
                    ───僕は、いつも待ってた
                    <br/>
                    君が現れるのを
                    <br/>
                    <br/>
                    そう？
                    <br/>
                    <br/>
                    僕は、言ったはずだけれどね
                    <br/>
                    人の別れが
                    <br/>
                    あまりに情けないと、いうことも
                    <br/>
                    <br/>
                    この石には…
                    <br/>
                    悲しい宿めがあるの
                    <br/>
                    決められた人以外へ渡るのなら
                    <br/>
                    その人を不幸に
                    <br/>
                    <br/>
                    そういうことは
                    <br/>
                    僕も認めざるを得ないな
                    <br/>
                    僕はただ───
                    <br/>
                    <br/>
                    私、聞きたかった
                    <br/>
                    どうして聞きたかった？
                    <br/>
                    <br/>
                    愛してしまっても、いいだろう？
                    <br/>
                    そう言って
                    <br/>
                    <br/>
                    もらえることを
                   </p>
                  </div>
