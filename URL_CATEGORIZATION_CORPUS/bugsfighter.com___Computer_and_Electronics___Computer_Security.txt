         <a href="https://www.bugsfighter.com/remove-aurora-ransomware-and-decrypt-aurora-files/">
          <img alt="" class="attachment-medium size-medium wp-post-image" height="211" sizes="(max-width: 300px) 100vw, 300px" src="http://bugsfighter.com/wp-content/uploads/2018/06/aurora-ransomware-1-300x211.png" srcset="http://bugsfighter.com/wp-content/uploads/2018/06/aurora-ransomware-1-300x211.png 300w, http://bugsfighter.com/wp-content/uploads/2018/06/aurora-ransomware-1-768x540.png 768w, http://bugsfighter.com/wp-content/uploads/2018/06/aurora-ransomware-1.png 800w" width="300"/>

        <p>
         <strong>
          Aurora Ransomware
         </strong>
         (sometimes called
         <strong>
          OneKeyLocker Ransomware
         </strong>
         ) is new crypto-virus, that started circulating the web since the end of May, 2018. It uses DES algorithm to encode files and adds
         <strong>
          .Aurora
         </strong>
         extension, after which it got its name. After encryption ransomware creates several text files
         <strong>
          HOW_TO_DECRYPT_YOUR_FILES.txt
         </strong>
         , newest version creates single
         <strong>
          #RECOVERY-PC#.txt
         </strong>
         file, containing ransom note with contact information and instructions. Usually, viruses of this type ask for $100 – $500 in BitCoins. At the moment, there are no public decryption tool available. Full recovery is only possible with help of backups. You can preserve your files till actual decryptor will be created. Some data can possibly be restored using instructions on this page.
        </p>

         <a href="https://www.bugsfighter.com/remove-bip-ransomware-and-decrypt-bip-files/">
          <img alt="" class="attachment-medium size-medium wp-post-image" height="164" sizes="(max-width: 300px) 100vw, 300px" src="http://bugsfighter.com/wp-content/uploads/2018/06/bip-ransomware-300x164.png" srcset="http://bugsfighter.com/wp-content/uploads/2018/06/bip-ransomware-300x164.png 300w, http://bugsfighter.com/wp-content/uploads/2018/06/bip-ransomware-768x420.png 768w, http://bugsfighter.com/wp-content/uploads/2018/06/bip-ransomware.png 800w" width="300"/>
          <div class="clearfix">
          </div>
         </a>
        </div>
        <p>
         <strong>
          Bip Ransomware
         </strong>
         is another successor of Dharma/Crysis Ransomware family. New variation adds complex suffix, that ends with
         <strong>
          .bip
         </strong>
         extension, to all affected files. Bip Ransomware encrypts almost all types of files, that can be valuable to users, such as documents, images, videos, databases, archives, project files, etc.  It is currently unknown, what type of encryption algorithm Bip Ransomware uses, but probably it is AES. Bip Ransomware usually demands from $1000 to $2000 in BitCoins for the decryption key. However, often hackers don’t send any keys and it is not recommended to pay the ransom. As for today, the 5-th of May 2018, decryption is not possible, however, you can attempt to decrypt your files from backups or trying file recovery software.
        </p>
