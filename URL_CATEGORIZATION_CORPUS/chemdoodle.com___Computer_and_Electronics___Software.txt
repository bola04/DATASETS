         <sup>

ChemDoodle works on all operating systems, has one of the largest feature sets in the industry, has the most customizable graphics, pastes scalable vector graphics and provides round trip editing into Microsoft Office, iWork and OpenOffice (among others), and can read and write all of our competitors’ formats.         </h1>
       </div>
       <article class="post-detail">
        <div class="entry">
         <h2>
          What is new in ChemDoodle 9
         </h2>
         <ol>
          <li>
           A
           <strong>
            <a href="/downloads/UserGuide.pdf">
             new user manual
            </a>
           </strong>
           discusses all the new features in detail over several pages, too many to list here. (
           <a href="/downloads/UserGuide.pdf">
            click to load manual
           </a>
           , section 1.2)
          </li>
          <li>
           <strong>
            Drawing and Graphics
           </strong>
           – Tons of new systems for making your graphics quicker. Auto-placement of attributes (charges/radicals/stereocenters/etc.). An improved text tool that can create both atom text and formatted captions. Draw chiral carbon nanotubes in addition to zigzag and armchair. New dynamic brackets and structure highlights. Better drawing tools for advanced figures.
          </li>
          <li>
           <strong>
            Chemistry
           </strong>
           – State-of-the-art implementation of the most recent CIP rules. A clearer and more powerful warning system. Advanced implicit hydrogen handling including the analysis of advanced aromatic resonance systems. Full support for the latest elements as defined by
           <strong>
            IUPAC
           </strong>
           and much more!
          </li>
          <li>
           <strong>
            Interface
           </strong>
           – A brand new customizable cursor system, improved
           <strong>
            IUPAC
           </strong>
           name-to-structure interface. Improved color palettes, now with
           <strong>
            Rasmol
           </strong>
           ,
           <strong>
            CPK
           </strong>
           and
           <strong>
            Custom
           </strong>
           color sets. HTTPS support for
           <strong>
            PubChem
           </strong>
           is now implemented for access in
           <strong>
            MolGrabber
           </strong>
           . Improved color choosers including alpha support and high resolution improvements across the entire application.
          </li>
          <li>
           <strong>
            Chemical Files
           </strong>
           – The
           <strong>
            Nature
           </strong>
           style sheet has been added.
           <strong>
            SMILES
           </strong>
           interpretation has seen significant work, with a focus on very advanced cheminformatics techniques. Added support for the
           <strong>
            RCSB MacroMolecular Transmission Format (MMTF)
           </strong>
           . More support for
           <strong>
            ChemDraw
           </strong>
           ,
           <strong>
            MDL CT
           </strong>
           ,
           <strong>
            MRV
           </strong>
           and
           <strong>
            ISIS/Sketch
           </strong>
           files.
          </li>
          <li>
           <strong>
            Images
           </strong>
           –
           <strong>
            TIFF
           </strong>
           images can now be exported with custom DPI settings.
           <strong>
            GIF
           </strong>
           image output can now have semi-transparent pixels merged with white. Added
           <em>
            viewBox
           </em>
           attribute for
           <strong>
            SVG
           </strong>
           . When saving files, you can now use alternate extensions and other image file chooser improvements. Control which image file types are shown in the save image choosers.
          </li>
          <li>
           <strong>
            Vector Art
           </strong>
           – New glassware graphics have been added as well as dozens of new BioArt.
          </li>
          <li>
           <strong>
            Customizability
           </strong>
           – The keyboard and tools shortcuts are now fully customizable by the user. The user settings folder location can now be controlled. Custom attribute names and values are now persisted through restarts.
          </li>
          <li>
           <strong>
            Windows
           </strong>
           – Full support for high-DPI screens, without the manual scaling required in the past. The OLE plugin has been rebuilt for the most current compliance with
           <strong>
            Windows
           </strong>
           libraries.
          </li>
          <li>
           <strong>
            macOS
           </strong>
           – Improved and full Retina support. Native file choosers.
          </li>
         </ol>
