      <div class="entry-summary">
       <p>
        うちの息子は学習塾に通っていて、娘は英語教室に通っています。３月までは息子も同じ英語教室に通っていたのですが、本人の希望と昨年、目標にしていた英語検定に合格したことで３月いっぱいで辞めることにしました。学習塾は自宅から自転車で通えるので自分で通っていますが、英語教室は自宅から車で約１０分ほどの場所にあり、私が送迎をしていました。レッスンの曜日は先生が他の生徒さんとの兼ね合いを考えてから決める事となっており、２年前までは同じ木曜日だったのですが昨年は娘が水曜日、息子が木曜日と別々になり送迎も大変でした。
        <br/>
        レッスンは１時間なので近くで待つこともできますが、ちょうど夕飯を準備する時間帯なのでほとんど自宅へ戻りまた迎えに行くというスタイルで週に２日は常にバタバタとしていました。でも、４月からは息子の送迎がなくなったので週に１度の送迎だけとなり負担が減っています。気持ち的にも余裕が出ましたし、お金の面でも少し負担が減りました。

      <!-- end .entry-meta -->
      <div class="entry-summary">
       <p>
        今日、小学生の息子を美容院に連れて行きました。
        <br/>
        今日の息子のヘアカットを担当してくださった美容師さんとはしばらくぶりの対面でした。
        <br/>
        ２か月前に行った時はその美容師さんが旅行中ということで違う方にカットしてもらったのです。
        <br/>
        久々に息子に会った美容師さんが私に、「お兄ちゃんのほうじゃないですよね？」と聞いてきました。
        <br/>
        次男がここ２か月ぐらいでものすごく身長が伸びたからでした。
        <br/>
        「いえいえ、長男は高校生なので、もっと背が高いですよ」と私は答えました。
        <br/>
        しかし、美容師さんがそういうのも無理ありません。
        <br/>
        次男は最近、声変わりもしてしまったし、１６０センチの身長の私よりも１０センチ近く背が高くなってしまったのです。
        <br/>
        子供が成長してくれるのはありがたいことですが、かわいい小学生の息子がだんだん、大きくなるのがちょっと寂しいような気もする私です。
        <br/>
        長男は次男よりも成長が早かったのですが、まだちびっこの次男がいたので、長男の成長を寂しいと感じたことがなかったのですが、ようやく、こういうことを言っていたママ友の心境がわかりました。

       <span class="sep">

       </span>
       <a href="http://www.mergeconverter.com/%e3%81%8a%e6%b4%97%e6%bf%af%e3%81%af%e6%b4%97%e5%89%a4%e9%81%b8%e3%81%b3%e3%81%8b%e3%82%89%e5%a7%8b%e3%81%be%e3%82%8a%e3%81%be%e3%81%99/#respond">

      <!-- end .entry-meta -->
      <div class="entry-summary">
       <p>
        元々家事があまり得意ではなかったので、結婚してからは色々と困ることがありました。洗濯も基本がわからないので、とりあえず全部洗濯機にぶち込んで洗うだけ。終わったら適当に干すだけです。そのため、しっかり洗えていなかったり、石けんかすが残っているようなことも…下着も実家にいる時より早く傷んでしまうようになりましたし、ちょっと困りました。そこで、一度インターネットや本などでしっかり覚えるようにしたんです。
        <br/>
        中でも単純だけど参考になるなと思ったのがお洗濯用の洗剤選びです。普段使いであれば液体洗剤を。少々詰め込んでしまっても大丈夫。ただし、汚れが目立つ時には粉の洗剤の方が洗浄力が強いので便利です。粉の洗剤は洗濯物がぎゅうぎゅうに詰め込まれた状態ですと、洗濯物に残ってしまいますよね。そのため、粉洗剤の時はゆとりのある状態で回すこと。もちろんおしゃれ着は専用の洗剤を使い、それだけで洗います。この3つを揃えておくだけで、かなり洗濯が楽しくそしてストレスフリーになりますよ。
