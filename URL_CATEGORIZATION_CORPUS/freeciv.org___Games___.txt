     <h2>
      Project news
     </h2>
     <ul>
      <li>
       <b>
        8th Apr 2018
       </b>
       - A group of volunteers has set up a new instance of Freeciv-web on a new server,
       <a href="https://www.freecivweb.org/">
        freecivweb.org
       </a>
       . It does not have the user accounts and saved games from play.freeciv.org, and not everything works yet; so far only the Longturn multiplayer game is working.
      </li>
      <li>
       <b>
        7th Apr 2018
       </b>
       - The
       <a href="http://www.freeciv.org/wiki/NEWS-2.6.0-beta3">
        third beta of Freeciv 2.6.0
       </a>
       has been released.
      </li>
      <li>
       <b>
        24th Mar 2018
       </b>
       -
       <a href="http://www.freeciv.org/wiki/NEWS-2.5.11">
        Freeciv 2.5.11
       </a>
       fixes a regression in 2.5.10: a crash in the Windows build when resizing a dialog.
      </li>
      <li>
       <p>
        <b>
         3rd Mar 2018
        </b>
        - The web version of Freeciv, play.freeciv.org, has been shut down. The sole volunteer who was running the service is no longer willing to continue doing so.
       </p>
       <p>
        We haven't identified anyone else on the (very small) Freeciv team who has the skills and time to run this service hosted on freeciv.org, so you should expect it to be down for the foreseeable future.
       </p>
      </li>
      <li>
       <b>
        20th Jan 2018
       </b>
       - Two new releases:
       <a href="http://www.freeciv.org/wiki/NEWS-2.5.10">
        Freeciv 2.5.10
       </a>
       (a bugfix release) and
       <a href="http://www.freeciv.org/wiki/NEWS-2.6.0-beta2">
        2.6.0-beta2
       </a>
       .
      </li>
      <li>
       <b>
        16th Sep 2017
       </b>
       - The first beta of
       <a href="http://www.freeciv.org/wiki/NEWS-2.6.0">
        Freeciv 2.6.0
       </a>
       has been released.
       <a href="download.html#beta">
        Download
       </a>
       it and report any
       <a href="http://www.freeciv.org/wiki/Bugs">
        bugs
       </a>
       you spot.
      </li>
     </ul>
